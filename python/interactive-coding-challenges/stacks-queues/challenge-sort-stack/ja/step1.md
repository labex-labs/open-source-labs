# スタックをソートする

## 問題

与えられたスタックを昇順にソートする関数を書きなさい。バッファとして別のスタックを使用できます。ソートされたスタックは、最上部に最大の要素がある必要があります。

## 要件

この問題を解くには、次の要件を考慮する必要があります。

- 最大の要素は、ソートされたスタックの最上部にある必要があります。
- スタックには重複する値があり得ます。
- この問題に対して既に使用できるスタッククラスがあると仮定できます。
- スタックはメモリに収まると仮定できます。

## 使い方の例

この関数がどのように動作するかのいくつかの例を示します。

- 空のスタック -> なし
- 1つの要素のスタック -> 同じ要素
- 2つ以上の要素のスタック（一般的な場合） -> 昇順にソートされたスタック
- 既にソートされたスタック -> 同じスタック
