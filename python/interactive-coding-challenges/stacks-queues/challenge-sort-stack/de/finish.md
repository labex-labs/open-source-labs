# Zusammenfassung

In dieser Herausforderung haben wir gelernt, wie man einen Stapel mit einem anderen Stapel als Puffer sortiert. Wir haben auch die Anforderungen und Beispiele dafür gesehen, wie die Funktion verhalten sollte. Indem wir diesen Leitlinien folgen, können wir eine Funktion schreiben, die einen Stapel in aufsteigender Reihenfolge sortiert.
