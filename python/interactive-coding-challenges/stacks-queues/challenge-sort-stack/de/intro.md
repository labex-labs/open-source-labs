In der Informatik ist ein Stapel (Stack) eine Datenstruktur, die das Prinzip Last In First Out (LIFO) befolgt. Ein Stapel sortieren bedeutet, seine Elemente in einer bestimmten Reihenfolge zu ordnen. In dieser Herausforderung werden wir einen Stapel mit einem anderen Stapel als Puffer sortieren.
