# Stapel sortieren

## Problem

Gegeben einen Stapel, schreiben Sie eine Funktion, um ihn in aufsteigender Reihenfolge zu sortieren. Sie können einen anderen Stapel als Puffer verwenden. Der sortierte Stapel sollte das größte Element oben haben.

## Anforderungen

Um dieses Problem zu lösen, müssen wir die folgenden Anforderungen berücksichtigen:

- Das größte Element sollte oben auf dem sortierten Stapel sein.
- Der Stapel kann doppelte Werte enthalten.
- Wir können davon ausgehen, dass wir bereits eine Stapelklasse haben, die für dieses Problem verwendet werden kann.
- Wir können davon ausgehen, dass der Stapel in den Arbeitsspeicher passt.

## Beispielverwendung

Hier sind einige Beispiele, wie die Funktion verhalten sollte:

- Leerer Stapel -> Kein Ergebnis
- Stapel mit einem Element -> Das gleiche Element
- Stapel mit zwei oder mehr Elementen (allgemeiner Fall) -> Sortierter Stapel in aufsteigender Reihenfolge
- Bereits sortierter Stapel -> Der gleiche Stapel
