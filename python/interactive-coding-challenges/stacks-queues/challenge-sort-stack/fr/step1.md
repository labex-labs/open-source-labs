# Trier une pile

## Problème

Étant donné une pile, écrire une fonction pour la trier par ordre croissant. Vous pouvez utiliser une autre pile comme tampon. La pile triée devrait avoir l'élément le plus grand en haut.

## Exigences

Pour résoudre ce problème, nous devons prendre en compte les exigences suivantes :

- L'élément le plus grand devrait être en haut de la pile triée.
- La pile peut avoir des valeurs dupliquées.
- Nous pouvons supposer qu'il existe déjà une classe pile qui peut être utilisée pour ce problème.
- Nous pouvons supposer que la pile tient dans la mémoire.

## Utilisation de l'exemple

Voici quelques exemples de comportement attendu de la fonction :

- Pile vide -> Aucun
- Pile à un élément -> Le même élément
- Pile à deux éléments ou plus (cas général) -> Pile triée par ordre croissant
- Pile déjà triée -> La même pile
