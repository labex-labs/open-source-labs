# Ordenar una pila

## Problema

Dada una pila, escribe una función para ordenarla en orden ascendente. Puedes utilizar otra pila como búfer. La pila ordenada debe tener el elemento más grande en la parte superior.

## Requisitos

Para resolver este problema, debemos considerar los siguientes requisitos:

- El elemento más grande debe estar en la parte superior de la pila ordenada.
- La pila puede tener valores duplicados.
- Podemos suponer que ya tenemos una clase de pila que se puede utilizar para este problema.
- Podemos suponer que la pila cabe en la memoria.

## Uso de ejemplo

A continuación se presentan algunos ejemplos de cómo debe comportarse la función:

- Pila vacía -> Ninguno
- Pila con un elemento -> El mismo elemento
- Pila con dos o más elementos (caso general) -> Pila ordenada en orden ascendente
- Pila ya ordenada -> La misma pila
