# Introducción

En informática, una pila es una estructura de datos que sigue el principio de último en, primero en salir (LIFO, por sus siglas en inglés). Ordenar una pila significa organizar sus elementos en un orden específico. En este desafío, ordenaremos una pila utilizando otra pila como búfer.
