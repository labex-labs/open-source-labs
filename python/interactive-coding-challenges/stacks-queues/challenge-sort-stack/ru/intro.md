# Введение

В информатике стек - это структура данных, которая работает по принципу «последним пришёл — первым вышел» (LIFO). Сортировка стека означает упорядочивание его элементов в определённом порядке. В этом испытании мы отсортируем стек, используя другой стек в качестве буфера.
