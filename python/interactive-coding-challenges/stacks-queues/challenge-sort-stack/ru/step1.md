# Отсортировать стек

## Задача

Дан стек. Напишите функцию для его сортировки по возрастанию. Вы можете использовать другой стек в качестве буфера. Отсортированный стек должен иметь наибольший элемент в вершине.

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Наибольший элемент должен быть в вершине отсортированного стека.
- В стеке могут быть дублирующиеся значения.
- Можно предположить, что у нас уже есть класс стека, который можно использовать для этой задачи.
- Можно предположить, что стек помещается в память.

## Пример использования

Ниже приведены примеры того, как должна работать функция:

- Пустой стек -> None
- Стек с одним элементом -> То же самое элемент
- Стек с двумя или более элементами (общий случай) -> Отсортированный стек по возрастанию
- Уже отсортированный стек -> То же самое стек
