# 简介

在计算机科学中，栈是一种抽象数据类型，它作为一组元素的集合，有两个主要操作：push（将一个元素添加到集合中）和 pop（移除最近添加且尚未移除的元素）。其他操作包括 peek（返回最近添加的元素而不移除它）和 is_empty（检查栈是否为空）。
