# Введение

В информатике стек - это абстрактный тип данных, представляющий собой коллекцию элементов с двумя основными операциями: push (добавление элемента в коллекцию) и pop (удаление наиболее недавно добавленного элемента, который еще не был удален). Другие операции включают peek (возвращение наиболее недавно добавленного элемента без его удаления) и is_empty (проверка, пуст ли стек).
