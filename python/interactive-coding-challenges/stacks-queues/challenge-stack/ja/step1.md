# スタック

## 問題

Python で連結リストを使ってスタックを実装し、以下のメソッドを持つこと：

- push: スタックのトップに要素を追加する
- pop: スタックのトップの要素を削除して返す。スタックが空の場合、None を返す。
- peek: スタックのトップの要素を削除せずに返す。スタックが空の場合、None を返す。
- is_empty: スタックが空の場合は True を、それ以外の場合は False を返す。

## 要件

以下の要件を満たすこと：

- 空のスタックから要素を削除する場合、None を返す。
- 実装は連結リストを使うこと。
- 実装は Python で行うこと。
- 実装は push、pop、peek、is_empty の 4 つのメソッドを含むこと。

## 使い方の例

### プッシュ

- 空のスタックに要素を追加する：stack.push(1)
- 要素があるスタックに要素を追加する：stack.push(2)

### ポップ

- 空のスタックから要素を削除する：stack.pop() -> None
- 要素が 1 つのスタックから要素を削除する：stack.pop() -> 1
- 要素が複数あるスタックから要素を削除する：stack.pop() -> 2

### ピーク

- 空のスタックのピークを取得する：stack.peek() -> None
- 要素が 1 つ以上あるスタックのピークを取得する：stack.peek() -> 2

### 空かどうかの判定

- 空のスタックの場合：stack.is_empty() -> True
- 要素が 1 つ以上あるスタックの場合：stack.is_empty() -> False
