# Резюме

В этом испытании мы реализовали класс SetOfStacks, который оборачивает список стеков, где каждый стек имеет ограниченную емкость. Класс предоставляет функциональные возможности для добавления и удаления элементов из стеков и автоматически создает или удаляет стеки по необходимости. Реализация соответствует требованиям использования существующего класса стека, ограничения всех стеков одинаковой емкостью и помещения в память. Класс SetOfStacks можно использовать в различных сценариях, таких как добавление и удаление элементов из пустых или непустых стеков, создание новых стеков при достижении емкости и уничтожение стеков, когда они становятся пустыми.
