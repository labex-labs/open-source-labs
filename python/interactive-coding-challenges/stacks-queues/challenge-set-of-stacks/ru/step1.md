# Набор стеков

## Задача

Реализуйте класс SetOfStacks, который оборачивает список стеков, где каждый стек имеет ограниченную емкость. Класс должен обладать следующими функциональными возможностями:

- Добавьте элемент на вершину последнего стека в списке. Если последний стек заполнен, создайте новый стек и добавьте элемент в новый стек.
- Удалите верхний элемент с последнего стека в списке. Если последний стек пуст, удалите его из списка и удалите верхний элемент с нового последнего стека в списке. Если список пуст, верните None.
- Удалите элемент из конкретного стека в списке. Если стек пуст, удалите его из списка. Если список пуст, верните None.

## Требования

Класс SetOfStacks должен соответствовать следующим требованиям:

- Класс должен использовать существующий класс стека.
- Все стеки в списке должны иметь одинаковую емкость.
- Если стек заполнится, новый стек должен создаваться автоматически для хранения дополнительных элементов.
- Если стек станет пустым, он должен быть удален из списка.
- Если мы пытаемся удалить элемент с пустого стека, метод должен вернуть None.
- Реализация должна помещаться в память.

## Пример использования

Класс SetOfStacks можно использовать в следующих сценариях:

- Добавление и удаление элементов из пустого стека.
- Добавление и удаление элементов из непустого стека.
- Добавление элементов в стек, достигший своей емкости, для создания нового.
- Удаление элементов из стека для его уничтожения.
