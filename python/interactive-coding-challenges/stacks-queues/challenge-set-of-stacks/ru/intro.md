# Введение

В информатике стек - это абстрактный тип данных, представляющий собой коллекцию элементов с двумя основными операциями: push (добавление элемента в коллекцию) и pop (удаление наиболее недавно добавленного элемента, который еще не был удален). В некоторых случаях может потребоваться реализовать набор стеков, где каждый стек имеет ограниченную емкость. Когда стек достигает своей емкости, создается новый стек для хранения дополнительных элементов. В этом испытании мы реализуем класс SetOfStacks, который оборачивает список стеков, где каждый стек ограничен емкостью.
