# スタックのセット

## 問題

各スタックが容量によって制限されるスタックのリストをラップするSetOfStacksクラスを実装します。このクラスは次の機能を持つ必要があります。

- 要素をリストの最後のスタックのトップにプッシュします。最後のスタックが満杯の場合、新しいスタックを作成して要素を新しいスタックに追加します。
- リストの最後のスタックからトップ要素をポップします。最後のスタックが空の場合、それをリストから削除し、リストの新しい最後のスタックからトップ要素をポップします。リストが空の場合、Noneを返します。
- リスト内の特定のスタックから要素をポップします。スタックが空の場合、それをリストから削除します。リストが空の場合、Noneを返します。

## 要件

SetOfStacksクラスは次の要件を満たす必要があります。

- クラスは既存のスタッククラスを使用する必要があります。
- リスト内のすべてのスタックは同じ容量によって制限される必要があります。
- スタックが満杯になると、追加要素を格納するために自動的に新しいスタックが作成される必要があります。
- スタックが空になると、それがリストから削除される必要があります。
- 空のスタックでポップする場合、メソッドはNoneを返す必要があります。
- 実装はメモリに収まる必要があります。

## 例の使用法

SetOfStacksクラスは次のシナリオで使用できます。

- 空のスタックでのプッシュとポップ。
- 空でないスタックでのプッシュとポップ。
- 容量スタックにプッシュして新しいスタックを作成する。
- スタックでポップして破棄する。
