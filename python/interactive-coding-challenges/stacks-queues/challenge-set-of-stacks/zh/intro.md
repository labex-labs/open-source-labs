# 简介

在计算机科学中，栈是一种抽象数据类型，它作为一组元素的集合，有两个主要操作：push（将一个元素添加到集合中）和 pop（移除最近添加且尚未移除的元素）。在某些情况下，我们可能需要实现一组栈，其中每个栈都有有限的容量。当一个栈达到其容量时，会创建一个新栈来存储额外的元素。在这个挑战中，我们将实现一个 `SetOfStacks` 类，它包装了一个栈列表，其中每个栈都有一个容量限制。
