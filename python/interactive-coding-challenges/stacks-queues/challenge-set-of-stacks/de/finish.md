# Zusammenfassung

In dieser Herausforderung haben wir eine SetOfStacks-Klasse implementiert, die eine Liste von Stapeln umschließt, wobei jeder Stapel durch eine Kapazität begrenzt ist. Die Klasse bietet Funktionalitäten zum Hinzufügen und Entfernen von Elementen auf und von den Stapeln und erstellt oder entfernt automatisch Stapeln, wenn erforderlich. Die Implementierung erfüllt die Anforderungen, eine vorhandene Stapelklasse zu verwenden, alle Stapel durch die gleiche Kapazität zu begrenzen und im Speicher zu passen. Die SetOfStacks-Klasse kann in verschiedenen Szenarien verwendet werden, wie zum Beispiel das Hinzufügen und Entfernen von Elementen auf leeren oder nicht-leeren Stapeln, das Erstellen neuer Stapel, wenn die Kapazität erreicht ist, und das Zerstören von Stapeln, wenn sie leer werden.
