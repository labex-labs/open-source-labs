# 简介

在计算机科学中，队列是一组按顺序维护的实体集合，可通过在序列一端添加实体以及从序列另一端移除实体来进行修改。它也被称为先进先出（FIFO，First In, First Out）数据结构。在本次挑战中，我们将使用链表来实现一个队列。
