# Einführung

In der Informatik ist eine Warteschlange eine Sammlung von Entitäten, die in einer Sequenz aufbewahrt werden und durch das Hinzufügen von Entitäten am einen Ende der Sequenz und das Entfernen von Entitäten am anderen Ende der Sequenz geändert werden können. Sie ist auch als FIFO (first in, first out) - Datenstruktur bekannt. In dieser Herausforderung werden wir eine Warteschlange mit einer verketteten Liste implementieren.
