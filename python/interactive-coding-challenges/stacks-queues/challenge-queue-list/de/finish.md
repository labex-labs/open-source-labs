# Zusammenfassung

In dieser Herausforderung haben wir gelernt, wie man eine Warteschlange mit einer verketteten Liste implementiert. Wir haben auch die Anforderungen an die Implementierung einer Warteschlange sowie die verschiedenen Szenarien kennengelernt, die auftreten können, wenn Elemente in die Warteschlange eingefügt oder aus ihr entfernt werden.
