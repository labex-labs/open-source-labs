# キューリスト

## 問題

連結リストを使って、エンキューとデキューメソッドを持つキューを実装します。エンキューメソッドはキューの末尾に要素を追加し、デキューメソッドはキューの先頭から要素を削除する必要があります。キューが空の場合、デキューは None を返す必要があります。

## 要件

キューを実装するには、次の要件に従う必要があります。

- リストに 1 つの項目がある場合、最初と最後のポインタは両方ともそれにポイントしている必要があります。
- リストに項目がない場合、最初と最後のポインタは None でなければなりません。
- 空のキューでデキューすると、None を返す必要があります。
- これがメモリに収まることを前提として構いません。

## 例の使用法

### エンキュー

- 空のキューにエンキューする：キューが空の場合、エンキューメソッドは要素をキューの最初と最後の要素として追加する必要があります。
- 空でないキューにエンキューする：キューが空でない場合、エンキューメソッドは要素をキューの末尾に追加する必要があります。

### デキュー

- 空のキューをデキューする -> None：キューが空の場合、デキューメソッドは None を返す必要があります。
- 1 つの要素を持つキューをデキューする：キューに要素が 1 つだけの場合、デキューメソッドは要素を削除し、最初と最後のポインタを None に設定する必要があります。
- 1 つ以上の要素を持つキューをデキューする：キューに 1 つ以上の要素がある場合、デキューメソッドは最初の要素を削除し、最初のポインタを次の要素に設定する必要があります。
