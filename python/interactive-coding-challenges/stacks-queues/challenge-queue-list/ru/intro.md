# Введение

В информатике очередь - это коллекция элементов, которая хранится в определенном порядке и может быть модифицирована путем добавления элементов в один конец последовательности и удаления элементов из другого конца последовательности. Она также называется структурой данных FIFO (first in, first out - "первым пришел - первым вышел"). В этом испытании мы реализуем очередь с использованием связанного списка.
