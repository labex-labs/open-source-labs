# Очередь в виде списка

## Задача

Реализовать очередь с методами enqueue (добавление элемента в очередь) и dequeue (удаление элемента из очереди) с использованием связанного списка. Метод enqueue должен добавлять элемент в конец очереди, а метод dequeue должен удалять элемент из начала очереди. Если очередь пуста, метод dequeue должен возвращать None.

## Требования

Для реализации очереди необходимо следовать следующим требованиям:

- Если в списке есть один элемент, указатели first и last должны указывать на него.
- Если в списке нет элементов, указатели first и last должны быть равны None.
- Если вызывать метод dequeue для пустой очереди, он должен вернуть None.
- Можно предположить, что это помещается в память.

## Пример использования

### Добавление элемента в очередь (enqueue)

- Добавление элемента в пустую очередь: Если очередь пуста, метод enqueue должен добавить элемент как первый и последний элемент очереди.
- Добавление элемента в непустую очередь: Если очередь не пуста, метод enqueue должен добавить элемент в конец очереди.

### Удаление элемента из очереди (dequeue)

- Удаление элемента из пустой очереди -> None: Если очередь пуста, метод dequeue должен вернуть None.
- Удаление элемента из очереди с одним элементом: Если в очереди есть только один элемент, метод dequeue должен удалить этот элемент и установить указатели first и last равными None.
- Удаление элемента из очереди с более чем одним элементом: Если в очереди есть более чем один элемент, метод dequeue должен удалить первый элемент и установить указатель first на следующий элемент.
