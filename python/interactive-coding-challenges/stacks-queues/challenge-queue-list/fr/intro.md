# Introduction

En informatique, une file est une collection d'entités maintenues dans une séquence et pouvant être modifiée par l'ajout d'entités à une extrémité de la séquence et la suppression d'entités de l'autre extrémité de la séquence. Elle est également connue sous le nom de structure de données FIFO (first in, first out). Dans ce défi, nous allons implémenter une file à l'aide d'une liste chaînée.
