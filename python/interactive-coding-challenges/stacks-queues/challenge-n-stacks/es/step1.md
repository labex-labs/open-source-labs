# N pilas

## Problema

Implementar n pilas utilizando una sola matriz. Las pilas y la matriz tienen un tamaño fijo y son de igual tamaño. Empujar a una pila llena resulta en una excepción, y desapilar de una pila vacía también resulta en una excepción. Podemos asumir que el índice de pila pasado por el usuario es válido y que esto cabe en la memoria.

## Requisitos

Para implementar n pilas utilizando una sola matriz, necesitamos cumplir con los siguientes requisitos:

- Las pilas y la matriz tienen un tamaño fijo.
- Las pilas son de igual tamaño.
- Empujar a una pila llena resulta en una excepción.
- Desapilar de una pila vacía resulta en una excepción.
- Podemos asumir que el índice de pila pasado por el usuario es válido.
- Podemos asumir que esto cabe en la memoria.

## Uso de ejemplo

Para probar la implementación de n pilas utilizando una sola matriz, podemos realizar las siguientes acciones en las tres pilas:

- Empujar a pila llena -> Excepción
- Empujar a pila no llena
- Desapilar en pila vacía -> Excepción
- Desapilar en pila no vacía
