# N Piles

## Problème

Implémenter n piles à l'aide d'un seul tableau. Les piles et le tableau ont une taille fixe et sont de taille égale. Empiler sur une pile pleine entraîne une exception, et dépiler sur une pile vide entraîne également une exception. On peut supposer que l'index de pile passé par l'utilisateur est valide et que cela tient dans la mémoire.

## Exigences

Pour implémenter n piles à l'aide d'un seul tableau, nous devons répondre aux exigences suivantes :

- Les piles et le tableau ont une taille fixe.
- Les piles sont de taille égale.
- Empiler sur une pile pleine entraîne une exception.
- Dépiler sur une pile vide entraîne une exception.
- On peut supposer que l'index de pile passé par l'utilisateur est valide.
- On peut supposer que cela tient dans la mémoire.

## Utilisation de l'exemple

Pour tester l'implémentation de n piles à l'aide d'un seul tableau, nous pouvons effectuer les actions suivantes sur les trois piles :

- Empiler sur une pile pleine -> Exception
- Empiler sur une pile non pleine
- Dépiler sur une pile vide -> Exception
- Dépiler sur une pile non vide
