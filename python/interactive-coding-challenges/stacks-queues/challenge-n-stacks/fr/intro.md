# Introduction

En informatique, une pile est un type abstrait de données qui sert de collection d'éléments, avec deux principales opérations : push, qui ajoute un élément à la collection, et pop, qui supprime l'élément le plus récemment ajouté qui n'a pas encore été supprimé. Dans ce défi, nous allons implémenter n piles à l'aide d'un seul tableau.
