# Введение

В информатике стек - это абстрактный тип данных, представляющий собой коллекцию элементов, с двумя основными операциями: push, которая добавляет элемент в коллекцию, и pop, которая удаляет наиболее недавно добавленный элемент, который еще не был удален. В этом испытании мы реализуем n стеков с использованием одного массива.
