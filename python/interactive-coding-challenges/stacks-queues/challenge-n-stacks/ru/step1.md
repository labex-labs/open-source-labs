# N стеков

## Задача

Реализовать n стеков с использованием одного массива. Стеки и массив имеют фиксированный размер и равные размеры. Добавление элемента в полный стек вызывает исключение, а удаление элемента из пустого стека также вызывает исключение. Можно предположить, что индекс стека, переданный пользователем, действителен и помещается в память.

## Требования

Для реализации n стеков с использованием одного массива необходимо выполнить следующие требования:

- Стеки и массив имеют фиксированный размер.
- Стеки имеют равные размеры.
- Добавление элемента в полный стек вызывает исключение.
- Удаление элемента из пустого стека вызывает исключение.
- Можно предположить, что индекс стека, переданный пользователем, действителен.
- Можно предположить, что это помещается в память.

## Пример использования

Для тестирования реализации n стеков с использованием одного массива можно выполнить следующие действия по отношению к трем стекам:

- Добавить элемент в полный стек -> Исключение
- Добавить элемент в неполный стек
- Удалить элемент из пустого стека -> Исключение
- Удалить элемент из непустого стека
