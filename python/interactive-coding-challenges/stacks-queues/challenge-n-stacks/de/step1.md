# N Stapel

## Problem

Implementiere n Stapel mit einem einzigen Array. Die Stapel und das Array haben eine feste Größe und sind gleich groß. Ein Push auf einen vollständigen Stapel führt zu einer Ausnahme, und ein Pop von einem leeren Stapel führt ebenfalls zu einer Ausnahme. Wir können davon ausgehen, dass der vom Benutzer übergebene Stapelindex gültig ist und dass dies im Speicher passt.

## Anforderungen

Um n Stapel mit einem einzigen Array zu implementieren, müssen wir die folgenden Anforderungen erfüllen:

- Die Stapel und das Array haben eine feste Größe.
- Die Stapel sind gleich groß.
- Ein Push auf einen vollständigen Stapel führt zu einer Ausnahme.
- Ein Pop von einem leeren Stapel führt zu einer Ausnahme.
- Wir können davon ausgehen, dass der vom Benutzer übergebene Stapelindex gültig ist.
- Wir können davon ausgehen, dass dies im Speicher passt.

## Beispielverwendung

Um die Implementierung von n Stapel mit einem einzigen Array zu testen, können wir die folgenden Aktionen auf den drei Stapel ausführen:

- Push auf vollständigen Stapel -> Ausnahme
- Push auf nicht vollständigen Stapel
- Pop auf leeren Stapel -> Ausnahme
- Pop auf nicht leeren Stapel
