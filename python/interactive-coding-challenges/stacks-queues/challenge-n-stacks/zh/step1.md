# 实现n个栈

## 问题

使用单个数组实现n个栈。栈和数组的大小是固定的，且大小相等。向已满的栈中压入元素会导致异常，从空栈中弹出元素也会导致异常。我们可以假设用户传入的栈索引是有效的，并且这在内存中是可行的。

## 要求

要使用单个数组实现n个栈，我们需要满足以下要求：

- 栈和数组的大小是固定的。
- 栈的大小相等。
- 向已满的栈中压入元素会导致异常。
- 从空栈中弹出元素会导致异常。
- 我们可以假设用户传入的栈索引是有效的。
- 我们可以假设这在内存中是可行的。

## 示例用法

为了测试使用单个数组实现n个栈的功能，我们可以对三个栈执行以下操作：

- 向已满的栈中压入元素 -> 异常
- 向未满的栈中压入元素
- 从空栈中弹出元素 -> 异常
- 从非空栈中弹出元素
