# 简介

在计算机科学中，栈是一种抽象数据类型，它作为一组元素的集合，有两个主要操作：push（将一个元素添加到集合中）和 pop（移除最近添加且尚未移除的元素）。在这个挑战中，我们将使用单个数组实现 n 个栈。
