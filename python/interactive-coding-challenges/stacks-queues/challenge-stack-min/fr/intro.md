# Introduction

En informatique, une pile est un type abstrait de données qui sert de collection d'éléments, avec deux principales opérations : push, qui ajoute un élément à la collection, et pop, qui supprime l'élément le plus récemment ajouté qui n'a pas encore été supprimé. Dans ce défi, nous allons implémenter une pile avec les méthodes push, pop et min qui fonctionnent en O(1) temps.
