# スタックの最小値

## 問題

push、pop、およびminメソッドがO(1)の時間で動作するスタックを実装する問題です。pushメソッドはコレクションに要素を追加し、popメソッドはまだ削除されていない最も新しく追加された要素を削除し、minメソッドはスタック内の最小要素を返します。この3つのメソッドはすべて一定時間O(1)で動作する必要があります。

## 要件

この問題を解くには、以下の要件を考慮する必要があります。

- スタックには整数のみが含まれます。
- pushの入力値は有効です。
- 空のスタックに対してminメソッドを呼び出した場合、sys.maxsizeを返す必要があります。
- この問題に使用できる既存のスタッククラスがあると仮定できます。
- スタックはメモリに収まると仮定できます。

## サンプルコード

以下のシナリオで実装をテストできます。

- 空のスタックに対するPush/pop
- 空でないスタックに対するPush/pop
- 空のスタックに対するMin
- 空でないスタックに対するMin
