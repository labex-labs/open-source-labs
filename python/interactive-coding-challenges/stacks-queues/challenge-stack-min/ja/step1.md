# スタックの最小値

## 問題

push、pop、および min メソッドが O(1) の時間で動作するスタックを実装する問題です。push メソッドはコレクションに要素を追加し、pop メソッドはまだ削除されていない最も新しく追加された要素を削除し、min メソッドはスタック内の最小要素を返します。この 3 つのメソッドはすべて一定時間 O(1) で動作する必要があります。

## 要件

この問題を解くには、以下の要件を考慮する必要があります。

- スタックには整数のみが含まれます。
- push の入力値は有効です。
- 空のスタックに対して min メソッドを呼び出した場合、sys.maxsize を返す必要があります。
- この問題に使用できる既存のスタッククラスがあると仮定できます。
- スタックはメモリに収まると仮定できます。

## サンプルコード

以下のシナリオで実装をテストできます。

- 空のスタックに対する Push/pop
- 空でないスタックに対する Push/pop
- 空のスタックに対する Min
- 空でないスタックに対する Min
