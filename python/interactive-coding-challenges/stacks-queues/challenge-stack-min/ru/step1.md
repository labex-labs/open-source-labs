# Стек с минимальным элементом

## Задача

Задача заключается в реализации стека с методами push, pop и min, работающими за O(1) время. Метод push добавляет элемент в коллекцию, метод pop удаляет наиболее недавно добавленный элемент, который еще не был удален, а метод min возвращает минимальный элемент в стеке. Все три метода должны выполняться за константное время, O(1).

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Стек содержит только целые числа.
- Входные значения для push являются допустимыми.
- Если мы вызываем метод min для пустого стека, мы должны вернуть sys.maxsize.
- Можно предположить, что у нас уже есть класс стека, который можно использовать для этой задачи.
- Можно предположить, что стек помещается в память.

## Пример использования

Мы можем протестировать нашу реализацию с помощью следующих сценариев:

- Push/pop для пустого стека.
- Push/pop для непустого стека.
- Min для пустого стека.
- Min для непустого стека.
