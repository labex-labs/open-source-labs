# 带有最小值操作的栈

## 问题

问题是要实现一个栈，使其具有时间复杂度为 O(1) 的 push、pop 和 min 方法。push 方法将一个元素添加到集合中，pop 方法移除最近添加且尚未移除的元素，min 方法返回栈中的最小元素。这三个方法都应该在常数时间 O(1) 内运行。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 栈中仅包含整数。
- push 的输入值是有效的。
- 如果我们对空栈调用 min 方法，应该返回 sys.maxsize。
- 我们可以假设已经有一个可用于此问题的栈类。
- 我们可以假设栈能装入内存。

## 示例用法

我们可以通过以下场景测试我们的实现：

- 对空栈进行 push/pop 操作。
- 对非空栈进行 push/pop 操作。
- 对空栈调用 min 方法。
- 对非空栈调用 min 方法。
