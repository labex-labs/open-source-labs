# Introducción

En informática, una cola es una estructura de datos que sigue el principio de Primer en Entrar, Primer en Salir (FIFO, por sus siglas en inglés). Es una lista ordenada de elementos donde un elemento se inserta por un extremo y se elimina por el otro. Por otro lado, una pila sigue el principio de Último en Entrar, Primer en Salir (LIFO, por sus siglas en inglés). Es una lista ordenada de elementos donde un elemento se inserta y se elimina por el mismo extremo. En este desafío, implementaremos una cola utilizando dos pilas.
