# 用栈实现队列

## 问题

用两个栈实现一个队列是一个具有挑战性的问题。基本思路是使用一个栈进行入队操作，另一个栈进行出队操作。当一个元素入队时，将其压入第一个栈。当一个元素出队时，从第二个栈弹出。如果第二个栈为空，我们将第一个栈中的所有元素弹出，并按相反顺序压入第二个栈。这确保了最先入队的元素是最先出队的元素。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 我们需要实现两个方法：入队（enqueue）和出队（dequeue）。
- 我们假设已经有一个可用于此问题的栈类。
- 我们不能将None值压入栈。
- 我们可以假设这个问题在内存中是可行的。

## 示例用法

以下是一些关于如何使用我们用两个栈实现的队列的示例：

- 在空栈上进行入队和出队操作：我们可以将一个元素入队，然后出队，以确保队列正常工作。
- 在非空栈上进行入队和出队操作：我们可以将多个元素入队，然后出队，以确保队列正常工作。
- 连续多次入队：我们可以连续将多个元素入队，然后出队，以确保队列正常工作。
- 连续多次出队：我们可以将多个元素入队，然后连续出队，以确保队列正常工作。
- 在出队后入队：我们可以将一个元素入队，出队，然后再将另一个元素入队，以确保队列正常工作。
- 在入队后出队：我们可以将一个元素入队，出队，然后再将另一个元素入队，以确保队列正常工作。
