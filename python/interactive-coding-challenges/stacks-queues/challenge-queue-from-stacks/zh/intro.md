# 简介

在计算机科学中，队列（queue）是一种遵循先进先出（FIFO，First-In-First-Out）原则的数据结构。它是一个元素的有序列表，其中元素在一端插入，并在另一端移除。另一方面，栈（stack）遵循后进先出（LIFO，Last-In-First-Out）原则。它是一个元素的有序列表，其中元素在同一端插入和移除。在这个挑战中，我们将使用两个栈来实现一个队列。
