# Introduction

En informatique, une file est une structure de données qui suit le principe First-In-First-Out (FIFO). C'est une liste ordonnée d'éléments où un élément est inséré à une extrémité et retiré de l'autre extrémité. Une pile, en revanche, suit le principe Last-In-First-Out (LIFO). C'est une liste ordonnée d'éléments où un élément est inséré et retiré de la même extrémité. Dans ce défi, nous allons implémenter une file à l'aide de deux piles.
