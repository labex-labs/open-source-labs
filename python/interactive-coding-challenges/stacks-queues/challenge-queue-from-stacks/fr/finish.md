# Résumé

Dans ce défi, nous avons appris à implémenter une file à l'aide de deux piles. Nous avons vu que ce problème peut être difficile, mais avec la bonne approche, nous pouvons le résoudre efficacement. Nous avons également vu quelques exemples de manière dont nous pouvons utiliser notre implémentation d'une file à l'aide de deux piles.
