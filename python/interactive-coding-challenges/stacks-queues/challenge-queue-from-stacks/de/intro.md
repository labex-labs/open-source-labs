# Einführung

In der Informatik ist eine Warteschlange (Queue) eine Datenstruktur, die dem First-In-First-Out (FIFO)-Prinzip folgt. Es ist eine geordnete Liste von Elementen, wobei ein Element am einen Ende eingefügt und am anderen Ende entfernt wird. Eine Stapelstruktur (Stack), anderseits, folgt dem Last-In-First-Out (LIFO)-Prinzip. Es ist eine geordnete Liste von Elementen, wobei ein Element am selben Ende eingefügt und entfernt wird. In dieser Herausforderung werden wir eine Warteschlange mit zwei Stapelstrukturen implementieren.
