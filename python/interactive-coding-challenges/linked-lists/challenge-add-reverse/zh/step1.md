# 反向相加

## 问题

给定两个非循环的单链表，将每个节点的值按相反顺序相加，并以相反顺序返回链表形式的结果。如果其中一个输入为 None，则因操作无效返回 None。这些数字能存入内存，并且我们可以假设已经有一个可用于此问题的链表类。

## 要求

要完成此挑战，必须满足以下要求：

- 链表是非循环的单链表。
- 返回值必须是相反顺序。
- 如果其中一个输入为 None，则因操作无效返回 None。
- 数字能存入内存。
- 我们可以假设已经有一个可用于此问题的链表类。
- 链表能存入内存。

## 示例用法

以下示例演示了此挑战的用法：

- 空链表 -> None
- 相加不同长度的值
  - 输入 1：6 -> 5 -> None
  - 输入 2：9 -> 8 -> 7
  - 结果：5 -> 4 -> 8
- 相加相同长度的值
  - 从不同长度的值中演练得出
  - 在此处列出以保证完整性
