# Сложение в обратном порядке

## Задача

Даны два нециклических односвязных списка. Сложите значения каждого узла в обратном порядке и верните результат также в обратном порядке в виде связанного списка. Если один из входных данных равен None, верните None в случае недопустимой операции. Числа помещаются в память, и мы можем предположить, что у нас уже есть класс связанного списка, который можно использовать для этой задачи.

## Требования

Для решения этой задачи необходимо выполнить следующие требования:

- Связанный список нециклический и односвязный.
- Возвращаемое значение должно быть в обратном порядке.
- Если один из входных данных равен None, верните None в случае недопустимой операции.
- Числа помещаются в память.
- Мы можем предположить, что у нас уже есть класс связанного списка, который можно использовать для этой задачи.
- Связанный список помещается в память.

## Пример использования

Следующие примеры демонстрируют использование этой задачи:

- Пустой список(ы) -> None
- Сложение значений разной длины
  - Вход 1: 6->5->None
  - Вход 2: 9->8->7
  - Результат: 5->4->8
- Сложение значений одинаковой длины
  - Демонстрируется на значениях разной длины
  - Здесь показано для полноты картины
