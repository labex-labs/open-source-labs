# Резюме

Связанный список нециклический и односвязный. Возвращаемое значение должно быть в обратном порядке. Если один из входных данных равен None, верните None в случае недопустимой операции. Числа помещаются в память, и мы можем предположить, что у нас уже есть класс связанного списка, который можно использовать для этой задачи.
