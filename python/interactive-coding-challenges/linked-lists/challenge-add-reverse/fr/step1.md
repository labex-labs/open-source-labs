# Ajouter en ordre inverse

## Problème

Étant donné deux listes chaînées non circulaires et à un seul sens, ajoutez les valeurs de chaque nœud dans l'ordre inverse et renvoyez le résultat dans une liste chaînée également dans l'ordre inverse. Si l'une des entrées est None, renvoyez None pour une opération invalide. Les nombres peuvent tenir en mémoire et on peut supposer qu'on a déjà une classe de liste chaînée qui peut être utilisée pour ce problème.

## Exigences

Pour résoudre ce défi, les exigences suivantes doivent être satisfaites :

- La liste chaînée est non circulaire et à un seul sens.
- La valeur de retour doit être dans l'ordre inverse.
- Si l'une des entrées est None, renvoyez None pour une opération invalide.
- Les nombres peuvent tenir en mémoire.
- On peut supposer qu'on a déjà une classe de liste chaînée qui peut être utilisée pour ce problème.
- La liste chaînée peut tenir en mémoire.

## Utilisation de l'exemple

Les exemples suivants démontrent l'utilisation de ce défi :

- Liste(s) vide(s) -> None
- Ajouter des valeurs de longueurs différentes
  - Entrée 1 : 6->5->None
  - Entrée 2 : 9->8->7
  - Résultat : 5->4->8
