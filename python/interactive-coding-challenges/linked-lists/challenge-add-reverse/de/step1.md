# Addieren Sie in umgekehrter Reihenfolge

## Problemstellung

Gegeben zwei nicht-zirkuläre, einfach verkettete Listen, addieren Sie die Werte jedes Knotens in umgekehrter Reihenfolge und geben das Ergebnis ebenfalls in einer umgekehrt verketteten Liste zurück. Wenn eine der Eingaben None ist, geben Sie None zurück, da dies eine ungültige Operation ist. Die Zahlen passen in den Arbeitsspeicher, und wir können davon ausgehen, dass wir bereits eine Klasse für verkettete Listen haben, die für dieses Problem verwendet werden kann.

## Anforderungen

Um diese Aufgabe zu lösen, müssen die folgenden Anforderungen erfüllt sein:

- Die verkettete Liste ist nicht-zirkulär und einfach verkettet.
- Der Rückgabewert muss in umgekehrter Reihenfolge vorliegen.
- Wenn eine der Eingaben None ist, geben Sie None zurück, da dies eine ungültige Operation ist.
- Die Zahlen passen in den Arbeitsspeicher.
- Wir können davon ausgehen, dass wir bereits eine Klasse für verkettete Listen haben, die für dieses Problem verwendet werden kann.
- Die verkettete Liste passt in den Arbeitsspeicher.

## Beispielverwendung

Die folgenden Beispiele veranschaulichen die Verwendung dieser Aufgabe:

- Leere Liste(n) -> None
- Addieren von Werten unterschiedlicher Länge
  - Eingabe 1: 6->5->None
  - Eingabe 2: 9->8->7
  - Ergebnis: 5->4->8
- Addieren von Werten gleicher Länge
  - Aus den Werten unterschiedlicher Länge geübt
  - Hier für Vollständigkeit wiederholt
