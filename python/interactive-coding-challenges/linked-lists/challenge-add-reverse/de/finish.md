# Zusammenfassung

Die verkettete Liste ist nicht-zirkulär und einfach verkettet. Der Rückgabewert muss in umgekehrter Reihenfolge vorliegen. Wenn eine der Eingaben None ist, geben Sie None zurück, da dies eine ungültige Operation ist. Die Zahlen passen in den Arbeitsspeicher, und wir können davon ausgehen, dass wir bereits eine Klasse für verkettete Listen haben, die für dieses Problem verwendet werden kann.
