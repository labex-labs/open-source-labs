# Sumar en orden inverso

## Problema

Dadas dos listas enlazadas simples no circulares, sumar los valores de cada nodo en orden inverso y devolver el resultado en una lista enlazada también en orden inverso. Si una de las entradas es None, devolver None por una operación no válida. Los números caben en memoria y podemos suponer que ya tenemos una clase de lista enlazada que se puede utilizar para este problema.

## Requisitos

Para completar este desafío, deben cumplirse los siguientes requisitos:

- La lista enlazada no es circular y es simple.
- El valor de retorno debe estar en orden inverso.
- Si una de las entradas es None, devolver None por una operación no válida.
- Los números caben en memoria.
- Podemos suponer que ya tenemos una clase de lista enlazada que se puede utilizar para este problema.
- La lista enlazada cabe en memoria.

## Uso de ejemplo

Los siguientes ejemplos demuestran el uso de este desafío:

- Lista(s) vacía(s) -> None
- Sumar valores de diferentes longitudes
  - Entrada 1: 6->5->None
  - Entrada 2: 9->8->7
  - Resultado: 5->4->8
- Sumar valores de la misma longitud
