# 逆順で加算

## 問題

非循環の単方向リンクリスト 2 つが与えられたとき、各ノードの値を逆順で加算し、結果も逆順のリンクリストで返します。入力の 1 つが None の場合、無効な操作として None を返します。数値はメモリに収まり、この問題に使用できるリンクリストクラスが既にあると仮定できます。

## 要件

このチャレンジを完了するには、以下の要件を満たす必要があります。

- リンクリストは非循環で単方向リンクされています。
- 返却値は逆順でなければなりません。
- 入力の 1 つが None の場合、無効な操作として None を返します。
- 数値はメモリに収まります。
- この問題に使用できるリンクリストクラスが既にあると仮定できます。
- リンクリストはメモリに収まります。

## 例の使用法

以下の例は、このチャレンジの使用方法を示しています。

- 空のリスト -> None
- 異なる長さの値を加算する
  - 入力 1: 6->5->None
  - 入力 2: 9->8->7
  - 結果：5->4->8
- 同じ長さの値を加算する
  - 異なる長さの値から実施
  - 完全性のためここで行う
