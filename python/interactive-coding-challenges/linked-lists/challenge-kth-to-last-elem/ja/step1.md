# 最後からk番目の要素

## 問題

非循環の単方向連結リストが与えられたとき、そのリストの最後からk番目の要素を見つけることが課題となります。kが0の場合、最後の要素が返されるべきです。kが連結リストの長さ以上の場合、Noneが返されるべきです。追加のデータ構造を使用できず、連結リストクラスが既に利用可能であると仮定します。

## 要件

この問題を解くには、以下の要件を満たす必要があります。

- 連結リストは非循環で単方向連結されています。
- kは有効な整数です。
- kが0の場合、最後の要素が返されるべきです。
- kが連結リストの長さ以上の場合、Noneが返されるべきです。
- 追加のデータ構造を使用できません。
- 連結リストクラスが既に利用可能である。

## 例の使用法

以下のシナリオを使って解決策をテストできます。

- 空のリストの場合、Noneが返されるべきです。
- kが連結リストの長さ以上の場合、Noneが返されるべきです。
- 連結リストが1つの要素のみを持ち、kが0の場合、その要素が返されるべきです。
- 多くの要素がある一般的なケースで、kが連結リストの長さ未満の場合、最後からk番目の要素が返されるべきです。
