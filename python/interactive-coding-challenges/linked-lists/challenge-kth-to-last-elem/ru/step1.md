# K-й элемент от конца

## Задача

Дан нециклический односвязный список. Задача — найти k-й элемент от конца списка. Если k равно 0, то должен быть возвращён последний элемент. Если k больше или равно длине связанного списка, то должен быть возвращён None. Не допускается использовать дополнительные структуры данных, предполагается, что класс связанного списка уже доступен.

## Требования

Для решения этой задачи должны быть выполнены следующие требования:

- Связанный список нециклический и односвязный.
- k — допустимое целое число.
- Если k равно 0, то должен быть возвращён последний элемент.
- Если k больше или равно длине связанного списка, то должен быть возвращён None.
- Не допускается использовать дополнительные структуры данных.
- Класс связанного списка уже доступен.

## Пример использования

Следующие сценарии можно использовать для тестирования решения:

- Пустой список должен возвращать None.
- Если k больше или равно длине связанного списка, то должен быть возвращён None.
- Если в связанном списке есть только один элемент и k равно 0, то должен быть возвращён этот элемент.
- В общем случае, когда есть много элементов и k меньше длины связанного списка, должен быть возвращён k-й элемент от конца.
