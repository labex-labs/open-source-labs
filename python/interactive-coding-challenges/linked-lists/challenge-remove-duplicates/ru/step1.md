# Удаление дубликатов

## Задача

Дана нециклическая односвязный список. Удалите из него дубликаты. Цель — модифицировать исходный список на месте и вернуть голову модифицированного списка.

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Список нециклический и односвязный.
- В список нельзя вставлять значения `None`.
- У нас уже есть класс односвязного списка, который можно использовать для этой задачи.
- Необходимо реализовать два решения: одно с использованием дополнительных структур данных, другое — без них.
- Задача помещается в память.

## Пример использования

Вот несколько примеров того, как должна работать функция:

- Пустой односвязный список -> []
- Односвязный список с одним элементом -> [элемент]
- Обычный случай без дубликатов -> [1, 2, 3, 4]
- Обычный случай с дубликатами -> [1, 2, 3]
