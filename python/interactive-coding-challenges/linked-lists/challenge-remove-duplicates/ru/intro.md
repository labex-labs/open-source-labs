# Введение

В информатике связанный список — это линейная коллекция элементов данных, в которой линейный порядок не определяется их физическим расположением в памяти. Вместо этого каждый элемент ссылается на следующий. Это структура данных, состоящая из коллекции узлов, которые вместе представляют последовательность. В этом испытании мы сосредоточимся на удалении дубликатов из связанного списка.
