# 简介

在计算机科学中，链表是数据元素的线性集合，其中线性顺序不是由它们在内存中的物理位置决定的。相反，每个元素指向下一个元素。它是一种由节点集合组成的数据结构，这些节点共同表示一个序列。在这个挑战中，我们将专注于从链表中删除重复项。
