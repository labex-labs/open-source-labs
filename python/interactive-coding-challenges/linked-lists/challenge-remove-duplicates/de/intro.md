# Einführung

In der Informatik ist eine verkettete Liste eine lineare Sammlung von Datenelementen, bei der die lineare Reihenfolge nicht durch ihre physische Platzierung im Speicher gegeben ist. Stattdessen weist jedes Element auf das nächste hin. Es ist eine Datenstruktur, die aus einer Sammlung von Knoten besteht, die zusammen eine Sequenz darstellen. In dieser Herausforderung werden wir uns auf das Entfernen von Duplikaten aus einer verketteten Liste konzentrieren.
