# Introducción

En informática, una lista enlazada es una colección lineal de elementos de datos, en la que el orden lineal no está dado por su ubicación física en la memoria. En cambio, cada elemento apunta al siguiente. Es una estructura de datos que consta de una colección de nodos que juntos representan una secuencia. En este desafío, nos centraremos en eliminar duplicados de una lista enlazada.
