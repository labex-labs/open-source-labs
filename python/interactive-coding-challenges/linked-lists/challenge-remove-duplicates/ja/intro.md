# はじめに

コンピュータサイエンスにおいて、リンクリストはデータ要素の線形集合であり、その線形順序はメモリ内の物理的配置によって与えられるものではありません。代わりに、各要素は次の要素を指し示します。これは、一緒にシーケンスを表すノードのコレクションで構成されるデータ構造です。このチャレンジでは、リンクリストから重複を削除することに焦点を当てます。
