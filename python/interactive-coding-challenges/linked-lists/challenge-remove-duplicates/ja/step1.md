# 重複削除

## 問題

非循環の単方向リンクリストが与えられた場合、そこから重複を削除します。目的は、元のリンクリストをそのまま変更し、変更後のリンクリストの先頭を返すことです。

## 要件

この問題を解くには、以下の要件を考慮する必要があります。

- リンクリストは非循環で単方向リンクされています。
- リストには null 値を挿入できません。
- この問題に対して既に使用できるリンクリストクラスがあります。
- 2 つの解決策を実装する必要があります。1 つは追加のデータ構造を使用したもので、もう 1 つは使用しないものです。
- この問題はメモリ内に収まります。

## 例の使用法

関数がどのように動作するかのいくつかの例を以下に示します。

- 空のリンクリスト -> []
- 要素が 1 つのリンクリスト -> [要素]
- 重複のない一般的なケース -> [1, 2, 3, 4]
- 重複のある一般的なケース -> [1, 2, 3]
