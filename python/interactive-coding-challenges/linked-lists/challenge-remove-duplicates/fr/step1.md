# Supprimer les doublons

## Problème

Étant donné une liste chaînée simple non circulaire, supprimer les doublons. Le but est de modifier la liste d'origine en place et de renvoyer la tête de la liste modifiée.

## Exigences

Pour résoudre ce problème, nous devons prendre en compte les exigences suivantes :

- La liste chaînée est non circulaire et simple.
- Les valeurs nulles ne peuvent pas être insérées dans la liste.
- Nous disposons déjà d'une classe de liste chaînée qui peut être utilisée pour ce problème.
- Deux solutions doivent être implémentées : une utilisant des structures de données supplémentaires et une sans.
- Le problème s'adapte à la mémoire.

## Utilisation exemple

Voici quelques exemples de comportement attendu de la fonction :

- Liste chaînée vide -> []
- Liste chaînée avec un élément -> [élément]
- Cas général sans doublons -> [1, 2, 3, 4]
- Cas général avec doublons -> [1, 2, 3]
