# Introduction

En informatique, une liste chaînée est une collection linéaire d'éléments de données, dans laquelle l'ordre linéaire n'est pas déterminé par leur emplacement physique en mémoire. Au lieu de cela, chaque élément pointe vers le suivant. C'est une structure de données composée d'une collection de nœuds qui représentent ensemble une séquence. Dans ce défi, nous nous concentrerons sur la suppression des doublons d'une liste chaînée.
