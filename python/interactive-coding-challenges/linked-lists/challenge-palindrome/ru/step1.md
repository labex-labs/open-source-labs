# Палиндром

## Задача

Дана односвязный список. Определить, является ли он палиндромом. Связанный список является палиндромом, если последовательность элементов в списке одинакова при чтении вперед и назад.

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Связанный список не является цикличным и односвязным.
- Односимвольная строка или число не считается палиндромом.
- У нас уже есть класс связанного списка, который можно использовать для этой задачи.
- Можно использовать дополнительные структуры данных.
- Связанный список помещается в память.

## Пример использования

Ниже приведены примеры того, как должна работать функция:

- Пустой список должен возвращать False.
- Список из одного элемента должен возвращать False.
- Список из двух и более элементов, который не является палиндромом, должен возвращать False.
- Палиндром с четной длиной должен возвращать True.
- Палиндром с нечетной длиной должен возвращать True.
