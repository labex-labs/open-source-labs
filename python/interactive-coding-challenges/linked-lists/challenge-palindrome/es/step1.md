# Palíndromo

## Problema

Dada una lista enlazada simple, determinar si es un palíndromo. Una lista enlazada es un palíndromo si la secuencia de elementos en la lista es la misma hacia adelante y hacia atrás.

## Requisitos

Para resolver este problema, debemos considerar los siguientes requisitos:

- La lista enlazada no es circular y es simple.
- Un solo carácter o número no se considera un palíndromo.
- Ya tenemos una clase de lista enlazada que se puede utilizar para este problema.
- Podemos utilizar estructuras de datos adicionales.
- La lista enlazada cabe en memoria.

## Uso de ejemplo

A continuación se presentan algunos ejemplos de cómo debe comportarse la función:

- Una lista vacía debe devolver False.
- Una lista con un solo elemento debe devolver False.
- Una lista con dos o más elementos que no es un palíndromo debe devolver False.
- Un palíndromo con una longitud par debe devolver True.
- Un palíndromo con una longitud impar debe devolver True.
