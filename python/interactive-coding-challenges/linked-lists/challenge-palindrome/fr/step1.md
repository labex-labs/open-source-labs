# Palindrome

## Problème

Étant donné une liste chaînée simple, déterminer si elle est un palindrome. Une liste chaînée est un palindrome si la séquence d'éléments dans la liste est la même dans les deux sens.

## Exigences

Pour résoudre ce problème, nous devons prendre en compte les exigences suivantes :

- La liste chaînée n'est pas circulaire et est simple.
- Un seul caractère ou un seul nombre n'est pas considéré comme un palindrome.
- Nous disposons déjà d'une classe de liste chaînée que nous pouvons utiliser pour ce problème.
- Nous pouvons utiliser des structures de données supplémentaires.
- La liste chaînée tient dans la mémoire.

## Utilisation de l'exemple

Voici quelques exemples de comportement attendu de la fonction :

- Une liste vide devrait renvoyer False.
- Une liste avec un seul élément devrait renvoyer False.
- Une liste avec deux éléments ou plus qui n'est pas un palindrome devrait renvoyer False.
- Un palindrome de longueur paire devrait renvoyer True.
- Un palindrome de longueur impaire devrait renvoyer True.
