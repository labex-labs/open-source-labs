# 回文

## 問題

単方向連結リストが与えられたとき、それが回文かどうかを判定します。連結リストが回文であるとは、リスト内の要素の順序が前向きと後ろ向きで同じであることを意味します。

## 要件

この問題を解くには、以下の要件を考慮する必要があります。

- 連結リストは非循環的で単方向連結です。
- 単一の文字または数字は回文とは見なされません。
- この問題に対して既に使用できる連結リストクラスがあります。
- 追加のデータ構造を使用できます。
- 連結リストはメモリに収まります。

## 例

この関数がどのように動作するかの例をいくつか示します。

- 空のリストはFalseを返すべきです。
- 要素が1つのリストはFalseを返すべきです。
- 2つ以上の要素を持ち、回文でないリストはFalseを返すべきです。
- 偶数の長さを持つ回文はTrueを返すべきです。
- 奇数の長さを持つ回文はTrueを返すべきです。
