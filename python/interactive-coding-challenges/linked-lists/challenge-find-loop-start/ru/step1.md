# Найти начало цикла

## Задача

Дан односвязный список. Нам нужно найти начало цикла, если оно существует. Цикл определяется как узел в списке, который ссылается на предыдущий узел, образуя цикл. Если цикла нет, возвращаем None. Если цикл есть, возвращаем узел, откуда начинается цикл.

## Требования

Для решения этой задачи нужно учитывать следующие требования:

- Список — это односвязный список.
- Мы не можем предполагать, что всегда передается циклический связный список.
- Когда находим цикл, возвращаем узел, откуда начинается цикл.
- Можем предположить, что у нас уже есть класс связного списка, который можно использовать для этой задачи.

## Пример использования

Вот несколько примеров использования этой функции:

- Пустой список -> None
- Не циклический связный список -> None
  - Один элемент
  - Два или более элементов
- Общий случай циклического связного списка
