# Введение

В информатике связанный список представляет собой линейную коллекцию элементов данных, при этом линейный порядок не определяется их физическим расположением в памяти. Вместо этого каждый элемент ссылается на следующий. Связанный список является динамической структурой данных, что означает, что мы можем добавлять или удалять элементы из него в любое время. В этом испытании мы рассмотрим, как найти начало цикла в связанном списке.
