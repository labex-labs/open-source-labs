# ループの開始点を見つける

## 問題

シングルリンクリストが与えられたとき、存在する場合はループの開始点を見つける必要があります。ループは、リスト内のノードが以前のノードを指し、サイクルを作成することで定義されます。ループがない場合は、None を返します。ループがある場合は、ループが開始するノードを返します。

## 要件

この問題を解くには、次の要件を考慮する必要があります。

- リンクリストはシングルリンクリストです。
- 常に循環リンクリストが渡されるとは仮定できません。
- ループを見つけたとき、ループが開始するノードを返します。
- この問題に使用できるリンクリストクラスが既にあると仮定できます。

## 例の使用法

この関数の使用方法の例をいくつか示します。

- 空のリスト -> None
- 循環リンクリストではない -> None
  - 1 つの要素
  - 2 つ以上の要素
- 循環リンクリストの一般的なケース
