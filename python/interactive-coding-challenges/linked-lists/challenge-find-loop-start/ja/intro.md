# はじめに

コンピュータサイエンスにおいて、リンクリストはデータ要素の線形集合であり、その線形順序はメモリ内の物理的配置によって与えられない。代わりに、各要素は次の要素を指す。リンクリストは動的データ構造であり、これは我々がいつでも要素を追加または削除できることを意味する。このチャレンジでは、リンクリストのループの開始点を見つける方法を見ていく。
