# Introducción

En informática, una lista enlazada es una colección lineal de elementos de datos, en la que el orden lineal no está dado por su ubicación física en la memoria. En cambio, cada elemento apunta al siguiente. Una lista enlazada es una estructura de datos dinámica, lo que significa que podemos agregar o eliminar elementos de ella en cualquier momento. En este desafío, veremos cómo encontrar el inicio de un ciclo en una lista enlazada.
