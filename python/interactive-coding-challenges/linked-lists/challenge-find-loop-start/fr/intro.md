# Introduction

En informatique, une liste chaînée est une collection linéaire d'éléments de données, dans laquelle l'ordre linéaire n'est pas déterminé par leur emplacement physique en mémoire. Au lieu de cela, chaque élément pointe vers le suivant. Une liste chaînée est une structure de données dynamique, ce qui signifie que nous pouvons y ajouter ou supprimer des éléments à tout moment. Dans ce défi, nous allons examiner comment trouver le début d'une boucle dans une liste chaînée.
