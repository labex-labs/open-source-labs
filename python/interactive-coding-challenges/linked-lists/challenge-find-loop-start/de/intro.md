# Einführung

In der Informatik ist eine verkettete Liste eine lineare Sammlung von Datenelementen, bei der die lineare Reihenfolge nicht durch ihre physische Platzierung im Arbeitsspeicher gegeben ist. Stattdessen weist jedes Element auf das nächste hin. Eine verkettete Liste ist eine dynamische Datenstruktur, was bedeutet, dass wir Elemente jederzeit hinzufügen oder entfernen können. In dieser Herausforderung werden wir uns ansehen, wie man den Anfang einer Schleife in einer verketteten Liste findet.
