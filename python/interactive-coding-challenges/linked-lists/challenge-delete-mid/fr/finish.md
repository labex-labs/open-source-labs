# Sommaire

Dans ce défi, nous avons exploré la manière de supprimer un nœud au milieu d'une liste chaînée, étant donné que nous n'avons accès qu'à ce nœud. Nous avons appris les exigences et les exemples d'utilisation de cette fonction. En suivant les exigences et les exemples, nous pouvons supprimer avec succès un nœud au milieu d'une liste chaînée en Python.
