# Eliminar el nodo del medio

## Problema

Dada una lista enlazada simple no circular, eliminar un nodo en el medio de la lista, dado solo acceso a ese nodo. Si se está eliminando el último nodo, conviértalo en un nodo ficticio con valor None. Se puede suponer que ya tenemos una clase de lista enlazada que se puede utilizar para este problema.

## Requisitos

Para resolver este problema, debemos considerar los siguientes requisitos:

- La lista enlazada es no circular y simple.
- Si se está eliminando el último nodo, conviértalo en un nodo ficticio con valor None.
- Ya tenemos una clase de lista enlazada que se puede utilizar para este problema.

## Uso de ejemplo

A continuación se presentan algunos ejemplos de cómo utilizar esta función:

- Eliminar en una lista vacía debe devolver None.
- Eliminar None debe devolver None.
- Eliminar en una lista con un solo nodo debe devolver [None].
- Eliminar en una lista con múltiples nodos debe devolver la lista actualizada con el nodo eliminado.
