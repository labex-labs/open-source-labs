# Introducción

En Python, una lista enlazada es una estructura de datos que consta de una secuencia de nodos, donde cada nodo contiene un valor y una referencia al siguiente nodo. Eliminar un nodo en el medio de una lista enlazada puede ser una tarea desafiante, especialmente cuando solo tenemos acceso a ese nodo. En este desafío, exploraremos cómo eliminar un nodo en el medio de una lista enlazada, dado solo acceso a ese nodo.
