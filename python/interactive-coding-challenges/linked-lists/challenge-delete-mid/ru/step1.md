# Удалить середину

## Задача

Дана нециклическая односвязный список. Удалите узел в середине списка, имея доступ только к этому узлу. Если удаляется последний узел, сделайте его "пустышкой" со значением None. Можно предположить, что у нас уже есть класс связанного списка, который можно использовать для этой задачи.

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Связанный список нециклический и односвязный.
- Если удаляется последний узел, сделайте его "пустышкой" со значением None.
- У нас уже есть класс связанного списка, который можно использовать для этой задачи.

## Пример использования

Вот несколько примеров использования этой функции:

- Удаление из пустого списка должно вернуть None.
- Удаление None должно вернуть None.
- Удаление из списка с одним узлом должно вернуть [None].
- Удаление из списка с несколькими узлами должно вернуть обновленный список с удаленным узлом.
