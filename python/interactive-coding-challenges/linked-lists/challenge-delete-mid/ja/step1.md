# 途中のノードを削除する

## 問題

非循環の単方向連結リストが与えられたとき、そのノードにのみアクセスできる状態で、リストの途中のノードを削除します。最終ノードを削除する場合は、値を None とするダミーにします。この問題に対して既に使用できる連結リストクラスがあると仮定します。

## 要件

この問題を解くには、以下の要件を考慮する必要があります。

- 連結リストは非循環で単方向連結です。
- 最終ノードを削除する場合は、値を None とするダミーにします。
- この問題に対して既に使用できる連結リストクラスがあります。

## 使い方の例

この関数の使い方の例をいくつか示します。

- 空のリストで削除を行うと None が返されます。
- None を削除すると None が返されます。
- 1 つのノードを持つリストで削除を行うと[None]が返されます。
- 複数のノードを持つリストで削除を行うと、削除されたノードを含む更新されたリストが返されます。
