# はじめに

Python において、連結リストは、各ノードが値と次のノードへの参照を含むノードのシーケンスで構成されるデータ構造です。連結リストの途中のノードを削除することは、そのノードにのみアクセスできる場合、特に難しいタスクになることがあります。このチャレンジでは、そのノードにのみアクセスできる状況下で、連結リストの途中のノードをどのように削除するかを検討します。
