# 分割

## 問題

シングルリンクリストが与えられたとき、それを値 x を中心に分割して、x 未満のすべてのノードが x 以上のすべてのノードの前にくるようにします。この関数は新しいリンクリストを返す必要があります。

## 要件

この問題を解くには、次の要件を考慮する必要があります。

- リンクリストは非循環でシングルリンクされています。
- 関数は新しいリンクリストを返す必要があります。
- 入力値 x は有効です。
- この問題に対して既に使えるリンクリストクラスがあります。
- 追加のデータ構造を作成できます。
- 問題はメモリに収まります。

## 例の使用法

ここに、関数がどのように機能するかのいくつかの例を示します。

- 空のリスト -> []
- 要素が 1 つのリスト -> [要素]
- 左側のリンクリストが空 -> [10, 11, 12]
- 右側のリンクリストが空 -> [1, 2, 3]
- 一般的なケース
  - 分割 = 10
  - 入力：4, 3, 7, 8, 10, 1, 10, 12
  - 出力：4, 3, 7, 8, 1, 10, 10, 12
