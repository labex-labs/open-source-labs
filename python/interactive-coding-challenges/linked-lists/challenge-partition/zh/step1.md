# 分区

## 问题

给定一个单链表，围绕值 x 对其进行分区，使得所有小于 x 的节点都出现在所有大于或等于 x 的节点之前。该函数应返回一个新的链表。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 链表是非循环的单链表。
- 函数应返回一个新的链表。
- 输入值 x 是有效的。
- 我们已经有一个可用于此问题的链表类。
- 我们可以创建额外的数据结构。
- 该问题在内存中可处理。

## 示例用法

以下是该函数应如何工作的一些示例：

- 空链表 -> []
- 单元素链表 -> [元素]
- 左链表为空 -> [10, 11, 12]
- 右链表为空 -> [1, 2, 3]
- 一般情况
  - 分区值 = 10
  - 输入：4, 3, 7, 8, 10, 1, 10, 12
  - 输出：4, 3, 7, 8, 1, 10, 10, 12
