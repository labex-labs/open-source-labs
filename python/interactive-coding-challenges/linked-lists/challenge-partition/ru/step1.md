# Разделение

## Задача

Дана односвязный список. Разделите его вокруг значения x так, чтобы все узлы, меньшие x, располагались перед всеми узлами, большими или равными x. Функция должна вернуть новый связный список.

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Связный список не является цикличным и односвязным.
- Функция должна возвращать новый связный список.
- Входное значение x является допустимым.
- У нас уже есть класс связного списка, который можно использовать для этой задачи.
- Мы можем создавать дополнительные структуры данных.
- Задача помещается в память.

## Пример использования

Ниже приведены примеры работы функции:

- Пустой список -> []
- Список из одного элемента -> [элемент]
- Левый связный список пуст -> [10, 11, 12]
- Правый связный список пуст -> [1, 2, 3]
- Общий случай
  - Разделение = 10
  - Вход: 4, 3, 7, 8, 10, 1, 10, 12
  - Выход: 4, 3, 7, 8, 1, 10, 10, 12
