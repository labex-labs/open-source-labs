# Связный список

## Задача

Реализовать связный список с методами:

- insert(value): вставляет новый узел с заданным значением в начало списка
- append(value): вставляет новый узел с заданным значением в конец списка
- find(value): возвращает первый узел в списке с заданным значением, или None, если такого узла нет
- delete(value): удаляет первый узел в списке с заданным значением, или ничего не делает, если такого узла нет
- length(): возвращает количество узлов в списке
- print(): выводит значения всех узлов в списке, разделенные пробелами

## Требования

Реализация связного списка должна соответствовать следующим требованиям:

- Связный список нециклический и односвязный.
- Реализация отслеживает только голову списка, а не хвост.
- Значения None нельзя вставлять в список.

## Пример использования

### Вставка в начало

- Вставка None: вызывает ошибку, так как значения None нельзя вставлять в список.
- Вставка в пустой список: вставляет значение в качестве первого узла в списке.
- Вставка в список с одним или более элементами: вставляет значение в качестве первого узла в списке, сдвигая существующие узлы вправо.

### Добавление в конец

- Добавление None: вызывает ошибку, так как значения None нельзя вставлять в список.
- Добавление в пустой список: вставляет значение в качестве первого узла в списке.
- Добавление в список с одним или более элементами: вставляет значение в качестве последнего узла в списке, обновляя ссылку предыдущего последнего узла, чтобы она указывала на новый узел.

### Поиск

- Поиск None: возвращает None, так как значения None не могут быть найдены в списке.
- Поиск в пустом списке: возвращает None, так как в списке нет узлов.
- Поиск в списке с одним или более совпадающими элементами: возвращает первый узел в списке с заданным значением.
- Поиск в списке без совпадений: возвращает None, так как в списке нет узлов с заданным значением.

### Удаление

- Удаление None: ничего не делает, так как значения None нельзя удалить из списка.
- Удаление в пустом списке: ничего не делает, так как в списке нет узлов.
- Удаление в списке с одним или более совпадающими элементами: удаляет первый узел в списке с заданным значением, сдвигая существующие узлы влево.
- Удаление в списке без совпадений: ничего не делает, так как в списке нет узлов с заданным значением.

### Длина

- Длина списка из нуля или более элементов: возвращает количество узлов в списке.

### Вывод

- Вывод пустого списка: ничего не выводит.
- Вывод списка с одним или более элементами: выводит значения всех узлов в списке, разделенные пробелами.
