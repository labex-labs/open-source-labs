# 連結リスト

## 問題

以下のメソッドを持つ連結リストを実装します。

- insert(value)：指定された値を持つ新しいノードをリストの先頭に挿入します。
- append(value)：指定された値を持つ新しいノードをリストの末尾に挿入します。
- find(value)：指定された値を持つリストの最初のノードを返します。存在しない場合はNoneを返します。
- delete(value)：指定された値を持つリストの最初のノードを削除します。存在しない場合は何もしません。
- length()：リスト内のノード数を返します。
- print()：リスト内のすべてのノードの値を空白区切りで表示します。

## 要件

連結リストの実装は、以下の要件を満たす必要があります。

- 連結リストは非循環的で単方向連結です。
- 実装では、リストの先頭のみを追跡し、末尾を追跡しません。
- None値はリストに挿入できません。

## 使い方の例

### 先頭に挿入

- Noneを挿入する：None値はリストに挿入できないため、エラーが発生します。
- 空のリストに挿入する：値をリストの最初のノードとして挿入します。
- 1つ以上の要素を持つリストに挿入する：値をリストの最初のノードとして挿入し、既存のノードを右にシフトします。

### 末尾に追加

- Noneを追加する：None値はリストに挿入できないため、エラーが発生します。
- 空のリストに追加する：値をリストの最初のノードとして挿入します。
- 1つ以上の要素を持つリストに追加する：値をリストの最後のノードとして挿入し、前の最後のノードの参照を新しいノードに向けるように更新します。

### 検索

- Noneを検索する：None値はリストに見つからないため、Noneを返します。
- 空のリストを検索する：リストにノードがないため、Noneを返します。
- 1つ以上の一致する要素を持つリストを検索する：指定された値を持つリストの最初のノードを返します。
- 一致する要素がないリストを検索する：指定された値を持つリストのノードがないため、Noneを返します。

### 削除

- Noneを削除する：None値はリストから削除できないため、何もしません。
- 空のリストを削除する：リストにノードがないため、何もしません。
- 1つ以上の一致する要素を持つリストを削除する：指定された値を持つリストの最初のノードを削除し、既存のノードを左にシフトします。
- 一致する要素がないリストを削除する：指定された値を持つリストのノードがないため、何もしません。

### 長さ

- 0個以上の要素の長さ：リスト内のノード数を返します。

### 表示

- 空のリストを表示する：何も表示されません。
- 1つ以上の要素を持つリストを表示する：リスト内のすべてのノードの値を空白区切りで表示します。
