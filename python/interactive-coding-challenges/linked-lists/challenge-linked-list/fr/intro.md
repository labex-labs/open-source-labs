# Introduction

Une liste chaînée est une structure de données composée d'une séquence de nœuds, où chaque nœud contient une valeur et une référence au nœud suivant dans la séquence. Les listes chaînées sont couramment utilisées pour implémenter d'autres structures de données, telles que les piles, les files d'attente et les tables de hachage.
