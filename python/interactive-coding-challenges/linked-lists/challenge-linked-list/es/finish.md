# Resumen

En resumen, una lista enlazada es una estructura de datos que consta de una secuencia de nodos, donde cada nodo contiene un valor y una referencia al siguiente nodo en la secuencia. La implementación de la lista enlazada debe tener métodos de inserción, inserción al final, búsqueda, eliminación, longitud y impresión, y cumplir con los requisitos de ser no circular y simplemente enlazada, solo llevar un registro de la cabeza de la lista y no permitir valores None. El uso de ejemplo proporciona escenarios para cada método para ilustrar su funcionalidad.
