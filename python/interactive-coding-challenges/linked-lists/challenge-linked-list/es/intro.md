# Introducción

Una lista enlazada es una estructura de datos que consta de una secuencia de nodos, donde cada nodo contiene un valor y una referencia al siguiente nodo en la secuencia. Las listas enlazadas se utilizan comúnmente para implementar otras estructuras de datos, como pilas, colas y tablas hash.
