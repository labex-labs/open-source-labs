# Резюме

Инверсия двоичного дерева - это распространённая задача в информатике. Обменом всех пар узлов слева и справа мы можем создать новое дерево, которое является зеркальным изображением исходного дерева. Чтобы решить эту задачу, нам нужно иметь класс `Node`, представляющий узел в двоичном дереве, и мы должны优雅но обрабатывать недействительные входные данные.
