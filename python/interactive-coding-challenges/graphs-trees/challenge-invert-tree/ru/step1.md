# Инвертировать дерево

## Задача

Дано двоичное дерево. Напишите функцию для инвертирования дерева. Функция должна принимать корневой узел дерева в качестве входных данных и возвращать новый корневой узел инвертированного дерева.

## Требования

Для решения этой задачи вам нужно удовлетворить следующим требованиям:

- Должен быть класс `Node`, представляющий узел в двоичном дереве.
- Нужно поменять местами все пары узлов слева и справа в двоичном дереве.
- Нужно优雅но обрабатывать недействительные входные данные.
- Ваше решение должно помещаться в память.

## Пример использования

Предположим, у нас есть следующее двоичное дерево:

```txt
     5
   /   \
  2     7
 / \   / \
1   3 6   9
```

После инвертирования дерева мы должны получить:

```txt
     5
   /   \
  7     2
 / \   / \
9   6 3   1
```
