# 2 分木を反転させる

## 問題

2 分木が与えられたとき、その木を反転させる関数を書きなさい。この関数は、木の根ノードを入力として受け取り、反転した木の新しい根ノードを返す必要があります。

## 要件

この問題を解くには、次の要件を満たす必要があります。

- 2 分木のノードを表す Node クラスが必要です。
- 2 分木のすべての左と右のノードのペアを交換する必要があります。
- 無効な入力を適切に処理する必要があります。
- ソリューションはメモリに収まる必要があります。

## 使い方の例

次の 2 分木があるとしましょう。

```txt
     5
   /   \
  2     7
 / \   / \
1   3 6   9
```

木を反転させた後、次のようになるはずです。

```txt
     5
   /   \
  7     2
 / \   / \
9   6 3   1
```
