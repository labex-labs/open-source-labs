# Résumé

Inverser un arbre binaire est un problème courant en informatique. En échangeant toutes les paires de nœuds gauche et droit, nous pouvons créer un nouvel arbre qui est l'image miroir de l'arbre original. Pour résoudre ce problème, nous devons avoir une classe `Node` qui représente un nœud dans l'arbre binaire, et nous devons gérer les entrées invalides de manière appropriée.
