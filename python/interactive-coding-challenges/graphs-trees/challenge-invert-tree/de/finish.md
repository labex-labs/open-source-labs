# Zusammenfassung

Das Invertieren eines binären Baumes ist ein häufiges Problem in der Informatik. Indem wir alle linken und rechten Knotenpaare austauschen, können wir einen neuen Baum erstellen, der das Spiegelbild des ursprünglichen Baumes ist. Um dieses Problem zu lösen, müssen wir eine Node-Klasse haben, die einen Knoten im binären Baum repräsentiert, und wir müssen ungültige Eingaben优雅地 (gracefully) behandeln.
