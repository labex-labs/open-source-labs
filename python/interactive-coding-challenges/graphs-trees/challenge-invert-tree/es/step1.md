# Invertir árbol

## Problema

Dado un árbol binario, escribe una función para invertir el árbol. La función debe tomar el nodo raíz del árbol como entrada y devolver el nuevo nodo raíz del árbol invertido.

## Requisitos

Para resolver este problema, debes cumplir con los siguientes requisitos:

- Deberías tener una clase `Node` que represente un nodo en el árbol binario.
- Debes intercambiar todos los pares de nodos izquierdos y derechos en el árbol binario.
- Debes manejar las entradas no válidas de manera adecuada.
- Tu solución debe caber en memoria.

## Uso de ejemplo

Supongamos que tenemos el siguiente árbol binario:

```txt
     5
   /   \
  2     7
 / \   / \
1   3 6   9
```

Después de invertir el árbol, deberíamos obtener:

```txt
     5
   /   \
  7     2
 / \   / \
9   6 3   1
```
