# Resumen

Invertir un árbol binario es un problema común en informática. Al intercambiar todos los pares de nodos izquierdos y derechos, podemos crear un nuevo árbol que es el espejo del árbol original. Para resolver este problema, necesitamos tener una clase `Node` que represente un nodo en el árbol binario, y debemos manejar adecuadamente las entradas no válidas.
