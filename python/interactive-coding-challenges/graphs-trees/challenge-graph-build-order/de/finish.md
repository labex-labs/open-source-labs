# Zusammenfassung

Zusammenfassend betrifft das Build-Order-Problem das Finden einer gültigen Reihenfolge, in der eine Liste von Projekten und ihren Abhängigkeiten gebaut werden sollen. Wir können dieses Problem lösen, indem wir die Anforderungen berücksichtigen und eine Graph- und Node-Klasse verwenden, um die Eingabe zu repräsentieren.
