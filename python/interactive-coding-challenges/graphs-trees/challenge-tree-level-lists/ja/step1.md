# 木のレベルのリスト

## 問題

二分探索木が与えられたとき、木の各レベルに対してリストを作成します。各リストは、そのレベルのノードを含む必要があります。リストは、2次元配列として返され、各サブ配列は木の1つのレベルを表します。

## 要件

この問題を解くには、以下の要件を満たす必要があります。

- 与えられた木は二分探索木です。
- 木の各レベルは、ノードのリストで表される必要があります。
- 挿入メソッドを持つNodeクラスが既に提供されています。
- ソリューションはメモリに収まる必要があります。

## 例の使用法

たとえば、以下の値を持つ二分探索木が与えられた場合：

```
5, 3, 8, 2, 4, 1, 7, 6, 9, 10, 11
```

関数は、以下の2次元配列を返す必要があります。

```
[[5], [3, 8], [2, 4, 7, 9], [1, 6, 10], [11]]
```

結果の各数字は、実際にはその数字を含むノードであることに注意してください。
