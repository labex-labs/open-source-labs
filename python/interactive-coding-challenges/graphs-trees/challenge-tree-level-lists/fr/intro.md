# Introduction

En informatique, un arbre binaire est une structure de données arborescente dans laquelle chaque nœud a au plus deux enfants, appelés l'enfant gauche et l'enfant droit. Dans ce défi, nous allons créer une liste pour chaque niveau d'un arbre binaire.
