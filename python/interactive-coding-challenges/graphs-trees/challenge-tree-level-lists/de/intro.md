# Einführung

In der Informatik ist ein Binärbaum eine Bäume-Datenstruktur, in der jeder Knoten maximal zwei Kinder hat, die als linker und rechter Kind bezeichnet werden. In dieser Herausforderung werden wir eine Liste für jede Ebene eines Binärbaums erstellen.
