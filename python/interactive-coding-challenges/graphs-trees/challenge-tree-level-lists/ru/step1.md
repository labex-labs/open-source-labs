# Списки по уровням дерева

## Задача

Дано двоичное дерево поиска. Создайте список для каждого уровня дерева. Каждый список должен содержать узлы соответствующего уровня дерева. Списки должны быть возвращены в виде массива массивов, где каждый подмассив представляет собой уровень дерева.

## Требования

Для решения этой задачи необходимо выполнить следующие требования:

- Данное дерево является двоичным деревом поиска.
- Каждый уровень дерева должен быть представлен списком узлов.
- Класс `Node` с методом `insert` уже предоставлен.
- Решение должно помещаться в память.

## Пример использования

Например, дано двоичное дерево поиска с такими значениями:

```
5, 3, 8, 2, 4, 1, 7, 6, 9, 10, 11
```

Функция должна вернуть следующий массив массивов:

```
[[5], [3, 8], [2, 4, 7, 9], [1, 6, 10], [11]]
```

Обратите внимание, что каждое число в результате на самом деле является узлом, содержащим это число.
