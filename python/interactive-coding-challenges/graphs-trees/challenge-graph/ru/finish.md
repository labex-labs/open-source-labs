# Резюме

В этом испытании мы реализовали граф, который может быть направленным или неориентированным, иметь взвешенные ребра и циклы. Мы также удостоверились, что добавление или удаление узлов, которые уже существуют или не существуют соответственно, не влияет на граф. Класс Graph можно использовать в качестве строительного блока для более сложных задач с графами.
