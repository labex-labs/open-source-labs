# Граф

## Задача

Реализовать граф, удовлетворяющий следующим требованиям:

### Требования

- Граф может быть направленным или неориентированным.
- Ребра могут иметь веса.
- Граф может содержать циклы.
- Если мы пытаемся добавить узел, который уже существует, мы ничего не делаем.
- Если мы пытаемся удалить узел, который не существует, мы ничего не делаем.
- Можно предположить, что это связный граф.
- Можно предположить, что входные данные действительны.
- Можно предположить, что это помещается в память.

## Пример использования

Вход:

```
graph.add_edge(0, 1, 5)
graph.add_edge(0, 5, 2)
graph.add_edge(1, 2, 3)
graph.add_edge(2, 3, 4)
graph.add_edge(3, 4, 5)
graph.add_edge(3, 5, 6)
graph.add_edge(4, 0, 7)
graph.add_edge(5, 4, 8)
graph.add_edge(5, 2, 9)
```

Результат:

- Узлы `0`, `1`, `2`, `3`, `4` и `5` соединены с указанными весами.

Примечание:

- Класс `Graph` будет использоваться в качестве строительного блока для более сложных задач с графами.
