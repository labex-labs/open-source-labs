# Resumen

En este desafío, implementamos un gráfico que puede ser dirigido o no dirigido, tener aristas con pesos y ciclos. También garantizamos que agregar o eliminar nodos que ya existen o no existen respectivamente, no afecte el gráfico. La clase `Graph` se puede utilizar como componente básico para desafíos de gráfico más complejos.
