# グラフ

## 問題

次の要件を満たすグラフを実装します。

### 要件

- グラフは有向グラフまたは無向グラフであることができる。
- エッジには重みを持たせることができる。
- グラフにサイクルがあってもよい。
- 既に存在するノードを追加しようとした場合、何もしない。
- 存在しないノードを削除しようとした場合、何もしない。
- これは連結グラフであると仮定できる。
- 入力が有効であると仮定できる。
- これがメモリに収まると仮定できる。

## 例の使用法

入力:

```
graph.add_edge(0, 1, 5)
graph.add_edge(0, 5, 2)
graph.add_edge(1, 2, 3)
graph.add_edge(2, 3, 4)
graph.add_edge(3, 4, 5)
graph.add_edge(3, 5, 6)
graph.add_edge(4, 0, 7)
graph.add_edge(5, 4, 8)
graph.add_edge(5, 2, 9)
```

結果:

- ノード `0`、`1`、`2`、`3`、`4`、および `5` が指定された重みで接続されている。

注:

- グラフクラスは、より複雑なグラフチャレンジのビルディングブロックとして使用されます。
