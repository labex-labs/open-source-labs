# Einführung

Ein Graph ist eine nicht-lineare Datenstruktur, die aus Knoten und Kanten besteht. Es wird verwendet, um Verbindungen oder Beziehungen zwischen Objekten darzustellen. Graphen werden in der Informatik, der Technik und den Sozialwissenschaften weit verbreitet eingesetzt.
