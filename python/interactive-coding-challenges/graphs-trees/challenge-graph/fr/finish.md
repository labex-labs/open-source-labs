# Sommaire

Dans ce défi, nous avons implémenté un graphe qui peut être dirigé ou non dirigé, avoir des arêtes pondérées et des cycles. Nous avons également veillé à ce que l'ajout ou la suppression de nœuds qui existent déjà ou n'existent pas respectivement, n'affecte pas le graphe. La classe Graph peut être utilisée comme base pour des défis de graphe plus complexes.
