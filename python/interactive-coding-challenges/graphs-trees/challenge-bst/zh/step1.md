# 二叉搜索树

## 问题

二叉搜索树是一种数据结构，它支持快速的查找、插入和删除操作。它是一棵树，其中每个节点最多有两个子节点，左子节点的值小于父节点，右子节点的值大于父节点。插入方法会根据值将一个新节点添加到树中的适当位置。

你的任务是在Python中实现一个带有插入方法的二叉搜索树。插入方法应接受一个值，并根据该值将一个新节点添加到树中的适当位置。如果根节点输入为None，则返回一个仅包含新根节点的树。

## 要求

要完成此挑战，你需要满足以下要求：

- 你不能插入None值。
- 你可以假设你处理的是有效的整数。
- 你可以假设所有左子节点都小于或等于该节点，所有右子节点都大于该节点。
- 你不必跟踪父节点，但这是可选的。
- 你可以假设这适合内存。
