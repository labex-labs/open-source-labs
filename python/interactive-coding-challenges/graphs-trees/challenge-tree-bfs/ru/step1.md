# Обход в ширину для дерева

## Задача

Дано двоичное дерево. Реализуйте функцию, которая выполняет обход в ширину для этого дерева. Функция должна вызывать метод `visit_func` для каждого узла при обработке узла.

## Требования

Для решения этой задачи необходимо выполнить следующие требования:

- Класс `Node` с методом `insert` уже доступен.
- Решение должно помещаться в память.
- Метод `visit_func` должен вызываться для каждого узла при обработке узла.

## Пример

### Обход в ширину

Предположим, у нас есть двоичное дерево с такой структурой:

```
    5
   / \
  2   8
 / \
1   3
```

Выполнение обхода в ширину для этого дерева приведет к последовательности посещенных узлов: `5, 2, 8, 1, 3`.
