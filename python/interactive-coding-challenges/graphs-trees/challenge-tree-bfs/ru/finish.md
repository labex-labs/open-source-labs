# Резюме

В этом задании на Python мы реализовали обход в ширину для двоичного дерева. Для решения требовался класс `Node` с методом `insert`, и метод `visit_func` вызывался для каждого узла при его обработке. Решение помещалось в память и при обходе в ширину по двоичному дереву выдавало правильную последовательность узлов.
