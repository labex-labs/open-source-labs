# Zusammenfassung

In dieser Python-Aufgabe haben wir die Breitensuche in einem binären Baum implementiert. Die Lösung erforderte eine `Node`-Klasse mit einer `insert`-Methode, und die `visit_func`-Methode wurde bei der Verarbeitung jedes Knotens aufgerufen. Die Lösung passte in den Speicher und erzeugte die korrekte Sequenz von Knoten, wenn eine Breitensuche auf einem binären Baum durchgeführt wurde.
