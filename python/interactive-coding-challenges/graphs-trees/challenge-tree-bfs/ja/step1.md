# 木の幅優先探索

## 問題

2 分木が与えられたとき、その木に対して幅優先探索を行う関数を実装します。この関数は、各ノードが処理される際に入力メソッド `visit_func` を呼び出す必要があります。

## 要件

この問題を解くには、以下の要件を満たす必要があります。

- 挿入メソッドを持つ `Node` クラスが既に用意されています。
- ソリューションはメモリに収まる必要があります。
- 各ノードが処理される際に `visit_func` メソッドが呼び出される必要があります。

## 例

### 幅優先探索

次の構造を持つ 2 分木があるとします。

```
    5
   / \
  2   8
 / \
1   3
```

この木に対して幅優先探索を行うと、次のノードの訪問順序が得られます。`5, 2, 8, 1, 3`。
