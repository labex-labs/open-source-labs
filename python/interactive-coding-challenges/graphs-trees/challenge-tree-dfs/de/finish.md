# Zusammenfassung

In dieser Herausforderung haben wir gelernt, wie man Tiefendurchläufe (in-order, pre-order, post-order) auf einem binären Baum implementiert. Wir haben auch gelernt, dass die Tiefensuche (Depth-First Search, DFS) ein beliebiger Algorithmus zum Durchlaufen von Bäumen und Graphen ist.
