# 木の DFS

## 問題

2 分木に対して深さ優先探索（中順、前順、後順）を実装します。中順探索では、左の部分木を訪問し、次に現在のノードを訪問し、そして右の部分木を訪問します。前順探索では、現在のノードを訪問し、次に左の部分木を訪問し、そして右の部分木を訪問します。後順探索では、左の部分木を訪問し、次に右の部分木を訪問し、そして現在のノードを訪問します。

## 要件

このチャレンジを完了するには、次の要件を満たす必要があります。

- 既に insert メソッドを持つ Node クラスがあると仮定できます。
- 各ノードを処理する際に、ノードに対して入力メソッド`visit_func`を呼び出す必要があります。
- これがメモリに収まると仮定できます。

## 使い方の例

以下は、DFS アルゴリズムの使い方のいくつかの例です。

### 中順探索

値が 5、2、8、1、3 の 2 分木の場合、中順探索の結果は 1、2、3、5、8 になります。値が 1、2、3、4、5 の 2 分木の場合、中順探索の結果は 1、2、3、4、5 になります。

### 前順探索

値が 5、2、8、1、3 の 2 分木の場合、前順探索の結果は 5、2、1、3、8 になります。値が 1、2、3、4、5 の 2 分木の場合、前順探索の結果は 1、2、3、4、5 になります。

### 後順探索

値が 5、2、8、1、3 の 2 分木の場合、後順探索の結果は 1、3、2、8、5 になります。値が 1、2、3、4、5 の 2 分木の場合、後順探索の結果は 5、4、3、2、1 になります。
