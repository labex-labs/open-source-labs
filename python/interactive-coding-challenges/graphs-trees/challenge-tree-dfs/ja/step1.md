# 木のDFS

## 問題

2分木に対して深さ優先探索（中順、前順、後順）を実装します。中順探索では、左の部分木を訪問し、次に現在のノードを訪問し、そして右の部分木を訪問します。前順探索では、現在のノードを訪問し、次に左の部分木を訪問し、そして右の部分木を訪問します。後順探索では、左の部分木を訪問し、次に右の部分木を訪問し、そして現在のノードを訪問します。

## 要件

このチャレンジを完了するには、次の要件を満たす必要があります。

- 既にinsertメソッドを持つNodeクラスがあると仮定できます。
- 各ノードを処理する際に、ノードに対して入力メソッド`visit_func`を呼び出す必要があります。
- これがメモリに収まると仮定できます。

## 使い方の例

以下は、DFSアルゴリズムの使い方のいくつかの例です。

### 中順探索

値が5、2、8、1、3の2分木の場合、中順探索の結果は1、2、3、5、8になります。値が1、2、3、4、5の2分木の場合、中順探索の結果は1、2、3、4、5になります。

### 前順探索

値が5、2、8、1、3の2分木の場合、前順探索の結果は5、2、1、3、8になります。値が1、2、3、4、5の2分木の場合、前順探索の結果は1、2、3、4、5になります。

### 後順探索

値が5、2、8、1、3の2分木の場合、後順探索の結果は1、3、2、8、5になります。値が1、2、3、4、5の2分木の場合、後順探索の結果は5、4、3、2、1になります。
