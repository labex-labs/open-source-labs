# 简介

二叉搜索树是一种用于以有序方式存储数据的数据结构。树中的每个节点最多有两个子节点，左子节点始终小于父节点，而右子节点始终大于父节点。在这个挑战中，我们将找到二叉搜索树中的第二大节点。
