# Введение

Двоичное дерево поиска - это структура данных, которая используется для хранения данных в отсортированном виде. Каждая вершина в дереве имеет не более двух дочерних вершин, и левая дочерняя вершина всегда меньше родительской, а правая дочерняя вершина всегда больше родительской. В этом испытании мы будем искать вторую по величине вершину в двоичном дереве поиска.
