# Einführung

Ein binärer Suchbaum ist eine Datenstruktur, die verwendet wird, um Daten in aufsteigender Reihenfolge zu speichern. Jeder Knoten im Baum hat maximal zwei Kinder, wobei das linke Kind immer kleiner als der Elternknoten ist, während das rechte Kind immer größer als der Elternknoten ist. In dieser Herausforderung werden wir den zweitgrößten Knoten in einem binären Suchbaum finden.
