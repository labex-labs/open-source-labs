# Introduction

Un arbre binaire de recherche est une structure de données utilisée pour stocker des données de manière triée. Chaque nœud de l'arbre a au plus deux enfants, et l'enfant gauche est toujours inférieur au parent, tandis que l'enfant droit est toujours supérieur au parent. Dans ce défi, nous allons trouver le deuxième nœud le plus grand dans un arbre binaire de recherche.
