# Introducción

Un árbol de búsqueda binaria es una estructura de datos que se utiliza para almacenar datos de manera ordenada. Cada nodo en el árbol tiene como máximo dos hijos, y el hijo izquierdo siempre es menor que el padre, mientras que el hijo derecho siempre es mayor que el padre. En este desafío, encontraremos el segundo nodo más grande en un árbol de búsqueda binaria.
