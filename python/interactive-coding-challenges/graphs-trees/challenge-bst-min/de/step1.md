# Bst Min

## Problem

Gegeben ein sortiertes Array, müssen wir einen binären Suchbaum mit minimaler Höhe erstellen. Die Höhe eines binären Suchbaums ist die Anzahl der Kanten zwischen der Wurzel des Baums und seinem weitesten Blatt. Ziel ist es, einen binären Suchbaum mit möglichst minimaler Höhe zu erstellen.

## Anforderungen

Um dieses Problem zu lösen, müssen wir die folgenden Anforderungen erfüllen:

- Das Array muss in aufsteigender Reihenfolge vorliegen.
- Die Arrayelemente müssen eindeutig sein.
- Wir müssen davon ausgehen, dass wir bereits eine Node-Klasse mit einer insert-Methode haben.
- Wir müssen davon ausgehen, dass dies in den Speicher passt.

## Beispielverwendung

Hier sind einige Beispiele, wie die Funktion verwendet werden kann:

- Eingabe: [0, 1, 2, 3, 4, 5, 6]
  Ausgabe: Ein binärer Suchbaum mit Höhe 3

- Eingabe: [0, 1, 2, 3, 4, 5, 6, 7]
  Ausgabe: Ein binärer Suchbaum mit Höhe 4
