# Bst Min

## 問題

ソート済み配列が与えられたとき、最小の高さを持つ二分探索木を作成する必要があります。二分探索木の高さは、木の根と最も遠い葉の間の辺の数です。目的は、可能な限り最小の高さを持つ二分探索木を作成することです。

## 要件

この問題を解くには、次の要件を満たす必要があります。

- 配列は昇順でなければなりません。
- 配列要素は一意でなければなりません。
- 既にinsertメソッドを持つNodeクラスがあると仮定します。
- これがメモリに収まると仮定します。

## 使い方の例

ここに、関数の使い方のいくつかの例を示します。

- 入力: [0, 1, 2, 3, 4, 5, 6]
  出力: 高さ3の二分探索木

- 入力: [0, 1, 2, 3, 4, 5, 6, 7]
  出力: 高さ4の二分探索木
