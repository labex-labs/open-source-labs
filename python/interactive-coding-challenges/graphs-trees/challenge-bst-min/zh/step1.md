# 最小高度二叉搜索树

## 问题

给定一个有序数组，我们需要创建一个高度最小的二叉搜索树。二叉搜索树的高度是树根与其最远叶子之间的边数。目标是创建一个尽可能高度最小的二叉搜索树。

## 要求

为了解决这个问题，我们需要满足以下要求：

- 数组必须是升序排列的。
- 数组元素必须是唯一的。
- 我们必须假设已经有一个带有插入方法的节点类（Node class）。
- 我们必须假设这在内存中是可行的。

## 示例用法

以下是一些如何使用该函数的示例：

- 输入：[0, 1, 2, 3, 4, 5, 6]
  输出：一个高度为3的二叉搜索树
- 输入：[0, 1, 2, 3, 4, 5, 6, 7]
  输出：一个高度为4的二叉搜索树
