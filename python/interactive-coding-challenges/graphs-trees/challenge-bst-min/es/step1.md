# Bst Min

## Problema

Dada una matriz ordenada, necesitamos crear un árbol de búsqueda binario con la altura mínima. La altura de un árbol de búsqueda binario es el número de aristas entre la raíz del árbol y su hoja más alejada. El objetivo es crear un árbol de búsqueda binario con la altura posiblemente mínima.

## Requisitos

Para resolver este problema, debemos cumplir con los siguientes requisitos:

- La matriz debe estar en orden ascendente.
- Los elementos de la matriz deben ser únicos.
- Debemos suponer que ya tenemos una clase Nodo con un método insertar.
- Debemos suponer que esto cabe en memoria.

## Uso de ejemplo

A continuación se presentan algunos ejemplos de cómo usar la función:

- Entrada: [0, 1, 2, 3, 4, 5, 6]
  Salida: Un árbol de búsqueda binario con altura 3

- Entrada: [0, 1, 2, 3, 4, 5, 6, 7]
  Salida: Un árbol de búsqueda binario con altura 4
