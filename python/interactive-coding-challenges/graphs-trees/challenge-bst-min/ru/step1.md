# Минимум в BST

## Задача

Дан отсортированный массив, и нам нужно создать двоичное дерево поиска с минимальной высотой. Высота двоичного дерева поиска — это количество ребер между корнем дерева и его самой удалённой листовой вершиной. Задача — создать двоичное дерево поиска с минимально возможной высотой.

## Требования

Для решения этой задачи необходимо выполнить следующие требования:

- Массив должен быть отсортирован по возрастанию.
- Элементы массива должны быть уникальными.
- Мы должны предположить, что у нас уже есть класс Node с методом insert.
- Мы должны предположить, что это помещается в память.

## Пример использования

Вот несколько примеров использования функции:

- Вход: [0, 1, 2, 3, 4, 5, 6]
  Выход: Двоичное дерево поиска с высотой 3

- Вход: [0, 1, 2, 3, 4, 5, 6, 7]
  Выход: Двоичное дерево поиска с высотой 4
