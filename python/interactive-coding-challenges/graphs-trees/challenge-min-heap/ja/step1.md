# ミニヒープ

## 問題

以下のメソッドを持つミニヒープを実装します。

- `extract_min()`: ヒープ内の最小値を削除して返します
- `insert(value)`: 新しい値をヒープに挿入しながら、ヒーププロパティを維持します

## 要件

実装は以下の要件を満たす必要があります。

- 入力は整数です
- 実装はメモリ内に収まる必要があります

## 使い方の例

以下のミニヒープを考えます。

```txt
          _5_
        /     \
       20     15
      / \    /  \
     22  40 25
```

- `extract_min()`: ヒープ内の最小値を削除して返します。最小値は 5 です。結果のヒープは以下の通りです。

```txt
          _15_
        /      \
       20      25
      / \     /  \
     22  40
```

- `insert(2)`: 値 2 をヒープに挿入しながら、ヒーププロパティを維持します。結果のヒープは以下の通りです。

```txt
          _2_
        /     \
       20      5
      / \     / \
     22  40  25  15
```
