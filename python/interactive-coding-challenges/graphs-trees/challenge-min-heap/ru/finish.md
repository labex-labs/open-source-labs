# Резюме

Кратко говоря, мини-куча - это двоичное дерево, представляющее собой структуру данных, которая сохраняет свойство кучи, при котором родительская вершина всегда меньше или равна своим дочерним вершинам. Реализация мини-кучи должна включать методы `extract_min()` и `insert(value)` и соответствовать требованиям целых чисел в качестве входных данных и помещения в память.
