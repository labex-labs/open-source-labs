# Мини-куча

## Задача

Реализовать мини-кучу с методами:

- `extract_min()`: удаляет и возвращает минимальное значение в куче
- `insert(value)`: вставляет новое значение в кучу, сохраняя свойство кучи

## Требования

Реализация должна соответствовать следующим требованиям:

- Входные данные - целые числа
- Реализация должна помещаться в память

## Пример использования

Рассмотрим следующую мини-кучу:

```txt
          _5_
        /     \
       20     15
      / \    /  \
     22  40 25
```

- `extract_min()`: удаляет и возвращает минимальное значение в куче, которое равно 5. Результат:

```txt
          _15_
        /      \
       20      25
      / \     /  \
     22  40
```

- `insert(2)`: вставляет значение 2 в кучу, сохраняя свойство кучи. Результат:

```txt
          _2_
        /     \
       20      5
      / \     / \
     22  40  25  15
```
