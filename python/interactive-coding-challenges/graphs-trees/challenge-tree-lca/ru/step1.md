# Дерево Lca

## Задача

Дано двоичное дерево и два узла. Найти их наименьшего общего предка.

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Данное дерево является двоичным деревом, а не двоичным деревом поиска.
- Мы не можем предполагать, что два узла уже находятся в дереве.
- Мы можем предположить, что двоичное дерево помещается в память.

## Пример использования

Рассмотрим следующее двоичное дерево:

```txt
          _10_
        /      \
       5        9
      / \      / \
     12  3    18  20
        / \       /
       1   8     40
```

Мы можем протестировать нашу функцию с следующими входными данными и ожидаемыми результатами:

- 0, 5 -> None (оба узла не находятся в дереве)
- 5, 0 -> None (оба узла не находятся в дереве)
- 1, 8 -> 3
- 12, 8 -> 5
- 12, 40 -> 10
- 9, 20 -> 9
- 3, 5 -> 5
