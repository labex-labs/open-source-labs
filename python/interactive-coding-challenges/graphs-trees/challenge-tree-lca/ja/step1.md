# 木のLca

## 問題

2分木と2つのノードが与えられたとき、それらの最小共通祖先を見つけます。

## 要件

この問題を解くには、次の要件を考慮する必要があります。

- 与えられた木は2分木であり、2分探索木ではありません。
- 2つのノードが既に木に含まれているとは仮定できません。
- 2分木がメモリに収まると仮定できます。

## 例の使用法

次の2分木を考えてみましょう。

```txt
          _10_
        /      \
       5        9
      / \      / \
     12  3    18  20
        / \       /
       1   8     40
```

次の入力と期待される出力で関数をテストできます。

- 0, 5 -> None (両方のノードが木に含まれていません)
- 5, 0 -> None (両方のノードが木に含まれていません)
- 1, 8 -> 3
- 12, 8 -> 5
- 12, 40 -> 10
- 9, 20 -> 9
- 3, 5 -> 5
