# 简介

在计算机科学中，二叉搜索树（BST）是一种二叉树数据结构，其中树中每个节点的值大于或等于其左子树中的任何值，并且小于或等于其右子树中的任何值。在这个挑战中，我们要在二叉搜索树中找到给定节点的中序后继。
