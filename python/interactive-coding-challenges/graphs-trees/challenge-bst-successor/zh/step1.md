# 二叉搜索树的后继节点

## 问题

给定一棵二叉搜索树和其中的一个节点，找到该节点的中序后继。节点的后继是在树的中序遍历中紧跟在给定节点之后出现的节点。如果没有后继节点，则返回 `None`。如果输入为 `None`，则抛出异常。我们可以假设我们已经有一个跟踪父节点的 `Node` 类，并且可以假设这符合内存要求。

## 要求

- 函数应返回二叉搜索树中给定节点的中序后继。
- 如果没有后继节点，函数应返回 `None`。
- 如果输入为 `None`，函数应抛出异常。
- 我们可以假设我们已经有一个跟踪父节点的 `Node` 类。
- 我们可以假设这符合内存要求。

## 示例用法

```txt
          _5_
        /     \
       3       8
      / \    /   \
     2   4  6    12
    /        \   / \
   1          7 10  15
               /
              9

输入：None  输出：异常
输入：4     输出：5
输入：5     输出：6
输入：8     输出：9
输入：15    输出：None
```
