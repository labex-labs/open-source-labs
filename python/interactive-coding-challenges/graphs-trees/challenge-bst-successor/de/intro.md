# Einführung

In der Informatik ist ein binärer Suchbaum (englisch: binary search tree, BST) eine binäre Baumdatenstruktur, bei der der Wert jedes Knotens im Baum größer oder gleich als jeder Wert in seinem linken Teilbaum und kleiner oder gleich als jeder Wert in seinem rechten Teilbaum ist. In dieser Herausforderung werden wir den in-order Nachfolger eines gegebenen Knotens in einem binären Suchbaum finden.
