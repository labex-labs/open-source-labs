# Zusammenfassung

In dieser Herausforderung haben wir gelernt, wie man den in-order Nachfolger eines gegebenen Knotens in einem binären Suchbaum findet. Wir haben auch die Anforderungen der Funktion gelernt, darunter das Zurückgeben von None, wenn es keinen Nachfolger gibt, das Werfen einer Ausnahme, wenn die Eingabe None ist, und das Vorraussetzen, dass wir bereits eine Node-Klasse haben, die die Eltern verfolgt.
