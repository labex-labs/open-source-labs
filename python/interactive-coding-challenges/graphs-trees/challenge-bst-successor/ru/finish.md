# Резюме

В этом испытании мы научились искать преемника в порядке возрастания для заданного узла в двоичном дереве поиска. Мы также узнали требования к функции, включая возврат `None`, если преемника нет, выброс исключения, если входной параметр равен `None`, и предположение, что у нас уже есть класс `Node`, который хранит информацию о родителях.
