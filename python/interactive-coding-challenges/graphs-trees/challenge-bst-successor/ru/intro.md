# Введение

В информатике двоичное дерево поиска (BST) — это двоичное дерево, в котором значение каждого узла дерева больше или равно любому значению в его левом поддереве и меньше или равно любому значению в его правом поддереве. В этом испытании мы будем искать преемника в порядке возрастания для заданного узла в двоичном дереве поиска.
