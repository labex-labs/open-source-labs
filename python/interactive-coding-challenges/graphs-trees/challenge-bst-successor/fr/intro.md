# Introduction

En informatique, un arbre binaire de recherche (ABR) est une structure de données d'arbre binaire dans laquelle la valeur de chaque nœud de l'arbre est supérieure ou égale à toute valeur de son sous-arbre gauche et inférieure ou égale à toute valeur de son sous-arbre droit. Dans ce défi, nous allons trouver le successeur dans l'ordre d'un nœud donné dans un arbre binaire de recherche.
