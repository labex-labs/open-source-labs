# Sommaire

Dans ce défi, nous avons appris à trouver le successeur dans l'ordre d'un nœud donné dans un arbre binaire de recherche. Nous avons également appris les exigences de la fonction, y compris le renvoi de None s'il n'y a pas de successeur, la levée d'une exception si l'entrée est None, et en supposant que nous avons déjà une classe Node qui suit les parents.
