# 二分探索木の次のノード

## 問題

二分探索木とその中のノードが与えられたとき、そのノードの中順巡回の次のノードを見つけます。ノードの次のノードとは、木の中順巡回で与えられたノードの直後に現れるノードです。次のノードが存在しない場合は、None を返します。入力が None の場合は、例外を投げます。既に親を追跡する Node クラスがあると仮定してよく、これがメモリに収まるとも仮定してよいものとします。

## 要件

- 関数は、二分探索木における与えられたノードの中順巡回の次のノードを返す必要があります。
- 次のノードが存在しない場合は、関数は None を返す必要があります。
- 入力が None の場合は、関数は例外を投げる必要があります。
- 既に親を追跡する Node クラスがあると仮定してよいものとします。
- これがメモリに収まると仮定してよいものとします。

## 例

```txt
          _5_
        /     \
       3       8
      / \    /   \
     2   4  6    12
    /        \   / \
   1          7 10  15
               /
              9

入力: None  出力: 例外
入力: 4     出力: 5
入力: 5     出力: 6
入力: 8     出力: 9
入力: 15    出力: None
```
