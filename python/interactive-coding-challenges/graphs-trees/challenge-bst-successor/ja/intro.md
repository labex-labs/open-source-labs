# はじめに

コンピュータサイエンスにおいて、二分探索木（BST：Binary Search Tree）は、木の各ノードの値がその左部分木の任意の値以上であり、かつその右部分木の任意の値以下である二分木データ構造です。このチャレンジでは、二分探索木における与えられたノードの中順巡回の次のノードを見つけます。
