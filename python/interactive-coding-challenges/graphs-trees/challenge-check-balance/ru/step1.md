# Проверить сбалансированность

## Задача

Дано двоичное дерево. Напишите функцию на Python для определения, является ли оно сбалансированным. Двоичное дерево считается сбалансированным, если высоты двух поддеревьев любого узла различаются не более чем на единицу. Функция должна принимать корневой узел двоичного дерева в качестве входных данных и возвращать True, если дерево сбалансировано, и False в противном случае. Если входные данные равны None, функция должна вызывать исключение.

## Требования

Для решения этой задачи необходимо удовлетворить следующим требованиям:

- Сбалансированным деревом считается такое, где высоты двух поддеревьев любого узла не различаются более чем на 1.
- Если входные данные равны None, функция должна вызывать исключение.
- Можно предположить, что у нас уже есть класс Node с методом insert.
- Можно предположить, что программа помещается в память.

## Пример использования

Ниже приведены примеры того, как должна работать функция:

- None -> вызывать исключение
- 1 -> True
- 5, 3, 8, 1, 4 -> True
- 5, 3, 8, 9, 10 -> False
