# 简介

在计算机科学中，二叉树是一种树状数据结构，其中每个节点最多有两个子节点，分别称为左子节点和右子节点。平衡二叉树是指任何节点的两个子树的高度差最多为 1 的树。在这个挑战中，我们将编写一个 Python 程序来检查一棵二叉树是否平衡。
