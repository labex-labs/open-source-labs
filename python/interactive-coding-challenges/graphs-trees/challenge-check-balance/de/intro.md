# Einführung

In der Informatik ist ein Binärbaum eine Bäume-Datenstruktur, in der jeder Knoten maximal zwei Kinder hat, die als linker und rechter Kind bezeichnet werden. Ein balancierter Binärbaum ist ein Baum, in dem die Höhen der beiden Teilbäume eines beliebigen Knotens um höchstens eins voneinander abweichen. In dieser Herausforderung werden wir ein Python-Programm schreiben, um zu überprüfen, ob ein Binärbaum balanciert ist.
