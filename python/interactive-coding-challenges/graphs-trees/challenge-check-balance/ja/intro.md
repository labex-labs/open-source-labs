# はじめに

コンピュータサイエンスにおいて、二分木は、各ノードが最大 2 つの子ノードを持つ木構造であり、それぞれ左の子ノードと右の子ノードと呼ばれます。平衡二分木は、任意のノードの 2 つの部分木の高さの差が最大 1 である木です。このチャレンジでは、Python プログラムを書いて二分木が平衡しているかどうかをチェックします。
