# 無重みグラフの最短経路

## 問題

辺の重みのない有向グラフが与えられたとき、2つのノード間の最短経路を見つける。

## 要件

この問題を解くには、以下の要件を満たす必要があります。

- グラフは有向である。
- グラフは無重みである。
- グラフとノードのクラスは既に利用可能である。
- 入力は2つのノードである。
- 出力は最短経路を構成するノードキーのリストである。
- 経路がない場合は、Noneを返す。
- グラフは連結されている。
- 入力は有効である。
- アルゴリズムはメモリに収まる必要がある。

## 例の使用法

次のようなグラフがあるとします。

```
graph.add_edge(0, 1)
graph.add_edge(0, 4)
graph.add_edge(0, 5)
graph.add_edge(1, 3)
graph.add_edge(1, 4)
graph.add_edge(2, 1)
graph.add_edge(3, 2)
graph.add_edge(3, 4)
```

`search_path`関数を使って2つのノード間の最短経路を見つけることができます。

- `search_path(start=0, end=2) -> [0, 1, 3, 2]`
- `search_path(start=0, end=0) -> [0]`
- `search_path(start=4, end=5) -> None`
