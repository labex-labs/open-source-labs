# Introducción

En teoría de grafos, el problema del camino más corto es el problema de encontrar un camino entre dos vértices (o nodos) en un grafo de modo que la suma de los pesos de sus aristas componentes sea minimizada. El problema de encontrar el camino más corto entre dos nodos en un grafo no ponderado se puede resolver utilizando un algoritmo de búsqueda en anchura.
