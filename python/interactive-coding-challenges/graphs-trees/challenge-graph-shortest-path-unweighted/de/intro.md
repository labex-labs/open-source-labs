# Einführung

Im Graphentheorie ist das kürzeste-Pfad-Problem das Problem, einen Pfad zwischen zwei Knotenpunkten (oder Knoten) in einem Graphen zu finden, sodass die Summe der Gewichte seiner zusammengesetzten Kanten minimiert ist. Das Problem, den kürzesten Pfad zwischen zwei Knoten in einem ungewichteten Graphen zu finden, kann mit einem Breitensuche-Algorithmus gelöst werden.
