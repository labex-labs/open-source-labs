# Einführung

In der Informatik ist ein Baum ein weit verbreitetes abstraktes Datentyp, der eine hierarchische Baumstruktur mit einer Menge verknüpfter Knoten simuliert. Jeder Knoten in einem Baum hat einen Elternknoten und null oder mehrere Kindknoten. Die Höhe eines Baums ist die Länge des längsten Pfads vom Wurzelknoten zu irgendeinem Blattknoten im Baum. In dieser Herausforderung werden wir die Höhe eines binären Baums bestimmen.
