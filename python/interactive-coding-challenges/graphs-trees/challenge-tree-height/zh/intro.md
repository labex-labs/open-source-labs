# 简介

在计算机科学中，树是一种广泛使用的抽象数据类型，它通过一组链接的节点来模拟分层树结构。树中的每个节点都有一个父节点和零个或多个子节点。树的高度是从根节点到树中任何叶节点的最长路径的长度。在这个挑战中，我们将确定二叉树的高度。
