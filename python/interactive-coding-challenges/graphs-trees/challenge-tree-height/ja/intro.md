# はじめに

コンピュータサイエンスにおいて、木（tree）は、一連のリンクされたノードで階層的な木構造をシミュレートする広く使用される抽象データ型です。木の各ノードは、親ノードとゼロ個以上の子ノードを持っています。木の高さは、木のルートノードから任意の葉ノードまでの最長パスの長さです。このチャレンジでは、二分木の高さを求めます。
