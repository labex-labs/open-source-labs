# 木の高さ

## 問題

二分木が与えられたとき、その木の高さを求める Python 関数を書きなさい。二分木の高さは、木のルートノードから任意の葉ノードまでの最長パスの長さである。

## 要件

この問題を解くために、以下の要件を満たす必要があります。

- 与えられた木は二分木である。
- すでに insert メソッドを持つ Node クラスがある。
- ソリューションはメモリに収まる。

## 使い方の例

この関数がどのように動作するかのいくつかの例を以下に示します。

- 木がただ 1 つのノードの場合、高さは 1 である。たとえば、入力が 5 -> 1 の場合、出力は 1 でなければならない。
- 木が複数のノードを持つ場合、高さはルートノードから任意の葉ノードまでの最長パスの長さである。たとえば、入力が 5, 2, 8, 1, 3 -> 3 の場合、出力は 3 でなければならない。
