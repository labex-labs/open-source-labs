# 木の高さ

## 問題

二分木が与えられたとき、その木の高さを求めるPython関数を書きなさい。二分木の高さは、木のルートノードから任意の葉ノードまでの最長パスの長さである。

## 要件

この問題を解くために、以下の要件を満たす必要があります。

- 与えられた木は二分木である。
- すでにinsertメソッドを持つNodeクラスがある。
- ソリューションはメモリに収まる。

## 使い方の例

この関数がどのように動作するかのいくつかの例を以下に示します。

- 木がただ1つのノードの場合、高さは1である。たとえば、入力が5 -> 1の場合、出力は1でなければならない。
- 木が複数のノードを持つ場合、高さはルートノードから任意の葉ノードまでの最長パスの長さである。たとえば、入力が5, 2, 8, 1, 3 -> 3の場合、出力は3でなければならない。
