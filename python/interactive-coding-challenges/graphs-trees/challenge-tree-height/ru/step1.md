# Высота дерева

## Задача

Дано двоичное дерево. Напишите функцию на Python для определения высоты дерева. Высота двоичного дерева - это длина самой длинной ветви от корневого узла до любого листового узла в дереве.

## Требования

Для решения этой задачи необходимо выполнить следующие требования:

- Данное дерево является двоичным деревом.
- У нас уже есть класс Node с методом insert.
- Решение должно соответствовать ограничениям по памяти.

## Пример использования

Ниже приведены примеры того, как должна работать функция:

- Если дерево содержит только один узел, высота равна 1. Например, если входные данные 5 -> 1, то выход должен быть 1.
- Если дерево содержит несколько узлов, высота равна длине самой длинной ветви от корневого узла до любого листового узла. Например, если входные данные 5, 2, 8, 1, 3 -> 3, то выход должен быть 3.
