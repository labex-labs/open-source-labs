# Введение

В информатике дерево - это широко используемый абстрактный тип данных, который моделирует иерархическую древовидную структуру с набором связанных узлов. Каждый узел в дереве имеет родительский узел и ноль или более дочерних узлов. Высота дерева - это длина самой длинной пути от корневого узла до любого листового узла в дереве. В этом испытании мы определим высоту двоичного дерева.
