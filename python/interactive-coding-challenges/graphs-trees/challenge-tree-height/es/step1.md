# Altura del árbol

## Problema

Dado un árbol binario, escribe una función en Python para determinar la altura del árbol. La altura de un árbol binario es la longitud del camino más largo desde el nodo raíz hasta cualquier nodo hoja en el árbol.

## Requisitos

Para resolver este problema, debemos cumplir con los siguientes requisitos:

- El árbol dado es un árbol binario.
- Ya tenemos una clase Nodo con un método insertar.
- La solución ajusta a la memoria.

## Uso de ejemplo

A continuación se presentan algunos ejemplos de cómo debe comportarse la función:

- Si el árbol tiene solo un nodo, la altura es 1. Por ejemplo, si la entrada es 5 -> 1, la salida debe ser 1.
- Si el árbol tiene múltiples nodos, la altura es la longitud del camino más largo desde el nodo raíz hasta cualquier nodo hoja. Por ejemplo, si la entrada es 5, 2, 8, 1, 3 -> 3, la salida debe ser 3.
