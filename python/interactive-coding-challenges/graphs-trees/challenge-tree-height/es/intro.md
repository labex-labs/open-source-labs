# Introducción

En informática, un árbol es un tipo de dato abstracto ampliamente utilizado que simula una estructura de árbol jerárquica con un conjunto de nodos enlazados. Cada nodo en un árbol tiene un nodo padre y cero o más nodos hijos. La altura de un árbol es la longitud del camino más largo desde el nodo raíz hasta cualquier nodo hoja en el árbol. En este desafío, determinaremos la altura de un árbol binario.
