# Introduction

En informatique, un arbre est un type abstrait de données largement utilisé qui simule une structure d'arbre hiérarchique avec un ensemble de nœuds liés. Chaque nœud dans un arbre a un nœud parent et zéro ou plusieurs nœuds enfants. La hauteur d'un arbre est la longueur du plus long chemin du nœud racine à n'importe quel nœud feuille de l'arbre. Dans ce défi, nous déterminerons la hauteur d'un arbre binaire.
