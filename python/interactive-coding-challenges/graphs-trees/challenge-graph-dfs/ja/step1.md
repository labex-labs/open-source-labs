# グラフのDFS

## 問題

有向グラフ上で深さ優先探索を実装します。アルゴリズムは、指定されたノードから開始し、グラフ内の到達可能なすべてのノードを訪問する必要があります。ノードが訪問される順序を記録し、リストとして返します。

## 要件

有向グラフ上でDFSを実装するには、以下の要件を満たす必要があります。

- グラフは有向です。
- GraphとNodeクラスは既に実装されています。
- グラフは連結されています。
- 入力は有効です。
- アルゴリズムはメモリに収まります。

## 例の使用法

次のエッジを持つ有向グラフがあるとします。

```
graph.add_edge(0, 1, 5)
graph.add_edge(0, 4, 3)
graph.add_edge(0, 5, 2)
graph.add_edge(1, 3, 5)
graph.add_edge(1, 4, 4)
graph.add_edge(2, 1, 6)
graph.add_edge(3, 2, 7)
graph.add_edge(3, 4, 8)
```

ノード0からDFSを開始する場合、訪問されるノードの順序は[0, 1, 3, 2, 4, 5]になるはずです。
