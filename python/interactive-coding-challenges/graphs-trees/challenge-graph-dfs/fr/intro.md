# Introduction

Le graphe est une structure de données non linéaire composée de nœuds et d'arêtes. La recherche en profondeur (DFS - Depth - First Search en anglais) est un algorithme de parcours de graphe qui explore le plus loin possible le long de chaque branche avant de revenir en arrière. Dans ce défi, nous allons implémenter la DFS sur un graphe dirigé.
