# Sommaire

Dans ce défi, nous avons implémenté la recherche en profondeur (DFS - Depth - First Search en anglais) sur un graphe dirigé. L'algorithme commence à un nœud donné et visite tous les nœuds atteignables dans le graphe. L'ordre dans lequel les nœuds sont visités est enregistré et renvoyé sous forme de liste. En répondant aux exigences, nous pouvons assurer la correction et l'efficacité de l'algorithme.
