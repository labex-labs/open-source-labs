# Резюме

В этом испытании мы реализовали обход графа в глубину для направленного графа. Алгоритм начинается с заданного узла и посещает все достижимые узлы в графе. Порядок посещения узлов записывается и возвращается в виде списка. Согласно требованиям, мы можем гарантировать правильность и эффективность алгоритма.
