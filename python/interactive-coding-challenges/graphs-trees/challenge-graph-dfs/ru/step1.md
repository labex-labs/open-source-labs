# Обход графа в глубину (Graph Dfs)

## Задача

Реализовать обход графа в глубину для направленного графа. Алгоритм должен начинать с заданного узла и посещать все достижимые узлы в графе. Порядок посещения узлов должен быть записан и возвращен в виде списка.

## Требования

Для реализации обхода в глубину для направленного графа необходимо выполнить следующие требования:

- Граф является направленным.
- Классы Graph и Node уже реализованы.
- Граф связный.
- Входные данные действительны.
- Алгоритм помещается в память.

## Пример использования

Предположим, что у нас есть направленный граф с следующими ребрами:

```
graph.add_edge(0, 1, 5)
graph.add_edge(0, 4, 3)
graph.add_edge(0, 5, 2)
graph.add_edge(1, 3, 5)
graph.add_edge(1, 4, 4)
graph.add_edge(2, 1, 6)
graph.add_edge(3, 2, 7)
graph.add_edge(3, 4, 8)
```

Если мы начнем обход в глубину с узла 0, порядок посещения узлов должен быть [0, 1, 3, 2, 4, 5].
