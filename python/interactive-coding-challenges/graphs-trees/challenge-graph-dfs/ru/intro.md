# Введение

Граф - это нелинейная структура данных, состоящая из узлов и ребер. Обход в глубину (Depth-first search, DFS) - это алгоритм обхода графа, который исследует граф вдоль каждого ветви до предела, прежде чем возвращаться назад. В этом испытании мы реализуем DFS для направленного графа.
