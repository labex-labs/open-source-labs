# В графе существует путь

## Задача

Дан направленный граф и два узла. Напишите функцию на Python для определения, существует ли путь между этими двумя узлами.

## Требования

Для решения этой задачи мы должны сделать следующие предположения:

- Граф направленный.
- У нас уже есть классы Graph и Node.
- Граф связный.
- Входные данные действительны.
- Решение помещается в память.

## Пример использования

Предположим, у нас есть следующий граф:

```
graph.add_edge(0, 1, 5)
graph.add_edge(0, 4, 3)
graph.add_edge(0, 5, 2)
graph.add_edge(1, 3, 5)
graph.add_edge(1, 4, 4)
graph.add_edge(2, 1, 6)
graph.add_edge(3, 2, 7)
graph.add_edge(3, 4, 8)
```

Мы можем использовать следующую функцию, чтобы определить, существует ли путь между двумя узлами:

```
search_path(start=0, end=2) -> True
search_path(start=0, end=0) -> True
search_path(start=4, end=5) -> False
```

Первые два вызова функции возвращают True, потому что между начальным и конечным узлами существует путь. Последний вызов функции возвращает False, потому что между начальным и конечным узлами нет пути.
