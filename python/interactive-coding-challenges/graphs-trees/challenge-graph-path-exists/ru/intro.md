# Введение

В информатике граф - это совокупность узлов, соединенных между собой ребрами. Графы широко используются для моделирования реальных сценариев, таких как социальные сети, транспортные системы и компьютерные сети. Одной из фундаментальных задач теории графов является определение наличия пути между двумя узлами в графе. В этом испытании мы напишем функцию на Python для решения этой проблемы.
