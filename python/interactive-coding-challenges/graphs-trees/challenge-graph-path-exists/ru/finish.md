# Резюме

В этом испытании мы написали функцию на Python для определения наличия пути между двумя узлами в направленном графе. Мы сделали несколько предположений о графе и входных данных, чтобы упростить задачу. Функция может быть использована для решения широкого спектра реальных задач, связанных с обходом графов.
