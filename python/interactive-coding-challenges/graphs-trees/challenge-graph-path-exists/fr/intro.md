# Introduction

En informatique, un graphe est une collection de nœuds connectés les uns aux autres par des arêtes. Les graphes sont largement utilisés pour modéliser des scénarios du monde réel tels que les réseaux sociaux, les systèmes de transport et les réseaux informatiques. Un des problèmes fondamentaux de la théorie des graphes est de déterminer s'il existe un chemin entre deux nœuds dans un graphe. Dans ce défi, nous allons écrire une fonction Python pour résoudre ce problème.
