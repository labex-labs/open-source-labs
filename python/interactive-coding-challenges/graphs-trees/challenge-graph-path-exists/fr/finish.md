# Sommaire

Dans ce défi, nous avons écrit une fonction Python pour déterminer s'il existe un chemin entre deux nœuds dans un graphe dirigé. Nous avons fait plusieurs hypothèses sur le graphe et les entrées pour simplifier le problème. La fonction peut être utilisée pour résoudre une large gamme de problèmes du monde réel qui impliquent la traversée de graphes.
