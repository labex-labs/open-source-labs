# Introducción

En informática, un grafo es una colección de nodos que están conectados entre sí a través de aristas. Los grafos se utilizan ampliamente para modelar escenarios del mundo real, como redes sociales, sistemas de transporte y redes informáticas. Uno de los problemas fundamentales en la teoría de grafos es determinar si existe un camino entre dos nodos en un grafo. En este desafío, escribiremos una función de Python para resolver este problema.
