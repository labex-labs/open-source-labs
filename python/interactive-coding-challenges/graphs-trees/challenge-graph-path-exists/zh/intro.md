# 简介

在计算机科学中，图是由节点组成的集合，这些节点通过边相互连接。图被广泛用于对现实世界场景进行建模，如社交网络、交通系统和计算机网络。图论中的一个基本问题是确定图中两个节点之间是否存在路径。在这个挑战中，我们将编写一个 Python 函数来解决这个问题。
