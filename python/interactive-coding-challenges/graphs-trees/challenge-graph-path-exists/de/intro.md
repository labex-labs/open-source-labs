# Einführung

In der Informatik ist ein Graph eine Sammlung von Knoten, die durch Kanten miteinander verbunden sind. Graphen werden weit verbreitet verwendet, um reale Szenarien wie soziale Netzwerke, Verkehrssysteme und Computernetzwerke zu modellieren. Ein der grundlegenden Probleme in der Graphentheorie ist es, zu bestimmen, ob es einen Pfad zwischen zwei Knoten in einem Graphen gibt. In dieser Herausforderung werden wir eine Python-Funktion schreiben, um dieses Problem zu lösen.
