# Обход графа в ширину (BFS)

## Задача

Реализация BFS для графа заключается в обходе всех вершин графа в порядке обхода в ширину, начиная от заданной исходной вершины. Алгоритм работает путём посещения исходной вершины, затем всех её соседей, затем всех соседей соседей и так далее. Порядок посещения вершин важен, так как он определяет кратчайший путь от исходной вершины до всех других вершин в графе.

## Требования

Для реализации BFS для графа необходимо выполнить следующие требования:

- Граф является направленным.
- Классы Graph и Node уже доступны.
- Граф связный.
- Входные данные валидны.
- Алгоритм помещается в память.

## Пример использования

Предположим, что у нас есть граф с такими ребрами:

```
graph.add_edge(0, 1, 5)
graph.add_edge(0, 4, 3)
graph.add_edge(0, 5, 2)
graph.add_edge(1, 3, 5)
graph.add_edge(1, 4, 4)
graph.add_edge(2, 1, 6)
graph.add_edge(3, 2, 7)
graph.add_edge(3, 4, 8)
```

Если мы начнём BFS от вершины 0, порядок посещения вершин будет [0, 1, 4, 5, 3, 2]. Это означает, что вершина 0 посещается первой, затем её соседи 1, 4 и 5, затем соседи 1 (3 и 4), а затем сосед 3 (2).
