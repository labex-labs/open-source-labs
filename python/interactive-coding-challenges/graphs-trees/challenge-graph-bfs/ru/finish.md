# Резюме

Кратко говоря, поиск в ширину (BFS) — это алгоритм обхода графа, который обходит все вершины графа в порядке обхода в ширину. Для реализации BFS для графа граф должен быть направленным, связным и входные данные должны быть валидными. Порядок посещения вершин важен, так как он определяет кратчайший путь от исходной вершины до всех других вершин в графе.
