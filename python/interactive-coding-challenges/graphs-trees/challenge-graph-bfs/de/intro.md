# Einführung

Der Breitensuche-Algorithmus (engl. Breadth-First Search, BFS) ist ein Algorithmus zum Abarbeiten von Graphen, der alle Knoten eines Graphen in breitfirstiger Reihenfolge durchläuft, d. h., er besucht alle Knoten auf derselben Ebene, bevor er zur nächsten Ebene übergeht.
