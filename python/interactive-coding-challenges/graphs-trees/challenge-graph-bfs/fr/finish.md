# Récapitulatif

En résumé, la recherche en largeur (BFS - Breadth-First Search en anglais) est un algorithme de parcours de graphe qui visite tous les sommets d'un graphe dans l'ordre de largeur. Pour implémenter la recherche en largeur sur un graphe, le graphe doit être dirigé, connexe et les entrées doivent être valides. L'ordre dans lequel les sommets sont visités est important, car il détermine le plus court chemin du sommet source à tous les autres sommets du graphe.
