# Введение

В теории графов задача нахождения кратчайшего пути заключается в поиске пути между двумя вершинами (или узлами) в графе таким образом, чтобы сумма весов его составляющих ребер была минимальной. Это фундаментальная задача в оптимизации сетей и имеет приложения в транспортном, коммуникационном и компьютерном сетевом оборудовании.
