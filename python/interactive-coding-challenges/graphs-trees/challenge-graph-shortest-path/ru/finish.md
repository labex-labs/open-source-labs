# Резюме

Задача нахождения кратчайшего пути - это фундаментальная задача в оптимизации сетей. Чтобы решить эту задачу, необходимо учитывать требования графа, такие как то, является ли он направленным, содержит ли он циклы и являются ли ребра взвешенными. Мы можем использовать класс ShortestPath для нахождения кратчайшего пути между двумя узлами в графе.
