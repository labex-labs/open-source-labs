# Introducción

En teoría de grafos, el problema del camino más corto es el problema de encontrar un camino entre dos vértices (o nodos) en un grafo de modo que la suma de los pesos de sus aristas componentes sea minimizada. Es un problema fundamental en la optimización de redes y tiene aplicaciones en transporte, comunicación y red de computadoras.
