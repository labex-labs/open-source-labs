# Einführung

In der Graphentheorie ist das kürzeste-Pfad-Problem das Problem, einen Pfad zwischen zwei Knotenpunkten (oder Knoten) in einem Graphen zu finden, sodass die Summe der Gewichte seiner zusammensetzenden Kanten minimiert ist. Es ist ein grundlegendes Problem in der Netzwerkoptimierung und hat Anwendungen in der Verkehrstechnik, Kommunikation und Computer-Netzwerken.
