# Bst 検証

## 問題

バイナリツリーのルートノードを入力として受け取り、それが有効な二分探索木であるかどうかを判断する Python 関数を書く問題です。バイナリツリーが有効な二分探索木であるための必要十分条件は次のとおりです。

1. ノードの左部分木には、そのノードの値より小さい値を持つノードのみ含まれます。
2. ノードの右部分木には、そのノードの値より大きい値を持つノードのみ含まれます。
3. 左部分木と右部分木の両方がそれ自体有効な二分探索木です。

## 要件

このチャレンジを解くには、次の要件を満たす必要があります。

- 関数は、重複する値を持つバイナリツリーを処理できる必要があります。
- 関数に None 入力が渡された場合、例外を発生させる必要があります。
- Node クラスは既に定義されている必要があります。
- バイナリツリーはメモリに収まる必要があります。

## 例の使用法

```txt
有効:
      5
    /   \
   5     8
  /     /
 4     6
        \
         7

無効:
      5
    /   \
   5     8
  / \   /
 4   9 7
```
