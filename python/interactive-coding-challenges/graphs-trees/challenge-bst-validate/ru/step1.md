# Проверка BST

## Задача

Задача — написать функцию на Python, которая принимает корневой узел двоичного дерева в качестве входных данных и определяет, является ли оно корректным двоичным деревом поиска. Двоичное дерево является корректным двоичным деревом поиска тогда и только тогда, когда выполняются следующие условия:

1. Левый поддерево узла содержит только узлы со значениями, меньшими значением узла.
2. Правый поддерево узла содержит только узлы со значениями, большими значением узла.
3. И левое, и правое поддеревья являются сами по себе корректными двоичными деревьями поиска.

## Требования

Для решения этого испытания должны быть выполнены следующие требования:

- Функция должна уметь обрабатывать двоичные деревья с дубликатами.
- Если функция вызывается с входным значением None, должна быть вызвана ошибка.
- Класс Node должен быть уже определён.
- Двоичное дерево должно помещаться в память.

## Пример использования

```txt
Корректное:
      5
    /   \
   5     8
  /     /
 4     6
        \
         7

Некорректное:
      5
    /   \
   5     8
  / \   /
 4   9 7
```
