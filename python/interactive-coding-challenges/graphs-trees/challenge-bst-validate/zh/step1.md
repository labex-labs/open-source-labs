# 验证二叉搜索树

## 问题

问题是编写一个 Python 函数，该函数以二叉树的根节点作为输入，并确定它是否是有效的二叉搜索树。当且仅当满足以下条件时，二叉树才是有效的二叉搜索树：

1. 节点的左子树仅包含值小于该节点值的节点。
2. 节点的右子树仅包含值大于该节点值的节点。
3. 左子树和右子树本身都是有效的二叉搜索树。

## 要求

为了解决这个挑战，必须满足以下要求：

- 该函数应该能够处理包含重复值的二叉树。
- 如果使用 None 作为输入调用该函数，它应该引发异常。
- 节点类 Node 应该已经定义。
- 二叉树应该能够装入内存。

## 示例用法

```txt
有效：
      5
    /   \
   5     8
  /     /
 4     6
        \
         7

无效：
      5
    /   \
   5     8
  / \   /
 4   9 7
```
