# Resumen

En este desafío, hemos aprendido cómo determinar si un árbol binario dado es un árbol de búsqueda binaria válido. También hemos discutido los requisitos para este desafío, incluyendo el hecho de que la función debe ser capaz de manejar duplicados, generar una excepción si se llama con una entrada None y asumir que la clase Node ya está definida.
