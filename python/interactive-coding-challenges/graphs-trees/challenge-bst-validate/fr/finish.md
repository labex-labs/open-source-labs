# Résumé

Dans ce défi, nous avons appris à déterminer si un arbre binaire donné est un arbre binaire de recherche valide. Nous avons également discuté des exigences de ce défi, y compris le fait que la fonction devrait être capable de gérer des doublons, lever une exception si elle est appelée avec une entrée None, et supposer que la classe Node est déjà définie.
