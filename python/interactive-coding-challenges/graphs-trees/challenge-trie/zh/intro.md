# 简介

字典树（Trie），也称为前缀树，是一种树形数据结构，用于存储和检索一组字符串。它常用于搜索引擎、拼写检查器和 IP 路由器中。在字典树中，每个节点代表字符串中的一个字符，从根节点到叶节点的路径代表一个完整的字符串。
