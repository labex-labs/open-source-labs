# トライ木

## 問題

あなたの課題は、以下のメソッドを持つトライ木を実装することです。

- `find(word)` - 与えられた単語がトライ木に存在する場合は `True` を返し、そうでない場合は `False` を返します。
- `insert(word)` - 与えられた単語をトライ木に挿入します。
- `remove(word)` - 与えられた単語をトライ木から削除します。
- `list_words()` - トライ木に含まれる、終端文字で終わるすべての単語のリストを返します。

## 要件

このチャレンジを完了するには、以下の要件を満たす必要があります。

- 実装は文字列と動作する必要があります。
- 文字列は ASCII コードであると仮定されます。
- `find` メソッドは、終端文字を持つ正確な単語のみを一致させる必要があります。
- `list_words` メソッドは、終端文字を持つ単語のみを返す必要があります。
- 実装はメモリに収まると仮定されます。

## 使い方の例

以下の例は、トライ木のメソッドの使い方を示しています。

```txt

         root
       /  |  \
      h   a*  m
     / \   \   \
    a   e*  t*  e*
   / \         / \
  s*  t*      n*  t*
             /
            s*

find

* 空のトライ木での検索
* 一致しない検索
* 一致する検索

insert

* 空のトライ木に挿入
* 葉の終端文字を作るために挿入
* 既存の終端文字を拡張するために挿入

remove

* 私を削除
* mens を削除
* a を削除
* has を削除

list_words

* 空のリストを表示
* 一般的なケースを表示
```
