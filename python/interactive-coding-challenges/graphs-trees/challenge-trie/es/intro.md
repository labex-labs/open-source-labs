# Introducción

Un trie, también conocido como árbol de prefijos, es una estructura de datos en forma de árbol que se utiliza para almacenar y recuperar un conjunto de cadenas. Se utiliza comúnmente en motores de búsqueda, verificadores ortográficos y routers IP. En un trie, cada nodo representa un carácter en una cadena, y el camino desde la raíz hasta un nodo hoja representa una cadena completa.
