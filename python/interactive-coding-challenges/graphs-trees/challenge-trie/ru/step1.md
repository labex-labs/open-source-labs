# Trie

## Задача

Ваша задача — реализовать trie с методами:

- `find(word)` — возвращает `True`, если данное слово есть в trie, и `False` в противном случае.
- `insert(word)` — вставляет данное слово в trie.
- `remove(word)` — удаляет данное слово из trie.
- `list_words()` — возвращает список всех слов в trie, которые заканчиваются на завершающий символ.

## Требования

Для решения этой задачи необходимо выполнить следующие требования:

- Реализация должна работать со строками.
- Строки предполагаются в ASCII.
- Метод `find` должен совпадать только с точными словами с завершающим символом.
- Метод `list_words` должен возвращать только слова с завершающим символом.
- Предполагается, что реализация помещается в память.

## Пример использования

Следующие примеры демонстрируют использование методов trie:

```txt

         root
       /  |  \
      h   a*  m
     / \   \   \
    a   e*  t*  e*
   / \         / \
  s*  t*      n*  t*
             /
            s*

find

* Найти в пустом trie
* Найти несовпадающее
* Найти совпадающее

insert

* Вставить в пустой trie
* Вставить, чтобы сделать терминатором листа
* Вставить, чтобы расширить существующий терминатор

remove

* Удалить меня
* Удалить mens
* Удалить a
* Удалить has

list_words

* Список пустой
* Список в общем случае
```
