# Введение

Trie, также известная как префиксное дерево, — это древовидная структура данных, используемая для хранения и извлечения набора строк. Она широко используется в поисковых системах, проверяющих орфографию, и IP-роутерах. В trie каждый узел представляет собой символ в строке, а путь от корня до листового узла представляет собой полную строку.
