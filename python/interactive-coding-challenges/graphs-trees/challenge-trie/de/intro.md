# Einführung

Ein Trie, auch als Präfixbaum bekannt, ist eine baumartige Datenstruktur, die verwendet wird, um eine Menge von Zeichenketten zu speichern und abzurufen. Es wird häufig in Suchmaschinen, Rechtschreibprüfern und IP-Routern verwendet. In einem Trie repräsentiert jeder Knoten ein Zeichen in einer Zeichenkette, und der Pfad vom Stamm zu einem Blattknoten repräsentiert eine vollständige Zeichenkette.
