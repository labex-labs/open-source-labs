# Введение

Сортировка - это фундаментальная операция в информатике. Это процесс упорядочивания данных в определенном порядке. Сортировка выбором - один из самых простых алгоритмов сортировки. Она работает путем повторного поиска минимального элемента из неотсортированной части списка и помещения его в начало. Этот процесс повторяется, пока весь список не будет отсортирован.
