# Сортировка выбором

## Задача

Реализуйте сортировку выбором на Python. Алгоритм должен принимать список целых чисел в качестве входных данных и возвращать отсортированный список. Алгоритм должен работать следующим образом:

1. Найти минимальный элемент в неотсортированной части списка.
2. Обменять его с первым элементом неотсортированной части списка.
3. Переместить границу отсортированной части списка на один элемент вправо.

Повторяйте шаги 1-3, пока весь список не будет отсортирован.

## Требования

Для реализации сортировки выбором на Python необходимо выполнить следующие требования:

- Алгоритм должен принимать список целых чисел в качестве входных данных.
- Алгоритм должен возвращать отсортированный список целых чисел.
- Алгоритм должен быть реализован с использованием алгоритма сортировки выбором.
- Алгоритм должен работать для списков любой длины.
- Алгоритм должен обрабатывать дубликаты элементов в списке.
- Входной список может быть неотсортированным.
- Входной список может содержать недопустимые данные, такие как нецелые значения.
- Алгоритм должен быть экономичным в использовании памяти и не использовать избыточную память.

## Пример использования

Следующие примеры демонстрируют использование алгоритма сортировки выбором:

- `selection_sort([])` возвращает `[]`
- `selection_sort([1])` возвращает `[1]`
- `selection_sort([3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5])` возвращает `[1, 1, 2, 3, 3, 4, 5, 5, 5, 6, 9]`
