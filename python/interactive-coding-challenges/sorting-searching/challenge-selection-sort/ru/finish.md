# Обзор

Сортировка выбором - простой алгоритм сортировки, который работает путем повторного поиска минимального элемента из неотсортированной части списка и помещения его в начало. Это не самый эффективный алгоритм сортировки, но он легко понять и реализовать. Алгоритм может обрабатывать дубликаты элементов и недопустимые входные данные. Согласно требованиям, мы можем реализовать сортировку выбором на Python и использовать ее для сортировки списков целых чисел.
