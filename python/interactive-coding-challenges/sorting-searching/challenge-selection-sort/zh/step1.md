# 选择排序

## 问题

用 Python 实现选择排序。该算法应以整数列表作为输入，并返回已排序的列表。该算法应按以下方式工作：

1. 在列表的未排序部分找到最小元素。
2. 将其与列表未排序部分的第一个元素交换。
3. 将列表已排序部分的边界向右移动一个元素。

重复步骤 1 至 3，直到整个列表被排序。

## 要求

要在 Python 中实现选择排序，必须满足以下要求：

- 该算法应以整数列表作为输入。
- 该算法应返回一个已排序的整数列表。
- 该算法应使用选择排序算法来实现。
- 该算法应适用于任何长度的列表。
- 该算法应处理列表中的重复元素。
- 输入列表可能未排序。
- 输入列表可能包含无效数据，例如非整数值。
- 该算法应具有内存效率，且不使用过多内存。

## 示例用法

以下示例演示了选择排序算法的用法：

- `selection_sort([])` 返回 `[]`
- `selection_sort([1])` 返回 `[1]`
- `selection_sort([3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5])` 返回 `[1, 1, 2, 3, 3, 4, 5, 5, 5, 6, 9]`
