# はじめに

ソートは、コンピュータサイエンスにおける基本的な操作です。特定の順序でデータを並べるプロセスです。選択ソートは、最も単純なソートアルゴリズムの 1 つです。リストの未ソート部分から最小要素を繰り返し見つけ、それを先頭に配置することで機能します。このプロセスを、リスト全体がソートされるまで繰り返します。
