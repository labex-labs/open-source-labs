# 選択ソート

## 問題

Pythonで選択ソートを実装します。このアルゴリズムは整数のリストを入力として受け取り、ソート済みのリストを返す必要があります。アルゴリズムは以下のように機能する必要があります。

1. リストの未ソート部分で最小要素を見つけます。
2. それをリストの未ソート部分の最初の要素と交換します。
3. リストのソート済み部分の境界を1要素右に移動します。

ステップ1から3を、リスト全体がソートされるまで繰り返します。

## 要件

Pythonで選択ソートを実装するには、以下の要件を満たす必要があります。

- アルゴリズムは整数のリストを入力として受け取る必要があります。
- アルゴリズムは整数のソート済みリストを返す必要があります。
- アルゴリズムは選択ソートアルゴリズムを使用して実装する必要があります。
- アルゴリズムは任意の長さのリストに対して機能する必要があります。
- アルゴリズムはリスト内の重複要素を処理する必要があります。
- 入力リストはソートされていなくてもよいです。
- 入力リストには非整数値などの無効なデータが含まれていてもよいです。
- アルゴリズムはメモリ効率が良く、過度のメモリを使用しない必要があります。

## 例の使用法

以下の例は、選択ソートアルゴリズムの使用方法を示しています。

- `selection_sort([])` は `[]` を返します
- `selection_sort([1])` は `[1]` を返します
- `selection_sort([3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5])` は `[1, 1, 2, 3, 3, 4, 5, 5, 5, 6, 9]` を返します
