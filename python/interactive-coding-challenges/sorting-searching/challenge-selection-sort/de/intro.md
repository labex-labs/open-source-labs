# Einführung

Sortieren ist eine grundlegende Operation in der Informatik. Es ist der Prozess, um Daten in einer bestimmten Reihenfolge zu ordnen. Der Selectionsort ist eines der einfachsten Sortieralgorithmen. Er funktioniert, indem er wiederholt das kleinste Element aus dem unsortierten Teil der Liste findet und es am Anfang platziert. Dieser Prozess wird wiederholt, bis die gesamte Liste sortiert ist.
