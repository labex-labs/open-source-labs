# Resumen

El ordenamiento por selección es un algoritmo de clasificación simple que funciona mediante la búsqueda repetida del elemento mínimo en la parte no clasificada de la lista y colocándolo al principio. No es el algoritmo de clasificación más eficiente, pero es fácil de entender e implementar. El algoritmo puede manejar elementos duplicados y entradas no válidas. Al cumplir con los requisitos, podemos implementar el ordenamiento por selección en Python y usarlo para ordenar listas de enteros.
