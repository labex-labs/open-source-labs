# Introducción

La clasificación es una operación fundamental en la ciencia de la computación. Es el proceso de organizar datos en un orden específico. El método de clasificación por selección es uno de los algoritmos de clasificación más simples. Funciona mediante la búsqueda repetida del elemento mínimo en la parte no clasificada de la lista y colocándolo al principio. Este proceso se repite hasta que toda la lista está clasificada.
