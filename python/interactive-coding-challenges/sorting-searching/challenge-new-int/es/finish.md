# Resumen

En este desafío, aprendimos cómo encontrar un entero faltante en una matriz de enteros no negativos utilizando la mínima memoria posible. Discutimos los requisitos y proporcionamos ejemplos de cómo debe comportarse la función. Siguiendo los requisitos y los ejemplos, podemos implementar una solución que resuelva el problema de manera eficiente y precisa.
