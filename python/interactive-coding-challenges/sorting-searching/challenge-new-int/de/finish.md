# Zusammenfassung

In dieser Herausforderung haben wir gelernt, wie man ein fehlendes Ganzzahl in einem Array von nicht-negativen ganzen Zahlen mit minimalem Speicher findet. Wir haben die Anforderungen diskutiert und Beispiele dafür gegeben, wie die Funktion verhalten sollte. Indem wir den Anforderungen und Beispielen folgen, können wir eine Lösung implementieren, die das Problem effizient und genau löst.
