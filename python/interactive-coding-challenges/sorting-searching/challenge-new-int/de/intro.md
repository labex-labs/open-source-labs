# Einführung

Beim Programmieren tritt es häufig auf, dass wir Probleme haben, bei denen wir ein fehlendes Element in einer gegebenen Datenstruktur finden müssen. In dieser Herausforderung erhalten wir ein Array von 32 nicht-negativen ganzen Zahlen und müssen eine Ganzzahl finden, die nicht im Eingabearray enthalten ist. Die Herausforderung besteht darin, dieses Problem mit minimalem Speicher zu lösen.
