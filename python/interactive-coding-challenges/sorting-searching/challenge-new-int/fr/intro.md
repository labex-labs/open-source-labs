# Introduction

En programmation, il est courant de rencontrer des problèmes où nous devons trouver un élément manquant dans une structure de données donnée. Dans ce défi, nous recevrons un tableau de 32 entiers non négatifs et nous devons trouver un entier qui n'est pas présent dans le tableau d'entrée. Le défi est de résoudre ce problème en utilisant la mémoire minimale.
