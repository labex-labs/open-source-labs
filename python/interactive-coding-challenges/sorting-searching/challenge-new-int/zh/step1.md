# 找出缺失的整数

## 问题

给定一个由 32 个非负整数组成的数组，找出一个不在输入数组中的整数。解决方案应使用最少的内存。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 输入数组包含非负整数。
- 整数的范围未指定，但我们需要讨论针对 40 亿个整数的方法。
- 我们需要为一个包含 32 个整数的数组实现解决方案。
- 我们不能假设输入数组是有效的。

## 示例用法

以下是该函数应有的一些行为示例：

- 如果输入为 None 或空数组，函数应引发异常。
- 如果有一个整数被排除在输入数组之外，函数应返回该整数。
- 如果没有整数被排除在输入数组之外，函数应返回 None。
