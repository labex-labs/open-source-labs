# Введение

В программировании часто возникают задачи, в которых необходимо найти отсутствующий элемент в заданной структуре данных. В этом испытании мы получим массив из 32 неотрицательных целых чисел и нам нужно найти целое число, которое отсутствует в входном массиве. Задача заключается в том, чтобы решить эту проблему с использованием минимального количества памяти.
