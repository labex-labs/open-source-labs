# Найти отсутствующее целое число

## Задача

Дан массив из 32 неотрицательных целых чисел. Найти целое число, которое отсутствует в входном массиве. Решение должно использовать минимальное количество памяти.

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Входной массив содержит неотрицательные целые числа.
- Диапазон целых чисел не указан, но необходимо обсудить подход для 4 миллиардов целых чисел.
- Необходимо реализовать решение для массива из 32 целых чисел.
- Мы не можем предполагать, что входной массив действителен.

## Пример использования

Ниже приведены примеры того, как должна работать функция:

- Если входные данные равны None или массив пуст, функция должна вызывать исключение.
- Если есть целое число, исключенное из входного массива, функция должна вернуть это целое число.
- Если нет целого числа, исключенного из входного массива, функция должна вернуть None.
