# Резюме

В этом испытании мы научились искать отсутствующее целое число в массиве неотрицательных целых чисел с использованием минимального количества памяти. Мы обсудили требования и дали примеры того, как должна работать функция. Следуя требованиям и примерам, мы можем реализовать решение, которое эффективно и точно решает задачу.
