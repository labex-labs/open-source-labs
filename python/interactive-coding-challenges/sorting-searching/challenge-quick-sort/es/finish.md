# Resumen

Quick Sort es un algoritmo de clasificación popular que utiliza un enfoque divide y vencerás para ordenar una matriz o una lista. El algoritmo funciona dividiendo la lista en dos sub-listas y luego ordenando recursivamente las sub-listas. El algoritmo Quick Sort es eficiente y simple, lo que lo hace ampliamente utilizado. Para implementar Quick Sort en Python, el algoritmo debe cumplir con los requisitos de no ser una solución in-place, manejar elementos duplicados, manejar entrada no válida y ser capaz de manejar listas de cualquier tamaño.
