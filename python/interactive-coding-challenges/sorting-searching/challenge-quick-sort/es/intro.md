# Introducción

Quick Sort es un algoritmo de clasificación popular que utiliza un enfoque divide y vencerás para ordenar una matriz o una lista. Es un algoritmo basado en comparaciones que funciona dividiendo una matriz en dos submatrices y luego ordenando recursivamente las submatrices. Quick Sort es ampliamente utilizado debido a su eficiencia y simplicidad.
