# 简介

快速排序是一种流行的排序算法，它采用分治策略对数组或列表进行排序。它是一种基于比较的算法，通过将数组划分为两个子数组，然后递归地对这些子数组进行排序。由于其效率和简单性，快速排序被广泛使用。
