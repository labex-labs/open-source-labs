# クイックソート

## 問題

Python でクイックソートアルゴリズムを実装します。このアルゴリズムは、ソートされていないリストを入力として受け取り、ソートされたリストを返す必要があります。このアルゴリズムは、空のリストや重複要素を含む任意のサイズのリストに対して機能する必要があります。また、無効な入力にも適切に対応する必要があります。

クイックソートアルゴリズムは以下のように機能します。

1. リストからピボット要素を選びます。
2. リストを 2 つのサブリストに分割します。1 つはピボットより小さい要素を含むサブリストで、もう 1 つはピボットより大きい要素を含むサブリストです。
3. 再帰的にサブリストにクイックソートアルゴリズムを適用します。
4. ソートされたサブリストをピボット要素を挟んで連結します。

## 要件

Python でクイックソートアルゴリズムを実装するには、以下の要件を満たす必要があります。

- アルゴリズムはインプレースソリューションではない必要があります。
- アルゴリズムはリスト内の重複要素を処理する必要があります。
- アルゴリズムは無効な入力、たとえば None やリストでない入力を処理する必要があります。
- アルゴリズムは、空のリストを含む任意のサイズのリストを処理できる必要があります。

## 使い方の例

以下は、Python でクイックソートアルゴリズムを使用する方法のいくつかの例です。

- None -> 例外

```python
quick_sort(None)
```

- 空の入力 -> []

```python
quick_sort([])
```

- 1 つの要素 -> [要素]

```python
quick_sort([5])
```

- 2 つ以上の要素

```python
quick_sort([5, 2, 8, 3, 1, 9])
```
