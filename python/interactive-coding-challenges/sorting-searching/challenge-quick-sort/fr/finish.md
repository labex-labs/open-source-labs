# Résumé

Le Tri Rapide est un algorithme de tri populaire qui utilise une approche diviser-pour-régner pour trier un tableau ou une liste. L'algorithme fonctionne en divisant la liste en deux sous-listes, puis en triant récursivement les sous-listes. L'algorithme de Tri Rapide est efficace et simple, ce qui en fait un algorithme largement utilisé. Pour implémenter le Tri Rapide en Python, l'algorithme devrait répondre aux exigences suivantes : ne pas être une solution en place, gérer les éléments dupliqués, gérer les entrées invalides et être capable de gérer des listes de toute taille.
