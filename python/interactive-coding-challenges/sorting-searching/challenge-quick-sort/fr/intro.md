# Introduction

Le Tri Rapide est un algorithme de tri populaire qui utilise une approche diviser-pour-régner pour trier un tableau ou une liste. C'est un algorithme basé sur les comparaisons qui fonctionne en divisant un tableau en deux sous-tableaux, puis en triant récursivement les sous-tableaux. Le Tri Rapide est largement utilisé en raison de son efficacité et de sa simplicité.
