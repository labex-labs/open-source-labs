# Резюме

Quick Sort - это популярный алгоритм сортировки, который использует подход "разделяй и властвуй" для сортировки массива или списка. Алгоритм работает путём разделения списка на два подсписка, а затем рекурсивной сортировки подсписков. Алгоритм Quick Sort эффективен и прост, что делает его широко используемым. Для реализации Quick Sort на Python алгоритм должен соответствовать требованиям не быть in-place решением, обрабатывать дублирующиеся элементы, обрабатывать некорректный ввод и быть способен обрабатывать списки любого размера.
