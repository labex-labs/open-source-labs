# Quick Sort

## Задача

Реализовать алгоритм Quick Sort на Python. Алгоритм должен принимать на вход неотсортированный список и возвращать отсортированный список. Алгоритм должен работать для списков любого размера, включая пустые списки и списки с дублирующимися элементами. Алгоритм также должен优雅но обрабатывать некорректный ввод.

Алгоритм Quick Sort работает следующим образом:

1. Выбрать элемент-опорный из списка.
2. Разделить список на два подсписка: один с элементами, меньшими, чем опорный, и другой с элементами, большими, чем опорный.
3. Рекурсивно применить алгоритм Quick Sort к подспискам.
4. Объединить отсортированные подсписки с элементом-опорным посередине.

## Требования

Для реализации алгоритма Quick Sort на Python должны быть выполнены следующие требования:

- Алгоритм не должен быть in-place решением.
- Алгоритм должен обрабатывать дублирующиеся элементы в списке.
- Алгоритм должен обрабатывать некорректный ввод, такой как None или несписковый ввод.
- Алгоритм должен быть способен обрабатывать списки любого размера, включая пустые списки.

## Пример использования

Ниже приведены некоторые примеры использования алгоритма Quick Sort в Python:

- None -> Исключение

```python
quick_sort(None)
```

- Пустой ввод -> []

```python
quick_sort([])
```

- Один элемент -> [элемент]

```python
quick_sort([5])
```

- Два или более элементов

```python
quick_sort([5, 2, 8, 3, 1, 9])
```
