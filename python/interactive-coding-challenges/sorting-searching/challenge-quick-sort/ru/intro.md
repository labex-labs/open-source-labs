# Введение

Quick Sort - это популярный алгоритм сортировки, который использует подход "разделяй и властвуй" для сортировки массива или списка. Это сравнительный алгоритм, который работает путём разделения массива на два подмассива, а затем рекурсивной сортировки подмассивов. Quick Sort широко используется из-за своей эффективности и простоты.
