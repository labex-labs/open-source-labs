# Introducción

El método de clasificación Radix es un algoritmo de clasificación de enteros no comparativo que clasifica datos con claves enteras agrupando las claves por los dígitos individuales que comparten la misma posición y valor significativo. Es un algoritmo de clasificación de tiempo lineal que a menudo se utiliza como subrutina en otros algoritmos de clasificación.
