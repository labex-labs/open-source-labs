# Radix sort

## Задача

Реализовать алгоритм Radix sort для сортировки списка целых чисел. Алгоритм сортирует целые числа, сравнивая их цифры, начиная от наименее значимой цифры до наиболее значимой цифры. Алгоритм сначала сортирует целые числа по наименее значимой цифре, затем по второй наименее значимой цифре и так далее, пока не будет отсортирована наиболее значимая цифра.

## Требования

Для реализации Radix sort необходимо выполнить следующие требования:

- Входными данными должен быть список целых чисел.
- Проверить наличие None вместо массива.
- Предполагается, что элементы массива - целые числа.
- Цифры должны быть в десятичной системе счисления.
- Алгоритм должен быть способен обрабатывать любое количество цифр.
- Алгоритм должен помещаться в память.

## Пример использования

Ниже приведены примеры использования алгоритма Radix sort:

- None -> Исключение
- [] -> []
- [128, 256, 164, 8, 2, 148, 212, 242, 244] -> [2, 8, 128, 148, 164, 212, 242, 244, 256]
