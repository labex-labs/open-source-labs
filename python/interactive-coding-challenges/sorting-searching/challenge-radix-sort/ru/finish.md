# Резюме

Radix sort - это эффективный алгоритм для сортировки целых чисел. Он сортирует целые числа, сравнивая их цифры, начиная от наименее значимой цифры до наиболее значимой цифры. Алгоритм имеет линейную сложность по времени и часто используется как подпрограмма в других алгоритмах сортировки.
