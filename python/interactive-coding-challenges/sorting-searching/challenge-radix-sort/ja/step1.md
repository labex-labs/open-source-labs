# ラディックスソート

## 問題

整数のリストをソートするために、ラディックスソートアルゴリズムを実装します。このアルゴリズムは、最下位桁から最上位桁までの桁を比較することで整数をソートします。アルゴリズムはまず最下位桁に基づいて整数をソートし、次に 2 番目に最下位桁、そしてそれ以降も同様にして最上位桁がソートされるまで続けます。

## 要件

ラディックスソートを実装するには、以下の要件を満たす必要があります。

- 入力は整数のリストでなければなりません。
- 配列の代わりに None をチェックします。
- 配列要素は整数であると仮定します。
- 桁は 10 進数でなければなりません。
- アルゴリズムは任意の桁数に対応できる必要があります。
- アルゴリズムはメモリに収まる必要があります。

## 使い方の例

以下は、ラディックスソートアルゴリズムの使い方の例です。

- None -> 例外
- [] -> []
- [128, 256, 164, 8, 2, 148, 212, 242, 244] -> [2, 8, 128, 148, 164, 212, 242, 244, 256]
