# Zusammenfassung

Der基数排序ist ein effizienter Algorithmus zum Sortieren von ganzen Zahlen. Es sortiert die ganzen Zahlen, indem es ihre Ziffern von der am wenigsten signifikanten Ziffer bis zur am meisten signifikanten Ziffer vergleicht. Der Algorithmus hat eine lineare Zeitkomplexität und wird oft als Unterprogramm in anderen Sortieralgorithmen verwendet.
