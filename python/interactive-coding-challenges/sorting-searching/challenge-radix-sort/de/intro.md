# Einführung

Der 基数排序 ist ein nicht-vergleichender ganzzahliger Sortieralgorithmus, der Daten mit ganzzahligen Schlüsseln sortiert, indem er die Schlüssel nach den einzelnen Ziffern gruppiert, die die gleiche signifikante Position und den gleichen Wert haben. Es ist ein Sortieralgorithmus in linearer Zeit, der oft als Unterprogramm in anderen Sortieralgorithmen verwendet wird.
