# Introduction

Le tri par radix est un algorithme de tri d'entiers non comparatif qui trie des données avec des clés entières en regroupant les clés par les chiffres individuels qui ont la même position et valeur significative. C'est un algorithme de tri en temps linéaire qui est souvent utilisé comme sous-routine dans d'autres algorithmes de tri.
