# Introduction

Le tri par insertion est un algorithme de tri simple qui fonctionne en insérant répétitivement l'élément suivant dans le tableau trié. C'est un algorithme de tri basé sur les comparaisons en place qui construit le tableau trié final un élément à la fois.

Les challenges sont généralement un peu difficiles. Si vous trouvez cela difficile, vous pouvez discuter avec Labby ou consulter la solution.
