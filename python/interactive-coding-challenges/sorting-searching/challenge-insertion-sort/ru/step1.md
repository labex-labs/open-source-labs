# Сортировка вставками

## Задача

Задача заключается в реализации сортировки вставками на Python. Дан неотсортированный список элементов, и алгоритм должен отсортировать список по возрастанию. Алгоритм работает путём итерации по списку и вставки каждого элемента в его правильную позицию в отсортированной части списка.

Алгоритм начинается с предположения, что первый элемент в списке уже отсортирован. Затем он итерируется по оставшимся элементам списка, сравнивая каждый элемент с элементами в отсортированной части списка. Если элемент меньше текущего элемента в отсортированной части списка, он вставляется перед этим элементом. Если элемент больше всех элементов в отсортированной части списка, он вставляется в конец отсортированной части списка.

## Требования

Для реализации сортировки вставками на Python должны быть выполнены следующие требования:

- Простое решение достаточно.
- Дубликаты допускаются.
- Входные данные не обязательно действительны, поэтому алгоритм должен优雅но обрабатывать недействительные входные данные.
- Алгоритм должен помещаться в память.

## Пример использования

Ниже приведены некоторые примеры использования алгоритма сортировки вставками:

- None -> Исключение: Если входные данные равны None, должно быть поднято исключение.
- Пустой ввод -> []: Если входные данные - это пустой список, выход должен также быть пустым списком.
- Один элемент -> [элемент]: Если входные данные - это список с одним элементом, выход должен быть тем же списком.
- Два или более элементов: Если входные данные - это список с двумя или более элементами, выход должен быть отсортированным списком по возрастанию.
