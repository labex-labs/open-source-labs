# Обзор

Сортировка вставками - это простой и эффективный алгоритм сортировки, который хорошо работает для небольших списков или частично отсортированных списков. В худшем случае его временная сложность составляет O(n^2), но он может быть быстрее других алгоритмов сортировки для небольших списков или частично отсортированных списков. Алгоритм работает путём последовательной вставки следующего элемента в отсортированный массив, формируя окончательный отсортированный массив по одному элементу за раз. Реализация сортировки вставками на Python должна соответствовать вышеупомянутым требованиям и优雅но обрабатывать недействительные входные данные.
