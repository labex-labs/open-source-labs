# Merge Sort

## Problema

Implementar el merge sort en Python. Dada una matriz no ordenada, el algoritmo debe ordenar la matriz en orden ascendente. El algoritmo debe ser capaz de manejar matrices de cualquier longitud y debe funcionar tanto para enteros como para cadenas.

## Requisitos

Para implementar el merge sort en Python, deben cumplirse los siguientes requisitos:

- El algoritmo debe utilizar el enfoque divide y vencerás.
- El algoritmo debe ser capaz de manejar matrices de cualquier longitud.
- El algoritmo debe funcionar tanto para enteros como para cadenas.
- El algoritmo debe ordenar la matriz en orden ascendente.
- La complejidad temporal del algoritmo debe ser O(n log n).
- El algoritmo debe implementarse en Python.

## Uso de ejemplo

Los siguientes ejemplos demuestran el uso del merge sort:

- Ninguno -> Excepción: El algoritmo debe generar una excepción si la entrada es None.
- Entrada vacía -> []: El algoritmo debe devolver una matriz vacía si la entrada es una matriz vacía.
- Un solo elemento -> [elemento]: El algoritmo debe devolver la misma matriz si la entrada contiene solo un elemento.
- Dos o más elementos: El algoritmo debe ordenar la matriz en orden ascendente. Si las submatrices izquierda y derecha tienen longitudes diferentes, el algoritmo todavía debe ser capaz de ordenar correctamente la matriz.
