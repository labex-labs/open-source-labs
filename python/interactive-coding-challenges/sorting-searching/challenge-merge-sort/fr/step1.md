# Tri fusion

## Problème

Implémentez le tri fusion en Python. Étant donné un tableau non trié, l'algorithme devrait trier le tableau par ordre croissant. L'algorithme devrait être capable de gérer des tableaux de toute longueur et devrait fonctionner pour les entiers et les chaînes de caractères.

## Exigences

Pour implémenter le tri fusion en Python, les exigences suivantes doivent être satisfaites :

- L'algorithme devrait utiliser l'approche diviser pour régner.
- L'algorithme devrait être capable de gérer des tableaux de toute longueur.
- L'algorithme devrait fonctionner pour les entiers et les chaînes de caractères.
- L'algorithme devrait trier le tableau par ordre croissant.
- La complexité temporelle de l'algorithme devrait être O(n log n).
- L'algorithme devrait être implémenté en Python.

## Utilisation de l'exemple

Les exemples suivants démontrent l'utilisation du tri fusion :

- None -> Exception : L'algorithme devrait lever une exception si l'entrée est None.
- Entrée vide -> [] : L'algorithme devrait renvoyer un tableau vide si l'entrée est un tableau vide.
- Un élément -> [élément] : L'algorithme devrait renvoyer le même tableau si l'entrée ne contient qu'un seul élément.
- Deux éléments ou plus : L'algorithme devrait trier le tableau par ordre croissant. Si les sous-tableaux gauche et droit ont des longueurs différentes, l'algorithme devrait toujours être capable de trier le tableau correctement.
