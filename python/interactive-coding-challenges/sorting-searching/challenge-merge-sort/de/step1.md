# Merge Sort

## Problem

Implementiere Merge Sort in Python. Gegeben ein unsortiertes Array, sollte der Algorithmus das Array aufsteigend sortieren. Der Algorithmus sollte in der Lage sein, Arrays beliebiger Länge zu verarbeiten und sowohl für Integer als auch für Strings funktionieren.

## Anforderungen

Um Merge Sort in Python zu implementieren, müssen die folgenden Anforderungen erfüllt werden:

- Der Algorithmus sollte die Divide-and-Conquer-Methode verwenden.
- Der Algorithmus sollte in der Lage sein, Arrays beliebiger Länge zu verarbeiten.
- Der Algorithmus sollte sowohl für Integer als auch für Strings funktionieren.
- Der Algorithmus sollte das Array aufsteigend sortieren.
- Die Zeitkomplexität des Algorithmus sollte O(n log n) betragen.
- Der Algorithmus sollte in Python implementiert werden.

## Beispielverwendung

Die folgenden Beispiele demonstrieren die Verwendung von Merge Sort:

- None -> Exception: Der Algorithmus sollte eine Exception werfen, wenn die Eingabe None ist.
- Leere Eingabe -> []: Der Algorithmus sollte ein leeres Array zurückgeben, wenn die Eingabe ein leeres Array ist.
- Ein Element -> [Element]: Der Algorithmus sollte das gleiche Array zurückgeben, wenn die Eingabe nur ein Element enthält.
- Zwei oder mehr Elemente: Der Algorithmus sollte das Array aufsteigend sortieren. Wenn die linke und rechte Teilarray unterschiedliche Längen haben, sollte der Algorithmus immer noch das Array richtig sortieren können.
