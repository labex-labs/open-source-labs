# マージソート

## 問題

Python でマージソートを実装します。未ソートの配列が与えられた場合、アルゴリズムは配列を昇順にソートする必要があります。アルゴリズムは任意の長さの配列を処理できるようにし、整数と文字列の両方に対して機能する必要があります。

## 要件

Python でマージソートを実装するには、次の要件を満たす必要があります。

- アルゴリズムは分割統治法を使用する必要があります。
- アルゴリズムは任意の長さの配列を処理できる必要があります。
- アルゴリズムは整数と文字列の両方に対して機能する必要があります。
- アルゴリズムは配列を昇順にソートする必要があります。
- アルゴリズムの時間計算量は O(n log n) である必要があります。
- アルゴリズムは Python で実装する必要があります。

## 例の使用法

次の例は、マージソートの使用方法を示しています。

- なし -> 例外: 入力が None の場合、アルゴリズムは例外を発生させる必要があります。
- 空の入力 -> []: 入力が空の配列の場合、アルゴリズムは空の配列を返す必要があります。
- 1 つの要素 -> [要素]: 入力が 1 つの要素のみ含む場合、アルゴリズムは同じ配列を返す必要があります。
- 2 つ以上の要素: アルゴリズムは配列を昇順にソートする必要があります。左と右のサブ配列の長さが異なる場合でも、アルゴリズムは正しく配列をソートできる必要があります。
