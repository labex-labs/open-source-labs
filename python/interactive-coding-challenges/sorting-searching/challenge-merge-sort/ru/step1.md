# Merge Sort

## Задача

Реализовать merge sort на Python. Дан неотсортированный массив, алгоритм должен отсортировать массив по возрастанию. Алгоритм должен быть способен обрабатывать массивы любого размера и работать как с целыми числами, так и со строками.

## Требования

Для реализации merge sort на Python необходимо выполнить следующие требования:

- Алгоритм должен использовать подход "разделяй и властвуй".
- Алгоритм должен быть способен обрабатывать массивы любого размера.
- Алгоритм должен работать как с целыми числами, так и со строками.
- Алгоритм должен отсортировать массив по возрастанию.
- Временная сложность алгоритма должна быть O(n log n).
- Алгоритм должен быть реализован на Python.

## Пример использования

Следующие примеры демонстрируют использование merge sort:

- None -> Исключение: Алгоритм должен вызывать исключение, если входные данные равны None.
- Пустой ввод -> []: Алгоритм должен возвращать пустой массив, если входные данные представляют собой пустой массив.
- Один элемент -> [элемент]: Алгоритм должен возвращать тот же массив, если входные данные содержат только один элемент.
- Два или более элементов: Алгоритм должен отсортировать массив по возрастанию. Если левая и правая подмассивы имеют разную длину, алгоритм все равно должен быть способен правильно отсортировать массив.
