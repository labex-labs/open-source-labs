# Резюме

Merge sort - это популярный алгоритм сортировки, который использует подход "разделяй и властвуй" для сортировки массива. Для реализации merge sort на Python алгоритм должен использовать подход "разделяй и властвуй", обрабатывать массивы любого размера, работать как с целыми числами, так и со строками, сортировать массив по возрастанию, иметь временную сложность O(n log n) и быть реализован на Python. Алгоритм должен вызывать исключение, если входные данные равны None, возвращать пустой массив, если входные данные представляют собой пустой массив, возвращать тот же массив, если входные данные содержат только один элемент, и правильно сортировать массив, если левая и правая подмассивы имеют разную длину.
