# Анаграммы

## Задача

Дан массив строк. Напишите функцию для сортировки массива так, чтобы все анаграммы были расположены рядом друг с другом. Анаграмма определяется как слово или фраза, образованные путём перестановки букв другого слова или фразы. Например, "act" и "cat" - это анаграммы друг друга.

## Требования

Для решения этой задачи должны быть выполнены следующие требования:

- Функция должна сгруппировать все анаграммы вместе в отсортированном массиве.
- Кроме группировки анаграмм нет других требований к сортировке.
- Входные данные могут быть недействительными, поэтому функция должна обрабатывать недействительные входы.
- Функция должна помещаться в память.

## Пример использования

Функция должна работать следующим образом в этих сценариях:

- None -> Исключение
- [] -> []
- Общий случай
  - Вход: ['ram', 'act', 'arm', 'bat', 'cat', 'tab']
  - Результат: ['arm', 'ram', 'act', 'cat', 'bat', 'tab']
