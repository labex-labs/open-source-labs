# Einführung

Ein Anagramm ist ein Wort oder eine Phrase, die durch Umordnung der Buchstaben eines anderen Wortes oder einer anderen Phrase gebildet wird. Beispielsweise sind "listen" und "silent" Anagramme voneinander. In dieser Herausforderung werden wir ein Array von Strings sortieren, sodass alle Anagramme nebeneinander stehen.
