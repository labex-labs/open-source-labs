# アナグラム

## 問題

文字列の配列が与えられたとき、その配列をソートして、すべてのアナグラムが隣り合うようにする関数を書きなさい。アナグラムとは、別の単語または句の文字を並べ替えて作られる単語または句のことです。たとえば、「act」と「cat」はお互いのアナグラムです。

## 要件

この問題を解くには、次の要件を満たす必要があります。

- 関数は、ソートされた配列の中ですべてのアナグラムをまとめる必要があります。
- アナグラムのグループ化以外には、他のソート要件はありません。
- 入力が有効でない場合があるため、関数は無効な入力を処理する必要があります。
- 関数はメモリ内に収まる必要があります。

## 例の使用法

この関数は、次のシナリオで次のように動作する必要があります。

- なし -> 例外
- [] -> []
- 一般的なケース
  - 入力：['ram', 'act', 'arm', 'bat', 'cat', 'tab']
  - 結果：['arm', 'ram', 'act', 'cat', 'bat', 'tab']
