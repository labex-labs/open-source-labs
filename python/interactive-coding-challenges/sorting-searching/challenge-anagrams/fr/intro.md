# Introduction

Un anagramme est un mot ou une phrase formée en réarrangeant les lettres d'un autre mot ou phrase. Par exemple, "listen" et "silent" sont des anagrammes l'un de l'autre. Dans ce défi, nous allons trier un tableau de chaînes de caractères de sorte que tous les anagrammes soient les uns à côté des autres.
