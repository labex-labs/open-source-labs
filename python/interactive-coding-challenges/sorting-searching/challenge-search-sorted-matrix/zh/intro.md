# 简介

在计算机科学中，矩阵是一个由数字、符号或表达式组成的二维数组，按行和列排列。有序矩阵是指行和列都按升序排序的矩阵。在这个挑战中，我们会得到一个有序矩阵，需要在其中搜索特定的元素。
