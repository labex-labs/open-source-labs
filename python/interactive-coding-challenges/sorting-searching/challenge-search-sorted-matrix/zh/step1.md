# 在有序矩阵中搜索

## 问题

给定一个有序矩阵，我们需要在其中搜索特定的元素。该矩阵的排序方式是每行和每列的元素都按升序排列。矩阵不一定是方阵。如果找到该元素，我们需要返回其在矩阵中的位置，以元组 (行, 列) 的形式表示；否则，返回 None。

## 要求

为了解决这个问题，我们需要做出以下假设：

- 矩阵每行的元素按升序排列。
- 矩阵每列的元素按升序排列。
- 矩阵不是锯齿状的，即它是一个矩形。
- 排序顺序是升序。
- 矩阵不一定是方阵。
- 输出是一个元组 (行, 列)。
- 我们要搜索的元素可能在矩阵中，也可能不在。
- 输入可能有效，也可能无效。
- 解决方案应能在内存中运行。

## 示例用法

我们可以使用以下测试用例来验证我们的解决方案：

- 如果输入为 None，函数应引发异常。
- 如果在矩阵中找到该元素，函数应返回其位置，以元组 (行, 列) 的形式表示。
- 如果在矩阵中未找到该元素，函数应返回 None。
