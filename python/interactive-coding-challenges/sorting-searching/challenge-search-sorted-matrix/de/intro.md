# Einführung

In der Informatik ist eine Matrix ein zweidimensionales Array von Zahlen, Symbolen oder Ausdrücken, das in Zeilen und Spalten angeordnet ist. Eine sortierte Matrix ist eine Matrix, in der die Zeilen und Spalten in aufsteigender Reihenfolge sortiert sind. In dieser Herausforderung erhalten wir eine sortierte Matrix und müssen darin ein bestimmtes Element suchen.
