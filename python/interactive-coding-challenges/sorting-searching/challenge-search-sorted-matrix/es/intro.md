# Introducción

En informática, una matriz es una matriz bidimensional de números, símbolos o expresiones dispuestas en filas y columnas. Una matriz ordenada es una matriz en la que las filas y columnas están ordenadas en orden ascendente. En este desafío, se nos da una matriz ordenada y debemos buscar un elemento específico en ella.
