# Поиск в отсортированной матрице

## Задача

Дана отсортированная матрица, и нам нужно искать в ней определенный элемент. Матрица отсортирована так, что элементы в каждой строке и столбце отсортированы в порядке возрастания. Матрица не обязательно является квадратной. Если элемент найден, мы должны вернуть его позицию в матрице в виде кортежа (строка, столбец), иначе вернуть None.

## Требования

Для решения этой задачи мы должны сделать следующие предположения:

- Элементы в каждой строке матрицы отсортированы в порядке возрастания.
- Элементы в каждом столбце матрицы отсортированы в порядке возрастания.
- Матрица не имеет ступенчатости, то есть является прямоугольником.
- Порядок сортировки - по возрастанию.
- Матрица не обязательно является квадратной.
- Выход - это кортеж (строка, столбец).
- Элемент, который мы ищем, может быть или не быть в матрице.
- Входные данные могут быть или не быть валидными.
- Решение должно помещаться в память.

## Пример использования

Мы можем использовать следующие тестовые случаи, чтобы проверить наше решение:

- Если входные данные равны None, функция должна вызывать исключение.
- Если элемент найден в матрице, функция должна вернуть его позицию в виде кортежа (строка, столбец).
- Если элемент не найден в матрице, функция должна вернуть None.
