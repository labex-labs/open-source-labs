# ソート済み行列の検索

## 問題

ソート済み行列が与えられたとき、その中から特定の項目を検索する必要があります。行列は各行と各列の項目が昇順にソートされた状態です。行列は必ずしも正方行列ではありません。見つかった場合、行列内の項目の位置をタプル (行, 列) として返す必要があります。そうでない場合は、None を返す必要があります。

## 要件

この問題を解くには、以下の仮定を行います。

- 行列の各行の項目は昇順にソートされています。
- 行列の各列の項目は昇順にソートされています。
- 行列は不規則ではなく、すなわち長方形です。
- ソート順は昇順です。
- 行列は必ずしも正方行列ではありません。
- 出力はタプル (行, 列) です。
- 検索対象の項目は行列に含まれている場合もあれば含まれていない場合もあります。
- 入力は有効である場合もあれば無効である場合もあります。
- 解はメモリに収まる必要があります。

## 例の使用方法

以下のテストケースを使用して解を検証できます。

- 入力が None の場合、関数は例外を発生させる必要があります。
- 項目が行列内に見つかった場合、関数はその位置をタプル (行, 列) として返す必要があります。
- 項目が行列内に見つからなかった場合、関数は None を返す必要があります。
