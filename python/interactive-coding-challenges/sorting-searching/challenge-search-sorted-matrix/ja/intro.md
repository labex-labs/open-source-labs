# はじめに

コンピュータサイエンスにおいて、行列（マトリックス）は、行と列に並べられた数値、記号、または式の二次元配列です。ソート済み行列は、行と列が昇順にソートされた行列です。このチャレンジでは、ソート済み行列が与えられ、その中から特定の項目を検索する必要があります。
