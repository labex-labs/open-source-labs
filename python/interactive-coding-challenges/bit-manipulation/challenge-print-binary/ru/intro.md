# Введение

В информатике двоичная система счисления - это система счисления, которая использует всего два цифры: 0 и 1. Она широко используется в цифровой электроннике и компьютерных системах. В этом испытании мы напишем программу на Python для вывода двоичного представления заданного вещественного числа от 0 до 1.
