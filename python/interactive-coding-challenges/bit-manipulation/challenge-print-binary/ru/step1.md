# Вывести двоичное представление

## Задача

Напишите функцию на Python, которая принимает в качестве входных данных вещественное число от 0 до 1 и возвращает его двоичное представление в виде строки. Если длина представления превышает 32, верните 'ERROR'.

## Требования

Для решения этой задачи необходимо обеспечить следующие требования:

- Входные данные должны быть вещественным числом (float).
- Выход должен быть строкой.
- Диапазон входных данных находится между 0 и 1, при этом значения 0 и 1 не включаются.
- Результат должен содержать конечный ноль и десятичную точку.
- Стартовый ноль и десятичная точка учитываются в пределах 32-символьного лимита.
- Мы не можем предполагать, что входные данные действительны.
- Мы можем предположить, что программа помещается в память.

## Пример использования

Ниже приведены примеры того, как должна работать функция:

- None -> 'ERROR'
- Вне допустимого диапазона (0, 1) -> 'ERROR'
- 0.625 -> 0.101
- 0.987654321 -> 'ERROR'
