# はじめに

コンピュータサイエンスにおいて、2 進数は 0 と 1 の 2 桁のみを使う数値表記法です。これはデジタルエレクトロニクスやコンピュータシステムで広く使われています。このチャレンジでは、0 から 1 の間の与えられた実数の 2 進数表現を出力する Python プログラムを書きます。
