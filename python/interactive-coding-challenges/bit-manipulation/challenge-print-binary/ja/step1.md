# 2 進数を出力する

## 問題

0 から 1 の間の実数を入力として受け取り、その 2 進数表現を文字列として返す Python 関数を書きます。表現の長さが 32 を超える場合は、'ERROR' を返します。

## 要件

この問題を解くには、次の要件を満たす必要があります。

- 入力は浮動小数点数でなければなりません。
- 出力は文字列でなければなりません。
- 入力の範囲は 0 から 1 の間ですが、0 と 1 の値は含まれません。
- 結果には末尾のゼロと小数点が含まれる必要があります。
- 先頭のゼロと小数点は 32 文字の制限に含まれます。
- 入力が有効であることを前提とすることはできません。
- プログラムがメモリに収まることを前提とすることができます。

## 使い方の例

この関数がどのように動作するかの例をいくつか示します。

- なし -> 'ERROR'
- 範囲外 (0, 1) -> 'ERROR'
- 0.625 -> 0.101
- 0.987654321 -> 'ERROR'
