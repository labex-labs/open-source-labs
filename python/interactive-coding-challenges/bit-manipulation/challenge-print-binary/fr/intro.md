# Introduction

En informatique, le binaire est un système de numérotation qui utilise seulement deux chiffres, 0 et 1. Il est largement utilisé en électronique numérique et dans les systèmes informatiques. Dans ce défi, nous allons écrire un programme Python pour afficher la représentation binaire d'un nombre réel donné compris entre 0 et 1.
