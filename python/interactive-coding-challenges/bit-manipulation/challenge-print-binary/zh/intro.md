# 简介

在计算机科学中，二进制是一种仅使用两个数字 0 和 1 的编号系统。它广泛应用于数字电子学和计算机系统。在这个挑战中，我们将编写一个 Python 程序来打印给定的 0 到 1 之间实数的二进制表示形式。
