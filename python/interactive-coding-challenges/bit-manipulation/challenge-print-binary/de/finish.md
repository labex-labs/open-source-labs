# Zusammenfassung

In dieser Herausforderung haben wir gelernt, wie man ein Python-Programm schreibt, um die binäre Darstellung einer gegebenen reellen Zahl zwischen 0 und 1 auszugeben. Wir haben auch die Anforderungen an das Programm definiert und einige Beispiele dafür gegeben, wie es sich verhalten sollte.
