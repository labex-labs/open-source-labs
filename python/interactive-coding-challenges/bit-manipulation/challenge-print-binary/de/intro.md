# Einführung

In der Informatik ist das Binärsystem ein Zahlensystem, das nur zwei Ziffern, 0 und 1, verwendet. Es wird in der Digitalelektronik und in Computernetzen weit verbreitet eingesetzt. In dieser Herausforderung werden wir ein Python-Programm schreiben, um die binäre Darstellung einer gegebenen reellen Zahl zwischen 0 und 1 auszugeben.
