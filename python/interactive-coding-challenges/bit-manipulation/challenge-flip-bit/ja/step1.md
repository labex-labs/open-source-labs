# ビット反転

## 問題

2進数が与えられたとき、そのビットのうち1つを0から1に反転させて、1の最長シーケンスを最大化する必要があります。たとえば、2進数 `000011110000` が与えられた場合、4番目のビットを0から1に反転させて `000111110000` を得ることができ、これは5つの1のシーケンスを持ちます。私たちの目標は、2進数を入力として受け取り、1ビットを反転させた後の1の最長シーケンスの長さを返すPython関数を書くことです。

## 要件

私たちのPython関数の要件は以下の通りです。

- 入力は2進数の整数でなければなりません。
- 入力は32ビット数であると仮定できます。
- 入力の長さを検証する必要はありません。
- 出力は整数でなければなりません。
- 入力が有効であることを前提とする必要はありません。
- Pythonに >>> 演算子がないため、正数を使用していると仮定できます。
- これがメモリに収まると仮定できます。

## 例の使用法

私たちのPython関数がどのように動作するかの例をいくつか示します。

- `None` -> 例外
- `11111111111111111111111111111111` -> 32
- `00000000000000000000000000000000` -> 1
- `00001111110111011111001111110000` -> 10
