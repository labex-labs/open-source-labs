# ビット反転

## 問題

2 進数が与えられたとき、そのビットのうち 1 つを 0 から 1 に反転させて、1 の最長シーケンスを最大化する必要があります。たとえば、2 進数 `000011110000` が与えられた場合、4 番目のビットを 0 から 1 に反転させて `000111110000` を得ることができ、これは 5 つの 1 のシーケンスを持ちます。私たちの目標は、2 進数を入力として受け取り、1 ビットを反転させた後の 1 の最長シーケンスの長さを返す Python 関数を書くことです。

## 要件

私たちの Python 関数の要件は以下の通りです。

- 入力は 2 進数の整数でなければなりません。
- 入力は 32 ビット数であると仮定できます。
- 入力の長さを検証する必要はありません。
- 出力は整数でなければなりません。
- 入力が有効であることを前提とする必要はありません。
- Python に >>> 演算子がないため、正数を使用していると仮定できます。
- これがメモリに収まると仮定できます。

## 例の使用法

私たちの Python 関数がどのように動作するかの例をいくつか示します。

- `None` -> 例外
- `11111111111111111111111111111111` -> 32
- `00000000000000000000000000000000` -> 1
- `00001111110111011111001111110000` -> 10
