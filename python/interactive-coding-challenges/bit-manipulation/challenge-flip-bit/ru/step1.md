# Переключение бита

## Задача

Дан двоичное число. Нам нужно переключить один из его битов с 0 на 1, чтобы максимизировать длину самой длинной последовательности единиц. Например, если у нас двоичное число `000011110000`, мы можем переключить четвертый бит с 0 на 1 и получить `000111110000`, в котором есть последовательность из пяти единиц. Наша задача - написать функцию на Python, которая принимает двоичное число и возвращает длину самой длинной последовательности единиц после переключения одного бита.

## Требования

Требования к нашей функции на Python следующие:

- Вход должен быть целым числом в двоичной системе счисления.
- Мы можем предположить, что вход - это 32-разрядное число.
- Не нужно проверять длину входных данных.
- Выход должен быть целым числом.
- Не можем предполагать, что входные данные действительны.
- Можем предположить, что используем положительное число, так как в Python нет оператора >>>.
- Можем предположить, что это помещается в память.

## Пример использования

Вот несколько примеров того, как должна работать наша функция на Python:

- `None` -> Исключение
- `11111111111111111111111111111111` -> 32
- `00000000000000000000000000000000` -> 1
- `00001111110111011111001111110000` -> 10
