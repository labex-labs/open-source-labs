# Einführung

In der Informatik ist das Umkehren eines Bits eine Technik, um einzelne Bits von Daten zu manipulieren. In dieser Herausforderung werden wir ein Bit von 0 auf 1 umkehren, um die längste Folge von Einsen zu maximieren.
