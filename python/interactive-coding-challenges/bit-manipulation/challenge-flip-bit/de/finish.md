# Zusammenfassung

In dieser Herausforderung haben wir gelernt, wie man ein Bit von 0 auf 1 umkehren kann, um die längste Folge von Einsen in einer binären Zahl zu maximieren. Wir haben auch eine Python-Funktion geschrieben, die eine binäre Zahl als Eingabe nimmt und die Länge der längsten Folge von Einsen nach dem Umkehren eines Bits zurückgibt.
