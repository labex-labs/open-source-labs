# Вставить M в N

## Задача

Даны два 16-битных числа, `n` и `m`, и два индекса `i` и `j`. Вставьте `m` в `n` так, чтобы `m` начиналось с бита `j` и заканчивалось на бите `i`. Программа должна обрабатывать следующие случаи:

- Если не задано ни одного входного значения, должно быть вызвано исключение.
- Если задан отрицательный индекс для `i` или `j`, должно быть вызвано исключение.
- Если входные данные недействительны, должно быть вызвано исключение.
- Если в диапазоне от `i` до `j` не хватает места для `m`, должно быть вызвано исключение.

Программа должна возвращать полученное 16-битное число после вставки.

## Требования

Программа должна соответствовать следующим требованиям:

- `j` должно быть больше, чем `i`.
- В диапазоне от `i` до `j` должно быть достаточно места для `m`.
- Входные данные должны быть действительными.
- Программа должна помещаться в память.

## Пример использования

Вот пример использования программы:

```txt
i      = 2
j      = 6
n      = 0000 0100 0000 0000
m      = 0000 0000 0001 0011
result = 0000 0100 0100 1100
```

В этом примере `m` вставляется в `n` так, чтобы `m` начиналось с бита `j = 6` и заканчивалось на бите `i = 2`. Полученное 16-битное число равно `0000 0100 0100 1100`.
