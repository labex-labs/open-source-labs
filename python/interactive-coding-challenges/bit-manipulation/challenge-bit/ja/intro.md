# はじめに

ビット操作はコンピュータサイエンスにおける基本概念であり、様々な分野で広く使用されています。これは、2 進数を構成する個々のビットを操作することを意味します。Python では、ビットを取得、設定、クリア、更新するなどの一般的なビット操作を行うことができます。このチャレンジでは、これらの操作を実装します。
