# Introducción

La manipulación de bits es un concepto fundamental en informática y se utiliza ampliamente en diversos campos. Consiste en manipular los bits individuales que forman un número binario. En Python, podemos realizar operaciones comunes de manipulación de bits, como obtener un bit, establecer un bit, borrar un bit y actualizar un bit. En este desafío, implementaremos estas operaciones.
