# Einführung

Das Bitmanipulation ist ein grundlegendes Konzept in der Informatik und wird in verschiedenen Bereichen weit verbreitet eingesetzt. Es beinhaltet das Manipulieren einzelner Bits, die eine binäre Zahl bilden. In Python können wir übliche Bitmanipulationsoperationen durchführen, wie das Abrufen eines Bits, das Setzen eines Bits, das Löschen eines Bits und das Aktualisieren eines Bits. In dieser Herausforderung werden wir diese Operationen implementieren.
