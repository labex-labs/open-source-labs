# Получить следующее

## Задача

Дано положительное целое число. Необходимо найти следующее наибольшее и следующее наименьшее числа, содержащие такое же количество единиц, как у заданного числа. Например, если входное значение 0000 0000 1101 0111, то в качестве результата для следующего наибольшего числа должно быть 0000 0000 1101 1011, а для следующего наименьшего - 0000 0000 1100 1111.

## Требования

Для решения этой задачи необходимо выполнить следующие требования:

- Выход должен быть положительным целым числом.
- Входные данные могут быть недействительными, поэтому необходимо обрабатывать исключения.
- Решение должно помещаться в память.

## Пример использования

Ниже приведены примеры использования этой функции:

- None -> Исключение
- 0 -> Исключение
- отрицательное целое число -> Исключение
- Общий случай:

```txt
    * Вход:         0000 0000 1101 0111
    * Следующее наибольшее:  0000 0000 1101 1011
    * Следующее наименьшее: 0000 0000 1100 1111
```
