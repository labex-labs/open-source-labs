# 求下一个数

## 问题

给定一个正整数，你需要找出与给定数字具有相同数量的 1 的下一个最大数和下一个最小数。例如，如果输入是 0000 0000 1101 0111，那么下一个最大数的输出应该是 0000 0000 1101 1011，下一个最小数的输出应该是 0000 0000 1100 1111。

## 要求

为了解决这个挑战，你需要满足以下要求：

- 输出应该是一个正整数。
- 输入可能无效，所以你需要处理异常。
- 解决方案应该适合内存。

## 示例用法

以下是一些如何使用此函数的示例：

- 无 -> 异常
- 0 -> 异常
- 负整数 -> 异常
- 一般情况：

```txt
    * 输入：0000 0000 1101 0111
    * 下一个最大数：0000 0000 1101 1011
    * 下一个最小数：0000 0000 1100 1111
```
