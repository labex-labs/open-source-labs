# 次の数を取得する

## 問題

与えられた正の整数に対して、その数と同じ数の1を持つ次に大きい数と次に小さい数を見つける必要があります。たとえば、入力が0000 0000 1101 0111の場合、次に大きい数の出力は0000 0000 1101 1011で、次に小さい数の出力は0000 0000 1100 1111でなければなりません。

## 要件

このチャレンジを解くには、次の要件を満たす必要があります。

- 出力は正の整数でなければなりません。
- 入力が有効でない場合があるため、例外を処理する必要があります。
- ソリューションはメモリに収まる必要があります。

## 例の使用方法

この関数の使用方法の例をいくつか示します。

- なし -> 例外
- 0 -> 例外
- 負の整数 -> 例外
- 一般的なケース：

```txt
    * 入力：         0000 0000 1101 0111
    * 次に大きい数： 0000 0000 1101 1011
    * 次に小さい数： 0000 0000 1100 1111
```
