# Einführung

In der Computergrafik ist eine Linie ein grundlegendes Element, das auf einem Bildschirm gezeichnet werden kann. Das Zeichnen einer Linie auf einem Bildschirm umfasst das Setzen der entsprechenden Bits im Speicher des Bildschirms. In dieser Herausforderung werden wir eine Methode implementieren, die eine Linie auf einem Bildschirm zeichnet, wenn seine Breite und zwei absolute Pixelpositionen angegeben werden.
