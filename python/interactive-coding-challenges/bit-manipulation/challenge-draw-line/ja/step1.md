# 線を描画する

## 問題

`draw_line(screen, width, x1, x2)` メソッドを実装します。ここで、`screen` はバイトのリストで、`width` は 8 で割り切れ、`x1`、`x2` は絶対ピクセル位置です。このメソッドは、`x1` から `x2` までの線を描画するために、`screen` 内の対応するビットを設定する必要があります。

### 要件

`draw_line` の実装は、次の要件を満たさなければなりません。

- 入力が妥当であることは想定されません。
- `screen` 内の対応するビットが設定されて、線が描画される必要があります。
- 実装がメモリに収まることが想定されます。

## 例

次の例は、`draw_line` の期待される動作を示しています。

- 無効な入力 -> 例外
  - `screen` が空
  - `width` = 0
  - 入力パラメータのいずれかが `None`
  - `x1` または `x2` が範囲外
- `len(screen)` = 20、`width` = 32 の一般的なケース：
  - `x1` = 2、`x2` = 6
    - `screen[0]` = `int('00111110', base=2)`
  - `x1` = 68、`x2` = 80
    - `screen[8]`, `int('00001111', base=2)`
    - `screen[9]`, `int('11111111', base=2)`
    - `screen[10]`, `int('10000000', base=2)`
