# Отрисовка линии

## Задача

Реализуйте метод `draw_line(screen, width, x1, x2)`, где `screen` - это список байтов, `width` делится на 8, а `x1`, `x2` - абсолютные позиции пикселей. Метод должен установить соответствующие биты в `screen`, чтобы нарисовать линию от `x1` до `x2`.

### Требования

Реализация `draw_line` должна соответствовать следующим требованиям:

- Не может быть предположено, что входные данные действительны.
- Соответствующие биты в `screen` должны быть установлены для рисования линии.
- Можно предположить, что реализация помещается в память.

## Пример использования

Следующие примеры иллюстрируют ожидаемое поведение `draw_line`:

- Недействительные входные данные -> Исключение
  - `screen` пуст
  - `width` = 0
  - любой входной параметр равен `None`
  - `x1` или `x2` выходит за границы
- Общий случай для `len(screen)` = 20, `width` = 32:
  - `x1` = 2, `x2` = 6
    - `screen[0]` = `int('00111110', base=2)`
  - `x1` = 68, `x2` = 80
    - `screen[8]`, `int('00001111', base=2)`
    - `screen[9]`, `int('11111111', base=2)`
    - `screen[10]`, `int('10000000', base=2)`
