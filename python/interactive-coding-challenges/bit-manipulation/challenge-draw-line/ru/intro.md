# Введение

В компьютерной графике линия - это базовый элемент, который можно нарисовать на экране. Отрисовка линии на экране включает настройку соответствующих битов в памяти экрана. В этом испытании мы реализуем метод, который рисует линию на экране по заданной ширине и двум абсолютным позициям пикселей.
