# Introduction

En graphisme informatique, une ligne est un élément de base qui peut être tracée sur un écran. Tracer une ligne sur un écran consiste à définir les bits correspondants dans la mémoire de l'écran. Dans ce défi, nous allons implémenter une méthode qui trace une ligne sur un écran en fonction de sa largeur et de deux positions absolues de pixels.
