# Introducción

En gráficos de computadora, una línea es un elemento básico que se puede dibujar en una pantalla. Dibujar una línea en una pantalla implica establecer los bits correspondientes en la memoria de la pantalla. En este desafío, implementaremos un método que dibuje una línea en una pantalla dada su anchura y dos posiciones absolutas de píxeles.
