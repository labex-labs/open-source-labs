# Introducción

En informática, la manipulación de bits es la acción de manipular algoritmicamente bits u otras piezas de datos más cortas que una palabra. La manipulación de bits se utiliza en criptografía, algoritmos de detección y corrección de errores, procesamiento de señales digitales y muchos otros campos. En este desafío, determinaremos el número de bits que se deben voltear para convertir el entero a en el entero b.
