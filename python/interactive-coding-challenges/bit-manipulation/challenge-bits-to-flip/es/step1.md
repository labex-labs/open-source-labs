# Bits To Flip

## Problema

Dados dos enteros A y B, determinar el número de bits que se deben voltear para convertir A en B. Por ejemplo, si A = 11101 y B = 01111, el número de bits que se deben voltear es 2.

## Requisitos

Para resolver este problema, debemos considerar los siguientes requisitos:

- A y B siempre son enteros.
- La salida siempre es un entero.
- A y B siempre tienen el mismo número de bits.
- Las entradas pueden no ser válidas (None).
- La solución debe caber en memoria.

## Uso de ejemplo

Aquí hay algunos ejemplos de cómo usar esta función:

- Si A o B es None, se debe generar una excepción.
- Si A = 11101 y B = 01111, el número de bits que se deben voltear es 2.
