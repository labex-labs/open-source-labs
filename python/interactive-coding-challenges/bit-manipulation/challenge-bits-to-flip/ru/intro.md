# Введение

В информатике манипуляция битами - это алгоритмическое манипулирование битами или другими фрагментами данных, короче слова. Манipulation битами используется в криптографии, алгоритмах обнаружения и исправления ошибок, цифровой обработке сигналов и многих других областях. В этом испытании мы определим количество бит, которые необходимо перевернуть, чтобы преобразовать целое число a в целое число b.
