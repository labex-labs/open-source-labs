# Переворачиваемые биты

## Задача

Даны два целых числа A и B. Определите количество бит, которые необходимо перевернуть, чтобы преобразовать A в B. Например, если A = 11101 и B = 01111, то количество переворачиваемых битов равно 2.

## Требования

Для решения этой задачи необходимо учитывать следующие требования:

- A и B всегда являются целыми числами.
- Выход всегда является целым числом.
- A и B всегда имеют одинаковое количество бит.
- Входные данные могут быть недействительными (None).
- Решение должно помещаться в память.

## Пример использования

Вот несколько примеров использования этой функции:

- Если A или B равно None, должно быть поднято исключение.
- Если A = 11101 и B = 01111, то количество переворачиваемых битов равно 2.
