# 反転させるビット

## 問題

2つの整数AとBが与えられたとき、AをBに変換するために反転させるビット数を求めます。たとえば、A = 11101でB = 01111の場合、反転させるビット数は2です。

## 要件

この問題を解くには、次の要件を考慮する必要があります。

- AとBは常に整数です。
- 出力は常に整数です。
- AとBは常に同じビット数です。
- 入力が有効でない場合があります（None）。
- ソリューションはメモリに収まる必要があります。

## 例の使用法

この関数の使い方の例をいくつか示します。

- AまたはBがNoneの場合、例外を発生させる必要があります。
- A = 11101でB = 01111の場合、反転させるビット数は2です。
