# 反転させるビット

## 問題

2 つの整数 A と B が与えられたとき、A を B に変換するために反転させるビット数を求めます。たとえば、A = 11101 で B = 01111 の場合、反転させるビット数は 2 です。

## 要件

この問題を解くには、次の要件を考慮する必要があります。

- A と B は常に整数です。
- 出力は常に整数です。
- A と B は常に同じビット数です。
- 入力が有効でない場合があります（None）。
- ソリューションはメモリに収まる必要があります。

## 例の使用法

この関数の使い方の例をいくつか示します。

- A または B が None の場合、例外を発生させる必要があります。
- A = 11101 で B = 01111 の場合、反転させるビット数は 2 です。
