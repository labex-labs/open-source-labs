# Einführung

In der Informatik ist die Bitmanipulation die algorithmische Manipulation von Bits oder anderen Datenfragmenten, die kürzer als ein Wort sind. Die Bitmanipulation wird in der Kryptographie, in Fehlertoleranz- und Fehlerkorrekturalgorithmen, in der digitalen Signalverarbeitung und vielen anderen Bereichen eingesetzt. In dieser Herausforderung werden wir die Anzahl der Bits bestimmen, die umgewandelt werden müssen, um int a in int b umzuwandeln.
