# Bits zum Umkehren

## Problem

Gegeben zwei Integer A und B, bestimmen Sie die Anzahl der Bits, die umgewandelt werden müssen, um A in B umzuwandeln. Beispielsweise ist, wenn A = 11101 und B = 01111, die Anzahl der Bits, die umgewandelt werden müssen, 2.

## Anforderungen

Um dieses Problem zu lösen, müssen wir die folgenden Anforderungen berücksichtigen:

- A und B sind immer Integer.
- Die Ausgabe ist immer eine Ganzzahl.
- A und B haben immer die gleiche Anzahl von Bits.
- Die Eingaben können ungültig sein (None).
- Die Lösung sollte in den Speicher passen.

## Beispielverwendung

Hier sind einige Beispiele, wie diese Funktion verwendet werden kann:

- Wenn A oder B None ist, sollte eine Ausnahme ausgelöst werden.
- Wenn A = 11101 und B = 01111, ist die Anzahl der Bits, die umgewandelt werden müssen, 2.
