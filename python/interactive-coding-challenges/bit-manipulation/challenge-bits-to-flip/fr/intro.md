# Introduction

En informatique, la manipulation de bits est l'acte de manipuler algorithmiquement des bits ou d'autres morceaux de données plus courts qu'un mot. La manipulation de bits est utilisée en cryptographie, dans les algorithmes de détection et de correction d'erreurs, le traitement du signal numérique et de nombreux autres domaines. Dans ce défi, nous allons déterminer le nombre de bits à inverser pour convertir l'entier a en l'entier b.
