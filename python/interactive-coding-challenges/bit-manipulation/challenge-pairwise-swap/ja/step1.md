# ペアワイズ交換

## 問題

正の整数が与えられたとき、可能な限り少ない演算で奇数ビットと偶数ビットを交換します。たとえば、入力が `1001 1111 0110` の場合、出力は `0110 1111 1001` でなければなりません。

## 要件

次の要件を満たす必要があります。

- 入力は常に正の整数です。
- プログラムは32ビットで動作します。
- 出力は整数です。
- 入力は有効です (Noneではありません)。
- プログラムはメモリに収まります。

## 例の使用法

次の例は、プログラムの使用法を示しています。

- なし -> 例外
- 0 -> 0
- -1 -> -1
- 一般的なケース:

```
    入力  = 1001 1111 0110
    結果 = 0110 1111 1001
```
