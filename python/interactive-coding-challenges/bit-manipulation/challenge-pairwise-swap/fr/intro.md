# Introduction

En informatique, les opérations bit-à-bit sont utilisées pour manipuler les bits individuels d'un nombre binaire. Un problème courant est de permuter les bits pairs et impairs d'un entier positif. Cela peut être réalisé avec le moins d'opérations possible en utilisant des opérateurs bit-à-bit.
