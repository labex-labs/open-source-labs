# Intercambio Par a Par

## Problema

Dado un número entero positivo, intercambie los bits impares y pares con el menor número posible de operaciones. Por ejemplo, si la entrada es `1001 1111 0110`, la salida debe ser `0110 1111 1001`.

## Requisitos

Los siguientes requisitos deben cumplirse:

- La entrada siempre es un número entero positivo.
- El programa funciona con 32 bits.
- La salida es un número entero.
- Las entradas son válidas (no son None).
- El programa ajusta en memoria.

## Uso de Ejemplo

Los siguientes ejemplos ilustran el uso del programa:

- None -> Excepción
- 0 -> 0
- -1 -> -1
- Caso general:

```
    entrada  = 1001 1111 0110
    resultado = 0110 1111 1001
```
