# Перестановка пар

## Задача

Дано положительное целое число. Переставьте нечетные и четные биты с использованием минимального количества операций. Например, если входное значение равно `1001 1111 0110`, то выходом должно быть `0110 1111 1001`.

## Требования

Следующие требования должны быть выполнены:

- Входное значение всегда является положительным целым числом.
- Программа работает с 32-разрядными числами.
- Выходом является целое число.
- Входные данные действительны (не равны None).
- Программа помещается в память.

## Пример использования

Следующие примеры иллюстрируют использование программы:

- None -> Исключение
- 0 -> 0
- -1 -> -1
- Общий случай:

```
    вход   = 1001 1111 0110
    результат = 0110 1111 1001
```
