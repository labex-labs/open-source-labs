# Introduction

En mathématiques, un nombre premier est un nombre naturel supérieur à 1 qui n'est pas le produit de deux nombres naturels plus petits. En d'autres termes, un nombre premier est un entier positif qui n'a pas de diviseurs positifs autres que 1 et lui-même. Dans ce défi Python, nous allons écrire un programme pour vérifier si un nombre donné est premier ou non.
