# 简介

在数学中，质数是大于 1 的自然数，且不是两个较小自然数的乘积。换句话说，质数是一个正整数，除了 1 和它本身之外没有其他正整数因数。在这个 Python 挑战中，我们将编写一个程序来检查给定的数是否为质数。
