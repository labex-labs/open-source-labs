# Математические операции

## Задача

Создайте класс на Python с методом insert, который может вставлять целое число в список. Класс также должен поддерживать вычисление максимума, минимума, среднего значения и моды списка с временной сложностью O(1). Класс должен обрабатывать следующие сценарии:

- Если входные данные недействительны, должно быть возбуждено исключение TypeError.
- Если список пуст, должно быть возбуждено исключение ValueError.
- Если есть несколько мод, можно вернуть любую из них.

## Требования

Для решения вышеуказанной задачи необходимо следовать следующим требованиям:

- Входные данные могут быть недействительными, поэтому нельзя предполагать, что они действительны.
- Диапазон входных данных находится между 0 и 100 включительно.
- Среднее значение должно быть возвращено в виде числа с плавающей точкой.
- Другие результаты должны быть возвращены в виде целого числа.
- Если есть несколько мод, класс может вернуть любую из них.
- Можно предположить, что список помещается в память.

## Пример использования

Ниже приведены примеры использования класса:

- None -> TypeError
- [] -> ValueError
- [5, 2, 7, 9, 9, 2, 9, 4, 3, 3, 2]
  - max: 9
  - min: 2
  - mean: 4.909090909090909
  - mode: 9 или 2
