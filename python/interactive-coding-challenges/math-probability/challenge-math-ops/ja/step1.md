# 数学演算

## 問題

整数をリストに挿入できる insert メソッドを持つ Python クラスを作成します。このクラスは、O(1) の時間計算量でリストの最大値、最小値、平均値、最頻値を計算する機能もサポートする必要があります。このクラスは以下のシナリオを処理する必要があります。

- 入力が無効な場合、TypeError を発生させる。
- リストが空の場合、ValueError を発生させる。
- 最頻値が複数ある場合、どれか 1 つの最頻値を返すことができる。

## 要件

上記の問題を達成するには、以下の要件を満たす必要があります。

- 入力が無効な場合があるため、入力が有効であることを前提としてはならない。
- 入力の範囲は 0 以上 100 以下である。
- 平均値は浮動小数点数で返す。
- その他の結果は整数で返す。
- 最頻値が複数ある場合、クラスはどれか 1 つの最頻値を返すことができる。
- リストはメモリに収まるものと仮定できる。

## 使い方の例

このクラスの使い方の例を以下に示します。

- なし -> TypeError
- [] -> ValueError
- [5, 2, 7, 9, 9, 2, 9, 4, 3, 3, 2]
  - 最大値：9
  - 最小値：2
  - 平均値：4.909090909090909
  - 最頻値：9 または 2
