# Вычесть два

## Задача

Напишите функцию на Python, которая принимает два целых числа в качестве входных данных и возвращает их разность без использования знака '+' или '-'. Функция должна обрабатывать следующие случаи:

- Если любой из входных данных равен None, функция должна вызывать исключение TypeError.
- Функция должна работать как с положительными, так и с отрицательными целыми числами.

## Требования

Для решения этой задачи необходимо выполнить следующие требования:

- Проверить на None входные данные и вызвать исключение TypeError, если необходимо.
- Можно предположить, что входные данные помещаются в память.

## Пример использования

Вот несколько примеров того, как должна работать функция:

```
sub_two(None, 5) -> TypeError
sub_two(7, 5) -> 2
sub_two(-5, -7) -> 2
sub_two(-5, 7) -> -12
sub_two(5, -7) -> 12
```
