# Introduction

En Python, nous pouvons facilement soustraire deux entiers en utilisant l'opérateur '-'. Cependant, que se passe-t-il si nous ne sommes pas autorisés à utiliser cet opérateur? Dans ce défi, nous allons explorer comment trouver la différence entre deux entiers sans utiliser le signe '+' ou '-'.
