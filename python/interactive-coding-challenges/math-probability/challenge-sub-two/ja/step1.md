# 2 つの数を引く

## 問題

'+' または '-' 記号を使わずに 2 つの整数を入力として受け取り、それらの差を返す Python 関数を書きましょう。この関数は以下のケースに対応する必要があります。

- どちらかの入力が None の場合、関数は TypeError を発生させる必要があります。
- 関数は正の整数と負の整数の両方に対して機能する必要があります。

## 要件

この問題を解くには、以下の要件を守る必要があります。

- None の入力をチェックし、必要に応じて TypeError を発生させます。
- 入力がメモリに収まることを想定して構いません。

## 使い方の例

ここに関数がどのように動作するかのいくつかの例を示します。

```
sub_two(None, 5) -> TypeError
sub_two(7, 5) -> 2
sub_two(-5, -7) -> 2
sub_two(-5, 7) -> -12
sub_two(5, -7) -> 12
```
