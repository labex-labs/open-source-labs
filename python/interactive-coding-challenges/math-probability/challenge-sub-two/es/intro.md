# Introducción

En Python, podemos restar dos enteros fácilmente utilizando el operador '-'. Sin embargo, ¿qué pasa si no se nos permite utilizar este operador? En este desafío, exploraremos cómo encontrar la diferencia entre dos enteros sin utilizar el signo '+' o '-'.
