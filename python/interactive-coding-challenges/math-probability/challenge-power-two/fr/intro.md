# Introduction

En informatique, il est souvent nécessaire de déterminer si un nombre donné est une puissance de deux. Cela peut être utile dans diverses applications, telles que l'optimisation d'algorithmes ou la vérification d'erreurs dans la transmission de données. Dans ce défi, nous allons écrire une fonction Python pour déterminer si un nombre est une puissance de deux.
