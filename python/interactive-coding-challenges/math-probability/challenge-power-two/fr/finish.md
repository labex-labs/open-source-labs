# Sommaire

Dans ce défi, nous avons écrit une fonction Python pour déterminer si un nombre donné est une puissance de deux. Nous avons appris qu'une puissance de deux est tout nombre qui peut s'écrire sous la forme 2^n, où n est un entier. Nous avons également appris à gérer les entrées invalides et à nous assurer que notre fonction répond aux contraintes de mémoire.
