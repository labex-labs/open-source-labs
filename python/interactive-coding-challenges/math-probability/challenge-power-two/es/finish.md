# Resumen

En este desafío, escribimos una función de Python para determinar si un número dado es una potencia de dos. Aprendimos que una potencia de dos es cualquier número que se puede expresar como 2^n, donde n es un entero. También aprendimos cómo manejar entradas no válidas y asegurarnos de que nuestra función cumpla con las restricciones de memoria.
