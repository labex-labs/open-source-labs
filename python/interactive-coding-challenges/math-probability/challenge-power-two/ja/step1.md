# 2 の累乗

## 問題

整数をパラメータとして受け取り、入力が 2 の累乗であれば`True`を返し、そうでなければ`False`を返す Python 関数`is_power_of_two`を書きます。2 の累乗とは、n が整数であるとき 2^n として表現できる任意の数です。たとえば、2、4、8、16 はすべて 2 の累乗です。

## 要件

`is_power_of_two`関数は以下の要件を満たさなければなりません。

- 入力数は整数でなければなりません。
- 関数は無効な入力を上手に処理しなければなりません。
- 出力はブール値でなければなりません。
- 関数はメモリ制約内に収まらなければなりません。

## 使い方の例

`is_power_of_two`関数の動作の例を以下に示します。

- `is_power_of_two(None)`は`TypeError`を発生させる必要があります。
- `is_power_of_two(0)`は`False`を返す必要があります。
- `is_power_of_two(1)`は`True`を返す必要があります。
- `is_power_of_two(2)`は`True`を返す必要があります。
- `is_power_of_two(15)`は`False`を返す必要があります。
- `is_power_of_two(16)`は`True`を返す必要があります。
