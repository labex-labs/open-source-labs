# Резюме

В этом испытании мы написали функцию на Python для определения, является ли заданное число степенью двойки. Мы узнали, что степенью двойки является любое число, которое может быть выражено в виде 2^n, где n - целое число. Мы также узнали, как обрабатывать недействительные входные значения и убедиться, что наша функция соответствует ограничениям по памяти.
