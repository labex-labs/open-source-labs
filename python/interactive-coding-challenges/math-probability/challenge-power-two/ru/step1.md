# Степень двойки

## Задача

Напишите функцию на Python под названием `is_power_of_two`, которая принимает целое число в качестве параметра и возвращает `True`, если входное значение является степенью двойки, и `False` в противном случае. Степенью двойки является любое число, которое может быть выражено в виде 2^n, где n - целое число. Например, 2, 4, 8 и 16 - все степени двойки.

## Требования

Функция `is_power_of_two` должна соответствовать следующим требованиям:

- Входное число должно быть целым числом.
- Функция должна优雅но обрабатывать недействительные входные значения.
- Выход должен быть булевым значением.
- Функция должна соответствовать ограничениям по памяти.

## Пример использования

Вот несколько примеров того, как должна работать функция `is_power_of_two`:

- `is_power_of_two(None)` должна вызывать `TypeError`.
- `is_power_of_two(0)` должна возвращать `False`.
- `is_power_of_two(1)` должна возвращать `True`.
- `is_power_of_two(2)` должна возвращать `True`.
- `is_power_of_two(15)` должна возвращать `False`.
- `is_power_of_two(16)` должна возвращать `True`.
