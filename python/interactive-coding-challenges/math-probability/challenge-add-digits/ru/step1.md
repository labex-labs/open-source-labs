# Сложение цифр

## Задача

Дано целое число, и нам нужно повторно суммировать его цифры, пока результат не будет однозначным числом. Например, если дано целое число 138, мы суммируем 1 + 3 + 8 = 12. Поскольку 12 не является однозначным числом, мы повторяем процесс и суммируем 1 + 2 = 3. Таким образом, конечный результат равен 3.

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Входное целое число неотрицательно.
- Входные данные могут не всегда быть валидными, поэтому мы должны обработать любые ошибки, которые могут возникнуть.
- Решение должно помещаться в память.

## Пример использования

Вот несколько примеров использования этой функции:

- Если мы передадим None в качестве входных данных, функция должна вызвать TypeError.
- Если мы передадим отрицательное целое число в качестве входных данных, функция должна вызвать ValueError.
- Если мы передадим 9 в качестве входных данных, функция должна вернуть 9, так как это уже однозначное число.
- Если мы передадим 138 в качестве входных данных, функция должна вернуть 3.
- Если мы передадим 65536 в качестве входных данных, функция должна вернуть 7.
