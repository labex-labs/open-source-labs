# 桁の和を求める

## 問題

与えられた整数に対して、桁の和を繰り返し求め、結果が 1 桁になるまで続けます。たとえば、整数 138 が与えられた場合、1 + 3 + 8 = 12 となります。12 は 1 桁ではないので、このプロセスを繰り返して 1 + 2 = 3 とします。したがって、最終的な結果は 3 になります。

## 要件

この問題を解くには、以下の要件を考慮する必要があります。

- 入力される整数は負ではありません。
- 入力が常に有効であるとは限らないので、発生する可能性のあるエラーを処理する必要があります。
- ソリューションはメモリに収まる必要があります。

## 使い方の例

この関数の使い方の例をいくつか示します。

- 入力として None を渡すと、関数は TypeError を発生させる必要があります。
- 入力として負の整数を渡すと、関数は ValueError を発生させる必要があります。
- 入力として 9 を渡すと、関数は既に 1 桁なので 9 を返す必要があります。
- 入力として 138 を渡すと、関数は 3 を返す必要があります。
- 入力として 65536 を渡すと、関数は 7 を返す必要があります。
