# 桁の和を求める

## 問題

与えられた整数に対して、桁の和を繰り返し求め、結果が1桁になるまで続けます。たとえば、整数138が与えられた場合、1 + 3 + 8 = 12 となります。12は1桁ではないので、このプロセスを繰り返して1 + 2 = 3 とします。したがって、最終的な結果は3になります。

## 要件

この問題を解くには、以下の要件を考慮する必要があります。

- 入力される整数は負ではありません。
- 入力が常に有効であるとは限らないので、発生する可能性のあるエラーを処理する必要があります。
- ソリューションはメモリに収まる必要があります。

## 使い方の例

この関数の使い方の例をいくつか示します。

- 入力としてNoneを渡すと、関数はTypeErrorを発生させる必要があります。
- 入力として負の整数を渡すと、関数はValueErrorを発生させる必要があります。
- 入力として9を渡すと、関数は既に1桁なので9を返す必要があります。
- 入力として138を渡すと、関数は3を返す必要があります。
- 入力として65536を渡すと、関数は7を返す必要があります。
