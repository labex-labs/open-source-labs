# はじめに

数学において、素数とは 1 より大きな自然数であって、1 とそれ自身以外の任意の数で割り切れない数のことです。素数のリストを生成することは、コンピュータサイエンスにおける一般的な問題であり、暗号学、数論、その他の分野において様々な応用があります。
