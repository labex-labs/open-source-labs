# Введение

В математике простые числа - это натуральные числа, большие 1, которые не делятся на другие числа, кроме 1 и самого себя. Генерация списка простых чисел - это распространённая задача в компьютерной науке и имеет различные приложения в криптографии, теории чисел и других областях.
