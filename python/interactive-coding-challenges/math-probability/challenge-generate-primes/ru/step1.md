# Генерация простых чисел

## Задача

Напишите функцию на Python, которая генерирует список простых чисел. Функция должна принимать целое число в качестве входных данных и возвращать список булевых значений, где каждое значение соответствует тому, является ли индекс простым числом или нет. Например, если входным значением является 20, то выходным значением должно быть [False, False, True, True, False, True, False, True, False, False, False, True, False, True, False, False, False, True, False, True], где значение с индексом 2 равно True, так как 2 - простое число, а значение с индексом 4 равно False, так как 4 - не простое число.

## Требования

- Функция не должна рассматривать 1 как простое число.
- Функция должна обрабатывать недопустимые входные данные, поднимая исключение.
- Функция должна генерировать список простых чисел в памяти.

## Пример использования

- Ничего -> Исключение
- Не целое число -> Исключение
- 20 -> [False, False, True, True, False, True, False, True, False, False, False, True, False, True, False, False, False, True, False, True]
