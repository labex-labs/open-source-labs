# 排列

## 问题

给定两个字符串，我们需要判断其中一个字符串是否是另一个字符串的排列。排列被定义为字符串中字符的重新排列。例如，“act”是“cat”的一个排列。

## 要求

为了解决这个问题，我们需要考虑以下要求：

- 字符串是 ASCII 编码。
- 空格很重要。
- 比较是区分大小写的。例如，“Nib”和“bin”不匹配。
- 我们可以使用额外的数据结构。
- 我们可以假设字符串能装入内存。

## 示例用法

以下是这个函数的一些使用示例：

- 一个或多个 None 输入 -> 假
- 一个或多个空字符串 -> 假
- “Nib”，“bin” -> 假
- “act”，“cat” -> 真
- “a ct”，“ca t” -> 真
