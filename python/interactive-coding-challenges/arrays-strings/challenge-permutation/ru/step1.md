# Перестановка

## Задача

Даны две строки, и нам нужно определить, является ли одна строка перестановкой другой. Перестановкой определяется переупорядочение символов в строке. Например, "act" - это перестановка "cat".

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Строка состоит из ASCII-символов.
- Пробелы имеют значение.
- Сравнение чувствительно к регистру. Например, "Nib" и "bin" не совпадают.
- Можно использовать дополнительные структуры данных.
- Можно предположить, что строки помещаются в память.

## Пример использования

Вот несколько примеров использования этой функции:

- Один или несколько входных значений None -> False
- Один или несколько пустых строк -> False
- 'Nib', 'bin' -> False
- 'act', 'cat' -> True
- 'a ct', 'ca t' -> True
