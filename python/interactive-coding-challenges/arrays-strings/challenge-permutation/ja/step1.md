# 順列

## 問題

2つの文字列が与えられたとき、一方の文字列がもう一方の文字列の順列であるかどうかを判断する必要があります。順列とは、文字列内の文字の並び替えのことを指します。たとえば、「act」は「cat」の順列です。

## 要件

この問題を解くために、以下の要件を考慮する必要があります。

- 文字列はASCIIです。
- 空白文字も重要です。
- 比較は大文字小文字を区別します。たとえば、「Nib」と「bin」は一致しません。
- 追加のデータ構造を使用できます。
- 文字列はメモリに収まると仮定できます。

## 使い方の例

この関数の使い方の例をいくつか示します。

- 1つ以上のNone入力 -> False
- 1つ以上の空文字列 -> False
- 'Nib', 'bin' -> False
- 'act', 'cat' -> True
- 'a ct', 'ca t' -> True
