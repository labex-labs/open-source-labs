# Einführung

In dieser Herausforderung werden wir einen String auf eine bestimmte Weise komprimieren, um Speicherplatz zu sparen. Die komprimierte Zeichenkette wird nur dann eine kürzere Länge als die ursprüngliche Zeichenkette haben, wenn dadurch Speicherplatz eingespart wird.
