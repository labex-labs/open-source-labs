# Resumen

En resumen, este desafío nos pide comprimir una cadena de texto de una manera específica para ahorrar espacio. Debemos asegurarnos de que la cadena comprimida sea más corta que la cadena original, de lo contrario devolvemos la cadena original. Los requisitos para este desafío incluyen utilizar cadenas ASCII, ser sensible a mayúsculas y minúsculas, permitir estructuras de datos adicionales y asumir que la cadena cabe en memoria.
