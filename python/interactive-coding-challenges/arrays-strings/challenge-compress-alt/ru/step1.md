# Сжатие Альтернативным способом

## Задача

Дана строка, сжать ее таким образом, чтобы последовательные вхождения одного и того же символа заменялись этим символом, за которым следует количество вхождений. Например, строка 'AAABCCDDDD' станет 'A3BCCD4'. Однако, если сжатая строка не короче исходной строки, вернуть исходную строку.

## Требования

Для решения этого испытания должны быть выполнены следующие требования:

- Предполагается, что строка состоит из ASCII-символов.
- Сжатие чувствительно к регистру.
- Можно использовать дополнительные структуры данных.
- Предполагается, что строка помещается в память.

## Пример использования

Ниже приведены примеры использования этой функции:

- `compress(None)` возвращает `None`
- `compress('')` возвращает `''`
- `compress('AABBCC')` возвращает `'AABBCC'`
- `compress('AAABCCDDDD')` возвращает `'A3BCCD4'`
