# 圧縮 Alt

## 問題

与えられた文字列を、同じ文字の連続する出現をその文字に続けて出現回数で置き換えることで圧縮します。たとえば、文字列 'AAABCCDDDD' は 'A3BCCD4' になります。ただし、圧縮された文字列が元の文字列より短くない場合は、元の文字列を返します。

## 要件

このチャレンジを解くには、次の要件を満たす必要があります。

- 文字列は ASCII と仮定されます。
- 圧縮は大文字小文字を区別します。
- 追加のデータ構造を使用できます。
- 文字列はメモリに収まると仮定されます。

## 例の使用法

この関数を使用する方法の例を次に示します。

- `compress(None)` は `None` を返します
- `compress('')` は `''` を返します
- `compress('AABBCC')` は `'AABBCC'` を返します
- `compress('AAABCCDDDD')` は `'A3BCCD4'` を返します
