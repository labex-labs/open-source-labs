# 另类压缩

## 问题

给定一个字符串，对其进行压缩，使得连续出现的相同字符被替换为该字符加上其出现的次数。例如，字符串“AAABCCDDDD”将变为“A3BCCD4”。但是，如果压缩后的字符串不比原始字符串短，则返回原始字符串。

## 要求

要解决此挑战，必须满足以下要求：

- 假设字符串为 ASCII 编码。
- 压缩区分大小写。
- 可以使用额外的数据结构。
- 假设字符串能装入内存。

## 示例用法

以下是该函数的使用示例：

- `compress(None)` 返回 `None`
- `compress('')` 返回 `''`
- `compress('AABBCC')` 返回 `'AABBCC'`
- `compress('AAABCCDDDD')` 返回 `'A3BCCD4'`
