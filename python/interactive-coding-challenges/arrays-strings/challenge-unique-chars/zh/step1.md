# 唯一字符

## 问题

给定一个字符串，任务是确定它是否包含所有唯一的字符。换句话说，字符串中不应有重复的字符。例如，字符串“hello”不包含所有唯一的字符，因为字母“l”出现了两次。另一方面，字符串“world”包含所有唯一的字符，因为每个字母只出现一次。

## 要求

要解决此问题，必须满足以下要求：

- 假设字符串为 ASCII 编码。
  - 根据所使用的语言，Unicode 字符串可能需要特殊处理。
- 假设大小写敏感。
- 可以使用额外的数据结构。
- 假设字符串适合内存。

## 示例用法

以下是该函数应有的行为示例：

- None -> False
- '' -> True
- 'foo' -> False
- 'bar' -> True
