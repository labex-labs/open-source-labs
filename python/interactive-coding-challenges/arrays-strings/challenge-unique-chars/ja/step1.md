# 一意の文字

## 問題

文字列が与えられたとき、それがすべて一意の文字を含んでいるかどうかを判断することが課題です。言い換えると、文字列には重複する文字がない必要があります。たとえば、文字列 "hello" はすべて一意の文字を持っていません。なぜなら、文字 "l" が2回登場するからです。一方、文字列 "world" はすべて一意の文字を持っています。なぜなら、各文字が1回だけ登場するからです。

## 要件

この問題を解くには、以下の要件を満たす必要があります。

- 文字列はASCIIと仮定されます。
  - Unicode文字列は使用言語に応じて特別な処理が必要になる場合があります。
- 大文字と小文字は区別されます。
- 追加のデータ構造を使用できます。
- 文字列はメモリに収まると仮定されます。

## 例の使用法

以下は、関数がどのように動作するかの例です。

- None -> False
- '' -> True
- 'foo' -> False
- 'bar' -> True
