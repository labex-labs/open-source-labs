# Уникальные символы

## Задача

Дана строка, и задача заключается в том, чтобы определить, содержит ли она все уникальные символы. Другими словами, в строке не должно быть повторяющихся символов. Например, строка "hello" не содержит все уникальные символы, потому что буква "l" встречается дважды. С другой стороны, строка "world" содержит все уникальные символы, так как каждая буква встречается только один раз.

## Требования

Для решения этой задачи должны быть выполнены следующие требования:

- Предполагается, что строка - это ASCII.
  - Строки Unicode могут требовать специальной обработки в зависимости от используемого языка.
- Предполагается, что регистр символов важен.
- Можно использовать дополнительные структуры данных.
- Предполагается, что строка помещается в память.

## Пример использования

Ниже приведены примеры того, как должна работать функция:

- None -> False
- '' -> True
- 'foo' -> False
- 'bar' -> True
