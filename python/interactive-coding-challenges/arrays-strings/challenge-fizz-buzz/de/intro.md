# Einführung

Fizz Buzz ist eine beliebte Programmieraufgabe, die oft bei Coding-Interviews eingesetzt wird. Es ist ein einfies Spiel, bei dem die Spieler nacheinander von 1 an zählen und jede Zahl, die durch 3 teilbar ist, durch das Wort "Fizz" und jede Zahl, die durch 5 teilbar ist, durch das Wort "Buzz" ersetzen. Wenn eine Zahl sowohl durch 3 als auch durch 5 teilbar ist, muss der Spieler "FizzBuzz" anstelle der Zahl sagen. In dieser Aufgabe werden wir Fizz Buzz mit Python implementieren.
