# はじめに

Fizz Buzz は、コーディング面接で頻繁に使用される人気のあるプログラミング チャレンジです。これは、1 から順番に数え上げるシンプルなゲームで、3 で割り切れる数を「Fizz」と言い換え、5 で割り切れる数を「Buzz」と言い換えます。3 と 5 の両方で割り切れる数の場合は、その数の代わりに「FizzBuzz」と言わなければなりません。このチャレンジでは、Python を使って Fizz Buzz を実装します。
