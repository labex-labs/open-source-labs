# Введение

Fizz Buzz - это популярная задача программирования, которая часто используется на собеседованиях по программированию. Это простая игра, в которой игроки по очереди должны считать от 1, заменяя любое число, которое делится на 3, на слово "Fizz", и любое число, которое делится на 5, на слово "Buzz". Если число делится на 3 и на 5 одновременно, игрок должен сказать "FizzBuzz" вместо числа. В этом задании мы будем реализовывать Fizz Buzz с использованием Python.
