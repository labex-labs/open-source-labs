# Fizz Buzz

## Задача

Реализуйте Fizz Buzz с использованием Python. Ваш функция должна принимать целое число n в качестве входных данных и возвращать список строк, представляющих числа от 1 до n, с следующими модификациями:

- Числа, кратные 3, должны быть заменены на строку "Fizz"
- Числа, кратные 5, должны быть заменены на строку "Buzz"
- Числа, кратные и 3, и 5, должны быть заменены на строку "FizzBuzz"

Ваша функция также должна обрабатывать следующие случаи:

- Если входное значение меньше 1, выбросить исключение
- Если входное значение не является допустимым целым числом, выбросить исключение

## Требования

Для реализации Fizz Buzz в Python нам нужно следовать следующим требованиям:

- Определить функцию, которая принимает целое число n в качестве входных данных
- Проверить, является ли входное значение допустимым целым числом, и выбросить исключение, если нет
- Проверить, меньше ли входное значение 1, и выбросить исключение, если да
- Создать список строк, представляющих числа от 1 до n, с описанными выше модификациями
- Вернуть список

## Пример использования

```python
assert fizz_buzz(15) == ['1', '2', 'Fizz', '4', 'Buzz', 'Fizz', '7', '8', 'Fizz', 'Buzz', '11', 'Fizz', '13', '14', 'FizzBuzz']
```
