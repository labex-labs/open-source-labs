# Резюме

В этом задании мы реализовали Fizz Buzz с использованием Python. Мы определили функцию, которая принимает целое число n в качестве входных данных и возвращает список строк, представляющих числа от 1 до n, с модификациями для кратных 3, 5 и обоим сразу. Мы также обработали случаи, когда входное значение не является допустимым целым числом или меньше 1, выбросив исключения.
