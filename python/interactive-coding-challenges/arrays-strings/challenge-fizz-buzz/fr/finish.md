# Sommaire

Dans ce défi, nous avons implémenté Fizz Buzz en utilisant Python. Nous avons défini une fonction qui prend un entier `n` en entrée et renvoie une liste de chaînes de caractères représentant les nombres de 1 à `n`, avec des modifications pour les multiples de 3, 5 et les deux. Nous avons également géré les cas où l'entrée n'est pas un entier valide ou est inférieure à 1 en levant des exceptions.
