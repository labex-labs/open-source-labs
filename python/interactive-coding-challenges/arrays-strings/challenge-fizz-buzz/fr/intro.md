# Introduction

Fizz Buzz est un défi de programmation populaire qui est souvent utilisé dans les entretiens de codage. C'est un jeu simple qui demande aux joueurs de compter à tour de rôle à partir de 1, en remplaçant tout nombre divisible par 3 par le mot "Fizz" et tout nombre divisible par 5 par le mot "Buzz". Si un nombre est divisible à la fois par 3 et 5, le joueur doit dire "FizzBuzz" au lieu du nombre. Dans ce défi, nous allons implémenter Fizz Buzz en utilisant Python.
