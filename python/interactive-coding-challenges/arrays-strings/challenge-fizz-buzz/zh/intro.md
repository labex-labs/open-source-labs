# 简介

Fizz Buzz 是一个广为人知的编程挑战，常用于编码面试。这是一个简单的游戏，要求玩家轮流从 1 开始计数，将任何能被 3 整除的数字替换为单词“Fizz”，将任何能被 5 整除的数字替换为单词“Buzz”。如果一个数字能同时被 3 和 5 整除，玩家必须说“FizzBuzz”而不是该数字。在这个挑战中，我们将使用 Python 实现 Fizz Buzz。
