# 简介

在计算机科学中，字符串旋转是将字符串中的字符向左或向右移动的过程。一个字符串相对于另一个字符串的旋转是指将一个字符串的字符以循环方式移动。例如，字符串“abcde”可以旋转形成字符串“cdeab”。在这个挑战中，我们要确定字符串 s1 是否是另一个字符串 s2 的旋转。
