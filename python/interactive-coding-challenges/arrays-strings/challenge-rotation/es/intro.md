# Introducción

En informática, la rotación de cadenas es un proceso de mover los caracteres de una cadena hacia la izquierda o hacia la derecha. Una rotación de una cadena con respecto a otra cadena es un proceso de tomar una cadena y mover sus caracteres de manera circular. Por ejemplo, la cadena "abcde" se puede rotar para formar la cadena "cdeab". En este desafío, vamos a determinar si una cadena s1 es una rotación de otra cadena s2.
