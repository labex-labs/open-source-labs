# 回転

## 問題

2 つの文字列 s1 と s2 が与えられたとき、関数 is_substring を（ただ 1 回だけ）呼び出すことで、s1 が s2 の回転であるかどうかを判定します。関数 is_substring は 2 つの文字列を入力として受け取り、最初の文字列が 2 番目の文字列の部分文字列である場合は True を返し、そうでない場合は False を返します。

## 要件

この問題を解くには、以下の要件を満たす必要があります。

- 文字列は ASCII です。
- 比較は大文字小文字を区別します。
- 追加のデータ構造を使用できます。
- 文字列はメモリに収まります。

## 使い方の例

関数がどのように動作するかのいくつかの例を以下に示します。

- 文字列のサイズが異なる場合、関数は False を返す必要があります。
- どちらかの文字列が None の場合、関数は False を返す必要があります。
- 一方の文字列が空白で、もう一方が空白でない場合、関数は False を返す必要があります。
- 両方の文字列が空白の場合、関数は True を返す必要があります。
- s1 が s2 の回転である場合、関数は True を返す必要があります。
