# はじめに

コンピュータサイエンスにおいて、文字列の回転とは、文字列内の文字を左または右に移動させるプロセスです。ある文字列の回転とは、その文字列をとって、その文字を円環状に移動させるプロセスです。たとえば、文字列 "abcde" を回転させると、文字列 "cdeab" が生成されます。このチャレンジでは、文字列 s1 が別の文字列 s2 の回転であるかどうかを判定します。

これは Challenge です。Guided Lab とは異なり、学習のためにラボの手順に従うのではなく、チャレンジタスクを独立して完了する必要があります。Challenge は通常、少し難しい課題です。難しいと感じた場合は、Labby と相談したり、解決策を確認したりできます。
