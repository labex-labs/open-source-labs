# Вращение

## Задача

Даны две строки s1 и s2. Определите, является ли s1 вращением s2, вызвав (только один раз) функцию is_substring. Функция is_substring принимает на вход две строки и возвращает True, если первая строка является подстрокой второй строки, и False в противном случае.

## Требования

Для решения этой задачи необходимо удовлетворить следующим требованиям:

- Строка состоит из ASCII-символов.
- Сравнение чувствительно к регистру.
- Можно использовать дополнительные структуры данных.
- Строки помещаются в память.

## Пример использования

Ниже приведены примеры того, как должна работать функция:

- Если строки имеют разный размер, функция должна вернуть False.
- Если любая из строк равна None, функция должна вернуть False.
- Если одна из строк - это пробел, а другая нет, функция должна вернуть False.
- Если обе строки - это пробелы, функция должна вернуть True.
- Если s1 является вращением s2, функция должна вернуть True.
