# Введение

В информатике вращение строки - это процесс перемещения символов в строке влево или вправо. Вращение одной строки вокруг другой - это процесс, при котором символы строки перемещаются по кругу. Например, строку "abcde" можно повернуть, чтобы получить строку "cdeab". В этом испытании мы должны определить, является ли строка s1 вращением другой строки s2.
