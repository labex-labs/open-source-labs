# Einführung

In der Informatik ist die Zeichenfolge-Drehung ein Prozess, bei dem die Zeichen in einer Zeichenfolge nach links oder rechts verschoben werden. Eine Zeichenfolge-Drehung einer anderen Zeichenfolge ist ein Prozess, bei dem man eine Zeichenfolge nimmt und ihre Zeichen in einem kreisförmigen Verfahren bewegt. Beispielsweise kann die Zeichenfolge "abcde" gedreht werden, um die Zeichenfolge "cdeab" zu bilden. In dieser Herausforderung werden wir bestimmen, ob eine Zeichenfolge s1 eine Drehung einer anderen Zeichenfolge s2 ist.
