# Zusammenfassung

In dieser Herausforderung haben wir gelernt, wie man bestimmt, ob eine Zeichenfolge eine Drehung einer anderen Zeichenfolge ist, indem man eine Funktion aufruft, die überprüft, ob eine Zeichenfolge ein Teilstring einer anderen Zeichenfolge ist. Wir haben auch die Anforderungen für die Lösung dieses Problems festgelegt und Beispiele dafür gegeben, wie die Funktion verhalten sollte.
