# Introduction

En informatique, la rotation de chaîne est un processus consistant à déplacer les caractères d'une chaîne vers la gauche ou la droite. Une rotation d'une chaîne par rapport à une autre chaîne est un processus consistant à prendre une chaîne et à déplacer ses caractères de manière circulaire. Par exemple, la chaîne "abcde" peut être tournée pour former la chaîne "cdeab". Dans ce défi, nous allons déterminer si une chaîne s1 est une rotation d'une autre chaîne s2.
