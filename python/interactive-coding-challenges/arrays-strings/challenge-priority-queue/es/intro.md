# Introducción

Una cola de prioridad es una estructura de datos que permite un acceso eficiente al elemento mínimo (o máximo) de una colección. Se utiliza comúnmente en algoritmos como el algoritmo de búsqueda de caminos más cortos de Dijkstra y la codificación Huffman. En este desafío, implementaremos una cola de prioridad respaldada por una matriz.
