# 简介

优先队列是一种数据结构，它允许高效地访问集合中的最小（或最大）元素。它通常用于诸如迪杰斯特拉最短路径算法和哈夫曼编码等算法中。在这个挑战中，我们将实现一个由数组支持的优先队列。
