# Einführung

Eine Prioritätswarteschlange ist eine Datenstruktur, die effizienten Zugang zum kleinsten (oder größten) Element in einer Sammlung ermöglicht. Sie wird häufig in Algorithmen wie Dijkstras kürzester-Pfad-Algorithmus und Huffman-Codierung verwendet. In dieser Herausforderung werden wir eine Prioritätswarteschlange implementieren, die von einem Array unterstützt wird.
