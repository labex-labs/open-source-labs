# Zusammenfassung

In dieser Herausforderung haben wir eine Prioritätswarteschlange implementiert, die von einem Array unterstützt wird. Die Prioritätswarteschlange unterstützt die Methoden `insert`, `extract_min` und `decrease_key`. Wir haben sichergestellt, dass es keine doppelten Schlüssel in der Prioritätswarteschlange gibt und dass die Eingaben nicht validiert werden müssen. Die Prioritätswarteschlange passt in den Arbeitsspeicher.
