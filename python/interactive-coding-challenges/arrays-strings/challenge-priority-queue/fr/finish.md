# Sommaire

Dans ce défi, nous avons implémenté une file d'attente prioritaire supportée par un tableau. La file d'attente prioritaire prend en charge les méthodes `insert`, `extract_min` et `decrease_key`. Nous avons veillé à ce qu'il n'y ait pas de clés dupliquées dans la file d'attente prioritaire et que les entrées n'aient pas besoin d'être validées. La file d'attente prioritaire tient en mémoire.
