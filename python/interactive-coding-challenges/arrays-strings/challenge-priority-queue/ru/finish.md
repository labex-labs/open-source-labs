# Резюме

В этом испытании мы реализовали приоритетную очередь, основанную на массиве. Приоритетная очередь поддерживает методы `insert`, `extract_min` и `decrease_key`. Мы удостоверились, что в приоритетной очереди нет дублирующихся ключей, и входные данные не требуют валидации. Приоритетная очередь помещается в память.
