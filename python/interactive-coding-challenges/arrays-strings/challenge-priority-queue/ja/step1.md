# 優先度付きキュー

## 問題

配列によってサポートされる優先度付きキューを実装します。優先度付きキューは、次のメソッドをサポートする必要があります。

- `insert`：優先度付きキューに新しい要素を挿入する
- `extract_min`：優先度付きキューから最小要素を削除して返す
- `decrease_key`：優先度付きキュー内の指定された要素のキーを減らす

## 要件

優先度付きキューを実装するには、次の要件を満たす必要があります。

- 優先度付きキューがサポートするメソッドは、`insert`、`extract_min`、および`decrease_key`でなければなりません。
- 優先度付きキューには、重複するキーはありません。
- 入力の検証は不要です。
- 優先度付きキューはメモリに収まる必要があります。

## 例の使用法

以下は、優先度付きキューのメソッドの使用方法の例です。

### insert

- `insert`の一般的なケース：優先度付きキューに新しいノードを挿入する。

### extract_min

- 空のリストからの`extract_min`：None を返す。
- `extract_min`の一般的なケース：優先度付きキューから最小ノードを削除して返す。

### decrease_key

- 無効なキーの`decrease_key`：None を返す。
- `decrease_key`の一般的なケース：優先度付きキュー内の指定されたノードのキーを減らす。
