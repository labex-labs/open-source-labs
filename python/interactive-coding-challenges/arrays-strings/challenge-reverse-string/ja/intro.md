# はじめに

プログラミングにおいては、文字列の操作が頻繁に必要となります。そのような操作の一つとして、文字列を逆順にすることが挙げられます。これは、文字列内の文字の順序を逆にすることを意味します。たとえば、文字列 "hello" を逆順にすると "olleh" になります。このチャレンジでは、文字列をインプレースで逆順にする関数を実装します。
