# 文字列を逆順にする

## 問題

文字列を逆順にする関数（文字のリスト）をインプレースで実装します。これは、関数が新しい文字列を作成するのではなく、元の文字列を変更することを意味します。関数は文字のリストを入力として受け取り、文字が逆順になった同じリストを返す必要があります。

この問題を解くには、いくつかの要件を考慮する必要があります。

## 要件

- 文字列はASCIIと仮定できます。
- インプレースで行う必要があるため、スライス演算子やreversed関数を使用できません。
- Pythonの文字列は不変であるため、代わりに文字のリストを使用できます。

## 使い方の例

関数がどのように動作するかの例をいくつか示します。

- None -> None
- [''] -> ['']
- ['f', 'o', 'o', ' ', 'b', 'a', 'r'] -> ['r', 'a', 'b', ' ', 'o', 'o', 'f']
