# Introduction

En programmation, il est souvent nécessaire de manipuler des chaînes de caractères. Une tâche courante consiste à inverser une chaîne de caractères. Cela signifie que l'ordre des caractères dans la chaîne est inversé. Par exemple, la chaîne de caractères "hello" deviendra "olleh" lorsqu'elle est inversée. Dans ce défi, nous allons implémenter une fonction pour inverser une chaîne de caractères en place.
