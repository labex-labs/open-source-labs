# Einführung

Beim Programmieren ist es oft erforderlich, Zeichenketten zu manipulieren. Eine häufige Aufgabe ist es, eine Zeichenkette umzukehren. Dies bedeutet, dass die Reihenfolge der Zeichen in der Zeichenkette umgekehrt wird. Beispielsweise würde die Zeichenkette "hello" bei der Umkehrung "olleh" werden. In dieser Herausforderung werden wir eine Funktion implementieren, um eine Zeichenkette in-place umzukehren.
