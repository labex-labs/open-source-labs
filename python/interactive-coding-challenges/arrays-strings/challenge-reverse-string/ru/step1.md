# Перевернуть строку

## Задача

Реализовать функцию для переворачивания строки (списка символов) на месте. Это означает, что функция должна изменять исходную строку, а не создавать новую. Функция должна принимать список символов в качестве входных данных и возвращать тот же список с перевернутыми символами.

Для решения этой задачи необходимо учесть несколько требований:

## Требования

- Можно предположить, что строка состоит из ASCII-символов.
- Поскольку нужно сделать это на месте, нельзя использовать оператор среза или функцию reversed.
- Поскольку строки в Python неизменяемы, можно использовать список символов вместо строки.

## Пример использования

Ниже приведены примеры того, как должна работать функция:

- None -> None
- [''] -> ['']
- ['f', 'o', 'o','', 'b', 'a', 'r'] -> ['r', 'a', 'b','', 'o', 'o', 'f']
