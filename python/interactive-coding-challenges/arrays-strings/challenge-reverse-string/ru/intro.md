# Введение

В программировании часто приходится манипулировать строками. Одна из распространенных задач - это перевернуть строку. Это означает, что порядок символов в строке меняется на обратный. Например, строка "hello" при переворачивании станет "olleh". В этом испытании мы реализуем функцию для переворачивания строки на месте.
