# Introducción

En la programación, a menudo es necesario manipular cadenas de texto. Una tarea común es invertir una cadena. Esto significa que el orden de los caracteres en la cadena se invierte. Por ejemplo, la cadena "hello" se convertirá en "olleh" cuando se invierta. En este desafío, implementaremos una función para invertir una cadena en el mismo lugar.
