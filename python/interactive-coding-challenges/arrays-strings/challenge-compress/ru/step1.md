# Сжатие

## Задача

Дана строка, сжать ее так, чтобы 'AAABCCDDDD' стала 'A3BC2D4'. Сжатую строку возвращают только в том случае, если это экономит место. Если сжатая строка длиннее исходной строки, возвращают исходную строку. Строка чувствительна к регистру и можно считать, что она ASCII. Допустимо использование дополнительных данных структур и можно считать, что строка помещается в память.

## Требования

Следующие требования должны быть выполнены:

- Строка ASCII.
- Строка чувствительна к регистру.
- Допустимо использование дополнительных данных структур.
- Строка помещается в память.

## Пример использования

Следующие примеры демонстрируют ожидаемый ввод и вывод функции:

- None -> None
- '' -> ''
- 'AABBCC' -> 'AABBCC'
- 'AAABCCDDDD' -> 'A3BC2D4'
