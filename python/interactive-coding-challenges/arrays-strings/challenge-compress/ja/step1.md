# 圧縮

## 問題

与えられた文字列を圧縮して、「AAABCCDDDD」を「A3BC2D4」に変換します。圧縮された文字列は、スペースを節約する場合のみ返されます。圧縮された文字列が元の文字列より長い場合は、元の文字列を返します。文字列は大文字小文字を区別し、ASCIIと仮定できます。追加のデータ構造を使用でき、文字列はメモリに収まると仮定できます。

## 要件

以下の要件を満たす必要があります。

- 文字列はASCIIです。
- 文字列は大文字小文字を区別します。
- 追加のデータ構造を使用できます。
- 文字列はメモリに収まります。

## 例の使用法

以下の例は、関数の予想される入力と出力を示しています。

- なし -> なし
- '' -> ''
- 'AABBCC' -> 'AABBCC'
- 'AAABCCDDDD' -> 'A3BC2D4'
