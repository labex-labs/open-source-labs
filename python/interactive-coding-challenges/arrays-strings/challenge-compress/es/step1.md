# Comprimir

## Problema

Dada una cadena, comprimirla de modo que 'AAABCCDDDD' se convierta en 'A3BC2D4'. La cadena comprimida solo se debe devolver si ahorra espacio. Si la cadena comprimida es más larga que la cadena original, devolver la cadena original. La cadena es sensible a mayúsculas y minúsculas y se puede suponer que es ASCII. Se pueden utilizar estructuras de datos adicionales y se puede suponer que la cadena cabe en memoria.

## Requisitos

Los siguientes requisitos deben cumplirse:

- La cadena es ASCII.
- La cadena es sensible a mayúsculas y minúsculas.
- Se pueden utilizar estructuras de datos adicionales.
- La cadena cabe en memoria.

## Uso de ejemplo

Los siguientes ejemplos demuestran la entrada y salida esperadas de la función:

- Ninguno -> Ninguno
- '' -> ''
- 'AABBCC' -> 'AABBCC'
- 'AAABCCDDDD' -> 'A3BC2D4'
