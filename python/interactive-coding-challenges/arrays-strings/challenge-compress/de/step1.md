# Komprimieren

## Problem

Gegeben einen String, komprimieren Sie ihn so, dass „AAABCCDDDD“ zu „A3BC2D4“ wird. Die komprimierte Zeichenfolge sollte nur zurückgegeben werden, wenn dadurch Speicher eingespart wird. Wenn die komprimierte Zeichenfolge länger als die ursprüngliche Zeichenfolge ist, geben Sie die ursprüngliche Zeichenfolge zurück. Die Zeichenfolge ist case-sensitive und kann als ASCII angenommen werden. Zusätzliche Datenstrukturen können verwendet werden, und es kann angenommen werden, dass die Zeichenfolge im Speicher passt.

## Anforderungen

Die folgenden Anforderungen müssen erfüllt werden:

- Die Zeichenfolge ist ASCII.
- Die Zeichenfolge ist case-sensitive.
- Zusätzliche Datenstrukturen können verwendet werden.
- Die Zeichenfolge passt im Speicher.

## Beispielverwendung

Die folgenden Beispiele veranschaulichen die erwarteten Eingaben und Ausgaben der Funktion:

- None -> None
- '' -> ''
- 'AABBCC' -> 'AABBCC'
- 'AAABCCDDDD' -> 'A3BC2D4'
