# Somme de deux éléments

## Problème

Étant donné un tableau d'entiers, trouver les deux indices dont la somme est égale à une valeur spécifique. La fonction devrait renvoyer les indices sous forme de liste.

## Exigences

Pour résoudre le problème de la somme de deux éléments, les exigences suivantes doivent être satisfaites :

- Il existe exactement une solution.
- Il existe toujours une solution.
- Le tableau est un tableau d'entiers.
- Le tableau n'est pas trié.
- Des valeurs négatives sont possibles.
- Les entrées ne sont pas nécessairement valides.
- La solution doit tenir en mémoire.

## Utilisation de l'exemple

Le problème de la somme de deux éléments peut être résolu à l'aide de l'exemple suivant :

- Aucune entrée -> TypeError
- [] -> ValueError
- [1, 3, 2, -7, 5], 7 -> [2, 4]
