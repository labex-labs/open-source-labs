# Two Sum

## 問題

整数の配列が与えられたとき、特定の値に合計する 2 つのインデックスを見つけます。この関数は、インデックスをリストとして返す必要があります。

## 要件

Two Sum 問題を解くには、次の要件を満たす必要があります。

- 正確に 1 つの解があります。
- 必ず解が存在します。
- 配列は整数の配列です。
- 配列はソートされていません。
- 負の値があり得ます。
- 入力は必ずしも有効ではありません。
- 解はメモリに収まる必要があります。

## 例の使用法

Two Sum 問題は、次の例を使って解くことができます。

- 入力なし -> TypeError
- [] -> ValueError
- [1, 3, 2, -7, 5], 7 -> [2, 4]
