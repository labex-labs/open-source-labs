# はじめに

Two Sum 問題は、コンピュータサイエンスにおける古典的なアルゴリズム問題です。配列内の特定の値に合計する 2 つのインデックスを見つける必要があります。この問題は、データサイエンス、機械学習、金融など、幅広い応用分野があります。

<div class="text-xs text-gray-500 dark:text-gray-400 mt-4 border-t border-l-2 border-gray-300 dark:border-gray-600 pt-2 pl-4">
これは Challenge です。Guided Lab とは異なり、学習のためにラボの手順に従うのではなく、チャレンジタスクを独立して完了する必要があります。Challenge は通常、少し難しい課題です。難しいと感じた場合は、Labby と相談したり、解決策を確認したりできます。過去のデータによると、この <span class="text-orange-600 dark:text-orange-400">中級</span> レベルのチャレンジの合格率は <span class="text-orange-600 dark:text-orange-400">68.18%</span>です。学習者から <span class="text-blue-600 dark:text-blue-400">88.89%</span> の好評価を得ています。
