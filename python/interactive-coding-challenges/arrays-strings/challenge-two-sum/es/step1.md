# Suma de Dos

## Problema

Dada una matriz de enteros, encontrar los dos índices que sumen un valor específico. La función debe devolver los índices como una lista.

## Requisitos

Para resolver el problema de la Suma de Dos, deben cumplirse los siguientes requisitos:

- Hay exactamente una solución.
- Siempre hay una solución.
- La matriz es una matriz de enteros.
- La matriz no está ordenada.
- Los valores negativos son posibles.
- Las entradas no son necesariamente válidas.
- La solución debe caber en memoria.

## Uso de ejemplo

El problema de la Suma de Dos se puede resolver utilizando el siguiente ejemplo:

- Entrada nula -> TypeError
- [] -> ValueError
- [1, 3, 2, -7, 5], 7 -> [2, 4]
