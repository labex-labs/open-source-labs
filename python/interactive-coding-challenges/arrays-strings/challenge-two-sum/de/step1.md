# Zwei Summen

## Problem

Gegeben ein Array von ganzen Zahlen, finde die zwei Indizes, deren Summe einem bestimmten Wert entspricht. Die Funktion sollte die Indizes als Liste zurückgeben.

## Anforderungen

Um das Zwei-Sum-Problem zu lösen, müssen die folgenden Anforderungen erfüllt sein:

- Es gibt genau eine Lösung.
- Es gibt immer eine Lösung.
- Das Array ist ein Array von ganzen Zahlen.
- Das Array ist nicht sortiert.
- Negative Werte sind möglich.
- Die Eingaben sind nicht notwendigerweise gültig.
- Die Lösung muss in den Arbeitsspeicher passen.

## Beispielverwendung

Das Zwei-Sum-Problem kann mit dem folgenden Beispiel gelöst werden:

- Keine Eingabe -> TypeError
- [] -> ValueError
- [1, 3, 2, -7, 5], 7 -> [2, 4]
