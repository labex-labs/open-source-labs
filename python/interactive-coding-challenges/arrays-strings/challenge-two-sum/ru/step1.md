# Два числа в сумме

## Задача

Дан массив целых чисел. Найти два индекса, сумма значений по которым равна определенному значению. Функция должна вернуть индексы в виде списка.

## Требования

Для решения задачи «Два числа в сумме» необходимо выполнить следующие требования:

- Есть ровно одно решение.
- Решение всегда существует.
- Массив состоит из целых чисел.
- Массив не отсортирован.
- Возможны отрицательные значения.
- Входные данные не обязательно действительны.
- Решение должно поместиться в память.

## Пример использования

Задачу «Два числа в сумме» можно решить с использованием следующего примера:

- Нет входных данных -> TypeError
- [] -> ValueError
- [1, 3, 2, -7, 5], 7 -> [2, 4]
