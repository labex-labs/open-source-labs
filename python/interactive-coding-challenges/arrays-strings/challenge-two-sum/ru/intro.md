# Введение

Проблема «Два числа в сумме» является классической алгоритмической задачей в информатике. Требуется найти два индекса в массиве, сумма значений по которым равна определенному значению. Эта задача имеет широкий спектр применений, в том числе в области.data science, machine learning и финансы.
