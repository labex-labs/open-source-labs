# 简介

哈希表是一种数据结构，它将键映射到值以便进行高效查找。它使用哈希函数来计算进入桶或槽数组的索引，从中可以找到所需的值。在这个挑战中，我们将实现一个具有插入、获取和删除方法的哈希表。
