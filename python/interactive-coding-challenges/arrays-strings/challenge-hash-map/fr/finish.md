# Résumé

Dans ce défi, nous avons implémenté une table de hachage avec des méthodes de définition, de récupération et de suppression. Nous avons utilisé le chaînage pour la résolution des collisions et avons supposé que les clés sont uniquement des entiers. Nous n'avons pas eu besoin de considérer les facteurs de charge ou de valider les entrées. La table de hachage tient en mémoire.
