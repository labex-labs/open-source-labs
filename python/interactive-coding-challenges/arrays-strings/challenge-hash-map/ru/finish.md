# Резюме

В этом испытании мы реализовали хеш-таблицу с методами set, get и remove. Для разрешения коллизий мы использовали цепочку и предположили, что ключи могут быть только целыми числами. Мы не должны были учитывать коэффициенты загрузки или проверять входные данные. Хеш-таблица помещается в память.
