# Разница между строками

## Задача

Даны две строки, и нам нужно найти одиночный отличный символ между ними. Предполагается, что строки состоят из ASCII-символов в нижнем регистре. Мы не можем предполагать, что входные данные действительны, поэтому нужно проверить на `None`. Если входные данные действительны, то можно предположить, что между двумя строками есть только один отличный символ. Также нужно убедиться, что решение помещается в память.

## Требования

Для решения этой задачи нужно учитывать следующие требования:

- Строки состоят из ASCII-символов.
- Строки в нижнем регистре.
- Нужно проверить на вход `None`.
- Между двумя строками есть только один отличный символ.
- Решение должно поместиться в память.

## Пример использования

Вот несколько примеров использования функции:

- Вход `None` -> `TypeError`
- 'ab', 'aab' -> 'a'
- 'aab', 'ab' -> 'a'
- 'abcd', 'abcde' -> 'e'
- 'aaabbcdd', 'abdbacade' -> 'e'
