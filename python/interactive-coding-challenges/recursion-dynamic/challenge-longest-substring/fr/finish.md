# Résumé

Ce défi Python consistait à trouver la plus longue sous-chaîne commune entre deux chaînes de caractères données. Le programme devrait supposer que les entrées ne sont pas nécessairement valides, mais que le problème tient en mémoire. La solution devrait être sensible à la casse et supposer que les chaînes sont en ASCII. La sortie devrait être une chaîne de caractères.
