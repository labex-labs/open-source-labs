# 最長部分文字列

## 問題

2つの文字列が与えられたとき、その最長共通部分文字列を見つけることが課題です。部分文字列とは、連続した文字のブロックです。解は大文字小文字を区別し、文字列はASCIIであると仮定します。出力は文字列でなければならず、プログラムは入力が必ずしも有効でないことを想定しています。ただし、問題がメモリに収まることを想定できます。

## 要件

この問題を解くために、プログラムは次の要件を満たさなければなりません。

- 入力は必ずしも有効でない。
- 文字列はASCIIである。
- 解は大文字小文字を区別する。
- 部分文字列は連続した文字のブロックである。
- 出力は文字列でなければならない。
- プログラムは問題がメモリに収まることを想定する。

## 例の使い方

プログラムは次のように動作する必要があります。

- str0またはstr1がNoneの場合、例外を発生させる。
- str0またはstr1が0に等しい場合、出力は空の文字列でなければならない。
- 一般的な場合、str0 = 'ABCDEFGHIJ' および str1 = 'FOOBCDBCDE' が与えられたとき、出力は 'BCDE' でなければならない。
