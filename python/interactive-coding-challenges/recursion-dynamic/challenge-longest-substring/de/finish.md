# Zusammenfassung

Diese Python-Aufgabe verlangte die Suche nach der längsten gemeinsamen Teilzeichenfolge zwischen zwei gegebenen Zeichenketten. Das Programm sollte davon ausgehen, dass die Eingaben nicht unbedingt gültig sind, aber dass das Problem in den Arbeitsspeicher passt. Die Lösung sollte case-sensitive sein und davon ausgehen, dass die Zeichenketten ASCII-Zeichen sind. Die Ausgabe sollte eine Zeichenkette sein.
