# Резюме

В этом испытании на Python требовалось найти самую длинную общую подстроку между двумя заданными строками. Программа должна предполагать, что входные данные не обязательно являются валидными, но что задача помещается в память. Решение должно быть чувствительным к регистру и предполагать, что строки состоят из ASCII-символов. Выход должен быть строкой.
