# Самая длинная подстрока

## Задача

Даны две строки, и задача заключается в нахождении самой длинной общей подстроки. Подстрока - это непрерывный блок символов. Решение должно быть чувствительным к регистру и предполагать, что строки состоят из ASCII-символов. Выход должен быть строкой, и программа должна предполагать, что входные данные не обязательно являются валидными. Однако, можно предположить, что задача помещается в память.

## Требования

Для решения этой задачи программа должна соответствовать следующим требованиям:

- Входные данные не обязательно являются валидными.
- Строки состоят из ASCII-символов.
- Решение должно быть чувствительным к регистру.
- Подстрока - это непрерывный блок символов.
- Выход должен быть строкой.
- Программа должна предполагать, что задача помещается в память.

## Пример использования

Программа должна работать следующим образом:

- Если str0 или str1 равно None, должно быть поднято исключение.
- Если str0 или str1 равно 0, выход должен быть пустой строкой.
- В общем случае, при str0 = 'ABCDEFGHIJ' и str1 = 'FOOBCDBCDE' выход должен быть 'BCDE'.
