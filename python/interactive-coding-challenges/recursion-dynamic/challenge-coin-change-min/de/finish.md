# Zusammenfassung

In dieser Python-Aufgabe haben wir gelernt, wie man die minimale Anzahl von Möglichkeiten bestimmt, um n Cent mit einer Menge von Münzen mit Denominationen kleiner als n Cent zu bilden. Wir haben auch die Anforderungen für das Lösen dieses Problems identifiziert und einige Beispiele dafür gegeben, wie diese Funktion verwendet werden kann.
