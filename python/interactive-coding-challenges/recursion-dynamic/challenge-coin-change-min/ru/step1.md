# Минимальное количество монет для сдачи

## Задача

Даны монеты с номиналами меньше n центов. Необходимо определить минимальное количество способов составить n центов с использованием этих монет. Монеты можно использовать в любом сочетании, при этом у нас бесконечное количество монет каждого номинала. Не нужно отчитываться о комбинациях монет, которые дают минимальное количество.

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Монеты должны точно составить n центов.
- Можно предположить, что у нас бесконечное количество монет для составления n центов.
- Не нужно отчитываться о комбинациях монет, которые дают минимальное количество.
- Не можно предполагать, что номиналы монет заданы в отсортированном порядке.
- Можно предположить, что это помещается в память.

## Пример использования

Вот несколько примеров использования этой функции:

- coins: None или n: None -> Исключение
- coins: [] или n: 0 -> 0
- coins: [1, 2, 3] или [3, 2, 1] -> 2
