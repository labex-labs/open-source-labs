# Обзор

Кратко говоря, задача с магическим индексом заключается в поиске элемента в отсортированном массиве, значение которого равно его индексу. Функция должна возвращать самый левый магический индекс, если их несколько, и -1, если магического индекса нет. Требования к функции включают отсортированный массив, возможность наличия дубликатов элементов и допуск отрицательных значений.
