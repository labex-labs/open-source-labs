# Einführung

In der Informatik ist ein magischer Index (auch als Fixpunkt bezeichnet) eines Arrays ein Index i im Array, dessen Wert gleich i ist. Mit anderen Worten, die Aufgabe besteht darin, ein Element im Array zu finden, dessen Wert gleich seinem Index ist.
