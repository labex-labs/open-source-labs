# Resumen

En resumen, el problema del índice mágico consiste en encontrar un elemento en una matriz ordenada tal que su valor sea igual a su índice. La función debe devolver el índice mágico más a la izquierda si hay múltiples índices mágicos, y -1 si no hay ningún índice mágico. Los requisitos para la función incluyen una matriz ordenada, la posibilidad de elementos duplicados y la aceptación de valores negativos.
