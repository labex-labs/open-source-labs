# Zusammenfassung

Zusammenfassend ist das Problem der längsten aufsteigenden Teilfolge die Suche nach der Länge der längsten Teilfolge einer gegebenen Sequenz, sodass alle Elemente der Teilfolge in aufsteigender Reihenfolge sortiert sind. Wir können dieses Problem durch Berücksichtigung der oben genannten Anforderungen lösen.
