# Einführung

In der Informatik ist das Problem der längsten aufsteigenden Teilfolge (engl. Longest Increasing Subsequence, LIS) die Suche nach der Länge der längsten Teilfolge einer gegebenen Sequenz, sodass alle Elemente der Teilfolge in aufsteigender Reihenfolge sortiert sind. Diese Teilfolge muss nicht notwendigerweise zusammenhängend oder eindeutig sein.
