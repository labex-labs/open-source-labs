# Самая длинная возрастающая подпоследовательность

## Задача

Дана последовательность целых чисел. Найти самую длинную возрастающую подпоследовательность. Подпоследовательность может быть не непрерывной и может содержать дубликаты. Если есть несколько решений, вернуть любое из них.

## Требования

Для решения этой задачи необходимо учитывать следующие требования:

- Возможны ли дубликаты?
  - Да
- Можем ли мы предполагать, что входные данные - целые числа?
  - Да
- Можем ли мы предполагать, что входные данные валидны?
  - Нет, необходимо обрабатывать невалидные входные данные.
- Ждем ли, что результат будет массивом самой длинной возрастающей подпоследовательности?
  - Да
- Можем ли мы предполагать, что это помещается в память?
  - Да

## Пример использования

Ниже приведены примеры использования этой функции:

- Ничего -> Исключение
- [] -> []
- [3, 4, -1, 0, 6, 2, 3] -> [-1, 0, 2, 3]
