# Резюме

Кратко говоря, задача о нахождении самой длинной возрастающей подпоследовательности (Longest Increasing Subsequence) заключается в определении длины самой длинной подпоследовательности заданной последовательности, таких что все элементы подпоследовательности упорядочены по возрастанию. Мы можем решить эту задачу, учитывая вышеупомянутые требования.
