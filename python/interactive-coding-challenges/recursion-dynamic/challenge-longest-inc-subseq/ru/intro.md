# Введение

В информатике задача о нахождении самой длинной возрастающей подпоследовательности (Longest Increasing Subsequence, LIS) заключается в определении длины самой длинной подпоследовательности заданной последовательности, таких что все элементы подпоследовательности упорядочены по возрастанию. Эта подпоследовательность не обязательно должна быть непрерывной или уникальной.
