# 最長増加部分列

## 問題

整数の数列が与えられたとき、最長の増加部分列を見つける。部分列は連続している必要はなく、重複する要素を含んでもよい。複数の解がある場合、どれか 1 つを返す。

## 要件

この問題を解くには、以下の要件を考慮する必要がある。

- 重複は許されるか？
  - はい
- 入力が整数であると仮定できるか？
  - はい
- 入力が有効であると仮定できるか？
  - いいえ、無効な入力を処理する必要がある。
- 結果が最長増加部分列の配列であることを期待するか？
  - はい
- これがメモリに収まると仮定できるか？
  - はい

## 使い方の例

この関数の使い方の例をいくつか示す。

- なし -> 例外
- [] -> []
- [3, 4, -1, 0, 6, 2, 3] -> [-1, 0, 2, 3]
