# 最大利益 k

## 問題

連続する各日の株価のリストが与えられたとき、k 回の取引での最大利益を求める。この問題では、k 回の取引を考慮して、連続する日付の株価のリストから得られる最大利益を求める必要がある。取引は株を買うことと売ることで構成され、取引の最大数は k に制限される。解は最大利益と買う日と売る日を返す必要がある。

## 要件

この問題を解くには、次の要件を満たす必要がある。

- k は売り取引の数を表す。
- prices 入力は整数の配列である。
- 入力が有効でない場合がある。
- 株価がすべて下落して利益を得るチャンスがない場合、解は 0 を返す。
- 出力は最大利益と買う日と売る日でなければならない。
- 解はメモリに収まる必要がある。

## 例の使用法

次の例は、解の使用法を示している。

- 株価: なし または k: なし -> なし
- 株価: [] または k <= 0 -> []
- 株価: [0, -1, -2, -3, -4, -5]
  - (最大利益, 取引のリスト)
  - (0, [])
- 株価: [2, 5, 7, 1, 4, 3, 1, 3] k: 3
  - (最大利益, 取引のリスト)
  - (10, [Type.SELL 日: 7 価格: 3,
    Type.BUY 日: 6 価格: 1,
    Type.SELL 日: 4 価格: 4,
    Type.BUY 日: 3 価格: 1,
    Type.SELL 日: 2 価格: 7,
    Type.BUY 日: 0 価格: 2])
