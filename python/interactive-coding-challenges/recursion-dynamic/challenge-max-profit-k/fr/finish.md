# Résumé

Le défi consiste à déterminer le profit maximal qui peut être obtenu à partir d'une liste de prix d'actions sur des jours consécutifs, en considérant k transactions. La solution devrait retourner le profit maximal et les jours pour acheter et vendre. Les exigences pour la solution comprennent le nombre de transactions, le format d'entrée, la validité des entrées, le format de sortie et l'utilisation de la mémoire.
