# Максимальная прибыль при k сделках

## Задача

Дан список цен на акции за каждый последовательный день. Определить максимальные прибыли при k сделках. Задача требует определения максимальной прибыли, которую можно получить из списка цен на акции за последовательные дни, учитывая k сделок. Сделки состоят из покупки и продажи акций, и максимальное количество сделок ограничено k. Решение должно вернуть максимальную прибыль и дни для покупки и продажи.

## Требования

Для решения задачи необходимо выполнить следующие требования:

- k представляет количество сделок продажи.
- Входные данные prices - это массив целых чисел.
- Входные данные могут быть недействительными.
- Если цены все убывают и нет возможности получить прибыль, решение должно вернуть 0.
- Выход должен быть максимальной прибылью и днями для покупки и продажи.
- Решение должно соответствовать требованиям к памяти.

## Пример использования

Следующие примеры иллюстрируют использование решения:

- Цены: None или k: None -> None
- Цены: [] или k <= 0 -> []
- Цены: [0, -1, -2, -3, -4, -5]
  - (максимальная прибыль, список сделок)
  - (0, [])
- Цены: [2, 5, 7, 1, 4, 3, 1, 3] k: 3
  - (максимальная прибыль, список сделок)
  - (10, [Type.SELL день: 7 цена: 3,
    Type.BUY день: 6 цена: 1,
    Type.SELL день: 4 цена: 4,
    Type.BUY день: 3 цена: 1,
    Type.SELL день: 2 цена: 7,
    Type.BUY день: 0 цена: 2])
