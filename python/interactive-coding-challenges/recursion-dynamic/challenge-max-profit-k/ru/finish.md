# Резюме

Задача требует определения максимальной прибыли, которую можно получить из списка цен на акции за последовательные дни, учитывая k сделок. Решение должно вернуть максимальную прибыль и дни для покупки и продажи. Требования к решению включают количество сделок, формат входных данных, валидность входных данных, формат вывода и использование памяти.
