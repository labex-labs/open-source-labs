# 最大利润 K

## 问题

给定连续每一天的股票价格列表，确定进行 k 笔交易时的最大利润。此问题要求在考虑 k 笔交易的情况下，确定从连续几天的股票价格列表中能够获得的最大利润。交易包括买卖股票，且交易的最大数量限制为 k。解决方案应返回最大利润以及买卖的日期。

## 要求

要解决此问题，必须满足以下要求：

- k 表示卖出交易的数量。
- prices 输入是一个整数数组。
- 输入可能无效。
- 如果价格全部下跌且没有获利机会，解决方案应返回 0。
- 输出应为最大利润以及买卖日期。
- 解决方案应符合内存要求。

## 示例用法

以下示例说明了该解决方案的用法：

- 价格：无 或 k：无 -> 无
- 价格：[] 或 k <= 0 -> []
- 价格：[0, -1, -2, -3, -4, -5]
  - （最大利润，交易列表）
  - （0，[]）
- 价格：[2, 5, 7, 1, 4, 3, 1, 3]，k：3
  - （最大利润，交易列表）
  - （10，[卖出类型 日期：7 价格：3，
    买入类型 日期：6 价格：1，
    卖出类型 日期：4 价格：4，
    买入类型 日期：3 价格：1，
    卖出类型 日期：2 价格：7，
    买入类型 日期：0 价格：2]）
