# 简介

在计算机科学中，最长公共子序列（LCS）问题是在一组序列（通常只有两个序列）中找到所有序列共有的最长子序列。它与寻找公共子串的问题不同：与子串不同，子序列不要求在原始序列中占据连续的位置。
