# 最长公共子序列

## 问题

给定两个字符串，找出它们的最长公共子序列。子序列是指通过删除一个序列中的某些元素或不删除任何元素，且不改变剩余元素的顺序而从该序列派生出来的序列。例如，“ACE”是“ABCDE”的子序列，但不是“AEDCA”的子序列。

## 要求

为了解决这个问题，必须满足以下要求：

- 输入可能无效，因此程序应处理无效输入。
- 字符串为 ASCII 编码。
- 程序应区分大小写。
- 子序列是字符的非连续块。
- 程序应以字符串形式返回结果。
- 程序应假定其能在内存中运行。

## 示例用法

以下是程序应有的一些行为示例：

- 如果 str0 或 str1 为 None，应引发异常。
- 如果 str0 或 str1 等于空字符串，应返回空字符串。
- 一般情况：

```
str0 = 'ABCDEFGHIJ'
str1 = 'FOOBCDBCDE'

结果: 'BCDE'
```
