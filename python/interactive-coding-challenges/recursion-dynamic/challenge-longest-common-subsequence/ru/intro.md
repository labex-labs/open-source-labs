# Введение

В информатике задача на нахождение самой длинной общей подпоследовательности (LCS) заключается в поиске самой длинной подпоследовательности, которая является общей для всех последовательностей в наборе последовательностей (чаще всего всего двух последовательностей). Она отличается от задач на нахождение общих подстрок: в отличие от подстрок подпоследовательности не обязаны занимать последовательные позиции в исходных последовательностях.
