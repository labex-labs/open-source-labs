# Самая длинная общая подпоследовательность

## Задача

Даны две строки, найти самую длинную общую подпоследовательность. Подпоследовательность - это последовательность, которая может быть получена из другой последовательности путём удаления некоторых или ни одного элементов без изменения порядка оставшихся элементов. Например, "ACE" - это подпоследовательность "ABCDE", но не "AEDCA".

## Требования

Для решения этой задачи должны быть выполнены следующие требования:

- Входные данные могут быть недействительными, поэтому программа должна обрабатывать недействительные входы.
- Строки состоят из ASCII-символов.
- Программа должна различать заглавные и строчные буквы.
- Подпоследовательность - это непрерывный блок символов.
- Программа должна возвращать строку в качестве результата.
- Программа должна предполагать, что в памяти достаточно места.

## Пример использования

Вот несколько примеров того, как должна работать программа:

- Если str0 или str1 равно None, должно быть поднято исключение.
- Если str0 или str1 равно 0, должна быть возвращена пустая строка.
- Общий случай:

```
str0 = 'ABCDEFGHIJ'
str1 = 'FOOBCDBCDE'

результат: 'BCDE'
```
