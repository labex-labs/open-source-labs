# Резюме

Кратко говоря, задача на нахождение самой длинной общей подпоследовательности - это классическая задача информатики, которая заключается в поиске самой длинной подпоследовательности, общей для двух строк. Программа должна обрабатывать недействительные входные данные, различать заглавные и строчные буквы и предполагать, что в памяти достаточно места.
