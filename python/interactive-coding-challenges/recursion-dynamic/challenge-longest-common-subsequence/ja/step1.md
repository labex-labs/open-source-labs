# 最長共通部分列

## 問題

2 つの文字列が与えられたとき、最長共通部分列を見つける。部分列とは、あるシーケンスからいくつかの要素を削除するか、または削除せずに、残りの要素の順序を変えることなく導き出すことができるシーケンスである。たとえば、「ACE」は「ABCDE」の部分列であるが、「AEDCA」の部分列ではない。

## 要件

この問題を解くには、以下の要件を満たす必要がある。

- 入力が有効でない場合があるので、プログラムは無効な入力を処理する必要がある。
- 文字列は ASCII である。
- プログラムは大文字小文字を区別する必要がある。
- 部分列は文字の非連続ブロックである。
- プログラムは結果として文字列を返す必要がある。
- プログラムはメモリに収まることを想定する必要がある。

## 例の使用法

以下は、プログラムがどのように動作するかの例である。

- str0 または str1 が None の場合、例外が発生する必要がある。
- str0 または str1 が 0 に等しい場合、空の文字列を返す必要がある。
- 一般的なケース：

```
str0 = 'ABCDEFGHIJ'
str1 = 'FOOBCDBCDE'

結果: 'BCDE'
```
