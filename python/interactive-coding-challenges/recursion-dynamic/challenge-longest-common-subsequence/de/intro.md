# Einführung

In der Informatik ist das Problem der längsten gemeinsamen Teilfolge (engl. longest common subsequence, LCS) darin, die längste Teilfolge zu finden, die allen Sequenzen in einer Menge von Sequenzen (oft nur zwei Sequenzen) gemeinsam ist. Es unterscheidet sich von Problemen bei der Suche nach gemeinsamen Teilzeichenfolgen: Anders als Teilzeichenfolgen müssen Teilfolgen nicht auf aufeinanderfolgenden Positionen in den ursprünglichen Sequenzen stehen.
