# Zusammenfassung

Zusammenfassend lässt sich sagen, dass das Problem der längsten gemeinsamen Teilfolge ein klassisches Problem der Informatik ist, das darin besteht, die längste Teilfolge zu finden, die zwei Strings gemeinsam haben. Das Programm sollte ungültige Eingaben verarbeiten, case-sensitive sein und davon ausgehen, dass es in den verfügbaren Speicher passt.
