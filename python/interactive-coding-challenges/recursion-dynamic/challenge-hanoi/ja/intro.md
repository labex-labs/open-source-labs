# はじめに

ハノイの塔は、コンピュータサイエンスと数学における古典的な問題です。これは、3 本の棒とさまざまなサイズの複数の円盤から構成されており、どの棒にもスライドさせることができます。パズルは、円盤が 1 本の棒にサイズの昇順できちんと積み重なり、一番上が最小のもので円錐形状を形成している状態から始まります。
