# ハノイ

## 問題

あなたのタスクは、3本の塔とN枚の円盤を持つハノイの塔を実装することです。目標は、すべての円盤を最初の塔から3番目の塔に移動させることであり、次の簡単なルールを守ります。

1. 一度に1枚の円盤のみを移動できます。
2. 各移動は、1つの積み重ねから最上部の円盤を取り出し、別の積み重ねの上または空の棒の上に置くことで構成されます。
3. 小さな円盤の上には、他の円盤を置くことはできません。

## 要件

この問題を解くには、次の要件を満たす必要があります。

- この問題に使用できるスタッククラスが必要です。
- 入力を処理する前に検証する必要があります。
- プログラムがメモリに収まることを確認する必要があります。

## 例の使用法

ここに、プログラムがどのように動作するかのいくつかの例を示します。

- 塔がない場合、例外が発生する必要があります。
- 円盤が0枚の場合、プログラムはNoneを返す必要があります。
- 円盤が1枚の場合、プログラムはそれを最初の塔から3番目の塔に移動させる必要があります。
- 円盤が2枚以上の場合、プログラムは上記のルールを守りながら、それらを最初の塔から3番目の塔に移動させる必要があります。
