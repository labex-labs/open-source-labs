# ハノイ

## 問題

あなたのタスクは、3 本の塔と N 枚の円盤を持つハノイの塔を実装することです。目標は、すべての円盤を最初の塔から 3 番目の塔に移動させることであり、次の簡単なルールを守ります。

1. 一度に 1 枚の円盤のみを移動できます。
2. 各移動は、1 つの積み重ねから最上部の円盤を取り出し、別の積み重ねの上または空の棒の上に置くことで構成されます。
3. 小さな円盤の上には、他の円盤を置くことはできません。

## 要件

この問題を解くには、次の要件を満たす必要があります。

- この問題に使用できるスタッククラスが必要です。
- 入力を処理する前に検証する必要があります。
- プログラムがメモリに収まることを確認する必要があります。

## 例の使用法

ここに、プログラムがどのように動作するかのいくつかの例を示します。

- 塔がない場合、例外が発生する必要があります。
- 円盤が 0 枚の場合、プログラムは None を返す必要があります。
- 円盤が 1 枚の場合、プログラムはそれを最初の塔から 3 番目の塔に移動させる必要があります。
- 円盤が 2 枚以上の場合、プログラムは上記のルールを守りながら、それらを最初の塔から 3 番目の塔に移動させる必要があります。
