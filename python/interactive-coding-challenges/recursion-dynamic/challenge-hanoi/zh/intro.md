# 简介

汉诺塔是计算机科学和数学中的一个经典问题。它由三根杆和若干个大小不同的圆盘组成，这些圆盘可以滑到任何一根杆上。谜题开始时，圆盘按大小升序整齐地堆叠在一根杆上，最小的在顶部，从而形成一个圆锥形。
