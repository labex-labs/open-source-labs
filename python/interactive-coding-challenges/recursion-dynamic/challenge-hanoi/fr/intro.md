# Introduction

Les Tours d'Hanoï est un problème classique en informatique et en mathématiques. Elle se compose de trois tiges et d'un certain nombre de disques de différentes tailles, qui peuvent glisser sur n'importe quelle tige. Le casse-tête commence avec les disques empilés proprement dans l'ordre croissant de taille sur une tige, le plus petit en haut, formant ainsi une forme conique.
