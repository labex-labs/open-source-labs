# Введение

Задача "Великий подвиг" (Towers of Hanoi) - это классическая задача в информатике и математике. Она состоит из трех стержней и нескольких дисков разного размера, которые могут скользить на любой стержень. Задача начинается с дисков, расположенных в ровном стопке в порядке возрастания размера на одном стержне, с самым маленьким диском сверху, образуя таким образом конус.
