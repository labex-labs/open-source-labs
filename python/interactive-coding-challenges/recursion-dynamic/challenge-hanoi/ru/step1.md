# Ханойские башни

## Задача

Ваша задача - реализовать задачу Ханойских башен для 3 башен и N дисков. Цель - переместить все диски с первой башни на третью башню, соблюдая следующие простые правила:

1. В каждый момент времени можно перемещать только один диск.
2. Каждый ход состоит в том, чтобы взять верхний диск из одной из стопок и поместить его на вершину другой стопки или на пустой стержень.
3. Ни один диск не может быть помещен на диск меньшего размера.

## Требования

Для решения этой задачи вам нужно удовлетворить следующим требованиям:

- Должен быть класс стека, который можно использовать для этой задачи.
- Перед обработкой входных данных их нужно проверить на корректность.
- Следует обеспечить, чтобы программа не потребляла слишком много памяти.

## Пример использования

Вот несколько примеров того, как должна работать программа:

- Если нет ни одной башни, должно быть поднято исключение.
- Если количество дисков равно 0, программа должна вернуть None.
- Если есть только 1 диск, программа должна переместить его с первой башни на третью башню.
- Если есть 2 или более дисков, программа должна переместить их с первой башни на третью башню, соблюдая вышеперечисленные правила.
