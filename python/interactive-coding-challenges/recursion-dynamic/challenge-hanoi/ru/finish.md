# Резюме

Кратко говоря, задача "Великий подвиг" (Towers of Hanoi) - это классическая задача, которая требует перемещения дисков с одной башни на другую, соблюдая определенные правила. Чтобы решить эту задачу, вам нужно иметь класс стека, проверить входные данные на корректность и обеспечить, чтобы программа не потребляла слишком много памяти.
