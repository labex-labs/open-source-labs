# Hanoi

## Problem

Ihre Aufgabe ist es, das Tower of Hanoi mit 3 Türmen und N Scheiben zu implementieren. Ziel ist es, alle Scheiben vom ersten Turm zum dritten Turm zu bewegen, wobei folgende einfache Regeln befolgt werden müssen:

1. Es kann nur eine Scheibe zu einem Zeitpunkt bewegt werden.
2. Jede Bewegung besteht darin, die obere Scheibe eines Stapels zu nehmen und sie auf einen anderen Stapel oder auf einen leeren Stab zu legen.
3. Keine Scheibe darf auf eine kleinere Scheibe gelegt werden.

## Anforderungen

Um dieses Problem zu lösen, müssen Sie die folgenden Anforderungen erfüllen:

- Sie sollten eine Stapelklasse haben, die für dieses Problem verwendet werden kann.
- Sie sollten die Eingaben vor der Verarbeitung validieren.
- Sie sollten sicherstellen, dass das Programm im Speicher passt.

## Beispielverwendung

Hier sind einige Beispiele dafür, wie das Programm verhalten sollte:

- Wenn es keine Türme gibt, sollte eine Ausnahme ausgelöst werden.
- Wenn es 0 Scheiben gibt, sollte das Programm None zurückgeben.
- Wenn es nur 1 Scheibe gibt, sollte das Programm sie vom ersten Turm zum dritten Turm bewegen.
- Wenn es 2 oder mehr Scheiben gibt, sollte das Programm sie vom ersten Turm zum dritten Turm bewegen, wobei die oben genannten Regeln befolgt werden.
