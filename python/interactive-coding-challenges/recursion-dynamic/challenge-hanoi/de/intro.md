# Einführung

Das Tower of Hanoi ist ein klassisches Problem der Informatik und Mathematik. Es besteht aus drei Stäben und einer Anzahl von Scheiben unterschiedlicher Größe, die auf jeden Stab gleiten können. Das Rätsel beginnt mit den Scheiben in einem ordentlichen Stapel in aufsteigender Reihenfolge der Größe auf einem Stab, wobei die kleinste oben ist, was eine kegelförmige Gestalt ergibt.
