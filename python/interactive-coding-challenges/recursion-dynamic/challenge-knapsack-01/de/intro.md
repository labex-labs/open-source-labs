# Einführung

Das Rucksackproblem ist ein klassisches Optimierungsproblem der Informatik. Es ist ein Problem der kombinatorischen Optimierung, bei dem das Ziel besteht, Gegenstände auszuwählen, um den Wert zu maximieren, während das Gewicht innerhalb der Kapazität des Rucksacks bleibt. Dieses Problem wird in vielen realen Anwendungen verwendet, wie z. B. in der Ressourcenallokation, der Portfoliooptimierung und den Schneidegutproblemen.
