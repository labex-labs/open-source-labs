# Introducción

El problema de la mochila es un problema clásico de optimización en informática. Es un problema de optimización combinatoria, donde el objetivo es seleccionar elementos para maximizar el valor mientras se mantiene el peso dentro de la capacidad de la mochila. Este problema se utiliza en muchas aplicaciones del mundo real, como la asignación de recursos, la optimización de carteras y los problemas de corte de existencias.
