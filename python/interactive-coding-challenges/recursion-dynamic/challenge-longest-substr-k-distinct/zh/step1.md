# 最长子串（最多 k 个不同字符）

## 问题

给定一个字符串和一个整数 k，找出包含最多 k 个不同字符的最长子串的长度。子串是连续的字符块。例如，在字符串 “abcabcdefgghiij” 中，最多包含 3 个不同字符的最长子串是 “abcabc”。如果有多个长度相同的子串，返回其中任何一个。

## 要求

为了解决这个挑战，必须满足以下要求：

- 输入可能无效，因此代码应能优雅地处理无效输入。
- 字符串为 ASCII 编码。
- 搜索区分大小写。
- 子串是连续的字符块。
- 结果应为整数。
- 代码应能在内存限制内处理输入。

## 示例用法

以下示例展示了代码的预期行为：

- 无 -> 类型错误
- '', k = 3 -> 0
- 'abcabcdefgghiij', k = 3 -> 6
- 'abcabcdefgghighij', k = 3 -> 7
