# 最長の部分文字列（k個の異なる文字）

## 問題

文字列と整数kが与えられたとき、最大でk個の異なる文字を含む最長の部分文字列の長さを求めます。部分文字列は、文字の連続したブロックです。たとえば、文字列 "abcabcdefgghiij" の場合、最大で3個の異なる文字を持つ最長の部分文字列は "abcabc" です。同じ長さの部分文字列が複数ある場合は、それらのいずれかを返します。

## 要件

このチャレンジを解くには、以下の要件を満たす必要があります。

- 入力が有効でない場合があるため、コードは無効な入力を適切に処理する必要があります。
- 文字列はASCIIです。
- 検索は大文字小文字を区別します。
- 部分文字列は、文字の連続したブロックです。
- 結果は整数でなければなりません。
- コードは、メモリ制限内で入力を処理できる必要があります。

## 例の使用法

以下の例は、コードの期待される動作を示しています。

- なし -> TypeError
- '', k = 3 -> 0
- 'abcabcdefgghiij', k=3 -> 6
- 'abcabcdefgghighij', k=3 -> 7
