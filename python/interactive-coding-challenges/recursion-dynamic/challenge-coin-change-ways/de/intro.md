# Einführung

Diese Herausforderung geht darum, die Möglichkeiten zu zählen, um für einen bestimmten Geldbetrag Wechselgeld zu geben, indem ein Array von unterschiedlichen Münzen verwendet wird. Ziel ist es, die Anzahl der einzigartigen Kombinationen zu finden, die zum Wechselgeld geben können.
