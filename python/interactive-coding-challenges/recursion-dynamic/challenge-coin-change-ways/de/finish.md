# Zusammenfassung

In dieser Herausforderung haben wir gelernt, wie man die Möglichkeiten zählt, um für einen bestimmten Geldbetrag Wechselgeld zu geben, indem ein Array von unterschiedlichen Münzen verwendet wird. Mit Hilfe von dynamischem Programmieren konnten wir die Anzahl der einzigartigen Kombinationen effizient berechnen.
