# Способы размена монет

## Задача

Даны целое число n и массив различных монет. Напишите функцию для подсчета количества способов размена суммы n с использованием монет из массива. Монету можно использовать любое количество раз, и мы считаем уникальные комбинации.

Например, если n = 4 и coins = [1, 2], то есть 3 способа размена: 1+1+1+1, 1+2+1 и 2+2.

## Требования

Для решения этой задачи вам понадобится:

- Написать функцию, которая принимает два аргумента: целое число n и массив различных монет.
- Использовать динамическое программирование для подсчета количества способов размена суммы n с использованием монет из массива.
- Вернуть количество уникальных комбинаций.

## Пример

Вход: n = 4, coins = [1, 2]

Выход: 3. 1+1+1+1, 1+2+1, 2+2 - это способы размена.
