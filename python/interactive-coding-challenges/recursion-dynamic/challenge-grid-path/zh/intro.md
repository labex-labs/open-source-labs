# 简介

在这个挑战中，我们将实现一种算法，用于将机器人从矩形网格的左上角移动到右下角。机器人只能向右和向下移动，并且某些单元格可能是禁区。

这是一个挑战（Challenge），与实验（Lab）不同，你需要独立完成挑战任务，而不是按照实验的步骤学习。挑战通常有一点难度。如果你觉得困难，可以与 Labby 讨论或查看解决方案。
