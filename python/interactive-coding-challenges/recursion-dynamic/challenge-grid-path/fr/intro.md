# Introduction

Dans ce défi, nous allons implémenter un algorithme pour déplacer un robot du coin supérieur gauche au coin inférieur droit d'une grille rectangulaire. Le robot ne peut se déplacer que vers la droite et vers le bas, et certaines cellules peuvent être interdites.
