# Introducción

En este desafío, implementaremos un algoritmo para mover un robot desde la esquina superior izquierda hasta la esquina inferior derecha de una cuadrícula rectangular. El robot solo puede moverse hacia la derecha y hacia abajo, y algunas celdas pueden estar fuera de límites.
