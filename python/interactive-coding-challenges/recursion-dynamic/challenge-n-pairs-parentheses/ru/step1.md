# N пар скобок

## Задача

Задача заключается в нахождении всех допустимых комбинаций из n-пар скобок. Допустимая комбинация - это та, в которой каждая открывающая скобка имеет соответствующую закрывающую скобку и пары скобок правильно вложены. Например, следующие комбинации допустимы для 3-пар скобок:

- ((()))
- (()())
- (())()
- ()(())
- ()()()

Следующие комбинации не допустимы для 3-пар скобок:

- )()(
- ((()
- ))((
- ()()()

## Требования

Для решения этой задачи необходимо ответить на следующие вопросы:

- Вход - это целое число, представляющее количество пар?
  - Да, вход - это целое число, представляющее количество пар.
- Можем ли мы предполагать, что входные данные валидны?
  - Нет, мы не можем предполагать, что входные данные валидны.
- Выход - это список допустимых комбинаций?
  - Да, выход - это список допустимых комбинаций.
- Должен ли выход содержать дубликаты?
  - Нет, выход не должен содержать дубликаты.
- Можем ли мы предполагать, что это помещается в память?
  - Да, мы можем предполагать, что это помещается в память.

## Пример использования

Следующие - примеры использования функции:

- None -> Исключение
- Отрицательное -> Исключение
- 0 -> []
- 1 -> ['()']
- 2 -> ['(())', '()()']
- 3 -> ['((()))', '(()())', '(())()', '()(())', '()()()']
