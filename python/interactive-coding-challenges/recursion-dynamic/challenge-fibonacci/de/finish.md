# Zusammenfassung

Zusammenfassend lässt sich sagen, dass die Fibonacci-Folge ein beliebtes Problem in der Informatik und Mathematik ist. Das Problem besteht darin, die Folge auf drei verschiedene Weise zu implementieren: rekursiv, dynamisch und iterativ. Das Programm sollte bei 0 und 1 beginnen, und die Eingaben sollten gültige nicht-negative ganze Zahlen sein. Sowohl die rekursive als auch die iterative Lösung sollten implementiert werden, und das Programm sollte in den Arbeitsspeicher passen.
