# Einführung

Die Fibonacci-Folge ist eine Zahlenfolge, bei der jede Zahl die Summe der beiden vorhergehenden Zahlen ist, wobei die Folge normalerweise mit 0 und 1 beginnt. Es ist ein beliebiges Problem in der Informatik und Mathematik.
