# Обзор

Кратко говоря, последовательность Фибоначчи - популярная задача в информатике и математике. Задача заключается в реализации последовательности тремя разными способами: рекурсивным, динамическим и итеративным. Программа должна начинаться с 0 и 1, и входными данными должны быть допустимые неотрицательные целые числа. Реализовать оба рекурсивный и итеративный решения, при этом программа должна помещаться в память.
