# Einführung

In der Mathematik ist eine Permutation eine Anordnung von Objekten in einer bestimmten Reihenfolge. In der Informatik müssen wir oft alle möglichen Permutationen eines gegebenen Eingabestrings finden. Dies ist ein übliches Problem, das mit verschiedenen Algorithmen gelöst werden kann.
