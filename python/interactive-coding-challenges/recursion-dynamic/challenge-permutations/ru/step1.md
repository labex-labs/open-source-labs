# Перестановки

## Задача

Дана входная строка, и задача заключается в нахождении всех возможных перестановок символов в этой строке. Выход должен быть списком строк, где каждая строка представляет собой уникальную перестановку входной строки. Входная строка может содержать дубликаты, но в выходе не должно быть дубликатов.

## Требования

Для решения этой задачи необходимо учитывать следующие требования:

- Вход может содержать дубликаты.
- В выходе не должно быть дубликатов.
- Выход должен быть списком строк.
- Результаты не требуют сортировки.
- Входные данные не всегда могут быть валидными.
- Решение должно помещаться в память.

## Пример использования

Ниже приведены примеры того, как должна работать функция:

- Если входное значение равно None, то выход должен быть None.
- Если входная строка пустая, то выход должен быть пустой строкой.
- Если входная строка 'AABC', то выход должен быть ['AABC', 'AACB', 'ABAC', 'ABCA', 'ACAB', 'ACBA', 'BAAC', 'BACA', 'BCAA', 'CAAB', 'CABA', 'CBAA'].
