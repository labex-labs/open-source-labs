# 順列

## 問題

入力文字列が与えられたとき、その文字列の文字のすべての可能な順列を見つけることが課題です。出力は文字列のリストでなければならず、各文字列は入力文字列の一意の順列を表します。入力文字列には重複が含まれていてもよいが、出力には重複がなければなりません。

## 要件

この問題を解くには、次の要件を考慮する必要があります。

- 入力には重複が含まれていてもよい。
- 出力には重複がなければならない。
- 出力は文字列のリストでなければならない。
- 結果はソートする必要はない。
- 入力は必ずしも有効であるとは限らない。
- ソリューションはメモリに収まる必要がある。

## 例の使い方

ここに、関数がどのように動作するかのいくつかの例を示します。

- 入力がNoneの場合、出力はNoneでなければならない。
- 入力が空文字列の場合、出力は空文字列でなければならない。
- 入力が 'AABC' の場合、出力は ['AABC', 'AACB', 'ABAC', 'ABCA', 'ACAB', 'ACBA', 'BAAC', 'BACA', 'BCAA', 'CAAB', 'CABA', 'CBAA'] でなければならない。
