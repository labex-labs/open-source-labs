# Introduction

En mathématiques, une permutation est un arrangement d'objets dans un ordre spécifique. En informatique, nous avons souvent besoin de trouver toutes les permutations possibles d'une chaîne de caractères d'entrée donnée. Il s'agit d'un problème courant qui peut être résolu à l'aide de divers algorithmes.
