# Introducción

El problema de la mochila es un problema clásico de optimización en informática. Consiste en seleccionar un subconjunto de elementos con el valor máximo, sujeto a una restricción de peso. El problema de la mochila ilimitada es una variación del problema de la mochila en la que los elementos se pueden seleccionar múltiples veces.
