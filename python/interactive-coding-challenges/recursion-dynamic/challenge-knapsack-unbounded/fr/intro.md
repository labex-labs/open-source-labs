# Introduction

Le problème du sac à dos est un problème classique d'optimisation en informatique. Il consiste à sélectionner un sous-ensemble d'objets ayant la valeur maximale, sous réserve d'une contrainte de poids. Le problème du sac à dos illimité est une variante du problème du sac à dos où les objets peuvent être sélectionnés plusieurs fois.
