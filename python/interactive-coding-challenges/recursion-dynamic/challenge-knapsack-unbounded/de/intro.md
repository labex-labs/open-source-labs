# Einführung

Das Rucksackproblem ist ein klassisches Optimierungsproblem der Informatik. Es besteht darin, eine Teilmenge von Gegenständen mit maximalem Wert auszuwählen, unter Berücksichtigung einer Gewichtsgrenze. Das unbeschränkte Rucksackproblem ist eine Variante des Rucksackproblems, bei der Gegenstände beliebig oft ausgewählt werden können.
