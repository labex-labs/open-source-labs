# Резюме

Задача о неограниченном рюкзаке - это вариант задачи о рюкзаке, в котором предметы можно выбирать несколько раз. Чтобы решить эту задачу, предметы можно заменить, когда они уже находятся в рюкзаке, и предмет нельзя разделить. Входные предметы не могут иметь вес или стоимость, равные 0, и требуется вернуть только общую стоимость. Входные данные могут быть недействительными, поэтому требуется валидация, и входные данные отсортированы по стоимости/весу. Ограничения по памяти не являются проблемой. Задача о неограниченном рюкзаке может быть использована в различных сценариях, таких как распределение ресурсов и оптимизация финансового портфеля.
