# Неограниченный рюкзак

## Задача

Даны рюкзак с общим вместимостью по весу и список предметов с весом w(i) и стоимостью v(i). Определите максимальную общую стоимость, которую можно унести. Предметы можно выбирать несколько раз.

## Требования

Для решения задачи о неограниченном рюкзаке необходимо выполнить следующие требования:

- Предметы можно заменить, когда они уже находятся в рюкзаке.
- Предмет нельзя разделить.
- Входные предметы не могут иметь вес или стоимость, равные 0.
- Требуется вернуть только общую стоимость, а не предметы, составляющие максимальную общую стоимость.
- Входные данные могут быть недействительными, поэтому требуется валидация.
- Входные данные отсортированы по стоимости/весу.
- Ограничения по памяти не являются проблемой.

## Пример использования

Задача о неограниченном рюкзаке может быть использована в различных сценариях, таких как распределение ресурсов и оптимизация финансового портфеля. Вот несколько примеров ее использования:

- Если общий вес или предметы равны None, должно быть поднято исключение.
- Если общий вес или предметы равны 0, результат должен быть 0.
- В общем случае предположим, что общий вес равен 8, а предметы следующие:

  | v   | w   |
  | --- | --- |
  | 0   | 0   |
  | 1   | 1   |
  | 3   | 2   |
  | 7   | 4   |

  Максимальная стоимость, которую можно унести, равна 14.
