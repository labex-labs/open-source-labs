# Введение

Задача о рюкзаке - это классическая задача оптимизации в информатике. Она заключается в выборе подмножества предметов с максимальной стоимостью, при этом соблюдая ограничение по весу. Задача о неограниченном рюкзаке - это вариант задачи о рюкзаке, в котором предметы можно выбирать несколько раз.
