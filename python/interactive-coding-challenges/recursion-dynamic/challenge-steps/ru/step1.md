# Python-вызов: ступеньки

## Задача

Представьте, что вы стоите внизу лестницы с n ступенями. Вы можете сделать один, два или три шага за раз. Задача — выяснить, сколько возможных способов есть подняться на n-ую ступень.

Например, если есть 3 ступени, вы можете подняться по лестнице следующими способами:

- 1-1-1
- 1-2
- 2-1
- 3

Таким образом, есть 4 возможных способа подняться на 3-ю ступень.

## Требования

Для решения этой задачи нужно иметь в виду следующие требования:

- Если n == 0, результат должен быть 1. Однако, существуют разные подходы к этой задаче, которые можно обсудить.
- Мы не можем предполагать, что входные данные действительны.
- Мы можем предположить, что задача помещается в память.

## Пример использования

Вот несколько примеров, как эту задачу можно решить с использованием Python:

- Отсутствующий или отрицательный ввод -> Исключение
- n == 0 -> 1
- n == 1 -> 1
- n == 2 -> 2
- n == 3 -> 4
- n == 4 -> 7
- n == 10 -> 274
