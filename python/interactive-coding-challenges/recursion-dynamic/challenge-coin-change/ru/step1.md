# Размена монет

## Задача

Даны набор монет различных номиналов и общая сумма денег n. Определить общее количество уникальных способов разменять n центов. Монеты, имеющиеся в наличии, имеют номиналы меньше n центов.

## Требования

Для решения этой задачи необходимо выполнить следующие требования:

- Монеты должны составлять ровно n центов.
- Можно предположить, что количество монет для составления n центов не ограничено.
- Не нужно отчитываться о комбинациях монет, которые представляют собой минимальное количество.
- Номиналы монет не заданы в отсортированном порядке.
- Решение должно помещаться в память.

## Пример использования

Следующие примеры демонстрируют использование задачи размена монет:

- Монеты: None или n: None -> Исключение
- Монеты: [] или n: 0 -> 0
- Монеты: [1, 2, 3], n: 5 -> 5
