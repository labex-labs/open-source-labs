# Introducción

El problema de cambio de monedas es un problema algorítmico clásico que implica determinar el número de maneras únicas de hacer cambio para una cantidad determinada de dinero. Este problema tiene una amplia gama de aplicaciones, desde máquinas expendedoras hasta sistemas financieros.
