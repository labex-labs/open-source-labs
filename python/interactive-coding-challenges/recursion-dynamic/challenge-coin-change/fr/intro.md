# Introduction

Le problème du rendu de monnaie est un problème algorithmique classique qui consiste à déterminer le nombre de façons uniques de rendre la monnaie pour un montant donné d'argent. Ce problème trouve de nombreuses applications, allant des distributeurs automatiques de boissons aux systèmes financiers.
