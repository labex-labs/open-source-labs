# Einführung

Das Münzwechselproblem ist ein klassisches algorithmisches Problem, das darin besteht, die Anzahl der einzigartigen Möglichkeiten zu bestimmen, um für einen bestimmten Geldbetrag den Wechsel zu erbringen. Dieses Problem hat eine Vielzahl von Anwendungen, von Verkaufsautomaten bis hin zu Finanzsystemen.
