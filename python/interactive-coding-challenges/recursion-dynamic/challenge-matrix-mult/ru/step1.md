# Умножение матриц

## Задача

Дана последовательность матриц 2x2, и нам нужно найти минимальную стоимость их последовательного умножения. Стоимость умножения двух матриц - это количество скалярных умножений, необходимых для этого. Например, если у нас есть матрицы A, B и C, и мы хотим вычислить произведение ABC, то стоимость будет равна количеству скалярных умножений, необходимых для вычисления каждого элемента результирующей матрицы.

Для решения этой задачи необходимо найти оптимальный порядок умножения матриц. Порядок умножения матриц влияет на общую стоимость умножения. Например, если у нас есть матрицы A, B и C, и мы хотим вычислить произведение ABC, то мы можем вычислить либо (AB)C, либо A(BC). Стоимость этих двух вычислений может различаться, и нам нужно найти оптимальный порядок, который минимизирует стоимость.

## Требования

Для решения этой задачи необходимо учесть следующие требования:

- Нам нужно только вычислить стоимость умножения матриц, а не перечислять фактический порядок операций.
- Мы не можем предполагать, что входные данные действительны, и нужно обрабатывать недействительные входные данные.
- Можно предположить, что задача помещается в память.

## Пример использования

Вот несколько примеров того, как должна работать функция:

- `None` -> `Exception`
- `[]` -> `0`
- `[Matrix(2, 3), Matrix(3, 6), Matrix(6, 4), Matrix(4, 5)]` -> `124`
