# Резюме

В этом испытании мы научились минимизировать стоимость умножения матриц, если дана последовательность матриц 2x2. Нам нужно найти оптимальный порядок умножения матриц, чтобы минимизировать стоимость. Для решения этой задачи необходимо учесть требования и обрабатывать недействительные входные данные.
