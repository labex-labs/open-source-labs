# Introduction

La multiplication de matrices est une opération fondamentale en algèbre linéaire et trouve de nombreuses applications dans divers domaines tels que la physique, l'ingénierie et l'informatique. Dans ce défi, nous nous concentrerons sur la minimisation du coût de la multiplication de matrices étant donné une liste de matrices 2x2.
