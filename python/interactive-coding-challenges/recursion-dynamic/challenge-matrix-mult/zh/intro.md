# 简介

矩阵乘法是线性代数中的一项基本运算，在物理、工程和计算机科学等各个领域都有广泛应用。在本次挑战中，我们将专注于在给定一系列 2x2 矩阵的情况下，最小化矩阵乘法的成本。
