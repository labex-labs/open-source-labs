# Introduction

En mathématiques, l'ensemble des parties d'un ensemble S est l'ensemble de tous les sous-ensembles de S, y compris l'ensemble vide et S lui-même. En Python, on peut utiliser une combinaison de boucles et de récursion pour générer l'ensemble des parties d'un ensemble donné.
