# Power Set

## Задача

Дано множество, вернуть все возможные подмножества этого множества. Подмножества должны быть уникальными, то есть если два подмножества имеют одинаковые элементы, они должны быть считаны одним и тем же подмножеством. Пустое множество также должно быть включено в качестве подмножества. Входные данные не обязательно уникальны, и мы не можем предполагать, что входные данные действительны. Однако, мы можем предположить, что задача помещается в память.

## Требования

Для генерации power set множества нам нужно удовлетворить следующие требования:

- Результат должен быть уникальным, считая подмножества с одинаковыми элементами одинаковыми.
- Пустое множество должно быть включено в качестве подмножества.
- Входные данные не обязательно уникальны.
- Мы не можем предполагать, что входные данные действительны.
- Мы можем предположить, что задача помещается в память.

## Пример использования

```txt
* None -> None
* [] -> [[]]
* ['a'] -> [[],
            ['a']]
* ['a', 'b'] -> [[],
                 ['a'],
                 ['b'],
                 ['a', 'b']]
* ['a', 'b', 'c'] -> [[],
                      ['a'],
                      ['b'],
                      ['c'],
                      ['a', 'b'],
                      ['a', 'c'],
                      ['b', 'c'],
                      ['a', 'b', 'c']]
```
