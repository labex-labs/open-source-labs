# 冪集合

## 問題

与えられた集合のすべての可能な部分集合を返す。部分集合は一意である必要があり、つまり2つの部分集合が同じ要素を持っている場合、それらは同じ部分集合として扱われる。空集合も部分集合として含める必要がある。入力は必ずしも一意ではなく、入力が有効であるとは仮定できない。ただし、問題はメモリに収まると仮定できる。

## 要件

集合の冪集合を生成するには、次の要件を満たす必要がある。

- 生成される部分集合は一意で、同じ要素を持つ部分集合は同じとして扱う。
- 空集合を部分集合として含める。
- 入力は必ずしも一意ではない。
- 入力が有効であるとは仮定できない。
- 問題はメモリに収まると仮定できる。

## 例の使用法

```txt
* None -> None
* [] -> [[]]
* ['a'] -> [[],
            ['a']]
* ['a', 'b'] -> [[],
                 ['a'],
                 ['b'],
                 ['a', 'b']]
* ['a', 'b', 'c'] -> [[],
                      ['a'],
                      ['b'],
                      ['c'],
                      ['a', 'b'],
                      ['a', 'c'],
                      ['b', 'c'],
                      ['a', 'b', 'c']]
```
