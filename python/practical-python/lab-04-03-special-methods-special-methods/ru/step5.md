# Вызов метода

Вызов метода представляет собой двухэтапный процесс.

1. Поиск: оператор `.`
2. Вызов метода: оператор `()`

```python
>>> s = stock.Stock('GOOG',100,490.10)
>>> c = s.cost  # Поиск
>>> c
<bound method Stock.cost of <Stock object at 0x590d0>>
>>> c()         # Вызов метода
49010.0
>>>
```
