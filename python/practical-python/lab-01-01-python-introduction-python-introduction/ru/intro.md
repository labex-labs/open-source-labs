# Введение

Python - это интерпретируемый высокоуровневый язык программирования. Он часто классифицируется как ["скриптовый язык"](https://en.wikipedia.org/wiki/Scripting_language) и считается похожим на такие языки, как Perl, Tcl или Ruby. Синтаксис Python в некой степени вдохновлен элементами языка программирования C.

Python был создан Гвидо ван Россумом примерно в 1990 году, который назвал его в честь комедийного коллектива "Монти Пайтон".
