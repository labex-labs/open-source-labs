# Introduction

Python est un langage de programmation interprété de haut niveau. Il est souvent classé comme un ["langage de script"](https://en.wikipedia.org/wiki/Scripting_language) et est considéré comme similaire à des langages tels que Perl, Tcl ou Ruby. La syntaxe de Python est vaguement inspirée d'éléments du langage de programmation C.

Python a été créé par Guido van Rossum vers 1990, qui l'a nommé en l'honneur de Monty Python.
