# Einführung

Python ist eine interpretierte Hochsprachen-Programmiersprache. Sie wird oft als ["Skriptsprache"](https://en.wikipedia.org/wiki/Scripting_language) klassifiziert und gilt als ähnlich zu Sprachen wie Perl, Tcl oder Ruby. Die Syntax von Python ist lose von Elementen der C-Programmierung inspiriert.

Python wurde um 1990 von Guido van Rossum geschaffen, der sie zu Ehren der Monty Python benannte.
