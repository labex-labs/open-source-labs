# Introducción

Python es un lenguaje de programación de alto nivel interpretado. A menudo se clasifica como un ["lenguaje de scripting"](https://en.wikipedia.org/wiki/Scripting_language) y se considera similar a lenguajes como Perl, Tcl o Ruby. La sintaxis de Python está vagamente inspirada en elementos de la programación en C.

Python fue creado por Guido van Rossum alrededor de 1990, quien lo nombró en honor a Monty Python.
