# はじめに

Python はインタープリタ型の高水準プログラミング言語です。これはしばしば["スクリプト言語"](https://en.wikipedia.org/wiki/Scripting_language)として分類され、Perl、Tcl、または Ruby などの言語に似ていると考えられています。Python の構文は、C 言語の要素に大まかに影響を受けています。

Python は 1990 年頃に Guido van Rossum によって作成され、Monty Python に敬意を表してその名が付けられました。
