# 소개

Python 은 인터프리터 방식의 고급 프로그래밍 언어입니다. 종종 ["스크립팅 언어 (scripting language)"](https://en.wikipedia.org/wiki/Scripting_language)로 분류되며, Perl, Tcl, 또는 Ruby 와 같은 언어와 유사하다고 여겨집니다. Python 의 문법은 C 프로그래밍의 요소에서 느슨하게 영감을 받았습니다.

Python 은 1990 년경 Guido van Rossum 에 의해 만들어졌으며, Monty Python 을 기리기 위해 이름을 지었습니다.
