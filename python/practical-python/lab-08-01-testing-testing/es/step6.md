# Ejecutando `unittest`

Para ejecutar las pruebas, convierte el c√≥digo en un script.

```python
# test_simple.py

...

if __name__ == '__main__':
    unittest.main()
```

Luego ejecuta Python en el archivo de prueba.

```bash
$ python3 test_simple.py
F.

    self.assertEqual(r, 5)

```
