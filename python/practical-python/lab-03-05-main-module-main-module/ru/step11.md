# Строка `#!`

В Unix строка `#!` может запускать скрипт как Python. Добавьте следующее в первую строку файла вашего скрипта.

```python
#!/usr/bin/env python3
#./prog.py
...
```

Для этого требуются права на выполнение.

```bash
$ chmod +x prog.py
# Затем вы можете выполнить
$./prog.py
... вывод...
```

_Примечание: Python Launcher для Windows также ищет строку `#!`, чтобы указать версию языка._
