# Основной модуль Python

В Python нет функции или метода _main_. Вместо этого есть _основной модуль_. Основной модуль - это исходный файл, который запускается первым.

```bash
$ python3 prog.py
...
```

Любой файл, который вы передаете интерпретатору при запуске, становится _основным_. Название не имеет значения.
