# Module principal de Python

Python n'a pas de fonction ou de méthode _main_. Au lieu de cela, il existe un _module principal_. Le _module principal_ est le fichier source qui s'exécute en premier.

```bash
$ python3 prog.py
...
```

Quel que soit le fichier que vous donnez à l'interpréteur au démarrage, il devient _main_. Le nom n'a pas d'importance.
