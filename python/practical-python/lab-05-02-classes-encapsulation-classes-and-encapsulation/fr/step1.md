# Public vs Private.

L'un des rôles principaux d'une classe est d'encapsuler les données et les détails d'implémentation interne d'un objet. Cependant, une classe définit également une interface _publique_ que le monde extérieur est censé utiliser pour manipuler l'objet. Cette distinction entre les détails d'implémentation et l'interface publique est importante.
