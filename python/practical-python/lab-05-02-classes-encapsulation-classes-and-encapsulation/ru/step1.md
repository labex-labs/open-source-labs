# Публичное vs Приватное

Одной из основных задач класса является инкапсуляция данных и внутренних деталей реализации объекта. Однако класс также определяет _публичный_ интерфейс, который предполагается использовать внешний мир для управления объектом. Различие между деталями реализации и публичным интерфейсом имеет важное значение.
