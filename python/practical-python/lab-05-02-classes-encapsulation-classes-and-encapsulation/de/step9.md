# Dekorator-Syntax

Die `@`-Syntax wird als "Dekoration" bezeichnet. Sie gibt einen Modifizierer an, der auf die unmittelbar folgende Funktionsdefinition angewendet wird.

```python
...
@property
def cost(self):
    return self.shares * self.price
```
