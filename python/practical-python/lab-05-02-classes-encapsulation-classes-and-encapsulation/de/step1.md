# Öffentlich vs Privat

Eine der primären Rollen einer Klasse besteht darin, Daten und interne Implementierungsdetails eines Objekts zu kapseln. Allerdings definiert eine Klasse auch eine _öffentliche_ Schnittstelle, die die Außenwelt verwenden soll, um das Objekt zu manipulieren. Diese Unterscheidung zwischen Implementierungsdetails und der öffentlichen Schnittstelle ist wichtig.
