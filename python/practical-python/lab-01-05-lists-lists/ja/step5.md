# リストのソート

リストは「その場で」ソートできます。

```python
s = [10, 1, 7, 3]
s.sort()                    # [1, 3, 7, 10]

# 逆順
s = [10, 1, 7, 3]
s.sort(reverse=True)        # [10, 7, 3, 1]

# 任意の順序付きデータにも対応
s = ['foo', 'bar','spam']
s.sort()                    # ['bar', 'foo','spam']
```

新しいリストを作成したい場合は、`sorted()` を使用します。

```python
t = sorted(s)               # s は変更されず、t にソート済みの値が格納される
```
