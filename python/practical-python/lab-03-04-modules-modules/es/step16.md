# Comentario

Cuando hayas terminado este ejercicio, deberías tener tres programas. `fileparse.py` que contiene una función general `parse_csv()`. `report.py` que produce un informe bonito, pero también contiene las funciones `read_portfolio()` y `read_prices()`. Y finalmente, `pcost.py` que calcula el costo de la cartera, pero utiliza la función `read_portfolio()` escrita para el programa `report.py`.
