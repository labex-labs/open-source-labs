# 연습 문제 3.12: 라이브러리 모듈 사용

2 장에서 다음과 같은 주식 보고서를 생성하는 프로그램 `report.py`를 작성했습니다.

          Name     Shares      Price     Change
    ---------- ---------- ---------- ----------
            AA        100       9.22     -22.98
           IBM         50     106.28      15.18
           CAT        150      35.46     -47.98
          MSFT        200      20.89     -30.34
            GE         95      13.48     -26.89
          MSFT         50      20.89     -44.21
           IBM        100     106.28      35.84

해당 프로그램을 가져와서 모든 입력 파일 처리가 `fileparse` 모듈의 함수를 사용하여 수행되도록 수정합니다. 이를 위해 `fileparse`를 모듈로 import 하고 `read_portfolio()` 및 `read_prices()` 함수를 변경하여 `parse_csv()` 함수를 사용합니다.

이 연습의 시작 부분에 있는 대화형 예제를 가이드로 사용하세요. 그런 다음, 이전과 정확히 동일한 출력을 얻어야 합니다.
