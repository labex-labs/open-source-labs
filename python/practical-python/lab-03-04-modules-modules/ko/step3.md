# 전역 정의 (Global Definitions)

_전역_ 범위에서 정의된 모든 것은 모듈 네임스페이스를 채웁니다. 동일한 변수 `x`를 정의하는 두 개의 모듈을 생각해 봅시다.

```python
# foo.py
x = 42
def grok(a):
    ...
```

```python
# bar.py
x = 37
def spam(a):
    ...
```

이 경우, `x` 정의는 서로 다른 변수를 참조합니다. 하나는 `foo.x`이고 다른 하나는 `bar.x`입니다. 서로 다른 모듈은 동일한 이름을 사용할 수 있으며, 해당 이름들은 서로 충돌하지 않습니다.

**모듈은 격리되어 있습니다.**
