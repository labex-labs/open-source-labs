# Модули в качестве окружений

Модули образуют окружающее окружение для всего кода, определенного внутри них.

```python
# foo.py
x = 42

def grok(a):
    print(x)
```

_Глобальные_ переменные всегда связываются с окружающим модулем (одним и тем же файлом). Каждый исходный файл представляет собой свою маленькую вселенную.
