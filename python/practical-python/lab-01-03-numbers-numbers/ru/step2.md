# Булевы значения (bool)

Булевы значения имеют два значения: `True` (истина), `False` (ложь).

```python
a = True
b = False
```

Числово, они оцениваются как целые числа со значением `1`, `0`.

```python
c = 4 + True # 5
d = False
if d == 0:
    print('d is False')
```

_Но не пишите код так. Это будет странно._
