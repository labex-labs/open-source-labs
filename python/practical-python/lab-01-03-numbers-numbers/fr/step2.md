# Booléens (bool)

Les booléens ont deux valeurs : `True` (vrai) et `False` (faux).

```python
a = True
b = False
```

Numériquement, ils sont évalués comme des entiers avec les valeurs `1` et `0`.

```python
c = 4 + True # 5
d = False
if d == 0:
    print('d est Faux')
```

_Mais n'écrivez pas votre code de cette manière. Cela serait étrange._
