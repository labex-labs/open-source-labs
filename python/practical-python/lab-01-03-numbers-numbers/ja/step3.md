# 整数型 (int)

任意のサイズと基数の符号付き値：

```python
a = 37
b = -299392993727716627377128481812241231
c = 0x7fa8      # 16 進数
d = 0o253       # 8 進数
e = 0b10001111  # 2 進数
```

一般的な演算：

    x + y      加算
    x - y      減算
    x * y      乗算
    x / y      除算 (浮動小数点数を生成)
    x // y     切り捨て除算 (整数を生成)
    x % y      剰余
    x ** y     累乗
    x << n     ビット左シフト
    x >> n     ビット右シフト
    x & y      ビット単位 AND
    x | y      ビット単位 OR
    x ^ y      ビット単位 XOR
    ~x         ビット単位 NOT
    abs(x)     絶対値
