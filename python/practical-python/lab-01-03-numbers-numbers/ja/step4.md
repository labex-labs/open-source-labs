# 浮動小数点数型 (float)

浮動小数点数値を指定するには、10 進数または指数表記を使用します。

```python
a = 37.45
b = 4e5 # 4 x 10**5 または 400,000
c = -1.345e-10
```

浮動小数点数は、ネイティブ CPU 表現 [IEEE 754](https://en.wikipedia.org/wiki/IEEE_754) を使用して倍精度で表されます。これは、プログラミング言語 C の `double` 型と同じです。

> 17 桁の精度\
> 指数は -308 から 308

小数を表す際には、浮動小数点数が不正確であることに注意してください。

```python
>>> a = 2.1 + 4.2
>>> a == 6.3
False
>>> a
6.300000000000001
>>>
```

これは **Python の問題ではなく**、CPU の根底にある浮動小数点数ハードウェアの問題です。

一般的な演算：

    x + y      加算
    x - y      減算
    x * y      乗算
    x / y      除算
    x // y     切り捨て除算
    x % y      剰余
    x ** y     累乗
    abs(x)     絶対値

これらは、ビット単位演算子を除いて、整数型と同じ演算子です。追加の数学関数は、`math` モジュールにあります。

```python
import math
a = math.sqrt(x)
b = math.sin(x)
c = math.cos(x)
d = math.tan(x)
e = math.log(x)
```
