# 숫자 변환 (Converting Numbers)

유형 이름을 사용하여 값을 변환할 수 있습니다:

```python
a = int(x)    # x 를 정수로 변환
b = float(x)  # x 를 부동 소수점으로 변환
```

시도해 보세요.

```python
>>> a = 3.14159
>>> int(a)
3
>>> b = '3.14159' # 숫자 문자열도 작동합니다.
>>> float(b)
3.14159
>>>
```
