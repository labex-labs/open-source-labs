# Booleans (bool)

Booleans haben zwei Werte: `True` und `False`.

```python
a = True
b = False
```

Numerisch werden sie als Ganzzahlen mit den Werten `1` und `0` ausgewertet.

```python
c = 4 + True # 5
d = False
if d == 0:
    print('d ist False')
```
