# Convertir Números

El nombre del tipo se puede utilizar para convertir valores:

```python
a = int(x)    # Convierte x a entero
b = float(x)  # Convierte x a flotante
```

Prueba.

```python
>>> a = 3.14159
>>> int(a)
3
>>> b = '3.14159' # También funciona con cadenas que contienen números
>>> float(b)
3.14159
>>>
```
