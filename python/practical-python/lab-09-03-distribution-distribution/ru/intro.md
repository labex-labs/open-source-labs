# Введение

В какой-то момент вам может захотеться передать свой код кому-то другому, возможно, просто коллеге. В этом разделе рассказывается о самых базовых методах этого. Для более подробной информации вам нужно будет обратиться к [Руководству пользователя по упаковке Python](https://packaging.python.org).
