# Tuplas vs. Listas

Las tuplas se parecen a listas de solo lectura. Sin embargo, las tuplas se utilizan con más frecuencia para un _único elemento_ que consta de múltiples partes. Las listas suelen ser una colección de elementos distintos, por lo general todos del mismo tipo.

```python
record = ('GOOG', 100, 490.1)       # Una tupla que representa un registro en un portafolio

symbols = [ 'GOOG', 'AAPL', 'IBM' ]  # Una Lista que representa tres símbolos de acciones
```
