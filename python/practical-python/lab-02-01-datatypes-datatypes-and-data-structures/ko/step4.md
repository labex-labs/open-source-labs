# 튜플 (Tuples)

튜플은 함께 그룹화된 값들의 모음입니다.

예시:

```python
s = ('GOOG', 100, 490.1)
```

때로는 구문에서 `()`가 생략됩니다.

```python
s = 'GOOG', 100, 490.1
```

특수한 경우 (0-튜플, 1-튜플).

```python
t = ()            # 빈 튜플
w = ('GOOG', )    # 1-항목 튜플
```

튜플은 종종 _단순한_ 레코드 또는 구조를 나타내는 데 사용됩니다. 일반적으로 여러 부분으로 구성된 단일 *객체 (object)*입니다. 좋은 비유는 다음과 같습니다: _튜플은 데이터베이스 테이블의 단일 행과 같습니다._

튜플 내용은 정렬됩니다 (배열과 유사).

```python
s = ('GOOG', 100, 490.1)
name = s[0]                 # 'GOOG'
shares = s[1]               # 100
price = s[2]                # 490.1
```

하지만 내용은 수정할 수 없습니다.

```python
>>> s[1] = 75
TypeError: object does not support item assignment
```

하지만 현재 튜플을 기반으로 새로운 튜플을 만들 수 있습니다.

```python
s = (s[0], 75, s[2])
```
