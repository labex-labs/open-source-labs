# Tuplas vs. Listas (Tuples vs. Lists)

Tuplas se parecem com listas somente leitura. No entanto, tuplas são frequentemente usadas para um _único item_ consistindo em múltiplas partes. Listas são geralmente uma coleção de itens distintos, geralmente todos do mesmo tipo.

```python
record = ('GOOG', 100, 490.1)       # Uma tupla representando um registro em um portfólio

symbols = [ 'GOOG', 'AAPL', 'IBM' ]  # Uma Lista representando três símbolos de ações
```
