# Кортежи по сравнению с списками

Кортежи похожи на неизменяемые списки. Однако, кортежи чаще всего используются для _единого элемента_, состоящего из нескольких частей. Списки обычно представляют собой коллекцию различных элементов, обычно всех одного типа.

```python
record = ('GOOG', 100, 490.1)       # Кортеж, представляющий запись в портфеле

symbols = [ 'GOOG', 'AAPL', 'IBM' ]  # Список, представляющий три символа акций
```
