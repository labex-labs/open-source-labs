# Упражнение 2.3: Некоторые дополнительные операции со словарями

Если вы преобразуете словарь в список, вы получите все его ключи:

```python
>>> list(d)
['name','shares', 'price', 'date', 'account']
>>>
```

Аналогично, если вы используете инструкцию `for` для итерации по словарю, вы получите ключи:

```python
>>> for k in d:
        print('k =', k)

k = name
k = shares
k = price
k = date
k = account
>>>
```

Попробуйте этот вариант, который выполняет поиск одновременно:

```python
>>> for k in d:
        print(k, '=', d[k])

name = AA
shares = 75
price = 32.2
date = (6, 11, 2007)
account = 12345
>>>
```

Вы также можете получить все ключи с помощью метода `keys()`:

```python
>>> keys = d.keys()
>>> keys
dict_keys(['name','shares', 'price', 'date', 'account'])
>>>
```

Метод `keys()` немного необычен тем, что возвращает специальный объект `dict_keys`.

Это оболочка над исходным словарем, которая всегда показывает вам текущие ключи - даже если словарь изменяется. Например, попробуйте это:

```python
>>> del d['account']
>>> keys
dict_keys(['name','shares', 'price', 'date'])
>>>
```

Обратите внимание, что ключ `'account'` исчез из `keys`, хотя вы не вызывали `d.keys()` снова.

Более элегантный способ работать с ключами и значениями одновременно - использовать метод `items()`. Это возвращает кортежи `(ключ, значение)`:

```python
>>> items = d.items()
>>> items
dict_items([('name', 'AA'), ('shares', 75), ('price', 32.2), ('date', (6, 11, 2007))])
>>> for k, v in d.items():
        print(k, '=', v)

name = AA
shares = 75
price = 32.2
date = (6, 11, 2007)
>>>
```

Если у вас есть кортежи, такие как `items`, вы можете создать словарь с использованием функции `dict()`. Попробуйте:

```python
>>> items
dict_items([('name', 'AA'), ('shares', 75), ('price', 32.2), ('date', (6, 11, 2007))])
>>> d = dict(items)
>>> d
{'name': 'AA','shares': 75, 'price':32.2, 'date': (6, 11, 2007)}
>>>
```
