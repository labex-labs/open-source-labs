# Структуры данных

Реальные программы работают с более сложными данными. Например, информация о портфеле акций:

```code
100 shares of GOOG at $490.10
```

Это "объект", состоящий из трех частей:

- Название или символ акции ("GOOG", строка)
- Количество акций (100, целое число)
- Цена (490,10, число с плавающей точкой)
