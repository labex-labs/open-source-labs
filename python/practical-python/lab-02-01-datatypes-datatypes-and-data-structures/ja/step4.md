# タプル

タプルは、一緒にグループ化された値のコレクションです。

例：

```python
s = ('GOOG', 100, 490.1)
```

時には、構文で `()` を省略します。

```python
s = 'GOOG', 100, 490.1
```

特殊なケース（0 要素タプル、1 要素タプル）。

```python
t = ()            # 空のタプル
w = ('GOOG', )    # 1 つの要素を持つタプル
```

タプルは、_単純な_ レコードや構造を表すためによく使用されます。通常、複数の部分から構成される単一の _オブジェクト_ です。良い例え：_タプルは、データベーステーブルの 1 行に似ています。_

タプルの内容は順序付けられています（配列のように）。

```python
s = ('GOOG', 100, 490.1)
name = s[0]                 # 'GOOG'
shares = s[1]               # 100
price = s[2]                # 490.1
```

ただし、内容を変更することはできません。

```python
>>> s[1] = 75
TypeError: object does not support item assignment
```

ただし、現在のタプルに基づいて新しいタプルを作成することはできます。

```python
s = (s[0], 75, s[2])
```
