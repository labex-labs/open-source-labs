# 演習2.2: データ構造としての辞書

タプルの代替として、代わりに辞書を作成することができます。

```python
>>> d = {
        'name' : row[0],
       'shares' : int(row[1]),
        'price'  : float(row[2])
    }
>>> d
{'name': 'AA','shares': 100, 'price': 32.2 }
>>>
```

この保有株の総コストを計算します。

```python
>>> cost = d['shares'] * d['price']
>>> cost
3220.0000000000005
>>>
```

この例を、上でタプルを使った同じ計算と比較してみましょう。株数を75に変更します。

```python
>>> d['shares'] = 75
>>> d
{'name': 'AA','shares': 75, 'price': 32.2 }
>>>
```

タプルとは異なり、辞書は自由に変更することができます。いくつかの属性を追加しましょう。

```python
>>> d['date'] = (6, 11, 2007)
>>> d['account'] = 12345
>>> d
{'name': 'AA','shares': 75, 'price':32.2, 'date': (6, 11, 2007), 'account': 12345}
>>>
```
