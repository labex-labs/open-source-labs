# Упражнение 7.1: Простой пример с переменными аргументами

Попробуйте определить следующую функцию:

```python
>>> def avg(x,*more):
        return float(x+sum(more))/(1+len(more))

>>> avg(10,11)
10.5
>>> avg(3,4,5)
4.0
>>> avg(1,2,3,4,5,6)
3.5
>>>
```

Заметьте, как параметр `*more` собирает все дополнительные аргументы.
