# Позиционные переменные аргументы (\*args)

Функция, которая принимает _любое количество_ аргументов, считается, что использует переменные аргументы. Например:

```python
def f(x, *args):
  ...
```

Вызов функции.

```python
f(1,2,3,4,5)
```

Дополнительные аргументы передаются в виде кортежа.

```python
def f(x, *args):
    # x -> 1
    # args -> (2,3,4,5)
```
