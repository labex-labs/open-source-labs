# パッケージとモジュールの違い

より大きなコードのコレクションの場合、モジュールをパッケージに整理するのが一般的です。

```code
# これから
pcost.py
report.py
fileparse.py

# これにする
porty/
    __init__.py
    pcost.py
    report.py
    fileparse.py
```

名前を選んでトップレベルのディレクトリを作成します。上の例では `porty` です（明らかにこの名前を選ぶことが一番重要な最初のステップです）。

ディレクトリに `__init__.py` ファイルを追加します。これは空でもかまいません。

ソースファイルをディレクトリに配置します。
