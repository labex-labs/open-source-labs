# Comentario: ¿No deberíamos estar usando Pandas para esto?

Los científicos de datos rápidamente señalan que bibliotecas como [Pandas](https://pandas.pydata.org) ya tienen una función para leer archivos CSV. Esto es cierto, y funciona bastante bien. Sin embargo, este no es un curso sobre el aprendizaje de Pandas. La lectura de archivos es un problema más general que los detalles específicos de los archivos CSV. La principal razón por la que estamos trabajando con un archivo CSV es que es un formato familiar para la mayoría de los programadores y es relativamente fácil de trabajar directamente, lo que ilustra muchas características de Python en el proceso. Entonces, sin duda, utiliza Pandas cuando vuelvas a trabajar. Sin embargo, durante el resto de este curso, vamos a seguir con la funcionalidad estándar de Python.
