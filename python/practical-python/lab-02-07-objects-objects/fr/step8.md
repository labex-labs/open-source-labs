# Noms, valeurs, types

Les noms de variables n'ont pas de _type_. C'est seulement un nom. Cependant, les valeurs _ont_ un type sous-jacent.

```python
>>> a = 42
>>> b = 'Hello World'
>>> type(a)
<type 'int'>
>>> type(b)
<type 'str'>
```

`type()` vous dira de quel type il s'agit. Le nom de type est généralement utilisé comme une fonction qui crée ou convertit une valeur en ce type.
