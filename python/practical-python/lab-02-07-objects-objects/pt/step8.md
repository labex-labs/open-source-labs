# Nomes, Valores, Tipos

Nomes de variáveis não têm um _tipo_. É apenas um nome. No entanto, os valores _têm_ um tipo subjacente.

```python
>>> a = 42
>>> b = 'Hello World'
>>> type(a)
<type 'int'>
>>> type(b)
<type 'str'>
```

`type()` irá dizer o que é. O nome do tipo é geralmente usado como uma função que cria ou converte um valor para esse tipo.
