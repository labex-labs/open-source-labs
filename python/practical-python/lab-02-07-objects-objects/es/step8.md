# Nombres, valores, tipos

Los nombres de variables no tienen un _tipo_. Es solo un nombre. Sin embargo, los valores _sí_ tienen un tipo subyacente.

```python
>>> a = 42
>>> b = 'Hello World'
>>> type(a)
<type 'int'>
>>> type(b)
<type 'str'>
```

`type()` te dirá de qué tipo es. El nombre del tipo se suele utilizar como una función que crea o convierte un valor en ese tipo.
