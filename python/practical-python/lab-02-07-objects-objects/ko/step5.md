# 동일성 (Identity) 과 참조 (References)

두 값이 정확히 동일한 객체인지 확인하려면 `is` 연산자를 사용하십시오.

```python
>>> a = [1,2,3]
>>> b = a
>>> a is b
True
>>>
```

`is`는 객체 동일성 (정수) 을 비교합니다. 동일성은 `id()`를 사용하여 얻을 수 있습니다.

```python
>>> id(a)
3588944
>>> id(b)
3588944
>>>
```

참고: 객체를 확인할 때는 거의 항상 `==`를 사용하는 것이 좋습니다. `is`의 동작은 종종 예상치 못한 결과를 보입니다.

```python
>>> a = [1,2,3]
>>> b = a
>>> c = [1,2,3]
>>> a is b
True
>>> a is c
False
>>> a == c
True
>>>
```
