# Имена, значения, типы

Имена переменных не имеют _типа_. Это всего лишь имя. Однако значения _имеют_ базовый тип.

```python
>>> a = 42
>>> b = 'Hello World'
>>> type(a)
<type 'int'>
>>> type(b)
<type 'str'>
```

`type()` покажет вам, что это за значение. Имя типа обычно используется в качестве функции, которая создает или преобразует значение в этот тип.
