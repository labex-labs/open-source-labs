# Funciones de biblioteca

Python viene con una gran biblioteca estándar. Los módulos de biblioteca se acceden utilizando `import`. Por ejemplo:

```python
import math
x = math.sqrt(10)

import urllib.request
u = urllib.request.urlopen('http://www.python.org/')
data = u.read()
```
