# Ejercicio 1.29: Definiendo una función

Intente definir una función simple:

```python
>>> def greeting(name):
        'Issues a greeting'
        print('Hello', name)

>>> greeting('Guido')
Hello Guido
>>> greeting('Paula')
Hello Paula
>>>
```

Si la primera declaración de una función es una cadena de texto, sirve como documentación. Intente escribir un comando como `help(greeting)` para verla mostrada.
