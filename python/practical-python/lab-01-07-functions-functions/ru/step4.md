# Захват и обработка исключений

Исключения можно ловить и обрабатывать.

Для захвата используйте оператор `try - except`.

```python
for line in file:
    fields = line.split(',')
    try:
        shares = int(fields[1])
    except ValueError:
        print("Couldn't parse", line)
  ...
```

Имя `ValueError` должно совпадать с типом ошибки, которую вы пытаетесь поймать.

Часто трудно заранее точно знать, какие виды ошибок могут возникнуть, в зависимости от выполняемой операции. Хорошо или плохо, обработка исключений часто добавляется _после_ того, как программа внезапно завершилась сбоем (то есть "о, мы забыли поймать эту ошибку. Мы должны обработать ее!").
