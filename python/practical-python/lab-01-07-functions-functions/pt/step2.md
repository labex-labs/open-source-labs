# Funções de Biblioteca

Python vem com uma grande biblioteca padrão. Módulos de biblioteca são acessados usando `import`. Por exemplo:

```python
import math
x = math.sqrt(10)

import urllib.request
u = urllib.request.urlopen('http://www.python.org/')
data = u.read()
```
