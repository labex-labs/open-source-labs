# 演習 4.2: メソッドの追加

クラスを使うと、オブジェクトに関数を付けることができます。これらはメソッドと呼ばれ、オブジェクト内に格納されたデータに対して操作を行う関数です。`Stock` オブジェクトに `cost()` と `sell()` のメソッドを追加しましょう。動作は次のようになるはずです。

```python
>>> import stock
>>> s = stock.Stock('GOOG', 100, 490.10)
>>> s.cost()
49010.0
>>> s.shares
100
>>> s.sell(25)
>>> s.shares
75
>>> s.cost()
36757.5
>>>
```
