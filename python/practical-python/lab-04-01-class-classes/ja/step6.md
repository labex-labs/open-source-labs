# クラスのスコープ

クラスは名前のスコープを定義しません。

```python
class Player:
 ...
    def move(self, dx, dy):
        self.x += dx
        self.y += dy

    def left(self, amt):
        move(-amt, 0)       # いいえ。グローバル関数 `move` を呼び出します
        self.move(-amt, 0)  # はい。上のメソッド `move` を呼び出します。
```

インスタンスで操作を行いたい場合は、常に明示的に参照します (たとえば、`self`)。
