# Instancias

Las instancias son los verdaderos _objetos_ que manipulas en tu programa.

Se crean llamando a la clase como si fuera una función.

```python
>>> a = Player(2, 3)
>>> b = Player(10, 20)
>>>
```

`a` y `b` son instancias de `Player`.

_Enfatiza: La declaración de clase es solo una definición (no hace nada por sí misma). Similar a la definición de una función._
