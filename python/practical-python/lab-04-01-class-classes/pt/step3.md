# Instâncias (Instances)

Instâncias são os _objetos_ reais que você manipula em seu programa.

Elas são criadas chamando a classe como uma função.

```python
>>> a = Player(2, 3)
>>> b = Player(10, 20)
>>>
```

`a` e `b` são instâncias de `Player`.

_Enfatizando: A declaração da classe é apenas a definição (ela não faz nada por si só). Semelhante a uma definição de função._
