# Instances

Les instances sont les vrais _objets_ que vous manipulez dans votre programme.

Elles sont créées en appelant la classe comme une fonction.

```python
>>> a = Player(2, 3)
>>> b = Player(10, 20)
>>>
```

`a` et `b` sont des instances de `Player`.

_Insistance : L'instruction class n'est que la définition (elle ne fait rien d'elle-même). Elle est similaire à la définition d'une fonction._
