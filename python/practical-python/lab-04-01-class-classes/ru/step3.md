# Экземпляры

Экземпляры - это фактические _объекты_, с которыми вы работаете в своем программе.

Они создаются путем вызова класса в качестве функции.

```python
>>> a = Player(2, 3)
>>> b = Player(10, 20)
>>>
```

`a` и `b` - это экземпляры класса `Player`.

_Подчеркните: Класс - это только определение (сам по себе он ничего не делает). Аналогично определению функции._
