# Instanzen

Instanzen sind die tatsächlichen _Objekte_, die Sie in Ihrem Programm manipulieren.

Sie werden erstellt, indem Sie die Klasse wie eine Funktion aufrufen.

```python
>>> a = Player(2, 3)
>>> b = Player(10, 20)
>>>
```

`a` und `b` sind Instanzen von `Player`.

_Betonen: Die class-Anweisung ist nur die Definition (sie macht nichts an sich). Ähnlich wie eine Funktionsdefinition._
