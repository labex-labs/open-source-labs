# Einbaut Exceptions

Es gibt etwa zwei Dutzend eingebaut Exceptions. Normalerweise ist der Name der Exception anzeigend für das, was schiefgeht (z.B. eine `ValueError` wird ausgelöst, weil Sie einen schlechten Wert angegeben haben). Dies ist keine erschöpfende Liste. Überprüfen Sie die [Dokumentation](https://docs.python.org/3/library/exceptions.html) für mehr Informationen.

```python
ArithmeticError
AssertionError
EnvironmentError
EOFError
ImportError
IndexError
KeyboardInterrupt
KeyError
MemoryError
NameError
ReferenceError
RuntimeError
SyntaxError
SystemError
TypeError
ValueError
```
