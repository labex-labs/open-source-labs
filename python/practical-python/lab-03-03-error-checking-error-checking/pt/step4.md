# Exceções Embutidas (Built-in Exceptions)

Existem cerca de duas dúzias de exceções embutidas. Normalmente, o nome da exceção é indicativo do que está errado (por exemplo, um `ValueError` é lançado porque você forneceu um valor inválido). Esta não é uma lista exaustiva. Consulte a [documentação](https://docs.python.org/3/library/exceptions.html) para mais informações.

```python
ArithmeticError
AssertionError
EnvironmentError
EOFError
ImportError
IndexError
KeyboardInterrupt
KeyError
MemoryError
NameError
ReferenceError
RuntimeError
SyntaxError
SystemError
TypeError
ValueError
```
