# Exceptions intégrées

Il existe environ une vingtaine d'exceptions intégrées. Généralement, le nom de l'exception indique ce qui ne va pas (par exemple, une `ValueError` est levée parce que vous avez fourni une mauvaise valeur). Ce n'est pas une liste exhaustive. Consultez la [documentation](https://docs.python.org/3/library/exceptions.html) pour en savoir plus.

```python
ArithmeticError
AssertionError
EnvironmentError
EOFError
ImportError
IndexError
KeyboardInterrupt
KeyError
MemoryError
NameError
ReferenceError
RuntimeError
SyntaxError
SystemError
TypeError
ValueError
```
