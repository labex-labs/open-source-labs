# Excepciones Integradas

Hay aproximadamente dos docenas de excepciones integradas. Por lo general, el nombre de la excepción indica lo que está mal (por ejemplo, se lanza una `ValueError` porque se proporcionó un valor incorrecto). Esta no es una lista exhaustiva. Consulte la [documentación](https://docs.python.org/3/library/exceptions.html) para obtener más información.

```python
ArithmeticError
AssertionError
EnvironmentError
EOFError
ImportError
IndexError
KeyboardInterrupt
KeyError
MemoryError
NameError
ReferenceError
RuntimeError
SyntaxError
SystemError
TypeError
ValueError
```
