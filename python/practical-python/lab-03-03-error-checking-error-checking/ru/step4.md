# Встроенные исключения

Есть около двух десятков встроенных исключений. Обычно название исключения говорит о том, что пошло не так (например, `ValueError` возникает, если вы подаете неправильное значение). Это не исчерпывающий список. Проверьте [документацию](https://docs.python.org/3/library/exceptions.html) для получения дополнительной информации.

```python
ArithmeticError
AssertionError
EnvironmentError
EOFError
ImportError
IndexError
KeyboardInterrupt
KeyError
MemoryError
NameError
ReferenceError
RuntimeError
SyntaxError
SystemError
TypeError
ValueError
```
