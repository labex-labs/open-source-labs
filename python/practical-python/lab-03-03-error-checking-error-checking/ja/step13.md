# `with` 文

現代のコードでは、`try - finally` はしばしば `with` 文に置き換えられます。

```python
lock = Lock()
with lock:
    # ロックが獲得されました
  ...
# ロックが解放されました
```

もっと身近な例：

```python
with open(filename) as f:
    # ファイルを使用する
  ...
# ファイルが閉じられました
```

`with` はリソースの使用 _コンテキスト_ を定義します。実行がそのコンテキストを抜けると、リソースが解放されます。`with` は、それをサポートするように特別にプログラムされた特定のオブジェクトのみと機能します。
