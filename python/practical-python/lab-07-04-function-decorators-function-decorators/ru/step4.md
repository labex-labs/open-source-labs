# Декораторы

Оборачивать функции в обертки - это чрезвычайно распространенная практика в Python. Такая распространенность привела к появлению специального синтаксиса для этого.

```python
def add(x, y):
    return x + y
add = logged(add)

# Особенный синтаксис
@logged
def add(x, y):
    return x + y
```

Особенный синтаксис выполняет те же самые шаги, что и показано выше. Декоратор - это просто новый синтаксис. Говорят, что он _декорирует_ функцию.
