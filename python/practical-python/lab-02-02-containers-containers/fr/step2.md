# Les listes en tant que conteneur

Utilisez une liste lorsque l'ordre des donn√©es est important. Rappelez-vous que les listes peuvent contenir n'importe quel type d'objet. Par exemple, une liste de tuples.

```python
portfolio = [
    ('GOOG', 100, 490.1),
    ('IBM', 50, 91.3),
    ('CAT', 150, 83.44)
]

portfolio[0]            # ('GOOG', 100, 490.1)
portfolio[2]            # ('CAT', 150, 83.44)
```
