# Композитные ключи

Почти любой тип значения может быть использован в качестве ключа словаря в Python. Ключ словаря должен быть неизменяемым типом. Например, кортежи:

```python
holidays = {
  (1, 1) : 'Новый год',
  (3, 14) : 'День Пи',
  (9, 13) : "День программиста",
}
```

Затем для доступа:

```python
>>> holidays[3, 14]
'День Пи'
>>>
```

_Ни список, ни множество, ни другой словарь не могут служить ключом словаря, потому что списки и словари являются изменяемыми._
