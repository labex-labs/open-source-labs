# Поиск в словаре

Вы можете проверить существование ключа.

```python
if key in d:
    # ДА
else:
    # НЕТ
```

Вы можете искать значение, которое может не существовать, и указать значение по умолчанию, если оно отсутствует.

```python
name = d.get(key, default)
```

Пример:

```python
>>> prices.get('IBM', 0.0)
93.37
>>> prices.get('SCOX', 0.0)
0.0
>>>
```
