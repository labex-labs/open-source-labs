# Словарь в качестве контейнера

Словари полезны, если вы хотите быстро осуществлять случайный поиск (по имени ключа). Например, словарь цен на акции:

```python
prices = {
   'GOOG': 513.25,
   'CAT': 87.22,
   'IBM': 93.37,
   'MSFT': 44.12
}
```

Вот некоторые простые поиски:

```python
>>> prices['IBM']
93.37
>>> prices['GOOG']
513.25
>>>
```
