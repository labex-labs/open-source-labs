# 文字列の表現

文字列の各文字は、内部的には整数であるいわゆる Unicode「コードポイント」として格納されます。次のエスケープシーケンスを使用して、正確なコードポイント値を指定することができます：

```python
a = '\xf1'          # a = 'ñ'
b = '\u2200'        # b = '∀'
c = '\U0001D122'    # c = '𝄢'
d = '\N{FOR ALL}'   # d = '∀'
```

[Unicode キャラクターデータベース](https://unicode.org/charts)は、利用可能なすべての文字コードのリファレンスです。
