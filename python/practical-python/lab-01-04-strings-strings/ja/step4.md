# 文字列のインデックス付け

文字列は、個々の文字にアクセスするための配列のように機能します。0 から始まる整数インデックスを使用します。負のインデックスは、文字列の末尾からの相対位置を指定します。

```python
a = 'Hello world'
b = a[0]          # 'H'
c = a[4]          # 'o'
d = a[-1]         # 'd' (文字列の末尾)
```

また、`:`を使ってインデックスの範囲を指定することで、部分文字列（サブストリング）をスライスまたは選択することもできます。

```python
d = a[:5]     # 'Hello'
e = a[6:]     # 'world'
f = a[3:8]    # 'lo wo'
g = a[-5:]    # 'world'
```

終了インデックスの文字は含まれません。欠落したインデックスは、文字列の先頭または末尾を想定します。
