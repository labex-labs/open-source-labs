# リテラルテキストの表現

文字列リテラルは、プログラム内で引用符を使って書かれます。

```python
# シングルクォート
a = 'Yeah but no but yeah but...'

# ダブルクォート
b = "computer says no"

# トリプルクォート
c = '''
Look into my eyes, look into my eyes, the eyes, the eyes, the eyes,
not around the eyes,
don't look around the eyes,
look into my eyes, you're under.
'''
```

通常、文字列は 1 行に収まります。トリプルクォートは、複数行にまたがる括られたすべてのテキスト（すべての書式設定も含む）をキャプチャします。

シングルクォート (') とダブルクォート (") を使う違いはありません。_ただし、文字列の始まりに使った同じ種類の引用符を、終わりにも使う必要があります_。
