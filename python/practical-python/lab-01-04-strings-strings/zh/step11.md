# f 字符串

一种带有格式化表达式替换的字符串。

```python
>>> name = 'IBM'
>>> shares = 100
>>> price = 91.1
>>> a = f'{name:>10s} {shares:10d} {price:10.2f}'
>>> a
'       IBM        100      91.10'
>>> b = f'Cost = ${shares*price:0.2f}'
>>> b
'Cost = $9110.00'
>>>
```

**注意：这需要 Python 3.6 或更高版本。** 格式代码的含义将在后面介绍。

在这些练习中，你将对 Python 字符串类型进行操作实验。你应该在 Python 交互式提示符下进行，这样可以轻松看到结果。重要提示：

> 在需要与解释器交互的练习中，`>>>` 是当 Python 希望你输入新语句时出现的解释器提示符。练习中的一些语句跨越多行 —— 要运行这些语句，你可能需要按几次“回车”。提醒一下，在处理这些示例时你 _不要_ 输入 `>>>`。
