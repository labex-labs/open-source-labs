# 문자열 이스케이프 코드

이스케이프 코드는 제어 문자 및 키보드에서 직접 쉽게 입력할 수 없는 문자를 나타내는 데 사용됩니다. 다음은 몇 가지 일반적인 이스케이프 코드입니다.

    '\n'      줄 바꿈 (Line feed)
    '\r'      캐리지 리턴 (Carriage return)
    '\t'      탭 (Tab)
    '\''      리터럴 작은 따옴표 (Literal single quote)
    '\"'      리터럴 큰 따옴표 (Literal double quote)
    '\\'      리터럴 백슬래시 (Literal backslash)
