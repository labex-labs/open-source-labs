# 문자열 표현

문자열의 각 문자는 내부적으로 정수인 소위 유니코드 "코드 포인트 (code-point)"로 저장됩니다. 다음 이스케이프 시퀀스를 사용하여 정확한 코드 포인트 값을 지정할 수 있습니다.

```python
a = '\xf1'          # a = 'ñ'
b = '\u2200'        # b = '∀'
c = '\U0001D122'    # c = '𝄢'
d = '\N{FOR ALL}'   # d = '∀'
```

[유니코드 문자 데이터베이스](https://unicode.org/charts)는 사용 가능한 모든 문자 코드에 대한 참조입니다.
