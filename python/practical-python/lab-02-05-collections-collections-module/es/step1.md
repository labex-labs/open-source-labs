# Ejemplo: Contando cosas

Digamos que quieres tabular las acciones totales de cada empresa.

```python
portfolio = [
    ('GOOG', 100, 490.1),
    ('IBM', 50, 91.1),
    ('CAT', 150, 83.44),
    ('IBM', 100, 45.23),
    ('GOOG', 75, 572.45),
    ('AA', 50, 23.15)
]
```

Hay dos entradas de `IBM` y dos entradas de `GOOG` en esta lista. Las acciones deben combinarse de alguna manera.
