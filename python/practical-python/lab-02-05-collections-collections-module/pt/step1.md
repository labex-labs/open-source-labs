# Exemplo: Contando Coisas

Digamos que você queira tabular o total de ações de cada ação.

```python
portfolio = [
    ('GOOG', 100, 490.1),
    ('IBM', 50, 91.1),
    ('CAT', 150, 83.44),
    ('IBM', 100, 45.23),
    ('GOOG', 75, 572.45),
    ('AA', 50, 23.15)
]
```

Existem duas entradas `IBM` e duas entradas `GOOG` nesta lista. As ações precisam ser combinadas de alguma forma.
