# Пример: подсчет количества элементов

Допустим, вы хотите составить таблицу общих акций каждой компании.

```python
portfolio = [
    ('GOOG', 100, 490.1),
    ('IBM', 50, 91.1),
    ('CAT', 150, 83.44),
    ('IBM', 100, 45.23),
    ('GOOG', 75, 572.45),
    ('AA', 50, 23.15)
]
```

В этом списке есть две записи для `IBM` и две записи для `GOOG`. Акции нужно как-то объединить.
