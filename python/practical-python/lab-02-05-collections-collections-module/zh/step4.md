# 示例：记录历史

问题：我们想要记录最近 N 件事情的历史。解决方案：使用 `deque`。

```python
from collections import deque

history = deque(maxlen=N)
with open(filename) as f:
    for line in f:
        history.append(line)
     ...
```

`collections` 模块可能是处理诸如制表和索引等特殊用途数据处理问题最有用的库模块之一。

