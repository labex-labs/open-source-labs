# Diskussion

Etwas sehr Potentes ist gerade passiert. Sie haben ein interessantes Iterationsmuster (das Lesen von Zeilen am Ende einer Datei) in seine eigene kleine Funktion verlagert. Die `follow()`-Funktion ist jetzt diese völlig allzweckdienliche Utility, die Sie in jedem Programm verwenden können. Beispielsweise könnten Sie sie verwenden, um Server-Logs, Debugging-Logs und andere ähnliche Datenquellen zu beobachten. Das ist ziemlich cool.
