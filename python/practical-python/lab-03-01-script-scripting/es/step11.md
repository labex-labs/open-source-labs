# Ejercicio 3.1: Estructurar un programa como una colección de funciones

Modifica tu programa `report.py` de modo que todas las operaciones principales, incluyendo cálculos y salida, se realicen mediante una colección de funciones. Específicamente:

- Crea una función `print_report(report)` que imprima el informe.
- Cambia la última parte del programa de modo que no sea más que una serie de llamadas a funciones y no tenga otras operaciones de cálculo.
