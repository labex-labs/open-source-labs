# ¿Qué es una función?

Una función es una secuencia nombrada de instrucciones.

```python
def funcname(args):
  instrucción
  instrucción
 ...
  return resultado
```

_Cualquier_ instrucción de Python se puede utilizar dentro.

```python
def foo():
    import math
    print(math.sqrt(2))
    help(math)
```

No hay instrucciones _especiales_ en Python (lo que la hace fácil de recordar).
