# Qu'est-ce qu'une fonction?

Une fonction est une séquence nommée d'instructions.

```python
def nom_de_la_fonction(arguments):
  instruction
  instruction
 ...
  return résultat
```

_N'importe quelle_ instruction Python peut être utilisée à l'intérieur.

```python
def foo():
    import math
    print(math.sqrt(2))
    help(math)
```

Il n'y a pas d'_instruction spéciale_ en Python (ce qui en facilite la mémorisation).
