# 下から上のスタイル

関数は構築ブロックとして扱われます。小さくて単純なブロックが先に来ます。

```python
# myprogram.py
def foo(x):
 ...

def bar(x):
 ...
    foo(x)          # 上で定義されています
 ...

def spam(x):
 ...
    bar(x)          # 上で定義されています
 ...

spam(42)            # 関数を使用するコードは最後に現れます
```

後の関数は前の関数をベースに構築されます。再び、これはただのスタイルの問題です。上記のプログラムで重要なのは、`spam(42)` の呼び出しが最後に来ることだけです。
