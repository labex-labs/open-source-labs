# Diccionarios, revisados

Recuerda que un diccionario es una colección de valores con nombre.

```python
stock = {
    'name' : 'GOOG',
   'shares' : 100,
    'price' : 490.1
}
```

Los diccionarios se utilizan comúnmente para estructuras de datos simples. Sin embargo, se utilizan para partes críticas del intérprete y pueden ser el _tipo de datos más importante en Python_.
