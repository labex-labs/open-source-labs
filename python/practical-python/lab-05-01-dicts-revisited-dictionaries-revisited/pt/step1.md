# Dicionários, Revisados

Lembre-se que um dicionário é uma coleção de valores nomeados.

```python
stock = {
    'name' : 'GOOG',
    'shares' : 100,
    'price' : 490.1
}
```

Dicionários são comumente usados para estruturas de dados simples. No entanto, eles são usados para partes críticas do interpretador e podem ser o _tipo de dado mais importante em Python_.
