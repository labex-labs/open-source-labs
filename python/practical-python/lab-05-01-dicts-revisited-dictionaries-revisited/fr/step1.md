# Dictionnaires, revisités

Rappelez-vous qu'un dictionnaire est une collection de valeurs nommées.

```python
stock = {
    'name' : 'GOOG',
   'shares' : 100,
    'price' : 490.1
}
```

Les dictionnaires sont couramment utilisés pour les structures de données simples. Cependant, ils sont utilisés pour des parties critiques de l'interpréteur et peuvent être le _type de données le plus important en Python_.
