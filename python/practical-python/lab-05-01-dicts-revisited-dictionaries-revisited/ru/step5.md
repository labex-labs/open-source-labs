# Экземпляры и классы

Экземпляры и классы связаны между собой. Атрибут `__class__` ссылается обратно на класс.

```python
>>> s = Stock('GOOG', 100, 490.1)
>>> s.__dict__
{ 'name': 'GOOG','shares': 100, 'price': 490.1 }
>>> s.__class__
<class '__main__.Stock'>
>>>
```

Словарь экземпляра хранит данные, уникальные для каждого экземпляра, в то время как словарь класса хранит данные, общими для _всех_ экземпляров.
