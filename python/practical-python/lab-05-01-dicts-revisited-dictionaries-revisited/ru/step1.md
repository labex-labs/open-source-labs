# Словарь, снова

Помните, что словарь - это коллекция именованных значений.

```python
stock = {
    'name' : 'GOOG',
   'shares' : 100,
    'price' : 490.1
}
```

Словари обычно используются для простых структур данных. Однако, они используются для важнейших частей интерпретатора и могут быть _самым важным типом данных в Python_.
