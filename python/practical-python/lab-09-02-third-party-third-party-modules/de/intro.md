# Einführung

Python verfügt über eine umfangreiche Bibliothek eingebauter Module (_batteries included_).

Es gibt noch viele weitere Drittanbieter-Module. Prüfen Sie sie im [Python Package Index](https://pypi.org/) oder PyPi. Oder führen Sie einfach eine Google-Suche zu einem bestimmten Thema durch.

Das Handling von Drittanbieterabhängigkeiten ist ein ständig sich entwickelndes Thema bei Python. Dieser Abschnitt behandelt lediglich die Grundlagen, um Ihnen zu helfen, die Funktionsweise zu verstehen.
