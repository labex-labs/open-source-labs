# Передача аргументов

Когда вы вызываете функцию, переменные аргументов - это имена, которые ссылаются на переданные значения. Эти значения НЕ являются копиями. Если передаются изменяемые типы данных (например, списки, словари), они могут быть изменены _на месте_.

```python
def foo(items):
    items.append(42)    # Изменяет входной объект

a = [1, 2, 3]
foo(a)
print(a)                # [1, 2, 3, 42]
```

**Основной момент: Функции не получают копию переданных аргументов.**
