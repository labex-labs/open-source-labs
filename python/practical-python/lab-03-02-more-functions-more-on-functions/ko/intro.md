# 소개

앞서 함수에 대한 소개가 있었지만, 실제로 함수가 더 깊은 수준에서 어떻게 작동하는지에 대한 세부 사항은 거의 제공되지 않았습니다. 이 섹션에서는 몇 가지 공백을 채우고 호출 규약 (calling conventions), 스코핑 규칙 (scoping rules) 등과 같은 문제들을 논의하는 것을 목표로 합니다.
