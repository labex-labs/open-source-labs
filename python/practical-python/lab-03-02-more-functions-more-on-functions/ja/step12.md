# 再代入と変更の違い

値を変更することと変数名を再代入することの微妙な違いを理解しておくことを確認しましょう。

```python
def foo(items):
    items.append(42)    # 入力オブジェクトを変更

a = [1, 2, 3]
foo(a)
print(a)                # [1, 2, 3, 42]

# VS
def bar(items):
    items = [4,5,6]    # ローカルの `items` 変数を別のオブジェクトに参照させるように変更

b = [1, 2, 3]
bar(b)
print(b)                # [1, 2, 3]
```

_メモ：変数の代入は決してメモリを上書きしません。名前は単に新しい値にバインドされるだけです。_
