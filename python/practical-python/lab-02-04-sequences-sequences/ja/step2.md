# スライシング

スライシングとは、シーケンスから部分シーケンスを取り出すことを意味します。構文は `s[start:end]` です。ここで、`start` と `end` は、取得したい部分シーケンスのインデックスです。

```python
a = [0,1,2,3,4,5,6,7,8]

a[2:5]    # [2,3,4]
a[-5:]    # [4,5,6,7,8]
a[:3]     # [0,1,2]
```

- インデックス `start` と `end` は整数でなければなりません。
- スライスは、終了値を含みません。数学の半開区間のようなものです。
- インデックスが省略された場合、それぞれリストの先頭または末尾がデフォルト値となります。
