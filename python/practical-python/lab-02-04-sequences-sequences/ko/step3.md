# 슬라이스 재할당 (Slice re-assignment)

리스트에서는 슬라이스를 재할당하고 삭제할 수 있습니다.

```python
# Reassignment
a = [0,1,2,3,4,5,6,7,8]
a[2:4] = [10,11,12]       # [0,1,10,11,12,4,5,6,7,8]
```

_참고: 재할당된 슬라이스는 같은 길이를 가질 필요가 없습니다._

```python
# Deletion
a = [0,1,2,3,4,5,6,7,8]
del a[2:4]                # [0,1,4,5,6,7,8]
```
