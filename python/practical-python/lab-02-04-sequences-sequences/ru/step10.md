# For и кортежи

Вы можете выполнять итерацию с несколькими переменными итерации.

```python
points = [
  (1, 4),(10, 40),(23, 14),(5, 6),(7, 8)
]
for x, y in points:
    # Цикл с x = 1, y = 4
    #            x = 10, y = 40
    #            x = 23, y = 14
    #           ...
```

При использовании нескольких переменных каждый кортеж _распаковывается_ в набор переменных итерации. Количество переменных должно соответствовать количеству элементов в каждом кортеже.
