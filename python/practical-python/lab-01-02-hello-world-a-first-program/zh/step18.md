# 练习 1.5：弹跳球

一个橡胶球从 100 米的高度落下，每次撞击地面时，它会反弹至下落高度的 3/5。编写一个名为 `bounce.py` 的程序，打印出显示前 10 次弹跳高度的表格。

以下是一个解决方案：

```python
# bounce.py

height = 100
bounce = 1
while bounce <= 10:
    height = height * (3 / 5)
    print(bounce, round(height, 4))
    bounce += 1
```
