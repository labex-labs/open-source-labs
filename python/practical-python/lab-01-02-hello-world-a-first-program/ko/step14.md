# 조건문 (Conditionals)

`if` 문은 조건부 실행에 사용됩니다.

```python
if a > b:
    print('Computer says no')
else:
    print('Computer says yes')
```

`elif`를 사용하여 추가적인 검사를 추가하여 여러 조건을 확인할 수 있습니다.

```python
if a > b:
    print('Computer says no')
elif a == b:
    print('Computer says yes')
else:
    print('Computer says maybe')
```
