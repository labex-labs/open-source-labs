# Чувствительность к регистру

Python чувствителен к регистру. Большие и маленькие буквы считаются разными символами. Все эти переменные разные:

```python
name = 'Jake'
Name = 'Elwood'
NAME = 'Guido'
```

Инструкции языка всегда пишутся с маленькой буквы.

```python
while x < 0:   # OK
WHILE x < 0:   # Ошибка
```
