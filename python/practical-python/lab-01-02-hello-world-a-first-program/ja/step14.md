# 条件分岐

`if` 文は条件分岐を実行するために使用されます。

```python
if a > b:
    print('Computer says no')
else:
    print('Computer says yes')
```

`elif` を使って追加のチェックを行うことで、複数の条件をチェックすることができます。

```python
if a > b:
    print('Computer says no')
elif a == b:
    print('Computer says yes')
else:
    print('Computer says maybe')
```
