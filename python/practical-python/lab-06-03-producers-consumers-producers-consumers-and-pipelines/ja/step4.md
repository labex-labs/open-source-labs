# 演習6.9: より複雑なパイプラインの設定

パイプラインの考え方をさらに進めて、より多くの操作を行いましょう。

```python
>>> from follow import follow
>>> import csv
>>> lines = follow('stocklog.csv')
>>> rows = csv.reader(lines)
>>> for row in rows:
        print(row)
```
