# Отношение "является"

Наследование устанавливает отношение типов.

```python
class Shape:
...

class Circle(Shape):
...
```

Проверьте объект на принадлежность к классу.

```python
>>> c = Circle(4.0)
>>> isinstance(c, Shape)
True
>>>
```

_Важно: По идее, любой код, работающий с экземплярами родительского класса, также должен работать с экземплярами дочернего класса._
