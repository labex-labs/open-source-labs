# Einführung

Dieser Abschnitt ist ein kleiner Ausflug, aber wenn Sie mit Daten arbeiten, möchten Sie oft strukturierte Ausgabe (Tabellen usw.) erzeugen. Beispielsweise:

```code
      Name      Anteile        Preis
----------  ----------  -----------
        AA         100        32,20
       IBM          50        91,10
       CAT         150        83,44
      MSFT         200        51,23
        GE          95        40,37
      MSFT          50        65,10
       IBM         100        70,44
```
