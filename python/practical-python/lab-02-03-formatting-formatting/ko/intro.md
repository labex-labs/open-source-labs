# 소개

이 섹션은 약간의 여담이지만, 데이터를 다룰 때 종종 구조화된 출력 (표 등) 을 생성하고 싶을 것입니다. 예를 들어 다음과 같습니다.

```code
      Name      Shares        Price
----------  ----------  -----------
        AA         100        32.20
       IBM          50        91.10
       CAT         150        83.44
      MSFT         200        51.23
        GE          95        40.37
      MSFT          50        65.10
       IBM         100        70.44
```
