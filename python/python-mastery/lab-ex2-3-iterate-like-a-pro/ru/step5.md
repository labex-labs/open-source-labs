# Генераторные выражения

Генераторное выражение практически идентично списочному включению, за исключением того, что оно не создает список. Вместо этого оно создает объект, который последовательно генерирует результаты - обычно для использования при итерации. Попробуйте простой пример:

```python
>>> nums = [1,2,3,4,5]
>>> squares = (x*x for x in nums)
>>> squares
<generator object <genexpr> at 0x37caa8>
>>> for n in squares:
        print(n)

1
4
9
16
25
>>>
```

Вы заметите, что генераторное выражение можно использовать только один раз. Посмотрите, что произойдет, если выполнить цикл `for` снова:

```python
>>> for n in squares:
         print(n)

>>>
```

Вы можете вручную получать результаты по одному, используя функцию `next()`. Попробуйте это:

```python
>>> squares = (x*x for x in nums)
>>> next(squares)
1
>>> next(squares)
4
>>> next(squares)
9
>>>
```

Продолжайте набирать `next()`, чтобы увидеть, что произойдет, когда данных больше нет.

Если задача, которую вы выполняете, более сложная, вы по-прежнему можете воспользоваться генераторами, написав генераторную функцию и используя оператор `yield`. Например:

```python
>>> def squares(nums):
        for x in nums:
            yield x*x

>>> for n in squares(nums):
        print(n)

1
4
9
16
25
>>>
```

Мы вернемся к генераторным функциям чуть позже в курсе - на данный момент просто воспринимайте такие функции как функции, обладающие интересным свойством подачи значений в инструкцию `for`.
