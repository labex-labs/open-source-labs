# Введение

**Цели:**

- Выяснить наиболее экономичный способ хранения большого количества данных в памяти.
- Изучить различные способы представления записей, включая кортежи, словари, классы и именованные кортежи.

В этом упражнении мы рассмотрим разные варианты представления структур данных с точки зрения использования памяти и эффективности. Многие люди используют Python для выполнения различных видов анализа данных, поэтому знание различных вариантов и их компромиссов представляет полезную информацию.
