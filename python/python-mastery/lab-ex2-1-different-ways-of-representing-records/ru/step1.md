# Затрудняемся в автобусе

Файл `ctabus.csv` - это CSV-файл, содержащий ежедневные данные о пассажиропотоке системы общественного транспорта Чикаго (CTA) с 1 января 2001 года по 31 августа 2013 года. Он содержит примерно 577000 строк данных. Используйте Python, чтобы просмотреть несколько строк данных и понять, как они выглядят:

```python
>>> f = open('/home/labex/project/ctabus.csv')
>>> next(f)
'route,date,daytype,rides\n'
>>> next(f)
'3,01/01/2001,U,7354\n'
>>> next(f)
'4,01/01/2001,U,9288\n'
>>>
```

В данных есть 4 столбца.

- route: Столбец 0. Название маршрута автобуса.
- date: Столбец 1. Строка даты в формате MM/DD/YYYY.
- daytype: Столбец 2. Код типа дня (U=воскресенье/праздник, A=суббота, W=рабочий день)
- rides: Столбец 3. Общее количество пассажиров (целое число)

Столбец `rides` записывает общее количество людей, которые сели на автобус по данному маршруту в определенный день. Таким образом, из примера видно, что 7354 человек сели на автобус номер 3 в 1 января 2001 года.
