# 소개

이 랩에서는 Python 에서 대용량 데이터 세트를 메모리 효율적으로 저장하는 방법을 배우게 됩니다. 또한 튜플, 딕셔너리, 클래스, 명명된 튜플 (named tuple) 과 같은 다양한 레코드 표현 방법을 살펴볼 것입니다.
