# Einführung

**Ziele:**

- Ermitteln der effizientesten Art, eine große Menge an Daten zu speichern, die den geringsten Speicherbedarf hat.
- Lernen über verschiedene Möglichkeiten, um Datensätze darzustellen, einschließlich Tupel, Wörterbücher, Klassen und benannte Tupel.

In dieser Übung betrachten wir verschiedene Optionen für die Darstellung von Datenstrukturen im Hinblick auf den Speicherbedarf und die Effizienz. Viele Menschen verwenden Python, um verschiedene Arten von Datenanalyse durchzuführen. Daher ist es nützliche Information, verschiedene Optionen und deren Kompromisse zu kennen.
