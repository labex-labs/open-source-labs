# バスに滞っている

`ctabus.csv` ファイルは、2001年1月1日から2013年8月31日までのシカゴ交通局 (CTA) のバスシステムの日次乗車者数データを含むCSVファイルです。約577,000行のデータが含まれています。Pythonを使ってデータの数行を表示して、その様子を見てみましょう。

```python
>>> f = open('/home/labex/project/ctabus.csv')
>>> next(f)
'route,date,daytype,rides\n'
>>> next(f)
'3,01/01/2001,U,7354\n'
>>> next(f)
'4,01/01/2001,U,9288\n'
>>>
```

4つのデータ列があります。

- route: 列0。バス路線名。
- date: 列1。MM/DD/YYYY形式の日付文字列。
- daytype: 列2。日の種類コード (U=日曜日/祝日、A=土曜日、W=平日)
- rides: 列3。乗車者の合計数 (整数)

`rides` 列は、特定の日にその路線のバスに乗車した人の合計数を記録しています。したがって、例から言えば、2001年1月1日には7354人が3番のバスに乗車しました。
