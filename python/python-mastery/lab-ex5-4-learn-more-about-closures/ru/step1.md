# Замыкания в качестве структуры данных

Одно из возможных применений замыканий - это их использование в качестве инструмента для инкапсуляции данных. Попробуйте этот пример:

```python
def counter(value):
    def incr():
        nonlocal value
        value += 1
        return value

    def decr():
        nonlocal value
        value -= 1
        return value

    return incr, decr
```

В этом коде определены две внутренние функции, которые манипулируют значением. Попробуйте его:

```python
>>> up, down = counter(0)
>>> up()
1
>>> up()
2
>>> up()
3
>>> down()
2
>>> down()
1
>>>
```

Заметьте, что здесь не используется определение класса. Кроме того, нет глобальных переменных. Тем не менее, функции `up()` и `down()` манипулируют некоторым "за кулисами" значением. Это довольно волшебно.
