# Introdução

Neste laboratório, você aprenderá mais sobre _closures_ (fechamentos) em Python. _Closures_ são um conceito de programação poderoso que permite que funções se lembrem e acessem variáveis de seu escopo envolvente, mesmo após a função externa ter completado a execução.

Você também entenderá _closures_ como uma estrutura de dados, explorará-os como um gerador de código e descobrirá como implementar a verificação de tipos com _closures_. Este laboratório o ajudará a descobrir alguns dos aspectos mais incomuns e poderosos dos _closures_ em Python.
