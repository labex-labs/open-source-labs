# インスタンスデータの変更

上記のインスタンスの1つに新しい属性を設定してみましょう。

```python
>>> goog.date = "6/11/2007"
>>> goog.__dict__
... 出力を見てください...
>>> ibm.__dict__
... 出力を見てください...
>>>
```

上記の出力では、`goog` インスタンスに `date` という属性があるのに対し、`ibm` インスタンスにはその属性がないことに気付くでしょう。重要なことは、Pythonは属性に対して本当に何の制限も設けていないということです。たとえば、インスタンスの属性は `__init__()` メソッドで設定されたものに限定されません。

属性を設定する代わりに、新しい値を直接 `__dict__` オブジェクトに入れてみましょう。

```python
>>> goog.__dict__['time'] = '9:45am'
>>> goog.time
'9:45am'
>>>
```

ここで、インスタンスが辞書の上にある層であるという事実がはっきりとわかります。
