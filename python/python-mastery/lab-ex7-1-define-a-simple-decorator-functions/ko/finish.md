# 요약

이 랩에서는 Python 의 데코레이터에 대해 배웠습니다. 데코레이터가 무엇인지, 어떻게 작동하는지 등을 포함합니다. 또한 함수에 동작을 추가하기 위한 간단한 로깅 데코레이터를 만드는 방법을 익혔고, 타입 어노테이션을 기반으로 함수 인수를 검증하기 위한 더 복잡한 데코레이터를 구축했습니다. 또한 함수 시그니처를 분석하기 위해 `inspect` 모듈을 사용하고 함수 메타데이터를 보존하기 위해 `functools.wraps`를 사용하는 방법을 배웠습니다.

데코레이터는 더 유지 관리 가능하고 재사용 가능한 코드를 작성할 수 있게 해주는 강력한 Python 기능입니다. 로깅, 접근 제어 및 캐싱과 같은 횡단 관심사 (cross-cutting concerns) 에 대해 Python 프레임워크 및 라이브러리에서 일반적으로 사용됩니다. 이제 이러한 기술을 자체 Python 프로젝트에 적용하여 더 깔끔하고 유지 관리 가능한 코드를 만들 수 있습니다.
