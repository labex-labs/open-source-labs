# 简介

在这个实验中，你将学习如何定义一个简单的 Python 类来表示股票信息。类是面向对象编程中的一个关键概念，它能让你对现实世界的实体进行建模，并更高效地组织你的代码。

本实验的目标包括学习定义一个简单的 Python 类、理解类属性和方法、创建类实例并与之交互，以及应用格式化来显示对象信息。在这个实验过程中，你将创建 `stock.py` 文件。
