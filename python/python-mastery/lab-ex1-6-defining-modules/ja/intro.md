# はじめに

**目的:**

- モジュールの定義
- import文の使用

**注意:** このモジュールに関する演習では、Pythonを適切な環境で実行していることを確認することが非常に重要です。import文が機能しない場合、`sys.path`の値を確認する必要があります。すべてが壊れているようであれば、援助を求めてください。

この演習を始める前に、まずPythonインタプリタセッションを再起動します。IDLEを使用している場合は、シェルウィンドウをクリックして、メニューオプション「シェル \> シェルを再起動」を探します。次のようなメッセージが表示されるはずです。

```python
>>> ##################== RESTART ##################==
>>>
```

Unixを使用している場合は、単にPythonを終了してインタプリタを再起動します。
