# Introduction

**Objectifs :**

- Définir des modules
- Utiliser l'instruction `import`

**Note :** Pour cet exercice impliquant des modules, il est crucial de s'assurer que vous exécutez Python dans un environnement approprié. Vous devrez peut-être vérifier la valeur de `sys.path` si vous ne parvenez pas à faire fonctionner les instructions `import`. Demandez de l'aide si tout semble cassé.

Avant de commencer cet exercice, redémarrez d'abord votre session d'interpréteur Python. Si vous utilisez IDLE, cliquez sur la fenêtre du shell et recherchez une option de menu "Shell \> Redémarrer le shell". Vous devriez obtenir un message comme celui-ci :

```python
>>> ##################== RESTART ##################==
>>>
```

Si vous utilisez Unix, quittez simplement Python et redémarrez l'interpréteur.
