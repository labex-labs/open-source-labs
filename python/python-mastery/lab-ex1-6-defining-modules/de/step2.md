# Hauptmodul

In Ihrem Programm `pcost.py` ruft der letzte Ausdruck eine Funktion auf und druckt das Ergebnis aus. Ändern Sie das Programm so, dass dieser Schritt nur erfolgt, wenn das Programm als Hauptprogramm ausgeführt wird. Versuchen Sie jetzt, das Programm auf zwei Arten auszuführen:

Zunächst führen Sie das Programm als Hauptprogramm aus:

```bash
python3 pcost.py
44671.15
```

Als nächstes führen Sie das Programm als Bibliotheksimport aus. Sie sollten keine Ausgabe sehen.

```python
>>> import pcost
>>>
```
