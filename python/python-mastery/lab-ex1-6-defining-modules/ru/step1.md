# Использование инструкции import

В предыдущих упражнениях вы написали два программы `pcost.py` и `stock.py`. Используйте инструкцию `import` для загрузки этих программ и использования их функциональности:

```python
>>> import pcost
44671.15
>>> pcost.portfolio_cost('portfolio2.dat')
19908.75
>>> from stock import Stock
>>> s = Stock('GOOG', 100, 490.10)
>>> s.name
'GOOG'
>>> s.cost()
49010.0
>>>
```

Если вы не можете заставить работать вышеперечисленные инструкции, вы, возможно, поместили свои программы в странный каталог. Убедитесь, что вы запускаете Python в том же каталоге, что и ваши файлы, или что каталог включен в `sys.path`.
