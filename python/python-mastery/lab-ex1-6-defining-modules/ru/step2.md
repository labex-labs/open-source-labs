# Главный модуль

В вашей программе `pcost.py` последняя инструкция вызывала функцию и выводила результат на печать. Измените программу так, чтобы этот шаг выполнялся только в том случае, если программа запускается как главный исполняемый файл. Теперь попробуйте запустить программу двумя способами:

Во - первых, запустите программу как главный исполняемый файл:

```bash
python3 pcost.py
44671.15
```

Далее, запустите программу как библиотеку для импорта. Вы не должны видеть никакого вывода.

```python
>>> import pcost
>>>
```
