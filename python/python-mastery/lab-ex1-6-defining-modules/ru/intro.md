# Введение

**Цели:**

- Определение модулей
- Использование инструкции import

**Примечание:** При выполнении этого упражнения по модулям крайне важно убедиться, что вы запускаете Python в правильной среде. Если вы не можете заставить инструкции import работать, вам может потребоваться проверить значение `sys.path`. Если все выглядит сломанным, обратитесь за помощью.

Прежде чем начать это упражнение, сначала перезапустите сеанс интерпретатора Python. Если вы используете IDLE, щелкните по окну консоли и найдите пункт меню "Shell \> Restart Shell". Вы должны получить сообщение такого вида:

```python
>>> ##################== RESTART ##################==
>>>
```

Если вы используете Unix, просто выйдите из Python и перезапустите интерпретатор.
