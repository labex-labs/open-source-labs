# はじめに

**目的**：

- コルーチンを使って処理パイプラインを設定する

**作成するファイル**：`cofollow.py`、`coticker.py`

**注意**

この演習では、`stocksim.py` プログラムがバックグラウンドで実行されている必要があります。

演習8.2では、ジェネレータを使って処理パイプラインを設定するコードを書きました。そのプログラムの重要な側面は、ジェネレータ関数間を流れるデータの考え方でした。非常に似た種類のデータフローをコルーチンを使って設定することができます。唯一の違いは、コルーチンでは、forループでデータを引き出すのとは逆に、異なる処理要素にデータを送信するということです。
