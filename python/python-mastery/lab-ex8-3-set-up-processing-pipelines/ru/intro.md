# Введение

**Цели:**

- Использование корутин для настройки конвейеров обработки

**Созданные файлы:** `cofollow.py`, `coticker.py`

**Примечание**

Для данного упражнения программа `stocksim.py` должна по-прежнему работать в фоновом режиме.

В упражнении 8.2 вы написали код, который использовал генераторы для настройки конвейера обработки. Главной особенностью этой программы была идея о потоке данных между функциями-генераторами. С использованием корутин можно настроить очень похожий тип потока данных. Единственное отличие заключается в том, что при использовании корутины вы отправляете данные в разные элементы обработки, в отличие от извлечения данных с помощью цикла for.
