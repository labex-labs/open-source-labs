# Подготовка

Эта практика посвящена некоторым более сложным аспектам библиотечных модулей. Начните выполнение практики с создания очень простого библиотечного модуля:

```python
# simplemod.py

x = 42        # Глобальная переменная

# Простая функция
def foo():
    print('x is', x)

# Простой класс
class Spam:
    def yow(self):
        print('Yow!')

# Скриптовая инструкция
print('Loaded simplemod')
```
