# Загрузка модуля и системный путь

Попробуйте импортировать модуль, который вы только что создали:

```python
>>> import simplemod
Loaded simplemod
>>> simplemod.foo()
x is 42
>>>
```

Если это завершилось ошибкой `ImportError`, то настройка вашего пути была нестабильной. Посмотрите на значение `sys.path` и исправьте его.

```python
>>> import sys
>>> sys.path
... посмотрите на результат...
>>>
```
