# Vorbereitung

Dieses Übungsbeispiel befasst sich mit einigen der schwierigeren Details von Bibliotheksmodulen. Beginnen Sie mit dieser Übung, indem Sie ein sehr einfaches Bibliotheksmodul erstellen:

```python
# simplemod.py

x = 42        # Eine globale Variable

# Eine einfache Funktion
def foo():
    print('x ist', x)

# Eine einfache Klasse
class Spam:
    def yow(self):
        print('Yow!')

# Ein Skript-Befehl
print('simplemod geladen')
```
