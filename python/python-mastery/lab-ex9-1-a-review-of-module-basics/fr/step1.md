# Préparation

Cet exercice porte sur certains des détails plus complexes des modules de bibliothèque. Commencez cet exercice en créant un module de bibliothèque très simple :

```python
# simplemod.py

x = 42        # Une variable globale

# Une fonction simple
def foo():
    print('x est', x)

# Une classe simple
class Spam:
    def yow(self):
        print('Yow!')

# Une instruction de script
print('Chargé simplemod')
```
