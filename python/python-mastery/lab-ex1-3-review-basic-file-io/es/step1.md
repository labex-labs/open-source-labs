# Trabajando con archivos

El archivo `portfolio.dat` contiene una lista de líneas con información sobre un portafolio de acciones. El archivo tiene este aspecto:

    AA 100 32.20
    IBM 50 91.10
    CAT 150 83.44
    MSFT 200 51.23
    GE 95 40.37
    MSFT 50 65.10
    IBM 100 70.44

La primera columna es el nombre de la acción, la segunda columna es la cantidad de acciones y la tercera columna es el precio de compra de una sola acción.

Escribe un programa llamado `pcost.py` que abra este archivo, lea todas las líneas y calcule cuánto costó comprar todas las acciones del portafolio. Para hacer esto, calcula la suma de la segunda columna multiplicada por la tercera columna. Finalmente, muestra los resultados del cálculo.
