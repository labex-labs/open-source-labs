# Работа с файлами

Файл `portfolio.dat` содержит список строк с информацией о портфеле акций. Файл выглядит так:

    AA 100 32.20
    IBM 50 91.10
    CAT 150 83.44
    MSFT 200 51.23
    GE 95 40.37
    MSFT 50 65.10
    IBM 100 70.44

Первый столбец - это название акции, второй столбец - количество акций, а третий столбец - цена покупки одной акции.

Напишите программу под названием `pcost.py`, которая открывает этот файл, читает все строки и вычисляет, сколько стоило приобрести все акции в портфеле. Для этого вычислите сумму второго столбца, умноженного на третий столбец. Наконец, выведите результаты вычислений.
