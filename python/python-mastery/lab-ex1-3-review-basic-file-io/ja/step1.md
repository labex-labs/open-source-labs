# ファイルの操作

ファイル `portfolio.dat` には、株式のポートフォリオに関する情報が記載された行のリストが含まれています。ファイルは以下のようになっています。

    AA 100 32.20
    IBM 50 91.10
    CAT 150 83.44
    MSFT 200 51.23
    GE 95 40.37
    MSFT 50 65.10
    IBM 100 70.44

1列目は銘柄名、2列目は株数、3列目は1株当たりの買い付け価格です。

`pcost.py` というプログラムを作成して、このファイルを開き、すべての行を読み取り、ポートフォリオ内のすべての株式を買い付けるのにどれだけの費用がかかったかを計算します。これを行うには、2列目を3列目で掛けた値の合計を計算します。最後に、計算結果を出力します。
