# Arbeiten mit Dateien

Die Datei `portfolio.dat` enthält eine Liste von Zeilen mit Informationen über ein Aktienportfolio. Die Datei sieht so aus:

    AA 100 32.20
    IBM 50 91.10
    CAT 150 83.44
    MSFT 200 51.23
    GE 95 40.37
    MSFT 50 65.10
    IBM 100 70.44

Die erste Spalte ist der Aktienname, die zweite Spalte ist die Anzahl der Aktien, und die dritte Spalte ist der Kaufpreis einer einzelnen Aktie.

Schreiben Sie ein Programm namens `pcost.py`, das diese Datei öffnet, alle Zeilen liest und berechnet, wie viel es gekostet hat, alle Aktien im Portfolio zu erwerben. Dazu berechnen Sie die Summe der zweiten Spalte multipliziert mit der dritten Spalte. Geben Sie schließlich das Ergebnis der Berechnung aus.
