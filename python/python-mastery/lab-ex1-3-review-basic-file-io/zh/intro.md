# 简介

在这个实验中，你将学习回顾 Python 中的基本文件输入和输出操作。你将创建一个 Python 程序，从包含股票投资组合信息的文件中读取数据，并计算该投资组合的总成本。

本实验的目标包括学习如何在 Python 中打开和读取文件、逐行处理文件中的数据、对数据进行计算以及输出结果。你将创建的文件是 `pcost.py`。
