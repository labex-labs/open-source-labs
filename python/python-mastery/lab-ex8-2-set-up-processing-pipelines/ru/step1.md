# Настройка конвейера обработки

Основная сила генераторов заключается в том, что они позволяют создавать программы, которые настраивают конвейеры обработки - очень похоже на трубы в Unix-системах. Проверьте этот концепт, выполнив следующие шаги:

```python
>>> from follow import follow
>>> import csv
>>> lines = follow('stocklog.csv')
>>> rows = csv.reader(lines)
>>> for row in rows:
        print(row)

['BA', '98.35', '6/11/2007', '09:41.07', '0.16', '98.25', '98.35', '98.31', '158148']
['AA', '39.63', '6/11/2007', '09:41.07', '-0.03', '39.67', '39.63', '39.31', '270224']
['XOM', '82.45', '6/11/2007', '09:41.07', '-0.23', '82.68', '82.64', '82.41', '748062']
['PG', '62.95', '6/11/2007', '09:41.08', '-0.12', '62.80', '62.97', '62.61', '454327']
...
```

Ну, это интересно. Здесь вы можете увидеть, что вывод функции `follow()` был направлен в функцию `csv.reader()`, и теперь мы получаем последовательность разделенных строк.
