# Добавление нового метода

Добавьте новый метод `sell(nshares)` в класс `Stock`, который продает определенное количество акций, уменьшая количество акций. Пусть он работает так:

```python
>>> s = Stock('GOOG',100,490.10)
>>> s.shares
100
>>> s.sell(25)
>>> s.shares
75
>>>
```

## Примечание:

Завершите функцию `sell(nshares)` в файле `stock.py`.
