# Agregar un nuevo método

Agrega un nuevo método `sell(nshares)` a Stock que vende un cierto número de acciones disminuyendo la cantidad de acciones. Haz que funcione así:

```python
>>> s = Stock('GOOG',100,490.10)
>>> s.shares
100
>>> s.sell(25)
>>> s.shares
75
>>>
```

## Nota:

Completa la función `sell(nshares)` en el archivo `stock.py`.
