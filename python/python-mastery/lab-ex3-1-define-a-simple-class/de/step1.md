# Hinzufügen einer neuen Methode

Fügen Sie der Klasse `Stock` eine neue Methode `sell(nshares)` hinzu, die eine bestimmte Anzahl von Aktien verkauft, indem der Aktienbestand verringert wird. Die Methode soll wie folgt funktionieren:

```python
>>> s = Stock('GOOG',100,490.10)
>>> s.shares
100
>>> s.sell(25)
>>> s.shares
75
>>>
```

## Hinweis:

Vervollständigen Sie die Funktion `sell(nshares)` in der Datei `stock.py`.
