# Копирование метаданных

Когда функция оборачивается декоратором, часто теряются сведения о имени функции, строках документации и других деталях. Проверьте это:

```python
>>> @logged
    def add(x,y):
        'Adds two things'
        return x+y

>>> add
<function wrapper at 0x4439b0>
>>> help(add)
... посмотрите на вывод...
>>>
```

Исправьте определение декоратора `logged`, чтобы он правильно копировал метаданные функции. Для этого используйте декоратор `@wraps(func)`, как показано в примечаниях.

После завершения убедитесь, что декоратор сохраняет имя функции и строку документации.

```python
>>> @logged
    def add(x,y):
        'Adds two things'
        return x+y

>>> add
<function add at 0x4439b0>
>>> add.__doc__
'Adds two things'
>>>
```

Исправьте декоратор `@validated`, который вы писали ранее, чтобы он также сохранял метаданные с использованием `@wraps(func)`.
