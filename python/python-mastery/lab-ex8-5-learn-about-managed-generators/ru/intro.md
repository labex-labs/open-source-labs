# Введение

**Цели:**

- Изучить управляемые генераторы

**Созданные файлы:** `multitask.py`, `server.py`

Генератор или сопрограмма никогда не может выполниться, не будучи управляемой другой программой. Например, генератор, используемый для итерации, ничего не делает, если итерация не выполняется с использованием цикла `for`. Аналогично, набор сопрограмм не запустится, если их метод `send()` не будет вызван каким-то образом.

В более сложных приложениях генераторов возможно управлять ими различными необычными способами. В этом упражнении рассмотрим несколько примеров.
