# Подготовка

В предыдущей задаче вы создали класс `Structure`, который упрощал определение структур данных. Например:

```python
class Stock(Structure):
    _fields = ('name','shares','price')
```

Это работает нормально, за исключением того, что много вещей выглядит довольно странно в функции `__init__()`. Например, если вы запрашиваете помощь с использованием `help(Stock)`, вы не получаете никакого полезного сигнатуры. Также передача аргументов по ключевым словам не работает. Например:

```python
>>> help(Stock)
... посмотрите на вывод...

>>> s = Stock(name='GOOG', shares=100, price=490.1)
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: __init__() got an unexpected keyword argument 'price'
>>>
```

В этой задаче мы рассмотрим другой подход к решению этой проблемы.
