# Захват исключений

Вместо того чтобы завершаться с ошибкой при обработке плохих данных, модифицируйте код так, чтобы он выдавал предупреждение. Финальным результатом должен быть список строк, которые были успешно преобразованы. Например:

```python
>>> port = read_csv_as_dicts('missing.csv', types=[str, int, float])
Строка 4: Плохая строка: ['C', '', '53.08']
Строка 7: Плохая строка: ['DIS', '50', 'N/A']
Строка 8: Плохая строка: ['GE', '', '37.23']
Строка 13: Плохая строка: ['INTC', '', '21.84']
Строка 17: Плохая строка: ['MCD', '', '51.11']
Строка 19: Плохая строка: ['MO', '', '70.09']
Строка 22: Плохая строка: ['PFE', '', '26.40']
Строка 26: Плохая строка: ['VZ', '', '42.92']
>>> len(port)
20
>>>
```

Примечание: внесение этого изменения может оказаться немного сложным, из-за предыдущего использования встроенной функции `map()`. Возможно, придется отказаться от этого подхода, так как нет простого способа поймать и обработать исключения в `map()`.
