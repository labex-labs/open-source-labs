# Введение

В этом практическом занятии (лабораторной работе) вы научитесь инкапсулировать внутренние данные объектов с использованием приватных атрибутов и реализовывать декораторы свойств (property decorators) для контроля доступа к атрибутам. Эти методы являются важными для сохранения целостности ваших объектов и обеспечения правильного доступа к данным.

Кроме того, вы узнаете, как ограничить создание атрибутов с использованием `__slots__`. Файл `stock.py` будет изменен в рамках этого практического занятия для применения этих концепций.
