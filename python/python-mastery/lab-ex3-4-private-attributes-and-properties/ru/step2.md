# Преобразование методов в свойства

В Python свойства (properties) - это мощная функциональность, которая позволяет получать вычисляемые значения так же, как и обычные атрибуты. Обычно, когда вы хотите получить значение из метода, вам нужно использовать скобки для вызова этого метода. Однако свойства устраняют необходимость в этих скобках, делая ваш код более чистым и более согласованным с тем, как вы обращаетесь к обычным атрибутам.

Давайте посмотрим на наш текущий класс `Stock`. У него есть метод `cost()`, который вычисляет общую стоимость акций. Этот метод умножает количество акций на цену за одну акцию, чтобы получить общую стоимость. Вот как выглядит метод `cost()`:

```python
def cost(self):
    return self.shares * self.price
```

Чтобы получить значение стоимости с помощью этого метода, мы должны вызвать его с использованием скобок, вот так:

```python
s = Stock('GOOG', 100, 490.10)
print(s.cost())  # Calls the method
```

Но мы можем сделать наш код лучше. Преобразовав метод `cost()` в свойство, мы можем получить доступ к значению стоимости так же, как и к другим атрибутам, без использования скобок. Вот как это будет выглядеть:

```python
s = Stock('GOOG', 100, 490.10)
print(s.cost)  # Accesses the property
```

## Инструкции:

1. Сначала нам нужно открыть файл `stock.py` в редакторе. Именно здесь определен класс `Stock`, и мы будем вносить изменения в него. Используйте следующую команду в терминале:

   ```bash
   code /home/labex/project/stock.py
   ```

2. Внутри файла `stock.py` мы собираемся заменить метод `cost()` на свойство. Мы будем использовать декоратор `@property` для этого. Декоратор `@property` сообщает Python, что следующий метод должен быть восприниматься как свойство. Замените метод `cost()` следующим кодом:

   ```python
   @property
   def cost(self):
       return self.shares * self.price
   ```

3. После внесения изменений сохраните файл `stock.py`. Это гарантирует, что наши изменения будут сохранены и могут быть использованы позже.

4. Теперь нам нужно создать простой скрипт Python для тестирования нашего нового свойства. Откройте новый файл с именем `test_property.py` в редакторе с помощью следующей команды:

   ```bash
   code /home/labex/project/test_property.py
   ```

5. В файле `test_property.py` мы добавим некоторый код для создания экземпляра класса `Stock` и доступа к свойству `cost`. Вот код, который вы должны добавить:

   ```python
   from stock import Stock

   # Create a stock instance
   s = Stock('GOOG', 100, 490.10)

   # Access cost as a property (no parentheses)
   print(f"Stock: {s.name}")
   print(f"Shares: {s.shares}")
   print(f"Price: {s.price}")
   print(f"Cost: {s.cost}")  # Using the property
   ```

6. Наконец, запустите тестовый скрипт, чтобы проверить, работает ли наше свойство как ожидается. Используйте следующую команду в терминале:
   ```bash
   python /home/labex/project/test_property.py
   ```

Вы должны увидеть вывод, похожий на следующий:

```
Stock: GOOG
Shares: 100
Price: 490.1
Cost: 49010.0
```

Обратите внимание, как теперь мы можем обращаться к `cost` как к атрибуту (без скобок), что делает наш код более согласованным с тем, как мы обращаемся к другим атрибутам, таким как `name`, `shares` и `price`. Это делает наш код легче читаемым и поддерживаемым.
