# Свойства для вычисляемых атрибутов

Ранее вы определили класс `Stock`. Например:

```python
>>> s = Stock('GOOG',100,490.10)
>>> s.name
'GOOG'
>>> s.shares
100
>>> s.price
490.1
>>> s.cost()
49010.0
>>>
```

С использованием свойства превратите `cost()` в атрибут, который больше не требует скобок. Например:

```python
>>> s = Stock('GOOG', 100, 490.1)
>>> s.cost               # Свойство. Вычисляет стоимость
49010.0
>>>
```
