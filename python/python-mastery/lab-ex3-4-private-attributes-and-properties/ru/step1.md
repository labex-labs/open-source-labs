# Приватные атрибуты

В общем случае, атрибуты, внутренние для класса, должны начинаться с нижнего подчеркивания. В предыдущем упражнении класс `Stock` имел переменную класса `types`, которая использовалась для преобразования строк данных. Измените код так, чтобы эта переменная начиналась с нижнего подчеркивания.
