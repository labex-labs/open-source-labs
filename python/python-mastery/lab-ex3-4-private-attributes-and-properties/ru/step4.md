# Добавление `__slots__`

Измените ваш новый класс `Stock` для использования `__slots__`. Вы обнаружите, что вам придется использовать другой набор имен атрибутов, чем раньше - конкретно, вам нужно будет перечислить имена приватных атрибутов (например, если свойство хранит значение в атрибуте `_shares`, это имя вы перечисляете в `__slots__`). Проверьте, что класс по-прежнему работает и что вы больше не можете добавлять новые атрибуты.

```python
>>> s = Stock('GOOG', 100, 490.10)
>>> s.spam = 42
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: 'Stock' object has no attribute 'spam'
>>>
```
