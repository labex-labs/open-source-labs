# 計算属性のためのプロパティ

以前、あなたは `Stock` というクラスを定義しました。たとえば：

```python
>>> s = Stock('GOOG',100,490.10)
>>> s.name
'GOOG'
>>> s.shares
100
>>> s.price
490.1
>>> s.cost()
49010.0
>>>
```

プロパティを使用して、`cost()` を丸括弧が不要な属性に変換します。たとえば：

```python
>>> s = Stock('GOOG', 100, 490.1)
>>> s.cost               # プロパティ。コストを計算する
49010.0
>>>
```
