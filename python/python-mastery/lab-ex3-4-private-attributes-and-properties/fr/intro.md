# Introduction

Dans ce laboratoire (lab), vous apprendrez à encapsuler les éléments internes d'un objet à l'aide d'attributs privés et à implémenter des décorateurs de propriétés pour contrôler l'accès aux attributs. Ces techniques sont essentielles pour maintenir l'intégrité de vos objets et garantir un accès approprié aux données.

De plus, vous comprendrez comment restreindre la création d'attributs à l'aide de `__slots__`. Le fichier `stock.py` sera modifié au cours de ce laboratoire pour appliquer ces concepts.
