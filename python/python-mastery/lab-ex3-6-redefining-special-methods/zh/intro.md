# 简介

在这个实验中，你将学习如何通过重新定义特殊方法来自定义对象的行为。你还将改变用户定义对象的打印方式，并使对象具有可比性。

此外，你将学习创建一个上下文管理器。本实验中需要修改的文件是 `stock.py`。
