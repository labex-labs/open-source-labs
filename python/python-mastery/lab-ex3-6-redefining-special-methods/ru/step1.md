# Лучший вывод для представления объектов

Все объекты Python имеют две строковые репрезентации. Первая репрезентация создается при преобразовании в строку с использованием `str()` (который вызывается при использовании `print`). Строковая репрезентация обычно представляет собой хорошо отформатированную версию объекта, предназначенную для людей. Вторая репрезентация - это кодовая репрезентация объекта, созданная с использованием `repr()` (или просто при просмотре значения в интерактивной оболочке). Кодовая репрезентация обычно показывает вам код, который нужно ввести, чтобы получить объект. Вот пример, иллюстрирующий использование дат:

```python
>>> from datetime import date
>>> d = date(2008, 7, 5)
>>> print(d)              # использует str()
2008-07-05
>>> d    # использует repr()
datetime.date(2008, 7, 5)
>>>
```

Существует несколько способов получить строку `repr()` в выводе:

```python
>>> print('The date is', repr(d))
The date is datetime.date(2008, 7, 5)
>>> print(f'The date is {d!r}')
The date is datetime.date(2008, 7, 5)
>>> print('The date is %r' % d)
The date is datetime.date(2008, 7, 5)
>>>
```

Измените объект `Stock`, который вы создали, так чтобы метод `__repr__()` выдавал более полезный вывод. Например:

```python
>>> goog = Stock('GOOG', 100, 490.10)
>>> goog
Stock('GOOG', 100, 490.1)
>>>
```
