# 第4部 : 辞書

ここまでの数パートでは、単に株価シンボルを扱ってきました。しかし、株価シンボルを価格などの他のデータにマッピングしたいとしたらどうでしょうか？辞書を使いましょう。

```python
>>> prices = { 'IBM': 91.1, 'GOOG': 490.1, 'AAPL':312.23 }
>>>
```

辞書はキーを値にマッピングします。アクセス方法は次の通りです。

```python
>>> prices['IBM']
91.1
>>> prices['IBM'] = 123.45
>>> prices['HPQ'] = 26.15
>>> prices
{'GOOG': 490.1, 'AAPL': 312.23, 'IBM': 123.45, 'HPQ': 26.15}
>>>
```

キーのリストを取得するには、次のようにします。

```python
>>> list(prices)
['GOOG', 'AAPL', 'IBM', 'HPQ']
>>>
```

値を削除するには、`del` を使います。

```python
>>> del prices['AAPL']
>>> prices
{'GOOG': 490.1, 'IBM': 123.45, 'HPQ': 26.15}
>>>
```
