# 第1部 : 数値

Python における数値計算は、おおよそ期待通りに機能します。たとえば：

```python
>>> 3 + 4*5
23
>>> 23.45 / 1e-02
2345.0
>>>
```

Python 2 と Python 3 では整数除算が異なることに注意してください。

```python
>>> 7 / 4      # Python 2 では、これは切り捨てて 1 になります
1.75
>>> 7 // 4     # 切り捨て除算
1
>>>
```

Python 2 で Python 3 の動作を得たい場合は、次のようにします：

```python
>>> from __future__ import division
>>> 7 / 4
1.75
>>> 7 // 4      # 切り捨て除算
1
>>>
```

数値にはいくつかのメソッドがあり、その多くは実際には最近のもので、経験豊富な Python プログラマーでさえ見落とされています。いくつか試してみてください。

```python
>>> x = 1172.5
>>> x.as_integer_ratio()
(2345, 2)
>>> x.is_integer()
False
>>> y = 12345
>>> y.numerator
12345
>>> y.denominator
1
>>> y.bit_length()
14
>>>
```
