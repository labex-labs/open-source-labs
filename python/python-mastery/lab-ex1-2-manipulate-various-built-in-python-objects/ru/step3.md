# Часть 3: Обработка списков

В первой части вы работали со строками, содержащими символы акций. Например:

```python
>>> symbols = 'HPQ AAPL IBM MSFT YHOO  GOOG'
>>>
```

Определите переменную выше и разбейте ее на список имен с использованием операции `split()` для строк:

```python
>>> symlist = symbols.split()
>>> symlist
['HPQ', 'AAPL', 'IBM', 'MSFT', 'YHOO', 'GOOG' ]
>>>
```

## Извлечение и переприсваивание элементов списка

Списки работают как массивы, где вы можете обращаться и изменять элементы по числовому индексу. Попробуйте несколько обращений:

```python
>>> symlist[0]
'HPQ'
>>> symlist[1]
'AAPL'
>>> symlist[-1]
'GOOG'
>>> symlist[-2]
'YHOO'
>>>
```

Попробуйте переприсвоить один из элементов:

```python
>>> symlist[2] = 'AIG'
>>> symlist
['HPQ', 'AAPL', 'AIG', 'MSFT', 'YHOO', 'GOOG' ]
>>>
```

## Цикл по элементам списка

Цикл `for` работает путем перебора данных в последовательности, такой как список. Проверьте это, введя следующий цикл и наблюдая, что происходит:

```python
>>> for s in symlist:
        print('s =', s)

... посмотрите на вывод...
```

## Проверка на принадлежность

Используйте оператор `in`, чтобы проверить, находятся ли `'AIG'`, `'AA'` и `'CAT'` в списке символов.

```python
>>> 'AIG' in symlist
True
>>> 'AA' in symlist
False
>>>
```

## Добавление, вставка и удаление элементов

Используйте метод `append()`, чтобы добавить символ `'RHT'` в конец `symlist`.

```python
>>> symlist.append('RHT')
>>> symlist
['HPQ', 'AAPL', 'AIG', 'MSFT', 'YHOO', 'GOOG', 'RHT']
>>>
```

Используйте метод `insert()`, чтобы вставить символ `'AA'` в качестве второго элемента в списке.

```python
>>> symlist.insert(1,'AA')
>>> symlist
['HPQ', 'AA', 'AAPL', 'AIG', 'MSFT', 'YHOO', 'GOOG', 'RHT']
>>>
```

Используйте метод `remove()`, чтобы удалить `'MSFT'` из списка.

```python
>>> symlist.remove('MSFT')
>>> symlist
['HPQ', 'AA', 'AAPL', 'AIG', 'YHOO', 'GOOG', 'RHT']
```

Попробуйте вызвать `remove()` снова, чтобы увидеть, что произойдет, если элемент не найден.

```python
>>> symlist.remove('MSFT')
... наблюдайте, что происходит...
>>>
```

Используйте метод `index()`, чтобы найти позицию `'YHOO'` в списке.

```python
>>> symlist.index('YHOO')
4
>>> symlist[4]
'YHOO'
>>>
```

## Сортировка списка

Хотите отсортировать список? Используйте метод `sort()`. Попробуйте:

```python
>>> symlist.sort()
>>> symlist
['AA', 'AAPL', 'AIG', 'GOOG', 'HPQ', 'RHT', 'YHOO']
>>>
```

Хотите отсортировать в обратном порядке? Попробуйте это:

```python
>>> symlist.sort(reverse=True)
>>> symlist
['YHOO', 'RHT', 'HPQ', 'GOOG', 'AIG', 'AAPL', 'AA']
>>>
```

Примечание: сортировка списка изменяет его содержимое "на месте". То есть элементы списка перемешиваются, но в результате не создается новый список.

## Списки любых объектов

Списки могут содержать любые виды объектов, включая другие списки (например, вложенные списки). Попробуйте это:

```python
>>> nums = [101,102,103]
>>> items = [symlist, nums]
>>> items
[['YHOO', 'RHT', 'HPQ', 'GOOG', 'AIG', 'AAPL', 'AA'], [101, 102, 103]]
```

Обратите особое внимание на вышеуказанный вывод. `items` - это список с двумя элементами. Каждый элемент - это список.

Попробуйте несколько обращений к вложенным спискам:

```python
>>> items[0]
['YHOO', 'RHT', 'HPQ', 'GOOG', 'AIG', 'AAPL', 'AA']
>>> items[0][1]
'RHT'
>>> items[0][1][2]
'T'
>>> items[1]
[101, 102, 103]
>>> items[1][1]
102
>>>
```
