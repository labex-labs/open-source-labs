# Часть 4: Словарь

В нескольких последних частях вы просто работали с символами акций. Однако, предположим, что вы хотели сопоставить символы акций с другими данными, такими как цена? Используйте словарь:

```python
>>> prices = { 'IBM': 91.1, 'GOOG': 490.1, 'AAPL':312.23 }
>>>
```

Словарь сопоставляет ключи со значениями. Вот, как получить доступ:

```python
>>> prices['IBM']
91.1
>>> prices['IBM'] = 123.45
>>> prices['HPQ'] = 26.15
>>> prices
{'GOOG': 490.1, 'AAPL': 312.23, 'IBM': 123.45, 'HPQ': 26.15}
>>>
```

Для получения списка ключей используйте это:

```python
>>> list(prices)
['GOOG', 'AAPL', 'IBM', 'HPQ']
>>>
```

Для удаления значения используйте `del`

```python
>>> del prices['AAPL']
>>> prices
{'GOOG': 490.1, 'IBM': 123.45, 'HPQ': 26.15}
>>>
```
