# Часть 2: Обработка строк

Определите строку, содержащую серию символов биржевого ticker, следующим образом:

```python
>>> symbols = 'AAPL IBM MSFT YHOO SCO'
```

Теперь давайте поэкспериментируем с разными операциями над строками:

## Извлечение отдельных символов и подстрок

Строки - это массивы символов. Попробуйте извлечь несколько символов:

```python
>>> symbols[0]
'A'
>>> symbols[1]
'A'
>>> symbols[2]
'P'
>>> symbols[-1]        # Последний символ
'O'
>>> symbols[-2]        # Второй с конца символ
'C'
>>>
```

Попробуйте взять несколько срезов:

```python
>>> symbols[:4]
'AAPL'
>>> symbols[-3:]
'SCO'
>>> symbols[5:8]
'IBM'
>>>
```

## Строки как объекты только для чтения

Строки только для чтения. Проверьте это, попробовав изменить первый символ `symbols` на строчную 'a'.

```python
>>> symbols[0] = 'a'
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: 'str' object does not support item assignment
>>>
```

## Конкатенация строк

Хотя данные строки только для чтения, вы всегда можете переприсвоить переменную новой созданной строке.
Попробуйте следующее выражение, которое конкатенирует новый символ "GOOG" в конец `symbols`:

```python
>>> symbols += ' GOOG'
>>> symbols
... посмотрите на результат...
```

Теперь попробуйте добавить "HPQ" в начало `symbols` следующим образом:

```python
>>> symbols = 'HPQ'+ symbols
>>> symbols
... посмотрите на результат...
```

Следует отметить, что в обоих этих примерах исходная строка `symbols` _не_ модифицируется "на месте". Вместо этого создается совершенно новая строка. Имя переменной `symbols` просто связывается с результатом. Затем старая строка уничтожается, так как она больше не используется.

## Проверка на принадлежность (проверка на подстроку)

П поэкспериментируйте с оператором `in`, чтобы проверить наличие подстрок. В интерактивном режиме попробуйте эти операции:

```python
>>> 'IBM' in symbols
True
>>> 'AA' in symbols
True
>>> 'CAT' in symbols
False
>>>
```

Убедитесь, что понимаете, почему проверка на "AA" вернула `True`.

## Методы строк

В интерактивном режиме Python попробуйте поэкспериментировать с некоторыми методами строк.

```python
>>> symbols.lower()
'hpq aapl ibm msft yhoo sco goog'
>>> symbols
'HPQ AAPL IBM MSFT YHOO SCO GOOG'
```

Помните, что строки всегда только для чтения. Если вы хотите сохранить результат операции, вам нужно поместить его в переменную:

```python
>>> lowersyms = symbols.lower()
>>> lowersyms
'hpq aapl ibm msft yhoo sco goog'
>>>
```

Попробуйте еще несколько операций:

```python
>>> symbols.find('MSFT')
13
>>> symbols[13:17]
'MSFT'
>>> symbols = symbols.replace('SCO','')
>>> symbols
'HPQ AAPL IBM MSFT YHOO  GOOG'
>>>
```
