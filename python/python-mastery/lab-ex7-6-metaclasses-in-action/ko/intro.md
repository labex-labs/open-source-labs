# 소개

이 랩에서는 Python 의 가장 강력하고 고급 기능 중 하나인 메타클래스 (metaclasses) 에 대해 배우게 됩니다. 메타클래스를 사용하면 클래스 생성을 사용자 정의하여 클래스가 정의되고 인스턴스화되는 방식을 제어할 수 있습니다. 실용적인 예제를 통해 메타클래스를 탐구할 것입니다.

이 랩의 목표는 메타클래스가 무엇이며 어떻게 작동하는지 이해하고, 실제 프로그래밍 문제를 해결하기 위해 메타클래스를 구현하며, Python 에서 메타클래스의 실용적인 응용 프로그램을 탐구하는 것입니다. 이 랩에서 수정되는 파일은 `structure.py`와 `validate.py`입니다.
