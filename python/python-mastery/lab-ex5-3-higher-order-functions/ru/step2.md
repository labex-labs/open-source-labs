# Отображение

Одной из наиболее распространенных операций в функциональном программировании является операция `map()`, которая отображает функцию на значения в последовательности. В Python есть встроенная функция `map()`, которая делает это. Например:

```python
>>> nums = [1,2,3,4]
>>> squares = map(lambda x: x*x, nums)
>>> for n in squares:
        print(n)

1
4
9
16
>>>
```

`map()` возвращает итератор, поэтому, если вы хотите получить список, вам нужно явно создать его:

```python
>>> squares = list(map(lambda x: x*x, nums))
>>> squares
[1, 4, 9, 16]
>>>
```
