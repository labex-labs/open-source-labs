# Добавление итерации к объектам

Если вы создали пользовательский класс, вы можете сделать его поддерживающим итерацию, определив специальный метод `__iter__()`. `__iter__()` возвращает итератор в качестве результата. Как показано в предыдущем примере, простой способ сделать это - определить `__iter__()` в виде генератора.

В предыдущих упражнениях вы определили базовый класс `Structure`. Добавьте метод `__iter__()` к этому классу, который будет выдавать значения атрибутов в порядке. Например:

```python
class Structure(metaclass=StructureMeta):
  ...
    def __iter__(self):
        for name in self._fields:
            yield getattr(self, name)
  ...
```

После этого вы должны быть able итерироваться по атрибутам экземпляра так:

```python
>>> from stock import Stock
>>> s = Stock('GOOG', 100, 490.1)
>>> for val in s:
        print(val)
GOOG
100
490.1
>>>
```
