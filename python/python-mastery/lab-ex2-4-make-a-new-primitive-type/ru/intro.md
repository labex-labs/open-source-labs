# Введение

**Цели:**

- Создать новый примитивный тип

В большинстве программ вы используете примитивные типы, такие как `int`, `float` и `str`, для представления данных. Однако вы не ограничены только этими типами. В стандартной библиотеке есть модули, такие как модуль `decimal` и `fractions`, которые реализуют новые примитивные типы. Вы также можете создать собственные типы, если только понимаете базовые протоколы, благодаря которым работают объекты Python. В этом упражнении мы создадим новый примитивный тип. Есть много мелких деталей, о которых нужно беспокоиться, но это даст вам общее представление о том, что требуется.
