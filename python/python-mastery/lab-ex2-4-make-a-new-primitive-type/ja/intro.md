# はじめに

**目的:**

- 新しいプリミティブ型を作成する

ほとんどのプログラムでは、`int`、`float`、`str` などのプリミティブ型を使ってデータを表現します。しかし、それだけの型には制限されません。標準ライブラリには、新しいプリミティブ型を実装する `decimal` や `fractions` モジュールなどのモジュールがあります。Python オブジェクトが機能する根本的なプロトコルを理解していれば、自分自身の型を作成することもできます。この演習では、新しいプリミティブ型を作成します。心配する小さな詳細はたくさんありますが、これにより必要なことの全体像が掴めるでしょう。
