# Introdução

Neste laboratório, você aprenderá a explorar os aspectos fundamentais das funções e métodos Python. Você também tornará as funções mais flexíveis ao projetar parâmetros de forma eficaz.

Além disso, você implementará _type hints_ (anotações de tipo) para aprimorar a legibilidade e a segurança do código, o que é crucial para escrever código Python de alta qualidade.
