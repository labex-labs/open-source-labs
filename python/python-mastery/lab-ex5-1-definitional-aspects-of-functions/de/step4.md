# Design Challenge: CSV-Header

Der Code geht davon aus, dass die erste Zeile der CSV-Daten immer Spaltenüberschriften enthält. Dies ist jedoch nicht immer der Fall. Beispielsweise enthält die Datei `portfolio_noheader.csv` Daten, aber keine Spaltenüberschriften.

Wie würden Sie den Code refaktorisieren, um fehlende Spaltenüberschriften zu berücksichtigen, die stattdessen vom Aufrufer manuell bereitgestellt werden?
