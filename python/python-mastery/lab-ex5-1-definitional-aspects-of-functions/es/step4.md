# Desafío de Diseño: Encabezados CSV

El código asume que la primera línea de datos CSV siempre contiene encabezados de columna. Sin embargo, esto no siempre es el caso. Por ejemplo, el archivo `portfolio_noheader.csv` contiene datos, pero no tiene encabezados de columna.
