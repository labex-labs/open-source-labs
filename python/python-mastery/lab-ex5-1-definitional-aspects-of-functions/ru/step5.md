# Вызов API: подсказки типов

Функции могут иметь необязательные подсказки типов, прикрепленные к аргументам и значениям возврата. Например:

```python
def add(x:int, y:int) -> int:
    return x + y
```

Модуль `typing` имеет дополнительные классы для выражения более сложных типов, включая контейнеры. Например:

```python
from typing import List

def sum_squares(nums: List[int]) -> int:
    total = 0
    for n in nums:
        total += n*n
    return total
```
