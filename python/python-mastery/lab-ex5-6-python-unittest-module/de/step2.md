# Unit-Testing

Verwenden Sie den Code in `teststock.py` als Leitfaden und erweitern Sie die `TestStock`-Klasse um Tests für Folgendes:

- Testen Sie, dass Sie eine `Stock`-Instanz mit Schlüsselwortargumenten wie `Stock(name='GOOG',shares=100,price=490.1)` erstellen können.
- Testen Sie, dass die `cost`-Eigenschaft einen korrekten Wert zurückgibt.
- Testen Sie, dass die `sell()`-Methode die Anzahl der Anteile korrekt aktualisiert.
- Testen Sie, dass die `from_row()`-Klassennmethode eine neue Instanz aus gültigen Daten erstellt.
- Testen Sie, dass die `__repr__()`-Methode einen korrekten Darstellung字符串 erstellt.
- Testen Sie die Vergleichsoperator-Methode `__eq__()`
