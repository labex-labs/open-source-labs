# Юнит-тестирование

С использованием кода в `teststock.py` в качестве руководства, расширьте класс `TestStock` тестами для следующего:

- Проверьте, что вы можете создать `Stock` с использованием именованных аргументов, таких как `Stock(name='GOOG',shares=100,price=490.1)`.
- Проверьте, что свойство `cost` возвращает правильное значение
- Проверьте, что метод `sell()` правильно обновляет количество акций.
- Проверьте, что метод класса `from_row()` создает новый экземпляр из корректных данных.
- Проверьте, что метод `__repr__()` создает правильную строку представления.
- Проверьте метод оператора сравнения `__eq__()`
