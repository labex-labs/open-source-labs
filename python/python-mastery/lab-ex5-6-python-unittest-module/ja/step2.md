# 単体テスト

`teststock.py` のコードを参考にして、`TestStock` クラスを拡張して、以下のテストを行います。

- `Stock(name='GOOG',shares=100,price=490.1)` のようなキーワード引数を使って `Stock` を作成できることをテストする。
- `cost` プロパティが正しい値を返すことをテストする。
- `sell()` メソッドが株式を正しく更新することをテストする。
- `from_row()` クラスメソッドが適切なデータから新しいインスタンスを作成することをテストする。
- `__repr__()` メソッドが適切な表現文字列を作成することをテストする。
- 比較演算子メソッド `__eq__()` をテストする
