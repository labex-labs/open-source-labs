# Pruebas unitarias

Usando el código en `teststock.py` como guía, extiende la clase `TestStock` con pruebas para lo siguiente:

- Prueba que puedas crear una instancia de `Stock` usando argumentos de palabras clave como `Stock(name='GOOG',shares=100,price=490.1)`.
- Prueba que la propiedad `cost` devuelva un valor correcto.
- Prueba que el método `sell()` actualice correctamente las acciones.
- Prueba que el método de clase `from_row()` cree una nueva instancia a partir de datos válidos.
- Prueba que el método `__repr__()` cree una cadena de representación adecuada.
- Prueba el método del operador de comparación `__eq__()`
