# Возвращение необязательных значений

Иногда функция может возвращать необязательное значение - возможно, в качестве механизма для индикации успеха или неудачи. Наиболее распространенный convention - использовать `None` в качестве представления для отсутствующего значения. Измените функцию `parse_line()` выше так, чтобы она возвращала либо кортеж в случае успеха, либо `None` при плохих данных. Например:

```python
>>> parse_line('email=guido@python.org')
('email', 'guido@python.org')
>>> parse_line('spam')       # Возвращает None
>>>
```
