# Задача по анализу данных

В прошлой лабораторной работе вы только написали код для чтения CSV-данных, связанных с Автобусным транспортом Чикаго. Например, вы можете получить данные в виде словарей так:

```python
>>> import readrides
>>> rows = readrides.read_rides_as_dicts('/home/labex/project/ctabus.csv')
>>>
```

Было бы стыдно сделать все это и потом ничего не сделать с этими данными.

В этом упражнении ваша задача состоит в том, чтобы написать программу, которая будет отвечать на следующие четыре вопросы:

1. Сколько маршрутов автобусов существует в Чикаго?

2. Сколько людей сели на автобус номер 22 в 2 февраля 2011 года? А что с любым маршрутом в любую дату вашего выбора?

3. Какова общая сумма пассажиропотока по каждому маршруту автобуса?

4. Какие пять маршрутов автобусов имели наибольший десятилетний прирост пассажиропотока с 2001 по 2011 год?

Вы можете свободно использовать любые методы для ответа на вышеперечисленные вопросы,只要 они являются частью стандартной библиотеки Python (например, встроенные типы данных, стандартные модули библиотеки и т.д.).
