# Introducción

**Objetivos:**

- Trabajar con varios contenedores
- Comprensiones de listas/conjuntos/diccionarios
- Módulo Collections
- Desafío de análisis de datos

La mayoría de los programadores de Python suelen estar familiarizados con listas, diccionarios, tuplas y otros tipos de datos básicos. En este ejercicio, pondremos ese conocimiento en práctica para resolver varios problemas de análisis de datos.
