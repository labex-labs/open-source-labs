# Einführung

**Ziele:**

- Arbeiten mit verschiedenen Containern
- List/Set/Dict Comprehensions
- Collections-Modul
- Data-Analyse-Aufgabe

Die meisten Python-Programmierer sind im Allgemeinen mit Listen, Wörterbüchern, Tupeln und anderen grundlegenden Datentypen vertraut. In dieser Übung werden wir diese Kenntnisse nutzen, um verschiedene Datenanalyseprobleme zu lösen.
