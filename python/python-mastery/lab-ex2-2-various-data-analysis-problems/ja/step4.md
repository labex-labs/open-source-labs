# データ分析チャレンジ

前回の実験では、シカゴ交通局に関連するCSVデータを読み取るためのコードを書きました。たとえば、次のように辞書としてデータを取得できます。

```python
>>> import readrides
>>> rows = readrides.read_rides_as_dicts('/home/labex/project/ctabus.csv')
>>>
```

そのような作業をすべて行って、そのデータを使わないのはもったいないことです。

この演習でのあなたの課題は、次の4つの質問に答えるプログラムを書くことです。

1. シカゴには何本のバス路線がありますか？

2. 2011年2月2日には何人が22番のバスに乗りましたか？好きな日付の任意の路線についてはどうでしょうか？

3. 各バス路線の乗車総数は何ですか？

4. 2001年から2011年までの10年間で乗客数が最も増加した5つのバス路線はどれですか？

Python標準ライブラリ（すなわち、組み込みデータ型、標準ライブラリモジュールなど）の一部である限り、上記の質問に答えるためにあなたは自由にどんな技術を使っても構いません。
