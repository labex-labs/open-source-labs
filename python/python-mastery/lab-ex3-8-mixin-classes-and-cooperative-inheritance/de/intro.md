# Einführung

In diesem Lab werden Sie sich mit Mixin-Klassen (Mixin-Klassen sind in der Informatik eine spezielle Art von Klasse) und ihrer Rolle bei der Verbesserung der Code-Wiederverwendbarkeit vertraut machen. Sie werden verstehen, wie Sie Mixins implementieren können, um die Funktionalität von Klassen zu erweitern, ohne den bestehenden Code zu verändern.

Sie werden auch die Techniken der kooperativen Vererbung in Python beherrschen. Die Datei `tableformat.py` wird während des Experiments modifiziert.
