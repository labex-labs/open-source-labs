# Introdução

Neste laboratório, você aprenderá sobre decoradores de classe em Python e revisitará e estenderá os descritores (descriptors) Python. Ao combinar esses conceitos, você pode criar estruturas de código poderosas e limpas.

Neste laboratório, você construirá sobre os conceitos de descritores anteriores e os estenderá usando decoradores de classe. Essa combinação permite que você crie um código mais limpo e sustentável, com capacidades de validação aprimoradas. Os arquivos a serem modificados são `validate.py` e `structure.py`.
