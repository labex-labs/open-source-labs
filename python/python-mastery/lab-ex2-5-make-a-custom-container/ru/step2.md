# Рост словаря/класса

В Python словарям (и классам) разрешается хранить до 5 значений, прежде чем выделенная для них память не утроится. Проверьте это, создав словарь и добавив несколько значений:

```python
>>> row = { 'route': '22', 'date': '01/01/2001', 'daytype': 'U', 'rides': 7354 }
>>> sys.getsizeof(row)
>>> sys.getsizeof(row)
240
>>> row['a'] = 1
>>> sys.getsizeof(row)
240
>>> row['b'] = 2
>>> sys.getsizeof(row)
368
>>>
```

Уменьшается ли память, если удалить только что добавленный элемент?

Думайте над тем: если вы создаете большое количество записей, представлять каждую запись в виде словаря может не быть наиболее эффективным подходом - вы можете платить высокую цену за удобство использования словаря. Возможно, лучше рассмотреть использование кортежей, именованных кортежей или классов, которые определяют `__slots__`.
