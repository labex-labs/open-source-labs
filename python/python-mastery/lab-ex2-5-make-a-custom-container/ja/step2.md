# 辞書/クラスの拡大

Pythonの辞書（およびクラス）は、予約されたメモリが2倍になるまで最大5つの値を格納することができます。辞書を作成していくつかの値を追加することで調査してみましょう。

```python
>>> row = { 'route': '22', 'date': '01/01/2001', 'daytype': 'U', 'rides': 7354 }
>>> sys.getsizeof(row)
>>> sys.getsizeof(row)
240
>>> row['a'] = 1
>>> sys.getsizeof(row)
240
>>> row['b'] = 2
>>> sys.getsizeof(row)
368
>>>
```

追加した項目を削除すると、メモリが減少しますか？

考えること：大量のレコードを作成する場合、各レコードを辞書として表現することは最も効率的な方法ではない場合があります。辞書を使う利便性のために高い代価を支払っている可能性があります。タプル、名前付きタプル、または`__slots__`を定義するクラスの使用を検討する方が良いかもしれません。
