# Использование getattr()

Функция `getattr()` чрезвычайно полезна для написания кода, который обрабатывает объекты по-очень общему принципу. Чтобы проиллюстрировать это, рассмотрите этот пример, который выводит набор пользовательских атрибутов:

```python
>>> s= Stock('GOOG', 100, 490.1)
>>> fields = ['name','shares','price']
>>> for name in fields:
           print(name, getattr(s, name))

name GOOG
shares 100
price 490.1
>>>
```
