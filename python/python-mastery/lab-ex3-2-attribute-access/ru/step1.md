# Три операции

Вся система объектов Python состоит всего из трех основных операций: получения, установки и удаления атрибутов. Обычно эти операции доступны через точку (.) следующим образом:

```python
>>> s = Stock('GOOG', 100, 490.1)
>>> s.name    # получение
'GOOG'
>>> s.shares = 50    # установка
>>> del s.shares     # удаление
>>>
```

Эти три операции также доступны в виде функций. Например:

```python
>>> getattr(s, 'name')            # То же, что и s.name
'GOOG'
>>> setattr(s,'shares', 50)      # То же, что и s.shares = 50
>>> delattr(s,'shares')          # То же, что и del s.shares
>>>
```

Дополнительная функция `hasattr()` может быть использована для проверки наличия атрибута в объекте:

```python
>>> hasattr(s, 'name')
True
>>> hasattr(s, 'blah')
False
>>>
```
