# Табличный вывод

В упражнении 3.1 вы написали функцию `print_portfolio()`, которая выводила красиво отформатированную таблицу. Эта функция была настраирована специально для списка объектов `Stock`. Однако, с использованием техники из части (b) ее можно полностью обобщить для работы с любым списком объектов.

Создайте новый модуль под названием `tableformat.py`. В этом модуле напишите функцию `print_table()`, которая принимает последовательность (список) объектов, список имен атрибутов и выводит красиво отформатированную таблицу. Например:

```python
>>> import stock
>>> import tableformat
>>> portfolio = stock.read_portfolio('portfolio.csv')
>>> tableformat.print_table(portfolio, ['name','shares','price'])
      name     shares      price
---------- ---------- ----------
        AA        100       32.2
       IBM         50       91.1
       CAT        150      83.44
      MSFT        200      51.23
        GE         95      40.37
      MSFT         50       65.1
       IBM        100      70.44

>>> tableformat.print_table(portfolio,['shares','name'])
    shares       name
---------- ----------
       100         AA
        50        IBM
       150        CAT
       200       MSFT
        95         GE
        50       MSFT
       100        IBM
>>>
```

Для простоты функция `print_table()` должна выводить каждое поле в столбце шириной в 10 символов.
