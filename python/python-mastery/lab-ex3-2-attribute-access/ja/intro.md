# はじめに

この実験では、Python の属性アクセスについて学びます。`getattr()` や `setattr()` などの関数を使って、オブジェクトの属性を効果的に操作する方法を探ります。

さらに、バインドメソッド (bound methods) を試してみます。この実験ではこれらの概念を案内し、その過程で `tableformat.py` という名前のファイルを作成します。
