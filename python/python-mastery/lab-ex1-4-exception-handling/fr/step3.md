# Expérimentation interactive

Exécutez votre programme `pcost.py` et appelez directement la fonction `portfolio_cost()` à partir de l'interpréteur interactif.

```python
>>> portfolio_cost('/home/labex/project/portfolio.dat')
44671.15
>>> portfolio_cost('/home/labex/project/portfolio2.dat')
19908.75
>>>
```

Note : Pour ce faire, vous devrez peut-être exécuter python en utilisant l'option `-i`. Par exemple :

```bash
python3 -i pcost.py
```

Nous allons écrire beaucoup de programmes où vous définissez des fonctions et expérimentez de manière interactive. Assurez-vous de savoir comment faire cela.
