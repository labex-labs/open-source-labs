# Définition d'une fonction

Prenez le programme `pcost.py` que vous avez écrit dans le dernier laboratoire et convertissez-le en une fonction `portfolio_cost(filename)` qui prend un nom de fichier en entrée, lit les données du portefeuille dans ce fichier et renvoie le coût total du portefeuille sous forme d'un nombre à virgule flottante. Une fois que vous avez écrit la fonction, faites appel à elle dans votre programme en ajoutant simplement cette instruction à la fin :

```python
print(portfolio_cost('/home/labex/project/portfolio.dat'))
```

Exécutez votre programme et assurez-vous qu'il produit la même sortie que précédemment.
