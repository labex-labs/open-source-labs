# Добавление обработки ошибок

При написании программ, обрабатывающих данные, часто возникают ошибки, связанные с плохими данными (некорректными, отсутствующими полями и т.д.). Измените программу `pcost.py` так, чтобы она читала файл с данными `portfolio3.dat` и запускала его (подсказка: программа должна завершиться с ошибкой).

Немного измените функцию, чтобы она могла восстановиться от строк с плохими данными. Например, функции преобразования `int()` и `float()` вызывают исключение `ValueError`, если они не могут преобразовать входные данные. Используйте `try` и `except`, чтобы перехватить и вывести предупреждающее сообщение о строках, которые нельзя распарсить. Например:

```shell
Не удалось распарсить: 'C - 53.08\n'
Причина: недопустимый литерал для int() с основанием 10: '-'
Не удалось распарсить: 'DIS - 34.20\n'
Причина: недопустимый литерал для int() с основанием 10: '-'
...
```

Попробуйте снова запустить программу на файле `portfolio3.dat`. Она должна успешно запуститься несмотря на выведенные предупреждающие сообщения.
