# Взаимодейственная экспериментация

Запустите программу `pcost.py` и вызовите функцию `portfolio_cost()` непосредственно из интерактивного интерпретатора.

```python
>>> portfolio_cost('/home/labex/project/portfolio.dat')
44671.15
>>> portfolio_cost('/home/labex/project/portfolio2.dat')
19908.75
>>>
```

Примечание: Для этого вам может потребоваться запускать python с использованием параметра `-i`. Например:

```bash
python3 -i pcost.py
```

Мы будем писать много программ, где вы определяете функции и экспериментируете интерактивно. Убедитесь, что знаете, как это делать.
