# Definieren einer Funktion

Nehmen Sie das Programm `pcost.py`, das Sie im letzten Labor geschrieben haben, und konvertieren Sie es in eine Funktion `portfolio_cost(filename)`, die einen Dateinamen als Eingabe nimmt, die Portfolio-Daten in dieser Datei liest und den Gesamtkosten des Portfolios als Gleitkommazahl zurückgibt. Nachdem Sie die Funktion geschrieben haben, rufen Sie Ihre Funktion im Programm auf, indem Sie einfach diesen Ausdruck am Ende hinzufügen:

```python
print(portfolio_cost('/home/labex/project/portfolio.dat'))
```

Führen Sie Ihr Programm aus und stellen Sie sicher, dass es die gleiche Ausgabe wie zuvor erzeugt.
