# Interaktive Experimentation

Führen Sie Ihr `pcost.py`-Programm aus und rufen Sie die `portfolio_cost()`-Funktion direkt aus dem interaktiven Interpreter auf.

```python
>>> portfolio_cost('/home/labex/project/portfolio.dat')
44671,15
>>> portfolio_cost('/home/labex/project/portfolio2.dat')
19908,75
>>>
```

Hinweis: Um dies zu tun, müssen Sie möglicherweise Python mit der Option `-i` ausführen. Beispielsweise:

```bash
python3 -i pcost.py
```

Wir werden viele Programme schreiben, in denen Sie Funktionen definieren und interaktiv experimentieren. Stellen Sie sicher, dass Sie wissen, wie dies geht.
