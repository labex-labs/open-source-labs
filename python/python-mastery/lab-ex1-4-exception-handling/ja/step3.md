# 対話型実験

`pcost.py` プログラムを実行し、対話型インタプリタから直接 `portfolio_cost()` 関数を呼び出します。

```python
>>> portfolio_cost('/home/labex/project/portfolio.dat')
44671.15
>>> portfolio_cost('/home/labex/project/portfolio2.dat')
19908.75
>>>
```

注: これを行うには、`-i` オプションを使用してpythonを実行する必要がある場合があります。たとえば:

```bash
python3 -i pcost.py
```

関数を定義して対話的に実験する多くのプログラムを書く予定です。これがどのように行うかを確認しておきましょう。
