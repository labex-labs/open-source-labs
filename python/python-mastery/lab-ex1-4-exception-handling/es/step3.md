# Experimentación interactiva

Ejecute su programa `pcost.py` y llame directamente a la función `portfolio_cost()` desde el intérprete interactivo.

```python
>>> portfolio_cost('/home/labex/project/portfolio.dat')
44671.15
>>> portfolio_cost('/home/labex/project/portfolio2.dat')
19908.75
>>>
```

Nota: Para hacer esto, es posible que tenga que ejecutar python utilizando la opción `-i`. Por ejemplo:

```bash
python3 -i pcost.py
```

Vamos a escribir muchos programas donde defina funciones y experimente de manera interactiva. Asegúrese de saber cómo hacer esto.
