# Definiendo una función

Tome el programa `pcost.py` que escribió en la última práctica y conviértalo en una función `portfolio_cost(filename)` que tome un nombre de archivo como entrada, lea los datos del portafolio en ese archivo y devuelva el costo total del portafolio como un número de punto flotante. Una vez que haya escrito la función, haga que su programa llame a la función simplemente agregando esta declaración al final:

```python
print(portfolio_cost('/home/labex/project/portfolio.dat'))
```

Ejecute su programa y asegúrese de que produzca la misma salida que antes.
