# Iniciar Python

Inicie Python3 en su máquina. Asegúrese de que pueda escribir declaraciones simples como el programa "hola mundo":

```python
>>> print('Hello World')
Hello World
>>>
```

En gran parte de este curso, deseará asegurarse de que pueda trabajar desde el REPL interactivo como este. Si está trabajando en un entorno diferente, como IPython o Jupyter Notebooks, está bien.
