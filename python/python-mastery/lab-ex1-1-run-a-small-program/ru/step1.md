# Запустите Python

Запустите Python3 на вашем компьютере. Убедитесь, что вы можете вводить простые инструкции, такие как программа "Hello World":

```python
>>> print('Hello World')
Hello World
>>>
```

В большей части этого курса вы будете хотеть убедиться, что можете работать из интерактивного REPL так. Если вы работаете в другом окружении, таком как IPython или Jupyter Notebooks, это нормально.
