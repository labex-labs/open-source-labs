# Starte Python

Starte Python3 auf Ihrem Computer. Stellen Sie sicher, dass Sie einfache Anweisungen wie das "Hello World"-Programm eingeben können:

```python
>>> print('Hello World')
Hello World
>>>
```

In vielen Teilen dieses Kurses möchten Sie sicherstellen, dass Sie so von der interaktiven REPL arbeiten können. Wenn Sie in einer anderen Umgebung wie IPython oder Jupyter Notebooks arbeiten, ist das in Ordnung.
