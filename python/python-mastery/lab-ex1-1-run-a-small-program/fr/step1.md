# Lancer Python

Démarrez Python3 sur votre machine. Assurez-vous que vous pouvez taper des instructions simples telles que le programme "bonjour le monde" :

```python
>>> print('Hello World')
Hello World
>>>
```

Dans la majeure partie de ce cours, vous devrez vous assurer que vous pouvez travailler à partir de l'interpréteur interactif REPL comme ceci. Si vous travaillez dans un environnement différent tel que IPython ou Jupyter Notebooks, cela va bien.
