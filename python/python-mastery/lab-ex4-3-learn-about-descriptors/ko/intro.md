# 소개

이 랩에서는 객체의 속성 접근을 사용자 정의하는 강력한 메커니즘인 Python 의 descriptor 에 대해 배우게 됩니다. Descriptor 를 사용하면 속성 접근, 설정 및 삭제 방법을 정의하여 객체 동작을 제어하고 유효성 검사 로직을 구현할 수 있습니다.

이 랩의 목표는 descriptor 프로토콜 이해, 사용자 정의 descriptor 생성 및 사용, descriptor 를 사용한 데이터 유효성 검사 구현, 그리고 descriptor 구현 최적화를 포함합니다. 랩 과정에서 `descrip.py`, `stock.py`, 그리고 `validate.py`를 포함한 여러 파일을 생성하게 됩니다.
