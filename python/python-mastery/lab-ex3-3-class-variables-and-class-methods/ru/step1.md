# Предыдущая лабораторная работа

Экземпляры класса `Stock`, определенного в предыдущей лабораторной работе, обычно создаются следующим образом:

```python
>>> s = Stock('GOOG', 100, 490.1)
>>>
```

Однако функция `read_portfolio()` также создает экземпляры из строк данных, прочитанных из файлов. Например, используется код, подобный следующему:

```python
>>> import csv
>>> f = open('portfolio.csv')
>>> rows = csv.reader(f)
>>> headers = next(rows)
>>> row = next(rows)
>>> row
['AA', '100', '32.20']
>>> s = Stock(row[0], int(row[1]), float(row[2]))
>>>
```
