# Введение

**Цели:**

- Изучить делегирование генераторов

**Измененные файлы:** `cofollow.py`, `server.py`

Одной из потенциальных проблем в коде, который использует генераторы, является проблема скрытия деталей от пользователя и написания библиотек. Как правило, для управления всеми процессами требуется много низкоуровневых механизмов, и часто весьма неудобно напрямую показывать их пользователям.

Начиная с Python 3.3, можно использовать новое выражение `yield from`, чтобы делегировать генераторы другой функции. Это полезный способ упростить код, который использует генераторы.
