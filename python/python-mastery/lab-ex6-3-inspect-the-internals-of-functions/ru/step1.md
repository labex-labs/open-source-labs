# Инспектирование функций

Определите простую функцию:

```python
>>> def add(x,y):
       'Adds two things'
       return x+y

>>>
```

Выполните `dir()` для функции, чтобы посмотреть на ее атрибуты.

```python
>>> dir(add)
... посмотрите на результат...
>>>
```

Получите некоторые базовые сведения, такие как имя функции, имя определяющего модуля и строка документации.

```python
>>> add.__name__
'add'
>>> add.__module__
'__main__'
>>> add.__doc__
'Adds two things'
>>>
```

Атрибут `__code__` функции содержит сведения низкого уровня о реализации функции. Попробуйте рассмотреть это и определить количество требуемых аргументов и имена локальных переменных.
