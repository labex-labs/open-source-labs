# 関数の調査

単純な関数を定義する：

```python
>>> def add(x,y):
       'Adds two things'
       return x+y

>>>
```

関数に対して `dir()` を実行して、その属性を見る。

```python
>>> dir(add)
... 結果を見る...
>>>
```

関数名、定義モジュール名、ドキュメント文字列などの基本情報を取得する。

```python
>>> add.__name__
'add'
>>> add.__module__
'__main__'
>>> add.__doc__
'Adds two things'
>>>
```

関数の `__code__` 属性には、関数の実装に関する低レベルの情報が含まれています。これを見て、必要な引数の数とローカル変数の名前を特定できるかどうかを確認してみましょう。
