# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie man alle Schlüssel in einem Dictionary findet, die einen bestimmten Wert haben. Sie haben eine Python-Funktion namens `find_keys(dictionary, value)` geschrieben, die ein Dictionary und einen Wert als Argumente übernimmt und eine Liste aller Schlüssel im Dictionary zurückgibt, die den angegebenen Wert haben. Sie haben die `dictionary.items()`-Methode und eine Listenkomprehension verwendet, um dieses Problem zu lösen.
