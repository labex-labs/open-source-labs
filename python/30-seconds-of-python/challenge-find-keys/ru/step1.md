# Найти ключи по значению

## Задача

Напишите функцию на Python под названием `find_keys(dictionary, value)`, которая принимает в качестве аргументов словарь и значение и возвращает список всех ключей в словаре, которые имеют данное значение. Если нет ключей с заданным значением, функция должна вернуть пустой список.

Для решения этой задачи вы можете использовать метод `dictionary.items()`, который возвращает генератор, выдающий пары ключ-значение словаря. Затем вы можете использовать списочное выражение для фильтрации ключей, имеющих заданное значение.

## Пример

```python
ages = {
  'Peter': 10,
  'Isabel': 11,
  'Anna': 10,
}
find_keys(ages, 10) # [ 'Peter', 'Anna' ]
```

В этом примере функция `find_keys()` вызывается с словарем `ages` и значением `10`. Функция возвращает список ключей, которые имеют значение `10`, то есть `'Peter'` и `'Anna'`.
