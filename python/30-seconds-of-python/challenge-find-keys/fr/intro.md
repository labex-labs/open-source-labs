# Introduction

Les dictionnaires sont une structure de données fondamentale en Python. Ils sont utilisés pour stocker des paires clé-valeur, où chaque clé est unique. Parfois, nous avons besoin de trouver toutes les clés dans un dictionnaire qui ont une valeur spécifique. Dans ce défi, vous devrez écrire une fonction qui trouve toutes les clés dans un dictionnaire qui ont une valeur donnée.
