# Einführung

In dieser Herausforderung werden Sie eine Python-Funktion erstellen, die eine Liste als Argument nimmt und ein Dictionary zurückgibt, wobei die einzigartigen Werte der Liste als Schlüssel und ihre Häufigkeiten als Werte verwendet werden. Dies ist eine häufige Aufgabe in der Datenanalyse und kann in vielen verschiedenen Anwendungen nützlich sein.
