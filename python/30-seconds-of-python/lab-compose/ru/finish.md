# Резюме

В этом испытании вы узнали, как использовать функцию `functools.reduce()` для выполнения правого-налево композиции функций в Python. Вы также написали функцию под названием `compose()`, которая принимает одну или более функций в качестве аргументов и возвращает новую функцию, которая является результатом композиции входных функций с права налево.
