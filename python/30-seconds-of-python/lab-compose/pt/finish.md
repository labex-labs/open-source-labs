# Resumo

Neste desafio, você aprendeu como usar a função `functools.reduce()` para realizar a composição de funções da direita para a esquerda (right-to-left function composition) em Python. Você também escreveu uma função chamada `compose()` que aceita uma ou mais funções como argumentos e retorna uma nova função que é o resultado da composição das funções de entrada da direita para a esquerda.
