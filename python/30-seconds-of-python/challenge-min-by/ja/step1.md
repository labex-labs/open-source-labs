# 関数に基づいたリストの最小値を見つける

## 問題

`min_by(lst, fn)` という名前の関数を書きます。この関数は、リスト `lst` と関数 `fn` を引数として受け取ります。この関数は、与えられた関数を使ってリストの各要素をある値にマッピングし、その後最小値を返す必要があります。

## 例

```python
min_by([{ 'n': 4 }, { 'n': 2 }, { 'n': 8 }, { 'n': 6 }], lambda v : v['n']) # 2
```

上記の例では、`min_by()` 関数に辞書のリストと、各辞書から `'n'` キーの値を抽出するラムダ関数が渡されています。この関数は、リストの最小値である `2` を返します。
