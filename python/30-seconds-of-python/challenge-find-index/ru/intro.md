# Введение

В Python часто необходимо найти индекс первого элемента в списке, который удовлетворяет определенному условию. Это можно сделать с использованием списочного выражения, `enumerate()` и `next()`. В этом испытании вам будет поручено написать функцию, которая находит индекс первого элемента в списке, который удовлетворяет заданной тестирующей функции.
