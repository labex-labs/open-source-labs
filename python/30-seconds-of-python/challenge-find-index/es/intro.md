# Introducción

En Python, a menudo es necesario encontrar el índice del primer elemento en una lista que cumpla una cierta condición. Esto se puede lograr utilizando una comprensión de lista, `enumerate()` y `next()`. En este desafío, se te pedirá escribir una función que encuentre el índice del primer elemento en una lista que cumpla una función de prueba dada.
