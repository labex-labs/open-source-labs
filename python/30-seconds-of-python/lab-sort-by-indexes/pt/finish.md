# Resumo

Neste desafio, você foi encarregado de criar uma função que ordena uma lista com base em outra lista que contém os índices desejados. Ao usar as funções `zip()` e `sorted()`, você foi capaz de combinar e ordenar as duas listas com base nos valores de `indexes`. Em seguida, você usou uma list comprehension para obter o primeiro elemento de cada par do resultado e retornou a lista ordenada.
