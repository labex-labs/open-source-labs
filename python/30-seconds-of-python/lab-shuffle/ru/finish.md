# Резюме

В этом испытании вы написали функцию, которая перемешивает элементы в списке с использованием алгоритма Фишера-Йетса. Этот алгоритм представляет собой простой и эффективный способ случайным образом перемешать элементы в списке. Завершив это испытание, вы приобрели более глубокое понимание того, как работают списки в Python и как их манипулировать с использованием функций.
