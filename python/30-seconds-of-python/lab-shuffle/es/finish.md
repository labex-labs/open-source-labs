# Resumen

En este desafío, escribiste una función que baraja los elementos en una lista utilizando el algoritmo de Fisher-Yates. Este algoritmo es una forma simple y eficiente de aleatorizar el orden de los elementos en una lista. Al completar este desafío, has adquirido una comprensión más profunda de cómo funcionan las listas en Python y de cómo manipularlas utilizando funciones.
