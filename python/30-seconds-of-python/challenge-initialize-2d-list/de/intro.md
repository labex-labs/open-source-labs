# Einführung

In Python ist eine 2D-Liste eine Liste von Listen. Es ist eine nützliche Datenstruktur zur Darstellung von Gittern, Tabellen und Matrizen. Das Initialisieren einer 2D-Liste umfasst das Erstellen einer Liste von Listen mit einer angegebenen Breite und Höhe und das Initialisieren jedes Elements mit einem Standardwert.
