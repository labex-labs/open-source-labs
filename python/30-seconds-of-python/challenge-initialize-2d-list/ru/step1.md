# Инициализация 2D списка

## Задача

Напишите функцию `initialize_2d_list(w, h, val=None)`, которая инициализирует 2D список заданной ширины и высоты значением. Функция должна возвращать список из `h` строк, где каждая строка - это список длиной `w`, инициализированный значением `val`. Если `val` не указано, значение по умолчанию должно быть `None`.

## Пример

```python
initialize_2d_list(2, 2, 0) # [[0, 0], [0, 0]]
initialize_2d_list(3, 3, "x") # [["x", "x", "x"], ["x", "x", "x"], ["x", "x", "x"]]
initialize_2d_list(2, 3) # [[None, None], [None, None], [None, None]]
```
