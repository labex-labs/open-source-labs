# Mittelwert-Herausforderung

## Problemstellung

Schreiben Sie eine Funktion namens `average`, die zwei oder mehr Zahlen entgegennimmt und deren Mittelwert zurückgibt. Ihre Funktion sollte den folgenden Richtlinien folgen:

- Verwenden Sie `sum()`, um alle übergebenen `args` zu summieren, und dividieren Sie durch `len()`.
- Die Funktion sollte beliebig viele Argumente verarbeiten können.
- Die Funktion sollte einen Float zurückgeben.

## Beispiel

```python
average(*[1, 2, 3]) # 2,0
average(1, 2, 3) # 2,0
```
