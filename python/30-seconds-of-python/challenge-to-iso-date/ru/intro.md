# Введение

В Python даты могут быть представлены в различных форматах. Одним из наиболее распространенных форматов является формат ISO-8601, который представляет даты и время в стандартизованном виде. В этом испытании вы напишете функцию, которая преобразует дату в ее представление в формате ISO-8601.
