# Einführung

In Python können Datumswerte in verschiedenen Formaten dargestellt werden. Eines der am häufigsten verwendeten Formate ist das ISO-8601-Format, das Datumswerte und Zeitangaben auf eine standardisierte Weise repräsentiert. In dieser Herausforderung werden Sie eine Funktion schreiben, die ein Datum in seine ISO-8601-Darstellung umwandelt.
