# Introdução

Dicionários são uma estrutura de dados fundamental em Python, e são usados para armazenar pares chave-valor. Às vezes, podemos precisar combinar os valores de dois ou mais dicionários em um único dicionário. Neste desafio, você escreverá uma função que recebe dois ou mais dicionários como argumentos e retorna um novo dicionário que combina os valores dos dicionários de entrada.
