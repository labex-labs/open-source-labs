# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie Sie eine Funktion schreiben, die beliebig viele iterierbare Objekte oder Objekte mit einer Länge-Eigenschaft annimmt und das längste zurückgibt. Sie haben `max()` mit `len()` als `key` verwendet, um das Element mit der größten Länge zurückzugeben. Wenn mehrere Elemente die gleiche Länge haben, wird das erste zurückgegeben.
