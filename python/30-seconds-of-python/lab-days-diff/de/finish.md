# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie man den Unterschied zwischen zwei Daten in Tagen mithilfe des Moduls `datetime` in Python berechnet. Sie haben auch die Praxis geübt, eine Funktion zu schreiben, die zwei Daten als Eingabe nimmt und die Anzahl der Tage zwischen ihnen zurückgibt.
