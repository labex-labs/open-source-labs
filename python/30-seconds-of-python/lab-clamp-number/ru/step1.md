# Клемпнуть число

Напишите функцию `clamp_number(num, a, b)`, которая принимает три параметра:

- `num` (целое или вещественное число): число, которое нужно клемпнуть
- `a` (целое или вещественное число): нижняя граница диапазона
- `b` (целое или вещественное число): верхняя граница диапазона

Функция должна клемпнуть `num` в заданном диапазоне, включая граничные значения. Если `num` попадает в диапазон (`a`, `b`), верните `num`. В противном случае верните ближайшее число в диапазоне.

```python
def clamp_number(num, a, b):
  return max(min(num, max(a, b)), min(a, b))
```

```python
clamp_number(2, 3, 5) # 3
clamp_number(1, -1, -5) # -1
```
