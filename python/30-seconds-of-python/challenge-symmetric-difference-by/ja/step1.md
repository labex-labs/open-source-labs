# 関数に基づく対称差

## 問題

`symmetric_difference_by(a, b, fn)` という関数を書きます。この関数は、2 つのリスト `a` と `b` と関数 `fn` を受け取ります。この関数は、与えられた関数を両方のリストの各要素に適用した後、元のリストのどちらかには含まれているが、両方には含まれていないすべての要素を含む新しいリストを返す必要があります。

この問題を解くには、次の手順を辿ることができます。

1. 各リストの各要素に `fn` を適用することで `set` を作成します。
2. それぞれに対して、`fn` と組み合わせたリスト内包表記を使用して、他方の先に作成したセットに含まれていない値のみを残します。
3. 手順 2 で得られた 2 つのリストを連結します。

この関数は次のパラメータを持つ必要があります。

- `a`: 要素のリスト
- `b`: 要素のリスト
- `fn`: 要素を受け取り、新しい値を返す関数

この関数は、与えられた関数を両方のリストの各要素に適用した後、元のリストのどちらかには含まれているが、両方には含まれていないすべての要素を含む新しいリストを返す必要があります。

## 例

```python
from math import floor

assert symmetric_difference_by([2.1, 1.2], [2.3, 3.4], floor) == [1.2, 3.4]
```
