# Резюме

В этом испытании вы узнали, как создавать список с использованием итераторной функции и начального значения семени. Вы реализовали функцию `unfold`, которая принимает в качестве аргументов итераторную функцию и начальное значение семени и возвращает список, который создается генератором, с использованием итераторной функции.
