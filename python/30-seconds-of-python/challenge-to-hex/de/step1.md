# Zahl in Hexadezimalzahl

## Problemstellung

Schreiben Sie eine Funktion `to_hex(dec)`, die eine Dezimalzahl als Argument nimmt und ihre hexadezimale Darstellung zurückgibt. Ihre Funktion sollte die folgenden Schritte ausführen:

1. Verwenden Sie `hex()`, um die Dezimalzahl in ihre hexadezimale Entsprechung umzuwandeln.
2. Geben Sie die hexadezimale Darstellung zurück.

## Beispiel

```python
to_hex(41) # "0x29"
to_hex(332) # "0x14c"
```
