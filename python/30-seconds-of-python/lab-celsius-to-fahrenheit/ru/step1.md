# Градусы Цельсия в градусы Фаренгейта

Напишите функцию `celsius_to_fahrenheit`, которая принимает температуру в градусах Цельсия в качестве аргумента и возвращает температуру в градусах Фаренгейта. Используйте формулу конверсии `F = 1.8 * C + 32` для преобразования градусов Цельсия в градусы Фаренгейта.

```python
def celsius_to_fahrenheit(degrees):
  return ((degrees * 1.8) + 32)
```

```python
celsius_to_fahrenheit(180) # 356.0
```
