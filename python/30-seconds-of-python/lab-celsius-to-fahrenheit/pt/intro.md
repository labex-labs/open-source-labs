# Introdução

No campo da ciência, a temperatura é frequentemente medida em Celsius ou Fahrenheit. Celsius é a unidade métrica de medição de temperatura, enquanto Fahrenheit é a unidade imperial de medição de temperatura. Às vezes, é necessário converter Celsius para Fahrenheit ou vice-versa. Neste desafio, você será encarregado de converter Celsius para Fahrenheit usando uma fórmula de conversão.
