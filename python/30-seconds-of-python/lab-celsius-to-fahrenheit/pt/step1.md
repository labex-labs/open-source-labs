# Celsius para Fahrenheit

Escreva uma função `celsius_to_fahrenheit` que recebe uma temperatura em Celsius como argumento e retorna a temperatura em Fahrenheit. Use a fórmula de conversão `F = 1.8 * C + 32` para converter Celsius para Fahrenheit.

```python
def celsius_to_fahrenheit(degrees):
  return ((degrees * 1.8) + 32)
```

```python
celsius_to_fahrenheit(180) # 356.0
```
