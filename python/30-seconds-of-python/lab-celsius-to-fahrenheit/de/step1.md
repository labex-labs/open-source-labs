# Celsius in Fahrenheit

Schreibe eine Funktion `celsius_to_fahrenheit`, die eine Temperatur in Celsius als Argument nimmt und die Temperatur in Fahrenheit zurÃ¼ckgibt. Verwende die Umrechnungsformel `F = 1,8 * C + 32`, um Celsius in Fahrenheit umzurechnen.

```python
def celsius_to_fahrenheit(degrees):
  return ((degrees * 1.8) + 32)
```
