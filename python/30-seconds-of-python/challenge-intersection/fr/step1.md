# Défi d'intersection de listes

## Problème

Écrivez une fonction `list_intersection(a, b)` qui prend deux listes `a` et `b` en entrée et renvoie une nouvelle liste ne contenant que les éléments présents dans les deux listes `a` et `b`. Si aucun élément commun n'est trouvé, la fonction devrait renvoyer une liste vide.

## Exemple

```python
list_intersection([1, 2, 3], [4, 3, 2]) # [2, 3]
list_intersection([1, 2, 3], [4, 5, 6]) # []
list_intersection([1, 2, 3], [3, 3, 2]) # [2, 3]
```
