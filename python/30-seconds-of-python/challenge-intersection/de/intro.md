# Einführung

In Python kannst du mithilfe der Schnittmengeoperation von Mengen leicht die gemeinsamen Elemente zwischen zwei Listen finden. In dieser Herausforderung wirst du aufgefordert, eine Funktion zu schreiben, die zwei Listen als Eingabe nimmt und eine neue Liste zurückgibt, die nur die Elemente enthält, die in beiden Eingabelisten vorhanden sind.
