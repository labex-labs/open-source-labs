# Введение

В Python мы можем использовать цикл `for` для перебора списка и выполнения функции для каждого элемента. Однако, что если мы хотим начать с последнего элемента и двигаться назад? В этом испытании вам нужно будет создать функцию, которая будет выполнять предоставленную функцию один раз для каждого элемента списка, начиная с последнего элемента списка.
