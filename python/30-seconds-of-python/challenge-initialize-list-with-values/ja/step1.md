# 値でリストを初期化する

## 問題

`initialize_list_with_values(n, val=0)` という関数を書きましょう。この関数は 2 つのパラメータを受け取ります。

- `n`（整数）は作成するリストの長さを表します。
- `val`（整数）はリストを埋めるために使用する値を表します。`val` が指定されない場合、デフォルト値の `0` が使用されます。

この関数は、指定された値で埋められた長さ `n` のリストを返す必要があります。

## 例

```python
initialize_list_with_values(5, 2) # [2, 2, 2, 2, 2]
initialize_list_with_values(3) # [0, 0, 0]
```
