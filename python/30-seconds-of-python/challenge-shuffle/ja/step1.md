# リストをシャッフルする

## 問題

関数 `shuffle(lst)` を書きましょう。この関数は、入力としてリストを受け取り、同じ項目をランダムな順序で含む新しいリストを返します。関数は、Fisher-Yates アルゴリズムを使ってリスト内の項目をシャッフルする必要があります。

Fisher-Yates アルゴリズムは以下のように機能します。

1. リストの最後の項目から始めます。
2. 0 から現在のインデックスまでのランダムなインデックスを生成します。
3. 現在のインデックスにある項目と、ランダムなインデックスにある項目を交換します。
4. リストの次の項目に移動し、手順 2 - 3 を繰り返します。すべての項目がシャッフルされるまで続けます。

関数は元のリストを変更してはいけません。代わりに、シャッフルされた項目を含む新しいリストを作成する必要があります。

入力リストには少なくとも 1 つの項目が含まれていると仮定して構いません。

## 例

```python
foo = [1, 2, 3, 4, 5]
shuffled_foo = shuffle(foo)
print(shuffled_foo) # [3, 1, 4, 5, 2]
print(foo) # [1, 2, 3, 4, 5]
```
