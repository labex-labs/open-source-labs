# Введение

В Python список представляет собой упорядоченную коллекцию элементов. Иногда мы можем захотеть случайным образом перемешать элементы в списке. Это может быть полезно для игр, симуляций или любых других ситуаций, где мы хотим ввести элемент случайности. В этом испытании вы напишете функцию, которая перемешивает элементы в списке.
