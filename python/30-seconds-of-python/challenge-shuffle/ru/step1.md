# Перемешать список

## Задача

Напишите функцию `shuffle(lst)`, которая принимает список в качестве входных данных и возвращает новый список с теми же элементами в случайном порядке. Ваша функция должна использовать алгоритм Фишера-Йетса для перемешивания элементов в списке.

Алгоритм Фишера-Йетса работает следующим образом:

1. Начните с последнего элемента в списке.
2. Сгенерируйте случайный индекс между 0 и текущим индексом.
3. Обменяйте элемент в текущем индексе с элементом в случайном индексе.
4. Перейдите к следующему элементу в списке и повторите шаги 2-3, пока все элементы не будут перемешаны.

Ваша функция не должна изменять исходный список. Вместо этого она должна создать новый список с перемешанными элементами.

Можно предположить, что входной список будет содержать по крайней мере один элемент.

## Пример

```python
foo = [1, 2, 3, 4, 5]
shuffled_foo = shuffle(foo)
print(shuffled_foo) # [3, 1, 4, 5, 2]
print(foo) # [1, 2, 3, 4, 5]
```
