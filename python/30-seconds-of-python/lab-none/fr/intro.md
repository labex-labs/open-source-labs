# Introduction

En Python, nous pouvons utiliser la fonction `all()` pour vérifier si tous les éléments d'une liste sont évalués comme vrais (`truthy`). Mais que faire si nous voulons vérifier si chaque élément d'une liste est évalué comme faux (`falsy`)? Dans ce défi, vous devrez créer une fonction qui vérifie si chaque élément d'une liste est évalué comme faux.
