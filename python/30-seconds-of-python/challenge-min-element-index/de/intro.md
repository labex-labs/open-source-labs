# Einführung

In Python kannst du mithilfe der `min()`-Funktion leicht den kleinsten Wert in einer Liste finden. Was passiert aber, wenn du auch den Index dieses kleinsten Werts kennen musst? In dieser Herausforderung wirst du eine Funktion erstellen, die den Index des Elements mit dem kleinsten Wert in einer Liste zurückgibt.
