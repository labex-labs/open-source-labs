# Индекс минимального элемента

## Задача

Напишите функцию `min_element_index(arr)`, которая принимает в качестве аргумента список целых чисел `arr` и возвращает индекс элемента с минимальным значением в списке.

Для решения этой задачи вы можете использовать функцию `min()` для получения минимального значения в списке, а затем использовать метод `list.index()`, чтобы вернуть его индекс.

## Пример

```python
min_element_index([3, 5, 2, 6, 10, 7, 9]) # 2
```

В этом примере минимальное значение в списке `[3, 5, 2, 6, 10, 7, 9]` равно `2`, которое находится по индексу `2`.
