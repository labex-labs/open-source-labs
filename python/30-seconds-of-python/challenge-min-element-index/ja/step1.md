# 最小要素のインデックス

## 問題

整数のリスト `arr` を引数として受け取り、リストの中で最小値を持つ要素のインデックスを返す関数 `min_element_index(arr)` を作成してください。

この問題を解くには、`min()` 関数を使ってリストの最小値を取得し、その後 `list.index()` メソッドを使ってそのインデックスを返します。

## 例

```python
min_element_index([3, 5, 2, 6, 10, 7, 9]) # 2
```

この例では、リスト `[3, 5, 2, 6, 10, 7, 9]` の最小値は `2` で、それはインデックス `2` にあります。
