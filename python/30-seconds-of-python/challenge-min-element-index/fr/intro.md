# Introduction

En Python, vous pouvez facilement trouver la valeur minimale dans une liste en utilisant la fonction `min()`. Cependant, que se passe-t-il si vous avez également besoin de connaître l'index de cette valeur minimale? Dans ce défi, vous allez créer une fonction qui renvoie l'index de l'élément ayant la valeur minimale dans une liste.
