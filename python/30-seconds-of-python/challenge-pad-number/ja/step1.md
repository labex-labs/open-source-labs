# 数字をパディングする

## 問題

`pad_number(n, l)` という関数を書きましょう。この関数は、数字 `n` と長さ `l` を受け取り、パディングされた数字を表す文字列を返します。関数は、数字の先頭にゼロを追加して、その長さを `l` 桁にします。もし数字がすでに `l` 桁の長さであれば、関数はその数字を文字列として返します。

数字をパディングするには、`str.zfill()` メソッドを使うことができます。このメソッドは、長さを受け取り、文字列の先頭にゼロを追加してその長さになるまでパディングします。たとえば、`"7".zfill(6)` は `"000007"` を返します。

## 例

```python
pad_number(1234, 6) # '001234'
pad_number(7, 6) # '000007'
pad_number(123456789, 9) # '123456789'
```
