# Функция для головы списка

Напишите функцию на Python под названием `head(lst)`, которая принимает список в качестве аргумента и возвращает первый элемент списка.

Ваша функция должна соответствовать следующим требованиям:

- Функция должна называться `head`
- Функция должна принимать один аргумент, который является списком
- Функция должна возвращать первый элемент списка
- Функция не должна изменять исходный список

```python
def head(lst):
  return lst[0]
```

```python
head([1, 2, 3]) # 1
```
