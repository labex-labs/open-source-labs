# Função Cabeça da Lista (List Head Function)

Escreva uma função Python chamada `head(lst)` que recebe uma lista como argumento e retorna o primeiro elemento da lista.

Sua função deve ter os seguintes requisitos:

- A função deve ser chamada `head`
- A função deve receber um único argumento, que é uma lista
- A função deve retornar o primeiro elemento da lista
- A função não deve modificar a lista original

```python
def head(lst):
  return lst[0]
```

```python
head([1, 2, 3]) # 1
```
