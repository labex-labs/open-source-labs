# Función de la cabeza de la lista

Escribe una función de Python llamada `head(lst)` que tome una lista como argumento y devuelva el primer elemento de la lista.

Tu función debe cumplir con los siguientes requisitos:

- La función debe llamarse `head`
- La función debe tomar un solo argumento, que es una lista
- La función debe devolver el primer elemento de la lista
- La función no debe modificar la lista original

```python
def head(lst):
  return lst[0]
```

```python
head([1, 2, 3]) # 1
```
