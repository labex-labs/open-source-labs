# Проверить, существует ли ключ в словаре

## Задача

Напишите функцию `key_in_dict(d, key)`, которая принимает на вход словарь `d` и ключ `key` и возвращает `True`, если ключ существует в словаре, и `False` в противном случае.

## Пример

```python
d = {'one': 1, 'three': 3, 'five': 5, 'two': 2, 'four': 4}
key_in_dict(d, 'three') # True
key_in_dict(d,'six') # False
```
