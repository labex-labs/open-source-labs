# 最後の一致する値を見つける

## 問題

`find_last(lst, fn)` という関数を書きなさい。この関数は、リスト `lst` とテスト関数 `fn` を引数として受け取ります。この関数は、`fn` が `True` を返す `lst` の最後の要素の値を返す必要があります。もし要素がテスト関数を満たさなければ、関数は `None` を返す必要があります。

この問題を解くために、リスト内包表記と `next()` を使って、リストを逆順に反復処理し、テスト関数を満たす最後の要素を返してください。

## 例

```python
find_last([1, 2, 3, 4], lambda n: n % 2 == 1) # 3
find_last([2, 4, 6, 8], lambda n: n % 2 == 1) # None
```

最初の例では、関数は `3` を返す必要があります。なぜなら、それはリストの最後の奇数だからです。2 番目の例では、関数は `None` を返す必要があります。なぜなら、リストに奇数がないからです。
