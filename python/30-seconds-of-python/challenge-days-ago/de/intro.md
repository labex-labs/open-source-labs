# Einführung

In Python bietet das Modul `datetime` Klassen zum Arbeiten mit Daten und Zeiten. Eine der Klassen in diesem Modul ist `date`, die ein Datum (Jahr, Monat, Tag) repräsentiert und verschiedene Methoden zum Arbeiten mit Daten bietet. Eine weitere Klasse ist `timedelta`, die eine Zeitspanne oder den Unterschied zwischen zwei Daten oder Zeiten repräsentiert.
