# Défi des jours passés

## Problème

Votre tâche est d'écrire une fonction appelée `days_ago(n)` qui prend un entier `n` en argument et renvoie la date il y a `n` jours à partir d'aujourd'hui.

Pour résoudre ce problème, vous devez utiliser la classe `date` du module `datetime` pour obtenir la date actuelle et la classe `timedelta` pour soustraire `n` jours à partir de la date actuelle.

## Exemple

```python
days_ago(5) # date(2022, 11, 23)
```
