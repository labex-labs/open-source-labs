# Вызов преобразования радиан в градусы

## Задача

Напишите функцию на Python под названием `rads_to_degrees`, которая принимает один аргумент `rad`, который является вещественным числом, представляющим угол в радианах. Функция должна возвращать угол в градусах в виде вещественного числа. Вы можете использовать следующую формулу для преобразования угла из радиан в градусы:

```
degrees = radians * (180 / pi)
```

где `pi` - это константа, представляющая отношение длины окружности к диаметру круга, которая приблизительно равна 3,14159.

Ваша функция должна импортировать константу `pi` из модуля `math`.

## Пример

Вот пример того, как должна работать ваша функция:

```python
from math import pi

assert rads_to_degrees(pi / 2) == 90.0
```
