# Introduction

En mathématiques, les angles peuvent être mesurés en différents unités telles que les degrés et les radians. Les radians sont une unité de mesure pour les angles qui sont couramment utilisées en mathématiques et en physique. Cependant, parfois, nous devons convertir un angle de radians en degrés pour le rendre plus compréhensible. Dans ce défi, vous devrez écrire une fonction Python qui convertit un angle de radians en degrés.
