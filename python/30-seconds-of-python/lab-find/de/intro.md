# Einführung

In Python können wir eine Listenkomprehension und `next()` verwenden, um den Wert des ersten Elements in einer gegebenen Liste zu finden, das einer bereitgestellten Testfunktion entspricht. Dies kann in vielen Szenarien nützlich sein, beispielsweise beim Finden der ersten ungeraden Zahl in einer Liste oder des ersten Strings, der mit einem bestimmten Buchstaben beginnt.
