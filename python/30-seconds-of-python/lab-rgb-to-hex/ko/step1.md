# RGB 를 Hex 로 변환하기

색상의 빨강, 녹색, 파랑 구성 요소의 값을 나타내는 세 개의 정수를 입력으로 받아 16 진수 색상 코드를 나타내는 문자열을 반환하는 함수 `rgb_to_hex(r, g, b)`를 작성하십시오. 출력 문자열은 `RRGGBB` 형식이어야 하며, 여기서 `RR`, `GG`, `BB`는 각각 빨강, 녹색, 파랑 구성 요소를 나타내는 두 자리 16 진수 값입니다.

예를 들어, 입력 값이 `255`, `165`, `1`이면 출력은 문자열 `'FFA501'`이어야 합니다.

```python
def rgb_to_hex(r, g, b):
  return ('{:02X}' * 3).format(r, g, b)
```

```python
rgb_to_hex(255, 165, 1) # 'FFA501'
```
