# Удаление элементов списка с правого края

## Задача

Напишите функцию `drop_right(a, n = 1)`, которая принимает список `a` и необязательное целое число `n` и возвращает новый список с `n` элементами, удаленными из правого конца списка `a`. Если `n` не указано, функция должна удалить только последний элемент из списка.

## Пример

```python
drop_right([1, 2, 3]) # [1, 2]
drop_right([1, 2, 3], 2) # [1]
drop_right([1, 2, 3], 42) # []
```
