# Introdução

Em matemática, ângulos podem ser medidos em diferentes unidades, como graus e radianos. Radianos são uma unidade de medida para ângulos comumente utilizada em matemática e física. No entanto, às vezes podemos precisar converter um ângulo de radianos para graus para torná-lo mais compreensível. Neste desafio, você será solicitado a escrever uma função Python que converte um ângulo de radianos para graus.
