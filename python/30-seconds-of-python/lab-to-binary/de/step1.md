# Zahl in Binärzahl umwandeln

Schreiben Sie eine Funktion `to_binary(n)`, die eine Dezimalzahl als Eingabe nimmt und ihre binäre Darstellung als Zeichenfolge zurückgibt. Ihre Funktion sollte die folgenden Schritte ausführen:

1. Verwenden Sie `bin()`, um die Dezimalzahl in ihre binäre Entsprechung umzuwandeln.
2. Geben Sie die binäre Darstellung als Zeichenfolge zurück.

```python
def to_binary(n):
  return bin(n)
```

```python
to_binary(100) # 0b1100100
```
