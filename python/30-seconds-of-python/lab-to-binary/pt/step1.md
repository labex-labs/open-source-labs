# Número para Binário

Escreva uma função `to_binary(n)` que recebe um número decimal como entrada e retorna sua representação binária como uma string. Sua função deve realizar as seguintes etapas:

1.  Use `bin()` para converter o número decimal em seu equivalente binário.
2.  Retorne a representação binária como uma string.

```python
def to_binary(n):
  return bin(n)
```

```python
to_binary(100) # 0b1100100
```
