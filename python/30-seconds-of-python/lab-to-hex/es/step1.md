# Número a Hexadecimal

Escribe una función `to_hex(dec)` que tome un número decimal como argumento y devuelva su representación hexadecimal. Tu función debe realizar los siguientes pasos:

1. Utiliza `hex()` para convertir el número decimal a su equivalente hexadecimal.
2. Devuelve la representación hexadecimal.

```python
def to_hex(dec):
  return hex(dec)
```

```python
to_hex(41) # 0x29
to_hex(332) # 0x14c
```
