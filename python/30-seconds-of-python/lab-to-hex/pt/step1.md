# Número para Hexadecimal

Escreva uma função `to_hex(dec)` que recebe um número decimal como argumento e retorna sua representação hexadecimal. Sua função deve realizar as seguintes etapas:

1.  Use `hex()` para converter o número decimal para seu equivalente hexadecimal.
2.  Retorne a representação hexadecimal.

```python
def to_hex(dec):
  return hex(dec)
```

```python
to_hex(41) # 0x29
to_hex(332) # 0x14c
```
