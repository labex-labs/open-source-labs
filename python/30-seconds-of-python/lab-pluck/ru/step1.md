# Извлечение значений из списка словарей

Напишите функцию `pluck(lst, key)`, которая принимает на вход список словарей `lst` и ключ `key` и возвращает список значений, соответствующих заданному `ключу`.

Необходимо:

- Использовать списочное выражение и `dict.get()` для получения значения `ключ` для каждого словаря в `lst`.
- Функция должна возвращать пустой список, если входной список пуст или если заданный ключ отсутствует в любом из словарей.

```python
def pluck(lst, key):
  return [x.get(key) for x in lst]
```

```python
simpsons = [
  { 'name': 'lisa', 'age': 8 },
  { 'name': 'homer', 'age': 36 },
  { 'name':'marge', 'age': 34 },
  { 'name': 'bart', 'age': 10 }
]
pluck(simpsons, 'age') # [8, 36, 34, 10]
```
