# Introdução

Em Python, um dicionário (dictionary) é uma coleção de pares chave-valor (key-value). Às vezes, precisamos extrair valores específicos de uma lista de dicionários. Neste desafio, você precisa escrever uma função que recebe uma lista de dicionários e uma chave (key) como entrada e retorna uma lista de valores correspondentes à chave especificada.
