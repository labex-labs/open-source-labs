# Мили в километры

Напишите функцию на Python с именем `miles_to_km`, которая принимает параметр `miles` (число с плавающей точкой или целое число) и возвращает эквивалентное расстояние в километрах. Вы должны использовать следующую формулу преобразования: `km = mi * 1.609344`.

```python
def miles_to_km(miles):
  return miles * 1.609344
```

```python
miles_to_km(5.03) # 8.09500032
```
