# Milhas para km

Escreva uma função Python chamada `miles_to_km` que recebe um parâmetro `miles` (um float ou um inteiro) e retorna a distância equivalente em quilômetros. Você deve usar a seguinte fórmula de conversão: `km = mi * 1.609344`.

```python
def miles_to_km(miles):
  return miles * 1.609344
```

```python
miles_to_km(5.03) # 8.09500032
```
