# Meilen in Kilometer

Schreiben Sie eine Python-Funktion namens `miles_to_km`, die einen Parameter `miles` (eine Fließkommazahl oder eine Ganzzahl) akzeptiert und die entsprechende Entfernung in Kilometern zurückgibt. Sie sollten die folgende Umrechnungsformel verwenden: `km = mi * 1.609344`.

```python
def miles_to_km(miles):
  return miles * 1.609344
```

```python
miles_to_km(5.03) # 8.09500032
```
