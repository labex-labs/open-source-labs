# Miles en kilomètres

Écrivez une fonction Python appelée `miles_to_km` qui prend un paramètre `miles` (un nombre flottant ou un entier) et retourne la distance équivalente en kilomètres. Vous devriez utiliser la formule de conversion suivante : `km = mi * 1.609344`.

```python
def miles_to_km(miles):
  return miles * 1.609344
```

```python
miles_to_km(5.03) # 8.09500032
```
