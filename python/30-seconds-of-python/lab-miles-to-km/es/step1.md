# Millas a kilómetros

Escribe una función de Python llamada `miles_to_km` que tome un parámetro `miles` (un número flotante o entero) y devuelva la distancia equivalente en kilómetros. Debes utilizar la siguiente fórmula de conversión: `km = mi * 1.609344`.

```python
def miles_to_km(miles):
  return miles * 1.609344
```

```python
miles_to_km(5.03) # 8.09500032
```
