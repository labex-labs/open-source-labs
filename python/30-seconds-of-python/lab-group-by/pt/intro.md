# Introdução

Em Python, podemos agrupar os elementos de uma lista com base em uma função fornecida. Isso pode ser útil em muitas situações, como quando queremos agrupar uma lista de palavras por seu comprimento ou uma lista de números por sua paridade. Neste desafio, você será solicitado a escrever uma função que agrupa os elementos de uma lista com base em uma função fornecida.
