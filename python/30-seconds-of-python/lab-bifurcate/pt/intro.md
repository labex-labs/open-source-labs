# Introdução

Em Python, podemos dividir uma lista em dois grupos com base em um filtro fornecido. Isso pode ser feito usando uma compreensão de lista (list comprehension) e a função `zip()`. Neste desafio, você será solicitado a escrever uma função que recebe uma lista e um filtro como entrada e retorna duas listas, uma contendo os elementos que passam no filtro e a outra contendo os elementos que não passam.
