# Resumo

Neste desafio, você aprendeu como descapitalizar a primeira letra de uma string em Python. Você usou _list slicing_ (fatiamento de lista) e `str.lower()` para descapitalizar a primeira letra da string, e `str.join()` para combinar a primeira letra em minúsculas com o restante dos caracteres. Você também aprendeu como usar um parâmetro opcional para converter o restante da string para maiúsculas.
