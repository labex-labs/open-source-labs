# Millas a kilómetros

## Problema

Escribe una función de Python llamada `miles_to_km` que tome un parámetro `miles` (un número de punto flotante o un entero) y devuelva la distancia equivalente en kilómetros. Debes utilizar la siguiente fórmula de conversión: `km = mi * 1.609344`.

## Ejemplo

```python
miles_to_km(5.03) # 8.09500032
```
