# Мили в километры

## Задача

Напишите функцию на Python под названием `miles_to_km`, которая принимает параметр `miles` (тип `float` или `integer`) и возвращает эквивалентное расстояние в километрах. Используйте следующую формулу преобразования: `km = mi * 1.609344`.

## Пример

```python
miles_to_km(5.03) # 8.09500032
```
