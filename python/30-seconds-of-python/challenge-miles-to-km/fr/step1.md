# Miles à km

## Problème

Écrivez une fonction Python appelée `miles_to_km` qui prend un paramètre `miles` (un nombre à virgule flottante ou un entier) et renvoie la distance équivalente en kilomètres. Vous devriez utiliser la formule de conversion suivante : `km = mi * 1.609344`.

## Exemple

```python
miles_to_km(5.03) # 8.09500032
```
