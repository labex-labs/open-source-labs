# Задача на нахождение медианы

## Проблема

Напишите функцию на Python под названием `find_median`, которая принимает список чисел в качестве аргумента и возвращает медиану этого списка. Ваша функция должна выполнять следующие шаги:

1. Отсортируйте числа в списке с использованием `list.sort()`.
2. Найдите медиану, которая представляет собой либо средний элемент списка, если длина списка нечетная, либо среднее арифметическое двух средних элементов, если длина списка четная.
3. Верните медиану.

Ваша функция не должна использовать никакие встроенные библиотеки или функции Python, которые непосредственно решают данную задачу.

## Пример

```python
find_median([1, 2, 3]) # 2.0
find_median([1, 2, 3, 4]) # 2.5
```
