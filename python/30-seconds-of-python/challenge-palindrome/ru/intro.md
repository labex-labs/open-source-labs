# Введение

Палиндром — это слово, фраза, число или другая последовательность символов, которая читается одинаково в обоих направлениях. Например, "racecar" — это палиндром, потому что при обратном чтении оно по-прежнему пишется как "racecar". В этом испытании вы напишете функцию, которая проверяет, является ли заданная строка палиндромом.
