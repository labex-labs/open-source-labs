# Einführung

Ein Palindrom ist ein Wort, eine Phrase, eine Zahl oder eine andere Zeichenfolge, die sowohl von vorne nach hinten als auch von hinten nach vorne gelesen gleich ist. Beispielsweise ist "racecar" ein Palindrom, da es sich selbst umkehrt und immer noch "racecar" geschrieben ist. In dieser Herausforderung werden Sie eine Funktion schreiben, die überprüft, ob eine gegebene Zeichenfolge ein Palindrom ist.
