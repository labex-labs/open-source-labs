# 文字列のバイトサイズ

## 問題

文字列 `s` を引数として受け取り、そのバイトサイズを返す関数 `byte_size(s)` を作成します。文字列のバイトサイズとは、メモリに文字列を格納するために必要なバイト数です。文字列をバイトサイズに変換するには、特定のエンコード方式を使用して文字列をエンコードする必要があります。このチャレンジでは、UTF-8 エンコード方式を使用します。

文字列をバイトサイズに変換するには、次の手順に従います。

1. UTF-8 エンコード方式を使用して文字列をエンコードします。
2. エンコードされた文字列の長さを取得します。

関数は、エンコードされた文字列の長さを返す必要があります。

## 例

```python
byte_size('😀') # 4
byte_size('Hello World') # 11
```

上記の例では、文字列 `'😀'` のバイトサイズは 4 です。なぜなら、メモリに UTF-8 エンコードされた文字列を格納するには 4 バイトが必要だからです。文字列 `'Hello World'` のバイトサイズは 11 です。なぜなら、メモリに UTF-8 エンコードされた文字列を格納するには 11 バイトが必要だからです。
