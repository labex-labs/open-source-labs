# Einführung

In Python ist ein String eine Folge von Zeichen. Jedes Zeichen in einem String beansprucht einen bestimmten Speicherplatz. Die Größe des Speichers, die ein String beansprucht, wird als seine Byte-Größe bezeichnet. In dieser Herausforderung werden Sie eine Funktion schreiben, die einen String als Eingabe nimmt und seine Byte-Größe zurückgibt.
