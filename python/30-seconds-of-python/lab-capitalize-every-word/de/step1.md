# Jedes Wort groß schreiben

Schreiben Sie eine Funktion `capitalize_every_word(s)`, die einen String `s` als Argument nimmt und einen neuen String zurückgibt, in dem der erste Buchstabe jedes Wortes groß geschrieben ist.

```python
def capitalize_every_word(s):
  return s.title()
```

```python
capitalize_every_word('hello world!') # 'Hello World!'
```
