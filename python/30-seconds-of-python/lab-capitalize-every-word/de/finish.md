# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie man den ersten Buchstaben jedes Wortes in einem String mithilfe einer integrierten Funktion in Python groß schreiben kann. Diese Fähigkeit kann in verschiedenen Anwendungen nützlich sein, wie z. B. die Formatierung von Texten für die Anzeige oder die Verarbeitung von Benutzereingaben.
