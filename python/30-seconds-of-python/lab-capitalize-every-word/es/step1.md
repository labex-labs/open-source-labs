# Poner en mayúscula cada palabra

Escribe una función `capitalize_every_word(s)` que tome una cadena `s` como argumento y devuelva una nueva cadena con la primera letra de cada palabra en mayúscula.

```python
def capitalize_every_word(s):
  return s.title()
```

```python
capitalize_every_word('hello world!') # 'Hello World!'
```
