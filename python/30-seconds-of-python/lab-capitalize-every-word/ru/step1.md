# Сделать заглавной первую букву каждого слова

Напишите функцию `capitalize_every_word(s)`, которая принимает строку `s` в качестве аргумента и возвращает новую строку с заглавными первыми буквами каждого слова.

```python
def capitalize_every_word(s):
  return s.title()
```

```python
capitalize_every_word('hello world!') # 'Hello World!'
```
