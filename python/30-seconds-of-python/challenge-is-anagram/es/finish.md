# Resumen

En este desafío, has aprendido cómo comprobar si dos cadenas son anagramas la una de la otra. Has utilizado `str.isalnum()` para filtrar los caracteres no alfanuméricos, `str.lower()` para transformar cada carácter a minúsculas y `collections.Counter` para contar los caracteres resultantes de cada cadena y comparar los resultados.
