# Задача на проверку анаграмм строк

## Задача

Напишите функцию `is_anagram(s1, s2)`, которая принимает две строки в качестве аргументов и возвращает `True`, если они являются анаграммами друг друга, и `False` в противном случае. Функция должна игнорировать регистр букв, пробелы, знаки препинания и специальные символы.

Для решения этой задачи вы можете следовать следующим шагам:

1. Используйте `str.isalnum()`, чтобы отфильтровать неалфавитно-цифровые символы, и `str.lower()`, чтобы преобразовать каждый символ в нижний регистр.
2. Используйте `collections.Counter`, чтобы подсчитать получившиеся символы для каждой строки и сравнить результаты.

## Пример

```python
is_anagram('#anagram', 'Nag a ram!')  # True
is_anagram('hello', 'world')  # False
```
