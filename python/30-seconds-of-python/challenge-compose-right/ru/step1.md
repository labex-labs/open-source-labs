# Обратная композиция функций

## Задача

## Пример

```python
add = lambda x, y: x + y
square = lambda x: x * x
add_and_square = compose_right(add, square)
assert add_and_square(1, 2) == 9
```

В приведенном выше примере мы определяем две функции `add` и `square`. Затем мы используем функцию `compose_right` для создания новой функции `add_and_square`, которая сначала складывает два числа, а затем возводит результат в квадрат. Затем мы вызываем функцию `add_and_square` с аргументами `1` и `2`, что возвращает `9`.
