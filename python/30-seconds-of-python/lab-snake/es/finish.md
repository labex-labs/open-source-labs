# Resumen

En este reto, aprendiste cómo escribir una función de Python que convierte una cadena de texto a snake case (notación de serpiente). Utilizaste `re.sub()` para encontrar y reemplazar palabras en la cadena, `str.lower()` para convertirlas a minúsculas y `str.join()` para combinarlas utilizando guiones bajos como separador. Con este conocimiento, ahora puedes convertir fácilmente cadenas de texto a snake case en tus programas de Python.
