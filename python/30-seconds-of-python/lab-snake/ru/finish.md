# Резюме

В этом задании вы научились писать функцию на Python, которая преобразует строку в "snake case" (змеиный регистр). Вы использовали `re.sub()` для поиска и замены слов в строке, `str.lower()` для преобразования их в нижний регистр и `str.join()` для объединения слов с использованием подчеркивания в качестве разделителя. С этими знаниями вы теперь можете легко преобразовывать строки в "snake case" в своих Python-программах.
