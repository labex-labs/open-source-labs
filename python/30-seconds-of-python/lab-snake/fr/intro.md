# Introduction

En Python, le snake case est une convention de nommage où les mots sont séparés par des underscores (tirets bas). Cette convention est couramment utilisée pour nommer les variables, les fonctions et autres identifiants. Cependant, parfois nous pouvons avoir des chaînes de caractères qui ne sont pas en snake case et nous devons les convertir. Dans ce défi, vous aurez pour tâche d'écrire une fonction Python qui convertit une chaîne de caractères en snake case.
