# Summary

Dans ce défi, vous avez appris à écrire une fonction Python qui convertit une chaîne de caractères en snake case. Vous avez utilisé `re.sub()` pour trouver et remplacer les mots dans la chaîne, `str.lower()` pour les convertir en minuscules et `str.join()` pour les combiner en utilisant des underscores (tirets bas) comme séparateur. Grâce à cette connaissance, vous pouvez maintenant facilement convertir des chaînes de caractères en snake case dans vos programmes Python.
