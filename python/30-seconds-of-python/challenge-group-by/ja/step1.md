# リスト要素をグループ化する

## 問題

関数 `group_by(lst, fn)` を作成してください。この関数は、リスト `lst` と関数 `fn` を引数として受け取り、辞書を返します。辞書のキーは、`fn` を `lst` の要素に適用した結果であり、値は、`fn` を適用したときに対応するキーを生成する `lst` の要素のリストです。

たとえば、数値のリスト `[6.1, 4.2, 6.3]` があり、それらを整数部分でグループ化したい場合、グループ化関数として `math` モジュールの `floor` 関数を使用できます。期待される出力は `{4: [4.2], 6: [6.1, 6.3]}` になります。

## 例

```python
from math import floor

group_by([6.1, 4.2, 6.3], floor) # {4: [4.2], 6: [6.1, 6.3]}
group_by(['one', 'two', 'three'], len) # {3: ['one', 'two'], 5: ['three']}
```
