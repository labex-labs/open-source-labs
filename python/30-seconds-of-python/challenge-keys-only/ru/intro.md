# Введение

В Python словарь - это коллекция пар ключ-значение. Каждый ключ уникален и используется для доступа к соответствующему значению. Иногда мы можем захотеть извлечь только ключи из словаря. В этом испытании вам нужно создать функцию, которая принимает на вход плоский словарь и возвращает список всех его ключей.
