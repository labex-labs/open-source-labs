# Einführung

In Python ist ein Dictionary eine Sammlung von Schlüssel-Wert-Paaren. Jeder Schlüssel ist einzigartig und wird verwendet, um seinen entsprechenden Wert zuzugreifen. Manchmal müssen wir möglicherweise nur die Schlüssel aus einem Dictionary extrahieren. In dieser Herausforderung haben Sie die Aufgabe, eine Funktion zu erstellen, die ein flaches Dictionary als Eingabe nimmt und eine Liste aller seiner Schlüssel zurückgibt.
