# 16 進数から RGB への変換

## 問題

16 進数の色コードを文字列として受け取り、その RGB コンポーネントに対応する整数のタプルを返す関数 `hex_to_rgb(hex_code)` を作成します。この関数は以下の手順を実行する必要があります。

1. `int()` とリストスライス表記を組み合わせたリスト内包表記を使用して、16 進数の文字列を元に RGB コンポーネントを取得します。
2. 得られたリストを `tuple()` を使ってタプルに変換します。

## 例

```python
hex_to_rgb('FFA501') # (255, 165, 1)
```
