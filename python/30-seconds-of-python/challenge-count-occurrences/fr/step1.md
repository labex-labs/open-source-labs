# Compter les occurrences - Défi

## Problème

Écrivez une fonction `count_occurrences(lst, val)` qui prend une liste `lst` et une valeur `val` en arguments et renvoie le nombre d'occurrences de `val` dans `lst`. Votre fonction devrait utiliser la méthode intégrée `list.count()` pour compter le nombre d'occurrences.

## Exemple

```python
count_occurrences([1, 1, 2, 1, 2, 3], 1) # 3
```
