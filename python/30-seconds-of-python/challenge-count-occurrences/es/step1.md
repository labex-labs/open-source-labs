# Desafío de conteo de ocurrencias

## Problema

Escribe una función `count_occurrences(lst, val)` que tome una lista `lst` y un valor `val` como argumentos y devuelva la cantidad de veces que aparece `val` en `lst`. Tu función debe utilizar el método `list.count()` incorporado para contar la cantidad de ocurrencias.

## Ejemplo

```python
count_occurrences([1, 1, 2, 1, 2, 3], 1) # 3
```
