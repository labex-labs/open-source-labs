# リストを辞書にマッピングする

## 問題

`map_dictionary(itr, fn)` という名前のPython関数を書きます。この関数には2つのパラメータがあります。

- `itr`：値のリスト
- `fn`：入力として値を取り、出力として値を返す関数

この関数は、キーと値のペアが元の値をキーとし、関数の結果を値とする辞書を返す必要があります。

この問題を解くには、次の手順に従います。

1. `map()` を使って、関数 `fn` をリストの各値に適用します。
2. `zip()` を使って、元の値と `fn` によって生成された値をペアにします。
3. `dict()` を使って適切な辞書を返します。

## 例

```python
map_dictionary([1, 2, 3], lambda x: x * x) # { 1: 1, 2: 4, 3: 9 }
```

この例では、`map_dictionary()` 関数はリスト `[1, 2, 3]` とラムダ関数 `lambda x: x * x` を入力として受け取ります。ラムダ関数は入力値を2乗します。この関数は、キーがリストの元の値で、値が2乗された値である辞書 `{ 1: 1, 2: 4, 3: 9 }` を返します。
