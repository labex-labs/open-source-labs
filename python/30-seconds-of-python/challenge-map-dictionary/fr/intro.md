# Introduction

En Python, un dictionnaire est une collection de paires clé-valeur. Parfois, nous avons besoin de créer un dictionnaire à partir d'une liste où les clés sont les éléments de la liste et les valeurs sont le résultat de l'application d'une fonction à ces éléments. Dans ce défi, vous devrez créer une fonction qui map les valeurs d'une liste à un dictionnaire en utilisant une fonction.
