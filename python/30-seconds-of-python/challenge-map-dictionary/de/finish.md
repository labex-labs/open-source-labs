# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie man ein Dictionary aus einer Liste erstellt, wobei die Schlüssel die Elemente der Liste sind und die Werte das Ergebnis der Anwendung einer Funktion auf diese Elemente. Sie haben die Funktionen `map()`, `zip()` und `dict()` verwendet, um das Problem zu lösen.
