# Введение

В Python словарь представляет собой коллекцию пар ключ-значение. Иногда нам нужно создать словарь из списка, где ключами являются элементы списка, а значениями — результат применения функции к этим элементам. В этом испытании вы создадите функцию, которая сопоставляет значения списка со словарем с использованием функции.
