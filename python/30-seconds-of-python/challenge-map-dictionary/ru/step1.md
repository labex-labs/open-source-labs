# Сопоставление списка со словарем

## Задача

Напишите функцию на Python под названием `map_dictionary(itr, fn)`, которая принимает два параметра:

- `itr`: список значений
- `fn`: функция, которая принимает значение в качестве входных данных и возвращает значение в качестве результата

Функция должна возвращать словарь, в котором пары ключ-значение состоят из исходного значения в качестве ключа и результата функции в качестве значения.

Для решения этой задачи следуйте шагам:

1. Используйте `map()`, чтобы применить `fn` к каждому значению списка.
2. Используйте `zip()`, чтобы сопоставить исходные значения с значениями, полученными с помощью `fn`.
3. Используйте `dict()`, чтобы вернуть соответствующий словарь.

## Пример

```python
map_dictionary([1, 2, 3], lambda x: x * x) # { 1: 1, 2: 4, 3: 9 }
```

В этом примере функция `map_dictionary()` принимает список `[1, 2, 3]` и лямбда-функцию `lambda x: x * x` в качестве входных данных. Лямбда-функция возводит входное значение в квадрат. Функция возвращает словарь `{ 1: 1, 2: 4, 3: 9 }`, где ключами являются исходные значения списка, а значениями — квадраты этих значений.
