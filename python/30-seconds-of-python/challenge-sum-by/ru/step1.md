# Сумма списка на основе функции

## Задача

Напишите функцию `sum_by(lst, fn)`, которая принимает список `lst` и функцию `fn` в качестве аргументов. Функция должна применить функцию к каждому элементу списка для получения значения и вернуть сумму этих значений.

## Пример

```python
sum_by([{ 'n': 4 }, { 'n': 2 }, { 'n': 8 }, { 'n': 6 }], lambda v : v['n']) # 20
```

В приведенном выше примере функция `sum_by()` принимает список словарей и лямбда-функцию, которая извлекает значение ключа `'n'` из каждого словаря. Функция применяет каждому словарю его значение `'n'` и возвращает сумму значений, которая равна `20`.
