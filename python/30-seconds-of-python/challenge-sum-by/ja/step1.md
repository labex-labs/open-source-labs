# 関数に基づくリストの合計

## 問題

`sum_by(lst, fn)` という関数を書きましょう。この関数は、リスト `lst` と関数 `fn` を引数として取ります。この関数は、提供された関数を使ってリストの各要素を値にマップし、その値の合計を返す必要があります。

## 例

```python
sum_by([{ 'n': 4 }, { 'n': 2 }, { 'n': 8 }, { 'n': 6 }], lambda v : v['n']) # 20
```

上記の例では、`sum_by()` 関数は辞書のリストと、各辞書から `'n'` キーの値を抽出するラムダ関数を取ります。この関数は各辞書をその `'n'` 値にマップし、値の合計である `20` を返します。
