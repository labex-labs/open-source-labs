# Удаление элементов списка

Напишите функцию `take(itr, n=1)`, которая принимает список `itr` и целое число `n` в качестве аргументов и возвращает новый список с `n` элементами, удаленными из начала списка. Если `n` больше длины списка, верните исходный список.

```python
def take(itr, n = 1):
  return itr[:n]
```

```python
take([1, 2, 3], 5) # [1, 2, 3]
take([1, 2, 3], 0) # []
```
