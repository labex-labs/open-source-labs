# 関数に基づくリストの共通部分

## 問題

`intersection_by(a, b, fn)` という関数を書きなさい。この関数は、2 つのリスト `a` と `b` と、関数 `fn` を受け取ります。この関数は、両方のリストの各要素に提供された関数を適用した後、両方のリストに存在する要素のリストを返す必要があります。

### 入力

- 2 つのリスト `a` と `b` (1 <= len(a), len(b) <= 1000)
- 1 つの引数を受け取り、値を返す関数 `fn`

### 出力

- 両方のリストの各要素に提供された関数を適用した後、両方のリストに存在する要素のリスト。

## 例

```python
intersection_by([2.1, 1.2], [2.3, 3.4], floor) # [2.1]
```

### 注

上記の例では、関数 `floor()` が両方のリストの各要素に適用されます。結果の集合は `{2, 3}` と `{2, 1}` です。これらの集合の共通部分は `{2}` であり、これがリストとして返されます。
