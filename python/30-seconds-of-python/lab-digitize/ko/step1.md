# 숫자 자릿수화 (Digitize Number)

`digitize(n)` 함수를 작성하세요. 이 함수는 음이 아닌 정수 `n`을 입력으로 받아 자릿수의 리스트를 반환합니다. 이 함수는 다음 단계를 수행하여 이를 달성해야 합니다.

1. 입력 숫자 `n`을 문자열로 변환합니다.
2. `map()` 함수를 `int` 함수와 함께 사용하여 문자열의 각 문자를 정수로 변환합니다.
3. 결과 정수 리스트를 반환합니다.

예를 들어, 입력 숫자가 `123`인 경우 함수는 리스트 `[1, 2, 3]`을 반환해야 합니다.

```python
def digitize(n):
  return list(map(int, str(n)))
```

```python
digitize(123) # [1, 2, 3]
```
