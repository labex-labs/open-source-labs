# Einführung

Wörterbücher sind eine leistungsstarke Datenstruktur in Python, die es Ihnen ermöglicht, Schlüssel-Wert-Paare zu speichern. Manchmal müssen Sie den Schlüssel finden, der mit einem bestimmten Wert in einem Wörterbuch assoziiert ist. In dieser Herausforderung werden Sie eine Funktion schreiben, die ein Wörterbuch und einen Wert als Eingabe erhält und den ersten Schlüssel im Wörterbuch zurückgibt, der den angegebenen Wert hat.
