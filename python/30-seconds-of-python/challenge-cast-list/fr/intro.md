# Introduction

En Python, parfois nous avons besoin de convertir une valeur en une liste. Cependant, si la valeur est déjà une liste, nous ne voulons pas créer une liste imbriquée. Dans ce défi, vous allez créer une fonction qui prend une valeur et la renvoie sous forme de liste, sauf si elle est déjà une liste ou un autre itérable.
