# Резюме

В этом испытании вы узнали, как создать функцию, которая принимает словарь и функцию в качестве аргументов и возвращает новый словарь с теми же ключами, что и исходный словарь, и значениями, сгенерированными путём запуска предоставленной функции для каждого значения. Это полезный метод для преобразования значений словаря.
