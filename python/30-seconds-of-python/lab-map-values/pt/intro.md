# Introdução

Em Python, um dicionário (dictionary) é uma coleção de pares chave-valor. Às vezes, precisamos transformar os valores de um dicionário usando uma função. Este desafio testará sua capacidade de criar uma função que recebe um dicionário e uma função como argumentos e retorna um novo dicionário com as mesmas chaves do dicionário original e valores gerados pela execução da função fornecida para cada valor.
