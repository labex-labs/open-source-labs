# Número a binario

## Problema

Escribe una función `to_binary(n)` que tome un número decimal como entrada y devuelva su representación binaria como una cadena. Tu función debe realizar los siguientes pasos:

1. Utiliza `bin()` para convertir el número decimal en su equivalente binario.
2. Devuelve la representación binaria como una cadena.

## Ejemplo

```python
to_binary(100) # '0b1100100'
to_binary(42) # '0b101010'
to_binary(255) # '0b11111111'
```
