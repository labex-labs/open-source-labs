# Einführung

In der Informatik ist das Binärsystem ein Zahlensystem zur Basis 2, das numerische Werte mit zwei Symbolen, typischerweise 0 und 1, repräsentiert. Das Binärsystem wird in der Informatik und der Digitalelektronik weit verbreitet eingesetzt. In dieser Herausforderung werden Sie eine Python-Funktion schreiben, die eine Dezimalzahl als Eingabe nimmt und ihre binäre Darstellung zurückgibt.
