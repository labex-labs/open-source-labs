# Conversion d'un nombre en binaire

## Problème

Écrivez une fonction `to_binary(n)` qui prend un nombre décimal en entrée et renvoie sa représentation binaire sous forme de chaîne de caractères. Votre fonction doit effectuer les étapes suivantes :

1. Utilisez `bin()` pour convertir le nombre décimal en son équivalent binaire.
2. Renvoyez la représentation binaire sous forme de chaîne de caractères.

## Exemple

```python
to_binary(100) # '0b1100100'
to_binary(42) # '0b101010'
to_binary(255) # '0b11111111'
```
