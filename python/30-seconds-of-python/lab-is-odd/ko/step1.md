# 숫자가 홀수인지 확인하기

`is_odd`라는 함수를 작성하세요. 이 함수는 단일 정수를 인수로 받아 숫자가 홀수이면 `True`를 반환하고, 짝수이면 `False`를 반환합니다. 함수는 다음 단계를 수행해야 합니다.

1. 나머지 연산자 (`%`) 를 사용하여 숫자가 짝수인지 홀수인지 확인합니다.
2. 숫자가 홀수이면 `True`를 반환합니다. 숫자가 짝수이면 `False`를 반환합니다.

```python
def is_odd(num):
  return num % 2 != 0
```

```python
is_odd(3) # True
```
