# Проверить, является ли число нечетным

Напишите функцию под названием `is_odd`, которая принимает целое число в качестве аргумента и возвращает `True`, если число нечетное, и `False`, если число четное. Ваша функция должна выполнять следующие шаги:

1. Используйте оператор модуля (`%`), чтобы проверить, является ли число четным или нечетным.
2. Если число нечетное, верните `True`. Если число четное, верните `False`.

```python
def is_odd(num):
  return num % 2!= 0
```

```python
is_odd(3) # True
```
