# 数が奇数であるかチェックする

`is_odd` という名前の関数を書きます。この関数は、単一の整数を引数として受け取り、その数が奇数の場合は `True` を返し、偶数の場合は `False` を返します。あなたの関数は次の手順を実行する必要があります。

1. 剰余 (`%`) 演算子を使用して、その数が偶数か奇数かをチェックします。
2. その数が奇数の場合は `True` を返します。その数が偶数の場合は `False` を返します。

```python
def is_odd(num):
  return num % 2!= 0
```

```python
is_odd(3) # True
```
