# Resumo

Neste desafio, você aprendeu a escrever uma função que verifica se um determinado número é ímpar ou par. Isso pode ser útil em vários cenários de programação, como ao trabalhar com loops, instruções condicionais ou operações matemáticas. Lembre-se de usar o operador de módulo para verificar se um número é ímpar ou par e retornar `True` se o número for ímpar e `False` se o número for par.
