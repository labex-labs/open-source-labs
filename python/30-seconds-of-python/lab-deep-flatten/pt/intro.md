# Introdução

Em Python, uma lista pode conter outras listas como elementos, e essas listas podem conter ainda mais listas como elementos, formando uma estrutura aninhada. O processo de _flattening_ (achatamento) de uma lista aninhada significa convertê-la em uma lista unidimensional, onde todos os elementos estão no mesmo nível. Neste desafio, você será solicitado a escrever uma função que realiza o _deep flatten_ (achatamento profundo) de uma lista, o que significa que ela achatará todas as listas aninhadas recursivamente.
