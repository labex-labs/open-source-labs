# Задача "Сумма степеней"

## Проблема

Напишите функцию на Python под названием `sum_of_powers`, которая принимает три параметра:

- `end` - целое число, представляющее конец диапазона (включительно)
- `power` - целое число, представляющее степень, в которую должно быть возведено каждое число в диапазоне (значение по умолчанию - 2)
- `start` - целое число, представляющее начало диапазона (значение по умолчанию - 1)

Функция должна возвращать сумму степеней всех чисел от `start` до `end` (оба числа включены).

Для решения этой задачи вы можете следовать следующим шагам:

1. Используйте `range()` в сочетании с списочным включением для создания списка элементов в заданном диапазоне, возведенных в заданную `power`.
2. Используйте `sum()`, чтобы сложить значения вместе.

## Пример

```python
sum_of_powers(10) # возвращает 385
sum_of_powers(10, 3) # возвращает 3025
sum_of_powers(10, 3, 5) # возвращает 2925
```
