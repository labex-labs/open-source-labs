# 累乗和のチャレンジ

## 問題

`sum_of_powers` という名前の Python 関数を書きましょう。この関数には 3 つのパラメータが渡されます。

- `end` - 範囲の終端を表す整数（両端含む）
- `power` - 範囲内の各数を累乗する指数を表す整数（デフォルト値は 2）
- `start` - 範囲の始端を表す整数（デフォルト値は 1）

この関数は、`start` から `end` までのすべての数の累乗の和を返す必要があります（両端含む）。

この問題を解くには、次の手順を辿ることができます。

1. `range()` とリスト内包表記を組み合わせて、指定された `power` 乗にした所望の範囲の要素のリストを作成します。
2. `sum()` を使って値を合計します。

## 例

```python
sum_of_powers(10) # 385 を返す
sum_of_powers(10, 3) # 3025 を返す
sum_of_powers(10, 3, 5) # 2925 を返す
```
