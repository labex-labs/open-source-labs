# Ключ максимального значения

## Задача

Напишите функцию `key_of_max(d)`, которая принимает словарь `d` в качестве аргумента и возвращает ключ максимального значения в словаре. Если есть несколько ключей с одинаковым максимальным значением, верните любой из них.

Для решения этой задачи вы можете использовать функцию `max()` с параметром `key`, установленным на `dict.get()`. Это вернет ключ максимального значения в словаре.

## Пример

```python
key_of_max({'a':4, 'b':0, 'c':13}) # 'c'
```
