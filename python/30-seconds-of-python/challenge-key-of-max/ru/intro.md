# Введение

В Python словари - это полезная структура данных, которая позволяет хранить пары ключ-значение. Иногда вам может потребоваться найти ключ максимального значения в словаре. В этом испытании вы напишете функцию, которая принимает словарь в качестве аргумента и возвращает ключ максимального значения в словаре.
