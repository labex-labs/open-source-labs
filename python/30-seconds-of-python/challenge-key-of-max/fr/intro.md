# Introduction

En Python, les dictionnaires sont une structure de données utile qui vous permet de stocker des paires clé-valeur. Parfois, vous pouvez avoir besoin de trouver la clé de la valeur maximale dans un dictionnaire. Dans ce défi, vous écrirez une fonction qui prend un dictionnaire en argument et renvoie la clé de la valeur maximale dans le dictionnaire.
