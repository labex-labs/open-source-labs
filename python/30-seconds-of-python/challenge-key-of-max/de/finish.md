# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie man mithilfe von Python den Schlüssel des maximalen Werts in einem Dictionary findet. Sie können die `max()`-Funktion mit dem `key`-Parameter auf `dict.get()` setzen, um den Schlüssel des maximalen Werts im angegebenen Dictionary zu finden und zurückzugeben.
