# Einführung

In Python sind Dictionaries eine nützliche Datenstruktur, die es Ihnen ermöglicht, Schlüssel-Wert-Paare zu speichern. Manchmal müssen Sie den Schlüssel des maximalen Werts in einem Dictionary finden. In dieser Herausforderung werden Sie eine Funktion schreiben, die ein Dictionary als Argument nimmt und den Schlüssel des maximalen Werts im Dictionary zurückgibt.
