# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie Sie in Python zwei oder mehr Listen zu einer Liste von Listen zusammenführen. Sie haben auch gelernt, wie Sie die `max()`-Funktion, Listenverständnis und die `range()`-Funktion verwenden, um das Problem zu lösen. Denken Sie daran, den `fill_value`-Parameter zu verwenden, um die fehlenden Werte in den kürzeren Listen auszufüllen.
