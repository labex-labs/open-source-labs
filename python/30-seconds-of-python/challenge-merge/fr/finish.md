# Résumé

Dans ce défi, vous avez appris à fusionner deux ou plusieurs listes en une liste de listes en Python. Vous avez également appris à utiliser la fonction `max()`, la compréhension de liste et la fonction `range()` pour résoudre le problème. N'oubliez pas d'utiliser le paramètre `fill_value` pour remplir les valeurs manquantes dans les listes les plus courtes.
