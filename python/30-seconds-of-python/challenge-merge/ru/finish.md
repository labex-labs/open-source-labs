# Резюме

В этом испытании вы узнали, как объединить два или более списков в список списков в Python. Вы также узнали, как использовать функцию `max()`, списочное выражение и функцию `range()` для решения задачи. Не забудьте использовать параметр `fill_value` для заполнения пропущенных значений в более коротких списках.
