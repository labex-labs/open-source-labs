# Einführung

In Python kannst du das Modul `datetime` verwenden, um mit Daten und Zeiten umzugehen. Eine häufige Aufgabe besteht darin, zu überprüfen, ob ein bestimmtes Datum ein Werktag oder ein Wochenende ist. In dieser Herausforderung schreibst du eine Funktion, die ein Datum als Eingabe erhält und `True` zurückgibt, wenn es ein Werktag ist, und `False`, wenn es ein Wochenende ist.
