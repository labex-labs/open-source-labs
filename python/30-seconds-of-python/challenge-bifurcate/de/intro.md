# Einführung

In Python können wir eine Liste basierend auf einem gegebenen Filter in zwei Gruppen unterteilen. Dies kann mit einer Listenkomprehension und der `zip()`-Funktion erreicht werden. In dieser Herausforderung müssen Sie eine Funktion schreiben, die eine Liste und einen Filter als Eingabe nimmt und zwei Listen zurückgibt, wobei eine die Elemente enthält, die den Filter überstehen, und die andere die Elemente, die es nicht tun.
