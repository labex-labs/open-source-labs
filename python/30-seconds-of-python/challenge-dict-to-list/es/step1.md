# Diccionario a Lista

## Problema

Escribe una funci√≥n `dict_to_list(d)` que tome un diccionario `d` como argumento y devuelva una lista de tuplas. Cada tupla debe contener un par clave-valor del diccionario. El orden de las tuplas en la lista debe ser el mismo que el orden de los pares clave-valor en el diccionario.

## Ejemplo

```python
d = {'one': 1, 'three': 3, 'five': 5, 'two': 2, 'four': 4}
dict_to_list(d)
# [('one', 1), ('three', 3), ('five', 5), ('two', 2), ('four', 4)]
```
