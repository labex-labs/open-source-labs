# Dictionnaire en Liste

## Problème

Écrivez une fonction `dict_to_list(d)` qui prend un dictionnaire `d` en argument et renvoie une liste de tuples. Chaque tuple devrait contenir une paire clé-valeur du dictionnaire. L'ordre des tuples dans la liste devrait être le même que l'ordre des paires clé-valeur dans le dictionnaire.

## Exemple

```python
d = {'one': 1, 'three': 3, 'five': 5, 'two': 2, 'four': 4}
dict_to_list(d)
# [('one', 1), ('three', 3), ('five', 5), ('two', 2), ('four', 4)]
```
