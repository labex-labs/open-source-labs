# Einführung

In Python können Wörterbücher und Listen geschachtelte Werte enthalten, die mithilfe einer Liste von Selektoren abgerufen werden können. In dieser Herausforderung werden Sie eine Funktion erstellen, die den Wert des geschachtelten Schlüssels aus einem Wörterbuch oder einer Liste abruft, der durch die gegebene Selektorenliste angegeben wird.
