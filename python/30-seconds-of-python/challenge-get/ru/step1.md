# Получить вложенное значение

## Задача

Напишите функцию `get(d, selectors)`, которая принимает в качестве аргументов словарь или список `d` и список селекторов `selectors` и возвращает значение вложенного ключа, указанного списком селекторов. Если ключ не существует, верните `None`.

Для реализации этой функции используйте `functools.reduce()` для перебора списка `selectors`. Примените `operator.getitem()` для каждого ключа в `selectors`, получая значение, которое будет использоваться в качестве элемента для следующей итерации.

## Пример

```python
users = {
  'freddy': {
    'name': {
      'first': 'fred',
      'last': 'smith'
    },
    'postIds': [1, 2, 3]
  }
}
get(users, ['freddy', 'name', 'last']) # 'smith'
get(users, ['freddy', 'postIds', 1]) # 2
get(users, ['freddy', 'age']) # None
```
