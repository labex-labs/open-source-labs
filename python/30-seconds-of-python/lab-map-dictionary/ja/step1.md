# リストを辞書にマッピングする

`map_dictionary(itr, fn)` という名前の Python 関数を作成してください。この関数は 2 つのパラメータを取ります。

- `itr`: 値のリスト
- `fn`: 値を入力として受け取り、値を出力として返す関数

この関数は、キーが元の値で、値が関数の結果である辞書を返す必要があります。

この問題を解くには、以下の手順に従ってください。

1. `map()` を使用して、リストの各値に `fn` を適用します。
2. `zip()` を使用して、元の値と `fn` によって生成された値をペアにします。
3. `dict()` を使用して、適切な辞書を返します。

```python
def map_dictionary(itr, fn):
  return dict(zip(itr, map(fn, itr)))
```

```python
map_dictionary([1, 2, 3], lambda x: x * x) # { 1: 1, 2: 4, 3: 9 }
```
