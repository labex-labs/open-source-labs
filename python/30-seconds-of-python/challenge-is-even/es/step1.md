# Comprobar si un número es par

## Problema

Escribe una función `is_even(num)` que tome un número como argumento y devuelva `True` si el número es par y `False` si el número es impar. Para comprobar si un número es par o impar, puedes usar el operador módulo (`%`). Si un número es par, no tendrá resto al dividirlo entre 2. Si un número es impar, tendrá un resto de 1 al dividirlo entre 2.

## Ejemplo

```python
is_even(4) # True
is_even(7) # False
is_even(0) # True
```
