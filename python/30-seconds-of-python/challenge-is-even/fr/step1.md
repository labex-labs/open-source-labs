# Vérifier si un nombre est pair

## Problème

Écrivez une fonction `is_even(num)` qui prend un nombre en argument et renvoie `True` si le nombre est pair et `False` s'il est impair. Pour vérifier si un nombre est pair ou impair, vous pouvez utiliser l'opérateur modulo (`%`). Si un nombre est pair, il n'aura pas de reste lorsqu'il est divisé par 2. Si un nombre est impair, il aura un reste de 1 lorsqu'il est divisé par 2.

## Exemple

```python
is_even(4) # True
is_even(7) # False
is_even(0) # True
```
