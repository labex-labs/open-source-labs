# 最大要素のインデックス

## 問題

リスト `arr` を引数として受け取り、最大値を持つ要素のインデックスを返す関数 `max_element_index(arr)` を作成してください。最大値を持つ要素が複数ある場合は、最初に出現する要素のインデックスを返します。

この問題を解くには、次の手順を辿ることができます。

1. リスト内の最大値を見つけるために、組み込みの `max()` 関数を使用します。
2. リスト内で最大値が最初に出現するインデックスを見つけるために、組み込みの `list.index()` 関数を使用します。
3. インデックスを返します。

## 例

```python
max_element_index([5, 8, 9, 7, 10, 3, 0]) # 4
```

この例では、リスト `[5, 8, 9, 7, 10, 3, 0]` の最大値は `10` で、インデックス `4` にあります。したがって、関数は `4` を返す必要があります。
