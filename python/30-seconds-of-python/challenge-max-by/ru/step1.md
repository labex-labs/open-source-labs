# Найти максимальное значение списка на основе функции

## Задача

Напишите функцию `max_by(lst, fn)`, которая принимает список `lst` и функцию `fn` в качестве аргументов. Функция должна сопоставить каждому элементу в `lst` значение с использованием заданной функции `fn`, а затем вернуть максимальное значение.

## Пример

```python
max_by([{ 'n': 4 }, { 'n': 2 }, { 'n': 8 }, { 'n': 6 }], lambda v : v['n']) # 8
```

## Ограничения

- Список `lst` будет содержать по крайней мере один элемент.
- Функция `fn` будет принимать один аргумент и возвращать значение.
