# Einführung

In Python ist ein String im Kebab-Case eine Zeichenkette, bei der jedes Wort durch ein Bindestrich getrennt ist. Beispielsweise ist "hello-world" ein String im Kebab-Case. In dieser Herausforderung müssen Sie eine Funktion schreiben, die einen gegebenen String in Kebab-Case umwandelt.
