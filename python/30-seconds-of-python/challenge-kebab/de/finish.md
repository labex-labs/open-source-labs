# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie man in Python einen String in Kebab-Case umwandelt. Sie haben `re.sub()` verwendet, um alle `-` oder `_` durch ein Leerzeichen zu ersetzen, `re.sub()` verwendet, um alle Wörter im String zu finden, `str.lower()` verwendet, um sie in Kleinbuchstaben zu konvertieren und `str.join()` verwendet, um alle Wörter mit `-` als Separator zu verbinden.
