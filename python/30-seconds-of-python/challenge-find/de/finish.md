# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie Sie mithilfe einer Listenkomprehension und `next()` den Wert des ersten Elements in einer Liste finden, das einer bereitgestellten Testfunktion entspricht. Diese Technik kann in vielen Szenarien nützlich sein und Ihnen helfen, effizienteres und präziseres Code zu schreiben.
