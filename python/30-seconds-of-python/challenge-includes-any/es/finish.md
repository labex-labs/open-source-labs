# Resumen

En este desafío, aprendiste cómo comprobar si algún elemento de una lista dada está incluido en otra lista dada. Usaste un bucle `for` para iterar a través de cada valor en `values` y el operador `in` para comprobar si el valor está incluido en `lst`. Luego devolviste `True` si se encontró cualquier valor, y `False` en caso contrario.
