# Einführung

In der funktionalen Programmierung ist das Curryieren eine Technik, um eine Funktion, die mehrere Argumente annimmt, in eine Sequenz von Funktionen zu transformieren, von denen jede nur ein einzelnes Argument annimmt. In Python können wir die `functools.partial()`-Funktion verwenden, um das Curryieren zu implementieren.
