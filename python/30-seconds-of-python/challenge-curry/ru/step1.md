# Функция каррирования

## Задача

Напишите функцию `curry(fn, *args)`, которая осуществляет каррирование заданной функции `fn`. Функция должна возвращать новую функцию, которая ведет себя как `fn` с заданными аргументами `args`, частично примененными.

## Пример

```python
add = lambda x, y: x + y
add10 = curry(add, 10)
add10(20) # 30
```
