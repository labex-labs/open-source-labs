# Introduction

En Python, vous pouvez facilement obtenir un élément aléatoire d'une liste en utilisant le module `random`. Ce module fournit une fonction appelée `choice()` qui renvoie un élément aléatoire d'une liste donnée. Dans ce défi, vous devrez écrire une fonction qui prend une liste en argument et renvoie un élément aléatoire de cette liste.
