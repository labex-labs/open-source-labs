# Введение

В Python мы можем легко найти разницу между двумя списками с использованием встроенной функции `set()`. Однако, что если мы хотим найти разницу на основе определенной функции, применяемой к каждому элементу обоих списков? В этом испытании вы создадите функцию, которая принимает два списка и функцию и возвращает разницу между двумя списками после применения указанной функции к каждому элементу списка обоих.
