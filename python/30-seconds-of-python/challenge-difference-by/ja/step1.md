# 関数に基づくリストの差分

## 問題

`difference_by(a, b, fn)` という名前の関数を作成して、次の3つのパラメータを受け取るようにします。

- `a`：要素のリスト
- `b`：要素のリスト
- `fn`：両方のリストの各要素に適用される関数

この関数は、両方のリストの各要素に提供された関数 `fn` を適用した後、リスト `a` にはありリスト `b` にはない要素のリストを返す必要があります。

この問題を解くには、次の手順を辿ることができます。

1. `map()` を使って `b` の各要素に `fn` を適用して `set` を作成します。
2. 以前作成したセット `_b` に含まれない値のみを残すために、`a` の `fn` と組み合わせてリスト内包表記を使います。

## 例

```python
difference_by([2.1, 1.2], [2.3, 3.4], floor) # [1.2]
difference_by([{ 'x': 2 }, { 'x': 1 }], [{ 'x': 1 }], lambda v : v['x'])
# [ { x: 2 } ]
```
