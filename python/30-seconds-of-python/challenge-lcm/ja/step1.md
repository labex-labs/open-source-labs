# 最小公倍数チャレンジ

## 問題

`lcm(numbers)` という関数を書きなさい。この関数は、数値のリストを引数として受け取り、それらの最小公倍数を返すものとする。関数は、2つの数 `x` と `y` の最小公倍数を計算するために次の式を使用すること。`lcm(x, y) = x * y / gcd(x, y)`。ここで、`gcd(x, y)` は `x` と `y` の最大公約数である。

この問題を解くには、`functools.reduce()` 関数を使って、`lcm()` の式をリスト内のすべての数に適用することができる。また、2つの数の最大公約数を計算するために `math.gcd()` 関数を使うこともできる。

## 例

```python
lcm([12, 7]) # 84
lcm([1, 3, 4, 5]) # 60
```
