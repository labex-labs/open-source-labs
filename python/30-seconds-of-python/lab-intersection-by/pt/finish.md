# Resumo

Neste desafio, você aprendeu como encontrar a intersecção de duas listas com base em uma função específica aplicada a cada elemento em ambas. Você escreveu uma função `intersection_by(a, b, fn)` que recebe duas listas e uma função, e retorna uma lista de elementos que existem em ambas as listas, após aplicar a função fornecida a cada elemento de ambas as listas.
