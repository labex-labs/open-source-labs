# Introdução

Em Python, podemos encontrar a intersecção de duas listas usando os métodos `set()` e `intersection()`. No entanto, e se quisermos encontrar a intersecção com base em uma função específica aplicada a cada elemento em ambas as listas? Neste desafio, você criará uma função que recebe duas listas e uma função, e retorna uma lista de elementos que existem em ambas as listas, após aplicar a função fornecida a cada elemento de ambas as listas.
