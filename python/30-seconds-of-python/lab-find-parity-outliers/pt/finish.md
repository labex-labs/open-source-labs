# Resumo

Neste desafio, você aprendeu como encontrar os _outliers_ de paridade em uma lista de inteiros. Você usou o módulo `collections.Counter` para contar os valores pares e ímpares na lista e, em seguida, encontrou a paridade mais comum. Finalmente, você usou uma _list comprehension_ para encontrar todos os elementos que não correspondem à paridade mais comum.
