# 範囲内の数

## 問題

`in_range(n, start, end = 0)` という関数を書きましょう。この関数は3つのパラメータを受け取ります。

- `n`：範囲内にあるかどうかを確認する数
- `start`：範囲の始点
- `end`：範囲の終点（省略可能で、デフォルト値は0）

与えられた数 `n` が指定された範囲内にある場合、関数は `True` を返し、それ以外の場合は `False` を返します。`end` パラメータが指定されていない場合、範囲は `0` から `start` までと見なされます。

## 例

```python
in_range(3, 2, 5) # True
in_range(3, 4) # True
in_range(2, 3, 5) # False
in_range(3, 2) # False
```
