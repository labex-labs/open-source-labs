# Число в диапазоне

## Задача

Напишите функцию `in_range(n, start, end = 0)`, которая принимает три параметра:

- `n`: число, которое необходимо проверить, попадает ли оно в диапазон
- `start`: начало диапазона
- `end`: конец диапазона (необязательный, значение по умолчанию равно 0)

Функция должна возвращать `True`, если заданное число `n` попадает в указанный диапазон, и `False` в противном случае. Если параметр `end` не указан, диапазон считается от `0` до `start`.

## Пример

```python
in_range(3, 2, 5) # True
in_range(3, 4) # True
in_range(2, 3, 5) # False
in_range(3, 2) # False
```
