# Binomischer Koeffizient

## Problemstellung

Schreiben Sie eine Funktion namens `binomial_coefficient(n, k)`, die zwei ganze Zahlen `n` und `k` als Eingabe erhält und den binomischen Koeffizienten von `n` und `k` zurückgibt. Ihre Funktion sollte die `math.comb()`-Methode verwenden, um den binomischen Koeffizienten zu berechnen.

## Beispiel

```python
binomial_coefficient(8, 2) # 28
binomial_coefficient(5, 3) # 10
binomial_coefficient(10, 5) # 252
```
