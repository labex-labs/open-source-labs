# Число является простым

## Задача

Напишите функцию на Python под названием `is_prime(n)`, которая принимает целое число `n` в качестве входных данных и возвращает `True`, если число является простым, и `False` в противном случае. Чтобы решить эту задачу, вам нужно следовать следующим правилам:

- Возвращайте `False`, если число равно `0`, `1`, отрицательному числу или является кратным `2`.
- Используйте `all()` и `range()` для проверки чисел от `3` до квадратного корня заданного числа.
- Возвращайте `True`, если ни одно из них не делит заданное число, `False` в противном случае.

## Пример

```python
is_prime(11) # True
```
