# List Initial

## Problème

Écrivez une fonction Python appelée `initial(lst)` qui prend une liste en argument et renvoie tous les éléments de la liste sauf le dernier.

## Exemple

```python
initial([1, 2, 3]) # [1, 2]
initial(['apple', 'banana', 'orange', 'grape']) # ['apple', 'banana', 'orange']
```
