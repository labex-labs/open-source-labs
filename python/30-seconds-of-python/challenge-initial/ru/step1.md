# Начало списка

## Задача

Напишите функцию на Python под названием `initial(lst)`, которая принимает список в качестве аргумента и возвращает все элементы списка, кроме последнего.

## Пример

```python
initial([1, 2, 3]) # [1, 2]
initial(['apple', 'banana', 'orange', 'grape']) # ['apple', 'banana', 'orange']
```
