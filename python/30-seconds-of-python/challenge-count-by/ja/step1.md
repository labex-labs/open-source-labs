# グループ化された要素のカウント

## 問題

引数としてリスト `lst` と関数 `fn` を受け取る関数 `count_by(lst, fn = lambda x: x)` を書きます。この関数は、与えられた関数に基づいてリストの要素をグループ化し、各グループの要素数を含む辞書を返す必要があります。

この問題を解くには、次の手順を辿ることができます。

1. `collections.defaultdict` を使用して辞書を初期化します。
2. `map()` を使用して与えられた関数をリストの各要素に適用します。
3. マッピングされた値を反復処理し、辞書内の各要素のカウントを増やします。

関数は結果の辞書を返す必要があります。

## 例

```python
count_by([6.1, 4.2, 6.3], floor) # {6: 2, 4: 1}
count_by(['one', 'two', 'three'], len) # {3: 2, 5: 1}
```

最初の例では、関数 `floor` がリスト `[6.1, 4.2, 6.3]` の要素をグループ化するために使用されます。結果の辞書には 2 つのキーがあります。`6` と `4` で、それぞれの値は `2` と `1` です。

2 番目の例では、関数 `len` がリスト `['one', 'two', 'three']` の要素をグループ化するために使用されます。結果の辞書には 2 つのキーがあります。`3` と `5` で、それぞれの値は `2` と `1` です。
