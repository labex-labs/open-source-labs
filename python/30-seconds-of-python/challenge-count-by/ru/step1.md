# Подсчет сгруппированных элементов

## Задача

Напишите функцию `count_by(lst, fn = lambda x: x)`, которая принимает список `lst` и функцию `fn` в качестве аргументов. Функция должна группировать элементы списка на основе заданной функции и возвращать словарь с количеством элементов в каждой группе.

Для решения этой задачи вы можете следовать следующим шагам:

1. Инициализировать словарь с использованием `collections.defaultdict`.
2. Использовать `map()`, чтобы применить заданную функцию к каждому элементу списка.
3. Перебирать сопоставленные значения и увеличивать счетчик каждого элемента в словаре.

Функция должна возвращать полученный словарь.

## Пример

```python
count_by([6.1, 4.2, 6.3], floor) # {6: 2, 4: 1}
count_by(['one', 'two', 'three'], len) # {3: 2, 5: 1}
```

В первом примере функция `floor` используется для группировки элементов списка `[6.1, 4.2, 6.3]`. Полученный словарь имеет два ключа: `6` и `4`, со значениями `2` и `1` соответственно.

Во втором примере функция `len` используется для группировки элементов списка `['one', 'two', 'three']`. Полученный словарь имеет два ключа: `3` и `5`, со значениями `2` и `1` соответственно.
