# Einführung

In Python können Sie die Elemente einer Liste basierend auf einer angegebenen Funktion gruppieren. Dies kann nützlich sein, wenn Sie die Anzahl der Vorkommen jeder Gruppe zählen möchten. In dieser Herausforderung müssen Sie eine Funktion erstellen, die die Elemente einer Liste basierend auf einer angegebenen Funktion gruppiert und die Anzahl der Elemente in jeder Gruppe zurückgibt.
