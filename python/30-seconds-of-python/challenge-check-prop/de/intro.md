# Einführung

In Python können wir Funktionen erstellen, die ein Dictionary entgegennehmen und eine Prädikatsfunktion auf eine bestimmte Eigenschaft des Dictionaries anwenden. Dies kann in Szenarien nützlich sein, in denen wir überprüfen müssen, ob eine bestimmte Bedingung für eine bestimmte Eigenschaft eines Dictionaries erfüllt ist.
