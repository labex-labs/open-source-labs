# プロパティのチェック

## 問題

`check_prop` という名前の関数を作成します。この関数は2つのパラメータを受け取ります。`fn` と `prop` です。`fn` パラメータは述語関数で、辞書の指定されたプロパティに適用されます。`prop` パラメータは文字列で、述語関数が適用されるプロパティの名前を表します。

`check_prop` 関数は、辞書を引数として受け取り、述語関数 `fn` を指定されたプロパティに適用するラムダ関数を返す必要があります。

## 例

```python
check_age = check_prop(lambda x: x >= 18, 'age')
user = {'name': 'Mark', 'age': 18}
check_age(user) # True
```

上記の例では、辞書の `age` プロパティの値が18以上であるかどうかをチェックする `check_age` 関数を作成します。次に、名前と年齢のプロパティを持つ `user` 辞書を作成します。最後に、`user` 辞書を引数として `check_age` 関数を呼び出します。年齢のプロパティが18に等しいため、`True` が返されます。
