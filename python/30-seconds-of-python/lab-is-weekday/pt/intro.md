# Introdução

Em Python, você pode usar o módulo `datetime` para trabalhar com datas e horas. Uma tarefa comum é verificar se uma determinada data é um dia de semana ou um fim de semana. Neste desafio, você escreverá uma função que recebe uma data como entrada e retorna `True` se for um dia de semana e `False` se for um fim de semana.
