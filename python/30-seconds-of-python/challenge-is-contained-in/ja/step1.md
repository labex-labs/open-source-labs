# リストの包含関係

## 問題

2つのリストを引数として受け取り、リスト `a` のすべての要素がリスト `b` に含まれている場合（要素の順序は問わない）に `True` を返す関数 `is_contained_in(a, b)` を作成してください。それ以外の場合は、関数は `False` を返す必要があります。

この問題を解くには、次のアプローチを使用できます。

1. リスト `a` の各一意の値をループ処理します。
2. 各値について、それがリスト `a` に現れる回数がリスト `b` に現れる回数よりも多いかどうかを確認します。
3. リスト `a` にある値がリスト `b` に現れる回数よりも多い場合、`False` を返します。
4. リスト `a` のすべての値がリスト `b` に少なくともそれらがリスト `a` に現れる回数だけ現れる場合、`True` を返します。

## 例

```python
assert is_contained_in([1, 4], [2, 4, 1]) == True
assert is_contained_in([1, 2, 3], [3, 2, 1]) == True
assert is_contained_in([1, 2, 3], [3, 2, 2, 1]) == False
assert is_contained_in([1, 2, 3], [4, 5, 6]) == False
```
