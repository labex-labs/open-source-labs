# Einführung

In Python ist es oft erforderlich, zu überprüfen, ob eine Liste in einer anderen Liste enthalten ist. Dies kann ein bisschen tricky sein, insbesondere wenn die Reihenfolge der Elemente in den Listen keine Rolle spielt. In dieser Herausforderung werden Sie eine Funktion schreiben, die überprüft, ob die Elemente einer Liste in einer anderen Liste enthalten sind, unabhängig von der Reihenfolge.
