# 最後の一致するインデックスを見つける

## 問題

`find_last_index(lst, fn)` という関数を書きましょう。この関数は、リスト `lst` と関数 `fn` を引数に取ります。この関数は、`fn` が `True` を返す `lst` の最後の要素のインデックスを返す必要があります。条件を満たす要素がない場合、関数は `-1` を返す必要があります。

## 例

```python
find_last_index([1, 2, 3, 4], lambda n: n % 2 == 1) # 2
find_last_index([2, 4, 6, 8], lambda n: n % 2 == 1) # -1
```
