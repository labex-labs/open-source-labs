# Fahrenheit in Celsius umrechnen

Schreiben Sie eine Funktion `fahrenheit_to_celsius(degrees)`, die eine Temperatur in Fahrenheit als Parameter nimmt und die Temperatur in Celsius zurÃ¼ckgibt. Die Funktion sollte die Umrechnungsformel `C = (F - 32) * 5 / 9` befolgen, wobei `C` die Temperatur in Celsius und `F` die Temperatur in Fahrenheit ist.

```python
def fahrenheit_to_celsius(degrees):
  return ((degrees - 32) * 5 / 9)
```

```python
fahrenheit_to_celsius(77) # 25.0
```
