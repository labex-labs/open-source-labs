# Conversão de Fahrenheit para Celsius

Escreva uma função `fahrenheit_to_celsius(degrees)` que recebe uma temperatura em Fahrenheit como parâmetro e retorna a temperatura em Celsius. A função deve seguir a fórmula de conversão `C = (F - 32) * 5 / 9`, onde `C` é a temperatura em Celsius e `F` é a temperatura em Fahrenheit.

```python
def fahrenheit_to_celsius(degrees):
  return ((degrees - 32) * 5 / 9)
```

```python
fahrenheit_to_celsius(77) # 25.0
```
