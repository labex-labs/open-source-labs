# Конвертация километров в мили: испытание

## Задача

Напишите функцию на Python под названием `km_to_miles`, которая принимает параметр `km` (тип `float`) и возвращает эквивалентное расстояние в милях (также `float`). Формула конвертации: `mi = km * 0.621371`.

## Пример

```python
km_to_miles(8.1) # 5.0331051
```
