# Desafío de kilómetros a millas

## Problema

Escribe una función de Python llamada `km_to_miles` que tome un parámetro `km` (un número de punto flotante) y devuelva la distancia equivalente en millas (también un número de punto flotante). La fórmula de conversión es `mi = km * 0.621371`.

## Ejemplo

```python
km_to_miles(8.1) # 5.0331051
```
