# Défi : kilomètres en miles

## Problème

Écrivez une fonction Python appelée `km_to_miles` qui prend un paramètre `km` (un nombre à virgule flottante) et renvoie la distance équivalente en miles (également un nombre à virgule flottante). La formule de conversion est `mi = km * 0.621371`.

## Exemple

```python
km_to_miles(8.1) # 5.0331051
```
