# N минимальных элементов

## Задача

Напишите функцию под названием `min_n(lst, n = 1)`, которая принимает список `lst` и необязательный целый числовой параметр `n` (значение по умолчанию `1`). Функция должна возвращать новый список, содержащий `n` наименьших элементов из исходного списка `lst`. Если параметр `n` не указан, функция должна вернуть список, содержащий наименьший элемент из `lst`.

Если `n` больше или равно длине списка `lst`, функция должна вернуть исходный список, отсортированный по возрастанию.

Ваша функция должна выполнить эти шаги:

1. Используйте встроенную функцию `sorted()` для сортировки списка по возрастанию.
2. Используйте запись среза для получения указанного количества элементов.
3. Верните полученный список.

## Пример

```python
min_n([1, 2, 3]) # [1]
min_n([1, 2, 3], 2) # [1, 2]
```
