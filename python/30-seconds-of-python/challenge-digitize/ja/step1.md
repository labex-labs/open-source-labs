# 数値を桁に分解する

## 問題

非負の整数 `n` を入力として受け取り、その桁のリストを返す関数 `digitize(n)` を書きなさい。この関数は、次の手順を行うことでこれを達成する必要があります。

1. 入力された数値 `n` を文字列に変換する。
2. `map()` 関数と `int` 関数を組み合わせて、文字列の各文字を整数に変換する。
3. 得られた整数のリストを返す。

たとえば、入力数値が `123` の場合、関数は `[1, 2, 3]` のリストを返す必要があります。

## 例

```python
assert digitize(123) == [1, 2, 3]
assert digitize(4567) == [4, 5, 6, 7]
assert digitize(0) == [0]
```
