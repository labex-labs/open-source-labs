# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie Sie mithilfe von Python eine Zahl in eine Liste ihrer Ziffern umwandeln. Mit der `map()`-Funktion und der `int`-Funktion können Sie jedes Zeichen in einem String einfach in eine Ganzzahl umwandeln und eine Liste dieser Ganzzahlen zurückgeben. Diese Technik kann für verschiedene Anwendungen nützlich sein, wie z. B. die Ausführung mathematischer Operationen auf einzelnen Ziffern oder die Manipulation von Zahlen auf feinerer Ebene.
