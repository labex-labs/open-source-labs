# Einführung

In Python ist es oft erforderlich, eine Zahl in eine Liste ihrer Ziffern umzuwandeln. Dies kann für verschiedene Anwendungen nützlich sein, wie z. B. die Ausführung mathematischer Operationen auf einzelnen Ziffern oder die Manipulation von Zahlen auf feinerer Ebene. In dieser Herausforderung müssen Sie eine Funktion schreiben, die eine Zahl als Eingabe nimmt und eine Liste ihrer Ziffern zurückgibt.
