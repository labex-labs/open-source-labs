# Преобразование числа в список цифр

## Задача

Напишите функцию `digitize(n)`, которая принимает неотрицательное целое число `n` в качестве входных данных и возвращает список его цифр. Функция должна выполнить следующие шаги, чтобы достичь этого:

1. Преобразуйте входное число `n` в строку.
2. Используйте функцию `map()`, объединенную с функцией `int`, чтобы преобразовать каждый символ в строке в целое число.
3. Верните результирующий список целых чисел.

Например, если входное число равно `123`, функция должна вернуть список `[1, 2, 3]`.

## Пример

```python
assert digitize(123) == [1, 2, 3]
assert digitize(4567) == [4, 5, 6, 7]
assert digitize(0) == [0]
```
