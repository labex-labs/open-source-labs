# Резюме

В этом испытании вы узнали, как преобразовать число в список его цифр с использованием Python. С помощью функции `map()` и функции `int` вы можете легко преобразовать каждый символ в строке в целое число и вернуть список этих целых чисел. Эта техника может быть полезна для различных приложений, таких как выполнение математических операций над отдельными цифрами или манипуляция числами более детально.
