# Введение

В Python часто необходимо преобразовать число в список его цифр. Это может быть полезно для различных приложений, таких как выполнение математических операций над отдельными цифрами или манипуляция числами более детально. В этом испытании вам будет поручено написать функцию, которая принимает число в качестве входных данных и возвращает список его цифр.
