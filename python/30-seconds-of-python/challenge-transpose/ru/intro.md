# Введение

В линейной алгебре транспонирование матрицы - это оператор, который переворачивает матрицу относительно своей главной диагонали. Транспонированная матрица получается путём замены строк на столбцы. В Python мы можем транспонировать двумерный список с помощью простой однострочной команды.
