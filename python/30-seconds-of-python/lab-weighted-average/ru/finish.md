# Резюме

В этом испытании вы узнали, как вычислять взвешенное среднее арифметическое списка чисел с использованием Python. Вы использовали функцию `sum()` для суммирования произведений чисел на их веса и для суммирования весов. Вы также использовали функцию `zip()` и списочное выражение для перебора пар значений и весов.
