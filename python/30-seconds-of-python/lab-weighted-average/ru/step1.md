# Взвешенное среднее арифметическое

Напишите функцию `weighted_average(nums, weights)`, которая принимает два списка одинаковой длины: `nums` и `weights`. Функция должна возвращать взвешенное среднее арифметическое чисел в `nums`, где каждое число умножается на соответствующий вес в `weights`. Взвешенное среднее арифметическое вычисляется путём деления суммы произведений каждого числа на его вес на сумму весов.

```python
def weighted_average(nums, weights):
  return sum(x * y for x, y in zip(nums, weights)) / sum(weights)
```

```python
weighted_average([1, 2, 3], [0.6, 0.2, 0.3]) # 1.72727
```
