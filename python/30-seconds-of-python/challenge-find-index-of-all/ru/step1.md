# Найти все совпадающие индексы

## Задача

Напишите функцию `find_index_of_all(lst, fn)`, которая принимает список `lst` и тестирующую функцию `fn` в качестве аргументов и возвращает список индексов всех элементов в `lst`, для которых `fn` возвращает `True`.

### Входные данные

- Список `lst` целых чисел.
- Тестирующая функция `fn`, которая принимает целое число в качестве входных данных и возвращает логическое значение.

### Выходные данные

- Список целых чисел, представляющих индексы всех элементов в `lst`, для которых `fn` возвращает `True`.

## Пример

```python
find_index_of_all([1, 2, 3, 4], lambda n: n % 2 == 1) # [0, 2]
find_index_of_all([1, 2, 3, 4], lambda n: n > 2) # [2, 3]
find_index_of_all([1, 2, 3, 4], lambda n: n < 0) # []
```

### Примечание

- В первом примере тестирующая функция `lambda n: n % 2 == 1` проверяет, является ли целое число нечетным. Функция возвращает `[0, 2]`, потому что элементы с индексами `0` и `2` в списке `[1, 2, 3, 4]` являются нечетными.
- Во втором примере тестирующая функция `lambda n: n > 2` проверяет, является ли целое число больше `2`. Функция возвращает `[2, 3]`, потому что элементы с индексами `2` и `3` в списке `[1, 2, 3, 4]` больше `2`.
- В третьем примере тестирующая функция `lambda n: n < 0` проверяет, является ли целое число отрицательным. Функция возвращает `[]`, потому что в списке `[1, 2, 3, 4]` нет отрицательных элементов.
