# Проверить, является ли число четным

Напишите функцию `is_even(num)`, которая принимает число в качестве аргумента и возвращает `True`, если число четное, и `False`, если число нечетное. Чтобы проверить, является ли число четным или нечетным, вы можете использовать оператор модуля (`%`). Если число четное, оно не будет иметь остатка при делении на 2. Если число нечетное, оно будет иметь остаток 1 при делении на 2.

```python
def is_even(num):
  return num % 2 == 0
```

```python
is_even(3) # False
```
