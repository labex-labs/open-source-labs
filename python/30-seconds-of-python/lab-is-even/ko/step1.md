# 숫자가 짝수인지 확인하기

`is_even(num)` 함수를 작성하세요. 이 함수는 숫자를 인수로 받아 숫자가 짝수이면 `True`를 반환하고, 홀수이면 `False`를 반환합니다. 숫자가 짝수인지 홀수인지 확인하려면 나머지 연산자 (`%`) 를 사용할 수 있습니다. 숫자가 짝수이면 2 로 나눴을 때 나머지가 없습니다. 숫자가 홀수이면 2 로 나눴을 때 나머지가 1 입니다.

```python
def is_even(num):
  return num % 2 == 0
```

```python
is_even(3) # False
```
