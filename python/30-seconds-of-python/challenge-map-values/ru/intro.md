# Введение

В Python словарь - это коллекция пар ключ-значение. Иногда нам нужно преобразовать значения словаря с использованием функции. Эта задача проверит вашу способность создать функцию, которая принимает словарь и функцию в качестве аргументов и возвращает новый словарь с теми же ключами, что и исходный словарь, и значениями, сгенерированными путём запуска предоставленной функции для каждого значения.
