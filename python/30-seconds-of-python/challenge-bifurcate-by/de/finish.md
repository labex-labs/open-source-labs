# Zusammenfassung

In dieser Aufgabe haben Sie gelernt, wie Sie eine Liste in zwei Gruppen aufteilen, basierend auf dem Ergebnis einer angegebenen Filterfunktion. Sie haben eine Listenkomprehension verwendet, um Elemente den Gruppen hinzuzufügen, basierend auf dem Wert, den die Filterfunktion für jedes Element zurückgibt. Dies ist eine nützliche Technik in der Programmierung, insbesondere wenn es um große Datensätze geht.
