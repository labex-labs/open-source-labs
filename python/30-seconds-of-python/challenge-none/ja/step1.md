# すべてのリスト要素が偽であるかどうかをテストする

## 問題

`none(lst, fn = lambda x: x)` という名前の Python 関数を書きます。この関数は、リスト `lst` とオプションの関数 `fn` を引数として受け取ります。関数は、リストのすべての要素が偽である場合は `True` を返し、そうでない場合は `False` を返します。オプションの関数 `fn` が提供された場合、それはリストの各要素の真偽性を判断するために使用されます。

要素が偽であるかどうかを判断するには、Python の `bool()` 関数と同じルールを使用できます。一般的に、次の値は偽と見なされます。

- `False`
- `None`
- `0`（整数）
- `0.0`（浮動小数点数）
- `''`（空の文字列）
- `[]`（空のリスト）
- `{}`（空の辞書）
- `()`（空のタプル）
- `set()`（空のセット）

オプションの関数 `fn` が提供された場合、それは 1 つの引数を取り、ブール値を返す必要があります。関数はリストの各要素に対して呼び出され、返り値は要素の真偽性を判断するために使用されます。

## 例

```python
assert none([0, 1, 2, 0], lambda x: x >= 2 ) == False
assert none([0, 0, 0]) == True
```
