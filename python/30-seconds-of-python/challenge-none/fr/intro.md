# Introduction

En Python, on peut utiliser la fonction `all()` pour vérifier si tous les éléments d'une liste sont considérés comme vrais (`truthy`). Mais que faire si on veut vérifier si chaque élément d'une liste est considéré comme faux (`falsy`)? Dans ce défi, vous devrez créer une fonction qui vérifie si chaque élément d'une liste est considéré comme faux (`falsy`).
