# リストから辞書へ

## 問題

2 つのリストを入力として受け取り、最初のリストの要素をキーとし、2 番目のリストの要素を値とする辞書を返す関数 `to_dictionary(keys, values)` を作成します。この関数は、`zip()` と `dict()` を組み合わせて 2 つのリストの値を辞書に結合する必要があります。2 つのリストの長さが等しくない場合、関数は `None` を返す必要があります。

## 例

```python
to_dictionary(['a', 'b'], [1, 2]) # { 'a': 1, 'b': 2 }
to_dictionary(['a', 'b', 'c'], [1, 2]) # None
```
