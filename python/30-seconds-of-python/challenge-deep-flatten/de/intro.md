# Einführung

In Python kann eine Liste andere Listen als Elemente enthalten, und diese Listen können wiederum noch weitere Listen als Elemente enthalten, was eine geschachtelte Struktur bildet. Der Prozess des Entschachtelns einer geschachtelten Liste besteht darin, sie in eine eindimensionale Liste umzuwandeln, wobei alle Elemente auf der gleichen Ebene stehen. In dieser Herausforderung werden Sie aufgefordert, eine Funktion zu schreiben, die eine Liste rekursiv vollständig entschachtelt, d. h., alle geschachtelten Listen werden rekursiv ausgeflacht.
