# Разбить на строки

## Задача

Напишите функцию `split_lines(s)`, которая принимает на вход многострочную строку `s` и возвращает список отдельных строк. Ваша функция должна разбивать строку при каждом символе конца строки (`\n`) и возвращать список полученных строк.

## Пример

```python
split_lines('This\nis a\nmultiline\nstring.\n')
# ['This', 'is a', 'multiline', 'string.', '']
```
