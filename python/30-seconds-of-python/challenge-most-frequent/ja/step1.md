# 最も頻繁に出現する要素

## 問題

整数のリストを入力として受け取り、そのリストの中で最も頻繁に出現する要素を返す `most_frequent(lst)` という Python 関数を書きなさい。同じ回数だけ出現し、最も高い頻度を持つ複数の要素がある場合は、リストで最初に出現する要素を返しなさい。

この問題を解くには、次の手順に従うことができます。

1. `set()` を使って `lst` の一意の値を取得します。
2. `max()` を使って最も多く出現する要素を見つけます。

あなたの関数は次のシグネチャを持つ必要があります。

```python
def most_frequent(lst: List[int]) -> int:
```

## 例

```python
assert most_frequent([1, 2, 1, 2, 3, 2, 1, 4, 2]) == 2
assert most_frequent([1, 2, 3, 4, 5]) == 1
assert most_frequent([1, 1, 1, 1, 1]) == 1
```
