# リストの和集合チャレンジ

## 問題

2 つのリストを入力として受け取り、両方のリストからのすべての一意の要素を含む新しいリストを返す、`list_union(a, b)` と呼ばれる Python 関数を書きなさい。あなたの関数は次の手順を実行する必要があります。

1. 2 つの入力リスト `a` と `b` を 1 つのリストに結合する。
2. 結合されたリストから重複する要素を削除する。
3. すべての一意の要素を含む新しいリストを返す。

あなたの関数は入力リスト `a` と `b` を変更してはならない。

## 例

```python
list_union([1, 2, 3], [4, 3, 2]) # [1, 2, 3, 4]
list_union(['apple', 'banana', 'orange'], ['pear', 'banana', 'kiwi']) # ['apple', 'banana', 'orange', 'pear', 'kiwi']
```
