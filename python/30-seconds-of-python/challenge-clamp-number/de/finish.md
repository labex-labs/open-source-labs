# Zusammenfassung

In dieser Herausforderung hast du gelernt, wie man eine Zahl innerhalb eines angegebenen Bereichs klammern kann. Dies ist eine nützliche Technik in der Programmierung, insbesondere wenn es um Benutzereingaben oder Daten geht, die auf einen bestimmten Bereich beschränkt sein müssen. Halte an der Übung und verfeinere deine Fähigkeiten, um ein besserer Programmierer zu werden!
