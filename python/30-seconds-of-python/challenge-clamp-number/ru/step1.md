# Задача по клемпингу числа

## Задача

Напишите функцию `clamp_number(num, a, b)`, которая принимает три параметра:

- `num` (целое или вещественное число): число, которое нужно заклемпить
- `a` (целое или вещественное число): нижняя граница диапазона
- `b` (целое или вещественное число): верхняя граница диапазона

Функция должна заклемпить `num` в заданном диапазоне, включая границы. Если `num` попадает в диапазон (`a`, `b`), верните `num`. В противном случае верните ближайшее число в диапазоне.

## Пример

```python
clamp_number(2, 3, 5) # 3
clamp_number(1, -1, -5) # -1
clamp_number(10, 1, 5) # 5
clamp_number(-10, -5, -1) # -5
```
