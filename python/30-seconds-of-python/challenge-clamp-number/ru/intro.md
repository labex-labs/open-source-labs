# Введение

В программировании часто необходимо ограничить число определенным диапазоном. Именно здесь приходит на помощь концепция клемпинга. Клемпинг числа означает ограничение его определенным диапазоном значений. В этом испытании вам будет поручено создать функцию, которая клемпит число в заданном диапазоне.
