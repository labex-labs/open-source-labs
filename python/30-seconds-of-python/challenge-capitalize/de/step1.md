# String in Großbuchstaben schreiben

## Problemstellung

Schreibe eine Python-Funktion namens `capitalize_string(s, lower_rest=False)`, die einen String als Argument nimmt und einen neuen String zurückgibt, bei dem der erste Buchstabe in Großbuchstaben geschrieben ist. Die Funktion sollte einen optionalen Parameter `lower_rest` haben, der, wenn auf `True` gesetzt, den Rest des Strings in Kleinbuchstaben umwandelt.

## Beispiel

```python
capitalize_string('hello world') # 'Hello world'
capitalize_string('hello world', True) # 'Hello world'
```
