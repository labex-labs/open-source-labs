# Перевести строку в верхний регистр

## Задача

Напишите функцию на Python под названием `capitalize_string(s, lower_rest=False)`, которая принимает строку в качестве аргумента и возвращает новую строку с первой буквой в верхнем регистре. Функция должна иметь необязательный параметр `lower_rest`, который, если установить в `True`, преобразует остальную часть строки в нижний регистр.

## Пример

```python
capitalize_string('hello world') # 'Hello world'
capitalize_string('hello world', True) # 'Hello world'
```
