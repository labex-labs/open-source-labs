# Mettre en majuscule une chaîne de caractères

## Problème

Écrivez une fonction Python appelée `capitalize_string(s, lower_rest=False)` qui prend une chaîne de caractères en argument et renvoie une nouvelle chaîne de caractères avec la première lettre en majuscule. La fonction devrait avoir un paramètre optionnel `lower_rest` qui, si défini sur `True`, convertit le reste de la chaîne en minuscules.

## Exemple

```python
capitalize_string('hello world') # 'Hello world'
capitalize_string('hello world', True) # 'Hello world'
```
