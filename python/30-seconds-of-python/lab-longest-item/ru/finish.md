# Резюме

В этом испытании вы узнали, как написать функцию, которая принимает любое количество итерируемых объектов или объектов с свойством длины и возвращает самый длинный из них. Вы использовали `max()` с `len()` в качестве `ключа`, чтобы вернуть элемент с наибольшей длиной. Если несколько элементов имеют одинаковую длину, будет возвращен первый из них.
