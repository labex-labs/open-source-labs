# Композиция функций

## Задача

Напишите функцию под названием `compose(*fns)`, которая принимает одну или несколько функций в качестве аргументов и возвращает новую функцию, которая является результатом композиции входных функций с права налево. Последняя (самая правая) функция может принимать один или несколько аргументов; оставшиеся функции должны быть унарными.

## Пример

```python
add5 = lambda x: x + 5
multiply = lambda x, y: x * y
multiply_and_add_5 = compose(add5, multiply)
multiply_and_add_5(5, 2) # 15
```

В приведенном выше примере мы определяем две функции `add5` и `multiply`. Затем мы используем функцию `compose()` для создания новой функции под названием `multiply_and_add_5`, которая сначала умножает свои два аргумента, а затем добавляет 5 к результату. Когда мы вызываем `multiply_and_add_5(5, 2)`, мы получаем результат `15`.
