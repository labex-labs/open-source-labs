# Le nombre est divisible

## Problème

Écrivez une fonction `is_divisible(dividende, diviseur)` qui prend deux entiers en arguments et renvoie `True` si le `dividende` est divisible par le `diviseur`, et `False` sinon.

## Exemple

```python
is_divisible(6, 3) # True
is_divisible(7, 2) # False
is_divisible(10, 5) # True
```
