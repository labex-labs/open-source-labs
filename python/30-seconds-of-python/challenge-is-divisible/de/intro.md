# Einführung

In Python können wir den Modulo-Operator (`%`) verwenden, um zu überprüfen, ob eine Zahl durch eine andere Zahl teilbar ist. Wenn der Rest gleich `0` ist, dann ist die Zahl teilbar. In dieser Herausforderung werden Sie eine Funktion erstellen, die überprüft, ob eine Zahl durch eine andere Zahl teilbar ist.
