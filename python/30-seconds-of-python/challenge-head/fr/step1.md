# Fonction pour obtenir la tête d'une liste

## Problème

Écrivez une fonction Python appelée `head(lst)` qui prend une liste en argument et renvoie le premier élément de la liste.

Votre fonction doit répondre aux exigences suivantes :

- La fonction doit s'appeler `head`
- La fonction doit prendre un seul argument, qui est une liste
- La fonction doit renvoyer le premier élément de la liste
- La fonction ne doit pas modifier la liste d'origine

## Exemple

```python
head([1, 2, 3]) # 1
```
