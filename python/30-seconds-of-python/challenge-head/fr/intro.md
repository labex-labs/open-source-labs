# Introduction

En Python, une liste est une collection ordonnée d'éléments. Chaque élément peut être accès par son indice - un entier non négatif qui représente la position de l'élément dans la liste. Le premier élément de la liste a l'indice 0, le second a l'indice 1, et ainsi de suite.

Dans ce défi, vous devrez écrire une fonction qui renvoie le premier élément d'une liste, également appelé "tête" de la liste.
