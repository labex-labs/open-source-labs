# Введение

В Python список - это упорядоченная коллекция элементов. Каждый элемент можно получить по его индексу - неотрицательному целому числу, представляющему позицию элемента в списке. Первый элемент в списке имеет индекс 0, второй - индекс 1 и так далее.

В этом испытании вы напишете функцию, которая возвращает первый элемент списка, также называемый «головой» списка.
