# Introducción

En Python, una lista es una colección ordenada de elementos. Cada elemento se puede acceder por su índice, un número entero no negativo que representa la posición del elemento en la lista. El primer elemento de la lista tiene índice 0, el segundo tiene índice 1, y así sucesivamente.

En este desafío, escribirás una función que devuelva el primer elemento de una lista, también conocido como la "cabeza" de la lista.
