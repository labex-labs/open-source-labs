# Liste Kopf-Funktion

## Problem

Schreiben Sie eine Python-Funktion namens `head(lst)`, die eine Liste als Argument nimmt und das erste Element der Liste zurückgibt.

Ihre Funktion sollte die folgenden Anforderungen erfüllen:

- Die Funktion sollte `head` heißen
- Die Funktion sollte ein einzelnes Argument akzeptieren, das eine Liste ist
- Die Funktion sollte das erste Element der Liste zurückgeben
- Die Funktion sollte die ursprüngliche Liste nicht modifizieren

## Beispiel

```python
head([1, 2, 3]) # 1
```
