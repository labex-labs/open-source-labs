# Introduction

En Python, nous pouvons utiliser la fonction `map()` pour appliquer une fonction à chaque élément d'une liste et renvoyer une nouvelle liste avec les éléments modifiés. Nous pouvons également utiliser la fonction `sum()` pour additionner tous les éléments d'une liste. Dans ce défi, nous allons combiner ces deux fonctions pour calculer la moyenne d'une liste, après avoir appliqué une fonction fournie à chaque élément pour obtenir une valeur modifiée.
