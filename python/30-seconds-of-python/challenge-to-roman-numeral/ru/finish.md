# Резюме

В этом испытании вы узнали, как преобразовать целое число в его римскую цифровую запись. Вы использовали список для поиска, содержащий кортежи в формате (римское значение, целое число), и цикл `for` для перебора значений в списке для поиска. Вы также использовали функцию `divmod()`, чтобы обновить `num` с остатком, добавляя римскую цифровую запись к результату.
