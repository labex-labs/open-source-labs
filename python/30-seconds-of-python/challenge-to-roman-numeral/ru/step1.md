# Задача по преобразованию целого числа в римскую цифру

## Задача

Напишите функцию `to_roman_numeral(num)`, которая принимает целое число `num` в диапазоне от 1 до 3999 (включительно) и возвращает его римскую цифровую запись в виде строки.

Для преобразования целого числа в его римскую цифровую запись можно использовать список для поиска, содержащий кортежи в формате (римское значение, целое число). Затем можно использовать цикл `for` для перебора значений в списке для поиска и функцию `divmod()`, чтобы обновить `num` с остатком, добавляя римскую цифровую запись к результату.

Ваша функция должна возвращать римскую цифровую запись входного целого числа.

## Пример

```python
to_roman_numeral(3) # 'III'
to_roman_numeral(11) # 'XI'
to_roman_numeral(1998) # 'MCMXCVIII'
```
