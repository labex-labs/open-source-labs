# Convertir la primera letra de una cadena a minúscula

## Problema

Escribe una función `decapitalize(s, upper_rest = False)` que tome una cadena `s` y devuelva una nueva cadena con la primera letra convertida a minúscula. La función también debe tener un parámetro opcional `upper_rest` que, cuando se establece en `True`, convertirá el resto de la cadena a mayúsculas.

## Ejemplo

```python
decapitalize('FooBar') # 'fooBar'
decapitalize('FooBar', True) # 'fOOBAR'
```
