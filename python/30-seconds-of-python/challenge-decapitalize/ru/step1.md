# Перевести первую букву строки в строчный регистр

## Задача

Напишите функцию `decapitalize(s, upper_rest = False)`, которая принимает строку `s` и возвращает новую строку с первой буквой в нижнем регистре. Функция также должна иметь необязательный параметр `upper_rest`, который, когда установлен в `True`, переводит остальную часть строки в верхний регистр.

## Пример

```python
decapitalize('FooBar') # 'fooBar'
decapitalize('FooBar', True) # 'fOOBAR'
```
