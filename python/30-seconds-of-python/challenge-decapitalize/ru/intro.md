# Введение

В Python строки являются неизменяемыми, что означает, что они не могут быть изменены после создания. Однако бывает, что нам нужно изменить строку, например, когда мы хотим сделать первую букву строчной. Это может быть полезно при работе с вводом пользователя или при форматировании строк для отображения. В этом испытании вам предстоит написать функцию, которая делает первую букву строки строчной.
