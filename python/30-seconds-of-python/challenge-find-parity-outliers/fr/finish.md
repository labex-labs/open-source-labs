# Sommaire

Dans ce défi, vous avez appris à trouver les anomalies de parité dans une liste d'entiers. Vous avez utilisé le module `collections.Counter` pour compter les valeurs paires et impaires dans la liste, puis trouvé la parité la plus fréquente. Enfin, vous avez utilisé une compréhension de liste pour trouver tous les éléments qui ne correspondent pas à la parité la plus fréquente.
