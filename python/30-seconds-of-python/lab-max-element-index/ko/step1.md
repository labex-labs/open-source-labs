# 최대 요소의 인덱스

`max_element_index(arr)` 함수를 작성하세요. 이 함수는 리스트 `arr`을 인수로 받아 최대값을 가진 요소의 인덱스를 반환합니다. 최대값을 가진 요소가 여러 개 있는 경우, 첫 번째 발생의 인덱스를 반환합니다.

이 문제를 해결하기 위해 다음 단계를 따를 수 있습니다.

1. 내장 함수 `max()`를 사용하여 리스트에서 최대값을 찾습니다.
2. 내장 함수 `list.index()`를 사용하여 리스트에서 최대값의 첫 번째 발생 인덱스를 찾습니다.
3. 인덱스를 반환합니다.

```python
def max_element_index(arr):
  return arr.index(max(arr))
```

```python
max_element_index([5, 8, 9, 7, 10, 3, 0]) # 4
```
