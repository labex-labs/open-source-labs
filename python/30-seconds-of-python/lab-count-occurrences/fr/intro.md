# Introduction

En Python, il est souvent nécessaire de compter le nombre d'occurrences d'une valeur spécifique dans une liste. Cela peut être utile dans diverses situations, telles que l'analyse de données ou le traitement de l'entrée utilisateur. Dans ce défi, vous devrez écrire une fonction qui compte le nombre d'occurrences d'une valeur donnée dans une liste.
