# Contar Ocurrencias

Escribe una función `count_occurrences(lst, val)` que tome una lista `lst` y un valor `val` como argumentos y devuelva la cantidad de veces que aparece `val` en `lst`. Tu función debe utilizar el método `list.count()` incorporado para contar la cantidad de veces que aparece.

```python
def count_occurrences(lst, val):
  return lst.count(val)
```

```python
count_occurrences([1, 1, 2, 1, 2, 3], 1) # 3
```
