# Einführung

In Python ist es oft erforderlich, die Anzahl der Vorkommen eines bestimmten Werts in einer Liste zu zählen. Dies kann in einer Vielzahl von Situationen nützlich sein, wie beispielsweise beim Analysieren von Daten oder der Verarbeitung von Benutzereingaben. In dieser Herausforderung müssen Sie eine Funktion schreiben, die die Anzahl der Vorkommen eines angegebenen Werts in einer Liste zählt.
