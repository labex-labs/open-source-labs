# Contar Ocorrências

Escreva uma função `count_occurrences(lst, val)` que recebe uma lista `lst` e um valor `val` como argumentos e retorna o número de ocorrências de `val` em `lst`. Sua função deve usar o método embutido `list.count()` para contar o número de ocorrências.

```python
def count_occurrences(lst, val):
  return lst.count(val)
```

```python
count_occurrences([1, 1, 2, 1, 2, 3], 1) # 3
```
