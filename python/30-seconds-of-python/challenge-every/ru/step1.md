# Проверить, являются ли все элементы списка истинными

## Задача

Напишите функцию под названием `every(lst, fn = lambda x: x)`, которая принимает список `lst` и функцию `fn` в качестве аргументов. Функция должна возвращать `True`, если `fn` возвращает `True` для каждого элемента в списке, и `False` в противном случае. Если функция не предоставлена, по умолчанию функция должна использовать тождественную функцию (`lambda x: x`).

Для решения этой задачи вам нужно будет использовать функцию `all()` в сочетании с `map()` и предоставленной функцией `fn`, чтобы проверить, возвращает ли `fn` `True` для всех элементов в списке.

## Пример

```python
every([4, 2, 3], lambda x: x > 1) # True
every([1, 2, 3]) # True
every([0, 1, 2]) # False
```
