# Введение

В Python мы можем использовать функцию `all()` для проверки, все ли элементы в списке истинны. Однако иногда мы можем захотеть проверить, истинно ли какое-то конкретное условие для каждого элемента в списке. В этом испытании вам нужно будет создать функцию, которая проверяет, возвращает ли предоставленная функция `True` для каждого элемента в списке.
