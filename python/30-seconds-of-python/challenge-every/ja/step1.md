# すべてのリスト要素が真であるかどうかをテストする

## 問題

`every(lst, fn = lambda x: x)` という名前の関数を作成します。この関数は、リスト `lst` と関数 `fn` を引数として受け取ります。関数は、`fn` がリストのすべての要素に対して `True` を返す場合に `True` を返し、そうでない場合は `False` を返します。関数が提供されない場合、関数は既定で恒等関数 (`lambda x: x`) を使用する必要があります。

この問題を解決するには、`all()` 関数を `map()` と併用して、提供された関数 `fn` がリストのすべての要素に対して `True` を返すかどうかを確認する必要があります。

## 例

```python
every([4, 2, 3], lambda x: x > 1) # True
every([1, 2, 3]) # True
every([0, 1, 2]) # False
```
