# Mettre en majuscule chaque mot

## Problème

Écrivez une fonction `capitalize_every_word(s)` qui prend une chaîne de caractères `s` en argument et renvoie une nouvelle chaîne de caractères avec la première lettre de chaque mot mise en majuscule.

## Exemple

```python
capitalize_every_word('hello world!') # 'Hello World!'
```
