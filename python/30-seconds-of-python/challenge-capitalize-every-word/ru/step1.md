# Сделать заглавной первую букву каждого слова

## Задача

Напишите функцию `capitalize_every_word(s)`, которая принимает строку `s` в качестве аргумента и возвращает новую строку с заглавной первой буквой каждого слова.

## Пример

```python
capitalize_every_word('hello world!') # 'Hello World!'
```
