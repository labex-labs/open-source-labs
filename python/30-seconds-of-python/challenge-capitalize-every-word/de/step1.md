# Jedes Wort großschreiben

## Problemstellung

Schreiben Sie eine Funktion `capitalize_every_word(s)`, die einen String `s` als Argument nimmt und einen neuen String zurückgibt, in dem der erste Buchstabe jedes Wortes großgeschrieben ist.

## Beispiel

```python
capitalize_every_word('hello world!') # 'Hello World!'
```
