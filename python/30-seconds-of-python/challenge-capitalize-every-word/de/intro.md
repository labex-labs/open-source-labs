# Einführung

In Python können wir den ersten Buchstaben jedes Wortes in einem String mithilfe einer integrierten Funktion großschreiben. In dieser Herausforderung werden Sie aufgefordert, eine Funktion zu schreiben, die einen String als Argument nimmt und einen neuen String zurückgibt, in dem der erste Buchstabe jedes Wortes großgeschrieben ist.
