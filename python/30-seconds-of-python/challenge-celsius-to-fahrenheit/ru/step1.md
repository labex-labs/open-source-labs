# Конвертация градусов Цельсия в градусы Фаренгейта

## Задача

Напишите функцию `celsius_to_fahrenheit`, которая принимает температуру в градусах Цельсия в качестве аргумента и возвращает температуру в градусах Фаренгейта. Используйте формулу преобразования `F = 1.8 * C + 32` для преобразования градусов Цельсия в градусы Фаренгейта.

## Пример

```python
celsius_to_fahrenheit(180) # 356.0
```
