# Défi : Convertir les degrés Celsius en degrés Fahrenheit

## Problème

Écrivez une fonction `celsius_to_fahrenheit` qui prend une température en degrés Celsius en argument et renvoie la température en degrés Fahrenheit. Utilisez la formule de conversion `F = 1,8 * C + 32` pour convertir les degrés Celsius en degrés Fahrenheit.

## Exemple

```python
celsius_to_fahrenheit(180) # 356,0
```
