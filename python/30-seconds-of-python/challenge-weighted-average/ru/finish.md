# Резюме

В этом испытании вы узнали, как вычислять взвешенное среднее арифметическое списка чисел с использованием Python. Вы использовали функцию `sum()` для нахождения суммы произведений чисел на их веса и для нахождения суммы весов. Вы также использовали функцию `zip()` и списочное выражение для перебора пар значений и весов.
