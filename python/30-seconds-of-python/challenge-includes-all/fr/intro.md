# Introduction

En Python, vous pouvez vérifier si tous les éléments d'une liste sont inclus dans une autre liste. Cela peut être utile dans de nombreuses situations, comme vérifier si l'entrée d'un utilisateur correspond à une liste d'options valides. Dans ce défi, vous créerez une fonction qui vérifie si toutes les valeurs d'une liste donnée sont incluses dans une autre liste.
