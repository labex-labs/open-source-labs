# Введение

В Python вы можете легко вычислить дату, которая будет через `n` дней от сегодняшней, используя модуль `datetime`. В этом испытании вам будет необходимо написать функцию, которая принимает целое число `n` в качестве входных данных и возвращает дату, которая будет через `n` дней от сегодняшней.
