# Introdução

Em Python, uma lista (list) é uma coleção de itens que são ordenados e mutáveis. Às vezes, precisamos obter todos os elementos de uma lista, exceto o primeiro. Isso pode ser feito usando o método de "cauda" (tail) da lista. Neste desafio, você será solicitado a implementar uma função que retorna todos os elementos em uma lista, exceto o primeiro.
