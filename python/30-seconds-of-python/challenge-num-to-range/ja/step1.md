# 数値を範囲にマッピングする

## 問題

`num_to_range` という関数を書きます。この関数は、`num`、`inMin`、`inMax`、`outMin`、`outMax` の5つの引数を取ります。この関数は、`inMin` - `inMax` の範囲内にある `num` を `outMin` - `outMax` の範囲にマッピングした値を返す必要があります。言い換えると、この関数は特定の範囲 (`inMin` - `inMax`) に含まれる数値 (`num`) を新しい範囲 (`outMin` - `outMax`) にマッピングする必要があります。

## 例

```python
num_to_range(5, 0, 10, 0, 100) # 50.0
```

この例では、0から10の範囲にある数値5を0から100の範囲にマッピングしています。結果は50.0になるはずです。
