# Zahl in Bereich abbilden

## Problem

Schreiben Sie eine Funktion namens `num_to_range`, die fÃ¼nf Argumente akzeptiert: `num`, `inMin`, `inMax`, `outMin` und `outMax`. Die Funktion sollte `num` abbilden, das sich zwischen `outMin`-`outMax` befindet, ausgehend von `inMin`-`inMax`. Mit anderen Worten, die Funktion sollte eine Zahl (`num`), die innerhalb eines bestimmten Bereichs (`inMin`-`inMax`) liegt, in einen neuen Bereich (`outMin`-`outMax`) abbilden.

## Beispiel

```python
num_to_range(5, 0, 10, 0, 100) # 50.0
```

In diesem Beispiel werden wir die Zahl 5 aus einem Bereich von 0 bis 10 in einen Bereich von 0 bis 100 abbilden. Das Ergebnis sollte 50.0 sein.
