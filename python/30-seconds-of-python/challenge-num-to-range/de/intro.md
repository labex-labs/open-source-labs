# Einführung

Beim Programmieren müssen wir oft eine Zahl aus einem Bereich in einen anderen Bereich abbilden. Beispielsweise können wir eine Zahl im Bereich von 0 bis 10 haben, aber müssen sie in einen Bereich von 0 bis 100 abbilden. Dies kann in vielen Anwendungen nützlich sein, wie zum Beispiel beim Skalieren von Daten oder der Umrechnung von Einheiten.
