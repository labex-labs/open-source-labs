# Introduction

En Python, une liste est une collection d'éléments qui est ordonnée et modifiable. Parfois, nous avons besoin d'obtenir tous les éléments d'une liste sauf le premier. Cela peut être fait en utilisant la méthode de queue de liste. Dans ce défi, vous devrez implémenter une fonction qui renvoie tous les éléments d'une liste sauf le premier.
