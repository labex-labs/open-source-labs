# Liste-End-Herausforderung

## Problemstellung

Schreiben Sie eine Funktion `tail(lst)`, die eine Liste als Argument nimmt und alle Elemente der Liste außer dem ersten zurückgibt. Wenn die Liste nur ein Element enthält, geben Sie die gesamte Liste zurück.

## Beispiel

```python
tail([1, 2, 3]) # [2, 3]
tail([1]) # [1]
```
