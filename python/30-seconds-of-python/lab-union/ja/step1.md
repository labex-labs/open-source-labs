# リストの和集合

`list_union(a, b)` という名前の Python 関数を書いて、2つのリストを入力として受け取り、両方のリストからのすべての一意の要素を含む新しいリストを返します。あなたの関数は次の手順を実行する必要があります。

1. 2つの入力リスト `a` と `b` を1つのリストに結合します。
2. 結合されたリストから重複する要素を削除します。
3. すべての一意の要素を含む新しいリストを返します。

あなたの関数は入力リスト `a` と `b` を変更してはいけません。

```python
def union(a, b):
  return list(set(a + b))
```

```python
union([1, 2, 3], [4, 3, 2]) # [1, 2, 3, 4]
```
