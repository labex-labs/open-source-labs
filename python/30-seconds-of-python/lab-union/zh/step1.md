# 列表并集

编写一个名为 `list_union(a, b)` 的 Python 函数，该函数接受两个列表作为输入，并返回一个包含两个列表中所有唯一元素的新列表。你的函数应执行以下步骤：

1. 将两个输入列表 `a` 和 `b` 合并为一个列表。
2. 从合并后的列表中删除任何重复项。
3. 返回包含所有唯一元素的新列表。

你的函数不应修改输入列表 `a` 和 `b`。

```python
def union(a, b):
  return list(set(a + b))
```

```python
union([1, 2, 3], [4, 3, 2]) # [1, 2, 3, 4]
```
