# Introducción

En Python, a veces necesitamos rellenar una cadena con un carácter específico para que tenga una cierta longitud. Por ejemplo, puede que queramos rellenar una cadena con espacios en ambos lados para que tenga una cierta longitud. En este desafío, se te pedirá escribir una función que rellene una cadena con el carácter especificado en ambos lados, si es más corta que la longitud especificada.
