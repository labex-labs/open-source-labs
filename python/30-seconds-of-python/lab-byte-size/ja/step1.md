# 文字列のバイトサイズ

文字列`s`を入力として受け取り、そのバイトサイズを返す関数`byte_size(s)`を書きます。文字列のバイトサイズは、メモリに文字列を格納するために必要なバイト数です。文字列をバイトサイズに変換するには、特定のエンコード方式を使用して文字列をエンコードする必要があります。この実験では、UTF-8エンコード方式を使用します。

文字列をバイトサイズに変換するには、次の手順に従います。

1. UTF-8エンコード方式を使用して文字列をエンコードします。
2. エンコードされた文字列の長さを取得します。

関数は、エンコードされた文字列の長さを返す必要があります。

```python
def byte_size(s):
  return len(s.encode('utf-8'))
```

```python
byte_size('😀') # 4
byte_size('Hello World') # 11
```
