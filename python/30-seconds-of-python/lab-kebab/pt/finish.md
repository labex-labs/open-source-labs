# Resumo

Neste desafio, você aprendeu como converter uma string para _kebab case_ em Python. Você usou `re.sub()` para substituir qualquer `-` ou `_` por um espaço, `re.sub()` para encontrar todas as palavras na string, `str.lower()` para colocá-las em minúsculas e `str.join()` para combinar todas as palavras usando `-` como separador.
