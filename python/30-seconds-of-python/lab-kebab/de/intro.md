# Einführung

In Python ist ein String im Kebab-Case-Format ein String, bei dem jedes Wort durch ein Bindestrich getrennt ist. Beispielsweise ist "hello-world" ein String im Kebab-Case-Format. In dieser Herausforderung müssen Sie eine Funktion schreiben, die einen gegebenen String in das Kebab-Case-Format umwandelt.
