# Разница между списками

## Задача

Напишите функцию на Python под названием `list_difference(a, b)`, которая принимает два списка в качестве аргументов и возвращает разницу между ними. Функция не должна фильтровать дубликаты значений. Чтобы решить эту задачу, вы можете следовать следующим шагам:

1. Создайте множество из второго списка `b`.
2. Используйте списочное выражение для первого списка `a`, чтобы оставить только значения, не содержащиеся в ранее созданном наборе `_b`.
3. Верните результирующий список.

## Пример

```python
list_difference([1, 2, 3], [1, 2, 4]) # [3]
```
