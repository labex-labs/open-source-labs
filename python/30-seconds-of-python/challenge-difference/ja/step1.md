# リストの差分

## 問題

2 つのリストを引数として受け取り、それらの差分を返す `list_difference(a, b)` という名前の Python 関数を書きます。この関数は重複する値をフィルタリングしないものとします。この問題を解くには、次の手順に従うことができます。

1. 2 番目のリスト `b` からセットを作成します。
2. 最初のリスト `a` に対してリスト内包表記を使用して、先に作成したセット `_b` に含まれていない値のみを残します。
3. 結果のリストを返します。

## 例

```python
list_difference([1, 2, 3], [1, 2, 4]) # [3]
```
