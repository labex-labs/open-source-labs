# 部分和のリスト

## 問題

`partial_sum(lst)` という関数を書きなさい。この関数は、数値のリストを引数として受け取り、部分和のリストを返すものとする。この関数は次の手順を実行する必要がある。

1. `itertools.accumulate()` を使って、リストの各要素に対する累積和を作成する。
2. `list()` を使って、結果をリストに変換する。
3. 部分和のリストを返す。

## 例

```python
partial_sum([1, 2, 3, 4, 5]) # [1, 3, 6, 10, 15]
partial_sum([2, 4, 6, 8, 10]) # [2, 6, 12, 20, 30]
partial_sum([5, 10, 15, 20, 25]) # [5, 15, 30, 50, 75]
```
