# リストの一部の要素が真であるかどうかをテストする

`some(lst, fn)` という関数を書きます。この関数は、リスト `lst` と関数 `fn` を引数として受け取ります。関数 `fn` がリスト `lst` の少なくとも 1 つの要素に対して `True` を返す場合、この関数は `True` を返す必要があります。リストの要素が条件を満たさない場合は、関数は `False` を返す必要があります。関数が提供されない場合は、関数は恒等関数（要素自体を返す関数）を使用する必要があります。

```python
def some(lst, fn = lambda x: x):
  return any(map(fn, lst))
```

```python
some([0, 1, 2, 0], lambda x: x >= 2 ) # True
some([0, 0, 1, 0]) # True
```
