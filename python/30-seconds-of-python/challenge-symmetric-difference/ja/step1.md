# 対称差

## 問題

2 つのリストを引数として受け取り、それらの対称差をリストとして返す関数 `symmetric_difference(a, b)` を書きます。この関数は、重複する値をフィルタリングしないでください。

この問題を解くには、次の手順を辿ることができます。

1. それぞれのリストからセットを作成します。
2. それぞれのセットに対して、リスト内包表記を使って、もう一方のセットに含まれていない値のみを残します。
3. 手順 2 で得られた 2 つのリストを連結します。

## 例

```python
symmetric_difference([1, 2, 3], [1, 2, 4]) # [3, 4]
```
