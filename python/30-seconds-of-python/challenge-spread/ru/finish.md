# Резюме

В этом испытании вы узнали, как использовать оператор расширения для сглаживания списка в Python. Вы также реализовали функцию, которая принимает список в качестве аргумента и возвращает новый список, содержащий все элементы исходного списка, сгладённые. Это полезный прием, который стоит знать при работе с вложенными списками в Python.
