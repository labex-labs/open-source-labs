# Repetir String (Repeat String)

Escreva uma função chamada `repeat_string` que recebe dois parâmetros: uma string `s` e um inteiro `n`. A função deve retornar uma nova string que contém `s` repetida `n` vezes.

Por exemplo, se `s` for `"hello"` e `n` for `3`, a função deve retornar `"hellohellohello"`. Se `s` for `"abc"` e `n` for `5`, a função deve retornar `"abcabcabcabcabc"`.

```python
def n_times_string(s, n):
  return (s * n)
```

```python
n_times_string('py', 4) #'pypypypy'
```
