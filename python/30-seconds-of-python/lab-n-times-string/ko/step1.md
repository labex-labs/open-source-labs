# 문자열 반복

`repeat_string`이라는 함수를 작성하세요. 이 함수는 두 개의 매개변수, 문자열 `s`와 정수 `n`을 받습니다. 이 함수는 `s`를 `n`번 반복한 새로운 문자열을 반환해야 합니다.

예를 들어, `s`가 `"hello"`이고 `n`이 `3`이면 함수는 `"hellohellohello"`를 반환해야 합니다. `s`가 `"abc"`이고 `n`이 `5`이면 함수는 `"abcabcabcabcabc"`를 반환해야 합니다.

```python
def n_times_string(s, n):
  return (s * n)
```

```python
n_times_string('py', 4) #'pypypypy'
```
