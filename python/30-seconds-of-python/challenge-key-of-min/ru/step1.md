# Ключ минимального значения

## Задача

Напишите функцию `key_of_min(d)`, которая принимает словарь `d` в качестве аргумента и возвращает ключ минимального значения в словаре.

Для решения этой задачи вы можете использовать встроенную функцию `min()` с параметром `key`, установленным на `dict.get()`. Это вернет ключ минимального значения в словаре.

## Пример

```python
key_of_min({'a':4, 'b':0, 'c':13}) # 'b'
```

В этом примере словарь `{'a':4, 'b':0, 'c':13}` передается в качестве аргумента функции `key_of_min()`. Функция возвращает ключ `'b'`, который соответствует минимальному значению в словаре.
