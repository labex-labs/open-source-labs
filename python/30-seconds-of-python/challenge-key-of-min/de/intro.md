# Einführung

In Python sind Wörterbücher eine nützliche Datenstruktur zum Speichern von Schlüssel-Wert-Paaren. Manchmal müssen wir den Schlüssel des kleinsten Werts in einem Wörterbuch finden. In dieser Herausforderung müssen Sie eine Funktion schreiben, die den Schlüssel des kleinsten Werts in einem gegebenen Wörterbuch findet.
