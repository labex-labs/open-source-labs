# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie man mithilfe von Python den Schlüssel des kleinsten Werts in einem Wörterbuch findet. Indem Sie die `min()`-Funktion mit dem Parameter `key` auf `dict.get()` setzen, können Sie leicht den Schlüssel des kleinsten Werts in einem gegebenen Wörterbuch finden.
