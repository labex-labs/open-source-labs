# Einführung

In Python können Datumswerte in verschiedenen Formaten dargestellt werden. Ein solches Format ist das ISO-8601-Format, das ein Standardformat zur Darstellung von Daten und Zeiten ist. In dieser Herausforderung müssen Sie eine Aufgabe lösen, bei der Sie ein Datum von seiner ISO-8601-Darstellung in ein `datetime.datetime`-Objekt umwandeln.
