# Введение

В Python даты могут быть представлены в различных форматах. Одним из таких форматов является формат ISO-8601, который является стандартным форматом для представления дат и времени. В этом испытании вам будет поручено преобразовать дату из ее представления в формате ISO-8601 в объект `datetime.datetime`.
