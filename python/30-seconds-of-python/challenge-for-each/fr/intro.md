# Introduction

En Python, il est courant de devoir exécuter une fonction pour chaque élément d'une liste. Cela peut être fait à l'aide d'une boucle `for`, mais il peut être fastidieux d'écrire la boucle à chaque fois. Dans ce défi, vous créerez une fonction qui prend une liste et une fonction en arguments et exécute la fonction pour chaque élément de la liste.
