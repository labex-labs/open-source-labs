# Einführung

In Python kannst du einfach ein zufälliges Element aus einer Liste mit dem Modul `random` erhalten. Dieses Modul bietet eine Funktion namens `choice()`, die ein zufälliges Element aus einer gegebenen Liste zurückgibt. In dieser Herausforderung wirst du aufgefordert sein, eine Funktion zu schreiben, die eine Liste als Argument nimmt und ein zufälliges Element aus dieser Liste zurückgibt.
