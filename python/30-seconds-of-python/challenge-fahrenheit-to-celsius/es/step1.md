# Conversión de grados Fahrenheit a grados Celsius

## Problema

Escribe una función `fahrenheit_to_celsius(grados)` que tome una temperatura en grados Fahrenheit como parámetro y devuelva la temperatura en grados Celsius. La función debe seguir la fórmula de conversión `C = (F - 32) * 5 / 9`, donde `C` es la temperatura en grados Celsius y `F` es la temperatura en grados Fahrenheit.

## Ejemplo

```python
fahrenheit_to_celsius(77) # 25.0
```
