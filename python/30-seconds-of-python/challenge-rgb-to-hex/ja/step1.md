# RGB から 16 進数への変換チャレンジ

## 問題

色の赤、緑、青コンポーネントの値を表す 3 つの整数を引数として受け取り、16 進数の色コードを表す文字列を返す関数 `rgb_to_hex(r, g, b)` を書きなさい。出力文字列は `RRGGBB` 形式でなければならず、ここで `RR`、`GG`、`BB` はそれぞれ赤、緑、青コンポーネントを表す 2 桁の 16 進数値です。

たとえば、入力値が `255`、`165`、`1` の場合、出力は文字列 `'FFA501'` でなければなりません。

## 例

```python
rgb_to_hex(255, 165, 1) # 'FFA501'
```
