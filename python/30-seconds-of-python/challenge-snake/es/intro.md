# Introducción

En Python, el nombre en minúsculas con guiones bajos (snake case) es una convención de nombrado donde las palabras están separadas por guiones bajos. Esto se utiliza comúnmente para nombrar variables, funciones y otros identificadores. Sin embargo, a veces puede que tengamos cadenas que no están en snake case y necesitamos convertirlas. En este desafío, se te encomendará escribir una función de Python que convierta una cadena a snake case.
