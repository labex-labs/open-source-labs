# Resumen

En este desafío, aprendiste cómo escribir una función de Python que convierte una cadena a formato snake case. Utilizaste `re.sub()` para coincidir y reemplazar palabras en la cadena, `str.lower()` para convertirlas a minúsculas y `str.join()` para combinarlas utilizando guiones bajos como separador. Con este conocimiento, ahora puedes convertir fácilmente cadenas a formato snake case en tus programas de Python.
