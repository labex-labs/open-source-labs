# Einführung

In Python ist snake case eine Namenskonvention, bei der Wörter durch Unterstriche getrennt werden. Dies wird üblicherweise bei der Namensgebung von Variablen, Funktionen und anderen Identifikatoren verwendet. Manchmal haben wir jedoch Strings, die nicht im snake case Format vorliegen, und müssen sie umwandeln. In dieser Herausforderung müssen Sie eine Python-Funktion schreiben, die einen String in snake case umwandelt.
