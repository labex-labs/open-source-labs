# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie Sie eine Python-Funktion schreiben, die einen String in snake case umwandelt. Sie haben `re.sub()` verwendet, um Wörter im String zu finden und zu ersetzen, `str.lower()` verwendet, um sie in Kleinbuchstaben umzuwandeln, und `str.join()` verwendet, um sie mit Unterstrichen als Separator zu kombinieren. Mit diesen Kenntnissen können Sie nun Strings in Ihren Python-Programmen leicht in snake case umwandeln.
