# Résumé

Dans ce défi, vous avez appris à écrire une fonction Python qui convertit une chaîne de caractères en snake case. Vous avez utilisé `re.sub()` pour correspondre et remplacer les mots dans la chaîne, `str.lower()` pour les mettre en minuscules et `str.join()` pour les combiner en utilisant des tirets du bas comme séparateur. Grâce à ces connaissances, vous pouvez désormais facilement convertir les chaînes de caractères en snake case dans vos programmes Python.
