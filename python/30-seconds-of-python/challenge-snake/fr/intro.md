# Introduction

En Python, le snake case est une convention de nommage où les mots sont séparés par des tirets du bas. Cela est couramment utilisé pour nommer les variables, les fonctions et autres identifiants. Cependant, parfois, nous pouvons avoir des chaînes de caractères qui ne sont pas au snake case et que nous devons convertir. Dans ce défi, vous devrez écrire une fonction Python qui convertit une chaîne de caractères en snake case.
