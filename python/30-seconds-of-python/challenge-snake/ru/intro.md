# Введение

В Python snake case - это соглашение о именовании, при котором слова разделяются нижними подчеркиваниями. Это широко используется для именования переменных, функций и других идентификаторов. Однако иногда у нас могут быть строки, которые не имеют snake case, и нам нужно их преобразовать. В этом испытании вам предстоит написать функцию на Python, которая преобразует строку в snake case.
