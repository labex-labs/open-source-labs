# Резюме

В этом испытании вы узнали, как написать функцию на Python, которая преобразует строку в snake case. Вы использовали `re.sub()`, чтобы найти и заменить слова в строке, `str.lower()`, чтобы привести их к нижнему регистру, и `str.join()`, чтобы объединить их с использованием нижних подчеркиваний в качестве разделителя. С этими знаниями вы теперь можете легко преобразовывать строки в snake case в своих программах на Python.
