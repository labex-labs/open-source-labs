# Введение

В Python модуль `datetime` предоставляет классы для работы с датами и временем. Одним из классов этого модуля является `date`, который представляет дату (год, месяц, день) и предоставляет различные методы для работы с датами. Другой класс - `timedelta`, который представляет длительность или разницу между двумя датами или временами.
