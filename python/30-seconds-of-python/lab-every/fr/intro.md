# Introduction

En Python, on peut utiliser la fonction `all()` pour vérifier si tous les éléments d'une liste sont considérés comme vrais (truthy). Cependant, parfois, on peut vouloir vérifier si une condition spécifique est vraie pour chaque élément de la liste. Dans ce défi, vous devrez créer une fonction qui vérifie si une fonction fournie renvoie `True` pour chaque élément de la liste.
