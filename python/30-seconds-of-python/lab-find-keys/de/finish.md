# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie Sie alle Schlüssel in einem Wörterbuch finden, die einen bestimmten Wert haben. Sie haben eine Python-Funktion namens `find_keys(dictionary, value)` geschrieben, die ein Wörterbuch und einen Wert als Argumente übernimmt und eine Liste aller Schlüssel im Wörterbuch zurückgibt, die den angegebenen Wert haben. Sie haben die `dictionary.items()`-Methode und eine Listenkomprehension verwendet, um dieses Problem zu lösen.
