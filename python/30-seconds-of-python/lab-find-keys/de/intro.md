# Einführung

Wörterbücher sind eine grundlegende Datenstruktur in Python. Sie werden verwendet, um Schlüssel-Wert-Paare zu speichern, wobei jeder Schlüssel einzigartig ist. Manchmal müssen wir alle Schlüssel in einem Wörterbuch finden, die einen bestimmten Wert haben. In dieser Herausforderung werden Sie aufgefordert, eine Funktion zu schreiben, die alle Schlüssel in einem Wörterbuch findet, die einen angegebenen Wert haben.
