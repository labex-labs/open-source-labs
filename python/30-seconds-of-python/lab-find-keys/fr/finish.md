# Récapitulatif

Dans ce défi, vous avez appris à trouver toutes les clés dans un dictionnaire qui ont une valeur donnée. Vous avez écrit une fonction Python appelée `trouver_cles(dictionnaire, valeur)` qui prend un dictionnaire et une valeur en arguments et renvoie une liste de toutes les clés dans le dictionnaire qui ont la valeur donnée. Vous avez utilisé la méthode `dictionnaire.items()` et une compréhension de liste pour résoudre ce problème.
