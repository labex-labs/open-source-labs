# Resumo

Neste desafio, você aprendeu como verificar se algum elemento em uma lista fornecida está incluído em outra lista fornecida. Você usou um loop `for` para iterar por cada valor em `values` e o operador `in` para verificar se o valor está incluído em `lst`. Em seguida, você retornou `True` se qualquer valor fosse encontrado e `False` caso contrário.
