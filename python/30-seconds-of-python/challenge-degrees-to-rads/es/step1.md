# Desafío de grados a radianes

## Problema

Escribe una función `grados_a_radianes(deg)` que tome un ángulo en grados como argumento y devuelva el ángulo en radianes. Tu función debe utilizar la siguiente fórmula para convertir grados a radianes:

```
radianes = (grados * pi) / 180.0
```

donde `pi` es un valor constante que representa la razón entre la circunferencia de un círculo y su diámetro (aproximadamente 3.14159).

Tu función debe devolver el ángulo en radianes redondeado a 4 decimales.

## Ejemplo

```python
grados_a_radianes(180) # devuelve 3.1416
grados_a_radianes(90) # devuelve 1.5708
grados_a_radianes(45) # devuelve 0.7854
```
