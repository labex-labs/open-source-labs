# 度からラジアンへのチャレンジ

## 問題

引数として度で表された角度を受け取り、その角度をラジアンで返す関数 `degrees_to_rads(deg)` を書きなさい。関数は、次の式を使って度をラジアンに変換する必要があります。

```
radians = (degrees * pi) / 180.0
```

ここで `pi` は、円の周の長さと直径の比を表す定数値です（約3.14159）。

関数は、4桁の小数で丸めたラジアンで表された角度を返す必要があります。

## 例

```python
degrees_to_rads(180) # 3.1416 を返す
degrees_to_rads(90) # 1.5708 を返す
degrees_to_rads(45) # 0.7854 を返す
```
