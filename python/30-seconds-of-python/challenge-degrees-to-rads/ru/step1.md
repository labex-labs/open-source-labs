# Вызов преобразования градусов в радианы

## Задача

Напишите функцию `degrees_to_rads(deg)`, которая принимает угол в градусах в качестве аргумента и возвращает угол в радианах. Ваша функция должна использовать следующую формулу для преобразования градусов в радианы:

```
radians = (degrees * pi) / 180.0
```

где `pi` - это константа, представляющая отношение длины окружности к диаметру (примерно 3,14159).

Ваша функция должна возвращать угол в радианах, округленный до 4 знаков после запятой.

## Пример

```python
degrees_to_rads(180) # возвращает 3,1416
degrees_to_rads(90) # возвращает 1,5708
degrees_to_rads(45) # возвращает 0,7854
```
