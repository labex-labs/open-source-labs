# Introducción

En Python, podemos encontrar fácilmente la diferencia entre dos listas utilizando la función integrada `set()`. Sin embargo, ¿y si queremos encontrar la diferencia basada en una función específica aplicada a cada elemento de ambas listas? En este desafío, crearás una función que tome dos listas y una función, y devuelva la diferencia entre las dos listas después de aplicar la función proporcionada a cada elemento de ambas listas.
