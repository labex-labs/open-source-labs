# Разница между списками на основе функции

Создайте функцию под названием `difference_by(a, b, fn)`, которая принимает три параметра:

- `a`: список элементов
- `b`: список элементов
- `fn`: функция, которая будет применяться к каждому элементу обоих списков

Функция должна возвращать список элементов, которые находятся в списке `a`, но не в списке `b`, после применения предоставленной функции `fn` к каждому элементу обоих списков.

Для решения этой проблемы вы можете следовать следующим шагам:

1. Создайте `set`, используя `map()` для применения `fn` к каждому элементу в `b`.
2. Используйте списочное выражение в сочетании с `fn` на `a`, чтобы оставить только значения, не содержащиеся в предварительно созданном наборе `_b`.

```python
def difference_by(a, b, fn):
  _b = set(map(fn, b))
  return [item for item in a if fn(item) not in _b]
```

```python
from math import floor

difference_by([2.1, 1.2], [2.3, 3.4], floor) # [1.2]
difference_by([{ 'x': 2 }, { 'x': 1 }], [{ 'x': 1 }], lambda v : v['x'])
# [ { x: 2 } ]
```
