# Einführung

In Python können wir mithilfe der integrierten `set()`-Funktion leicht den Unterschied zwischen zwei Listen finden. Was passiert jedoch, wenn wir den Unterschied basierend auf einer bestimmten Funktion, die auf jedes Element in beiden Listen angewendet wird, ermitteln möchten? In dieser Herausforderung werden Sie eine Funktion erstellen, die zwei Listen und eine Funktion als Eingabe nimmt und den Unterschied zwischen den beiden Listen zurückgibt, nachdem die bereitgestellte Funktion auf jedes Listelement beider Listen angewendet wurde.
