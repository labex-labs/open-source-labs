# Introdução

Em Python, podemos facilmente encontrar a diferença entre duas listas usando a função `set()` integrada. No entanto, e se quisermos encontrar a diferença com base em uma função específica aplicada a cada elemento em ambas as listas? Neste desafio, você criará uma função que recebe duas listas e uma função, e retorna a diferença entre as duas listas após aplicar a função fornecida a cada elemento de ambas as listas.
