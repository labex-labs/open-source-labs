# Введение

В Python иногда необходимо задержать выполнение функции на определенное время. Это может быть полезно в сценариях, когда мы хотим дождаться наступления определенного события перед выполнением функции. В этом испытании вам будет предложено создать функцию, которая задерживает выполнение другой функции на указанное время.
