# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie Sie den `*`-Operator verwenden, um einen String eine bestimmte Anzahl von Malen zu wiederholen. Sie haben auch eine Funktion namens `repeat_string` geschrieben, die einen String und eine Ganzzahl als Parameter akzeptiert und einen neuen String zurückgibt, der den ursprünglichen String `n` Mal wiederholt enthält.
