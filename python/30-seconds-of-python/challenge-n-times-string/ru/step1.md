# Повторение строки

## Задача

Напишите функцию под названием `repeat_string`, которая принимает два параметра: строку `s` и целое число `n`. Функция должна вернуть новую строку, которая содержит `s`, повторенную `n` раз.

Например, если `s` равно `"hello"` и `n` равно `3`, функция должна вернуть `"hellohellohello"`. Если `s` равно `"abc"` и `n` равно `5`, функция должна вернуть `"abcabcabcabcabc"`.

## Пример

```python
assert repeat_string("hello", 3) == "hellohellohello"
assert repeat_string("abc", 5) == "abcabcabcabcabc"
assert repeat_string("123", 2) == "123123"
```
