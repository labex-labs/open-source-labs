# Sommaire

Dans ce défi, vous avez appris à utiliser l'opérateur `*` pour répéter une chaîne de caractères un certain nombre de fois. Vous avez également écrit une fonction appelée `repeat_string` qui prend une chaîne de caractères et un entier en paramètres et renvoie une nouvelle chaîne de caractères qui contient la chaîne d'origine répétée `n` fois.
