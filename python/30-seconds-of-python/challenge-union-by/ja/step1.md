# 関数に基づくリストの和集合

## 問題

`union_by(a, b, fn)` という関数を書きましょう。この関数は、2 つのリスト `a` と `b` と、関数 `fn` を引数に取ります。この関数は、両方のリストの各要素に提供された関数を適用した後、2 つのリストのいずれかに存在するすべての要素を 1 度だけ含むリストを返す必要があります。

この問題を解くには、次の手順を辿ることができます。

1. `a` の各要素に `fn` を適用して `set` を作成します。
2. 以前作成したセット `_a` に含まれていない値のみを残すために、`b` に対して `fn` と組み合わせたリスト内包表記を使用します。
3. 最後に、以前の結果と `a` から `set` を作成し、それを `list` に変換します。

この関数は次の入力パラメータを持つ必要があります。

- `a`: 要素のリスト
- `b`: 要素のリスト
- `fn`: 要素を受け取り、値を返す関数

この関数は要素のリストを返す必要があります。

## 例

`union_by()` が行うべきことの例を以下に示します。

```python
from math import floor

union_by([2.1], [1.2, 2.3], floor) # [2.1, 1.2]
```

この例では、`union_by()` は 2 つのリスト `[2.1]` と `[1.2, 2.3]` と、関数 `floor()` を引数に取ります。この関数は、両方のリストの各要素に `floor()` を適用して、`{2}` のセットを作成します。次に、そのセットに含まれていない値のみを残すためにリスト内包表記を使用します。これは `[1.2]` です。最後に、以前の結果と `[2.1]` から `{1.2, 2.1}` のセットを作成し、それを `[1.2, 2.1]` のリストに変換します。
