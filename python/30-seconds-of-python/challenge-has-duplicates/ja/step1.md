# リスト内の重複を確認する

## 問題

引数としてリストを受け取り、リストに重複する要素が含まれている場合は `True` を、そうでない場合は `False` を返す、`has_duplicates(lst)` という Python 関数を作成してください。

この問題を解くには、次の手順を使用できます。

1. `set()` 関数を使用して、リストから重複する要素を削除します。
2. 元のリストの長さとセットの長さを比較します。もし同じであれば、重複する要素はありません。異なる場合、重複する要素があります。

## 例

```python
x = [1, 2, 3, 4, 5, 5]
y = [1, 2, 3, 4, 5]
print(has_duplicates(x)) # True
print(has_duplicates(y)) # False
```
