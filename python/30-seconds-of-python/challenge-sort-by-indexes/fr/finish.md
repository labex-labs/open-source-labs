# Résumé

Dans ce défi, vous deviez créer une fonction qui trie une liste en fonction d'une autre liste contenant les index souhaités. En utilisant les fonctions `zip()` et `sorted()`, vous avez pu combiner et trier les deux listes en fonction des valeurs de `indexes`. Vous avez ensuite utilisé une compréhension de liste pour obtenir le premier élément de chaque paire du résultat et renvoyé la liste triée.
