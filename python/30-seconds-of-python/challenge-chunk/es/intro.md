# Introducción

En Python, a menudo necesitamos dividir una lista en listas más pequeñas de un tamaño especificado. Esto puede ser útil para varios propósitos, como procesar datos por lotes o mostrar datos en un formato paginado. En este desafío, tendrás la tarea de escribir una función que pueda dividir una lista en listas más pequeñas de un tamaño especificado.
