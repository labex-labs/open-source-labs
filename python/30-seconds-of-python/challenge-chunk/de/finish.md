# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie Sie eine Liste in kleinere Listen von einer bestimmten Größe unterteilen. Sie haben eine Funktion geschrieben, die eine Liste und eine positive ganze Zahl als Argumente nimmt und eine Liste von kleineren Listen zurückgibt, von denen jede eine maximale Größe der angegebenen ganzen Zahl hat. Dies kann für verschiedene Zwecke nützlich sein, beispielsweise beim Verarbeiten von Daten in Batches oder beim Anzeigen von Daten in einer seitenbasierten Formatierung.
