# Introduction

En Python, nous avons souvent besoin de diviser une liste en listes plus petites d'une taille spécifiée. Cela peut être utile à diverses fins, telles que le traitement des données par lots ou l'affichage des données sous forme paginée. Dans ce défi, vous devrez écrire une fonction qui peut diviser une liste en listes plus petites d'une taille spécifiée.
