# Km para Milhas

Escreva uma função Python chamada `km_to_miles` que recebe um parâmetro `km` (um float) e retorna a distância equivalente em milhas (também um float). A fórmula de conversão é `mi = km * 0.621371`.

```python
def km_to_miles(km):
  return km * 0.621371
```

```python
km_to_miles(8.1) # 5.0331051
```
