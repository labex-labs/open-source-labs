# Километры в мили

Напишите функцию на Python под названием `km_to_miles`, которая принимает параметр `km` (тип `float`) и возвращает эквивалентное расстояние в милях (также `float`). Формула преобразования: `mi = km * 0.621371`.

```python
def km_to_miles(km):
  return km * 0.621371
```

```python
km_to_miles(8.1) # 5.0331051
```
