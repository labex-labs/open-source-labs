# 辞書の値

## 問題

フラットな辞書が与えられ、辞書内のすべての値のフラットなリストを返す関数を作成する必要があります。あなたのタスクは、`values_only(flat_dict)`関数を実装することです。この関数は、フラットな辞書を引数として取り、辞書内のすべての値のリストを返します。

この問題を解くには、与えられた辞書の値を返すために`dict.values()`メソッドを使用できます。その後、`list()`関数を使用して結果をリストに変換できます。

## 例

```python
ages = {
  'Peter': 10,
  'Isabel': 11,
  'Anna': 9,
}
values_only(ages) # [10, 11, 9]
```
