# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie Sie alle Werte aus einem flachen Dictionary extrahieren und als Liste zurückgeben. Sie haben die Methode `dict.values()` verwendet, um die Werte zu erhalten und anschließend das Ergebnis mithilfe der Funktion `list()` in eine Liste umgewandelt. Dies ist eine nützliche Technik, wenn Sie mit Dictionaries in Python arbeiten.
