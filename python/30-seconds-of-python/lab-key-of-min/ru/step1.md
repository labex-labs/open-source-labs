# Ключ минимального значения

Напишите функцию `key_of_min(d)`, которая принимает словарь `d` в качестве аргумента и возвращает ключ минимального значения в словаре.

Для решения этой проблемы вы можете использовать встроенную функцию `min()` с параметром `key`, установленным на `dict.get()`. Это вернет ключ минимального значения в словаре.

```python
def key_of_min(d):
  return min(d, key = d.get)
```

```python
key_of_min({'a':4, 'b':0, 'c':13}) # b
```
