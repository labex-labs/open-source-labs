# Zusammenfassung

In dieser Herausforderung haben Sie gelernt, wie man mithilfe von Python den Schlüssel des kleinsten Werts in einem Dictionary findet. Indem Sie die Funktion `min()` mit dem Parameter `key` auf `dict.get()` setzen, können Sie leicht den Schlüssel des kleinsten Werts in einem gegebenen Dictionary ermitteln.
