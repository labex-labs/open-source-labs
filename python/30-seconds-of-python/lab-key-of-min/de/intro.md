# Einführung

In Python sind Dictionaries eine nützliche Datenstruktur zum Speichern von Schlüssel-Wert-Paaren. Manchmal müssen wir den Schlüssel des kleinsten Werts in einem Dictionary finden. In dieser Herausforderung haben Sie die Aufgabe, eine Funktion zu schreiben, die den Schlüssel des kleinsten Werts in einem gegebenen Dictionary findet.
