# Градусы в радианы

Напишите функцию `degrees_to_rads(deg)`, которая принимает угол в градусах в качестве аргумента и возвращает угол в радианах. Ваша функция должна использовать следующую формулу для преобразования градусов в радианы:

```
radians = (degrees * pi) / 180.0
```

где `pi` - это константное значение, представляющее отношение длины окружности к диаметру (примерно 3,14159).

Ваша функция должна возвращать угол в радианах, округленный до 4 десятичных знаков.

```python
from math import pi

def degrees_to_rads(deg):
  return (deg * pi) / 180.0
```

```python
degrees_to_rads(180) # ~3.1416
```
