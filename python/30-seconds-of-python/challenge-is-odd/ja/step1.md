# 数が奇数であるかチェックする

## 問題

`is_odd` という名前の関数を書きましょう。この関数は、1つの整数を引数として受け取り、その数が奇数の場合は `True` を返し、偶数の場合は `False` を返します。あなたの関数は以下の手順を実行する必要があります。

1. 剰余 (`%`) 演算子を使用して、その数が偶数か奇数かをチェックします。
2. その数が奇数の場合は `True` を返します。その数が偶数の場合は `False` を返します。

## 例

あなたの関数がどのように動作するかの例を以下に示します。

```python
>>> is_odd(3)
True
>>> is_odd(4)
False
```
