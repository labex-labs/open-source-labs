# Vérifier si un nombre est impair

## Problème

Écrivez une fonction appelée `is_odd` qui prend un seul entier en argument et renvoie `True` si le nombre est impair et `False` si le nombre est pair. Votre fonction devrait effectuer les étapes suivantes :

1. Utilisez l'opérateur modulo (`%`) pour vérifier si le nombre est pair ou impair.
2. Si le nombre est impair, renvoyez `True`. Si le nombre est pair, renvoyez `False`.

## Exemple

Voici un exemple de ce que devrait faire votre fonction :

```python
>>> is_odd(3)
True
>>> is_odd(4)
False
```
