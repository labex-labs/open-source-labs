# Résumé

Dans ce défi, vous avez appris à écrire une fonction qui vérifie si un nombre donné est impair ou pair. Cela peut être utile dans diverses situations de programmation, par exemple lorsqu'on travaille avec des boucles, des instructions conditionnelles ou des opérations mathématiques. N'oubliez pas d'utiliser l'opérateur modulo pour vérifier si un nombre est impair ou pair, et de renvoyer `True` si le nombre est impair et `False` si le nombre est pair.
