# Einführung

Beim Programmieren ist es oft erforderlich, zu überprüfen, ob eine gegebene Zahl ungerade oder gerade ist. Dies kann in verschiedenen Szenarien nützlich sein, wie beispielsweise bei der Arbeit mit Schleifen, bedingten Anweisungen oder mathematischen Operationen. In dieser Herausforderung werden Sie aufgefordert, eine Funktion zu schreiben, die überprüft, ob eine gegebene Zahl ungerade oder gerade ist.
