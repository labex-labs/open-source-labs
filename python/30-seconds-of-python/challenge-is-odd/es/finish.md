# Resumen

En este desafío, has aprendido cómo escribir una función que comprueba si un número dado es impar o par. Esto puede ser útil en varios escenarios de programación, como cuando se trabaja con bucles, declaraciones condicionales o operaciones matemáticas. Recuerda usar el operador módulo para comprobar si un número es impar o par, y devolver `True` si el número es impar y `False` si el número es par.
