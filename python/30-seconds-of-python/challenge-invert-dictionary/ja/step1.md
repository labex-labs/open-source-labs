# 辞書を反転させる

## 問題

`invert_dictionary(obj)` という名前の Python 関数を書きましょう。この関数は、辞書 `obj` を引数として受け取り、キーと値を反転させた新しい辞書を返します。入力辞書 `obj` は一意のハッシュ可能な値を持つものとします。出力辞書は、入力辞書と同じキーを持つ必要がありますが、値は入力辞書のキーになる必要があります。

新しい辞書を作成するには、`dictionary.items()` とリスト内包表記を組み合わせて使用する必要があります。

## 例

```python
ages = {
  'Peter': 10,
  'Isabel': 11,
  'Anna': 9,
}
invert_dictionary(ages) # { 10: 'Peter', 11: 'Isabel', 9: 'Anna' }
```
