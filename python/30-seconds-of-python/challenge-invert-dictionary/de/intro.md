# Einführung

Ein Wörterbuch ist eine Sammlung von Schlüssel-Wert-Paaren, wobei jeder Schlüssel einzigartig ist. In Python werden Wörterbücher weit verbreitet eingesetzt, um Daten effizient zu speichern und abzurufen. Manchmal müssen wir jedoch ein Wörterbuch umkehren, d. h., die Schlüssel und Werte tauschen. Dies kann in vielen Szenarien nützlich sein, beispielsweise beim Suchen nach einem Schlüssel basierend auf seinem Wert. In dieser Herausforderung werden Sie eine Python-Funktion zum Umkehren eines Wörterbuchs schreiben.
