# はじめに

この実験では、HTML、CSS、JavaScript を使用してスクロール進捗バーを作成する方法を探ります。スクロール進捗バーは、ユーザーのページ上の現在位置と、まだ見るべきコンテンツの量を示す優れた方法です。この実験の終了時には、この機能をあなたのウェブサイトに実装する方法をより深く理解することができるでしょう。
