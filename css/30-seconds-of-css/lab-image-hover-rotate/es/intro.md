# Introducción

En este laboratorio, aprenderemos cómo crear un efecto de hover que rote una imagen utilizando propiedades de CSS. Al utilizar `scale()`, `rotate()` y `transition`, podemos hacer que la imagen parezca rotar cuando el usuario pase el cursor sobre el elemento padre. También utilizaremos `overflow: hidden` para ocultar lo que sobra de la transformación de la imagen, creando un efecto limpio y pulido.
