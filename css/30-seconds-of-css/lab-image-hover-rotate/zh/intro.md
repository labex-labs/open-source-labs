# 简介

在本实验中，我们将学习如何使用 CSS 属性创建一个悬停效果，使图像旋转。通过使用 `scale()`、`rotate()` 和 `transition`，当用户将鼠标悬停在父元素上时，我们可以使图像看起来旋转。我们还将使用 `overflow: hidden` 来隐藏图像变换中多余的部分，从而创建一个简洁精致的效果。
