# Анимация увеличения и уменьшения

В виртуальной машине уже предоставлены файлы `index.html` и `style.css`.

Для создания анимации увеличения и уменьшения следуйте этим шагам:

1. Определите трехэтапную анимацию с использованием `@keyframes`. На `0%` и `100%` задайте элементу его исходный размер с помощью `scale(1,1)`. На `50%` увеличьте его размер в 1,5 раза с помощью `scale(1.5,1.5)`.

2. Задайте элементу определенные размеры с помощью `width` и `height`.

3. Используйте `animation`, чтобы задать соответствующие значения элементу и сделать его анимированным.

Вот пример кода HTML и CSS:

```html
<div class="zoom-in-out-box"></div>
```

```css
.zoom-in-out-box {
  margin: 24px;
  width: 50px;
  height: 50px;
  background: #f50057;
  animation: zoom-in-zoom-out 1s ease infinite;
}

@keyframes zoom-in-zoom-out {
  0% {
    transform: scale(1, 1);
  }
  50% {
    transform: scale(1.5, 1.5);
  }
  100% {
    transform: scale(1, 1);
  }
}
```
