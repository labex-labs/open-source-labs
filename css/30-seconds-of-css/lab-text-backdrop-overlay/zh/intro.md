# 简介

在这个实验中，我们将学习如何使用 CSS 实现图像文本叠加效果。本实验的目的是教你如何在图像上叠加显示文本，使其在任何背景图像和颜色下都清晰可读。你将学习如何使用 `backdrop-filter` 属性为文本应用模糊和亮度效果，从而创建出专业且视觉上吸引人的设计。
