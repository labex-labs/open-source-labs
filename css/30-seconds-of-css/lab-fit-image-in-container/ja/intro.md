# はじめに

この実験では、画像のアスペクト比を維持しながら、CSS を使って画像をコンテナ内に収まるように調整し、配置する方法を学びます。`object-fit` と `object-position` のプロパティを調べ、それらを使ってさまざまな結果を得る方法を学びます。この実験が終わるとき、指定されたスペース内に完璧に収まる視覚的に魅力的な画像を作成できるようになります。
