# Introdução

Este efeito pode ser usado para adicionar interesse visual ao texto e melhorar a experiência do usuário de um website. Ao seguir os passos descritos neste laboratório, você obterá uma melhor compreensão de como usar CSS para criar conteúdo web dinâmico e envolvente.
