# 简介

在本实验中，我们将学习如何使用 CSS 创建一个在悬停时具有放大动画效果的按钮。此效果可用于为网站或应用程序增添交互性和视觉吸引力。通过应用 `transform` 属性和 `:hover` 伪类，我们可以使按钮在用户悬停时放大，从而创建一个微妙而有效的动画。
