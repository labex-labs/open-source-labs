# Стилизация ссылок без текста

В ВМ уже предоставлены `index.html` и `style.css`.

Для отображения URL-адреса для ссылок, которые не содержат текст, вы можете использовать псевдо-класс `:empty` для выбора таких ссылок, псевдо-класс `:not` для исключения ссылок с текстом и свойство `content` в сочетании с функцией `attr()`, чтобы отобразить URL-адрес ссылки в псевдо-элементе `::before`. Вот пример фрагмента кода:

```html
<a href="https://30secondsofcode.org"></a>
```

```css
a[href^="http"]:empty::before {
  content: attr(href);
}
```

Пожалуйста, нажмите на кнопку "Go Live" в нижнем правом углу, чтобы запустить веб-сервис на порту 8080. Затем вы можете обновить вкладку **Web 8080**, чтобы просмотреть веб-страницу.
