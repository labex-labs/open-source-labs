# 简介

在本实验中，我们将学习如何使用 CSS 创建在状态更改时带有动画效果的自定义复选框。我们将使用 SVG 元素创建选中符号，使用弹性盒模型（Flexbox）布局复选框，并使用 CSS 动画创建缩放效果。完成本实验后，你将具备为你的网页项目创建时尚且交互式复选框的技能。
