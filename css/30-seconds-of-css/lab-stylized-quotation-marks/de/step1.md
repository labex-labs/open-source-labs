# Stilvolle Anführungszeichen

`index.html` und `style.css` wurden bereits in der VM bereitgestellt.

Um die Inline-Anführungszeichen anzupassen, modifizieren Sie die `quotes`-Eigenschaft innerhalb eines `<q>`-Elements.

Beispielsweise:

```html
<p><q>Do or do not, there is no try.</q> – Yoda</p>
```

kann mit geschweiften Anführungszeichen mit CSS formatiert werden:

```css
q {
  quotes: "“" "”";
}
```

Bitte klicken Sie auf 'Go Live' in der unteren rechten Ecke, um den Webdienst auf Port 8080 auszuführen. Anschließend können Sie die Registerkarte **Web 8080** aktualisieren, um die Webseite anzuschauen.
