# はじめに

この実験では、CSS の `quotes` プロパティを調べます。これにより、インライン引用符に使用する引用符をカスタマイズできます。このプロパティを使って独自の開始引用符と終了引用符を定義し、HTML の `<q>` 要素に適用する方法を学びます。この実験が終わるとき、Web ページのスタイリングに個性を加える方法をより深く理解しているでしょう。
