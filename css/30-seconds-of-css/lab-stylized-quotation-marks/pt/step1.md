# Aspas Estilizadas

`index.html` e `style.css` já foram fornecidos na VM (Virtual Machine).

Para personalizar as aspas inline, modifique a propriedade `quotes` dentro de um elemento `<q>`.

Por exemplo:

```html
<p><q>Do or do not, there is no try.</q> – Yoda</p>
```

pode ser estilizado com aspas curvas usando CSS:

```css
q {
  quotes: "“" "”";
}
```

Por favor, clique em 'Go Live' no canto inferior direito para executar o serviço web na porta 8080. Em seguida, você pode atualizar a aba **Web 8080** para visualizar a página web.
