# 简介

在本实验中，我们将学习 CSS 中的首字下沉（Drop Cap）技术。首字下沉是一种排版技术，用于为第一段的首字母增添视觉吸引力。通过本实验，你将了解如何使用`:first-child`选择器和`::first-letter`伪元素，以独特且引人注目的方式为段落的首字母设置样式。
