# 简介

在本实验中，我们将通过创建一个在状态变化时带有动画效果的自定义单选按钮来探索 CSS 编程概念。本实验将指导你完成使用弹性盒模型（Flexbox）为单选按钮创建布局、重置 `<input>` 元素的样式以及使用 `::before` 元素创建单选按钮内圈的过程。在实验结束时，你将对 CSS 样式和动画技术有更深入的理解。
