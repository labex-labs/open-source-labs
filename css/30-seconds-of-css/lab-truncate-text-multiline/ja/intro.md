# はじめに

この実験では、コンテナの寸法を超える複数行のテキストを切り詰める方法を学びます。この技術は、クリーンで整理されたデザインを維持しながら、長いテキストの抜粋を表示するのに役立ちます。`overflow`、`height`、`background` などの CSS プロパティを使用することで、視覚的に魅力的な切り詰め効果を作成して、ユーザー エクスペリエンスを向上させることができます。
