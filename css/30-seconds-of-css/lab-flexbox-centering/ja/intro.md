# はじめに

この実験では、CSS におけるフレックスボックスの中央揃えの概念を探ります。親要素内の子要素を水平および垂直方向に中央揃えする方法を学びます。この実験が終了すると、フレックスボックスの中央揃え技術を使って応答性が高く視覚的に魅力的なレイアウトを作成できるようになります。
