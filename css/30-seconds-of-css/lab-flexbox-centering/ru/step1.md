# Центрирование с использованием flexbox

В ВМ уже предоставлены файлы `index.html` и `style.css`.

Для горизонтального и вертикального центрирования дочернего элемента внутри родительского элемента с использованием flexbox выполните следующие шаги:

1. Создайте макет с использованием flexbox, установив для родительского элемента свойство `display` равным `flex`.
2. Используйте свойство `justify-content`, установив его значение равным `center`, чтобы центрировать дочерний элемент по горизонтали.
3. Используйте свойство `align-items`, установив его значение равным `center`, чтобы центрировать дочерний элемент по вертикали.
4. Добавьте дочерний элемент внутри родительского элемента.

Ниже представлен пример кода:

```html
<div class="flexbox-centering">
  <div>Centered content.</div>
</div>
```

```css
.flexbox-centering {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100px;
}
```
