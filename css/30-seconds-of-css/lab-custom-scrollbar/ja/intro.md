# はじめに

この実験では、CSS を使ってスクロールバーのスタイルをカスタマイズする方法を探ります。`::-webkit-scrollbar` プロパティを使って、スクロールバーのトラックやサムなどのスクロールバーのスタイルを設定します。この実験が終わるとき、カスタムなスクロールバーを作成し、ウェブサイトのユーザーエクスペリエンスを向上させる方法をより深く理解しているでしょう。
