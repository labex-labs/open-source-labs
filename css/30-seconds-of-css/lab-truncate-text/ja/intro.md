# はじめに

この実験では、CSS プログラミングの世界に入り、1 行の幅を超えるテキストを省略する方法を学びます。`overflow`、`white-space`、および `text-overflow` プロパティを組み合わせることで、省略されたテキストの末尾に省略記号を表示できるようになります。この実験を通じて、ウェブサイト上のテキストの表示を最適化する方法を深く理解することができます。
