# Einführung

In diesem Lab werden wir untersuchen, wie man für Elemente mit variierender Breite ein konstantes Verhältnis von Breite zu Höhe erstellt. Indem wir die `padding-top`-Eigenschaft auf einem `::before`-Pseudo-Element verwenden, können wir gewährleisten, dass die Höhe des Elements proportional zu seiner Breite bleibt. Diese Technik kann verwendet werden, um responsive Quadrate und andere Formen mit bestimmten Seitenverhältnissen zu erstellen.
