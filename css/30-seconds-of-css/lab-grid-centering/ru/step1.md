# Центрирование элементов сетки

В ВМ уже предоставлены файлы `index.html` и `style.css`.

Для горизонтального и вертикального центрирования дочернего элемента внутри родительского элемента следуйте шагам:

1. Создайте макет сетки с использованием `display: grid`.
2. Используйте `justify-content: center`, чтобы центрировать дочерний элемент по горизонтали.
3. Используйте `align-items: center`, чтобы центрировать дочерний элемент по вертикали.

Вот пример структуры HTML:

```html
<div class="parent">
  <div class="child">Centered content.</div>
</div>
```

И соответствующий CSS:

```css
.parent {
  display: grid;
  justify-content: center;
  align-items: center;
  height: 100px;
}
```
