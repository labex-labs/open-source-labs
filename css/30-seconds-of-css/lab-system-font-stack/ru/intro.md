# Введение

В этом практическом занятии мы изучим концепцию системных шрифтовых стеков в CSS. Вы научитесь определять список шрифтов, которые браузер будет искать на операционной системе пользователя, и как использовать эту технику для создания ощущения нативного приложения на ваших веб-страницах. В конце практического занятия у вас будет твердый понимание того, как реализовать системные шрифтовые стеки в своих собственных проектах.
