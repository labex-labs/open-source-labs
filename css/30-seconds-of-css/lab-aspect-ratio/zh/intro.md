# 简介

在本实验中，我们将学习如何使用 CSS 创建具有指定宽高比的响应式容器。你将了解 `--aspect-ratio` 自定义属性，以及如何使用 `calc()` 函数来计算容器的高度。你还将学习如何使用 `object-fit: cover` 属性设置子元素以保持宽高比。在本实验结束时，你将更好地理解如何在网页上创建视觉上吸引人的响应式容器。
