# 简介

在本实验中，我们将学习如何使用 CSS 为按钮创建悬停时的填充动画。我们将使用 `:hover` 伪类在用户悬停在按钮上时更改按钮的 `背景` 和 `颜色`，并添加过渡效果来为这些更改设置动画。完成本实验后，你将能够通过为按钮添加引人入胜的悬停效果来提升网站上的用户体验。
