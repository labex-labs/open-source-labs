# 소개

이 랩에서는 CSS 를 사용하여 펄스 효과 로더 애니메이션을 만드는 방법을 배웁니다. `animation-delay` 속성을 사용하여 리듬 효과를 만들고, `@keyframes`를 사용하여 사이클의 두 지점에서 애니메이션을 정의합니다. 이 랩을 마치면 웹 페이지의 사용자 경험을 향상시킬 수 있는 매력적인 애니메이션을 만드는 방법에 대한 이해를 높일 수 있습니다.
