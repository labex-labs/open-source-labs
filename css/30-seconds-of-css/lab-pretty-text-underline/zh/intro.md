# 简介

在本实验中，我们将学习如何使用 CSS 创建一个美观的文本下划线，且不会裁剪下行字母。通过结合使用 text-shadow 和带有线性渐变的 background-image，我们可以创建一个渐变效果，该渐变将充当实际的下划线，同时确保文本仍可选中。此技术为 text-decoration: underline 提供了一种更具视觉吸引力的替代方案。
