# はじめに

この実験では、CSS を使ってホバー時にボタンの枠線をアニメーションさせる方法を探ります。`::before` と `::after` 疑似要素を使うことで、ボタンの上下に 2 つのボックスを追加し、ホバー時にそれらの幅を `100%` に変化させることができます。この実験は、CSS のスキルを向上させ、Web ページにインタラクティビティを追加するのに役立ちます。
