# Шахматный узор фона

В виртуальной машине (VM) уже предоставлены файлы `index.html` и `style.css`.

Для создания шахматного узора фона следуйте этим шагам:

1. Установите свойство `background-color` в значение белого цвета.
2. Используйте `background-image` с двумя значениями `linear-gradient()`, каждый с разным углом, чтобы создать шахматный узор. Например, установите один угол в `45deg`, а другой в `-45deg`.
3. Укажите размер узора с помощью `background-size`. Например, `60px 60px` создаст узор размером 60 на 60 пикселей.
4. Используйте `background-repeat` для настройки повторения узора. Например, `repeat` заставит узор повторяться в обоих направлениях.
5. Обратите внимание, что свойства `height` и `width` элемента фиксированы на 240 пикселя для демонстрации.

Вот пример HTML-элемента с классом `.checkerboard`:

```html
<div class="checkerboard"></div>
```

А вот соответствующий CSS:

```css
.checkerboard {
  width: 240px;
  height: 240px;
  background-color: #fff;
  background-image:
    linear-gradient(
      45deg,
      #000 25%,
      transparent 25%,
      transparent 75%,
      #000 75%,
      #000
    ),
    linear-gradient(
      -45deg,
      #000 25%,
      transparent 25%,
      transparent 75%,
      #000 75%,
      #000
    );
  background-size: 60px 60px;
  background-repeat: repeat;
}
```

Пожалуйста, нажмите на кнопку 'Go Live' в правом нижнем углу, чтобы запустить веб-сервис на порту 8080. Затем вы можете обновить вкладку **Web 8080**, чтобы предварительно просмотреть веб-страницу.
