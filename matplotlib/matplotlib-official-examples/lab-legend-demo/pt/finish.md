# Resumo

Neste laboratório, aprendemos como criar e personalizar legendas no Matplotlib. Demonstramos como criar legendas para linhas específicas, rótulos complexos e gráficos mais complexos. Também mostramos como escrever classes personalizadas para estilizar legendas. As legendas são uma parte importante de qualquer gráfico, e entender como criá-las e personalizá-las é essencial para criar visualizações eficazes.
