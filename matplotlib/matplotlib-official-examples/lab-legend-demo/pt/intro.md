# Introdução

Neste laboratório, exploraremos como criar e personalizar legendas no Matplotlib. As legendas são usadas para explicar o significado dos elementos em um gráfico, incluindo linhas, barras e marcadores. Demonstraremos como criar legendas para linhas específicas, rótulos complexos e gráficos mais complexos. Finalmente, mostraremos como escrever classes personalizadas para estilizar legendas.
