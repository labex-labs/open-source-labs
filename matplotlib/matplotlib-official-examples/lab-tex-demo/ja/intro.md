# はじめに

この実験では、Matplotlib ライブラリを使ってさまざまなグラフを作成する方法を学びます。Matplotlib は、データの可視化に使用される Python ライブラリです。NumPy と SciPy ライブラリの上に構築されており、折れ線グラフ、散布図、棒グラフなど、幅広い可視化を作成できます。

## VM のヒント
