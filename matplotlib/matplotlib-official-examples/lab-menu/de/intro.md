# Einführung

In diesem Lab lernst du, wie du mithilfe von Matplotlib ein einfaches Menü erstellen kannst. Das Menü wird eine Liste von Elementen enthalten, von denen jedes vom Benutzer ausgewählt werden kann. Das ausgewählte Element löst dann eine Callback-Funktion aus, die eine Aktion ausführt.
