# Bibliotheken importieren

Um loszulegen, müssen wir die erforderlichen Bibliotheken importieren. In diesem Fall müssen wir `matplotlib.pyplot`, `matplotlib.artist`, `matplotlib.patches` und `IdentityTransform` aus `matplotlib.transforms` importieren.

```python
import matplotlib.pyplot as plt
import matplotlib.artist as artist
import matplotlib.patches as patches
from matplotlib.transforms import IdentityTransform
```
