# Создать фигуру и оси

В этом шаге вы создадите фигуру и оси для графика. Также вы будете настраивать позицию осей, чтобы腾出 место для ползунков.

```python
fig, ax = plt.subplots()
fig.subplots_adjust(bottom=0.25)
l, = ax.plot(t, s, lw=2)

ax_freq = fig.add_axes([0.25, 0.1, 0.65, 0.03])
ax_amp = fig.add_axes([0.25, 0.15, 0.65, 0.03])
```

注：“腾出”为根据语境意译，原文“make room for”直译为“为……腾出空间” 。
