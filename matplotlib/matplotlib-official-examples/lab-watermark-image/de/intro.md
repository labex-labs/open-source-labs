# Einführung

Dieses Tutorial zeigt Ihnen, wie Sie ein Bild auf einem Matplotlib-Diagramm platzieren, indem Sie es nach vorne bringen und halbtransparent machen. Das Tutorial verwendet die Methode `figimage` aus der Klasse `matplotlib.figure.Figure` und die Methode `imread` aus dem Modul `matplotlib.image`.
