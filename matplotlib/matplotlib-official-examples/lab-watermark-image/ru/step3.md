# Создание графика

Теперь мы можем создать график, на который будем накладывать изображение. В этом примере мы создадим простой столбчатый график с использованием случайных данных.

```python
fig, ax = plt.subplots()

np.random.seed(19680801)
x = np.arange(30)
y = x + np.random.randn(30)
ax.bar(x, y, color='#6bbc6b')
ax.grid()
```
