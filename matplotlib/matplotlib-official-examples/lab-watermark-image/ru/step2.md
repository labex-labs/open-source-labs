# Загрузка изображения

Далее нам нужно загрузить изображение, которое мы хотим наложить на график. Мы можем использовать метод `get_sample_data` из модуля `matplotlib.cbook` для загрузки образцового изображения. В этом примере мы будем использовать изображение `logo2.png`.

```python
with cbook.get_sample_data('logo2.png') as file:
    im = image.imread(file)
```
