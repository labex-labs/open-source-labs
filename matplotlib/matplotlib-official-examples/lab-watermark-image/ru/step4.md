# Наложение изображения

Для наложения изображения на график можно использовать метод `figimage` из класса `matplotlib.figure.Figure`. Необходимо указать изображение, позицию изображения на графике, z-порядок (чтобы переместить изображение на передний план) и альфа-канал (чтобы сделать изображение полупрозрачным).

```python
fig.figimage(im, 25, 25, zorder=3, alpha=.7)
```
