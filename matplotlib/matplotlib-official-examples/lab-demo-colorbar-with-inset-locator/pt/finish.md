# Resumo

Neste tutorial, aprendemos como usar `mpl_toolkits.axes_grid1.inset_locator.inset_axes` para adicionar uma barra de cores a um gráfico com tamanho e posição personalizados. A função `inset_axes` nos permite controlar a posição, altura e largura da barra de cores, facilitando a criação de gráficos com aparência profissional.
