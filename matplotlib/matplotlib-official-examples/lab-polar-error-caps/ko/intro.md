# 소개

데이터 시각화에서 오차 막대 (error bar) 는 데이터 포인트의 불확실성 또는 변동성을 나타내는 데 사용됩니다. Matplotlib 은 Python 에서 널리 사용되는 데이터 시각화 라이브러리이며, 오차 막대를 위한 내장 지원을 제공합니다. 이 랩에서는 Matplotlib 을 사용하여 극좌표계에서 오차 막대 플롯을 만드는 방법을 배웁니다.
