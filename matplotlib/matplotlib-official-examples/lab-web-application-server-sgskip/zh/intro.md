# 简介

本实验教程将指导你在 Flask Web 应用服务器中使用 Matplotlib。你将学习如何使用`.Figure`构造函数创建图形并将其保存到内存缓冲区，将生成的图形嵌入 HTML 输出中，以及使用`flask`命令行工具运行 Flask 应用程序。
