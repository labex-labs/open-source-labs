# Resumen

Este tutorial mostró cómo trazar flechas que se comportan de manera diferente cuando se cambian los límites de datos en un gráfico. Cubrió tres casos de uso para trazar flechas, dependiendo de si la cabeza o los puntos de anclaje deben ser fijos en el espacio de datos o de visualización. Estos casos de uso pueden ser útiles para anotar gráficos y asegurarse de que la flecha quede en la posición correcta incluso si el gráfico se desplaza o escala.
