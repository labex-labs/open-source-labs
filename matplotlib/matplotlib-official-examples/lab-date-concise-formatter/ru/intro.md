# Введение

`~.dates.ConciseDateFormatter` - полезный инструмент для форматирования меток дат при работе с Matplotlib. В этом практическом занятии вы научитесь использовать этот форматтер для улучшения строк, выбранных для меток делений оси времени, и для минимизации количества строк, используемых в этих метках делений.
