# はじめに

この実験では、`transforms.offset_copy` を使用して、任意の座標で指定された場所に対して画面座標における指定されたオフセットで、文字列などの描画要素を配置する方法を学びます。Python の Matplotlib ライブラリを使用して散布図と極座標グラフを作成し、その後 `offset_copy` を使用して各グラフにテキストを追加します。

## VM のヒント

VM の起動が完了したら、左上隅をクリックして **ノートブック** タブに切り替え、Jupyter Notebook を使用して練習します。

時々、Jupyter Notebook が読み込み完了するまで数秒待つ必要がある場合があります。Jupyter Notebook の制限により、操作の検証を自動化することはできません。
