# 소개

이 랩에서는 `transforms.offset_copy`를 사용하여 텍스트 문자열과 같은 드로잉 요소를 임의의 좌표계에서 주어진 위치를 기준으로 화면 좌표에서 지정된 오프셋으로 배치하는 방법을 배웁니다. Python 의 Matplotlib 라이브러리를 사용하여 산점도와 극좌표 플롯을 생성한 다음, `offset_copy`를 사용하여 각 플롯에 텍스트를 추가합니다.
