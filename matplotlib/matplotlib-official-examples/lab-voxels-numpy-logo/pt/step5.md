# Reduzir as lacunas

Reduzimos as lacunas entre cada voxel modificando as coordenadas de cada voxel usando a função `indices` do NumPy.

```python
x, y, z = np.indices(np.array(filled_2.shape) + 1).astype(float) // 2
x[0::2, :, :] += 0.05
y[:, 0::2, :] += 0.05
z[:, :, 0::2] += 0.05
x[1::2, :, :] += 0.95
y[:, 1::2, :] += 0.95
z[:, :, 1::2] += 0.95
```
