# はじめに

Matplotlib は、Python における強力なデータ可視化ライブラリです。これを使うと、散布図、ヒストグラム、棒グラフなど、様々なプロットを作成できます。スタイルシートのリファレンススクリプトは、共通のセットの例プロットに対して利用可能な異なるスタイルシートを示しています。この実験では、Matplotlib のスタイルシートを使ってプロットの外観をカスタマイズする方法を学びます。

## VM のヒント
