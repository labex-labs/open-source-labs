# 소개

이 랩에서는 Python 의 Matplotlib 라이브러리를 사용하여 두 신호의 coherence (일관성) 를 플롯하는 방법을 보여줍니다. 두 신호의 coherence 는 선형 관계의 척도이며, 값 1 은 완벽한 coherence 를, 값 0 은 coherence 가 없음을 나타냅니다.
