# 소개

Matplotlib 는 Python 에서 널리 사용되는 데이터 시각화 라이브러리입니다. 때로는 동일한 그래프에 서로 다른 스케일의 데이터를 표시해야 할 때가 있는데, 이때 보조 축 (secondary axis) 의 개념이 유용하게 사용됩니다. 이 랩에서는 Matplotlib 에서 보조 축을 만드는 방법을 배우겠습니다.
