# Резюме

В этом практическом занятии мы узнали, как использовать Matplotlib для создания графика с логотипом Firefox. Мы проанализировали данные SVG-пути с использованием настраиваемой функции и создали график с использованием объектов `PathPatch`. Это практическое занятие хорошо знакомит с возможностями Matplotlib для создания сложных форм и дизайнов.
