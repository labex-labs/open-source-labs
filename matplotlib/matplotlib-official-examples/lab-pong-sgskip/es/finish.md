# Resumen

En este laboratorio, aprendiste cómo crear un juego interactivo de Pong utilizando Matplotlib. Aprendiste cómo definir las clases `Pad` y `Puck`, así como la clase `Game`, que se encarga de manejar la lógica del juego y de dibujar el juego en la pantalla. También aprendiste cómo crear la animación del juego instanciando un objeto `Game` y llamando a su método `draw()` en un bucle. Con este conocimiento, ahora puedes crear tus propios juegos interactivos utilizando Python y Matplotlib.
