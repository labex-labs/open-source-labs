# Introducción

En este laboratorio, aprenderá a crear un juego interactivo de Pong utilizando Matplotlib. Pong es un juego arcade clásico que implica dos jugadores golpeando una pelota de un lado a otro a través de un campo de juego virtual utilizando palas. Este laboratorio lo guiará a través del proceso de construcción de su propio juego de Pong utilizando Python y Matplotlib.
