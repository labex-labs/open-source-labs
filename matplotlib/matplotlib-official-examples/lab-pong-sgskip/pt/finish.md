# Resumo

Neste laboratório, você aprendeu como criar um jogo interativo de Pong usando Matplotlib. Você aprendeu como definir as classes `Pad` e `Puck`, bem como a classe `Game`, que é responsável por lidar com a lógica do jogo e desenhar o jogo na tela. Você também aprendeu como criar a animação do jogo instanciando um objeto `Game` e chamando seu método `draw()` em um loop. Com este conhecimento, você agora pode criar seus próprios jogos interativos usando Python e Matplotlib.
