# Füge der zweiten Parasitenachse eine rechte y-Achse hinzu

Wir fügen der zweiten Parasitenachse eine rechte y-Achse hinzu mit der `par2.axis["right2"] = par2.new_fixed_axis(loc="right", offset=(60, 0))`-Methode.

```python
par2.axis["right2"] = par2.new_fixed_axis(loc="right", offset=(60, 0))
```
