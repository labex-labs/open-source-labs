# Resumo

Neste laboratório, aprendemos como criar eixos parasitas usando os módulos `mpl_toolkits.axes_grid1.parasite_axes.HostAxes` e `mpl_toolkits.axes_grid1.parasite_axes.ParasiteAxes` em Matplotlib. Também aprendemos como plotar dados nos eixos parasitas e compartilhar a escala x com os eixos host.
