# 소개

이 랩은 Python 의 Matplotlib 라이브러리를 사용하여 이동 및 클릭 이벤트를 연결하여 플로팅 캔버스와 상호 작용하는 방법을 보여주는 예시입니다. Matplotlib 는 사용자가 Python 에서 정적, 애니메이션 및 대화형 시각화를 생성할 수 있도록 하는 데이터 시각화 라이브러리입니다.
