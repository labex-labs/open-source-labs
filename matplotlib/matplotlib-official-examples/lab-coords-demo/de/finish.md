# Zusammenfassung

In diesem Lab wurde gezeigt, wie man mit einem Matplotlib-Plot mithilfe von Mausbewegung- und Mausklickerevents interagieren kann. Indem wir uns an diese Events anmelden, können wir verschiedene Aktionen ausführen, wie das Ausgeben der Koordinaten der Mauszeiger, das Trennen von Callbacks usw. Diese Technik kann hilfreich sein, um interaktive Visualisierungen in Python zu erstellen.
