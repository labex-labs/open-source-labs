# Einführung

In diesem Lab wird ein Beispiel gezeigt, wie man mit der Zeichenfläche interagiert, indem man mit Hilfe der Matplotlib-Bibliothek in Python auf Bewegungs- und Klickereignisse reagiert. Matplotlib ist eine Datenvisualisierungsbibliothek, die es Benutzern ermöglicht, statische, animierte und interaktive Visualisierungen in Python zu erstellen.
