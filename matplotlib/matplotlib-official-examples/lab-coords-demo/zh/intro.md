# 简介

本实验展示了一个示例，演示如何使用 Python 中的 Matplotlib 库连接到移动和点击事件，从而与绘图画布进行交互。Matplotlib 是一个数据可视化库，允许用户在 Python 中创建静态、动画和交互式可视化。
