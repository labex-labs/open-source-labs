# Einführung

In diesem Tutorial wird Ihnen der Prozess des Zeichnens und Manipulieren von Formen mithilfe der Klassen `matplotlib.path.Path`, `matplotlib.patches.PathPatch` und `matplotlib.transforms` erläutert. Sie werden lernen, wie Sie einen Plot mit einem Kreis, einer Delfin-Form und einer um 60 Grad gedrehten Delfin-Form erstellen.
