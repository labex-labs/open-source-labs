# Introducción

Este tutorial lo guiará a través del proceso de dibujar y manipular formas utilizando las clases `matplotlib.path.Path`, `matplotlib.patches.PathPatch` y `matplotlib.transforms`. Aprenderá cómo crear un gráfico con un círculo, una forma de delfín y una forma de delfín rotada 60 grados.
