# Resumen

En este tutorial, has aprendido cómo crear un gráfico con un círculo, una forma de delfín y una forma de delfín rotada 60 grados. Has utilizado las clases `matplotlib.path.Path`, `matplotlib.patches.PathPatch` y `matplotlib.transforms` para dibujar y manipular formas en el gráfico. Ahora puedes utilizar este conocimiento para crear formas y gráficos más complejos.
