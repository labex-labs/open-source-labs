# 소개

이 튜토리얼에서는 Python Matplotlib 을 사용하여 풍향 막대 그래프를 만드는 방법을 배웁니다. 풍향 막대는 깃발, 선, 점의 조합을 사용하여 풍속과 풍향을 시각적으로 표현한 것입니다. 선의 길이는 풍속을 나타내고, 깃발과 점의 방향은 풍향을 나타냅니다.
