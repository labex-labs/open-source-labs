# はじめに

この実験では、Matplotlib でキーワード引数を使ってフォントプロパティを設定する方法を学びます。異なるフォントファミリー、スタイル、バリアント、ウェイト、サイズを調べて、テキストの外観をカスタマイズします。Matplotlib の `fig.text()` メソッドを使って、異なるフォントオプションを表示します。

## VM のヒント

VM の起動が完了したら、左上隅をクリックして **ノートブック** タブに切り替えて、Jupyter Notebook を使って練習します。

時々、Jupyter Notebook が読み込み終わるまで数秒待つ必要があります。Jupyter Notebook の制限により、操作の検証を自動化できません。
