# Введение

Matplotlib - это библиотека для визуализации данных, которая позволяет создавать различные визуализации на Python. Одним из важных аспектов создания визуализаций является выбор подходящей шрифта для текста. В этом руководстве мы узнаем, как использовать шрифты PGF в Matplotlib.
