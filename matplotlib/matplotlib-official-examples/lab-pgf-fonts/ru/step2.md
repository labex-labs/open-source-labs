# Задаем семейство шрифтов

Мы установим семейство шрифтов на "serif" с использованием параметра `font.family`. Кроме того, мы установим параметр `font.serif` в пустой список, чтобы использовать стандартный LaTeX-шрифт с засечками.

```python
plt.rcParams.update({
    "font.family": "serif",
    "font.serif": [],
})
```
