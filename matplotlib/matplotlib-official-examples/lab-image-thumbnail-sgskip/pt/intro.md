# Introdução

Este laboratório irá guiá-lo através do processo de geração de miniaturas (thumbnails) a partir de imagens existentes, utilizando a biblioteca Matplotlib em Python. Miniaturas são versões menores de imagens que podem ser usadas para exibir uma pré-visualização da imagem maior. Matplotlib depende da biblioteca Pillow para ler imagens e suporta todos os formatos suportados por Pillow.
