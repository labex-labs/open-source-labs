# Resumo

Neste laboratório, você aprendeu como gerar miniaturas a partir de imagens existentes usando Matplotlib em Python. Você aprendeu como importar bibliotecas, analisar argumentos, verificar diretórios, criar diretórios de saída e gerar miniaturas. Seguindo as etapas deste laboratório, você pode facilmente gerar miniaturas para todas as imagens em um diretório.
