# Introdução

Matplotlib é uma popular biblioteca Python utilizada para visualização de dados. Uma das principais características do Matplotlib é a capacidade de personalizar os rótulos dos ticks (tick labels) em um gráfico. Neste laboratório, você aprenderá como rotacionar rótulos de ticks personalizados usando Matplotlib.
