# Introducción

Matplotlib es una popular biblioteca de Python utilizada para la visualización de datos. Una de las características clave de Matplotlib es la capacidad de personalizar las etiquetas de los ejes en un gráfico. En este laboratorio, aprenderás cómo rotar las etiquetas de los ejes personalizadas utilizando Matplotlib.
