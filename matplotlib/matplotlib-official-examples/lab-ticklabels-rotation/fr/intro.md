# Introduction

Matplotlib est une bibliothèque Python populaire utilisée pour la visualisation de données. L'une des principales fonctionnalités de Matplotlib est la capacité à personnaliser les étiquettes d'échelle sur un graphique. Dans ce laboratoire, vous allez apprendre à faire pivoter des étiquettes d'échelle personnalisées à l'aide de Matplotlib.
