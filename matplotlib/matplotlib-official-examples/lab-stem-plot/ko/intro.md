# 소개

줄기 플롯 (stem plot) 은 데이터 시각화에 사용되는 플롯 유형입니다. 기준선에서 y 좌표까지 수직선을 그리고, 그 끝에 마커를 배치하는 데 사용됩니다. 이 랩에서는 Python 의 Matplotlib 라이브러리에서 줄기 플롯 함수를 사용하는 방법을 배웁니다.
