# はじめに

Matplotlib は、Python でデータ可視化を行うためのツールを幅広く提供する人気のあるデータ可視化ライブラリです。Matplotlib の興味深い機能の 1 つは、グラフに十字マーカーカーソルを追加する機能です。この実験では、Matplotlib のグラフに十字マーカーカーソルを追加する方法を学びます。

## VM のヒント

VM の起動が完了したら、左上隅をクリックして**ノートブック**タブに切り替え、Jupyter Notebook を使って練習しましょう。

Jupyter Notebook の読み込みには数秒かかる場合があります。Jupyter Notebook の制限により、操作の検証は自動化できません。
