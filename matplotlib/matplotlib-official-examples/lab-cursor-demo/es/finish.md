# Resumen

En este laboratorio, aprendió cómo agregar un cursor de cruz a una gráfica de Matplotlib. Creó un cursor simple, un cursor que utiliza el bliteado para una representación más rápida y un cursor que se ajusta automáticamente a los puntos de datos de un objeto `Line2D`. Estos cursores pueden ser útiles para explorar datos y obtener información valiosa a partir de visualizaciones.
