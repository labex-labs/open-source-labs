# Introducción

Matplotlib es una popular biblioteca de visualización de datos que ofrece una amplia variedad de herramientas para crear visualizaciones en Python. Una de las características interesantes de Matplotlib es la capacidad de agregar un cursor de cruz a una gráfica. En este laboratorio, aprenderá cómo agregar un cursor de cruz a una gráfica de Matplotlib.
