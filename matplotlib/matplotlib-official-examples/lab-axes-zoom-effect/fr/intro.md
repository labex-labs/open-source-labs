# Introduction

Ce tutoriel vous guidera sur la manière de créer un effet de zoom à l'aide de matplotlib. L'effet de zoom vous permettra de vous connecter et de zoomer sur deux axes différents.

## Conseils sur la machine virtuelle

Une fois le démarrage de la machine virtuelle terminé, cliquez sur le coin supérieur gauche pour basculer vers l'onglet **Carnet de notes** pour accéder à Jupyter Notebook pour pratiquer.
