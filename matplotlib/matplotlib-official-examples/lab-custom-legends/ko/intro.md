# 소개

Python Matplotlib 라이브러리는 플롯에서 범례를 생성하고 사용자 정의하는 유연한 방법을 제공합니다. 범례는 플롯에 표시된 데이터를 명확하고 간결하게 설명하므로 모든 플롯의 필수 구성 요소입니다. 이 랩에서는 Matplotlib 객체를 사용하여 사용자 정의 범례를 구성하는 과정을 안내합니다.
