# Introdução

Matplotlib é uma biblioteca Python utilizada para visualização de dados. Permite aos utilizadores criar uma vasta gama de gráficos, diagramas e plots. Uma das principais características do Matplotlib é a sua capacidade de aplicar transformações de escala aos eixos. Isto permite uma maior flexibilidade na apresentação de dados, especialmente quando se lida com números muito grandes ou muito pequenos. Neste laboratório, aprenderemos como aplicar várias transformações de escala aos eixos usando Matplotlib.
