# Резюме

В этом практическом занятии мы узнали, как создавать гистограммы рядом для нескольких наборов данных с использованием Matplotlib. Мы вычислили необходимые величины для построения и использовали метод `barh` для создания горизонтальных столбцов для каждой гистограммы. С этими навыками мы можем создавать информативные визуализации наших данных, чтобы получить представление о них и передать свои результаты другим.
