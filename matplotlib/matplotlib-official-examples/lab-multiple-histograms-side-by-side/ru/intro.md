# Введение

Гистограммы - прекрасный способ визуализировать распределение набора данных, а Matplotlib - одна из самых популярных библиотек Python для создания визуализаций данных. В этом практическом занятии мы будем использовать Matplotlib для создания бок о бок гистограмм для нескольких наборов данных.
