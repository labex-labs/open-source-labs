# 소개

히스토그램은 데이터 세트의 분포를 시각화하는 훌륭한 방법이며, Matplotlib 은 데이터 시각화를 생성하기 위한 가장 인기 있는 Python 라이브러리 중 하나입니다. 이 랩에서는 Matplotlib 을 사용하여 여러 데이터 세트에 대한 나란히 히스토그램을 생성합니다.
