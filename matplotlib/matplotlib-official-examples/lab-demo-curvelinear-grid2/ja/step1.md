# 必要なライブラリをインポートする

最初のステップは、必要なライブラリをインポートすることです。これには、`matplotlib.pyplot`、`numpy`、`ExtremeFinderSimple`、`MaxNLocator`、および`GridHelperCurveLinear`が含まれます。

```python
import matplotlib.pyplot as plt
import numpy as np
from mpl_toolkits.axisartist.axislines import Axes
from mpl_toolkits.axisartist.grid_finder import (ExtremeFinderSimple, MaxNLocator)
from mpl_toolkits.axisartist.grid_helper_curvelinear import GridHelperCurveLinear
```
