# 소개

이 랩에서는 Python Matplotlib 을 사용하여 라쏘 도구로 데이터 포인트를 대화식으로 선택하는 방법을 배웁니다. 산점도를 그리고 그래프의 점 주위에 라쏘 루프를 그려서 몇 개의 점을 선택합니다. 그리려면 그래프를 클릭하고, 누른 상태에서 선택하려는 점 주위로 드래그하면 됩니다.
