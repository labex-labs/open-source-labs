# 소개

이 랩에서는 Matplotlib 의 `figimage` 함수를 사용하여 Axes 객체 없이 이미지를 그림에 직접 배치하는 방법을 배웁니다. 이는 로고나 워터마크와 같이 플롯의 일부가 아닌 이미지를 포함하려는 경우 유용할 수 있습니다.

## VM 팁
