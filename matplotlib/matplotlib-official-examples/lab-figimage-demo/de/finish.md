# Zusammenfassung

In diesem Lab haben wir gelernt, wie man Matplotlib's `figimage`-Funktion verwendet, um Bilder direkt in eine Figur zu platzieren. Dies kann nützlich sein, wenn Sie Bilder einfügen möchten, die kein Teil Ihres Plots sind. Indem Sie den in diesem Lab beschriebenen Schritten folgen, sollten Sie jetzt ein besseres Verständnis dafür haben, wie man Bilder in Matplotlib-Figuren erstellt und anzeigt.
