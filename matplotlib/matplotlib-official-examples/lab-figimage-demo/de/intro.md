# Einführung

In diesem Lab lernst du, wie du die `figimage`-Funktion von Matplotlib verwendest, um Bilder direkt in eine Figur zu platzieren, ohne dass Axes-Objekte erforderlich sind. Dies kann nützlich sein, wenn du Bilder einfügen möchtest, die kein Teil deines Plots sind, wie z. B. ein Logo oder ein Wasserzeichen.
