# Resumo

Neste laboratório, aprendemos como criar um gráfico com múltiplos conjuntos de dados usando eixos parasitas (parasite axis) em Python Matplotlib. Usamos `mpl_toolkits.axes_grid1.parasite_axes.host_subplot` e `mpl_toolkits.axisartist.axislines.Axes` para criar um gráfico com múltiplos eixos y. Também aprendemos como ajustar a posição dos eixos parasitas, plotar os dados, definir os limites e rótulos dos eixos, adicionar legenda e cor ao gráfico, e exibir o gráfico.
