# 简介

在本实验中，我们将学习如何使用 Python Matplotlib 中的寄生轴创建多数据集图。我们将使用 `mpl_toolkits.axes_grid1.parasite_axes.host_subplot` 和 `mpl_toolkits.axisartist.axislines.Axes` 创建具有多个 y 轴的图。
