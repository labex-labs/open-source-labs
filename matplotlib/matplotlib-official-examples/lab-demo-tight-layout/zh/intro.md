# 简介

本实验旨在提供一份使用 Matplotlib（一个用于创建可视化的 Python 库）的分步指南。Matplotlib 是科学和工程领域中流行的数据可视化工具。本教程将引导你完成使用 Matplotlib 创建可视化的过程。
