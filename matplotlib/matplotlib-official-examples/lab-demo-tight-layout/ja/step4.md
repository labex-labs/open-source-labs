# 複数のプロットの作成

同じグラフ内に複数のプロットを作成することもできます。

```python
x = [1, 2, 3, 4, 5]
y1 = [2, 4, 6, 8, 10]
y2 = [1, 3, 5, 7, 9]

plt.subplot(1, 2, 1)
plt.plot(x, y1)
plt.title('Plot 1')

plt.subplot(1, 2, 2)
plt.plot(x, y2)
plt.title('Plot 2')

plt.show()
```

ここでは、`subplot`関数を使用して、同じグラフ内に 2 つのプロットを横並びに作成しています。`subplot`には 3 つの引数を渡します。行数、列数、およびプロット番号です。その後、各サブプロットにプロットを作成します。
