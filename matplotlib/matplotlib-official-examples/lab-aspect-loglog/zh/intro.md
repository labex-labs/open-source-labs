# 简介

在本教程中，我们将学习如何使用 Python 中的 Matplotlib 创建对数 - 对数图。对数 - 对数图是一种图表类型，其中 x 轴和 y 轴均采用对数刻度。这使我们能够以紧凑且信息丰富的方式可视化跨越多个数量级的数据。
