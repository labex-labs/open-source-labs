# Introducción

Este tutorial lo guiará a través del uso básico de la función `errorbar()` en Matplotlib. La función `errorbar()` se utiliza para trazar barras de error en un gráfico. Las barras de error indican la variabilidad o incertidumbre de un punto de datos en un gráfico. La función se puede utilizar para trazar barras de error tanto en la dirección x como en la dirección y.
