# 소개

이 랩은 Python 의 Matplotlib 라이브러리를 사용하여 비 시뮬레이션을 만드는 방법에 대한 단계별 튜토리얼입니다. 시뮬레이션은 표면에 떨어지는 빗방울을 시뮬레이션하기 위해 50 개의 산점 (scatter point) 의 크기와 불투명도를 애니메이션화합니다.
