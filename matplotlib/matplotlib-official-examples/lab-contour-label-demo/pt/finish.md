# Resumo

Neste tutorial, aprendemos como criar rótulos de contorno em Matplotlib. Cobrimos algumas técnicas avançadas para criar rótulos de contorno personalizados, incluindo formatadores de nível personalizados, rotulando contornos com strings arbitrárias e usando um _formatter_ (formatador) para formatar os rótulos de contorno. Essas técnicas podem ser úteis para criar visualizações que são informativas e esteticamente agradáveis.
