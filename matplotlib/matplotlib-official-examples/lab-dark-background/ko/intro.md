# 소개

Matplotlib 는 Python 에서 정적, 애니메이션, 대화형 시각화를 생성하는 데 사용되는 데이터 시각화 라이브러리입니다. 이 랩에서는 Matplotlib 에서 'dark_background' 스타일 시트를 사용하여 어두운 배경의 플롯을 만드는 방법을 배웁니다. 어두운 배경 스타일 시트는 어두운 환경에서 눈에 편안한 시각화를 표시하는 데 특히 유용합니다.
