# Введение

В этом руководстве показано, как нарисовать кривую с ошибочной полосой с использованием Python Matplotlib. Ошибочная полоса используется для показа неопределенности кривой. В этом примере предполагается, что ошибка может быть задана в виде скалярного значения _err_, которое описывает неопределенность перпендикулярно кривой в каждой точке. Мы визуализируем эту ошибку в виде окрашенной полосы вокруг пути с использованием `.PathPatch`. Патч создается из двух сегментов пути _(xp, yp)_ и _(xn, yn)_, которые сдвинуты на +/- _err_ перпендикулярно кривой _(x, y)_.
