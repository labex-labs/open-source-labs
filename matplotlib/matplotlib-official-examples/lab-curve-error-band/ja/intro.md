# はじめに

このチュートリアルでは、Python の Matplotlib を使って誤差帯付きの曲線を描画する方法を説明します。誤差帯は、曲線の不確定性を示すために使用されます。この例では、誤差をスカラー値 _err_ として与えることができ、これは曲線に垂直な各点の不確定性を表します。この誤差を、`.PathPatch` を使って曲線の周りに色付きの帯として可視化します。パッチは、曲線 _(x, y)_ に垂直に +/- _err_ だけシフトされた 2 つのパスセグメント _(xp, yp)_ と _(xn, yn)_ から作成されます。
