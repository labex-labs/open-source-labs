# 소개

이 튜토리얼에서는 Python 의 Matplotlib 을 사용하여 극좌표계에 산점도를 만드는 방법을 배웁니다. 극좌표 플롯은 극좌표로 표시되는 데이터의 그래픽 표현입니다. 시간이나 방향에 따라 측정된 데이터와 같이 데이터가 순환적이거나 원형인 경우 유용합니다.
