# Создание файла Glade 3

Далее мы создадим файл Glade 3 для создания пользовательского интерфейса для нашего приложения. Откройте Glade 3 и создайте новый проект. Добавьте виджет `ScrolledWindow` и виджет `Window`. Назовите окно `window1`, а прокручиваемое окно - `scrolledwindow1`. Сохраните файл под именем `mpl_with_glade3.glade`.
