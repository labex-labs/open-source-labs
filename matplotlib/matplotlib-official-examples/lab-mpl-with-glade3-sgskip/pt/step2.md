# Criando um Arquivo Glade 3

Em seguida, criaremos um arquivo Glade 3 para criar a interface do usuário para nossa aplicação. Abra o Glade 3 e crie um novo projeto. Adicione um widget `ScrolledWindow` e um widget `Window`. Nomeie a janela como `window1` e a janela com rolagem como `scrolledwindow1`. Salve o arquivo como `mpl_with_glade3.glade`.
