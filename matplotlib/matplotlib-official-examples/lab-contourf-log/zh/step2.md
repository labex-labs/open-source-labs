# 生成数据集

我们将使用`numpy`生成一个包含正值和负值的数据集：

```python
N = 100
x = np.linspace(-3.0, 3.0, N)
y = np.linspace(-2.0, 2.0, N)

X, Y = np.meshgrid(x, y)

# 一个带有尖峰的低隆起。
# 需要将 z/颜色轴设置为对数刻度，这样我们就能同时看到隆起和尖峰。
# 线性刻度只会显示尖峰。
Z1 = np.exp(-X**2 - Y**2)
Z2 = np.exp(-(X * 10)**2 - (Y * 10)**2)
z = Z1 + 50 * Z2

# 在左下角放入一些负值，以便在对数运算时产生问题：
z[:5, :5] = -1

# 以下内容并非绝对必要，但它将消除
# 一个警告。注释掉它以查看警告。
z = ma.masked_where(z <= 0, z)
```
