# Resumen

En este laboratorio, aprendiste cómo posicionar títulos de gráficos en Matplotlib. Puedes centrar el título utilizando la función `title()`, alinearlo a la izquierda o a la derecha utilizando el parámetro `loc`, o posicionarlo en la parte superior utilizando la función `set_xlabel()`. También puedes especificar manualmente la posición vertical utilizando el parámetro `y` o ajustarla utilizando los parámetros `titley` y `titlepad` de `rcParams`.
