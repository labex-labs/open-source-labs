# Zusammenfassung

In diesem Lab haben Sie gelernt, wie Sie Diagrammtitel in Matplotlib positionieren können. Sie können den Titel mithilfe der Funktion `title()` zentrieren, ihn mit dem Parameter `loc` links oder rechts ausrichten oder ihn mit der Funktion `set_xlabel()` oben platzieren. Sie können auch die vertikale Position manuell mit dem Parameter `y` angeben oder sie mit den Parametern `titley` und `titlepad` von `rcParams` anpassen.
