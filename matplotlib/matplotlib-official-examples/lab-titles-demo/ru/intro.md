# Введение

В этом практическом занятии (лабораторной работе) вы научитесь позиционировать заголовки графиков в Matplotlib. Matplotlib может отображать заголовки графиков по центру, выровненными по левому краю системы координат и выровненными по правому краю системы координат. В этом практическом занятии показано, как использовать Matplotlib для позиционирования заголовков графиков различными способами.
