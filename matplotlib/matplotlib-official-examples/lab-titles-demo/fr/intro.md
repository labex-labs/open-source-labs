# Introduction

Dans ce laboratoire, vous apprendrez à positionner les titres des graphiques dans Matplotlib. Matplotlib peut afficher les titres de graphiques centrés, alignés sur le côté gauche d'un ensemble d'axes et alignés sur le côté droit d'un ensemble d'axes. Ce laboratoire vous montrera comment utiliser Matplotlib pour positionner les titres de graphiques de différentes manières.
