# Resumo

Este laboratório demonstrou como sobrepor métodos básicos para que um artista possa conter outro artista em Python Matplotlib. Criamos uma classe personalizada chamada `MyLine`, que herdou de `lines.Line2D` e continha uma instância `mtext.Text` para rotulá-la. Em seguida, adicionamos a linha a um objeto de eixo e exibimos o gráfico.
