# Résumé

Ce laboratoire a montré comment surcharger des méthodes de base pour que l'artiste puisse contenir un autre artiste dans Matplotlib de Python. Nous avons créé une classe personnalisée appelée `MyLine` qui héritait de `lines.Line2D` et contenait une instance `mtext.Text` pour la légender. Nous avons ensuite ajouté la ligne à un objet d'axe et affiché la figure.
