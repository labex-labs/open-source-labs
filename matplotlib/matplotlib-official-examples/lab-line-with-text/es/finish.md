# Resumen

Esta práctica demostró cómo anular métodos básicos para que un artista pueda contener a otro artista en Python Matplotlib. Creamos una clase personalizada llamada `MyLine` que heredó de `lines.Line2D` y contenía una instancia de `mtext.Text` para etiquetarla. Luego agregamos la línea a un objeto de eje y mostramos la gráfica.
