# はじめに

この実験では、Python の Matplotlib ライブラリを使って、三角グリッドからのデータを四角グリッドに補間する方法を学びます。まず三角分割を作成し、その後線形および 3 次の方法を使ってデータを補間します。最後に結果をプロットします。

## VM のヒント
