# Resumo

Neste laboratório, aprendemos como criar um editor de polígonos usando o tratamento de eventos do Matplotlib. Criamos uma classe para lidar com todas as interações com o polígono e usamos a classe `Polygon` para criar o próprio polígono. Em seguida, criamos o gráfico e adicionamos o polígono a ele. Com este conhecimento, você pode criar suas próprias aplicações interativas usando o Matplotlib.
