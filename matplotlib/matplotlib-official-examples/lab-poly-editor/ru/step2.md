# Создаем функцию для вычисления расстояния

Нам нужно создать функцию, которая вычисляет расстояние между точкой и отрезком. Эта функция будет использоваться позже для определения, следует ли добавить новую вершину в многоугольник.

```python
def dist_point_to_segment(p, s0, s1):
    """
    Получить расстояние от точки *p* до отрезка (*s0*, *s1*), где
    *p*, *s0*, *s1* - массивы ``[x, y]``.
    """
    s01 = s1 - s0
    s0p = p - s0
    if (s01 == 0).all():
        return np.hypot(*s0p)
    # Проецируем на отрезок, не выходя за его концы.
    p1 = s0 + np.clip((s0p @ s01) / (s01 @ s01), 0, 1) * s01
    return np.hypot(*(p - p1))
```
