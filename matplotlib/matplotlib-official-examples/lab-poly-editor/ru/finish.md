# Резюме

В этом практическом занятии мы научились создавать редактор многоугольников с использованием обработки событий Matplotlib. Мы создали класс для обработки всех взаимодействий с многоугольником и использовали класс `Polygon` для создания самого многоугольника. Затем мы создали график и добавили в него многоугольник. С этими знаниями вы можете создавать собственные интерактивные приложения с использованием Matplotlib.
