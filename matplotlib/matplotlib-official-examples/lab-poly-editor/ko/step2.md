# 거리 계산 함수 생성

점과 선분 사이의 거리를 계산하는 함수를 생성해야 합니다. 이 함수는 나중에 새로운 정점을 다각형에 추가해야 하는지 여부를 결정하는 데 사용됩니다.

```python
def dist_point_to_segment(p, s0, s1):
    """
    점 *p*에서 세그먼트 (*s0*, *s1*) 까지의 거리를 구합니다. 여기서
    *p*, *s0*, *s1*은 ``[x, y]`` 배열입니다.
    """
    s01 = s1 - s0
    s0p = p - s0
    if (s01 == 0).all():
        return np.hypot(*s0p)
    # 세그먼트 끝을 지나지 않고 세그먼트에 투영합니다.
    p1 = s0 + np.clip((s0p @ s01) / (s01 @ s01), 0, 1) * s01
    return np.hypot(*(p - p1))
```
