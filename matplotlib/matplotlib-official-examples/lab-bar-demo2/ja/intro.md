# はじめに

このチュートリアルでは、Python の Matplotlib ライブラリを使って棒グラフを作成する手順を段階的に案内します。このチュートリアルの例では、センチメートルとインチの既定の単位を使う方法、さまざまなキーワードを使って x と y の単位を設定する方法、およびスカラーまたは単位を使って x の制限を設定する方法を示しています。
