# 소개

이 랩에서는 Python 의 Matplotlib 을 사용하여 quiver plot 을 만드는 방법을 안내합니다. Quiver plot 은 벡터 필드를 화살표로 표시합니다. 유체 흐름, 전기 및 자기장, 그리고 다른 유형의 벡터 필드를 시각화하는 데 유용합니다.
