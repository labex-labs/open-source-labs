# 소개

이 랩에서는 Matplotlib 에서 점선 스타일을 사용자 정의하는 방법을 배웁니다. `.Line2D.set_dashes()`를 사용하여 대시 시퀀스를 수정하고, `property_cycle`을 사용하여 대시 스타일을 구성하며, `~.Line2D.set_dash_capstyle()`, `~.Line2D.set_dash_joinstyle()`, 그리고 `~.Line2D.set_gapcolor()`와 같은 관련 메서드를 사용하여 대시의 다른 속성을 설정하는 방법을 다룹니다.
