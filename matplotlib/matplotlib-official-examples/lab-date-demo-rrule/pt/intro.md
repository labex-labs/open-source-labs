# Introdução

Neste laboratório, você aprenderá como usar regras de recorrência no Matplotlib para posicionar marcas de data. O `iCalendar RFC`\_ especifica regras de recorrência (rrules), que definem sequências de datas. Você pode usar rrules no Matplotlib para posicionar marcas de data. Este exemplo define marcas de data personalizadas em cada 5ª Páscoa.
