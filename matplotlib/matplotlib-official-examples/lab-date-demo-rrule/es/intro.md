# Introducción

En este laboratorio, aprenderá a usar reglas de recurrencia en Matplotlib para colocar marcas de fecha. El `iCalender RFC`\_ especifica reglas de recurrencia (rrules), que definen secuencias de fechas. Puede usar rrules en Matplotlib para colocar marcas de fecha. Este ejemplo establece marcas de fecha personalizadas en cada quinto domingo de Pascua.
