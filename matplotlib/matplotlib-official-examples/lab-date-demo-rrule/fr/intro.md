# Introduction

Dans ce laboratoire, vous allez apprendre à utiliser les règles de récurrence dans Matplotlib pour placer les repères de dates. Le `iCalender RFC`\_ spécifie les règles de récurrence (rrules), qui définissent des séquences de dates. Vous pouvez utiliser les rrules dans Matplotlib pour placer les repères de dates. Cet exemple définit des repères de dates personnalisés pour chaque cinquième Pâques.
