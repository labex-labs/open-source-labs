# はじめに

Matplotlib は、Python プログラミング言語とその数値数学拡張である NumPy 用のグラフ描画ライブラリです。Tkinter、wxPython、Qt、または GTK のような汎用 GUI ツールキットを使用して、グラフをアプリケーションに埋め込むためのオブジェクト指向 API を提供します。この実験では、Matplotlib を使ってさまざまな種類の画像を描画する方法を学びます。
