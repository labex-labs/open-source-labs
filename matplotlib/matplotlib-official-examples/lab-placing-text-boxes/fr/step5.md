# Création des propriétés de la boîte de texte

Nous allons créer un dictionnaire contenant les propriétés de la boîte de texte. Dans cet exemple, nous utiliserons une boîte arrondie avec une couleur de fond blé (wheat) et une opacité (alpha) de 0,5.

```python
props = dict(boxstyle='round', facecolor='wheat', alpha=0.5)
```
