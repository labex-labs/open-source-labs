# Резюме

В этом практическом занятии (lab) мы научились добавлять текстовые поля на график в Matplotlib с использованием Python. Мы использовали функцию `matplotlib.pyplot.text()` для добавления текстового поля и указали расположение поля в координатах осей. Мы также использовали свойство `bbox` для добавления закругленной рамки вокруг текста и установили свойства рамки с помощью словаря. Этот метод можно использовать для добавления аннотаций к графикам и предоставления дополнительной информации аудитории.
