# Introducción

En la visualización de datos, a menudo es necesario agregar anotaciones a las gráficas para proporcionar información adicional al público. Una forma de hacer esto es agregando cuadros de texto a una gráfica. En Matplotlib, es posible colocar cuadros de texto en coordenadas de los ejes, de modo que el texto no se mueva con cambios en los límites de x o y. También se puede usar la propiedad `bbox` del texto para rodear el texto con una instancia de `~matplotlib.patches.Patch`.

En este laboratorio, aprenderemos cómo colocar cuadros de texto en Matplotlib utilizando Python.
