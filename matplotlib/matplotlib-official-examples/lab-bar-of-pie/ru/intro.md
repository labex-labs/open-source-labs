# Введение

В этом уроке вы научитесь создавать график "Бар из пирога" с использованием библиотеки Matplotlib для Python. График "Бар из пирога" - это комбинация круговой диаграммы и накопленной столбчатой диаграммы, где первая секция пирога раскрывается в столбчатую диаграмму с дальней разбивкой ее характеристик. Этот график полезен, когда вы хотите показать распределение целого набора данных, выделяя при этом конкретные категории.
