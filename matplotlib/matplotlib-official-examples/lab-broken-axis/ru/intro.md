# Введение

В визуализации данных бывают случаи, когда нам приходится иметь дело с выбросами, которые затрудняют просмотр деталей большинства данных. В таких случаях можно использовать разрывную ось для увеличения масштаба большинства данных, сохраняя при этом отображение выбросов. В этом руководстве мы научимся создавать графики с разрывной осью с использованием Matplotlib в Python.
