# Установка пределов по оси Y

Мы ограничим ось y первого подграфика так, чтобы отображались только выбросы, а второго подграфика - чтобы отображалось большинство данных. Мы будем использовать `ax1.set_ylim` и `ax2.set_ylim` для установки пределов по оси y.

```python
ax1.set_ylim(.78, 1.)  # только выбросы
ax2.set_ylim(0,.22)  # большинство данных
```
