# Создание наклонных линий разрыва

Наконец, мы создадим наклонные линии разрыва. Мы создадим объекты линий в координатах осей и используем `ax1.transAxes` и `ax2.transAxes` для преобразования их в координаты каждого подграфика. Мы будем использовать `ax1.plot` и `ax2.plot` для построения линий. Также мы будем использовать `marker` для указания стиля маркера, `markersize` для установки размера маркеров, `linestyle` для установки стиля линии, `color` для установки цвета линии, `mec` для установки цвета края маркера и `mew` для установки ширины края маркера.

```python
d =.5
kwargs = dict(marker=[(-1, -d), (1, d)], markersize=12,
              linestyle="none", color='k', mec='k', mew=1, clip_on=False)
ax1.plot([0, 1], [0, 0], transform=ax1.transAxes, **kwargs)
ax2.plot([0, 1], [1, 1], transform=ax2.transAxes, **kwargs)
```
