# Einführung

In diesem Tutorial lernen wir, wie man mithilfe von Matplotlib in Python einen zusammengesetzten Pfad erstellt. Ein zusammengesetzter Pfad ist eine Sammlung einfacher Pfade, die verwendet werden können, um komplexe Formen zu erstellen. Wir werden einen zusammengesetzten Pfad erstellen, indem wir zwei einfache Polygone, ein Rechteck und ein Dreieck kombinieren.
