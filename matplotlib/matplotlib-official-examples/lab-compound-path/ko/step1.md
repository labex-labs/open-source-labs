# 필요한 라이브러리 가져오기

필요한 라이브러리를 가져오는 것으로 시작합니다. 플롯을 생성하기 위해 `matplotlib.pyplot`, 패치를 생성하기 위해 `matplotlib.patches`, 경로를 생성하기 위해 `matplotlib.path.Path`, 그리고 배열을 생성하기 위해 `numpy`가 필요합니다.

```python
import matplotlib.pyplot as plt
from matplotlib.patches import PathPatch
from matplotlib.path import Path
import numpy as np
```
