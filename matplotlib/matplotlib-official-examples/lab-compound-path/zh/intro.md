# 简介

在本教程中，我们将学习如何使用 Python 中的 Matplotlib 创建复合路径。复合路径是一组简单路径的集合，可用于创建复杂形状。我们将通过组合两个简单多边形（一个矩形和一个三角形）来创建一个复合路径。
