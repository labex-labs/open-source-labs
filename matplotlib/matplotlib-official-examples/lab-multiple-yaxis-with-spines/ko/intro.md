# 소개

데이터 시각화에서, 서로 다른 측정 단위를 가진 여러 변수를 동일한 그래프에 플롯해야 하는 경우가 많습니다. 이를 달성하는 일반적인 방법 중 하나는 여러 y 축을 사용하는 것입니다. 각 y 축은 서로 다른 변수에 해당합니다. 이 랩에서는 Matplotlib 를 사용하여 여러 y 축이 있는 그래프를 만드는 방법을 배웁니다.
