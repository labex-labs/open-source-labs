# はじめに

この実験では、Python の Matplotlib ライブラリを使って SkewT-logP 図を作成する方法を学びます。SkewT-logP 図は、気象学で温度の垂直プロファイルを表示するために一般的に使用されます。非直交な X 軸と Y 軸が関係するため、複雑なプロットです。このプロットを作成するには、Matplotlib の変換とカスタム射影 API を使用します。
