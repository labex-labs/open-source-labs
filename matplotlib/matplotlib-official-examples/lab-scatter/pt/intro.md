# Introdução

Neste laboratório, aprenderemos como criar um gráfico de dispersão simples usando a biblioteca Matplotlib do Python. Um gráfico de dispersão (scatter plot) é um tipo de gráfico que exibe valores para duas variáveis como uma coleção de pontos. Cada ponto representa os valores das duas variáveis, e a posição do ponto representa os valores das duas variáveis. Gráficos de dispersão são úteis para identificar relações entre variáveis e para identificar _outliers_ (valores discrepantes).
