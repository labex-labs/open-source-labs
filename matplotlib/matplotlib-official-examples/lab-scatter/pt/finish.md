# Resumo

Neste laboratório, aprendemos como criar um gráfico de dispersão simples usando a biblioteca Matplotlib do Python. Geramos dados aleatórios para o gráfico usando a biblioteca NumPy, definimos o tamanho e a cor dos pontos e criamos o gráfico usando a função `scatter` da biblioteca Matplotlib. Gráficos de dispersão são úteis para identificar relações entre variáveis e para identificar _outliers_ (valores discrepantes).
