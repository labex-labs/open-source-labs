# 소개

이 랩에서는 Matplotlib 라이브러리를 사용하여 mathtext 와 usetex 에 대해 계산된 텍스트 기준선을 비교합니다. mathtext 와 usetex 를 각각 사용하는 두 개의 서브플롯을 포함하는 플롯을 생성합니다. 각 서브플롯은 서로 다른 스타일과 위치를 가진 네 개의 테스트 문자열을 표시합니다.
