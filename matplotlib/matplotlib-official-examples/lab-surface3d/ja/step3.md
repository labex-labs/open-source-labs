# データの作成

```python
X = np.arange(-5, 5, 0.25)
Y = np.arange(-5, 5, 0.25)
X, Y = np.meshgrid(X, Y)
R = np.sqrt(X**2 + Y**2)
Z = np.sin(R)
```

ここでは、グラフ用のデータを作成します。`X` と `Y` の値を、-5 から 5 まで 0.25 の間隔で等間隔に設定された配列として作成します。次に、`np.meshgrid()` を使って `X` と `Y` の値のメッシュグリッドを作成します。このメッシュグリッドを使って、原点からの距離である `R` の値を計算します。そして、`R` の `sin()` 関数を使って `Z` の値を計算します。
