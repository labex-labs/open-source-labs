# 简介

本实验是一个关于如何使用 Python 中的 Matplotlib 绘制 3D 曲面的分步教程。3D 曲面使用 coolwarm 颜色映射进行着色，并通过使用“antialiased=False”使其不透明。本教程还演示了如何使用 `.LinearLocator` 和自定义格式设置 z 轴刻度标签。
