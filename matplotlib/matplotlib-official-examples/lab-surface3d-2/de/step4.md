# Einstellung des Seitenverhältnisses

Um sicherzustellen, dass der Plot ein gleichmäßiges Seitenverhältnis hat, können wir die `set_aspect()`-Funktion verwenden. Wir übergeben den String-Wert `'equal'`, um das Seitenverhältnis auf 1:1 einzustellen.

```python
# Set an equal aspect ratio
ax.set_aspect('equal')
```
