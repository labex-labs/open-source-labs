# 소개

이 튜토리얼은 Python 의 Matplotlib 라이브러리에서 단일 colormap, norm, 그리고 colorbar 를 사용하여 이미지 세트를 만드는 과정을 안내합니다. 데이터를 생성하고, 색상 스케일을 설정하며, 다른 이미지의 norm 변화에 대응하도록 이미지를 업데이트하는 방법을 배우게 됩니다.
