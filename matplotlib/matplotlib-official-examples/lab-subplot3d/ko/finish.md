# 요약

이 랩에서는 Matplotlib 을 사용하여 서브플롯으로 3D plot 을 생성하는 방법을 배웠습니다. 두 개의 서브플롯, 3D surface plot 및 3D wireframe plot 이 있는 figure 를 생성했습니다. NumPy 를 사용하여 3D surface plot 에 대한 데이터를 생성하고, `mpl_toolkits.mplot3d.axes3d`의 `get_test_data` 함수를 사용하여 3D wireframe plot 에 대한 데이터를 생성했습니다. 또한 3D surface plot 에 color bar 를 추가하고 z-축의 limits 를 설정했습니다.
