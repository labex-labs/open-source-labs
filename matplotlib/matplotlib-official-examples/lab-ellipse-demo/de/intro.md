# Einführung

Dieses Lab demonstriert, wie man mit Python Matplotlib Ellipsen zeichnet. Das Lab umfasst zwei Beispiele:

- Zeichnen einzelner Ellipsen
- Zeichnen von Ellipsen mit verschiedenen Winkeln
  > Sie können die Datei `ellipse-demo.ipynb` in der WebIDE öffnen, um mit den Übungen zu beginnen. Erfahren Sie, wie Sie [Jupyter Notebooks in VS Code](https://code.visualstudio.com/docs/datascience/jupyter-notebooks) verwenden.
  > ![Beispiel für das Zeichnen von Ellipsen mit Matplotlib](https://file.labex.io/upload/u/1991/fzLMg1oHuQrI.png)
  > Labby kann die Antworten nicht automatisch überprüfen, da es nicht auf das Notebook zugreifen kann.
