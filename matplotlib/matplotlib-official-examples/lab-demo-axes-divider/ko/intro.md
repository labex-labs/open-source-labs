# 소개

Matplotlib 는 Python 에서 정적, 애니메이션, 대화형 시각화를 생성할 수 있게 해주는 Python 라이브러리입니다. 과학 컴퓨팅, 데이터 분석, 머신 러닝 등 다양한 분야에서 널리 사용됩니다. 이 랩에서는 Matplotlib 를 사용하여 이미지를 플롯하는 방법과 축 및 컬러바의 위치를 조작하는 방법을 배우게 됩니다.
