# Introdução

Matplotlib é uma biblioteca Python que permite a criação de visualizações estáticas, animadas e interativas em Python. É amplamente utilizada em computação científica, análise de dados, _machine learning_ e muito mais. Neste laboratório, você aprenderá como plotar imagens usando Matplotlib e como manipular a localização de eixos e barras de cores.
