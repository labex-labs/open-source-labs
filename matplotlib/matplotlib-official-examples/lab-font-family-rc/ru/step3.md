# Выбор стандартного моноширинного шрифта

Стандартный моноширинный шрифт в Matplotlib определяется операционной системой. Мы можем выбрать использовать стандартный моноширинный шрифт, установив параметр `font.family` в `'monospace'`. Для этого мы можем использовать следующий код:

```python
import matplotlib.pyplot as plt

plt.rcParams["font.family"] = "monospace"
```
