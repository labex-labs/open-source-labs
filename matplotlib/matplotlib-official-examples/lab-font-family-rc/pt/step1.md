# Escolha da Fonte Sans-Serif Padrão

A família de fontes padrão no Matplotlib é sans-serif. Podemos optar por usar a família de fontes padrão definindo o parâmetro `font.family` como `'sans-serif'`. Para fazer isso, podemos usar o seguinte código:

```python
import matplotlib.pyplot as plt

plt.rcParams["font.family"] = "sans-serif"
```
