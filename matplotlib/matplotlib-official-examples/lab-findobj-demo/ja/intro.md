# はじめに

このチュートリアルでは、Python プログラミング言語用のグラフ作成ライブラリである Python Matplotlib の基本を学びます。このチュートリアルでは、Matplotlib を使ってさまざまな種類のグラフを作成してカスタマイズする方法を学びます。

## VM のヒント

VM の起動が完了したら、左上隅をクリックして **ノートブック** タブに切り替えて、Jupyter Notebook を使って練習します。

時々、Jupyter Notebook が読み込み終わるまで数秒待つ必要がある場合があります。Jupyter Notebook の制限により、操作の検証を自動化することはできません。

学習中に問題に遭遇した場合は、Labby にお問い合わせください。セッション後にフィードバックを提供してください。そうすれば、迅速に問題を解決します。
