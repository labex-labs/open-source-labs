# はじめに

このチュートリアルでは、Python プログラミング言語用のグラフ作成ライブラリである Python Matplotlib の基本を学びます。このチュートリアルでは、Matplotlib を使ってさまざまな種類のグラフを作成してカスタマイズする方法を学びます。
