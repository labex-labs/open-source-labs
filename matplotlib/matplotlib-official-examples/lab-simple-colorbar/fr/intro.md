# Introduction

Dans ce laboratoire, nous allons apprendre à ajouter une simple barre de couleur à un tracé Matplotlib. Une barre de couleur est un outil pratique pour visualiser la plage de valeurs représentées par les couleurs dans un tracé. Nous allons utiliser la fonction `imshow` pour créer un tracé 2D simple puis lui ajouter une barre de couleur.
