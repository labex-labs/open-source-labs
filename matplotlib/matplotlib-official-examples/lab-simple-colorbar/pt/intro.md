# Introdução

Neste laboratório, aprenderemos como adicionar uma barra de cores (colorbar) simples a um gráfico Matplotlib. Uma barra de cores é uma ferramenta útil para visualizar a gama de valores representados por cores em um gráfico. Usaremos a função `imshow` para criar um gráfico 2D simples e, em seguida, adicionaremos uma barra de cores a ele.
