# Resumo

Neste laboratório, aprendemos como adicionar uma barra de cores simples a um gráfico Matplotlib. Usamos a função `imshow` para criar um gráfico 2D e, em seguida, adicionamos uma barra de cores usando a função `colorbar`. Também personalizamos a aparência da barra de cores e adicionamos um rótulo a ela. As barras de cores são uma ferramenta útil para visualizar a gama de valores representados pelas cores em um gráfico.
