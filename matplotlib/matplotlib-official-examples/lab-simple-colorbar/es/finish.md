# Resumen

En este laboratorio, aprendimos cómo agregar una barra de colores simple a una gráfica de Matplotlib. Usamos la función `imshow` para crear una gráfica 2D y luego agregamos una barra de colores usando la función `colorbar`. También personalizamos la apariencia de la barra de colores y le agregamos una etiqueta. Las barras de colores son una herramienta útil para visualizar el rango de valores representados por los colores en una gráfica.
