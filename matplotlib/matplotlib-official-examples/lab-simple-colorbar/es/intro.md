# Introducción

En este laboratorio, aprenderemos cómo agregar una barra de colores simple a una gráfica de Matplotlib. Una barra de colores es una herramienta útil para visualizar el rango de valores representados por los colores en una gráfica. Usaremos la función `imshow` para crear una gráfica 2D simple y luego agregaremos una barra de colores a ella.
