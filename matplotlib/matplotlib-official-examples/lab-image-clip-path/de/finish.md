# Zusammenfassung

In diesem Lab haben wir gelernt, wie man ein Bild mit Patches zuschneidet, indem man die Matplotlib-Bibliothek von Python verwendet. Wir haben ein Beispielbild geladen, einen kreisförmigen Patch erstellt und das Bild auf die Form des Patches zugeschnitten. Das Zuschneiden von Bildern mit Patches kann nützlich sein, um bestimmte Bereiche eines Bildes hervorzuheben oder das Bild auf eine bestimmte Form zu zuschneiden.
