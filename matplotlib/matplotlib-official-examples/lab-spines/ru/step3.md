# Создаем подграфики

Мы создадим три подграфика, чтобы продемонстрировать разные настройки "хребтов" (spines). Мы будем использовать ограниченный макет (constrained layout), чтобы убедиться, что подписи не накладываются на оси.

```python
fig, (ax0, ax1, ax2) = plt.subplots(nrows=3, layout='constrained')
```
