# Настройка "хребтов" (spines) для всех четырёх сторон

В первом подграфике мы будем отображать "хребты" по всем четырем сторонам графика. Мы можем получить доступ к "хребтам" каждого подграфика с использованием контейнера `ax.spines`. Затем мы можем настроить "хребты" с использованием различных методов.

```python
ax0.plot(x, y)
ax0.set_title('Normal Spines')
```
