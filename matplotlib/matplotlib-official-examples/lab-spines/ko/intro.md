# 소개

이 랩에서는 Matplotlib 에서 스파인 (spine) 을 사용자 정의하는 방법을 배우게 됩니다. 스파인은 축 눈금 표시를 연결하고 데이터 영역의 경계를 나타내는 선입니다. 기본적으로 Matplotlib 는 플롯의 네 면 모두에 스파인을 표시합니다. 그러나 데이터를 더 잘 강조하기 위해 이러한 스파인을 사용자 정의할 수 있습니다.
