# Introdução

No Matplotlib, `patheffects` podem ser usados para alterar como os caminhos são desenhados. `TickedStroke` é um tipo de `patheffect` que desenha linhas com um estilo tracejado. Este tutorial irá guiá-lo através dos passos para criar TickedStroke no Matplotlib.
