# 简介

在本实验中，我们将学习如何使用 Matplotlib 通过矩形和 PolyCollections 来构建直方图。我们将使用 numpy 生成随机数据，然后使用 Matplotlib 将数据可视化为直方图。本实验假设你对 Python 和 Matplotlib 有基本的了解。
