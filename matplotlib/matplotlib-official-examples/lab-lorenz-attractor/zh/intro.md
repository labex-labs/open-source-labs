# 简介

本实验是一个循序渐进的教程，用于使用 mplot3d 在三维空间中绘制爱德华·洛伦兹 1963 年的《确定性非周期流》。我们将使用 Python 和 Matplotlib，它是 Python 编程语言的一个绘图库。
