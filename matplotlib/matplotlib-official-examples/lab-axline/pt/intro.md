# Introdução

Matplotlib é uma biblioteca Python utilizada para criar visualizações para análise de dados. Neste tutorial, aprenderemos como usar `axvline`, `axhline` e `axline` para desenhar linhas infinitas em Matplotlib.

## Dicas para a VM

Após a inicialização da VM, clique no canto superior esquerdo para mudar para a aba **Notebook** e acessar o [Jupyter Notebook](https://support.labex.io/en/labex-vm/jupyter) para praticar.
