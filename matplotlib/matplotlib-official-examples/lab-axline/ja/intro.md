# はじめに

Matplotlib は、データ分析用のビジュアライゼーションを作成するために使用される Python ライブラリです。このチュートリアルでは、Matplotlib で無限の線を描画するための`axvline`、`axhline`、および`axline`の使い方を学びます。

## VM のヒント
