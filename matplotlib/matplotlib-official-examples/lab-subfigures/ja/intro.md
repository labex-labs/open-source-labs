# はじめに

Matplotlib は、グラフ、チャート、図などのビジュアライゼーションを作成するために使用される Python ライブラリです。この実験では、Matplotlib でサブフィギュアを作成する方法を学びます。サブフィギュアは、同じ図の中に 2 つ以上のプロットを含む図です。サブフィギュアは、異なるデータセットを比較したり、同じデータの異なるビューを表示したりするのに役立ちます。複数のプロットを一緒に表示したい場合に便利です。
