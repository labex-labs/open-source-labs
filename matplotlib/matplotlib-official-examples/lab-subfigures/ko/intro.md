# 소개

Matplotlib 는 그래프, 차트, 그림과 같은 시각화를 생성하는 데 사용되는 Python 라이브러리입니다. 이 랩에서는 Matplotlib 에서 서브피겨 (subfigure) 를 만드는 방법을 배우게 됩니다. 서브피겨는 동일한 그림 내에 두 개 이상의 플롯을 포함하는 그림입니다. 서브피겨는 서로 다른 데이터 세트를 비교하거나 동일한 데이터의 다른 보기를 표시하는 데 도움이 될 수 있습니다. 여러 플롯을 함께 표시하려는 경우 유용합니다.
