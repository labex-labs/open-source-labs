# Resumo

Neste tutorial, aprendemos como usar o módulo RGBAxes do toolkit AxesGrid em Matplotlib para exibir canais RGB. Cobrimos as seguintes etapas:

1. Importar as bibliotecas necessárias
2. Definir uma função para obter os canais RGB
3. Definir uma função para criar um cubo RGB
4. Criar um gráfico RGBAxes
5. Criar um gráfico RGBAxes com canais separados
6. Exibir os gráficos.
