# Introdução

Neste laboratório, você aprenderá como criar uma imagem de contorno usando a biblioteca Matplotlib do Python. A imagem de contorno é uma técnica de visualização que representa uma superfície tridimensional usando um gráfico bidimensional. A imagem de contorno consiste em linhas de contorno, que são linhas que conectam pontos de igual valor na superfície, e contornos preenchidos, que são regiões entre as linhas de contorno.
