# Resumo

Neste laboratório, você aprendeu como criar uma imagem de contorno usando a biblioteca Matplotlib do Python. Você começou importando as bibliotecas necessárias, definindo os dados a serem plotados, criando a imagem de contorno e exibindo a imagem. A imagem de contorno é uma técnica de visualização útil que pode ser usada para representar uma superfície tridimensional usando um gráfico bidimensional.
