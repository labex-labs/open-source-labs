# Введение

В этом лабе вы научитесь создавать контурное изображение с использованием библиотеки Matplotlib для Python. Контурное изображение - это техника визуализации, которая представляет трехмерную поверхность с использованием двухмерной диаграммы. Контурное изображение состоит из контуров, которые представляют собой линии, соединяющие точки одинакового значения на поверхности, и заполненных контуров, которые представляют собой области между контурами.
