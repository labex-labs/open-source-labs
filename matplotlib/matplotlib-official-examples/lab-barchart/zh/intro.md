# 简介

在本实验中，我们将学习如何使用 Python Matplotlib 创建分组柱状图，以及如何用标签注释柱子。我们将使用帕尔默企鹅数据集（Palmer Penguins dataset）中的数据来创建一个按物种显示企鹅属性的图表。
