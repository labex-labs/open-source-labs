# Evaluar los datos para graficar

Ahora, evalúa algunos datos para graficar. En este ejemplo, graficaremos una función objetivo, `g1`, `g2` y `g3`.

```python
# Evaluar algunos elementos para graficar
obj = x1**2 + x2**2 - 2*x1 - 2*x2 + 2
g1 = -(3*x1 + x2 - 5.5)
g2 = -(x1 + 2*x2 - 4.5)
g3 = 0.8 + x1**-3 - x2
```
