# 简介

Matplotlib 是 Python 中一个强大的数据可视化库。它提供了各种可视化工具来创建令人惊叹的图形和图表。在本实验中，我们将了解 ConnectionPatch，它用于在不同坐标系和/或轴中定义的点之间绘制线条。
