# Introdução

Matplotlib é uma poderosa biblioteca de visualização de dados em Python. Ela oferece uma variedade de ferramentas de visualização para criar gráficos e diagramas impressionantes. Neste laboratório, aprenderemos sobre `ConnectionPatch`, que é usado para desenhar uma linha entre pontos definidos em diferentes sistemas de coordenadas e/ou eixos.
