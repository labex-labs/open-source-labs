# Простое форматирование

В этом шаге мы покажем, как использовать простой форматтер, передав строку или функцию в `~.Axis.set_major_formatter` или `~.Axis.set_minor_formatter`. Мы создадим два графика, один с использованием строкового форматтера, а другой - с использованием функционного форматтера.

```python
fig0, axs0 = plt.subplots(2, 1, figsize=(8, 2))
fig0.suptitle('Simple Formatting')

# Строку, использующую синтаксис функции форматирования строк, можно
# использовать напрямую в качестве форматтера. Переменная ``x`` - это значение
# деления, а переменная ``pos`` - позиция деления. Это автоматически
# создает StrMethodFormatter.
setup(axs0[0], title="'{x} км'")
axs0[0].xaxis.set_major_formatter('{x} км')

# Функцию также можно использовать напрямую в качестве форматтера. Функция
# должна принимать два аргумента: ``x`` для значения деления и ``pos`` для
# позиции деления, и должна возвращать строку. Это автоматически создает
# FuncFormatter.
setup(axs0[1], title="lambda x, pos: str(x-5)")
axs0[1].xaxis.set_major_formatter(lambda x, pos: str(x-5))

fig0.tight_layout()
```
