# Resumo

Neste tutorial, aprendemos como usar a interface pyplot em Matplotlib para definir e obter propriedades de objetos para visualização de dados. Usamos o método `setp` para definir as propriedades de um objeto e o método `getp` para obter as propriedades de um objeto. Também aprendemos sobre aliases (alias) para propriedades para reduzir as batidas de teclado. Matplotlib é uma biblioteca poderosa que permite criar uma ampla gama de visualizações em Python.
