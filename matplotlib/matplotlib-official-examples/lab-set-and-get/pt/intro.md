# Introdução

Matplotlib é uma biblioteca Python utilizada para criar visualizações estáticas, animadas e interativas em Python. Ela permite criar uma ampla gama de visualizações, como gráficos de linhas, gráficos de dispersão (scatter plots), gráficos de barras, histogramas e gráficos 3D. Neste tutorial, aprenderemos como usar a interface pyplot para definir e obter propriedades de objetos para visualizar dados.
