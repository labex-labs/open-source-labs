# 소개

Matplotlib 는 Python 에서 정적, 애니메이션, 대화형 시각화를 생성하는 데 사용되는 Python 라이브러리입니다. 선 차트, 산점도, 막대 차트, 히스토그램, 3D 플롯 등 다양한 시각화를 생성할 수 있습니다. 이 튜토리얼에서는 pyplot 인터페이스를 사용하여 데이터를 시각화하기 위한 객체 속성을 설정하고 가져오는 방법을 배웁니다.
