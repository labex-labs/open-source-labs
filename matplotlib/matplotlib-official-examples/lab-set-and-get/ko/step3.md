# 속성 설정

pyplot 인터페이스를 사용하면 데이터를 시각화하기 위한 객체 속성을 설정하고 가져올 수 있습니다. `setp` 메서드를 사용하여 객체의 속성을 설정할 수 있습니다. 예를 들어, 선의 선 스타일을 파선으로 설정하려면 다음 코드를 사용합니다.

```python
line, = plt.plot([1, 2, 3])
plt.setp(line, linestyle='--')
```

유효한 인수의 유형을 알고 싶다면, 값을 지정하지 않고 설정하려는 속성의 이름을 제공할 수 있습니다.

```python
plt.setp(line, 'linestyle')
```

그러면 다음과 같은 출력이 반환됩니다.

```
linestyle: {'-', '--', '-.', ':', '', (offset, on-off-seq), ...}
```

설정할 수 있는 모든 속성과 가능한 값을 보려면 다음 코드를 사용할 수 있습니다.

```python
plt.setp(line)
```

그러면 속성 및 가능한 값의 긴 목록이 반환됩니다.
