# Resumen

En este tutorial, aprendimos cómo usar la interfaz de pyplot en Matplotlib para establecer y obtener propiedades de objetos para visualizar datos. Usamos el método `setp` para establecer las propiedades de un objeto y el método `getp` para obtener las propiedades de un objeto. También aprendimos sobre los alias de las propiedades para reducir las pulsaciones de teclado. Matplotlib es una biblioteca poderosa que te permite crear una amplia variedad de visualizaciones en Python.
