# Introducción

Matplotlib es una biblioteca de Python que se utiliza para crear visualizaciones estáticas, animadas e interactivas en Python. Te permite crear una amplia variedad de visualizaciones, como diagramas de líneas, diagramas de dispersión, diagramas de barras, histogramas y diagramas 3D. En este tutorial, aprenderemos cómo usar la interfaz de pyplot para establecer y obtener propiedades de objetos para visualizar datos.
