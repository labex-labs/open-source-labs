# 소개

데이터 분석에서, 우리는 종종 두 개의 독립 변수에 의존하는 데이터를 히트맵 (heatmap) 이라고 하는 색상 코딩된 이미지 플롯으로 표시하고자 합니다. 이 랩에서는 Matplotlib 의 `imshow` 함수를 사용하여 주석이 달린 히트맵을 생성할 것입니다. 간단한 예시로 시작하여 범용 함수로 사용할 수 있도록 확장할 것입니다.
