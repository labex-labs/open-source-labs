# Einführung

Matplotlib ist eine leistungsstarke Bibliothek zur Erstellung von Visualisierungen in Python. Manchmal müssen Sie beim Erstellen von Diagrammen die Subplot-Parameter manuell anpassen. In diesem Lab erfahren Sie, wie Sie die Subplot-Parameter programmgesteuert anhand der Größe der Beschriftungen anpassen können.
