# Introdução

Matplotlib é uma biblioteca poderosa para criar visualizações em Python. Às vezes, ao criar gráficos, pode ser necessário ajustar os parâmetros dos subplots manualmente. Este laboratório mostrará como ajustar programaticamente os parâmetros dos subplots com base no tamanho dos rótulos.
