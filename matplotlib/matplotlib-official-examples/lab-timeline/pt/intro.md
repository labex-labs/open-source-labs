# Introdução

Neste laboratório, você aprenderá como criar uma linha do tempo simples usando as datas de lançamento do Matplotlib. Uma linha do tempo é uma representação gráfica de uma sequência de eventos em ordem cronológica. Linhas do tempo podem ser criadas com uma coleção de datas e texto. Neste exemplo, mostraremos como criar uma linha do tempo simples usando as datas dos lançamentos recentes do Matplotlib.
