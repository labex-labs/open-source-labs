# 소개

이 랩은 Python Matplotlib 을 사용하여 등고선 플롯을 만드는 방법에 대한 단계별 튜토리얼입니다. 등고선 플롯은 3 차원 데이터를 2 차원으로 시각화하는 데 유용합니다. 이 튜토리얼에서는 간단한 등고선 플로팅, 등고선에 대한 이미지와 등고선에 대한 컬러바, 그리고 레이블이 지정된 등고선을 설명합니다.
