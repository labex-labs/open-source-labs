# Resumo

Neste laboratório, aprendemos como exibir duas escalas no eixo y esquerdo e direito usando Python Matplotlib. Usamos uma função de closure (fechamento) para registrar como um callback (retorno de chamada) para atualizar o segundo eixo de acordo com o primeiro eixo. Isso é útil quando queremos plotar dois conjuntos de dados com unidades diferentes.
