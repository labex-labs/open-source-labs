# 소개

이 튜토리얼에서는 Python 의 Matplotlib 라이브러리를 사용하여 Sankey 다이어그램을 만드는 방법을 배웁니다. Sankey 다이어그램은 시스템을 통해 에너지 또는 물질의 흐름을 시각화하는 데 일반적으로 사용되는 흐름 다이어그램의 한 유형입니다. 여러 입력과 출력이 있는 복잡한 시스템을 분석하는 데 특히 유용합니다.
