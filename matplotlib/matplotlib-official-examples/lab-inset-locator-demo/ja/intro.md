# はじめに

Matplotlib は Python のデータ可視化ライブラリです。`.inset_locator`機能を使うと、軸の角にインセットを簡単に配置できます。この実験では、Matplotlib のプロットにインセットを作成するために`.inset_locator`機能をどのように使うかを調べます。
