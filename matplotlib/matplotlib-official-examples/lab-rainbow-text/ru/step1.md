# Создайте первый текстовый объект

Первым шагом является создание первого текстового объекта с использованием `~.Axes.text`. Этот текстовый объект будет точкой начала процесса конкатенации. Следующий код создает красный текстовый объект с словом "Matplotlib" в позиции (0,1, 0,5) на графике.

```python
text = ax.text(.1,.5, "Matplotlib", color="red")
```
