# Criar o Primeiro Objeto de Texto

O primeiro passo é criar o primeiro objeto de texto usando `~.Axes.text`. Este objeto de texto será o ponto de partida para o processo de concatenação. O código a seguir cria um objeto de texto vermelho com a palavra "Matplotlib" na posição (0.1, 0.5) no gráfico.

```python
text = ax.text(.1, .5, "Matplotlib", color="red")
```
