# Crear el primer objeto de texto

El primer paso es crear el primer objeto de texto utilizando `~.Axes.text`. Este objeto de texto será el punto de partida para el proceso de concatenación. El siguiente código crea un objeto de texto rojo con la palabra "Matplotlib" en la posición (0.1, 0.5) en la gráfica.

```python
text = ax.text(.1,.5, "Matplotlib", color="red")
```
