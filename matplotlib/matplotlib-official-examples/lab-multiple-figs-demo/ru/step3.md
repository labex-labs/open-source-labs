# Создать фигуру 1

Начнем с создания первой фигуры, которая будет содержать два подграфика. В верхнем подграфике мы построим первую синусоидальную волну, а в нижнем подграфике - волну с удвоенной амплитудой первой синусоидальной волны.

```python
plt.figure(1)

# Верхний подграфик
plt.subplot(211)
plt.plot(t, s1)

# Нижний подграфик
plt.subplot(212)
plt.plot(t, 2*s1)
```
