# 简介

在本实验中，你将学习如何在 Matplotlib 的 pyplot 中管理多个图形。Matplotlib 是 Python 中一个流行的数据可视化库，而 pyplot 是 Matplotlib 中的一个子库，它为创建、定制和组织绘图提供了一个简单的接口。你将学习如何创建多个图形并在它们之间切换，在每个图形中创建子图，以及对特定子图进行更改。
