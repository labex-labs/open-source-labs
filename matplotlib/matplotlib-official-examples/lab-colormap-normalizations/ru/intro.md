# Введение

В этом практическом занятии мы научимся использовать Matplotlib для отображения цветовых карты на данные нелинейным способом. Мы покажем, как использовать `norm` для создания логарифмических, степенных, симметричных логарифмических и пользовательских нормализаций. Также мы научимся использовать `BoundaryNorm` для задания границ для цветов.
