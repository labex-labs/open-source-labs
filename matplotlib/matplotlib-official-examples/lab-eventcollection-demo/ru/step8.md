# Задаем пределы и добавляем заголовок

Мы зададим пределы по осям x и y и добавим заголовок к графику с использованием функций `matplotlib.pyplot.xlim()`, `matplotlib.pyplot.ylim()` и `matplotlib.pyplot.title()`.

```python
# set the limits
ax.set_xlim([0, 1])
ax.set_ylim([0, 1])
ax.set_title('line plot with data points')
```
