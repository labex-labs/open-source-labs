# 필요한 라이브러리 가져오기

먼저, 필요한 라이브러리를 가져와야 합니다. 무작위 데이터를 생성하기 위해 `numpy`를 사용하고, 그래프를 생성하기 위해 `matplotlib.pyplot`을 사용하며, 데이터 포인트의 위치를 표시하기 위해 `matplotlib.collections`에서 `EventCollection`을 사용합니다.

```python
import matplotlib.pyplot as plt
import numpy as np
from matplotlib.collections import EventCollection
```
