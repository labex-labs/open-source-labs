# 简介

在本实验中，我们将学习如何使用 Python 中的 Matplotlib 创建带有数据点的折线图。我们将使用 Matplotlib 中的 `EventCollection` 类在每条曲线的相应轴上标记 x 和 y 数据点的位置。
