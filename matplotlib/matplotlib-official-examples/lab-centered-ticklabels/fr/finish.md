# Récapitulatif

Dans ce laboratoire, nous avons appris à centrer les étiquettes entre les graduations dans un graphique Matplotlib à l'aide de Python. Nous avons utilisé des données financières sur le prix d'actions de Google pour démontrer cette technique et avons suivi les étapes suivantes :

1. Chargé les données financières
2. Créé le graphique
3. Configuré les localiseurs et les formatteurs principaux et secondaires
4. Supprimé les étiquettes des graduations majeures et les graduations mineures
5. Ajusté l'alignement des étiquettes des graduations mineures au centre entre les graduations majeures
6. Affiché le graphique.
