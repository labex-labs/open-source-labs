# Introduction

Matplotlib est une bibliothèque de tracé pour le langage de programmation Python. Elle fournit une API orientée objet pour intégrer des graphiques dans des applications utilisant des kits de bibliothèques GUI de type général tels que Tkinter, wxPython, Qt ou GTK. Matplotlib permet aux développeurs de créer une large gamme de visualisations statiques, animées et interactives en Python.

Dans ce laboratoire, nous allons apprendre à créer des unités personnalisées dans Matplotlib et à tracer des données à l'aide de ces unités personnalisées.
