# Subplot-Abstände und -Ränder anpassen

Jetzt lassen wir uns die Subplot-Abstände und -Ränder mit `pyplot.subplots_adjust` anpassen. Wir können die Werte für den unteren, rechten, oberen und linken Rand als Bruchteil der Figurbreite und -höhe angeben. Führen Sie folgenden Code aus:

```python
plt.subplots_adjust(bottom=0.1, right=0.8, top=0.9)
```
