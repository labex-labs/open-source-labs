# Resumo

Neste laboratório, você aprendeu como gerar gráficos usando strings correspondentes a variáveis, utilizando o formato de dados `numpy.recarray` ou `pandas.DataFrame` com o argumento de palavra-chave `data` em Matplotlib. Você também aprendeu como interpretar um gráfico de dispersão (scatter plot) usando as variáveis `a`, `b`, `c` e `d`.
