# Генерируем график

В этом шаге мы сгенерируем точечный график, используя словарь `data` в качестве входных данных для функции `scatter()`. Мы будем использовать строки, соответствующие переменным `a`, `b`, `c` и `d`, для генерации графика.

```python
fig, ax = plt.subplots()
ax.scatter('a', 'b', c='c', s='d', data=data)
ax.set(xlabel='entry a', ylabel='entry b')
plt.show()
```
