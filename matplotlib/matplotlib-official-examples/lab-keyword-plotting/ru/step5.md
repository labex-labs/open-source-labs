# Интерпретируем график

В этом шаге мы интерпретируем точечный график, сгенерированный на шаге 4. График показывает связь между переменными `a` и `b`, при этом переменная `c` используется для определения цвета каждой точки, а переменная `d` - для определения размера каждой точки. На оси x представлено `entry a`, а на оси y - `entry b`.
