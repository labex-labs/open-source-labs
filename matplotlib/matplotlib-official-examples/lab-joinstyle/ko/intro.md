# 소개

이 랩은 Python 의 Matplotlib 에서 `JoinStyle` 모듈을 사용하는 방법에 대한 단계별 가이드를 제공합니다. `JoinStyle`은 Matplotlib 이 서로 다른 두 개의 선분 (line segment) 이 만나는 모서리를 그리는 방식을 제어합니다. 이 랩은 Matplotlib 을 처음 사용하는 사용자를 위해 설계되었습니다.
