# Самый простой штрихованный график с цветовой шкалой

В этом шаге мы создадим самый простой штрихованный график с цветовой шкалой. Мы будем использовать функцию `contourf` для создания заполненного контурного графика и задавать штрихи с использованием параметра `hatches`.

```python
fig1, ax1 = plt.subplots()
cs = ax1.contourf(x, y, z, hatches=['-', '/', '\\', '//'],
                  cmap='gray', extend='both', alpha=0.5)
fig1.colorbar(cs)
```
