# Resumo

Neste laboratório, aprendemos como criar gráficos de contorno preenchidos com padrões de hachura usando Matplotlib. Usamos as funções `contour` e `contourf` para criar os gráficos e especificamos as hachuras usando o parâmetro `hatches`. Também adicionamos uma barra de cores (colorbar) e uma legenda aos nossos gráficos.
