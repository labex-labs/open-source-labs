# Введение

В этом практическом занятии вы научитесь создавать многостраничный PDF-файл с использованием Python Matplotlib. PDF-файл будет содержать несколько страниц с различными графиками и метаданными. Также вы научитесь прикреплять аннотации к PDF-файлу.
