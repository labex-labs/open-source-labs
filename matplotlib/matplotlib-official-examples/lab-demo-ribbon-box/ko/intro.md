# 소개

Matplotlib 는 Python 을 위한 인기 있는 데이터 시각화 라이브러리입니다. 이 튜토리얼은 데이터를 시각화하는 독특한 방법인 리본 박스 차트 (ribbon box chart) 를 만드는 과정을 안내합니다. 리본 박스 차트는 각 막대가 "리본 모양"이고 그라데이션 색상 체계를 갖는 누적 막대 차트입니다.
