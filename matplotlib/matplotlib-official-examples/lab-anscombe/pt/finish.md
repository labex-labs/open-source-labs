# Resumo

Neste tutorial, você aprendeu como usar Matplotlib para criar visualizações em Python. Você aprendeu como criar uma figura com subplots, plotar pontos de dados, adicionar uma linha de regressão linear e adicionar uma caixa de texto com estatísticas sobre os dados. Com Matplotlib, você pode criar uma ampla variedade de visualizações para explorar e analisar seus dados.
