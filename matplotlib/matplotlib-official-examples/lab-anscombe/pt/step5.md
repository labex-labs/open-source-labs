# Interpretando os Resultados

A visualização resultante é um conjunto de quatro subplots, cada um mostrando um conjunto de dados diferente. Os limites de x e y são os mesmos para todos os subplots. Cada subplot contém os pontos de dados x e y, bem como uma linha de regressão linear. A caixa de texto no canto inferior direito de cada subplot mostra algumas estatísticas sobre os dados, incluindo a média, o desvio padrão e o coeficiente de correlação.
