# Introdução

Neste tutorial, aprenderemos como criar um histograma usando a biblioteca Matplotlib. Um histograma é uma representação gráfica da distribuição de um conjunto de dados. É uma estimativa da distribuição de probabilidade de uma variável contínua. Para criar um histograma, precisamos dividir toda a gama de valores em uma série de intervalos ou _bins_ (classes), e então contar quantos valores caem em cada intervalo.
