# 소개

이 랩은 Python Matplotlib 라이브러리를 사용하여 히스토그램을 만드는 방법에 대한 단계별 튜토리얼입니다. 히스토그램은 수치 데이터의 분포를 시각적으로 표현한 것입니다. 연속 변수의 확률 분포를 추정하는 데 사용됩니다.
