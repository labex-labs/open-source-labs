# Die Position der x-Achsenbeschriftung festlegen

Wir können die Position der x-Achsenbeschriftung mit der Methode `set_xlabel` festlegen. Wir können die Position auf `'left'` (links), `'center'` (mittig) oder `'right'` (rechts) setzen. In diesem Beispiel setzen wir die Position auf `'left'`.

```python
ax.set_xlabel('XLabel', loc='left')
```
