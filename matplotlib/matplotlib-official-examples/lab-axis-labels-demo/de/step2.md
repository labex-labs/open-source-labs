# Die Position der y-Achsenbeschriftung festlegen

Wir können die Position der y-Achsenbeschriftung mit der Methode `set_ylabel` festlegen. Wir können die Position auf `'top'` (oben), `'center'` (mittig) oder `'bottom'` (unten) setzen. In diesem Beispiel setzen wir die Position auf `'top'`.

```python
ax.set_ylabel('YLabel', loc='top')
```
