# 라이브러리 임포트

먼저, 필요한 라이브러리를 임포트해야 합니다. 플롯을 생성하기 위해 `matplotlib.pyplot`을 사용하고, 임의의 데이터를 생성하기 위해 `numpy`를 사용하며, 다양한 유형의 아티스트를 생성하기 위해 `matplotlib.collections`, `matplotlib.lines`, `matplotlib.patches`, 그리고 `matplotlib.text`를 사용합니다.

```python
import matplotlib.pyplot as plt
import numpy as np
import matplotlib.collections as collections
import matplotlib.lines as lines
import matplotlib.patches as patches
import matplotlib.text as text
```
