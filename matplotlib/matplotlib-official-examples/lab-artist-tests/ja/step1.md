# ライブラリのインポート

まず、必要なライブラリをインポートする必要があります。グラフを作成するために `matplotlib.pyplot` を、ランダムなデータを生成するために `numpy` を、さまざまな種類のアーティストを作成するために `matplotlib.collections`、`matplotlib.lines`、`matplotlib.patches`、および `matplotlib.text` を使用します。

```python
import matplotlib.pyplot as plt
import numpy as np
import matplotlib.collections as collections
import matplotlib.lines as lines
import matplotlib.patches as patches
import matplotlib.text as text
```
