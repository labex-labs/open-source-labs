# 소개

Matplotlib 는 시각적으로 매력적인 그래프를 생성하기 위한 다양한 도구와 기술을 제공하는 Python 데이터 시각화 라이브러리입니다. 때로는 텍스트가 겹치거나 잘리는 것을 방지하기 위해 y-label 을 위한 공간을 확보해야 합니다. 이 랩에서는 axes_grid 모듈을 사용하여 Matplotlib 에서 y-label 을 위한 공간을 만드는 방법을 보여줍니다.
