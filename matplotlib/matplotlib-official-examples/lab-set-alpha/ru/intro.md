# Введение

В этом практическом занятии (лабораторной работе) рассказывается о настройке альфа-значения цвета с использованием библиотеки Python Matplotlib. Альфа-значение является мерой прозрачности, где значение 0 означает полную прозрачность, а значение 1 - полную непрозрачность. Мы рассмотрим два способа установки альфа-значения в Matplotlib: с использованием ключевого аргумента `alpha` и с использованием формата цвета `(matplotlib_color, alpha)`.
