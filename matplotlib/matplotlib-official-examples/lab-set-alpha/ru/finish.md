# Итог

В этой практической работе (лабораторной работе) мы узнали два способа установки альфа-значения цвета в Matplotlib: с использованием ключевого аргумента `alpha` и с использованием формата цвета `(matplotlib_color, alpha)`. Ключевой аргумент `alpha` устанавливает одно и то же альфа-значение для всех столбцов на диаграмме, в то время как формат цвета `(matplotlib_color, alpha)` позволяет установить разные альфа-значения для каждого столбца на основе его y-значения.
