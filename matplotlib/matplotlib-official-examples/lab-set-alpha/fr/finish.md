# Résumé

Dans ce laboratoire, nous avons appris deux méthodes pour définir la valeur alpha d'une couleur dans Matplotlib : en utilisant l'argument mot-clé `alpha` et en utilisant le format de couleur `(matplotlib_color, alpha)`. L'argument mot-clé `alpha` définit la même valeur alpha pour toutes les barres d'un diagramme, tandis que le format de couleur `(matplotlib_color, alpha)` nous permet de définir des valeurs alpha différentes pour chaque barre en fonction de sa valeur y.
