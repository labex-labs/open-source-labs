# Introduction

Ce laboratoire porte sur la définition de la valeur alpha d'une couleur à l'aide de la bibliothèque Python Matplotlib. La valeur alpha est une mesure de la transparence, où une valeur de 0 signifie complètement transparente et une valeur de 1 signifie complètement opaque. Nous explorerons deux méthodes pour définir la valeur alpha dans Matplotlib : en utilisant l'argument mot-clé `alpha` et en utilisant le format de couleur `(matplotlib_color, alpha)`.
