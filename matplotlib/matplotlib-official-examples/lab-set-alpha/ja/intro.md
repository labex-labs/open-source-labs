# はじめに

この実験は、Python の Matplotlib ライブラリを使用して色のアルファ値を設定することに関するものです。アルファ値は透明度を表す指標で、0 は完全に透明、1 は完全に不透明を意味します。Matplotlib でアルファ値を設定する 2 つの方法を探っていきます。具体的には、`alpha` キーワード引数を使用する方法と、`(matplotlib_color, alpha)` という色の書式を使用する方法です。
