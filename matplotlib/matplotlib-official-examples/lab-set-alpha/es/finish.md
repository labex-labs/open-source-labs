# Resumen

En este laboratorio, aprendimos dos formas de establecer el valor alfa de un color en Matplotlib: utilizando el argumento de palabra clave `alpha` y utilizando el formato de color `(matplotlib_color, alpha)`. El argumento de palabra clave `alpha` establece el mismo valor alfa para todas las barras de un gráfico, mientras que el formato de color `(matplotlib_color, alpha)` nos permite establecer diferentes valores alfa para cada barra en función de su valor en el eje y.
