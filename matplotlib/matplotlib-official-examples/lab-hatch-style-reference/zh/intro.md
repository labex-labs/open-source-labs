# 简介

本实验将指导你在 Matplotlib 中为大多数多边形添加阴影线的过程，包括 Axes.bar、Axes.fill_between、Axes.contourf 以及 patches.Polygon 的子对象。你将学习如何创建不同的阴影线图案，重复它们以增加密度，并将它们组合以创建其他图案。
