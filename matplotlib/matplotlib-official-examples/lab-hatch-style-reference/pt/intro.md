# Introdução

Este laboratório irá guiá-lo através do processo de adição de hachuras (hatches) à maioria dos polígonos no Matplotlib, incluindo `Axes.bar`, `Axes.fill_between`, `Axes.contourf` e filhos de `patches.Polygon`. Você aprenderá como criar diferentes padrões de hachura, repeti-los para aumentar a densidade e combiná-los para criar padrões adicionais.
