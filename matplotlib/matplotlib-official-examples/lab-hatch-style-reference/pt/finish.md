# Resumo

Neste laboratório, você aprendeu como adicionar hachuras (hatches) à maioria dos polígonos no Matplotlib, incluindo `Axes.bar`, `Axes.fill_between`, `Axes.contourf` e filhos de `patches.Polygon`. Você aprendeu como criar diferentes padrões de hachura, repeti-los para aumentar a densidade e combiná-los para criar padrões adicionais. Você também aprendeu como usar a função `add_patch()` para adicionar um objeto `Rectangle` a um eixo e como usar a função `text()` para adicionar texto ao gráfico.
