# Введение

В визуализации данных диаграммы рассеяния используются для показа взаимосвязи между двумя переменными. Кроме того, гистограммы полезны для показа распределения одной переменной. В этом руководстве вы узнаете, как создать диаграмму рассеяния с гистограммами с использованием библиотеки Matplotlib для Python.
