# Resumen

En este tutorial, aprendiste cómo crear un diagrama de dispersión con histogramas utilizando la biblioteca Matplotlib de Python. Primero importaste las bibliotecas necesarias, luego creaste datos aleatorios para el diagrama de dispersión. A continuación, creaste el diagrama de dispersión y los histogramas utilizando `make_axes_locatable`. Finalmente, mostraste el gráfico.
