# Introducción

En la visualización de datos, los diagramas de dispersión se utilizan para mostrar la relación entre dos variables. Además, los histogramas son útiles para mostrar la distribución de una variable única. En este tutorial, aprenderá a crear un diagrama de dispersión con histogramas utilizando la biblioteca Matplotlib de Python.
