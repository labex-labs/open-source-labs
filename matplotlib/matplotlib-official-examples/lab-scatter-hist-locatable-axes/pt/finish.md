# Resumo

Neste tutorial, você aprendeu como criar um gráfico de dispersão com histogramas usando a biblioteca Matplotlib do Python. Primeiro, você importou as bibliotecas necessárias, em seguida, criou dados aleatórios para o gráfico de dispersão. Depois, você criou o gráfico de dispersão e os histogramas usando `make_axes_locatable`. Finalmente, você exibiu o gráfico.
