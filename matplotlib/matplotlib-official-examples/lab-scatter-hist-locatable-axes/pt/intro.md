# Introdução

Na visualização de dados, os gráficos de dispersão (scatter plots) são usados para mostrar a relação entre duas variáveis. Adicionalmente, os histogramas são úteis para mostrar a distribuição de uma única variável. Neste tutorial, você aprenderá como criar um gráfico de dispersão com histogramas usando a biblioteca Matplotlib do Python.
