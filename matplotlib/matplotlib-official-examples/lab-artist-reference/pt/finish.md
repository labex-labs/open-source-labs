# Resumo

Neste laboratório, aprendemos como desenhar várias formas usando a biblioteca Matplotlib em Python. Aprendemos como definir formas, desenhar formas, personalizar formas e salvar o gráfico como um arquivo de imagem. Matplotlib fornece uma API (Application Programming Interface) fácil de usar para desenhar vários tipos de gráficos e é amplamente utilizado na visualização de dados.
