# Resumen

En este laboratorio, aprendimos cómo dibujar varias formas utilizando la biblioteca Matplotlib en Python. Aprendimos cómo definir formas, dibujar formas, personalizar formas y guardar la gráfica como un archivo de imagen. Matplotlib proporciona una API fácil de usar para dibujar varios tipos de gráficas y se utiliza ampliamente en la visualización de datos.
