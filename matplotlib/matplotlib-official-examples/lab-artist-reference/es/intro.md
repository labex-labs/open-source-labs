# Introducción

En este laboratorio, aprenderemos a dibujar diversas formas utilizando la librería Matplotlib en Python. Matplotlib es una librería de trazado para el lenguaje de programación Python y su extensión matemática numérica NumPy. Proporciona una API orientada a objetos para incrustar trazados en aplicaciones utilizando kits de herramientas GUI de propósito general como Tkinter, wxPython, Qt o GTK.
