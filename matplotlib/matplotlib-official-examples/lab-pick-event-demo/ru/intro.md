# Введение

Этот практикум旨在向您介绍 Matplotlib 中的拾取概念。能够拾取艺术家对象是一个强大的工具，可用于构建响应用户操作的交互式可视化。我们将介绍简单拾取、使用自定义命中测试函数进行拾取、在散点图上进行拾取以及拾取图像。
