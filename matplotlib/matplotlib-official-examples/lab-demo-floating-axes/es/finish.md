# Resumen

En este laboratorio, aprendimos cómo usar el módulo `floating_axes` de Matplotlib para crear gráficos personalizados con formas no rectangulares. Creamos un gráfico de dispersión, un gráfico de barras y un gráfico sectorial usando diferentes sistemas de coordenadas. Usamos `GridHelperCurveLinear` para crear las transformaciones necesarias, `FixedLocator` y `MaxNLocator` para establecer las líneas de cuadrícula y `DictFormatter` y `FormatterHMS` para formatear las etiquetas de los ticks.
