# Introducción

En este laboratorio, aprenderemos a usar el módulo `floating_axes` de Matplotlib para crear gráficos personalizados con formas no rectangulares. Este módulo es útil cuando necesitamos representar datos en sistemas de coordenadas no cartesianas, como gráficos polares o logarítmicos. Demonstraremos cómo crear un gráfico de dispersión, un gráfico de barras y un gráfico sectorial usando diferentes sistemas de coordenadas.
