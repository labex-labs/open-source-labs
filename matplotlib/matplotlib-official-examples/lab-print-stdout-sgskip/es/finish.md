# Resumen

En este laboratorio, aprendiste cómo usar Matplotlib para crear un gráfico de líneas simple y guardarlo como un archivo PNG. También aprendiste cómo personalizar el gráfico agregando un título, etiquetas para el eje x y el eje y, y una cuadrícula. Matplotlib es una biblioteca poderosa que se puede usar para muchos tipos de visualizaciones de datos y puede ayudarte a obtener información sobre tus datos.
