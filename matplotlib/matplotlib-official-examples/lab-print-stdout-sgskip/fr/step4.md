# Personnaliser le graphique

Nous pouvons personnaliser le graphique en ajoutant un titre, des étiquettes pour l'axe des x et l'axe des y, et une grille. Le code correspondant est montré ci-dessous :

```python
plt.title("My Plot")
plt.xlabel("X-axis Label")
plt.ylabel("Y-axis Label")
plt.grid(True)
```
