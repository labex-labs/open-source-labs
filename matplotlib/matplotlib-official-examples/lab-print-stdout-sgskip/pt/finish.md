# Resumo

Neste laboratório, você aprendeu como usar o Matplotlib para criar um gráfico de linha simples e salvá-lo como um arquivo PNG. Você também aprendeu como personalizar o gráfico adicionando um título, rótulos para o eixo x e o eixo y, e uma grade. Matplotlib é uma biblioteca poderosa que pode ser usada para muitos tipos de visualizações de dados e pode ajudá-lo a obter insights sobre seus dados.
