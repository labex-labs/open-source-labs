# Anpassen des Plots

Wir können den Plot anpassen, indem wir einen Titel, Beschriftungen für die x-Achse und die y-Achse sowie ein Gitter hinzufügen. Der Code dazu ist unten dargestellt:

```python
plt.title("Mein Plot")
plt.xlabel("Beschriftung der x-Achse")
plt.ylabel("Beschriftung der y-Achse")
plt.grid(True)
```
