# 简介

在本实验中，我们将学习如何使用 Matplotlib 库在 Python 中创建一个帽形图。帽形图是堆叠柱状图的一种变体，其中每个柱子都是帽形。我们将使用一个按游戏数量和玩家分类的分数数据集来创建该图表。
