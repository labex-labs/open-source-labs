# Introducción

En este laboratorio, aprenderemos cómo crear un gráfico de Sombrero en Python utilizando la librería Matplotlib. Un gráfico de Sombrero es una variante de un gráfico de barras apiladas, donde cada barra tiene la forma de un sombrero. Utilizaremos un conjunto de datos de puntuaciones por número de juegos y jugadores para crear el gráfico.
