# Resumen

En este laboratorio, aprendimos cómo crear un gráfico de Sombrero en Python utilizando la biblioteca Matplotlib. Definimos una función que crea el gráfico de Sombrero, preparamos los datos y creamos el gráfico utilizando la función `hat_graph`. El gráfico de Sombrero es una variante de un gráfico de barras apiladas, donde cada barra tiene la forma de un sombrero. Utilizamos un conjunto de datos de puntuaciones según el número de partidas y jugadores para crear el gráfico.
