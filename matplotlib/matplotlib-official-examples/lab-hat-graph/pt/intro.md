# Introdução

Neste laboratório, aprenderemos como criar um Gráfico de Chapéu (Hat Graph) em Python usando a biblioteca Matplotlib. Um Gráfico de Chapéu é uma variante de um gráfico de barras empilhadas, onde cada barra tem a forma de um chapéu. Usaremos um conjunto de dados de pontuações por número de jogo e jogadores para criar o gráfico.
