# Resumo

Neste laboratório, aprendemos como criar um Gráfico de Chapéu em Python usando a biblioteca Matplotlib. Definimos uma função que cria o Gráfico de Chapéu, preparamos os dados e criamos o gráfico usando a função `hat_graph`. O Gráfico de Chapéu é uma variante de um gráfico de barras empilhadas (stacked bar chart), onde cada barra tem a forma de um chapéu. Usamos um conjunto de dados de pontuações por número de jogo e jogadores para criar o gráfico.
