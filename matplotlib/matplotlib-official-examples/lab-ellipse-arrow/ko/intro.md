# 소개

이 랩에서는 Matplotlib 을 사용하여 방향 화살표가 있는 타원을 그리는 방법을 배우게 됩니다. 타원은 데이터를 표현하기 위해 데이터 시각화에서 일반적으로 사용되는 도형의 한 유형입니다. 타원에 방향 화살표를 추가하면 데이터의 방향에 대한 추가 정보를 제공할 수 있습니다.
