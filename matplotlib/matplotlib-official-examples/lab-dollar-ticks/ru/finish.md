# Резюме

В этом практическом занятии (лабораторной работе) мы научились использовать библиотеку Matplotlib для форматирования меток по оси Y с использованием знака доллара. Мы использовали класс `StrMethodFormatter` из модуля `matplotlib.ticker` для форматирования меток и настроили параметры делений, чтобы дополнительно изменить внешний вид нашего графика.
