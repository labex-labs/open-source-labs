# Форматирование меток по оси Y с использованием знака доллара

Теперь давайте отформатируем метки по оси Y так, чтобы они отображали знак доллара. Мы будем использовать класс `StrMethodFormatter` из модуля `matplotlib.ticker` для форматирования меток.

```python
import matplotlib.ticker as ticker

# Format y-axis labels with dollar signs
fmt = '${x:,.2f}'
tick = ticker.StrMethodFormatter(fmt)
ax.yaxis.set_major_formatter(tick)
```

В приведенном выше коде мы создаем объект `StrMethodFormatter` с форматной строкой `'${x:,.2f}'`. Эта форматная строка указывает, что мы хотим видеть знак доллара, за которым следует пробел, а затем число с разделением тысяч запятыми и двумя знаками после запятой.

Далее мы создаем объект `Tick` с использованием только что созданного объекта `StrMethodFormatter`. Наконец, мы устанавливаем основным форматером для оси Y созданный объект `Tick`.
