# Resumen

En este laboratorio, aprendimos cómo utilizar Matplotlib para formatear las etiquetas del eje y con signos de dólar. Utilizamos la clase `StrMethodFormatter` del módulo `matplotlib.ticker` para formatear las etiquetas y personalizamos los parámetros de las marcas de escala (ticks) para ajustar aún más la apariencia de nuestro gráfico.
