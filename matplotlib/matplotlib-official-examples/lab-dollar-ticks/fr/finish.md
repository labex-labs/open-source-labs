# Résumé

Dans ce laboratoire (lab), nous avons appris à utiliser Matplotlib pour formater les étiquettes de l'axe des ordonnées (y-axis) avec des signes dollars. Nous avons utilisé la classe `StrMethodFormatter` du module `matplotlib.ticker` pour formater les étiquettes, et nous avons personnalisé les paramètres des graduations (ticks) pour ajuster davantage l'apparence de notre graphique.
