# Introduction

Dans ce laboratoire, nous allons apprendre à utiliser Matplotlib pour formater les étiquettes de l'axe des ordonnées (y-axis) d'un graphique afin d'afficher des signes dollars. Cela est particulièrement utile lorsque vous travaillez avec des données financières ou tout type de données nécessitant un formatage monétaire.
