# はじめに

この実験では、Python の Matplotlib ライブラリを使って、マーカーのさまざまなプロパティを使って多変量データセットをプロットする方法を学びます。具体的には、成功した野球の投げを、マーカーのサイズを投手の技術に、マーカーの回転を離陸角度に、そして推力をマーカーの色にマッピングしたスマイリーフェイスとして表現する方法を学びます。
