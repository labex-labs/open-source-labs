# 简介

Matplotlib 是一个在 Python 中广泛使用的数据可视化库。它允许用户创建各种各样的可视化图表，包括折线图、散点图、柱状图等等。在本实验中，你将学习如何在 Matplotlib 中使用 `symlog` 轴缩放来创建对称对数图。
