# 소개

이 랩에서는 Python Matplotlib 을 사용하여 음영 기복도 (shaded relief plot) 를 만드는 방법을 배웁니다. 음영 기복도는 고도 변화를 음영으로 표현하여 지형 데이터를 시각화하는 데 유용합니다.
