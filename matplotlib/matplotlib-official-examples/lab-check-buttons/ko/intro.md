# 소개

이 랩에서는 Python Matplotlib 에서 체크 버튼을 사용하는 방법을 시연합니다. 체크 버튼은 사용자가 체크 박스와 유사하게 체크 버튼을 사용하여 시각적 요소를 켜고 끌 수 있도록 합니다. `CheckButtons` 함수를 사용하여 세 개의 서로 다른 사인파가 있는 플롯을 생성하고, 체크 버튼으로 어떤 파형을 표시할지 선택할 수 있는 기능을 구현합니다.
