# Resumo

Neste laboratório, aprendemos como usar botões de seleção (check buttons) em Python Matplotlib. Criamos um gráfico com três diferentes ondas senoidais e a capacidade de escolher quais ondas são exibidas com os botões de seleção. Usamos a função `CheckButtons` para criar os botões e definimos uma função de _callback_ para alternar a visibilidade da linha correspondente no gráfico.
