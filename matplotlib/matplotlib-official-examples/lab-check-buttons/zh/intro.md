# 简介

本实验将演示如何在 Python Matplotlib 中使用复选按钮。复选按钮允许用户通过复选按钮来打开和关闭视觉元素，类似于复选框。我们将使用 `CheckButtons` 函数创建一个包含三个不同正弦波的图表，并能够通过复选按钮选择显示哪些波形。
