# Bibliotheken importieren

Wir müssen beginnen, indem wir die erforderlichen Bibliotheken für dieses Tutorial importieren. Wir werden `matplotlib.pyplot` für die Darstellung, `matplotlib.cm` für Farbskalen und `mpl_toolkits.mplot3d` für die 3D-Darstellung verwenden.

```python
import matplotlib.pyplot as plt
from matplotlib import cm
from mpl_toolkits.mplot3d import axes3d
```
