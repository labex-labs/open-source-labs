# 简介

本实验展示了如何使用 Matplotlib 在 Python 中创建不规则间隔数据的等高线图。等高线图使用等值线或等高线显示数据值的二维分布。在本示例中，我们将比较在规则网格上插值的不规则间隔数据的等高线图与非结构化三角形网格的三角等高线图。
