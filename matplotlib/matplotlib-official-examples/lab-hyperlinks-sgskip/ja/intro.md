# はじめに

Matplotlib は、Python で幅広い静的、アニメーション、インタラクティブなデータ可視化を作成できる Python データ可視化ライブラリです。この実験では、Matplotlib を使用してプロットにハイパーリンクを追加する方法を学びます。この実験が終了するとき、Matplotlib の散布図や画像にハイパーリンクを追加できるようになります。
