# Introdução

Neste laboratório, você aprenderá como usar o Matplotlib, uma biblioteca Python para criar visualizações, para criar uma figura e anotar sua anatomia. Você aprenderá como criar uma figura, plotar dados, definir limites de eixos, adicionar rótulos e títulos, e anotar a figura com texto e marcadores.
