# Resumo

Neste laboratório, você aprendeu como usar o Matplotlib para criar uma figura e anotar sua anatomia. Você aprendeu como criar uma figura, plotar dados, definir limites de eixos, adicionar rótulos e títulos, e anotar a figura com texto e marcadores. Seguindo os passos deste laboratório, você agora deve ter uma boa compreensão de como usar o Matplotlib para criar e anotar figuras em Python.
