# 简介

在本实验中，你将学习如何使用 Matplotlib（一个用于创建可视化的 Python 库）来创建一个图形并注释其各个部分。你将学习如何创建图形、绘制数据、设置轴限制、添加标签和标题，以及用文本和标记注释图形。
