# 소개

이 랩에서는 시각화를 생성하기 위한 Python 라이브러리인 Matplotlib 을 사용하여 그림을 만들고 해부학적 구조에 주석을 다는 방법을 배우게 됩니다. 그림을 만들고, 데이터를 플롯하고, 축 제한을 설정하고, 레이블과 제목을 추가하고, 텍스트와 마커로 그림에 주석을 다는 방법을 배우게 됩니다.
