# はじめに

この実験では、Matplotlib ライブラリを使って Python で非構造的な三角形グリッドを作成して描画する方法を学びます。まず、一連の点のデロネ三角分割を作成し、その三角分割を描画します。次に、各三角形を構成する 3 つの点のインデックスを指定することで独自の三角分割を指定します。最後に、ユーザーが指定した三角分割を描画します。
