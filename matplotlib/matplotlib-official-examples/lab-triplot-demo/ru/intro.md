# Введение

В этом практическом занятии мы научимся создавать и отображать неструктурированные треугольные сетки в Python с использованием библиотеки Matplotlib. Сначала мы создадим триангуляцию Делоне для набора точек, а затем отобразим триангуляцию. Далее мы укажем собственную триангуляцию, предоставив индексы трех точек, которые составляют каждый треугольник. Наконец, мы отобразим триангуляцию, заданную пользователем.
