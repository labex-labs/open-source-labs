# Einführung

Matplotlib ist eine Python-Bibliothek, mit der Benutzer verschiedene Arten von Graphen und Diagrammen erstellen können. In diesem Tutorial wird Ihnen der Prozess des Erstellens einer Legende in einem Matplotlib-Diagramm beigeführt.
