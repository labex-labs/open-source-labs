# Introducción

Este tutorial lo guiará a través de la creación de un gráfico simple utilizando la biblioteca Matplotlib de Python. Matplotlib es una biblioteca de visualización de datos ampliamente utilizada en el cálculo científico para crear visualizaciones estáticas, animadas e interactivas en Python.
