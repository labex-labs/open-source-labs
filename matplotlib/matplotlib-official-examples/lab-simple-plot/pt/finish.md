# Resumo

Este tutorial guiou você através da criação de um gráfico simples usando Matplotlib. Começamos importando as bibliotecas necessárias, geramos dados para o gráfico, criamos o gráfico, adicionamos rótulos e um título, e adicionamos uma grade. Matplotlib é uma biblioteca poderosa para criar visualizações em Python, e este tutorial é apenas o começo do que você pode fazer com ela.
