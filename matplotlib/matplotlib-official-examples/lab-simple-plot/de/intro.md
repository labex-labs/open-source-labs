# Einführung

In diesem Tutorial wird Ihnen die Erstellung eines einfachen Diagramms mit der Matplotlib-Bibliothek in Python gezeigt. Matplotlib ist eine Datenvisualisierungsbibliothek, die in der wissenschaftlichen Informatik weit verbreitet ist, um statische, animierte und interaktive Visualisierungen in Python zu erstellen.
