# 定义数据

我们将定义用于创建堆叠柱状图的数据。

```python
species = (
    "阿德利企鹅\n $\\mu=$3700.66 克",
    "帽带企鹅\n $\\mu=$3733.09 克",
    "巴布亚企鹅\n $\\mu=5076.02 克$",
)
weight_counts = {
    "低于": np.array([70, 31, 58]),
    "高于": np.array([82, 37, 66]),
}
宽度 = 0.5
```

需注意，这里的“宽度”在代码中实际是指柱状图的宽度参数`width`，为了更符合中文表达习惯，将其翻译为“宽度”，但在实际编程语境中，它就是`width`这个变量名。
