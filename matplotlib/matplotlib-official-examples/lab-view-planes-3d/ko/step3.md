# 기본 3D 뷰 평면과 해당 각도 정의

기본 3D 뷰 평면과 해당 고도 (elevation), 방위각 (azimuth), 롤 (roll) 각도를 정의합니다.

```python
views = [('XY',   (90, -90, 0)),
         ('XZ',    (0, -90, 0)),
         ('YZ',    (0,   0, 0)),
         ('-XY', (-90,  90, 0)),
         ('-XZ',   (0,  90, 0)),
         ('-YZ',   (0, 180, 0))]
```
