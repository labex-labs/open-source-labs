# Définissez la disposition du graphique 3D

Nous définissons la disposition du graphique 3D à l'aide d'une liste de listes. Le `'.'` dans la liste représente un sous-graphique vide.

```python
layout = [['XY',  '.',   'L',   '.'],
          ['XZ', 'YZ', '-XZ', '-YZ'],
          ['.',   '.', '-XY',   '.']]
```
