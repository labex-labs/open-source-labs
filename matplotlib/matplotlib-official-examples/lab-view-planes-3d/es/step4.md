# Definir el diseño del gráfico tridimensional

Definimos el diseño del gráfico tridimensional utilizando una lista de listas. El `'.'` en la lista representa un subgráfico vacío.

```python
layout = [['XY',  '.',   'L',   '.'],
          ['XZ', 'YZ', '-XZ', '-YZ'],
          ['.',   '.', '-XY',   '.']]
```
