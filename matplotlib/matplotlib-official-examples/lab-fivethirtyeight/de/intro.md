# Einführung

Matplotlib ist eine beliebte Datenvisualisierungsbibliothek in Python. Es bietet eine benutzerfreundliche Schnittstelle zum Erstellen einer Vielzahl von Visualisierungen, von einfachen Linienplots bis hin zu komplexen Heatmaps. In diesem Lab werden wir die Grundlagen von Matplotlib durchgehen und einen einfachen Linienplot mit der Stilscheibe "fivethirtyeight" erstellen.
