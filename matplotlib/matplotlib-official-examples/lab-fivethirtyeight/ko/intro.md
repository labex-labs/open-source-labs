# 소개

Matplotlib 는 Python 에서 널리 사용되는 데이터 시각화 라이브러리입니다. 간단한 선 그래프에서 복잡한 히트맵까지 다양한 시각화를 생성하기 위한 사용하기 쉬운 인터페이스를 제공합니다. 이 랩에서는 Matplotlib 의 기본 사항을 살펴보고 "fivethirtyeight" 스타일 시트를 사용하여 간단한 선 그래프를 생성합니다.
