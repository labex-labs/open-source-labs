# 简介

Matplotlib 是 Python 中一个流行的数据可视化库。它提供了一个易于使用的接口，用于创建从简单折线图到复杂热图等各种各样的可视化图表。在本实验中，我们将介绍 Matplotlib 的基础知识，并使用“fivethirtyeight”样式表创建一个简单的折线图。
