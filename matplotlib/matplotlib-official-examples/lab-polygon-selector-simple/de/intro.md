# Einführung

Matplotlib ist eine Python-Bibliothek, die zur Datenvisualisierung verwendet wird. In diesem Lab verwenden wir Matplotlib, um programmgesteuert oder interaktiv ein Polygon zu erstellen. Das Polygon kann verwendet werden, um einen interessanten Bereich hervorzuheben oder einen bestimmten Bereich eines Diagramms auszublenden.
