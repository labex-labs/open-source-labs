# Einführung

Matplotlib ist eine Bibliothek, die zur Datenvisualisierung in Python verwendet wird. Es bietet eine Vielzahl von Tools zum Erstellen verschiedener Arten von Diagrammen, Graphen und Tabellen. Eine nützliche Funktion von Matplotlib ist die Möglichkeit, den Koordinatenformatter anzupassen. In diesem Lab werden wir die Schritte durchgehen, um den Koordinatenformatter in Matplotlib zu modifizieren, um den Bild-„z“-Wert des nächsten Pixels anhand von x und y anzuzeigen.
