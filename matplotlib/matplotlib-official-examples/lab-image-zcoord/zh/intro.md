# 简介

Matplotlib 是一个用于在 Python 中进行数据可视化的库。它提供了广泛的工具来创建各种类型的绘图、图表和图形。Matplotlib 的一个有用特性是能够自定义坐标格式化器。在本实验中，我们将介绍在 Matplotlib 中修改坐标格式化器的步骤，以便根据给定的 x 和 y 报告最近像素的图像 “z” 值。
