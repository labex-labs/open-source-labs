# パラメータを設定する

次に、シミュレーション用のパラメータを定義します。これらのパラメータには、重力加速度、各振り子の長さと質量、およびシミュレーションの時間間隔が含まれます。

```python
G = 9.8  # 重力加速度、m/s^2
L1 = 1.0  # 振り子 1 の長さ、m
L2 = 1.0  # 振り子 2 の長さ、m
L = L1 + L2  # 合成振り子の最大長さ
M1 = 1.0  # 振り子 1 の質量、kg
M2 = 1.0  # 振り子 2 の質量、kg
t_stop = 2.5  # シミュレートする秒数
history_len = 500  # 表示する軌道点の数
```
