# Agregar cadenas laterales y enlaces de esquina

Ahora, agregamos las cadenas laterales y los enlaces de esquina usando las funciones `side()` y `corner()`.

```python
side(sankey, n=links_per_side)
corner(sankey)
side(sankey, n=links_per_side)
corner(sankey)
side(sankey, n=links_per_side)
corner(sankey)
side(sankey, n=links_per_side)
```
