# Resumo

Neste laboratório, você aprendeu como usar o Matplotlib para criar um gráfico interativo que permite aos usuários selecionar um conjunto de pontos usando uma ferramenta lasso. Você também aprendeu como usar uma função de callback para alterar a cor dos pontos selecionados. Essa funcionalidade pode ser estendida para outros projetos que exigem seleção interativa de dados.
