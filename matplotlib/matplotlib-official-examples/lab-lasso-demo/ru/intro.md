# Введение

В этом лабе вы научитесь использовать библиотеку Matplotlib для создания интерактивной диаграммы, которая позволяет пользователям выбирать набор точек с помощью инструмента лассо. Также вы научитесь использовать обратный вызов-функцию для изменения цвета выбранных точек.
