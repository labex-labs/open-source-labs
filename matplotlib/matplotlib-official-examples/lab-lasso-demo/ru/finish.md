# Резюме

В этом лабе вы узнали, как использовать Matplotlib для создания интерактивной диаграммы, которая позволяет пользователям выбирать набор точек с помощью инструмента лассо. Также вы узнали, как использовать обратный вызов-функцию для изменения цвета выбранных точек. Эта функциональность может быть расширена для других проектов, требующих интерактивного выбора данных.
