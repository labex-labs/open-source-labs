# Introdução

Neste laboratório, aprenderemos como criar um diagrama de Ishikawa, também conhecido como diagrama de espinha de peixe (fishbone diagram) ou diagrama de causa e efeito. Os diagramas de Ishikawa são comumente usados para identificar problemas em um sistema, mostrando como causas e efeitos estão interligados. Usaremos Python e a biblioteca Matplotlib para criar o diagrama.
