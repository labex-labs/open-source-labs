# 글꼴 딕셔너리 정의

다음으로, 텍스트와 레이블에 대한 스타일 옵션을 포함할 글꼴 딕셔너리를 정의합니다. 이 예제에서는 글꼴 패밀리 (font family) 를 'serif'로, 색상 (color) 을 'darkred'로, 굵기 (weight) 를 'normal'로, 크기 (size) 를 16 으로 설정합니다.

```python
font = {'family': 'serif',
        'color':  'darkred',
        'weight': 'normal',
        'size': 16,
        }
```
