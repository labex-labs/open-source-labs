# 소개

Matplotlib 는 Python 의 데이터 시각화 라이브러리입니다. 다양한 그래프와 차트를 생성하기 위한 강력한 도구입니다. Matplotlib 의 기능 중 하나는 y-값을 기준으로 다른 색상의 선을 그릴 수 있다는 것입니다. 이 랩에서는 마스크된 배열 (masked arrays) 을 사용하여 y-값에 따라 다른 색상으로 선을 그리는 방법을 보여줍니다.
