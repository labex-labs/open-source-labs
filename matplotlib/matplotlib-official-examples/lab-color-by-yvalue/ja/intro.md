# はじめに

Matplotlib は Python のデータ可視化ライブラリです。あらゆる種類のグラフやチャートを作成する強力なツールです。Matplotlib の特徴の 1 つは、y 値に基づいて異なる色の線を描画する機能です。この実験では、マスク配列を使用して y 値に基づいて異なる色の線を描画する方法を示します。
