# 简介

Matplotlib 是 Python 中的一个数据可视化库。它是创建各种图形和图表的强大工具。Matplotlib 的特性之一是能够根据 y 值绘制不同颜色的线条。本实验将演示如何使用掩码数组根据 y 值绘制不同颜色的线条。
