# Introdução

Este laboratório irá guiá-lo sobre como interagir com dados em múltiplas _canvases_. Ao selecionar e destacar um ponto em um eixo, você irá gerar os dados desse ponto no outro eixo. Usaremos Python Matplotlib para este laboratório.

## Dicas para a VM

Após a inicialização da VM, clique no canto superior esquerdo para mudar para a aba **Notebook** e acessar o [Jupyter Notebook](https://support.labex.io/en/labex-vm/jupyter) para praticar.

Às vezes, pode ser necessário aguardar alguns segundos para que o Jupyter Notebook termine de carregar. A validação das operações não pode ser automatizada devido às limitações do Jupyter Notebook.

Se você enfrentar problemas durante o aprendizado, sinta-se à vontade para perguntar ao Labby. Forneça feedback após a sessão, e resolveremos o problema prontamente para você.
