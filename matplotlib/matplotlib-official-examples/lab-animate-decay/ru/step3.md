# Настраиваем график

Теперь нам нужно настроить график. Мы создадим фигуру и объект оси с использованием функции `subplots()` Matplotlib. Также создадим объект линии для представления синусоидальной волны.

```python
fig, ax = plt.subplots()
line, = ax.plot([], [], lw=2)
ax.grid()
xdata, ydata = [], []
```
