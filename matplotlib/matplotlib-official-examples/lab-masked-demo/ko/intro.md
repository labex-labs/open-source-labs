# 소개

데이터 시각화에서, 플롯해야 할 누락된 데이터를 만나는 것은 흔한 일입니다. 이 튜토리얼에서는 Matplotlib 을 사용하여 누락된 값을 가진 데이터를 플롯하는 방법을 배웁니다. 세 가지 방법을 살펴볼 것입니다: 원치 않는 데이터 포인트 제거, 포인트 마스킹, 그리고 값을 NaN 으로 설정하기.
