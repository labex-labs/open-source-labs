# Введение

В этом практическом занятии вы научитесь создавать трехмерный воксельный график с использованием цилиндрических координат в Matplotlib. График покажет, как использовать параметры _x_, _y_ и _z_ метода `.Axes3D.voxels` для создания трехмерного воксельного графика.
