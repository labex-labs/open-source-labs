# Introduction

Dans ce laboratoire, vous allez apprendre à créer un tracé de voxels 3D en utilisant les coordonnées cylindriques dans Matplotlib. Le tracé démontrera comment utiliser les paramètres _x_, _y_ et _z_ de `.Axes3D.voxels` pour créer un tracé de voxels 3D.
