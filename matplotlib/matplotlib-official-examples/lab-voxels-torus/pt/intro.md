# Introdução

Neste laboratório, você aprenderá como criar um gráfico de voxel 3D usando coordenadas cilíndricas em Matplotlib. O gráfico demonstrará como usar os parâmetros _x_, _y_ e _z_ de `.Axes3D.voxels` para criar um gráfico de voxel 3D.
