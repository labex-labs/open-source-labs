# Resumo

_Constrained layout_ é um recurso útil no Matplotlib que redimensiona automaticamente os subplots para evitar sobreposições entre objetos de eixos e rótulos. É particularmente útil ao criar múltiplos subplots em uma figura. Seguindo os passos descritos neste tutorial, você pode criar gráficos claros e legíveis sem ter que se preocupar com rótulos sobrepostos.
