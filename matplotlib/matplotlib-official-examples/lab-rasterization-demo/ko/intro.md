# 소개

이 랩에서는 벡터 그래픽의 래스터화에 대해 배우게 됩니다. 래스터화는 벡터 그래픽을 래스터 이미지 (픽셀) 로 변환하는 과정입니다. 렌더링 속도를 높이고 대규모 데이터 세트에 대해 더 작은 파일을 생성할 수 있지만, 고정된 해상도라는 단점이 있습니다. Python Matplotlib 라이브러리를 사용하여 래스터화 개념을 설명합니다.
