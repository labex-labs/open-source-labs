# 简介

在本实验中，你将学习矢量图形的光栅化。光栅化是将矢量图形转换为光栅图像（像素）的过程。它可以加快渲染速度，并为大型数据集生成更小的文件，但代价是分辨率固定。我们将使用 Python Matplotlib 库来说明光栅化的概念。
