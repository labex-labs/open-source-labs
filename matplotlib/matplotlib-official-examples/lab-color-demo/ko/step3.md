# 색상 지정

Matplotlib 은 다음과 같은 여러 가지 방법으로 색상을 지정합니다.

1.  \[0, 1] 범위의 부동 소수점 값으로 구성된 RGB 또는 RGBA 튜플.
2.  16 진수 RGB 또는 RGBA 문자열.
3.  약식 16 진수 RGB 또는 RGBA 문자열.
4.  회색조에 대한 \[0, 1] 범위를 포함하는 부동 소수점 값의 문자열 표현.
5.  단일 문자열, 즉 {'b', 'g', 'r', 'c', 'm', 'y', 'k', 'w'} 중 하나.
6.  X11/CSS4 ("html") 색상 이름.
7.  'xkcd:' 접두사가 붙은 xkcd 색상 설문 조사에서 가져온 이름.
8.  "Cn" 색상 사양, 즉 숫자 다음에 'C'.
9.  {'tab:blue', 'tab:orange', 'tab:green', 'tab:red', 'tab:purple', 'tab:brown', 'tab:pink', 'tab:gray', 'tab:olive', 'tab:cyan'} 중 하나.

이러한 방법 중 일부를 사용하여 차트의 색상을 지정합니다.
