# 简介

散点图用于可视化两个变量之间的关系。当数据中有多个组，并且我们希望在图中区分它们时，带有图例的散点图会很有用。在本实验中，我们将学习如何使用 Matplotlib 库在 Python 中创建带有图例的散点图。
