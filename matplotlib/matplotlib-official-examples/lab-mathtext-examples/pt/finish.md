# Resumo

Este laboratório demonstrou como criar uma figura que exibe os recursos selecionados do motor de renderização matemática do Matplotlib. A figura demonstrou como escrever expressões matemáticas com exemplos de subscritos, sobrescritos, frações, binômios, números empilhados, radicais, fontes, acentos, grego, hebraico, delimitadores, funções e símbolos.
