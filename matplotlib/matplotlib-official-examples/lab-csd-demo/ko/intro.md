# 소개

신호 처리에서 교차 스펙트럼 밀도 (CSD, Cross Spectral Density) 는 주파수 영역에서 두 신호 간의 상관 관계를 측정하는 지표입니다. 두 신호가 주파수 성분 측면에서 서로 얼마나 관련되어 있는지 파악하는 데 사용됩니다. 이 랩에서는 Python 의 Matplotlib 라이브러리를 사용하여 두 신호의 CSD 를 계산하는 방법을 배우게 됩니다.
