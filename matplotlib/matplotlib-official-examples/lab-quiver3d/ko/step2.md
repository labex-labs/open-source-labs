# 그리드 생성

다음으로, 벡터 필드를 표시할 점들의 그리드를 생성합니다. 이 예제에서는 NumPy 의 `meshgrid` 함수를 사용하여 점들의 메쉬 그리드를 생성합니다. `arange` 함수는 지정된 간격 내에서 균등하게 간격을 둔 점들의 배열을 생성하는 데 사용됩니다.

```python
x, y, z = np.meshgrid(np.arange(-0.8, 1, 0.2),
                      np.arange(-0.8, 1, 0.2),
                      np.arange(-0.8, 1, 0.8))
```
