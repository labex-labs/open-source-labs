# 소개

이 랩에서는 Python Matplotlib 을 사용하여 3D 쿼버 플롯 (quiver plot) 을 만드는 방법을 배우게 됩니다. 쿼버 플롯은 벡터 필드를 화살표로 표시합니다. 화살표는 벡터의 방향을 가리키며, 화살표의 길이는 벡터의 크기를 나타냅니다.
