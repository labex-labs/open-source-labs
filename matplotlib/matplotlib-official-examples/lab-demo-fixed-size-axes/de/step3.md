# Definieren der Größe der Achsen

Wir werden die Größe der Achsen mithilfe der `Size`-Klasse definieren. In diesem Beispiel werden wir eine Achse mit einer festen physikalischen Größe von 4,5 Zoll mal 5 Zoll erstellen.

```python
h = [Size.Fixed(1.0), Size.Fixed(4.5)]
v = [Size.Fixed(0.7), Size.Fixed(5.)]
```
