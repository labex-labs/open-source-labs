# 서브플롯 생성

다음으로, `plt.subplot_mosaic`를 사용하여 서브플롯을 생성합니다. 3x2 그리드의 서브플롯을 생성하고 다음과 같이 레이블을 지정합니다.

- 왼쪽 상단 플롯은 "a)"로 레이블이 지정됩니다.
- 왼쪽 하단 플롯은 "b)"로 레이블이 지정됩니다.
- 오른쪽 상단 및 오른쪽 하단 플롯은 각각 "c)"와 "d)"로 레이블이 지정됩니다.

```python
fig, axs = plt.subplot_mosaic([['a)', 'c)'], ['b)', 'c)'], ['d)', 'd)']], layout='constrained')
```
