# 소개

Matplotlib 는 Python 에서 널리 사용되는 데이터 시각화 라이브러리입니다. 서브플롯을 포함하여 다양한 유형의 플롯을 생성하기 위한 다양한 도구를 제공합니다. 서브플롯을 생성할 때, 독자가 제시된 정보를 더 쉽게 이해할 수 있도록 각 플롯에 레이블을 지정하는 것이 종종 도움이 됩니다. 이 랩에서는 Matplotlib 에서 제공하는 다양한 방법을 사용하여 서브플롯에 레이블을 지정하는 방법을 배웁니다.
