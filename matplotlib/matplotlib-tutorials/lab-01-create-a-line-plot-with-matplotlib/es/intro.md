# Introducción

En este laboratorio, aprenderemos cómo crear un gráfico de líneas utilizando Matplotlib. Los gráficos de líneas son una visualización básica que se puede utilizar para representar puntos de datos conectados por segmentos de línea recta. Utilizaremos la biblioteca Matplotlib en Python para crear un gráfico de líneas.
