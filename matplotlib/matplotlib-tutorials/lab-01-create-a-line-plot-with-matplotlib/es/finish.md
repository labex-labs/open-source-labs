# Resumen

En este laboratorio, aprendimos cómo crear un gráfico de líneas utilizando Matplotlib. Importamos las bibliotecas necesarias, generamos algunos datos de muestra, creamos el gráfico de líneas, personalizamos el gráfico y mostramos el gráfico. Los gráficos de líneas son una visualización útil para representar puntos de datos conectados por segmentos de línea recta.
