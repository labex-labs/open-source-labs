# 소개

이 랩에서는 pandas 에서 텍스트 데이터를 처리하는 방법을 소개합니다. 텍스트 데이터를 저장하고, 데이터 전처리 및 변환을 위해 문자열 메서드를 사용하며, 정규 표현식을 사용하여 부분 문자열을 추출하는 방법을 배우게 됩니다. 또한 머신 러닝 알고리즘을 위한 더미 변수를 생성하는 방법도 배웁니다.
