# 소개

이 랩에서는 pandas 라이브러리를 사용하여 Python 에서 시간 델타 (time delta) 를 다루는 과정을 안내합니다. 시간 델타는 지속 시간 또는 시간의 차이를 나타냅니다. 시간 델타를 구성, 조작 및 연산하는 다양한 방법을 살펴볼 것입니다.
