# はじめに

この実験では、データ分析における一般的な問題である pandas での欠損値の処理方法を学びます。欠損値の識別方法、欠損値の埋め込み方法、不要なデータの削除方法について説明します。また、欠損値を表すために使用できる pandas の実験的な `NA` スカラーについても議論します。
