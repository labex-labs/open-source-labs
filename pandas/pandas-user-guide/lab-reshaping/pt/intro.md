# Introdução

Na análise de dados, é comum encontrar conjuntos de dados que não estão organizados da maneira desejada. O _reshaping_ de dados (remodelagem de dados) refere-se ao processo de alterar a forma como os dados são organizados em linhas e colunas. Neste laboratório, exploraremos alguns dos métodos-chave no Pandas para _reshaping_ de dados, incluindo Pivot, Stack, Unstack, Melt, Cross tabulations (tabelas cruzadas), Tiling (telhamento), Factorizing (fatorização) e Exploding (explosão).
