# 简介

本实验重点介绍如何使用 pandas 将数据分析扩展到更大的数据集。它涵盖了诸如加载更少数据、使用高效数据类型、分块以及利用 Dask 等其他库的方法。需要注意的是，pandas 更适合内存分析，对于非常大的数据集可能不是最佳工具。
