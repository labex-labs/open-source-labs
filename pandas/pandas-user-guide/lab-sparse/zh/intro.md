# 简介

本实验将指导你如何在 pandas 库中使用稀疏数据结构。在我们有大量数据，其中大部分数据相似（如零或 NaN）的情况下，这很有用，因此可以在内存中更有效地表示。我们将学习 `SparseArray`、`SparseDtype`、稀疏访问器、稀疏计算以及与 scipy 稀疏矩阵的交互。
