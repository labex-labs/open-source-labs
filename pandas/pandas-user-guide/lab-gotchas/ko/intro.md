# 소개

Pandas 기본 Lab 에 오신 것을 환영합니다! 이 Lab 에서는 Pandas 라이브러리의 몇 가지 기본적인 측면을 탐구합니다: DataFrame 메모리 사용량, if/truth 문 처리, 사용자 정의 함수 (UDF, User Defined Function) 메서드 사용, NA 값 처리, NumPy 와의 차이점, 그리고 스레드 안전성 고려 사항입니다.
