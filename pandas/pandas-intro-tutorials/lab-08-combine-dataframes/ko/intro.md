# 소개

이 랩에서는 Python 의 Pandas 라이브러리를 사용하여 여러 테이블을 결합하는 방법을 탐구하기 위해 대기 질 데이터를 사용합니다. `concat` 및 `merge` 함수를 사용하여 이러한 작업을 수행합니다. 이 랩은 데이터 프레임을 효과적으로 연결하고 병합하는 방법을 이해하는 데 도움이 될 것입니다.
