# 简介

在这个实验中，我们将学习如何使用 Python 的 Pandas 库来计算数据的汇总统计信息。我们将使用泰坦尼克号数据集，其中包含泰坦尼克号沉船事件中乘客的数据。我们将学习如何计算汇总统计信息、聚合统计信息以及按类别统计记录数量。
