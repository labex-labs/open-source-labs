# 简介

Pandas 是由 Python 开发的强大数据处理工具。由于其灵活性和易用性，它常用于数据分析和清理。在本实验中，我们将学习如何使用 Pandas 执行基本操作，如加载数据、创建数据框、访问数据以及进行简单统计。
