# 소개

이 랩에서는 Python 의 강력한 데이터 조작 라이브러리인 Pandas 를 사용하여 플롯을 만드는 방법을 배웁니다. 실제 대기 질 데이터를 사용하여 실질적인 예시를 보여드리겠습니다. 이 랩을 마치면 Pandas 를 사용하여 선 그래프, 산점도, 상자 그림을 만들고 플롯을 사용자 정의할 수 있게 됩니다.
