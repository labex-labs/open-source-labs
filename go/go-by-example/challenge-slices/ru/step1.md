# Слайсы

Задача, которую нужно решить в этом испытании, — создать и манипулировать срезами в Go. Вам понадобится создать пустой срез с ненулевой длиной, задавать и получать значения в срезе, использовать функцию `len` для получения длины среза, функцию `append` для добавления новых значений в срез, функцию `copy` для копирования среза и оператор среза для получения среза элементов из существующего среза.

## Требования

Для завершения этого испытания вам понадобится иметь базовое понимание синтаксиса Go и типа данных среза. Также вам нужно будет быть знакомым с функциями `make`, `append` и `copy`, а также с оператором среза.

## Пример

```sh
# Обратите внимание, что хотя срезы имеют другой тип, чем массивы,

$ go run slices.go
emp: [ ]
set: [a b c]
get: c
len: 3
apd: [a b c d e f]
cpy: [a b c d e f]
sl1: [c d e]
sl2: [a b c d e]
sl3: [c d e f]

# реализации срезов в Go.

# Теперь, когда мы рассмотрели массивы и срезы, мы перейдем к
```
