# Форматирование строк

Вам необходимо форматировать различные типы данных с использованием различных глаголов форматирования в Golang.

## Требования

- Вы должны использовать пакет `fmt` для форматирования данных.
- Вы должны использовать правильный глагол форматирования для каждого типа данных.
- Вы должны быть в состоянии форматировать целые числа, числа с плавающей точкой, строки и структуры.
- Вы должны быть в состоянии контролировать ширину и точность вывода.
- Вы должны быть в состоянии выравнивать вывод по левому или правому краю.

## Пример

```sh
$ go run string-formatting.go
struct1: {1 2}
struct2: {x:1 y:2}
struct3: main.point{x:1, y:2}
type: main.point
bool: true
int: 123
bin: 1110
char:!
hex: 1c8
float1: 78.900000
float2: 1.234000e+08
float3: 1.234000E+08
str1: "string"
str2: "\"string\""
str3: 6865782074686973
pointer: 0xc0000ba000
width1: | 12 | 345 \
  | width2: | 1.20 | 3.45 \
  | width3: | 1.20 | 3.45 \
  | width4: | foo | b \
  | width5: | foo | b \
  | sprintf: a string
io: an error
```
