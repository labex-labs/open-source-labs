# Путь к файлу

В этом задании вам нужно использовать пакет `filepath` для выполнения различных операций с путями к файлам, таких как построение путей в переносимом формате, разделение пути на компоненты директории и файла, проверка, является ли путь абсолютным, нахождение расширения файла и нахождение относительного пути между двумя путями.

## Требования

- Используйте `Join`, чтобы построить пути в переносимом формате.
- Используйте `Dir` и `Base`, чтобы разделить путь на компоненты директории и файла.
- Используйте `IsAbs`, чтобы проверить, является ли путь абсолютным.
- Используйте `Ext`, чтобы найти расширение файла.
- Используйте `TrimSuffix`, чтобы удалить расширение из имени файла.
- Используйте `Rel`, чтобы найти относительный путь между двумя путями.

## Пример

```sh
$ go run file-paths.go

dir1/filename
dir1/filename

Base(p): filename

true

```
