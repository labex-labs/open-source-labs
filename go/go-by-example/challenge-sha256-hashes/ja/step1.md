# SHA256ハッシュ

文字列を与えられた場合、そのSHA256ハッシュを計算します。

## 要件

- プログラムは `crypto/sha256` と `fmt` パッケージをインポートする必要があります。
- プログラムは `sha256.New()` 関数を使用して新しいハッシュを作成する必要があります。
- プログラムは `Write` 関数を使用して文字列を構成するバイトをハッシュに書き込む必要があります。
- プログラムは `Sum` 関数を使用して、最終的なハッシュ結果をバイトスライスとして取得する必要があります。
- プログラムは、元の文字列と16進数形式のハッシュ結果を表示する必要があります。

## 例

```sh
# プログラムを実行すると、ハッシュが計算され、
# 人が読みやすい16進形式で表示されます。
$ go run sha256-hashes.go
sha256 this string
1af1dfa857bf1d8814fe1af8983c18080019922e557f15a8a...

# 上記に示すパターンと同様に、他のハッシュを計算することができます。
# たとえば、SHA512ハッシュを計算するには、`crypto/sha512` をインポートして
# `sha512.New()` を使用します。

# 暗号学的に安全なハッシュが必要な場合は、
# [ハッシュの強さ](https://en.wikipedia.org/wiki/Cryptographic_hash_function) について慎重に調査する必要があります！
```
