# 定时器与定时器通道

在这个挑战中，你需要创建一个每 500 毫秒触发一次的定时器通道，直到我们停止它。你将使用一个通道来等待值的到来。

## 要求

- 使用 `time` 包创建一个定时器通道。
- 使用一个通道来等待值的到来。
- 使用 `select` 语句从通道接收值。
- 在 1600 毫秒后停止定时器通道。

## 示例

```sh
# 当我们运行这个程序时，定时器通道应该在我们停止它之前触发 3 次。
$ go run tickers.go
Tick at 2012-09-23 11:29:56.487625 -0700 PDT
Tick at 2012-09-23 11:29:56.988063 -0700 PDT
Tick at 2012-09-23 11:29:57.488076 -0700 PDT
Ticker stopped
```
