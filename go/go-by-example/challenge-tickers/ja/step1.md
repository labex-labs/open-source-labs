# タイマーとタイマー

このチャレンジでは、停止するまで 500 ミリ秒ごとにタイマーを発生させるタイマーを作成する必要があります。値が到着するたびにチャネルを使用して値を待ち受けます。

## 要件

- `time`パッケージを使用してタイマーを作成します。
- 値が到着するたびにチャネルを使用して値を待ち受けます。
- `select`文を使用してチャネルから値を受信します。
- 1600 ミリ秒後にタイマーを停止します。

## 例

```sh
# このプログラムを実行すると、タイマーは停止する前に 3 回タイマーを発生させる必要があります。
$ go run tickers.go
Tick at 2012-09-23 11:29:56.487625 -0700 PDT
Tick at 2012-09-23 11:29:56.988063 -0700 PDT
Tick at 2012-09-23 11:29:57.488076 -0700 PDT
Ticker stopped
```
