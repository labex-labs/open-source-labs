# Zusammenfassung

Diese Herausforderung hat gezeigt, wie man in Go verschiedene Arten von Zahlen aus Zeichenketten mit dem Paket `strconv` extrahiert. Durch das Erledigen dieser Herausforderung sollten Sie ein besseres Verständnis dafür haben, wie Sie die Zahlensyntaxanalyse in Ihren Go-Programmen behandeln können.
