# Introducción

Este desafío tiene como objetivo probar tu capacidad para definir y usar subcomandos con su propio conjunto de flags en Golang.

Los challenges suelen ser un poco difíciles. Si lo encuentras difícil, puedes discutir con Labby o revisar la solución.
