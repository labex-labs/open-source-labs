# Случайные числа

Требуется написать программу, которая генерирует случайные целые числа и числа с плавающей точкой в заданном диапазоне. Программа также должна быть способна генерировать различные последовательности чисел, меняя зерно (seed).

## Требования

- Программа должна использовать пакет `math/rand` для генерации случайных чисел.
- Программа должна генерировать случайные целые числа в заданном диапазоне.
- Программа должна генерировать случайные числа с плавающей точкой в заданном диапазоне.
- Программа должна быть способна генерировать различные последовательности чисел, меняя зерно (seed).

## Пример

```sh
# В зависимости от того, где вы запускаете этот пример,
# некоторые из сгенерированных чисел могут отличаться.
# Обратите внимание, что на Go playground при
# инициализации зерна с использованием `time.Now()`
# все равно получаются детерминированные результаты
# из-за особенностей реализации playground.
$ go run random-numbers.go
81,87
0.6645600532184904
7.123187485356329,8.434115364335547
0,28
5,87
5,87

# См. документацию по пакету [`math/rand`](https://pkg.go.dev/math/rand),
# чтобы узнать, какие другие случайные величины может
# предоставить Go.
```
