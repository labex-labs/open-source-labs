# Закрытие каналов

В этом задании вам нужно изменить заданный код, чтобы закрыть канал `jobs`, когда у воркера больше нет задач. Также вам нужно использовать канал `done` для уведомления о том, когда все задачи будут выполнены.

## Требования

- Используйте буферизованный канал `jobs` для передачи задач от горутины `main()` к воркеру.
- Используйте канал `done` для уведомления о том, когда все задачи будут выполнены.
- Используйте воркера для повторного приема из `jobs` с использованием `j, more := <-jobs`.
- Используйте специальный двухзначный прием для уведомления по каналу `done`, когда все задачи будут выполнены.
- Отправьте 3 задачи в воркера по каналу `jobs`, а затем закройте его.
- Используйте подход [синхронизации](channel-synchronization), чтобы дождаться завершения работы воркера.

## Пример

```sh
$ go run closing-channels.go
sent job 1
received job 1
sent job 2
received job 2
sent job 3
received job 3
sent all jobs
received all jobs

# The idea of closed channels leads naturally to our next
# example: `range` over channels.
```
