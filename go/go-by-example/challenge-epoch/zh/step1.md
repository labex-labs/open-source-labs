# 纪元

此挑战中要解决的问题是编写一个Go语言程序，该程序能够计算出自Unix纪元以来的秒数、毫秒数或纳秒数。

## 要求

要完成此挑战，你需要对Go语言有基本的了解，并满足以下要求：

- 熟悉Go语言中的 `time` 包。
- 了解如何使用 `time` 包中的 `Unix`、`UnixMilli` 和 `UnixNano` 函数。

## 示例

```sh
$ go run epoch.go
2012-10-31 16:13:58.292387 +0000 UTC
1351700038
1351700038292
1351700038292387000
2012-10-31 16:13:58 +0000 UTC
2012-10-31 16:13:58.292387 +0000 UTC
```
