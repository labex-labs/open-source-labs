# Epoch

Le problème à résoudre dans ce défi est d'écrire un programme en Golang qui peut calculer le nombre de secondes, de millisecondes ou de nanosecondes depuis l'époque Unix.

## Exigences

Pour terminer ce défi, vous devez avoir une compréhension de base du Golang et les exigences suivantes :

- Familiarité avec le package `time` en Golang.
- Connaissance de la manière d'utiliser les fonctions `Unix`, `UnixMilli` et `UnixNano` dans le package `time`.

## Exemple

```sh
$ go run epoch.go
2012-10-31 16:13:58.292387 +0000 UTC
1351700038
1351700038292
1351700038292387000
2012-10-31 16:13:58 +0000 UTC
2012-10-31 16:13:58.292387 +0000 UTC

# Ensuite, nous examinerons une autre tâche liée au temps : l'analyse
# et la mise en forme du temps.
```
