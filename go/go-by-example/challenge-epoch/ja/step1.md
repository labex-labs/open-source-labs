# エポック

このチャレンジで解くべき問題は、Unix エポック以降の秒数、ミリ秒数、またはナノ秒数を計算できる Golang プログラムを書くことです。

## 要件

このチャレンジを完了するには、Golang の基本的な知識と以下の要件が必要です。

- Golang の `time` パッケージに慣れていること。
- `time` パッケージの `Unix`、`UnixMilli`、および `UnixNano` 関数の使い方を知っていること。

## 例

```sh
$ go run epoch.go
2012-10-31 16:13:58.292387 +0000 UTC
1351700038
1351700038292
1351700038292387000
2012-10-31 16:13:58 +0000 UTC
2012-10-31 16:13:58.292387 +0000 UTC

# 次に、別の時間関連のタスクである時間の解析とフォーマットについて見てみましょう。
```
