# Época

El problema que se debe resolver en este desafío es escribir un programa de Golang que pueda calcular el número de segundos, milisegundos o nanosegundos desde la época Unix.

## Requisitos

Para completar este desafío, debes tener un conocimiento básico de Golang y los siguientes requisitos:

- Familiaridad con el paquete `time` en Golang.
- Conocimiento de cómo utilizar las funciones `Unix`, `UnixMilli` y `UnixNano` en el paquete `time`.

## Ejemplo

```sh
$ go run epoch.go
2012-10-31 16:13:58.292387 +0000 UTC
1351700038
1351700038292
1351700038292387000
2012-10-31 16:13:58 +0000 UTC
2012-10-31 16:13:58.292387 +0000 UTC

# A continuación, veremos otra tarea relacionada con la hora: la
# análisis y formateo de fechas y horas.
```
