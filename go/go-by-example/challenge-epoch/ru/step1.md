# Эпоха

Задача, которую необходимо решить в этом испытании, - написать программу на Golang, которая может вычислить количество секунд, миллисекунд или наносекунд с момента Unix-эпохи.

## Требования

Для завершения этого испытания вам необходимо иметь базовое понимание Golang и следующие требования:

- Ознакомленность с пакетом `time` в Golang.
- Знание, как использовать функции `Unix`, `UnixMilli` и `UnixNano` в пакете `time`.

## Пример

```sh
$ go run epoch.go
2012-10-31 16:13:58.292387 +0000 UTC
1351700038
1351700038292
1351700038292387000
2012-10-31 16:13:58 +0000 UTC
2012-10-31 16:13:58.292387 +0000 UTC
```
