# Форматирование и разбор времени

Задача заключается в форматировании и разборе времени в Golang с использованием предоставленных макетов.

## Требования

- Использовать пакет `time` для форматирования и разбора времени.
- Использовать макет `time.RFC3339` для форматирования и разбора времени.
- Использовать эталонное время `Mon Jan 2 15:04:05 MST 2006` для показа шаблона форматирования/разбора заданного времени/строки.
- Использовать функцию `Parse` для разбора времени.
- Использовать функцию `Format` для форматирования времени.
- Использовать функцию `fmt.Println` для вывода отформатированного времени.
- Использовать функцию `fmt.Printf` для вывода отформатированного времени с извлеченными компонентами.

## Пример

```sh
$ go run time-formatting-parsing.go
2014-04-15T18:00:15-07:00
2012-11-01 22:08:41 +0000 +0000
6:00PM
Tue Apr 15 18:00:15 2014
2014-04-15T18:00:15.161182-07:00
0000-01-01 20:41:00 +0000 UTC
2014-04-15T18:00:15-00:00
parsing time "8:41PM" as "Mon Jan _2 15:04:05 2006":...
```
