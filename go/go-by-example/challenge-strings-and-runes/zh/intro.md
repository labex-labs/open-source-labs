# 简介

本次挑战聚焦于 Go 语言中的字符串和符文（rune）。在 Go 语言中，字符串是只读的字节切片，而符文是表示 Unicode 码点的整数。本次挑战将探索如何在 Go 语言中处理字符串和符文。
