# Форматирование и разбор времени

Задача заключается в форматировании и разборе времени в Golang с использованием предоставленных макетов.

- Используйте пакет `time` для форматирования и разбора времени.
- Используйте макет `time.RFC3339` для форматирования и разбора времени.
- Используйте эталонное время `Mon Jan 2 15:04:05 MST 2006` для показа шаблона, с помощью которого форматируется/разбирается заданное время/строка.
- Используйте функцию `Parse` для разбора времени.
- Используйте функцию `Format` для форматирования времени.
- Используйте функцию `fmt.Println` для вывода отформатированного времени.
- Используйте функцию `fmt.Printf` для вывода отформатированного времени с извлеченными компонентами.

```sh
$ go run time-formatting-parsing.go
2014-04-15T18:00:15-07:00
2012-11-01 22:08:41 +0000 +0000
6:00PM
Tue Apr 15 18:00:15 2014
2014-04-15T18:00:15.161182-07:00
0000-01-01 20:41:00 +0000 UTC
2014-04-15T18:00:15-00:00
parsing time "8:41PM" as "Mon Jan _2 15:04:05 2006":...
```

Ниже представлен полный код:

```go
// Go поддерживает форматирование и разбор времени с помощью
// шаблонных макетов.

package main

import (
	"fmt"
	"time"
)

func main() {
	p := fmt.Println

	// Вот простой пример форматирования времени
	// в соответствии с RFC3339, с использованием соответствующего константного макета.
	t := time.Now()
	p(t.Format(time.RFC3339))

	// Разбор времени использует те же значения макетов, что и `Format`.
	t1, e := time.Parse(
		time.RFC3339,
		"2012-11-01T22:08:41+00:00")
	p(t1)

	// `Format` и `Parse` используют примерные макеты. Обычно
	// вы будете использовать константу из пакета `time` для таких макетов, но
	// вы также можете предоставить собственный макет. Макеты должны использовать
	// эталонное время `Mon Jan 2 15:04:05 MST 2006` для показа
	// шаблона, с помощью которого форматируется/разбирается заданное время/строка.
	// Эталонное время должно быть именно таким: год 2006,
	// 15 для часа, понедельник для дня недели и т.д.
	p(t.Format("3:04PM"))
	p(t.Format("Mon Jan _2 15:04:05 2006"))
	p(t.Format("2006-01-02T15:04:05.999999-07:00"))
	form := "3 04 PM"
	t2, e := time.Parse(form, "8 41 PM")
	p(t2)

	// Для чисто числовых представлений вы также можете
	// использовать стандартное строковое форматирование с извлеченными
	// компонентами значения времени.
	fmt.Printf("%d-%02d-%02dT%02d:%02d:%02d-00:00\n",
		t.Year(), t.Month(), t.Day(),
		t.Hour(), t.Minute(), t.Second())

	// `Parse` вернет ошибку при неправильном вводе,
	// объясняя проблему разбора.
	ansic := "Mon Jan _2 15:04:05 2006"
	_, e = time.Parse(ansic, "8:41PM")
	p(e)
}

```
