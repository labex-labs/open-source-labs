# Парсинг URL

Задача требует разбора примера URL, который включает в себя схему, информацию об аутентификации, хост, порт, путь, параметры запроса и фрагмент запроса. Разобранный URL должен использоваться для извлечения отдельных компонентов URL.

## Требования

- Необходимо импортировать пакеты `url` и `net`.
- Пример URL должен быть разобран и проверен на ошибки.
- Схема, информация об аутентификации, хост, порт, путь, параметры запроса и фрагмент запроса должны быть извлечены из разобранного URL.
- Функция `SplitHostPort` должна использоваться для извлечения имени хоста и порта из поля `Host`.
- Функция `ParseQuery` должна использоваться для разбора параметров запроса в map.
