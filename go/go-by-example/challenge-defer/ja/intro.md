# はじめに

`defer` ステートメントは、囲まれた関数が返るまで関数の実行を遅らせるために使用されます。関数が完了した後にクリーンアップ処理が行われるようにするために、関数が完了するまでにどのような経路をたどったかに関係なく、頻繁に使用されます。
