# Arrays

Se le pide que cree un array de enteros con una longitud de 5. Luego establecerá un valor en un índice específico y recuperará un valor de un índice específico. También se le pedirá que encuentre la longitud del array y declare e inicialice un array en una sola línea. Finalmente, creará un array bidimensional e inicializará con valores.

## Requisitos

- Crear un array de enteros con una longitud de 5
- Establecer un valor en un índice específico y recuperar un valor de un índice específico
- Encontrar la longitud del array
- Declarar e inicializar un array en una sola línea
- Crear un array bidimensional e inicializarlo con valores

## Ejemplo

```sh
# Tenga en cuenta que los arrays aparecen en la forma `[v1 v2 v3...]`
# cuando se imprimen con `fmt.Println`.
$ go run arrays.go
emp: [0 0 0 0 0]
set: [0 0 0 0 100]
get: 100
len: 5
dcl: [1 2 3 4 5]
2d: [[0 1 2] [1 2 3]]
```
