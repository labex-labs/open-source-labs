# Tableaux

Vous devrez créer un tableau d'entiers de longueur 5. Vous devrez ensuite définir une valeur à un index spécifique et récupérer une valeur à partir d'un index spécifique. Vous devrez également trouver la longueur du tableau et déclarer et initialiser un tableau en une seule ligne. Enfin, vous créerez un tableau à deux dimensions et le initialiserez avec des valeurs.

## Exigences

- Créer un tableau d'entiers de longueur 5
- Définir une valeur à un index spécifique et récupérer une valeur à partir d'un index spécifique
- Trouver la longueur du tableau
- Déclarer et initialiser un tableau en une seule ligne
- Créer un tableau à deux dimensions et le initialiser avec des valeurs

## Exemple

```sh
# Notez que les tableaux apparaissent sous la forme `[v1 v2 v3...]`
# lorsqu'ils sont imprimés avec `fmt.Println`.
$ go run arrays.go
emp: [0 0 0 0 0]
set: [0 0 0 0 100]
get: 100
len: 5
dcl: [1 2 3 4 5]
2d: [[0 1 2] [1 2 3]]
```
