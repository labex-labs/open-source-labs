# `for`

Ниже приведен код, содержащий различные виды циклов `for`. Однако некоторые части кода неполны, и вам нужно заполнить пропуски, чтобы код работал правильно.

## Требования

- Основы синтаксиса Go
- Знание циклов `for` в Go

## Пример

```sh
$ go run for.go

3
7
8
9

5
```
