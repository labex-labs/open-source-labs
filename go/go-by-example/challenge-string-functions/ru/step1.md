# Строковые функции

Завершите код ниже, чтобы вывести результат работы различных строковых функций, предоставляемых пакетом `strings`.

## Требования

- Используйте пакет `strings` для решения задачи.
- Используйте функцию `fmt.Println` для вывода результата.
- Не изменяйте имя функции или параметры.

## Пример

```sh
$ go run string-functions.go
Contains: true
Count: 2
HasPrefix: true
HasSuffix: true
Index: 1
Join: a-b
Repeat: aaaaa
Replace: f00
Replace: f0o
Split: [a b c d e]
ToLower: test
ToUpper: TEST
```
