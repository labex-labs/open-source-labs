# Введение

В этом практическом занятии мы будем изучать использование хука `useGetSet` в React. Этот хук позволяет нам создавать состояние и получать или обновлять его значение с использованием функций-аксессоров и функций-менеджеров соответственно. В конце практического занятия вы будете лучше понимать, как использовать этот хук в своих React-приложениях.
