# Introdução

Neste laboratório, aprenderemos como criar um elemento `<select>` não controlado (uncontrolled) em React, utilizando funções de callback para passar seu valor ao componente pai. Usaremos a prop `selectedValue` para definir o valor inicial do elemento `<select>`, e o evento `onChange` para enviar o novo valor ao pai. Este laboratório também abordará como usar `Array.prototype.map()` para criar elementos `<option>` para cada valor passado.
