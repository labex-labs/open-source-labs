# はじめに

この実験では、コールバック関数を使って React で制御されていない`<select>`要素を作成し、その値を親コンポーネントに渡す方法を学びます。`<select>`要素の初期値を設定するために`selectedValue`プロップを使い、新しい値を親に送信するために`onChange`イベントを使います。この実験では、渡された各値に対して`<option>`要素を作成するために`Array.prototype.map()`をどのように使うかも説明します。
