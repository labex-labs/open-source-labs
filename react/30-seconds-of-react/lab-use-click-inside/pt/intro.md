# Introdução

Neste laboratório, aprenderemos como criar um custom React hook chamado `useClickInside`. Este hook irá lidar com o evento de clicar dentro de um componente encapsulado e acionar uma função de callback. Usaremos os hooks `useEffect()` e `useRef()` para anexar e limpar o evento `click`. Ao final deste laboratório, você terá uma melhor compreensão de como criar custom hooks e lidar com eventos no React.
