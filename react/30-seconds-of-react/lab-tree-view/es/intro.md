# Introducción

En este laboratorio, aprenderemos a crear una vista de árbol de objetos expandible en React. Este componente es útil para mostrar estructuras de datos complejas en un formato estructurado y collapsible. Usaremos el hook useState y la recursividad para renderizar una vista de árbol de un objeto JSON o matriz con contenido collapsible. Al final de este laboratorio, serás capaz de crear y personalizar un componente de vista de árbol para mostrar tus datos de manera clara y organizada.
