# Introdução

Neste laboratório, aprenderemos como criar uma visualização de árvore de objetos expansível em React. Este componente é útil para exibir estruturas de dados complexas em um formato estruturado e recolhível. Usaremos o hook `useState` e recursão para renderizar uma visualização de árvore de um objeto ou array JSON com conteúdo recolhível. Ao final deste laboratório, você será capaz de criar e personalizar um componente de visualização de árvore para exibir seus dados de forma clara e organizada.
