# 简介

在本实验中，我们将学习如何在 React 中创建一个 `Email Link` 组件，该组件生成一个格式化的用于发送电子邮件的链接。此组件将使用 `mailto:` 链接格式，并接受用于电子邮件收件人、主题和正文的属性。我们还将学习如何使用 `encodeURIComponent` 安全地对电子邮件主题和正文进行编码。
