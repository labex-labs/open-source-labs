# 소개

이 랩에서는 이메일을 보낼 수 있도록 형식이 지정된 링크를 생성하는 React 의 `Email Link` 컴포넌트를 만드는 방법을 배웁니다. 이 컴포넌트는 `mailto:` 링크 형식을 사용하며, 이메일 수신자, 이메일 제목 및 이메일 본문에 대한 props 를 받습니다. 또한 `encodeURIComponent`를 사용하여 이메일 제목과 본문을 안전하게 인코딩하는 방법도 배웁니다.
