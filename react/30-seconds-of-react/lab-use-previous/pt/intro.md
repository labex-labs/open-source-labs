# Introdução

Neste laboratório, exploraremos o uso do hook `usePrevious` em React. Este hook customizado nos permite armazenar o estado ou as props anteriores de um componente, o que pode ser útil em uma variedade de cenários. Ao criar um componente de contador simples e usar o hook `usePrevious`, demonstraremos como implementar essa funcionalidade em seus projetos React.
