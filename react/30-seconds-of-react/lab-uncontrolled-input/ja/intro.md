# はじめに

この実験では、React を使って制御されていない入力フィールドを作成する方法を探ります。制御されていない入力フィールドは、親コンポーネントで値の状態を管理せずにユーザー入力をキャプチャするためのシンプルで柔軟な方法です。値の更新について親に通知するためにコールバック関数を使い、他のコンポーネントや API との統合を容易にします。
