# Introduction

Dans ce laboratoire, nous allons explorer l'utilisation du hook `useToggler` dans React. Ce hook fournit un moyen simple et efficace de créer une variable d'état booléenne qui peut être basculée entre ses deux états. À la fin de ce laboratoire, vous aurez une meilleure compréhension de la manière d'utiliser ce hook dans vos applications React pour créer une fonctionnalité de basculement pour vos composants.
