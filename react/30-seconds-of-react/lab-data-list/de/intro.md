# Einführung

In diesem Lab werden wir untersuchen, wie man in React mithilfe der Komponente `DataList` eine dynamische Listenkomponente erstellt. Mit dieser Komponente können wir ein Array von primitiven Werten als nummerierte oder aufzählungsfreie Liste rendern. Am Ende des Labs werdet ihr ein besseres Verständnis dafür haben, wie man `Array.prototype.map()` verwendet und in React bedingte Elemente basierend auf Props rendert.
