# Introdução

Neste laboratório, aprenderemos como usar o hook `useNavigatorOnLine` em React para verificar se um cliente está online ou offline. Criaremos uma função para obter o status online do cliente usando a API web `Navigator.onLine`, usaremos o hook `useState()` para criar uma variável de estado apropriada e adicionaremos listeners para eventos apropriados usando o hook `useEffect()` para atualizar o estado e limpar esses listeners ao desmontar. Finalmente, retornaremos a variável de estado do status online para exibir uma mensagem com base no status online atual.
