# 소개

이 랩에서는 React 에서 `useNavigatorOnLine` 훅을 사용하여 클라이언트가 온라인인지 오프라인인지 확인하는 방법을 배웁니다. `Navigator.onLine` 웹 API 를 사용하여 클라이언트의 온라인 상태를 가져오는 함수를 만들고, `useState()` 훅을 사용하여 적절한 상태 변수를 생성하며, `useEffect()` 훅을 사용하여 적절한 이벤트에 대한 리스너를 추가하여 상태를 업데이트하고 언마운트 시 해당 리스너를 정리합니다. 마지막으로, 현재 온라인 상태에 따라 메시지를 표시하기 위해 온라인 상태 상태 변수를 반환합니다.
