# Introducción

En este laboratorio, exploraremos cómo crear un componente de puntuación de estrellas en React. El componente mostrará un conjunto de estrellas con las que los usuarios pueden interactuar para establecer el valor de la puntuación. Usaremos el hook useState() para administrar el estado del componente y manejar las interacciones del usuario mediante los eventos onMouseOver, onMouseLeave y onClick.
