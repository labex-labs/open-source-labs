# Introduction

Dans ce laboratoire, nous allons explorer la manière de créer un composant d'évaluation étoilée en React. Le composant affichera un ensemble d'étoiles avec lesquelles les utilisateurs peuvent interagir pour définir la valeur de l'évaluation. Nous utiliserons le hook useState() pour gérer l'état du composant et gérer les interactions de l'utilisateur à l'aide des événements onMouseOver, onMouseLeave et onClick.
