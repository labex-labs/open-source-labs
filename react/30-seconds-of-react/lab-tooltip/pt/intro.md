# Introdução

Neste laboratório, aprenderemos como criar um componente de _tooltip_ (dica de ferramenta) em React usando _state_ (estado) e _event handlers_ (manipuladores de eventos). _Tooltips_ são pequenas caixas pop-up que aparecem quando o usuário passa o mouse sobre um elemento, fornecendo informações adicionais. Ao final do laboratório, você será capaz de criar _tooltips_ personalizáveis que podem ser facilmente integrados em suas aplicações React.
