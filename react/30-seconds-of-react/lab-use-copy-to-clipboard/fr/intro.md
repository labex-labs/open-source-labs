# Introduction

Dans ce laboratoire, nous allons apprendre à utiliser le hook `useCopyToClipboard` en React pour copier un texte donné dans le presse-papiers. Ce hook est utile pour implémenter une fonctionnalité "copier dans le presse-papiers" dans les applications web. Nous verrons également comment utiliser les hooks `useState`, `useCallback` et `useEffect` dans cette implémentation.
