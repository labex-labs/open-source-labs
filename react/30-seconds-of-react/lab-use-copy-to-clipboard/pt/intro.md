# Introdução

Neste laboratório, aprenderemos como usar o hook `useCopyToClipboard` em React para copiar um texto fornecido para a área de transferência (clipboard). Este hook é útil para implementar a funcionalidade "copiar para a área de transferência" em aplicações web. Também veremos como usar os hooks `useState`, `useCallback` e `useEffect` nesta implementação.
