# Einführung

In diesem Lab werden wir untersuchen, wie man den `useRequestAnimationFrame`-Hook in React verwendet, um eine Funktion zu animieren und sicherzustellen, dass sie vor jedem Neuzeichnen ausgeführt wird. Dieser Hook ist nützlich für das Erstellen von glatten und effizienten Animationen in Webanwendungen. Wir werden den Prozess des Erstellens des Hooks und dessen Implementierung in einem einfachen Zähler-Component durchgehen, das in Echtzeit aktualisiert wird.
