# Introduction

Dans ce laboratoire, nous allons apprendre à implémenter le hook `useBodyScrollLock()` dans une application React. Ce hook nous permet de désactiver le défilement sur l'élément `body` d'une page web lorsqu'un modal ou une autre couche est ouverte. En utilisant ce hook, nous pouvons empêcher les utilisateurs de défiler accidentellement loin du contenu avec lequel ils tentent d'interagir.
