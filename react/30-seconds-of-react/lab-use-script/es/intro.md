# Introducción

En este laboratorio, aprenderemos a usar el hook `useScript` en React para cargar dinámicamente un script externo. Este hook se puede utilizar para manejar eficientemente la carga y descarga de scripts, y puede ser especialmente útil cuando se trabaja con bibliotecas o APIs de terceros. Al final de este laboratorio, serás capaz de usar el hook `useScript` para cargar scripts externos en tus aplicaciones React.
