# Введение

В этом практическом занятии мы научимся использовать хук `useScript` в React для динамической загрузки внешнего скрипта. Этот хук позволяет эффективно обрабатывать загрузку и выгрузку скриптов и особенно полезен при работе с библиотеками или API третьих лиц. В конце практического занятия вы сможете использовать хук `useScript` для загрузки внешних скриптов в своих React-приложениях.
