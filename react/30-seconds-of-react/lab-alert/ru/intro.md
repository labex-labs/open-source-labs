# Введение

В этом практическом занятии (lab) мы создадим компонент Closable Alert с использованием React. Этот компонент позволит пользователям отображать сообщение определенного типа и закрывать уведомление (alert), когда они закончат с ним работу. К концу этого практического занятия вы научитесь использовать хуки `useState()` и `useEffect()` для создания динамического компонента, который можно легко настроить под свои нужды.
