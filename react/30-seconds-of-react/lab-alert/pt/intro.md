# Introdução

Neste laboratório, criaremos um componente de alerta fechável usando React. O componente permitirá que os usuários exibam uma mensagem de um tipo específico e fechem o alerta quando terminarem de usá-lo. Ao final deste laboratório, você terá aprendido a usar os hooks `useState()` e `useEffect()` para criar um componente dinâmico que pode ser facilmente personalizado para atender às suas necessidades.
