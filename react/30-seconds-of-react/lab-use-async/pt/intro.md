# Introdução

Neste laboratório, aprenderemos como usar o hook `useAsync` em React para lidar com chamadas assíncronas. Este hook customizado nos permite definir uma função handler (manipuladora) e, em seguida, executá-la de forma assíncrona, enquanto atualizamos o estado do nosso componente usando o hook `useReducer`. Ao final deste laboratório, você será capaz de criar código reutilizável para lidar com dados assíncronos em suas aplicações React.
