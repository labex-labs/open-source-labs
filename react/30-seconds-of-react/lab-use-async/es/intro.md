# Introducción

En este laboratorio, aprenderemos a usar el hook `useAsync` en React para manejar llamadas asincrónicas. Este hook personalizado nos permite definir una función manejadora y luego ejecutarla de manera asincrónica mientras actualizamos el estado de nuestro componente usando el hook `useReducer`. Al final de este laboratorio, serás capaz de crear código reusable para manejar datos asincrónicos en tus aplicaciones React.
