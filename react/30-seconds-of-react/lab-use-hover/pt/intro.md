# Introdução

Neste laboratório, exploraremos a implementação de um _hook_ customizado chamado `useHover` em React. Este _hook_ irá lidar com o evento de _hover_ (passar o mouse por cima) de um componente encapsulado e atualizar o estado de acordo. Ao final deste laboratório, você terá uma melhor compreensão de como criar e usar _hooks_ customizados em React para aprimorar a funcionalidade de seus componentes.
