# Introdução

Neste laboratório, aprenderemos como criar um componente de arrastar e soltar arquivos (file drag and drop) em React usando os hooks `useState` e `useEffect`. Este componente permite que os usuários arrastem e soltem arquivos em uma área designada e aciona uma função de callback com o arquivo solto como argumento. Ao final deste laboratório, você terá uma melhor compreensão de como lidar com a funcionalidade de arrastar e soltar em React.
