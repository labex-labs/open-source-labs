# Einführung

In diesem Lab werden wir lernen, wie man in React mithilfe der useState- und useEffect-Hooks eine Datei-Ziehen-und-Zwischenlegen-Komponente erstellt. Diese Komponente ermöglicht es Benutzern, Dateien einfach auf einen bestimmten Bereich zu ziehen und zu legen und löst eine Rückruffunktion mit der abgelegten Datei als Argument aus. Am Ende dieses Labs werden Sie ein besseres Verständnis dafür haben, wie man in React die Ziehen-und-Zwischenlegen-Funktionalität behandelt.
