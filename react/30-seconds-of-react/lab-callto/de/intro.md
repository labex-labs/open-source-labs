# Einführung

In diesem Lab werden wir lernen, wie man in React einen aufrufbaren Telefonlink mit dem `tel:`-Protokoll erstellt. Dieser Link ermöglicht es Benutzern, eine Telefonnummer direkt von ihrem Gerät aus mit einem Klick anzurufen. Am Ende dieses Labs werden Sie in der Lage sein, diese Funktion in Ihre eigenen React-Anwendungen zu integrieren und so deren Benutzerfreundlichkeit und Zugänglichkeit zu verbessern.
