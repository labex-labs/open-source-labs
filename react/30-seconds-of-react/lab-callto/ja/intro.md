# はじめに

この実験では、`tel:` プロトコルを使用して React で呼び出し可能な電話リンクを作成する方法を学びます。このリンクを使うと、ユーザーはデバイスからただ 1 回のクリックで電話番号を直接呼び出すことができます。この実験が終わると、あなたはこの機能を自分の React アプリケーションに実装し、その使いやすさとアクセシビリティを向上させることができるようになります。
