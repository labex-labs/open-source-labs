# Introdução

Neste laboratório, aprenderemos como criar uma _checkbox_ (caixa de seleção) com estado e seleção múltipla usando React. Este laboratório irá guiá-lo através do processo de renderização de uma lista de _checkboxes_, atualização do seu estado e passagem do valor/valores selecionados para o componente pai usando uma função _callback_. Ao final deste laboratório, você será capaz de criar um componente de _checkbox_ reutilizável que pode ser usado em qualquer projeto React.
