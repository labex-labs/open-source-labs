# Einführung

In diesem Lab werden wir den `useOnGlobalEvent`-Hook in React erkunden. Dieser Hook ermöglicht es uns, Ereignissen zu lauschen, die auf dem globalen Objekt auftreten, wie beispielsweise das `mousemove`-Ereignis auf dem `window`-Objekt. Indem wir diesen Hook verwenden, können wir einfach eine Callback-Funktion ausführen, wenn ein bestimmtes globales Ereignis auftritt.
