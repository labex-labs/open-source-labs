# 简介

在本实验中，我们将探索 React 中的 `useOnGlobalEvent` 钩子。这个钩子使我们能够监听在全局对象上发生的事件，比如 `window` 对象上的 `mousemove` 事件。通过使用这个钩子，我们可以在特定的全局事件发生时轻松地执行一个回调函数。
