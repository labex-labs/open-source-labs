# 简介

在本实验中，我们将探索如何在 React 中创建一个不受控的范围输入元素，该元素允许用户通过沿水平线滑动按钮来从一系列数字中选择值。我们将使用回调函数将选定的值传递给父组件。在本实验结束时，你将更好地理解如何在 React 中使用 `onChange` 事件和 `defaultValue` 属性。
