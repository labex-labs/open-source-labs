# Introdução

Neste laboratório, exploraremos como criar um elemento de entrada de intervalo (range input) não controlado em React que permite aos usuários selecionar valores de um intervalo de números deslizando um botão ao longo de uma linha horizontal. Usaremos uma função de callback para passar o valor selecionado para o componente pai. Ao final deste laboratório, você terá uma melhor compreensão de como usar o evento `onChange` e a propriedade `defaultValue` em React.
