# Introdução

Neste laboratório, exploraremos como criar um componente carousel (carrossel) usando React. Um carousel é um elemento de interface do usuário (UI) popular que permite aos usuários percorrer um conjunto de imagens ou conteúdo. Ao usar os hooks `useState()` e `useEffect()`, podemos construir um carousel simples, mas funcional, que rola automaticamente por um conjunto de itens.
