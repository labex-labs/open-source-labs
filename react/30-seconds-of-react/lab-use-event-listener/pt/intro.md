# Introdução

Neste laboratório, exploraremos o uso do hook `useEventListener` em React. O objetivo deste laboratório é ajudá-lo a entender como adicionar event listeners (ouvintes de eventos) a elementos em uma aplicação React e como limpá-los adequadamente para evitar vazamentos de memória. Através de exemplos práticos, você aprenderá como usar este hook para criar interfaces de usuário mais interativas e responsivas.
