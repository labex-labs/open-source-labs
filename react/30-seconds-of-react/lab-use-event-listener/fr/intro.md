# Introduction

Dans ce laboratoire, nous allons explorer l'utilisation du hook `useEventListener` dans React. Le but de ce laboratoire est de vous aider à comprendre comment ajouter des écouteurs d'événements à des éléments dans une application React, et comment les nettoyer correctement pour éviter les fuites mémoire. Par des exemples pratiques, vous allez apprendre à utiliser ce hook pour créer des interfaces utilisateur plus interactives et réactives.
