# Introduction

Dans ce laboratoire (lab), nous allons apprendre à créer un bouton avec une animation d'effet d'onde (ripple effect) en utilisant React. Le but de ce laboratoire est de vous aider à comprendre comment utiliser les hooks React, tels que `useState()` et `useEffect()`, pour gérer l'état du bouton et déclencher l'animation. À la fin de ce laboratoire, vous serez en mesure de créer des boutons interactifs et esthétiquement agréables pour vos applications React.
