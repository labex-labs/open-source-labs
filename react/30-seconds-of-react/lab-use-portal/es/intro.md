# Introducción

En este laboratorio, exploraremos el uso del hook `usePortal` en React. El propósito de este hook es crear un portal que permita renderizar hijos fuera del componente padre. Aprenderemos cómo usar los hooks `useState()`, `useCallback()` y `useEffect()` para crear y administrar un portal, así como cómo usar `ReactDOM.createPortal()` y `ReactDOM.unmountComponentAtNode()` para renderizar y eliminar el portal.
