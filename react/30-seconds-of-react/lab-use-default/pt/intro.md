# Introdução

Neste laboratório, exploraremos como criar um _hook_ personalizado chamado `useDefault` em React. O objetivo deste _hook_ é criar um valor com estado com um _fallback_ padrão se for `null` ou `undefined`, e uma função para atualizá-lo. Ao final do laboratório, você terá uma melhor compreensão de como usar _hooks_ personalizados para simplificar seu código React e torná-lo mais eficiente.
