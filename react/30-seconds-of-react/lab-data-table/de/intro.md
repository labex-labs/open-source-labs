# Einführung

In diesem Lab werden wir untersuchen, wie man React verwendet, um eine Tabelle dynamisch zu rendern, deren Zeilen aus einem Array von primitiven Werten erstellt werden. Insbesondere werden wir die `Array.prototype.map()`-Methode verwenden, um jedes Element im Datenarray zu einem `<tr>`-Element mit einem geeigneten Schlüssel zuzuordnen und es in einer Tabelle mit zwei Spalten (ID und Wert) anzuzeigen. Am Ende dieses Labs werden Sie ein besseres Verständnis dafür erwerben, wie man React verwendet, um dynamische und reaktive Tabellen zu erstellen.
