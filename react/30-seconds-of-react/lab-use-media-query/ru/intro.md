# Введение

В этом лабораторном задании мы будем изучать использование хука `useMediaQuery` в React. Этот хук позволяет проверить, соответствует ли текущая среда заданному медиа-запросу, и возвращать соответствующее значение. Мы узнаем, как использовать этот хук для создания отзывчивых компонентов, которые изменяют свое поведение в зависимости от размера экрана.
