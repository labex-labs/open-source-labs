# Introduction

Dans ce laboratoire, nous allons explorer l'utilisation du hook `useMediaQuery` dans React. Ce hook nous permet de vérifier si l'environnement actuel correspond à une requête média donnée et de retourner la valeur appropriée. Nous allons apprendre à utiliser ce hook pour créer des composants réactifs qui changent leur comportement en fonction de la taille de l'écran.
