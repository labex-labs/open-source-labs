# Introducción

En este laboratorio, exploraremos cómo implementar la carga diferida de imágenes en una aplicación React. La carga diferida mejora el rendimiento de la página al retrasar la carga de las imágenes hasta que se necesiten, lo que reduce el tiempo de carga inicial y mejora la experiencia del usuario. Utilizaremos la API Intersection Observer y los hooks de React para crear un componente reusable que admita la carga diferida de imágenes.
