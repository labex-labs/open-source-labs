# Introducción

En este laboratorio, aprenderemos cómo crear un elemento `<textarea>` no controlado en React. El objetivo de este laboratorio es mostrar cómo usar las propiedades `defaultValue` y `onChange` para pasar el valor del área de texto a una función de devolución de llamada en el componente padre. Al final de este laboratorio, tendrás una mejor comprensión de cómo crear campos de entrada no controlados en React.
