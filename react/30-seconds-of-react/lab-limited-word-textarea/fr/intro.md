# Introduction

Dans ce laboratoire, nous allons créer un composant React qui affiche une zone de texte avec une limite de mots. Le composant utilisera des hooks React tels que `useState()`, `useCallback()` et `useEffect()` pour gérer l'état de la zone de texte et limiter le nombre de mots qui peuvent être saisis. Ce laboratoire fournit un exemple pratique de la manière d'utiliser des hooks pour créer des composants React réutilisables.
