# Introduction

Dans ce laboratoire, nous allons apprendre à utiliser le hook `useComponentDidUpdate` dans React. Ce hook nous permet d'exécuter une fonction de rappel immédiatement après qu'un composant a été mis à jour, de manière similaire à la méthode de cycle de vie `componentDidUpdate()` des composants de classe. Nous allons explorer comment utiliser ce hook pour suivre les modifications de variables d'état spécifiques et exécuter une logique personnalisée en conséquence.
