# Introdução

Neste laboratório, aprenderemos sobre o hook `useComponentDidUpdate` em React. Este hook nos permite executar um callback imediatamente após um componente ser atualizado, de forma semelhante ao método de ciclo de vida `componentDidUpdate()` dos componentes de classe. Exploraremos como usar este hook para rastrear alterações em variáveis de estado específicas e executar lógica personalizada de acordo.
