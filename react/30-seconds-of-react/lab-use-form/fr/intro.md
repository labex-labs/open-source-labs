# Introduction

Dans ce laboratoire, nous allons apprendre à utiliser le crochet `useForm` en React pour créer une valeur avec état à partir des champs d'un formulaire. Cela nous permettra de gérer facilement l'état de nos entrées de formulaire et de les mettre à jour selon les besoins. À la fin de ce laboratoire, vous aurez une meilleure compréhension de la manière de créer et de gérer des formulaires en React en utilisant le crochet `useForm`.
