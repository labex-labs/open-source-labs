# Einführung

In diesem Lab werden wir lernen, wie man ein Passwort-Eingabefeld mit einem Umschaltknopf erstellt, der es Benutzern ermöglicht, ihr Passwort anzuzeigen oder zu verbergen. Dies wird mit der `useState()`-Hook in React durchgeführt. Am Ende dieses Labs werdet ihr ein besseres Verständnis dafür haben, wie man im React den Zustand verwaltet und wie man einen einfachen, aber nützlichen Komponenten für die Benutzeroberfläche erstellt.
