# Introduction

Dans ce laboratoire, nous allons explorer comment utiliser le hook `useOnWindowScroll` en React pour exécuter une fonction de rappel chaque fois que la fenêtre est défilée. Ce hook nous permet d'ajouter un écouteur de défilement à l'objet global `Window` et de le supprimer lorsque le composant est démonté. En utilisant ce hook, nous pouvons facilement ajouter un comportement de défilement personnalisé à nos composants React.
