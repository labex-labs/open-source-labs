# Einführung

In diesem Lab werden wir untersuchen, wie man den `useOnWindowScroll`-Hook in React verwendet, um einen Callback auszuführen, wenn das Fenster scrollt. Dieser Hook ermöglicht es uns, einen Scroll-Listener zum globalen `Window`-Objekt hinzuzufügen und ihn zu entfernen, wenn die Komponente entladen wird. Mit diesem Hook können wir unserer React-Komponenten leicht benutzerdefiniertes Scrollverhalten hinzufügen.
