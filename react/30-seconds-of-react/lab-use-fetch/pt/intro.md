# Introdução

Neste laboratório, aprenderemos sobre o hook `useFetch` em React. Este hook nos permite implementar o método `fetch()` de forma declarativa, facilitando a obtenção de dados de APIs e a atualização do estado do componente. Criaremos um hook personalizado que recebe uma URL e opções, e chama assincronamente `fetch()` para atualizar as variáveis de estado de resposta, erro e abortar (abort).
