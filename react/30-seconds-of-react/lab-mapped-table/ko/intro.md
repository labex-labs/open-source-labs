# 소개

이 랩에서는 객체 배열과 속성 이름 목록을 사용하여 React 에서 동적 테이블 컴포넌트를 만드는 방법을 살펴봅니다. 이 컴포넌트는 `Object.keys()`, `Array.prototype.filter()`, `Array.prototype.includes()`, 그리고 `Array.prototype.reduce()`와 같은 다양한 JavaScript 배열 메서드를 사용하여 데이터를 필터링하고 지정된 속성으로 테이블을 렌더링합니다. 이 랩을 마치면 React 에서 동적 테이블을 렌더링하는 방법에 대한 이해도가 높아질 것입니다.
