# Introdução

Neste laboratório, aprenderemos como criar um hook React personalizado chamado `useKeyPress` que monitora as mudanças no estado pressionado de uma determinada tecla. Este hook será útil em cenários onde precisamos detectar se uma tecla específica está sendo pressionada pelo usuário. Usaremos os hooks `useState` e `useEffect` para lidar com o estado e os listeners de eventos, respectivamente.
