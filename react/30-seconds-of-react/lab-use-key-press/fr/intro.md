# Introduction

Dans ce laboratoire, nous allons apprendre à créer un hook React personnalisé appelé `useKeyPress` qui écoute les changements de l'état appuyé d'une touche donnée. Ce hook sera utile dans des scénarios où nous devons détecter si une touche spécifique est enfoncée par l'utilisateur. Nous utiliserons les hooks `useState` et `useEffect` pour gérer l'état et les écouteurs d'événements respectivement.
