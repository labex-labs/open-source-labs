# Introduction

Dans ce laboratoire, nous allons explorer la manière de créer un composant Modal en React qui peut être contrôlé via des événements. Le composant Modal aura un bouton de fermeture, et un écouteur d'événements clavier sera utilisé pour fermer le modal lorsque la touche Échap est pressée. À la fin de ce laboratoire, vous aurez une bonne compréhension de la manière de créer un composant Modal réutilisable qui peut être facilement intégré dans n'importe quel projet React.
