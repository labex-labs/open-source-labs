# Introducción

En este laboratorio, exploraremos cómo crear un componente Modal en React que se pueda controlar a través de eventos. El componente Modal tendrá un botón de cierre, y se utilizará un oyente de eventos del teclado para cerrar el modal cuando se presione la tecla Esc. Al final de este laboratorio, tendrás una buena comprensión de cómo crear un componente Modal reusable que se puede integrar fácilmente en cualquier proyecto de React.
