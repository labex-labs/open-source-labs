# Introdução

Neste laboratório, exploraremos como criar um componente Modal em React que pode ser controlado através de eventos. O componente Modal terá um botão de fechar, e o ouvinte de eventos do teclado será usado para fechar o modal quando a tecla Esc for pressionada. Ao final deste laboratório, você terá uma boa compreensão de como criar um componente Modal reutilizável que pode ser facilmente integrado em qualquer projeto React.
