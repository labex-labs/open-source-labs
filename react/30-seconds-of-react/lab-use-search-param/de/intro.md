# Einführung

In diesem Lab arbeiten wir mit dem `useSearchParam`-Hook, der die Suchparameter der Browseradresse verfolgt. Mit diesem Hook können wir den aktuellen Wert eines bestimmten Parameters in der URL abrufen und ihn dynamisch aktualisieren, wenn sich die URL ändert. Am Ende dieses Labs werden Sie einen besseren Überblick darüber haben, wie Sie diesen Hook in Ihren React-Anwendungen verwenden, um URL-Parameter zu verarbeiten.
