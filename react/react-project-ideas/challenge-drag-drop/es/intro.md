# Introducción

En este desafío desarrollaremos un arrastrar y soltar en React similar a un tablero de JIRA sin ninguna biblioteca externa.
Aprenderás cómo hacer que un elemento HTML sea arrastrable y cómo hacer que un div capture ese elemento arrastrable.
Los eventos de JavaScript utilizados en este desafío son onDragOver, onDrop, onDragStart.
