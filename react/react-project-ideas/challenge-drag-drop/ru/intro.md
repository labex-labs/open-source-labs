# Введение

В этом задании мы разработаем компонент для перетаскивания элементов в React, подобный доске задач JIRA, не используя никаких внешних библиотек.
Вы научитесь делать HTML-элемент перетаскиваемым и как заставить div ловить этот перетаскиваемый элемент.
В этом задании используются события JavaScript onDragOver, onDrop, onDragStart.
