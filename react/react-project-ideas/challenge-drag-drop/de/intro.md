# Einführung

In dieser Herausforderung werden wir eine React-Drag-and-Drop-Funktion ähnlich einer JIRA-Board-View entwickeln, ohne jede externe Bibliothek zu verwenden.
Sie werden lernen, wie Sie ein HTML-Element als ziehbar machen und wie Sie ein Div-Element dafür nutzen, um das ziehbare Element aufzunehmen.
Die in dieser Herausforderung verwendeten JavaScript-Events sind onDragOver, onDrop und onDragStart.
