# 简介

在这个挑战中，我们将在不使用任何外部库的情况下开发一个类似于 JIRA 看板的 React 拖放功能。
你将学习如何使一个 HTML 元素可拖动，以及如何使一个 div 捕获该可拖动元素。
本挑战中使用的 JavaScript 事件有 onDragOver、onDrop、onDragStart。
