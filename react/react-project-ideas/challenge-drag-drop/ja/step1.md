# ドラッグアンドドロップ

始めるには、エディタを開きます。エディタから次のファイルが見えます。

```txt
├── public
├── src
│   ├── App.js
│   ├── App.css
│   ├── index.css
│   └── index.js
├── package-lock.json
└── package.json
```

## 要件

- プロジェクトの依存関係をインストールするには、次のコマンドを使用します。

  ```bash
  npm i
  ```

- このチャレンジは`App.js`ファイルで完了してください。
- `onDragStart`関数が定義されています。これは、タスクカードの dragstart イベントのイベントハンドラです。タスクの名前プロパティにデータ転送データを設定し、ドロップされたときにタスクを識別するために使用されます。
- `onDrop`関数が定義されています。これは、タスクボードの drop イベントのイベントハンドラです。データ転送データからタスクの名前を取得し、ドロップ位置（cat）に基づいてタスクのカテゴリをタスク状態で更新し、setTasks を使用して更新されたタスク状態を設定します。

## 例

コードを完了したら、次のコマンドで実行します。

```bash
npm start
```
