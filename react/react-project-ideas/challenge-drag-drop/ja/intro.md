# はじめに

このチャレンジでは、外部ライブラリを一切使用せずに、JIRA ボードに似た React のドラッグアンドドロップを開発します。
HTML 要素をドラッグ可能にする方法と、そのドラッグ可能な要素をキャッチする div を作成する方法を学びます。
このチャレンジで使用される JS イベントは、onDragOver、onDrop、onDragStart です。
