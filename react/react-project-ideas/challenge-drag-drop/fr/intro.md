# Introduction

Dans ce défi, nous allons développer un glisser-déposer React similaire à un tableau JIRA sans utiliser de bibliothèque externe.
Vous allez apprendre à rendre un élément HTML draggable et à faire en sorte qu'un div attrape cet élément draggable.
Les événements JS utilisés dans ce défi sont onDragOver, onDrop, onDragStart.
