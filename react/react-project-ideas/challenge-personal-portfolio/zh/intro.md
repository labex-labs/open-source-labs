# 简介

在本教程中，我们将从零开始在 React 中构建一个响应式的个人作品集项目。本教程将教你如何使用 ReactJS、HTML 和 CSS 开发自己的、适用于 2023 年的现代个人作品集网站。
开发过程很简单，我们不会进行任何 API 调用。此网站的内容/文本将在你的组件中进行硬编码。
