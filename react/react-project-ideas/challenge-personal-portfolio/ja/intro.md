# はじめに

このチュートリアルでは、React を使って応答性のあるポートフォリオ プロジェクトをゼロから構築します。このチュートリアルでは、ReactJS、HTML および CSS を使って、自分だけの最新の 2023 年向けの個人ポートフォリオ ウェブサイトを開発する方法を学びます。
開発プロセスは簡単で、API コールは一切行いません。このウェブサイトのコンテンツ/テキストは、コンポーネントにハードコードされます。
