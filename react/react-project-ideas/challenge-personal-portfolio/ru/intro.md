# Введение

В этом уроке мы построим отзывчивый проект-портфолио на React с нуля. В этом уроке вы научитесь создавать свою современную персональную веб-страницу-портфолио, готовую к 2023 году, с использованием ReactJS, HTML и CSS.
Процесс разработки простой, и мы не будем совершать никаких вызовов API. Содержание/Текст для этого сайта будет жестко закодирован в ваших компонентах.
