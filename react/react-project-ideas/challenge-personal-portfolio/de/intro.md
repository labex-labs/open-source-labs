# Einführung

In diesem Tutorial werden wir von Grund auf ein responsives Portfolio-Projekt in React erstellen. In diesem Tutorial lernen Sie, wie Sie Ihre eigene moderne, auf 2023 vorbereitete persönliche Portfolio-Website mit ReactJS, HTML und CSS entwickeln können.
Der Entwicklungsprozess ist einfach und wir werden keine API-Aufrufe durchführen. Der Inhalt/Text für diese Website wird in Ihren Komponenten hartcodiert.
