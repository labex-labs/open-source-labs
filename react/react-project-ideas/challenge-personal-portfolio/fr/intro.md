# Introduction

Dans ce tutoriel, nous allons construire un projet de portefeuille responsive en React从零开始. Ce tutoriel vous apprendra à développer votre propre site web de portefeuille personnel moderne prêt pour 2023 en utilisant reactjs, html & css.
Le processus de développement est simple et nous n'effectuerons pas d'appels API. Le contenu / le texte de ce site web sera codé en dur dans vos composants.
