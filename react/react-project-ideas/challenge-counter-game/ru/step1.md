# Игра-счетчик

Для начала работы откройте редактор. Вы можете увидеть следующие файлы в своем редакторе.

```txt
├── public
├── src
│   ├── components
│   │  └── HomePage
│   │       ├── HomePage.css
│   │       └── HomePage.js
│   ├── App.js
│   ├── index.css
│   └── index.js
├── package-lock.json
└── package.json
```

## Требования

- Чтобы установить зависимости проекта, используйте следующую команду:

  ```bash
  npm i
  ```

- Пожалуйста, завершите это испытание в файле `components/HomePage/HomePage.js`.
- Используйте хук `useState`, чтобы определить две переменные состояния: `count` и `timer`.
- Используйте хук `useEffect`, чтобы запустить таймер, когда переменная состояния таймера изменяется.
- Проверьте значение `timer`. Если оно равно нулю, эффект возвращается раньше и ничего не делает.
- Если значение `timer` не равно нулю, оно настраивает интервал, который уменьшает значение `timer` на 1 каждую секунду (1000 миллисекунд).
- Возвращает функцию очистки, которая отменяет интервал, когда компонент демонтируется или когда значение таймера изменяется.

## Пример

После завершения кода запустите его с помощью следующей команды:

```bash
npm start
```
