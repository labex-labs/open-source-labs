# Введение

В этом испытании вы будете разрабатывать игру-счетчик, то есть счетчик с увеличением и уменьшением, с использованием хуков React, то есть с использованием useState и useEffect, в котором вы можете нажать кнопку, чтобы увеличить значение счетчика. Это можно играть с друзьями и предлагать им нажимать кнопку... как можно чаще за заданное время.
