# はじめに

このチャレンジでは、React フック（useState と useEffect）を使って、カウンターゲーム（つまりインクリメントデクリメントカウンター）を開発します。このゲームでは、ボタンをクリックするとカウント値が増えます。これを友達と一緒に遊んで、与えられた時間内にできるだけ多く「クリック！クリック！クリック！」とクリックするように挑戦してみましょう。
