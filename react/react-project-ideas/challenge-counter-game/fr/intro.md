# Introduction

Dans ce défi, vous développerez un jeu de compteur, également appelé compteur d'incrémentation/décrémentation, en utilisant les hooks React, c'est-à-dire useState et useEffect, dans lequel vous pouvez cliquer sur un bouton pour augmenter la valeur du compteur. Vous pouvez jouer avec vos amis et les défier à cliquer autant que possible dans un laps de temps donné.
