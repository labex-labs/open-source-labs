# Introduction

Dans ce défi, vous développerez un jeu de compteur, également appelé compteur d'incrémentation/décrémentation, en utilisant les hooks React, c'est-à-dire useState et useEffect, dans lequel vous pouvez cliquer sur un bouton pour augmenter la valeur du compteur. Vous pouvez jouer avec vos amis et les défier à cliquer autant que possible dans un laps de temps donné.

<div class="text-xs text-gray-500 dark:text-gray-400 mt-4 border-t border-l-2 border-gray-300 dark:border-gray-600 pt-2 pl-4">
Ceci est un Challenge, qui diffère d'un Guided Lab en ce sens que vous devez essayer de réaliser la tâche du challenge de manière indépendante, plutôt que de suivre les étapes d'un laboratoire pour apprendre. Les challenges sont généralement un peu difficiles. Si vous trouvez cela difficile, vous pouvez discuter avec Labby ou consulter la solution.
