# 简介

在这个挑战中，你将使用 React 钩子（即 useState 和 useEffect）开发一个计数器游戏，也就是递增递减计数器，在这个游戏中你可以点击一个按钮来增加计数值。你可以和你的朋友一起玩这个游戏，并挑战他们在给定的时间内尽可能多地点击……
