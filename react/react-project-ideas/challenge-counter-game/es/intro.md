# Introducción

En este desafío, desarrollarás un juego de contador, también conocido como contador de incremento y decremento, utilizando React hooks, es decir, useState y useEffect, en el que puedes hacer clic en un botón para aumentar el valor del contador. Puedes jugar con tus amigos y desafiarles a hacer clic, clic, clic... lo máximo posible en un tiempo determinado.
