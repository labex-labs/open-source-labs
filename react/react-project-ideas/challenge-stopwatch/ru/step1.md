# Секундомер

Для начала работы откройте редактор. В редакторе вы можете увидеть следующие файлы.

```txt
├── public
├── src
│   ├── components
│   │   ├──common
│   │   ├── stopwatch
│   │   ├── timer
│   │   ├── App.css
│   │   └── App.js
│   ├── index.css
│   └── index.js
├── package-lock.json
└── package.json
```

## Требования

- Чтобы установить зависимости проекта, используйте следующую команду:

  ```bash
  npm i
  ```

- Пожалуйста, завершите это испытание в файле `src/components/timer/Timer.js`.
- Функция `onStart` вызывается каждую секунду хуком useEffect.
  - Она проверяет, достиг ли таймер 0 часов, 0 минут и 0 секунд. Если да, она устанавливает isStarted в false и возвращает.
  - Если таймер не запущен, она возвращает без каких-либо изменений.
  - Если таймер работает, она уменьшает таймер на 1 секунду.
  - Если минуты или секунды достигают 0, она соответствующим образом настраивает часы, минуты или секунды с использованием функции setTimer.
- Функция `onReset` вызывается при нажатии кнопки "Сброс".
  - Она устанавливает isStarted в false и сбрасывает таймер до 0 часов, 0 минут и 0 секунд.

## Пример

После завершения кода запустите его с помощью следующей команды:

```bash
npm start
```
