
## Требования

- Чтобы установить зависимости проекта, используйте следующую команду:

  ```bash
  npm i
  ```

- Пожалуйста, завершите это испытание в файле `src/components/Dino/Dino.js`.
- Инициализируйте два хука `useRef`: `dinoRef` и `cactusRef`. Эти хуки будут использоваться для ссылки на DOM-элементы динозавра и кактуса.
- Инициализируйте хуок `useState` под названием score с начальным значением 0. Этот хуок будет отслеживать счет игрока.
- Определите функцию jump. Она добавляет класс "jump" к DOM-элементу динозавра, ссылка на который хранится в dinoRef. Это запускает CSS-анимацию, которая заставляет динозавра прыгать. Через 300 миллисекунд класс "jump" удаляется, и динозавр возвращается в исходное положение.
- Используйте хуок `useEffect` для обработки игровой логики. Он настраивает интервал, который запускается каждые 10 миллисекунд. Внутри функции интервала он получает текущую позицию динозавра (dinoTop) и кактуса (cactusLeft) с использованием функции getComputedStyle.
- Он проверяет, произошло ли столкновение, сравнивая позицию кактуса (cactusLeft) с позицией динозавра (dinoTop). Если кактус находится в определенном диапазоне и на том же уровне, что и динозавр, то происходит столкновение. В этом случае показывается alert с результатом игрока, и счет сбрасывается до 0 с использованием функции setScore. В противном случае счет увеличивается на 1 с использованием setScore.
- Первый хуок `useEffect` также возвращает функцию очистки, которая очищает интервал, когда компонент демонтируется. Это гарантирует, что интервал корректно очищается, чтобы избежать утечки памяти.
- Второй хуок `useEffect` отвечает за установку и удаление слушателя события "keydown". Когда нажимается клавиша, вызывается функция jump. Это позволяет игроку заставить динозавра прыгать, нажав любую клавишу.

## Пример

После завершения кода запустите его с помощью следующей команды:

```bash
npm start
```
