## Требования

- Чтобы установить зависимости проекта, используйте следующую команду:

  ```bash
  npm i
  ```

- Пожалуйста, завершите это испытание в файле `src/components/Game.js`.
- Используйте хук `useState` для определения трех переменных состояния: board, xTurn и winner.
  - `board` представляет состояние игровой доски. Она инициализируется как массив из 9 элементов, где каждый элемент изначально установлен в null.
  - `xTurn` - это логический флаг, указывающий,轮到 X 走 (является ли сейчас ходом X).
  - `winner` хранит результат функции `calculateWinner`, которая определяет победителя на основе текущего состояния доски.
- Функция `handleClick` вызывается при нажатии на квадрат игровой доски.
  - Она создает копию текущего состояния доски с использованием оператора расширения (`[...board]`) и присваивает ее tmpBoard.
  - Если уже есть победитель (winner истинно) или нажатый квадрат уже отмечен (`tmpBoard[i]` истинно), функция возвращает без каких-либо изменений.
  - В противном случае она обновляет значение нажатого квадрата в `tmpBoard` на "X" или "O" в зависимости от значения `xTurn`.
  - Затем обновленный `tmpBoard` устанавливается в качестве нового значения для состояния доски с использованием `setBoard`, а xTurn переключается с использованием setXTurn.
