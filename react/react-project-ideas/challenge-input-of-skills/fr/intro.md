# Introduction

Dans ce défi, nous allons développer un composant d'entrée d'étiquettes React qui peut ajouter des étiquettes sans utiliser de bibliothèque externe.
Avec seulement un composant, vous pouvez avoir une zone de saisie avec la capacité d'entrer des étiquettes ou des mots clés.
Dans l'entrée de composant d'étiquettes, il n'y aura qu'un seul état qui portera toutes les entrées de l'utilisateur avec l'aide de l'événement JavaScript onKeyUp.
