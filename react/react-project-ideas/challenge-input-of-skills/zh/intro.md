# 简介

在这个挑战中，我们将开发一个 React 标签输入组件，它可以在不使用任何外部库的情况下添加标签。
仅用一个组件，你就可以拥有一个能够输入标签或关键词的输入框。
在标签输入组件的输入中，将只有一个状态，它将借助 onKeyUp JavaScript 事件来承载所有用户输入。
