# Einführung

In dieser Herausforderung werden wir einen React-Tag-Eingabekomponente entwickeln, die Tags hinzufügen kann, ohne eine externe Bibliothek zu verwenden.
Mit nur einer Komponente können Sie einen Eingabefeld haben, das die Möglichkeit bietet, Tags oder Schlüsselwörter einzugeben.
In der Tags-Eingabekomponente wird es nur einen Zustand geben, der alle Benutzereingaben mit Hilfe des JavaScript-Events onKeyUp enthält.
