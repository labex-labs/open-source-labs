# はじめに

このチャレンジでは、外部ライブラリを一切使用せずにタグを追加できる React タグ入力コンポーネントを開発します。
1 つのコンポーネントだけで、タグまたはキーワードを入力できる入力ボックスを持つことができます。
タグコンポーネントの入力では、onKeyUp JavaScript イベントの助けを借りてすべてのユーザー入力を保持する 1 つの状態のみがあります。
