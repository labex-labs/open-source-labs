# Introducción

En este desafío desarrollaremos un componente de entrada de etiquetas de React que puede agregar etiquetas sin ninguna biblioteca externa.
Con solo un componente, puede tener una caja de entrada con la capacidad de ingresar etiquetas o palabras clave.
En la entrada de componentes de etiquetas, solo habrá un estado que llevará todas las entradas del usuario con la ayuda del evento JavaScript onKeyUp.
