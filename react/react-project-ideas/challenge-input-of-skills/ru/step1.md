# Ввод навыков

Для начала откройте редактор. Вы можете увидеть следующие файлы в редакторе.

```txt
├── public
├── src
│   ├── components
│   │   └── TagInput.js
│   ├── App.css
│   ├── App.js
│   ├── index.css
│   └── index.js
├── package-lock.json
└── package.json
```

## Требования

- Чтобы установить зависимости проекта, используйте следующую команду:

  ```bash
  npm i
  ```

- Пожалуйста, завершите это испытание в файле `src/component/TagInput.js`.
- Функция `handleAddTag` вызывается при нажатии клавиши в поле ввода. Если нажатая клавиша не клавиша Enter, функция возвращает управление и ничего не делает. В противном случае она проверяет значение ввода и добавляет его в состояние тегов, если оно не пусто и еще не было добавлено. Затем она очищает поле ввода.
- Функция `onDeleteTag` вызывается при нажатии на тег. Она фильтрует состояние тегов, чтобы удалить нажатый тег, и обновляет состояние отфильтрованными тегами.

## Пример

После завершения кода запустите его с помощью следующей команды:

```bash
npm start
```
