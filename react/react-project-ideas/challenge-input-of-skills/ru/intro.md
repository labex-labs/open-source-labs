# Введение

В этом испытании мы разработаем компонент ввода тегов для React, который может добавлять теги без использования внешней библиотеки.
Только с помощью одного компонента вы можете получить поле ввода, которое позволяет вводить теги или ключевые слова.
В компоненте ввода тегов будет только одно состояние, которое будет хранить все вводы пользователя с помощью события onKeyUp JavaScript.
