# 简介

在这个挑战中，你将使用函数式组件在 React 中开发一个可复用的自定义表单，也就是提交表单。通过将输入字段和按钮作为属性传递，你可以根据需要添加任意数量的输入字段和按钮。在未来的项目中，你也可以使用这个单一组件来创建登录表单、注册表单或用于任何 CRUD 功能的数据创建/更新表单。

这个表单不会动态添加输入字段，但它能够通过将字段作为属性传递来创建多个字段。因此，如果你有一个包含多个表单的大型项目，你可以创建一个单一的自定义表单组件，并在任何需要的地方调用它。
