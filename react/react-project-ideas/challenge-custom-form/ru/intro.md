# Введение

В этом задании вы будете разрабатывать повторно используемую пользовательскую форму, то есть форму отправки, в React с использованием функционального компонента, в которой вы можете добавлять столько полей ввода и кнопок, сколько хотите, передав их в качестве пропсов. Вы также можете использовать этот ОДИН компонент в своих будущих проектах для создания форм входа, регистрации или формы создания/обновления данных для любой функциональности CRUD.

Эта форма не добавляет поля ввода динамически, но имеет возможность создавать несколько полей, передав их в качестве пропсов. Таким образом, если у вас есть крупный проект, в котором может быть несколько форм, вы можете создать ОДИН ВСЕГДА компонент пользовательской формы и вызывать его в любом месте.
