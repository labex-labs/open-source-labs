# Пользовательская форма

Для начала работы откройте редактор. Вы можете увидеть следующие файлы в редакторе.

```txt
├── public
├── src
│   ├── components
│   │   └── CustomForm
│   │       ├── CustomForm.css
│   │       └── CustomForm.js
│   ├── App.js
│   ├── index.css
│   └── index.js
├── package-lock.json
└── package.json
```

## Требования

- Чтобы установить зависимости проекта, используйте следующую команду:

  ```bash
  npm i
  ```

- Пожалуйста, завершите это задание в файле `App.js`.
- Используйте хук `useRef` для создания двух объектов ссылок, `usernameRef` и `passwordRef`. Эти ссылки будут использоваться для доступа к значениям полей ввода.
- Функция `handleLogin`: Эта функция вызывается при нажатии кнопки "Войти". Она выводит значения полей ввода имени пользователя и пароля в консоль и отображает предупреждение с введенным именем пользователя и паролем.
- Функция `handleRegister`: Эта функция вызывается при нажатии кнопки "Зарегистрироваться". Она выводит значения полей ввода имени пользователя и пароля в консоль.

## Пример

После завершения кода запустите его с помощью следующей команды:

```bash
npm start
```
