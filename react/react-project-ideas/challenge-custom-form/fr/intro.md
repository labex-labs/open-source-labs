# Introduction

Dans ce défi, vous développerez un formulaire personnalisé réutilisable, c'est-à-dire un formulaire de soumission en React avec un composant fonctionnel dans lequel vous pouvez ajouter autant de champs de saisie et de boutons que vous le souhaitez en les envoyant en tant que props. Vous pouvez également utiliser ce COMPOSANT UNIQUE dans vos futurs projets pour créer un formulaire de connexion, un formulaire d'inscription ou un formulaire de création / de mise à jour de données pour toute fonctionnalité CRUD.

Ce formulaire ne rajoute pas de champs de saisie dynamiquement, mais il est capable de créer plusieurs champs en les envoyant en tant que props. Ainsi, si vous avez un grand projet qui peut avoir plusieurs formulaires, vous pouvez créer UN SEUL COMPOSANT DE FORMULAIRE PERSONNALISÉ et l'appeler où vous le souhaitez.
