# はじめに

このチャレンジでは、React の機能コンポーネントを使って再利用可能なカスタムフォーム（すなわち送信フォーム）を開発します。このフォームでは、プロップとして送信することで、必要なだけの入力フィールドやボタンを追加できます。この単一のコンポーネントを、将来のプロジェクトでログインフォーム、登録フォーム、または任意の CRUD 機能のデータ作成/更新フォームとしても使用できます。

このフォームは動的に入力フィールドを追加しませんが、プロップとして送信することで複数のフィールドを作成する機能を備えています。したがって、複数のフォームを持つ大規模なプロジェクトでは、1 つの単一のカスタムフォームコンポーネントを作成し、必要な場所で呼び出すことができます。
