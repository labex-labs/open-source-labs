# Einführung

In dieser Herausforderung entwickeln Sie ein wiederverwendbares benutzerdefiniertes Formular, auch Submit-Formular genannt, in React mit einem funktionalen Komponenten, in dem Sie beliebig viele Eingabefelder und Schaltflächen hinzufügen können, indem Sie sie als Props senden. Sie können auch in Ihren zukünftigen Projekten dieses EINZIGEN Komponenten verwenden, um ein Login-Formular, ein Registrierungsformular oder ein Datenerstellungs-/aktualisierungsformular für jede CRUD-Funktionalität zu erstellen.

Dieses Formular fügt keine Eingabefelder dynamisch hinzu, hat jedoch die Möglichkeit, mehrere Felder zu erstellen, indem Sie sie als Props senden. Wenn Sie also ein großes Projekt haben, das mehrere Formulare haben kann, können Sie EINZIGEN BENUTZERDEFINIERTEN FORMULARKOMPONENTEN erstellen und sie überall aufrufen, wo Sie möchten.
