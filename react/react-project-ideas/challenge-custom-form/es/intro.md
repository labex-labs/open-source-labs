# Introducción

En este desafío, desarrollarás un formulario personalizado reusable, es decir, un formulario de envío en React con un componente funcional en el que puedes agregar tantos campos de entrada y botones como desees enviándolos como propiedades. También puedes usar ese COMPONENTE ÚNICO en tus futuros proyectos para tener un formulario de inicio de sesión, un formulario de registro o un formulario de creación/actualización de datos para cualquier funcionalidad CRUD.

Este formulario no agrega dinámicamente campos de entrada, pero tiene la capacidad de crear múltiples campos enviándolos como propiedades. Entonces, si tienes un proyecto grande que puede tener múltiples formularios, puedes crear UN SOLO COMPONENTE DE FORMULARIO PERSONALIZADO y llamarlo dondequiera que desees.
