# Sitio web de comercio electrónico

Para comenzar, abra el editor. Puede ver los siguientes archivos desde su editor.

```txt
├── public
├── src
│   ├── components
│   ├── context
│   ├── mockData
│   ├── App.css
│   ├── App.js
│   ├── index.css

├── package-lock.json
└── package.json
```

## Requisitos

- Para instalar las dependencias del proyecto, use el siguiente comando:

  ```bash
  npm i
  ```

- Complete este desafío en el archivo `src/components/itemDetail/ItemDetail.js`.
- La función `getItemDetail` está definida. Toma un parámetro de id y devuelve el objeto del artículo del array de artículos que coincide con el id dado.
- Implementación de la capacidad de hacer clic en un producto para mostrar la página de detalles del producto.

## Ejemplo

Una vez que haya terminado el código, ejecútelo con el siguiente comando:

```bash
npm start
```
