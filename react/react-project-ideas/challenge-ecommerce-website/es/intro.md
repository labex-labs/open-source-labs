# Introducción

En este tutorial desarrollaremos un sitio web de compras, también conocido como proyecto de Reactjs Ecommerce (con código fuente abierto), utilizando la API de contexto de Reactjs. En este tutorial aprenderá a desarrollar un sitio web de comercio electrónico con funcionalidades básicas, es decir, mostrar productos, agregar al carrito, revisar y finalizar la compra, y colocar y ver pedidos.
Para la enrutación utilizaremos el paquete npm'react-router-dom'.
