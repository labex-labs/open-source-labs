# Site web de commerce électronique

Pour commencer, ouvrez l'éditeur. Vous pouvez voir les fichiers suivants dans votre éditeur.

```txt
├── public
├── src
│   ├── components
│   ├── context
│   ├── mockData
│   ├── App.css
│   ├── App.js
│   ├── index.css
│   └── index.js
├── package-lock.json
└── package.json
```

## Exigences

- Pour installer les dépendances du projet, utilisez la commande suivante :

  ```bash
  npm i
  ```

- Veuillez compléter ce défi dans le fichier `src/components/itemDetail/ItemDetail.js`.
- La fonction `getItemDetail` est définie. Elle prend un paramètre d'identifiant et renvoie l'objet d'article du tableau d'articles qui correspond à l'identifiant donné.
- Implémentation de la capacité de cliquer sur un produit pour afficher la page de détail du produit.

## Exemple

Une fois que vous avez terminé le code, exécutez-le avec la commande suivante :

```bash
npm start
```
