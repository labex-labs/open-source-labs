# Introduction

Dans ce tutoriel, nous allons développer un site web de commerce électronique, également appelé projet Reactjs Ecommerce (avec code source open source), en utilisant l'API Reactjs Context. Ce tutoriel vous apprendra à développer un site web de commerce électronique avec les fonctionnalités de base, à savoir afficher les produits, ajouter au panier, passer en revue et passer la commande, ainsi qu'à placer et à visualiser les commandes.
Pour la routage, nous utiliserons le package npm 'react-router-dom'.
