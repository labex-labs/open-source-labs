# はじめに

このチュートリアルでは、Reactjs コンテキスト API を使用して、ショッピングサイト（つまり、オープンソースコード付きの Reactjs Ecommerce）プロジェクトを開発します。このチュートリアルでは、基本的な機能（つまり、商品の表示、カートへの追加、レビューとチェックアウト、注文の入力と表示）を備えた e コマースサイトを開発する方法を学びます。
ルーティングには、「react-router-dom」npm パッケージを使用します。
