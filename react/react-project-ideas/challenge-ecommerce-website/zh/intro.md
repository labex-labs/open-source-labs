# 简介

在本教程中，我们将使用 Reactjs 上下文 API 开发一个购物网站，即 Reactjs 电子商务（开源代码）项目。本教程将教你开发一个具有基本功能的电子商务网站，即展示商品、添加到购物车、查看和结账以及下单和查看订单。
对于路由，我们将使用“react-router-dom”npm 包。
