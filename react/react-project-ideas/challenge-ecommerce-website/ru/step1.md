# Сайт для онлайн-магазина

Для начала работы откройте редактор. В редакторе вы можете увидеть следующие файлы.

```txt
├── public
├── src
│   ├── components
│   ├── context
│   ├── mockData
│   ├── App.css
│   ├── App.js
│   ├── index.css
│   └── index.js
├── package-lock.json
└── package.json
```

## Требования

- Чтобы установить зависимости проекта, используйте следующую команду:

  ```bash
  npm i
  ```

- Пожалуйста, завершите этот вызов в файле `src/components/itemDetail/ItemDetail.js`.
- Функция `getItemDetail` определена. Она принимает параметр id и возвращает объект элемента из массива элементов, который соответствует заданному id.
- Реализация возможности нажатия на товар для отображения страницы с деталями товара.

## Пример

После завершения кода запустите его с помощью следующей команды:

```bash
npm start
```
