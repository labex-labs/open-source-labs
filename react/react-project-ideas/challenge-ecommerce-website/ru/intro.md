# Введение

В этом руководстве мы разработаем сайт для онлайн-магазина (Reactjs Ecommerce) с открытым исходным кодом с использованием Reactjs Context API. В этом руководстве вы научитесь создавать сайт для онлайн-магазина с базовыми функциями, такими как отображение товаров, добавление в корзину, просмотр и оформление заказа и размещение и просмотр заказов.
Для маршрутизации мы будем использовать npm-пакет 'react-router-dom'.
