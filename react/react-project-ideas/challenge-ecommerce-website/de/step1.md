# E-Commerce-Website

Um zu beginnen, öffnen Sie den Editor. Sie können die folgenden Dateien in Ihrem Editor sehen.

```txt
├── public
├── src
│   ├── components
│   ├── context
│   ├── mockData
│   ├── App.css
│   ├── App.js
│   ├── index.css
│   └── index.js
├── package-lock.json
└── package.json
```

## Anforderungen

- Um die Projektabhängigkeiten zu installieren, verwenden Sie den folgenden Befehl:

  ```bash
  npm i
  ```

- Bitte beenden Sie diese Herausforderung in der Datei `src/components/itemDetail/ItemDetail.js`.
- Die Funktion `getItemDetail` ist definiert. Sie nimmt einen `id`-Parameter entgegen und gibt das Artikelobjekt aus dem Artikelarray zurück, das der angegebenen `id` entspricht.
- Implementierung der Möglichkeit, auf ein Produkt zu klicken, um die Produkt-Detailseite anzuzeigen.

## Beispiel

Wenn Sie den Code abgeschlossen haben, führen Sie ihn mit dem folgenden Befehl aus:

```bash
npm start
```
