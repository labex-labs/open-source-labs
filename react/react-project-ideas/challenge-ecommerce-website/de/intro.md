# Einführung

In diesem Tutorial werden wir eine Einkaufswebsite, aka Reactjs Ecommerce (mit quelloffenen Code) Projekt, mit der Reactjs Context API entwickeln. In diesem Tutorial lernen Sie, eine E-Commerce-Website mit grundlegenden Funktionen zu entwickeln, nämlich Produkte anzuzeigen, in den Warenkorb hinzuzufügen, zu bewerten und abzuschließen sowie Bestellungen zu stellen und anzuzeigen.
Für die Navigation werden wir das npm-Paket 'react-router-dom' verwenden.
