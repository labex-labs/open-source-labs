# Игра "Змейка"

Для начала работы откройте редактор. В редакторе вы можете увидеть следующие файлы.

```txt
├── public
├── src
│   ├── components
│   │   ├── Food.js
│   │   └── Snake.js
│   ├── App.css
│   ├── App.js
│   ├── index.css
│   └── index.js
├── package-lock.json
└── package.json
```

## Требования

- Чтобы установить зависимости проекта, используйте следующую команду:

  ```bash
  npm i
  ```

- Пожалуйста, завершите это испытание в файле `src/App.js`.
- Функция `randomFoodPosition` определена для генерации случайной позиции для элемента еды на игровом поле.
- Внутри функции App определены несколько переменных состояния с использованием хука useState:
  - `snake` представляет текущее состояние змейки.
  - `lastDirection` представляет последнее направление, в котором змейка двигалась.
  - `foodPosition` представляет текущую позицию элемента еды.
  - `isStarted` определяет, началась ли игра.
  - `gameOver` указывает, закончена ли игра.
  - `playgroundRef` - это ссылка на элемент игрового поля.

## Пример

После завершения кода запустите его с помощью следующей команды:

```bash
npm start
```
