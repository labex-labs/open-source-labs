# Introducción

El agrupamiento jerárquico es un método de agrupamiento jerárquico utilizado para agrupar objetos similares. Comienza con cada objeto como su propio grupo y luego fusiona iterativamente los grupos más similares hasta que se cumple un criterio de parada. En este laboratorio, demostraremos el efecto de diferentes métricas en el agrupamiento jerárquico utilizando el algoritmo de agrupamiento jerárquico.
