# 简介

在本次实验中，我们将探索鸢尾花数据集上的半监督分类器。我们将比较标签传播、自训练和支持向量机（SVM）在鸢尾花数据集上生成的决策边界。我们将使用流行的 Python 机器学习库 scikit-learn 来实现分类器并可视化决策边界。
