# 简介

本实验展示了如何使用 RBFSampler 和 Nystroem 来近似 RBF 核的特征图，以便在数字数据集上使用支持向量机（SVM）进行分类。我们比较了在原始空间中使用线性 SVM、使用近似映射的线性 SVM 以及使用核化 SVM 的结果。展示了在近似映射中，不同数量的蒙特卡罗采样（对于使用随机傅里叶特征的 RBFSampler）和训练集的不同大小子集（对于 Nystroem）的时间和准确性。
