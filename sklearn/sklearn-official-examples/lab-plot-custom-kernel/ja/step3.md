# カスタムカーネルの作成

このステップでは、カスタムカーネルを作成します。カスタムカーネルは 2 つの行列の内積になります。値 [[2, 0], [0, 1.0]] の行列 M を作成します。その後、行列 X と Y に M を掛けて内積を求めます。

```python
def my_kernel(X, Y):
    """
    カスタムカーネルを作成します。

                 (2  0)
    k(X, Y) = X  (    ) Y.T
                 (0  1)
    """
    M = np.array([[2, 0], [0, 1.0]])
    return np.dot(np.dot(X, M), Y.T)
```
