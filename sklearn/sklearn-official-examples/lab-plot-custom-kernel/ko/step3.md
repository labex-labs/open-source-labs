# 사용자 지정 커널 생성

이 단계에서는 사용자 지정 커널을 생성합니다. 사용자 지정 커널은 두 행렬의 내적 (dot product) 입니다. 값이 [[2, 0], [0, 1.0]]인 행렬 M 을 생성합니다. 그런 다음 행렬 X 와 Y 를 M 으로 곱하고 그 내적을 계산합니다.

```python
def my_kernel(X, Y):
    """
    사용자 지정 커널을 생성합니다.

                 (2  0)
    k(X, Y) = X  (    ) Y.T
                 (0  1)
    """
    M = np.array([[2, 0], [0, 1.0]])
    return np.dot(np.dot(X, M), Y.T)
```
