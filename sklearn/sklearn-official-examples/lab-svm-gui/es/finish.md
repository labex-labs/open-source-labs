# Resumen

En este tutorial, aprendiste cómo usar la interfaz gráfica de usuario (GUI) de Scikit-learn Libsvm para crear puntos de datos mediante clics y visualizar la región de decisión inducida por diferentes kernels y configuraciones de parámetros. También aprendiste cómo crear las clases Model, Controller, View y ControllBar, así como cómo ejecutar el programa.
