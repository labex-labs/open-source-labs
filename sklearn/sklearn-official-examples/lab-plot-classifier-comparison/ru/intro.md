# Введение

В этом практическом занятии мы сравним несколько классификаторов из Scikit-Learn на синтетических наборах данных. Цель этого практического занятия - проиллюстрировать природу границ решений различных классификаторов. Мы предобработаем наборы данных, разделим их на обучающую и тестовую части и построим графики для наборов данных. Затем мы пройдемся по классификаторам, обучим классификаторы на обучающих данных, построим границы решений и построим графики для тестовых данных. Наконец, мы покажем точность классификации на тестовом наборе.
