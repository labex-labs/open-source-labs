# Sommaire

Dans ce laboratoire, nous avons comparé plusieurs classifieurs dans Scikit-Learn sur des jeux de données synthétiques. Nous avons prétraité les jeux de données, les avons divisés en parties d'entraînement et de test, et avons tracé les jeux de données. Nous avons ensuite itéré sur les classifieurs, ajusté les classifieurs sur les données d'entraînement, tracé les limites de décision et tracé les données de test. Enfin, nous avons affiché la précision de classification sur l'ensemble de test.
