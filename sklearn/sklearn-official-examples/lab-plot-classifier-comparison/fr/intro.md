# Introduction

Dans ce laboratoire, nous comparerons plusieurs classifieurs dans Scikit-Learn sur des jeux de données synthétiques. Le but de ce laboratoire est d'illustrer la nature des limites de décision de différents classifieurs. Nous allons prétraiter les jeux de données, les diviser en parties d'entraînement et de test, et tracer les jeux de données. Nous allons ensuite itérer sur les classifieurs, ajuster les classifieurs sur les données d'entraînement, tracer les limites de décision et tracer les données de test. Enfin, nous afficherons la précision de classification sur l'ensemble de test.
