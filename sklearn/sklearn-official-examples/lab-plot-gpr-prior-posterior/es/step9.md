# Resumen

En este laboratorio, aprendimos cómo usar diferentes kernels para la regresión de procesos gaussianos en la biblioteca Scikit-learn de Python. Cubrimos los kernels más comúnmente utilizados en la RGP, incluyendo el kernel de función de base radial, el kernel de racional cuadrático, el kernel Exp-Sine-Squared, el kernel de producto punto y el kernel de Matérn. También aprendimos cómo trazar muestras extraídas del proceso gaussiano usando una función auxiliar.
