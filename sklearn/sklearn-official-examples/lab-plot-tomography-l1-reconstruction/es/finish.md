# Resumen

En este laboratorio, demostramos el uso de la detección por compresión para la reconstrucción de una imagen bidimensional a partir de un número reducido de proyecciones adquiridas a diferentes ángulos. Comparamos el rendimiento de los métodos de penalización L1 y L2 y observamos que la penalización L1 produce un mejor resultado con menos artefactos.
