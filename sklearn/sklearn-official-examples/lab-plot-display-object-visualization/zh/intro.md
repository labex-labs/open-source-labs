# 简介

在本实验中，我们将学习如何使用 scikit-learn 通过显示对象创建可视化。显示对象是允许我们直接从其各自的指标创建可视化的类。在本实验中，我们将使用“混淆矩阵显示”、“ROC 曲线显示”和“精确率召回率显示”显示对象为二分类问题创建可视化。
