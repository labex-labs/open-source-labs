# 简介

在本教程中，我们将使用 Scikit-Learn（Python 中一个流行的机器学习库）比较核岭回归（KRR）和支持向量回归（SVR）。这两种模型都通过使用核技巧来学习非线性函数。KRR 和 SVR 在损失函数和拟合方法上有所不同。我们将使用一个人工数据集，该数据集由一个正弦目标函数和每隔五个数据点添加的强噪声组成。
