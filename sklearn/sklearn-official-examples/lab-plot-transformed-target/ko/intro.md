# 소개

이 실습에서는 scikit-learn 라이브러리의 TransformedTargetRegressor 를 사용하는 방법을 배웁니다. 선형 회귀 모델을 학습하기 전에 대상 값을 변환하는 이점을 관찰하기 위해 두 개의 다른 데이터 세트에 적용합니다. 합성 데이터와 Ames 주택 데이터 세트를 사용하여 대상 값을 변환하는 영향을 보여줍니다.
