# Резюме

В этом практическом занятии мы узнали, как использовать PowerTransformer для преобразования данных из различных распределений в нормальное распределение с использованием преобразований Бокса-Кокса и Йо-Джонсона. Мы также узнали, как использовать QuantileTransformer для приведения любого произвольного распределения к гауссовскому распределению. Важно визуализировать данные до и после преобразования, так как некоторые преобразования могут быть неэффективными для определенных наборов данных.
