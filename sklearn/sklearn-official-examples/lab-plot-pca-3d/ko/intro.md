# 소개

주성분 분석 (PCA) 은 데이터를 단순화하는 통계적 기법입니다. 선형 변환 기법으로 데이터에서 가장 중요한 특징이나 패턴을 찾습니다. PCA 는 차원 축소, 데이터 압축 및 특징 추출을 위해 데이터 분석 및 머신 러닝에서 널리 사용됩니다. 이 실습에서는 파이썬의 scikit-learn 라이브러리를 사용하여 데이터 세트에 PCA 를 적용하고 결과를 시각화할 것입니다.
