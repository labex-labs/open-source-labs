# 简介

在本实验中，我们将探索如何使用多任务套索算法进行联合特征选择。我们将使用流行的 Python 机器学习库 scikit-learn 生成一些示例数据并对其拟合模型。然后，我们将绘制模型的结果以查看它们的比较情况。
