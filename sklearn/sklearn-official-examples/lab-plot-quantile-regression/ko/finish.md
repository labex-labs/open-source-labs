# 요약

이 튜토리얼에서는 scikit-learn 을 사용하여 분위수 회귀를 수행하는 방법을 배웠습니다. 두 개의 합성 데이터 세트를 생성하여 분위수 회귀가 어떻게 중요하지 않은 조건부 분위수를 예측할 수 있는지 보여주었습니다. `QuantileRegressor` 클래스를 사용하여 중앙값과 5% 및 95% 로 고정된 하위 및 상위 분위수를 각각 추정했습니다. `QuantileRegressor`를 `LinearRegression`과 비교하고 평균 절대 오차 (MAE) 와 평균 제곱 오차 (MSE) 를 사용하여 성능을 평가했습니다.
