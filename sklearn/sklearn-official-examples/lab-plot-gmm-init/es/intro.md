# Introducción

En este laboratorio, aprenderemos sobre diferentes métodos de inicialización para los Modelos Mixtos Gausianos (GMM). Usaremos la biblioteca scikit-learn para generar datos de muestra y visualizar los resultados de clustering. Hay cuatro métodos diferentes para el parámetro de inicialización _init_param_: `kmeans` (predeterminado), `random`, `random_from_data` y `k-means++`.
