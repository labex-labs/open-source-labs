# Введение

В этом лабораторном занятии мы узнаем о различных методах инициализации для Gaussian Mixture Models (GMM). Мы будем использовать библиотеку scikit-learn для генерации выборочных данных и визуализации результатов кластеризации. Существует четыре различных метода для параметра инициализации _init_param_: `kmeans` (по умолчанию), `random`, `random_from_data` и `k-means++`.
