# Introduction

Dans ce laboratoire, nous allons apprendre les différentes méthodes d'initialisation des modèles de mélange gaussien (GMM). Nous utiliserons la bibliothèque scikit-learn pour générer des données d'échantillonnage et visualiser les résultats de regroupement. Il existe quatre méthodes différentes pour le paramètre d'initialisation _init_param_ : `kmeans` (valeur par défaut), `random`, `random_from_data` et `k-means++`.
