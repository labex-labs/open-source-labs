# 简介

在本实验中，我们将学习如何为套索回归模型选择最佳超参数 alpha。我们将讨论两种方法：（1）仅使用训练集和一些信息准则来选择 alpha 的最优值，以及（2）使用交叉验证来选择最佳超参数。在本示例中，我们将使用糖尿病数据集。
