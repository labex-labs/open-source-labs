# 定义分类器

我们将定义两个分类器：一个使用 K 近邻（KNN），另一个使用邻域成分分析（NCA）和 KNN。我们将使用管道（pipelines）来缩放数据并应用分类器。

```python
names = ["KNN", "NCA, KNN"]

classifiers = [
    Pipeline(
        [
            ("scaler", StandardScaler()),
            ("knn", KNeighborsClassifier(n_neighbors=n_neighbors)),
        ]
    ),
    Pipeline(
        [
            ("scaler", StandardScaler()),
            ("nca", NeighborhoodComponentsAnalysis()),
            ("knn", KNeighborsClassifier(n_neighbors=n_neighbors)),
        ]
    ),
]
```
