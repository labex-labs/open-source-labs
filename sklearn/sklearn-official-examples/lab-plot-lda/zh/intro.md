# 简介

本实验将向你介绍线性判别分析（LDA）及其在分类中的应用。我们将使用 Python 中流行的机器学习库 scikit-learn 来实现 LDA。我们还将探讨协方差的 Ledoit-Wolf 和 Oracle 收缩近似（OAS）估计器如何改进分类。
