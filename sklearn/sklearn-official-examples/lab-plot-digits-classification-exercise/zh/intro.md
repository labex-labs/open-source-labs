# 简介

本实验是一个关于如何使用 scikit-learn 对数字数据集应用分类技术的分步教程。在本实验中，我们将加载数据集、预处理数据、将数据集拆分为训练集和测试集，然后使用两种不同的分类技术（K 近邻和逻辑回归）对数字进行分类。最后，我们将比较这两种技术的准确性。
