# 소개

이 실습에서는 비선형 서포트 벡터 머신 (SVM) 과 방사 기저 함수 (RBF) 커널을 사용하여 이진 분류를 수행하는 과정을 안내합니다. 예측 대상은 입력값의 XOR 입니다. 색상 맵은 SVM 이 학습한 결정 함수를 보여줍니다. 이 작업을 위해 Python scikit-learn 라이브러리를 사용합니다.
