# 소개

이 실습에서는 Scikit-Learn 라이브러리의 `make_multilabel_classification` 함수를 사용하여 다중 레이블 데이터셋을 생성하는 방법을 배웁니다. 이 함수는 각 샘플이 두 개의 특징의 개수를 가지며, 두 클래스 각각에서 서로 다른 분포를 보이는 다중 레이블 데이터의 랜덤 샘플을 생성합니다.

## VM 팁
