# 简介

特征缩放是许多机器学习算法的重要预处理步骤。它涉及对每个特征进行重新缩放，使其标准差为 1，均值为 0。在本实验中，我们将使用 Python 中的 scikit-learn 库探讨特征缩放的重要性及其对机器学习模型的影响。
