# Introduction

Ce laboratoire démontre l'utilisation de l'algorithme de régression Lasso de Scikit-learn sur des données denses et creuses. L'algorithme Lasso est une méthode de régression linéaire qui ajoute une pénalité aux coefficients de régression. Cette pénalité encourage le modèle à produire des coefficients sparsiﬁés. L'algorithme Lasso est utile dans les situations où le nombre de caractéristiques est important par rapport au nombre d'échantillons.
