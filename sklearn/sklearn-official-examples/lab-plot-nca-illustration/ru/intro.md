# Введение

Этот лабораторный практикум посвящен демонстрации использования анализа соседних компонентов (Neighborhood Components Analysis, NCA) для обучения метрики расстояния, которая максимизирует точность классификации ближайших соседей. Он представляет визуальное изображение этой метрики по сравнению с исходным пространством точек.

## Советы по использованию ВМ

После запуска ВМ нажмите в левом верхнем углу, чтобы переключиться на вкладку **Notebook** и получить доступ к Jupyter Notebook для практики.

Иногда вам может потребоваться подождать несколько секунд, пока Jupyter Notebook загрузится полностью. Валидация операций не может быть автоматизирована из-за ограничений Jupyter Notebook.

Если вы сталкиваетесь с проблемами во время обучения, не стесняйтесь обращаться к Labby. Оставьте отзыв после занятия, и мы оперативно решим проблему для вас.
