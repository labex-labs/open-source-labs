# Introdução

Este laboratório demonstra como traçar a probabilidade de classificação de diferentes classificadores usando o Python Scikit-learn. Usaremos um conjunto de dados de 3 classes e o classificaremos com um classificador de Máquinas de Vetores de Suporte, regressão logística penalizada L1 e L2 com um ajuste One-Vs-Rest ou multinomial, e classificação de processos gaussianos.
