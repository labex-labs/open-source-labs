# 拟合 K 均值模型

我们将在图像数据的一个小子样本上拟合 K 均值模型，并使用它来预测整个图像的颜色索引。

```python

from sklearn.utils import shuffle
from time import time

n_colors = 64

# 在数据的一个小子样本上拟合 K 均值模型
print("在数据的一个小子样本上拟合模型")
```
