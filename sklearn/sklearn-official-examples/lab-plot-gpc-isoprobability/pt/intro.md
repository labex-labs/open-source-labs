# Introdução

Este laboratório demonstra como utilizar a Classificação de Processo Gaussiano (GPC) na biblioteca scikit-learn para Python. O objetivo é mostrar como treinar e testar um modelo GPC, plotar linhas de iso-probabilidade e avaliar o desempenho da classificação.
