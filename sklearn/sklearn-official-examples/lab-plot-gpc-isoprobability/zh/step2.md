# 准备数据

我们将生成一些用于分类的合成数据。要分类的函数定义如下：

```python
def g(x):
    """要预测的函数（分类将在于预测 g(x) <= 0 是否成立）"""
    return 5.0 - x[:, 1] - 0.5 * x[:, 0] ** 2.0
```
