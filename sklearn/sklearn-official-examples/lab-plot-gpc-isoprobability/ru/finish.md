# Резюме

В этом практическом занятии мы научились использовать классификацию с использованием гауссовых процессов (Gaussian Process Classification, GPC) из библиотеки scikit - learn для Python. Мы показали, как обучать и тестировать модель GPC, строить линии равной вероятности и оценивать производительность классификации.
