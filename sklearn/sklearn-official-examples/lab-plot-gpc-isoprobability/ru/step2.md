# Подготовка данных

Мы сгенерируем некоторые синтетические данные для классификации. Функция, которую нужно классифицировать, определена как:

```python
def g(x):
    """The function to predict (classification will then consist in predicting
    whether g(x) <= 0 or not)"""
    return 5.0 - x[:, 1] - 0.5 * x[:, 0] ** 2.0
```
