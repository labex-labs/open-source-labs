# Введение

В этом практическом занятии показано, как использовать классификацию с помощью гауссовых процессов (GPC) из библиотеки scikit-learn для Python. Цель - показать, как обучать и тестировать модель GPC, строить линии равномерной вероятности и оценивать производительность классификации.
