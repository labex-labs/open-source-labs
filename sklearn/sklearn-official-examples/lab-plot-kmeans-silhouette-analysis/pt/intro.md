# Introdução

O agrupamento (Clustering) é uma técnica popular de aprendizagem não supervisionada que envolve agrupar pontos de dados semelhantes com base em suas características. O Método Silhouette é uma técnica comumente utilizada para determinar o número ideal de clusters em um conjunto de dados. Neste laboratório, usaremos o Método Silhouette para determinar o número ideal de clusters usando o algoritmo KMeans.
