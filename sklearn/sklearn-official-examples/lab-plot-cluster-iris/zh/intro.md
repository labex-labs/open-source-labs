# 简介

在本实验中，我们将探索 K 均值聚类算法及其在 Python 中使用 scikit-learn 库的实现。聚类是一种无监督学习，它涉及根据数据点的相似性将它们分组到不同的簇中。K 均值聚类是一种流行的聚类算法，广泛应用于图像处理、生物信息学和市场研究等各个领域。
