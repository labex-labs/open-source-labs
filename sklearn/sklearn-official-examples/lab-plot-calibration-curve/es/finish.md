# Resumen

Aprendimos cómo usar curvas de calibración para evaluar las probabilidades predichas de un modelo de clasificación. Comparamos varios clasificadores con curvas de calibración y los evaluamos con varias métricas de clasificación. También aprendimos que la calibración sigmoide paramétrica puede manejar situaciones donde la curva de calibración del clasificador base es sigmoide, pero no donde es sigmoide transpuesta. La calibración isotónica no paramétrica puede manejar ambas situaciones, pero puede requerir más datos para producir buenos resultados.
