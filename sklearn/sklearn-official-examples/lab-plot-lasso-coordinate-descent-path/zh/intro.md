# 简介

在本教程中，我们将学习套索回归（Lasso）和弹性网络（Elastic Net），它们是用于线性回归的技术，并通过坐标下降法实现。我们将学习如何使用套索回归和弹性网络计算正则化路径，以及如何使用matplotlib显示结果。
