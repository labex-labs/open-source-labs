# 简介

在本实验中，我们将学习如何使用决策树回归算法来拟合带有额外噪声观测值的正弦曲线。决策树将用于学习近似正弦曲线的局部线性回归。我们将看到，如果树的最大深度设置得太高，决策树会学习到训练数据的过于精细的细节并从噪声中学习，即它们会过拟合。
