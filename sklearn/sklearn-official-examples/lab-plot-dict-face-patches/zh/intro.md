# 简介

本实验展示了如何使用 scikit-learn API 来处理一个大型人脸数据集，并学习一组代表人脸的 20 x 20 图像块。本实验的关键在于使用在线学习，即我们一次加载并处理一张图像，并从每张图像中提取 50 个随机图像块。我们累积 500 个图像块（来自 10 张图像），然后运行在线 KMeans 对象 MiniBatchKMeans 的 partial_fit 方法。
