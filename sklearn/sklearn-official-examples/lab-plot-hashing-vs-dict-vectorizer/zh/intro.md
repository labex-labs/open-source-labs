# 简介

在本实验中，我们将探索文本向量化，即将非数值输入数据（如字典或文本文档）表示为实数向量的过程。我们将通过使用两种方法对借助自定义 Python 函数进行预处理（分词）的文本文档进行向量化，来比较 `FeatureHasher` 和 `DictVectorizer` 这两种方法。
