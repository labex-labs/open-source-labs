# 소개

이 실습은 머신 러닝에서 과소적합 (underfitting) 과 과대적합 (overfitting) 의 문제점을 보여주고, 다항식 특징을 가진 선형 회귀를 사용하여 비선형 함수를 근사하는 방법을 보여줍니다. scikit-learn 을 사용하여 데이터를 생성하고, 모델을 학습시키며, 모델 성능을 평가할 것입니다.
