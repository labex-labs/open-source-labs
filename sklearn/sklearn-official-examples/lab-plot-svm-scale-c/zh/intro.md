# 简介

本实验展示了在使用支持向量机（SVM）进行分类时缩放正则化参数的效果。在SVM分类中，我们关注以下方程的风险最小化：

```math
C \sum_{i=1, n} \mathcal{L} (f(x_i), y_i) + \Omega (w)
```

其中：

- `C` 用于设置正则化的程度
- `L` 是我们的样本和模型参数的损失函数
- `Ω` 是我们的模型参数的惩罚函数
