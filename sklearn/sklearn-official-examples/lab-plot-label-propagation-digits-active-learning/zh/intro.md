# 简介

本实验展示了一种使用标签传播来学习手写数字的主动学习技术。标签传播是一种半监督学习方法，它使用基于图的方法在数据点之间传播标签。主动学习是一个允许我们迭代选择数据点进行标注，并使用这些标注点重新训练模型的过程。
