# 简介

在本实验中，我们将使用 Python 的 scikit-learn 库对鸢尾花数据集进行主成分分析（PCA）。PCA 是一种用于降低数据集维度，同时尽可能保留其方差的技术。简单来说，它有助于识别数据集中最重要的特征，并舍弃不太重要的特征。鸢尾花数据集是机器学习领域中一个著名的数据集，包含了三种不同类型鸢尾花的物理属性信息。
