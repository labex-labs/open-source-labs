# 简介

在本实验中，我们将学习高斯混合模型（Gaussian Mixture Models，GMM），以及如何使用 Python 中的 scikit-learn 库将其用于聚类和密度估计。高斯混合模型是一种概率模型，它假设数据点是由高斯分布的混合生成的。它们是 k 均值聚类的推广，融合了有关数据协方差结构的信息。
