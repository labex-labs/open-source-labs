# 简介

在机器学习中，交叉验证是一种用于在独立数据集上评估模型性能的技术。它通过更好地估计模型对新的、未见过的数据的泛化能力，有助于防止过拟合。

在本实验中，我们将探讨交叉验证的概念，以及如何使用 Python 中的 scikit-learn 库来实现它。
