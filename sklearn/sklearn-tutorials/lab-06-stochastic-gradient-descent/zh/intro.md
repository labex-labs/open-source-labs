# 简介

随机梯度下降（Stochastic Gradient Descent，SGD）是机器学习中一种常用的优化算法。它是梯度下降算法的一种变体，在每次迭代时使用随机选择的训练数据子集。这使得它在计算上效率很高，适合处理大型数据集。在本实验中，我们将逐步介绍如何使用 scikit-learn 在 Python 中实现 SGD。
