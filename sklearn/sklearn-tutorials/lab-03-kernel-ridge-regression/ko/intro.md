# 소개

이 실습에서는 커널 릿지 회귀 (KRR) 와 Python 의 scikit-learn 라이브러리를 사용한 구현 방법을 배웁니다. KRR 은 커널 트릭을 릿지 회귀와 결합하여 커널에 의해 유도된 공간에서 선형 함수를 학습합니다. 입력과 출력 변수 간의 비선형 관계를 처리할 수 있는 비선형 회귀 방법입니다.
