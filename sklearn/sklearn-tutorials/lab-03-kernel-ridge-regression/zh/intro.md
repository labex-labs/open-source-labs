# 简介

在本实验中，我们将学习核岭回归（Kernel Ridge Regression，KRR）及其在 Python 中使用 scikit-learn 库的实现。KRR 将岭回归与核技巧相结合，以在核诱导的空间中学习线性函数。它是一种非线性回归方法，可以处理输入和输出变量之间的非线性关系。
