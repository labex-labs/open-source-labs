# Introdução

Neste laboratório, você aprenderá os fundamentos de como trabalhar com arrays NumPy. NumPy é uma biblioteca poderosa para computação numérica em Python. Ela fornece estruturas de dados e funções eficientes para realizar operações matemáticas em arrays.

> Nota: Você pode escrever código em `06-copies-and-views.ipynb`. Algumas operações de impressão são omitidas nas etapas, e você pode imprimir a saída conforme necessário.
