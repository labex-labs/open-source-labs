# Resumo (Summary)

Neste laboratório, você aprendeu os fundamentos de como trabalhar com arrays NumPy. Você aprendeu sobre cópias (copies) e visualizações (views), e como criá-las. Você também aprendeu sobre operações de indexação e outras operações que podem criar visualizações ou cópias. Finalmente, você aprendeu como determinar se um array é uma visualização ou uma cópia usando o atributo `base`.

Ao entender esses conceitos e usar os métodos apropriados, você pode manipular e analisar dados de forma eficiente usando NumPy.
