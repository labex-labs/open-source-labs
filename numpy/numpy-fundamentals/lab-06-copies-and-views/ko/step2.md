# 뷰 생성하기

뷰는 배열의 특정 메타데이터를 변경하여 생성할 수 있습니다. 이렇게 하면 데이터를 복사하지 않고 데이터를 보는 새로운 방식이 생성됩니다. 뷰를 생성하려면 `ndarray` 객체의 `view()` 메서드를 사용할 수 있습니다.

```python
import numpy as np

# 배열 생성
x = np.array([1, 2, 3, 4, 5])

# 뷰 생성
y = x.view()

# 뷰 수정
y[0] = 10

# 원본 배열 출력
print(x)  # 출력: [10, 2, 3, 4, 5]
```

위의 예제에서 뷰 `y`를 사용하면 원본 배열 `x`를 수정할 수 있습니다.
