# 인덱싱 연산

NumPy 의 인덱싱 연산은 인덱싱 유형에 따라 뷰 또는 복사본을 생성할 수 있습니다.

- 기본 인덱싱은 항상 뷰를 생성합니다. 예를 들어:

```python
import numpy as np

# 배열 생성
x = np.arange(10)

# 뷰 생성
y = x[1:3]

# 뷰 수정
y[0] = 10

# 원본 배열 출력
print(x)  # 출력: [0, 10, 2, 3, 4, 5, 6, 7, 8, 9]
```

위의 예제에서 뷰 `y`는 원본 배열 `x`에 가해진 변경 사항을 반영합니다.

- 고급 인덱싱은 항상 복사본을 생성합니다. 예를 들어:

```python
import numpy as np

# 배열 생성
x = np.arange(9).reshape(3, 3)

# 복사본 생성
y = x[[1, 2]]

# 원본 배열 수정
x[[1, 2]] = [[10, 11, 12], [13, 14, 15]]

# 복사본 출력
print(y)  # 출력: [[3, 4, 5], [6, 7, 8]]
```

위의 예제에서 복사본 `y`는 원본 배열 `x`를 수정한 후에도 변경되지 않습니다.
