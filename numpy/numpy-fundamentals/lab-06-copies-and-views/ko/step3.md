# 복사본 생성하기

복사본은 배열의 데이터 버퍼와 메타데이터를 모두 복제하여 생성할 수 있습니다. 복사본을 생성하려면 `ndarray` 객체의 `copy()` 메서드를 사용할 수 있습니다.

```python
import numpy as np

# 배열 생성
x = np.array([1, 2, 3, 4, 5])

# 복사본 생성
y = x.copy()

# 복사본 수정
y[0] = 10

# 원본 배열 출력
print(x)  # 출력: [1, 2, 3, 4, 5]
```

위의 예제에서 복사본 `y`는 원본 배열 `x`와 독립적입니다.
