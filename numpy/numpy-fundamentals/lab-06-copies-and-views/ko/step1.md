# 복사본과 뷰 이해하기

NumPy 배열은 데이터 버퍼와 메타데이터의 두 부분으로 구성됩니다. 데이터 버퍼는 실제 데이터 요소를 포함하고, 메타데이터는 데이터 유형 및 스트라이드 (strides) 와 같은 정보를 포함합니다.

NumPy 배열을 조작할 때 복사본과 뷰의 차이점을 이해하는 것이 중요합니다.

- **뷰 (view)**는 데이터 버퍼를 변경하지 않고 특정 메타데이터를 변경하여 배열에 다르게 접근할 수 있도록 합니다. 뷰에 가해진 모든 변경 사항은 원본 배열에 반영됩니다.

- **복사본 (copy)**은 데이터 버퍼와 메타데이터를 모두 복제하는 새로운 배열입니다. 복사본에 가해진 변경 사항은 원본 배열에 영향을 미치지 않습니다.
