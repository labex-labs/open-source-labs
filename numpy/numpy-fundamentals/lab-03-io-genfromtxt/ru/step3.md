# Разделение строк на столбцы

Аргумент `delimiter` используется для определения того, как строки должны быть разделены на столбцы. По умолчанию `numpy.genfromtxt` предполагает, что `delimiter=None`, что означает, что строка разделяется по пробельным символам (включая табуляции).

```python
np.genfromtxt(StringIO(data), delimiter=",")
```
