# Установка типа данных

Аргумент `dtype` используется для управления тем, как строки преобразуются в другие типы. Это может быть один тип, последовательность типов, строка, разделенная запятыми, словарь, последовательность кортежей, существующий объект `numpy.dtype` или `None`, чтобы определить тип из данных самих.

```python
np.genfromtxt(StringIO(data), dtype=float)
```
