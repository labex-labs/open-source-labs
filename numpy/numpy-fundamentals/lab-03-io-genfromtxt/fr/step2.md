# Définition de l'entrée

La fonction `numpy.genfromtxt` nécessite la source des données comme seul argument obligatoire. Cela peut être une chaîne de caractères, une liste de chaînes de caractères, un générateur ou un objet ressemblant à un fichier ouvert avec une méthode `read`.

```python
data = "1, 2, 3\n4, 5, 6"
```
