# Récapitulatif

Dans ce laboratoire, nous avons appris à importer des données à l'aide de la fonction `numpy.genfromtxt`. Nous avons exploré différentes options pour définir l'entrée, diviser les lignes en colonnes, choisir des colonnes, définir le type de données et ajuster la conversion. Nous avons également appris à propos des fonctions raccourcies fournies par le module `numpy.lib.npyio`.
