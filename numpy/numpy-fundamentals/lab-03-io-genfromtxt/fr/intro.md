# Introduction

Dans ce laboratoire, nous allons apprendre à importer des données à l'aide de la fonction `numpy.genfromtxt`. Cette fonction nous permet de lire des données tabulaires à partir de diverses sources et de les convertir en tableaux NumPy. Nous allons explorer différentes options pour définir l'entrée, diviser les lignes en colonnes, choisir des colonnes, définir le type de données et ajuster la conversion.

> Note: Vous pouvez écrire du code dans `03-io-genfromtxt.ipynb`. Certaines opérations d'impression sont omises dans les étapes, et vous pouvez imprimer la sortie selon vos besoins.
