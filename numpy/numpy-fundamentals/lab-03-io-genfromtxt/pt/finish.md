# Resumo

Neste laboratório, aprendemos como importar dados usando a função `numpy.genfromtxt`. Exploramos diferentes opções para definir a entrada, dividir as linhas em colunas, escolher colunas, definir o tipo de dados e ajustar a conversão. Também aprendemos sobre as funções de atalho fornecidas pelo módulo `numpy.lib.npyio`.
