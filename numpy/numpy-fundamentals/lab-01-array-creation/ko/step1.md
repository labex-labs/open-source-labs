# Python 시퀀스를 NumPy 배열로 변환하기

NumPy 배열을 생성하기 위해 리스트 및 튜플과 같은 Python 시퀀스를 변환할 수 있습니다. 방법은 다음과 같습니다.

```python
import numpy as np

# 리스트에서 1D 배열 생성
a1D = np.array([1, 2, 3, 4])

# 리스트의 리스트에서 2D 배열 생성
a2D = np.array([[1, 2], [3, 4]])

# 중첩된 리스트에서 3D 배열 생성
a3D = np.array([[[1, 2], [3, 4]], [[5, 6], [7, 8]]])
```

배열을 생성할 때 `dtype` 매개변수를 사용하여 데이터 유형을 지정할 수도 있습니다. 오버플로우 또는 예상치 못한 결과를 방지하기 위해 데이터 유형 할당에 주의하십시오.
