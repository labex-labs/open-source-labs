# Resumo

Neste laboratório, você aprendeu como criar arrays usando NumPy. Você aprendeu diferentes métodos, incluindo a conversão de sequências Python, o uso de funções intrínsecas de criação de arrays NumPy, a replicação e junção de arrays existentes, a leitura de arrays do disco, a criação de arrays a partir de bytes brutos e o uso de funções especiais de bibliotecas. Com essas técnicas, você pode criar e manipular arrays de forma eficiente para várias tarefas de análise de dados e científicas.
