# Einführung

Dieses Lab bietet einen schrittweisen Leitfaden zur Erstellung von Arrays mit NumPy, einer grundlegenden Bibliothek für Array-Container in Python. Sie werden verschiedene Methoden zur Array-Erstellung lernen, darunter das Konvertieren von Python-Sequenzen, das Verwenden von eingebauten NumPy-Array-Erstellungsfunktionen, das Replizieren und Verbinden von vorhandenen Arrays, das Einlesen von Arrays von der Festplatte, das Erstellen von Arrays aus rohen Bytes und das Verwenden spezieller Bibliotheksfunktionen.

> Hinweis: Sie können im `01-array-creation.ipynb` Code schreiben. Einige Druckvorgänge werden in den Schritten weggelassen, und Sie können die Ausgabe nach Bedarf drucken.
