# Introdução

O broadcasting (difusão) é um recurso poderoso no NumPy que permite que arrays com diferentes formatos sejam usados em operações aritméticas. Ele oferece uma maneira de vetorizar operações de array e melhorar a eficiência computacional. Este laboratório irá guiá-lo pelos fundamentos do broadcasting no NumPy.

> Nota: Você pode escrever código em `05-broadcasting.ipynb`. Algumas operações de impressão são omitidas nas etapas, e você pode imprimir a saída conforme necessário.
