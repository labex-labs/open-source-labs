# Resumo

Neste laboratório, aprendemos sobre os fundamentos das Funções Universais (ufuncs) do NumPy. Exploramos os diferentes métodos das _ufuncs_, as regras de _broadcasting_, as regras de _type casting_ e como substituir o comportamento das _ufuncs_. As _ufuncs_ são uma ferramenta poderosa para realizar operações elemento a elemento em _arrays_ de forma eficiente.
