# Введение

В этом практическом занятии будет приведен пошаговый гайд по пониманию различных типов данных, доступных в NumPy, и по изменению типа данных массива. NumPy поддерживает широкий спектр числовых типов, включая логические, целочисленные, вещественные и комплексные числа. Понимание этих типов данных важно для выполнения различных численных вычислений и задач анализа данных с использованием NumPy.

> Примечание: Вы можете писать код в `04-data-types.ipynb`. В некоторых шагах опущены некоторые операции вывода, и вы можете выводить результат по необходимости.
