# Получение типа данных массива

Для определения типа данных массива вы можете получить доступ к атрибуту `dtype`. Например:

```python
z.dtype
# возвращает тип данных массива z, который является uint8
```

Объект `dtype` также содержит информацию о типе, такой как его битовая ширина и порядок байт. Вы можете использовать объект `dtype` для запроса свойств типа, например, того, является ли он целым числом. Например:

```python
d = np.dtype(int)
# создает объект dtype для int

np.issubdtype(d, np.integer)
# возвращает True, что указывает на то, что d является подтипом np.integer

np.issubdtype(d, np.floating)
# возвращает False, что указывает на то, что d не является подтипом np.floating
```
