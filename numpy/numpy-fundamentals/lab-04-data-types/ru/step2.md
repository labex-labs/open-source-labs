# Работа с типами данных

Типы данных NumPy представляются в виде объектов `dtype` (тип данных). После импорта NumPy с использованием `import numpy as np` вы можете получить доступ к типам данных с использованием `np.bool_`, `np.float32` и т.д.

Вы можете использовать типы данных в качестве функций для преобразования чисел Python в массивные скаляры, последовательностей чисел Python в массивы данного типа или в качестве аргументов для ключевого слова dtype в многих функциях или методах NumPy. Вот несколько примеров:

```python
x = np.float32(1.0)
# x теперь является массивным скаляром с типом float32 и значением 1.0

y = np.int_([1,2,4])
# y теперь является массивом целых чисел с значениями [1, 2, 4]

z = np.arange(3, dtype=np.uint8)
# z теперь является массивом с типом uint8 и значениями [0, 1, 2]
```

Вы также можете ссылаться на типы массивов с использованием кодов символов, хотя рекомендуется использовать объекты dtype вместо этого. Например:

```python
np.array([1, 2, 3], dtype='f')
# возвращает массив с значениями [1., 2., 3.] и типом данных float32
```

Для преобразования типа массива вы можете использовать метод `.astype()` или тип сам в качестве функции. Например:

```python
z.astype(float)
# возвращает массив z с типом данных float64

np.int8(z)
# возвращает массив z с типом данных int8
```
