# Резюме

В этом практическом занятии вы узнали о различных числовых типах данных, доступных в NumPy, и о том, как с ними работать. Вы можете использовать объекты `dtype` для преобразования чисел Python в массивные скаляры, создания массивов определенных типов и изменения типа данных массива. Понимание типов данных важно для выполнения числовых вычислений и задач анализа данных с использованием NumPy.
