# 소개

이 랩에서는 NumPy 에서 사용할 수 있는 다양한 데이터 유형과 배열의 데이터 유형을 수정하는 방법에 대한 단계별 가이드를 제공합니다. NumPy 는 부울, 정수, 부동 소수점 숫자 및 복소수를 포함한 광범위한 숫자 유형을 지원합니다. 이러한 데이터 유형을 이해하는 것은 NumPy 를 사용하여 다양한 숫자 계산 및 데이터 분석 작업을 수행하는 데 중요합니다.

> 참고: `04-data-types.ipynb`에서 코드를 작성할 수 있습니다. 일부 인쇄 작업은 단계에서 생략되었으며, 필요에 따라 출력을 인쇄할 수 있습니다.
