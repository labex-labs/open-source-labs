# Resumo

Neste laboratório, você aprendeu sobre os diferentes tipos de dados numéricos disponíveis no NumPy e como trabalhar com eles. Você pode usar objetos `dtype` para converter números Python em escalares de array, criar arrays de tipos específicos e modificar o tipo de dados de um array. Compreender os tipos de dados é importante para realizar cálculos numéricos e tarefas de análise de dados usando NumPy.
