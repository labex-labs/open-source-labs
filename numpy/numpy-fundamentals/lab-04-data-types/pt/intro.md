# Introdução

Este laboratório fornecerá um guia passo a passo para entender os diferentes tipos de dados disponíveis no NumPy e como modificar o tipo de dados de um array. NumPy suporta uma ampla gama de tipos numéricos, incluindo booleanos, inteiros, números de ponto flutuante e números complexos. Compreender esses tipos de dados é importante para realizar várias computações numéricas e tarefas de análise de dados usando NumPy.

> Nota: Você pode escrever código em `04-data-types.ipynb`. Algumas operações de impressão são omitidas nas etapas, e você pode imprimir a saída conforme necessário.
