# Сравнение структурированных массивов

Если типы данных двух структурированных массивов совпадают, мы можем сравнить их с использованием оператора равенства (`==`). Это вернет булевский массив, указывающий, какие элементы имеют одинаковые значения для всех полей.

```python
# Сравнить два структурированных массива
y = np.array([('Alice', 25), ('Bob', 30)], dtype=[('name', 'U10'), ('age', int)])
comparison = x == y
```
