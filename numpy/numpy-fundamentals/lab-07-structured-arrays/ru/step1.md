# Создание структурированного массива

Для создания структурированного массива мы можем использовать функцию `np.array` и указать тип данных с помощью параметра `dtype`. Тип данных должен быть списком кортежей, где каждый кортеж представляет поле в структурированном массиве. Каждый кортеж должен содержать имя поля и тип данных этого поля.

```python
import numpy as np

# Создать структурированный массив
x = np.array([('Alice', 25), ('Bob', 30)], dtype=[('name', 'U10'), ('age', int)])
```
