# Создание записного массива

Записной массив - это подкласс ndarray, который позволяет получать доступ к полям по атрибуту, а не по индексу. Мы можем создать записной массив с использованием функции `np.rec.array`.

```python
# Создать записной массив
recordarr = np.rec.array([('Alice', 25), ('Bob', 30)], dtype=[('name', 'U10'), ('age', int)])
```
