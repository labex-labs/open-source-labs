# Преобразование записного массива в структурированный массив

Для преобразования записного массива обратно в структурированный массив мы можем использовать метод `view` и указать исходный тип данных (`dtype`) структурированного массива.

```python
# Преобразовать записной массив в структурированный массив
x = recordarr.view(dtype=[('name', 'U10'), ('age', int)])
```
