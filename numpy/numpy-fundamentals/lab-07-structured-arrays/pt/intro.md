# Introdução

Neste laboratório, aprenderemos sobre arrays estruturados em NumPy. Arrays estruturados são ndarrays cujo tipo de dado (datatype) é uma composição de tipos de dados mais simples, organizados como uma sequência de campos nomeados. Eles são úteis para trabalhar com dados estruturados, como dados tabulares, onde cada campo representa um atributo diferente dos dados.

> Nota: Você pode escrever código em `07-structured-arrays.ipynb`. Algumas operações de impressão são omitidas nos passos, e você pode imprimir a saída conforme necessário.
