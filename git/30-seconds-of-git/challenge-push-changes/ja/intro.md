# はじめに

<p>🧑‍💻 Git や LabEx の初心者ですか？ 「<b><a style="color: unset;text-decoration: underline;" href="https://labex.io/courses/quick-start-with-git" target="_blank">Git クイックスタート</a></b>」コースから始めることをおすすめします。</p>

Git は、開発者がコードベースの変更を追跡できる人気のあるバージョン管理システムです。Git の主な機能の 1 つは、ローカルの変更をリモートリポジトリにプッシュする機能です。これにより、複数の開発者が同じコードベースで共同作業し、最新状態を保つことができます。
