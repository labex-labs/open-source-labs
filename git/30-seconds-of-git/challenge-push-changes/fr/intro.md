Git est un système de contrôle de versions populaire qui permet aux développeurs de suivre les modifications dans leur base de code. L'une des principales fonctionnalités de Git est la capacité de pousser les modifications locales vers un référentiel distant. Cela permet à plusieurs développeurs de collaborer sur la même base de code et de la maintenir à jour.
