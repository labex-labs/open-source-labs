Git ist ein beliebiges Versionsverwaltungssystem, das es Entwicklern ermöglicht, Änderungen in ihrer Codebasis zu verfolgen. Ein Schlüsselmerkmal von Git ist die Möglichkeit, lokale Änderungen in ein Remote-Repository zu pushen. Dadurch können mehrere Entwickler an der gleichen Codebasis zusammenarbeiten und sie aktuell halten.
