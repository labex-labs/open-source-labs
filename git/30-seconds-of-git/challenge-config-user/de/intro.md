Git ist ein beliebiges Versionsverwaltungssystem, das von Entwicklern verwendet wird, um ihren Code zu verwalten. Wenn Sie Änderungen an einem Repository vornehmen, verfolgt Git, wer diese Änderungen vorgenommen hat. Dazu muss Git Ihre Benutzereinformationen kennen, wie Ihren Namen und Ihre E-Mail-Adresse. In dieser Herausforderung lernen Sie, wie Sie Ihre Benutzereinformationen für Git konfigurieren.
