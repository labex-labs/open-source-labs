Git es un popular sistema de control de versiones utilizado por los desarrolladores para administrar su código. Cuando realizas cambios en un repositorio, Git registra quién realizó esos cambios. Para hacer esto, Git necesita conocer tu información de usuario, como tu nombre y dirección de correo electrónico. En este desafío, aprenderás cómo configurar tu información de usuario para Git.
