No Git, um _branch_ (ramificação) é um ponteiro leve e móvel para um _commit_. Renomear um _branch_ é uma tarefa comum no Git. No entanto, renomear um _branch_ tanto localmente quanto no repositório remoto pode ser um pouco complicado. Neste laboratório, você aprenderá como renomear um _branch_ remoto no Git.
