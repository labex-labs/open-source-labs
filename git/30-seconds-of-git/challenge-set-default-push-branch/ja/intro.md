Git を使用する際、ローカルの変更をリモート リポジトリにプッシュすることが一般的です。既定では、Git は変更をローカル ブランチと同じ名前のブランチにプッシュします。ただし、場合によっては、別のブランチに変更をプッシュしたい場合もあります。このチャレンジでは、既定のプッシュ ブランチ名を現在のローカル ブランチ名に設定する方法を学びます。
