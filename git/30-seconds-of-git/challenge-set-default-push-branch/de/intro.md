Wenn Sie mit Git arbeiten, ist es üblich, Ihre lokalen Änderungen in ein Remote-Repository zu pushen. Standardmäßig wird Git Ihre Änderungen in einen Branch mit demselben Namen wie Ihr lokaler Branch pushen. Manchmal möchten Sie jedoch Ihre Änderungen in einen anderen Branch pushen. In dieser Herausforderung lernen Sie, wie Sie den Standard-Push-Branch-Namen auf den Namen des aktuellen lokalen Branches festlegen.
