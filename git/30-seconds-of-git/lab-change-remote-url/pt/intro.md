No Git, um _remote_ (remoto) é uma versão do seu repositório que está hospedada em um servidor ou outro computador. Quando você clona um repositório, o Git cria automaticamente um _remote_ chamado `origin` que aponta para o repositório original. No entanto, às vezes você pode precisar alterar a URL do repositório remoto. Este laboratório testará sua capacidade de alterar a URL remota usando comandos Git.
