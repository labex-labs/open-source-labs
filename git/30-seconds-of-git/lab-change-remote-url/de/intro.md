In Git ist ein Remote eine Version Ihres Repositorys, das auf einem Server oder einem anderen Computer gehostet wird. Wenn Sie ein Repository klonen, erstellt Git automatisch einen Remote namens `origin`, der auf das ursprüngliche Repository zeigt. Manchmal müssen Sie jedoch die URL des Remote-Repositorys ändern. In diesem Lab werden Ihre Fähigkeiten getestet, um die Remote-URL mit Git-Befehlen zu ändern.
