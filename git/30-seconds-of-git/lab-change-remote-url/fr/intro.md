En Git, un dépôt distant est une version de votre référentiel hébergée sur un serveur ou un autre ordinateur. Lorsque vous clonez un référentiel, Git crée automatiquement un dépôt distant appelé `origin` qui pointe vers le référentiel original. Cependant, parfois vous devrez peut-être modifier l'URL du référentiel distant. Ce laboratoire vous permettra de tester votre capacité à modifier l'URL du dépôt distant à l'aide de commandes Git.
