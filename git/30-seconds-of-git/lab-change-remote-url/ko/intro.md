Git 에서 remote 는 서버 또는 다른 컴퓨터에 호스팅되는 리포지토리의 버전입니다. 리포지토리를 clone 하면 Git 은 자동으로 `origin`이라는 remote 를 생성하여 원래 리포지토리를 가리킵니다. 하지만 때로는 remote 리포지토리의 URL 을 변경해야 할 수 있습니다. 이 랩에서는 Git 명령어를 사용하여 remote URL 을 변경하는 능력을 테스트합니다.
