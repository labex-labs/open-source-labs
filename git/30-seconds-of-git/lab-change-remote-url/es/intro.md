En Git, un repositorio remoto es una versión de tu repositorio que se hospeda en un servidor o en otro computador. Cuando clonas un repositorio, Git automáticamente crea un repositorio remoto llamado `origin` que apunta al repositorio original. Sin embargo, a veces es posible que necesites cambiar la URL del repositorio remoto. Esta práctica te ayudará a probar tu habilidad para cambiar la URL remota usando comandos de Git.
