Git を使用する際、作業ディレクトリにトラックされていない変更があるのは一般的です。これらの変更は、Git が現在トラッキングしていないファイルです。時々、これらの変更を破棄して新しく始めたい場合があります。この実験では、現在のブランチのすべてのトラックされていない変更を破棄する方法を学びます。
