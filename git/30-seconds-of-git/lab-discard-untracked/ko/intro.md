Git 을 사용할 때 작업 디렉토리에 추적되지 않은 변경 사항이 있는 것은 흔한 일입니다. 이러한 변경 사항은 Git 이 현재 추적하지 않는 파일입니다. 때로는 이러한 변경 사항을 버리고 처음부터 다시 시작하고 싶을 수 있습니다. 이 랩에서는 현재 브랜치에 대한 추적되지 않은 모든 변경 사항을 폐기하는 방법을 배우게 됩니다.
