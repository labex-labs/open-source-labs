Git 은 개발자가 코드베이스를 효율적으로 관리할 수 있도록 해주는 강력한 버전 관리 시스템입니다. Git 의 가장 유용한 기능 중 하나는 대화형 리베이스 (interactive rebase) 로, 개발자가 브랜치의 커밋 기록을 수정할 수 있게 해줍니다. 이 챌린지에서는 Git 을 사용하여 대화형 리베이스를 수행하는 방법을 배우게 됩니다.
