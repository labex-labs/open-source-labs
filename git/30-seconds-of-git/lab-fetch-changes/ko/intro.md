Git 은 코드 변경 사항을 추적하는 데 사용되는 널리 사용되는 버전 관리 시스템입니다. Git 의 주요 기능 중 하나는 원격 저장소 (remote repositories) 로 작업할 수 있는 기능입니다. 이 랩에서는 Git 을 사용하여 원격 저장소에서 최신 변경 사항을 가져오는 방법을 배우게 됩니다.
