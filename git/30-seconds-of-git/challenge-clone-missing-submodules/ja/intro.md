Git では、サブモジュールはリポジトリ内のリポジトリです。これにより、他のリポジトリをプロジェクトの依存関係として含めることができます。サブモジュールを含む Git リポジトリをクローンするとき、サブモジュールは自動的にクローンされません。これは、プロジェクトをビルドまたは実行しようとするときに問題を引き起こす可能性があります。このチャレンジでは、欠落しているサブモジュールをクローンしてコミットをチェックアウトする方法を学びます。
