Git é um sistema de controle de versão (version control system) popular que permite aos desenvolvedores rastrear as alterações em seu código-base. O arquivo de configuração do Git é um arquivo que contém configurações que afetam o comportamento do Git. Neste laboratório, você aprenderá como editar o arquivo de configuração do Git usando o editor de texto do Git.
