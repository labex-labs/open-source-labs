Git 을 사용할 때, 로컬 브랜치가 원격 브랜치와 동기화되지 않는 경우가 흔히 발생합니다. 이는 원격 브랜치에 변경 사항이 생겼지만 로컬 브랜치에 반영되지 않은 경우에 발생할 수 있습니다. 이러한 경우, 로컬 브랜치를 원격 브랜치와 일치하도록 재설정해야 합니다. 이 랩에서는 로컬 `master` 브랜치를 원격 브랜치와 일치하도록 재설정하는 단계를 안내합니다.
