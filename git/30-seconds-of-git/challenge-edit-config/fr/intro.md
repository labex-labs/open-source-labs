Git est un système de contrôle de versions populaire qui permet aux développeurs de suivre les modifications dans leur base de code. Le fichier de configuration Git est un fichier qui contient des paramètres qui affectent le comportement de Git. Dans ce défi, vous allez apprendre à éditer le fichier de configuration Git à l'aide de l'éditeur de texte Git.
