Git es un popular sistema de control de versiones que permite a los desarrolladores rastrear los cambios en su repositorio de código. El archivo de configuración de Git es un archivo que contiene configuraciones que afectan el comportamiento de Git. En este desafío, aprenderás cómo editar el archivo de configuración de Git utilizando el editor de texto de Git.
