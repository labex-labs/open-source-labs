Git ist ein beliebtes Versionsverwaltungssystem, das Entwicklern ermöglicht, Änderungen in ihrer Codebasis zu verfolgen. Die Git-Konfigurationsdatei ist eine Datei, die Einstellungen enthält, die das Verhalten von Git beeinflussen. In dieser Herausforderung lernen Sie, wie Sie die Git-Konfigurationsdatei mit dem Git-Texteditor bearbeiten.
