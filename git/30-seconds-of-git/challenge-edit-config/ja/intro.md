Git は、開発者がコードベースの変更を追跡できる人気のあるバージョン管理システムです。Git 設定ファイルは、Git の動作に影響を与える設定を含むファイルです。このチャレンジでは、Git のテキストエディタを使用して Git 設定ファイルを編集する方法を学びます。
