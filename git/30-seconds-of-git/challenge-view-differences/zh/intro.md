Git 是一种流行的版本控制系统，它允许开发者跟踪对其代码库所做的更改。Git 的一项基本功能是能够查看对文件所做更改之间的差异。当你想要查看对文件所做的更改或比较文件不同版本之间的更改时，此功能非常有用。
