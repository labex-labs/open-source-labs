Git 을 사용할 때, 레포지토리 (repository) 내에 여러 개의 브랜치 (branch) 를 갖는 것은 흔한 일입니다. 하지만 모든 브랜치와 마지막 커밋 (commit) 날짜를 추적하는 것은 어려울 수 있습니다. 이 랩 (lab) 에서는 Git 을 사용하여 브랜치를 날짜별로 정렬하는 방법을 배우고, 이를 통해 레포지토리를 더 쉽게 관리할 수 있게 됩니다.
