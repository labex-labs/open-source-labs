Cuando se trabaja con Git, es común tener múltiples ramas en un repositorio. Sin embargo, puede ser difícil mantener un seguimiento de todas las ramas y sus fechas de último commit. En este laboratorio, aprenderás a usar Git para ordenar las ramas por fecha, lo que facilita la gestión de tu repositorio.
