Wenn Sie mit Git arbeiten, ist es üblich, in einem Repository mehrere Branches zu haben. Es kann jedoch schwierig sein, alle Branches und ihre letzten Commit-Daten im Auge zu behalten. In diesem Lab werden Sie lernen, wie Sie Git verwenden, um Branches nach Datum zu sortieren, was es einfacher macht, Ihr Repository zu verwalten.
