Wenn Sie mit Git arbeiten, ist es üblich, detached branches zu erstellen und umzuschalten. Diese Branches sind nicht mit einem bestimmten Branch assoziiert und werden normalerweise für Tests oder Experimente verwendet. Mit der Zeit können sich jedoch diese Branches anhäufen und Ihren Repository verschmutzen. In dieser Herausforderung lernen Sie, wie Sie alle detached branches in Ihrem Git-Repository löschen.
