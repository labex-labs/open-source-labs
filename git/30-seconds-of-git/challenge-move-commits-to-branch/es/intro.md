Git es un sistema de control de versiones poderoso que permite a los desarrolladores llevar un registro de los cambios realizados en su repositorio de código. Una de las características útiles de Git es la capacidad de mover commits de una rama a otra. Esto puede ser útil cuando te das cuenta de que algunos cambios que hiciste en la rama `master` deberían haber sido hechos en una rama separada. En este desafío, aprenderás cómo mover commits de la rama `master` a una nueva rama.
