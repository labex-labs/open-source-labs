Git 是一个强大的版本控制系统，它允许开发者跟踪对其代码库所做的更改。Git 的一个实用功能是能够将提交从一个分支移动到另一个分支。当你意识到对 `master` 分支所做的某些更改本应在一个单独的分支上进行时，这会很有帮助。在这个挑战中，你将学习如何将提交从 `master` 分支移动到一个新分支。

这是一个挑战（Challenge），与实验（Lab）不同，你需要独立完成挑战任务，而不是按照实验的步骤学习。挑战通常有一点难度。如果你觉得困难，可以与 Labby 讨论或查看解决方案。
