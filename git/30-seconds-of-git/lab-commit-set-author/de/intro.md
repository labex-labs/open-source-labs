Git ist ein beliebiges Versionsverwaltungssystem, das von Entwicklern verwendet wird, um ihre Codebasis zu verwalten. Eine der wesentlichen Funktionen von Git ist die Möglichkeit, Commits zu erstellen, die Momentaufnahmen des Codes zu einem bestimmten Zeitpunkt sind. Standardmäßig verwendet Git den Benutzernamen und die E-Mail-Adresse des Benutzers, um den Autor eines Commits zu identifizieren. Es können jedoch Situationen auftreten, in denen Sie einen Commit von einem anderen Autor erstellen müssen. In dieser Herausforderung lernen Sie, wie Sie in Git einen Commit von einem anderen Autor erstellen.
