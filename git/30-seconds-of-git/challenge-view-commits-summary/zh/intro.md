Git 是一个强大的版本控制系统，它允许开发者随时间追踪对其代码库所做的更改。Git 最有用的特性之一是能够查看对一个仓库所做的所有提交的摘要。当试图了解一个项目的历史或确定何时进行了特定更改时，这会很有帮助。
