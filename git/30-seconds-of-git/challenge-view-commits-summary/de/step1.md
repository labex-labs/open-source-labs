# Zeigen Sie einen kurzen Commit-Zusammenfassung an

Als Entwickler arbeiten Sie an einem Projekt mit mehreren Mitwirkenden. Sie müssen eine Zusammenfassung aller Commits für das Projekt anzeigen, um die vorgenommenen Änderungen zu verstehen und potenzielle Probleme zu identifizieren. möchten jedoch nicht viel Zeit damit verbringen, durch alle Commit-Nachrichten zu durchsuchen, um die benötigten Informationen zu finden.

## Aufgaben

Zeigen Sie eine kurze Zusammenfassung aller Commits in einem Git-Repository an. Angenommen, Sie arbeiten an einem Projekt auf GitHub namens `git-playground`.

1. Nachdem Sie das Repository geklont haben, navigieren Sie zum Projektverzeichnis und zeigen Sie eine kurze Zusammenfassung aller Commits an.

Dies wird eine Liste aller Commits für das Repository ausgeben, zusammen mit einer kurzen Zusammenfassung jeder Commit-Nachricht. Beispielsweise:

```shell
d22f46b (HEAD -> master, origin/master, origin/feature-branch, origin/HEAD) Added file2.txt
cf80005 Added file1.txt
b00b937 Initial commit
```
