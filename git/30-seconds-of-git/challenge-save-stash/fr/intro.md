Git est un puissant système de contrôle de versions qui permet aux développeurs de suivre les modifications apportées à leur base de code. L'une des fonctionnalités utiles de Git est la possibilité de créer un cache. Un cache vous permet de sauvegarder l'état actuel de votre répertoire de travail et de votre index, de sorte que vous pouvez basculer sur une autre branche ou travailler sur une autre fonctionnalité sans perdre vos modifications.
