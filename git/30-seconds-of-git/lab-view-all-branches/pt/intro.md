No Git, as branches (ramificações) são usadas para desenvolver funcionalidades ou isolar as alterações feitas no codebase (código base). As branches locais (local branches) são as branches que existem apenas na sua máquina local. Neste desafio, você aprenderá como visualizar uma lista de todas as branches locais em um repositório Git.
