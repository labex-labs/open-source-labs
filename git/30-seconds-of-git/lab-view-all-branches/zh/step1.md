# 查看本地分支

你一直在处理一个名为 `https://github.com/labex-labs/git-playground` 的 Git 仓库。你想要查看该仓库中所有本地分支的列表。

1. 使用以下命令导航到克隆的仓库：
   ```
   cd git-playground
   ```
2. 使用以下命令查看所有本地分支的列表：
   ```
   git branch
   ```
   你将看到仓库中所有本地分支的列表。使用箭头键浏览列表，然后按 <kbd>Q</kbd> 退出。
