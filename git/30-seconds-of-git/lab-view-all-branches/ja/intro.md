Git では、ブランチを使用して機能を開発したり、コードベースに対して行われた変更を分離したりします。 ローカルブランチは、ローカルマシン上にのみ存在するブランチです。 このチャレンジでは、Git リポジトリ内のすべてのローカルブランチの一覧を表示する方法を学びます。
