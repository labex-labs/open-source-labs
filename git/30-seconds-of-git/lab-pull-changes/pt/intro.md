Git é um sistema de controle de versão (version control system) popular que permite aos desenvolvedores colaborar em projetos e acompanhar as alterações feitas no código-base (codebase). Uma das características mais importantes do Git é a capacidade de puxar (pull) as últimas alterações de um repositório remoto. Neste laboratório, você aprenderá como puxar as últimas alterações de um repositório remoto usando o Git.
