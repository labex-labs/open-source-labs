Git 에서 stash 는 아직 커밋할 준비가 되지 않은 변경 사항을 저장하는 방법입니다. 변경 사항을 잃지 않고 임시로 작업을 저장하고 다른 브랜치 또는 커밋으로 전환할 수 있습니다. 변경 사항 작업을 계속할 준비가 되면 stash 를 작업 디렉토리에 적용할 수 있습니다. 이 랩에서는 최신 stash 를 Git 저장소에 적용하는 방법을 배우게 됩니다.
