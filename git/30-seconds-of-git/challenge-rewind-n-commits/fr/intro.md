Git est un outil puissant qui permet aux développeurs de suivre les modifications de leur base de code. L'une des fonctionnalités utiles de Git est la capacité de remonter les commits. Cela peut être utile lorsque vous avez besoin d'annuler des modifications ou de revenir à une version précédente de votre code.
