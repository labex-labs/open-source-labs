# Просмотр визуального графа репозитория

В качестве разработчика вы, возможно, захотите просмотреть историю репозитория, чтобы понять, как код меняется с течением времени. Однако просто просмотр списка коммитов может быть подавляющим и трудно понять. Именно здесь на помощь приходит граф Git.可视化 репозитория, вы можете быстро увидеть, как код развивался, и выявить любые проблемы или ошибки, которые могли быть введены.

## Задачи

Чтобы просмотреть визуальный граф Git-репозитория, скажем, вы хотите просмотреть историю репозитория `git-playground` на GitHub.

1. После клонирования репозитория вы можете перейти в директорию и просмотреть граф.
2. Отобразить визуальный граф всех коммитов и ветвей в репозитории, позволяя вам увидеть, как код развивался с течением времени.

Вот конечный результат:

```
* d22f46ba8c2d4e07d773c5126e9c803933eb5898 (HEAD -> master, origin/master, origin/feature-branch, origin/HEAD) Added file2.txt
* cf80005e40a3c661eb212fcea5fad06f8283f08f Added file1.txt
* b00b9374a7c549d1af111aa777fdcc868d8a2a01 Initial commit
```
