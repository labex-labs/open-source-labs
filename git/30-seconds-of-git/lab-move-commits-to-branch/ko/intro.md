Git 은 개발자가 코드베이스에 적용된 변경 사항을 추적할 수 있도록 하는 강력한 버전 관리 시스템입니다. Git 의 유용한 기능 중 하나는 커밋을 한 브랜치에서 다른 브랜치로 이동하는 기능입니다. 이는 `master` 브랜치에 적용한 일부 변경 사항이 별도의 브랜치에서 이루어졌어야 한다는 것을 깨달았을 때 유용할 수 있습니다. 이 랩에서는 `master` 브랜치에서 새로운 브랜치로 커밋을 이동하는 방법을 배우게 됩니다.
