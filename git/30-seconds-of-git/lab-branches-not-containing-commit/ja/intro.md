Git は、開発者がコードベースの変更を追跡できる人気のあるバージョン管理システムです。Git の便利な機能の 1 つは、ブランチを作成できることです。ブランチは、基本的に独立して変更できるコードベースの別個のコピーです。ただし、特定のコミットが含まれているブランチを追跡するのは難しい場合があります。この実験では、特定のコミットを含まないブランチを見つける方法を学びます。
