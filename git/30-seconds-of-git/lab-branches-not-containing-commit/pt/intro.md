Git é um sistema de controle de versão popular que permite aos desenvolvedores rastrear as alterações em seu código-base. Uma das funcionalidades úteis do Git é a capacidade de criar branches (branches), que são essencialmente cópias separadas do código-base que podem ser modificadas independentemente. No entanto, às vezes pode ser difícil acompanhar quais branches contêm certos commits. Neste laboratório, você aprenderá como encontrar branches que não contêm um commit específico.
