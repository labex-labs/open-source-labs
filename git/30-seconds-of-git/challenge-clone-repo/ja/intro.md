Git は、開発者がコードの変更を追跡し、他の人と協力できるバージョン管理システムです。Git で最も一般的なタスクの 1 つは、既存のリポジトリのローカルコピーを作成するリポジトリのクローン化です。このチャレンジでは、Git を使用してリポジトリをクローンする方法を学びます。
