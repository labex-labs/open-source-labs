Git ist ein Versionsverwaltungssystem, das es Entwicklern ermöglicht, Änderungen in ihrem Code zu verfolgen und mit anderen zusammenzuarbeiten. Eine der häufigsten Aufgaben in Git ist das Klonen eines Repositorys, das eine lokale Kopie eines bestehenden Repositorys erstellt. In dieser Herausforderung lernst du, wie du ein Repository mit Git klonst.
