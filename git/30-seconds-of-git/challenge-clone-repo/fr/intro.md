Git est un système de contrôle de versions qui permet aux développeurs de suivre les modifications de leur code et de collaborer avec d'autres. L'une des tâches les plus courantes dans Git est le clonage d'un référentiel, qui crée une copie locale d'un référentiel existant. Dans ce défi, vous allez apprendre à cloner un référentiel à l'aide de Git.
