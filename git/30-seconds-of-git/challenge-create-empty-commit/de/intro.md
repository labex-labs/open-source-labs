In Git ist ein Commit ein Momentaufnahme der Änderungen an einem Repository. Jeder Commit hat eine Nachricht, die die vorgenommenen Änderungen beschreibt. Manchmal müssen Sie einen leeren Commit ohne Änderungen erstellen, um als Platzhalter zu dienen oder um einen Build-Prozess auszulösen. In dieser Herausforderung lernen Sie, wie Sie in Git einen leeren Commit erstellen.
