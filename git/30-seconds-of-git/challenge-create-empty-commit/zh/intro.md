在 Git 中，提交（commit）是对仓库所做更改的一个快照。每个提交都有一条描述所做更改的消息。有时，你可能需要创建一个没有更改的空提交，用作占位符或触发构建过程。在这个挑战中，你将学习如何在 Git 中创建空提交。
