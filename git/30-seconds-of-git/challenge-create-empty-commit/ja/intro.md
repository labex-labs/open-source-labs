Git では、コミットはリポジトリに対して行われた変更のスナップショットです。各コミットには、行われた変更を説明するメッセージがあります。時々、ダミーのコミット（変更がないコミット）を作成して、プレースホルダーとして使用したり、ビルドプロセスをトリガーしたりする必要がある場合があります。このチャレンジでは、Git で空のコミットを作成する方法を学びます。
