Git est un outil puissant de contrôle de versions, permettant aux développeurs de suivre les modifications apportées à leur code au fil du temps. L'une des fonctionnalités les plus utiles de Git est la possibilité de visualiser l'historique d'un référentiel à l'aide d'un graphe. Ce graphe montre tous les commits et les branches du référentiel, facilitant la compréhension de l'évolution du code au fil du temps.
