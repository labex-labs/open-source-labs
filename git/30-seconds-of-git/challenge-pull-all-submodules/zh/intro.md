在使用 Git 仓库时，可以使用子模块来包含其他仓库作为依赖项。然而，当这些依赖项在各自的远程仓库中更新时，更改不会自动反映在主仓库中。为了更新子模块，你需要从它们各自的远程仓库中拉取。
