Git est un puissant système de contrôle de versions qui permet aux développeurs de suivre les modifications apportées à leur base de code. L'une des fonctionnalités les plus utiles de Git est la possibilité d'annuler les modifications qui ont été validées. Dans ce défi, vous allez apprendre à annuler une validation à l'aide de Git.
