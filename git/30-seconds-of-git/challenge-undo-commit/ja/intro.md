Git は、開発者がコードベースの変更を追跡できる強力なバージョン管理システムです。Git の最も便利な機能の 1 つは、コミットされた変更を元に戻す機能です。このチャレンジでは、Git を使ってコミットを元に戻す方法を学びます。
