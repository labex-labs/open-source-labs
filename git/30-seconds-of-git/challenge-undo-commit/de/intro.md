Git ist ein leistungsstarkes Versionsverwaltungssystem, das es Entwicklern ermöglicht, Änderungen an ihrer Codebasis zu verfolgen. Eine der nützlichsten Funktionen von Git ist die Möglichkeit, bereits committete Änderungen rückgängig zu machen. In dieser Herausforderung lernen Sie, wie Sie einen Commit mit Git rückgängig machen.
