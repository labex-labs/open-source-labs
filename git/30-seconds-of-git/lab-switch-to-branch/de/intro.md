In Git werden Branches verwendet, um verschiedene Versionen eines Projekts zu erstellen. Das Umschalten zwischen Branches ist eine häufige Aufgabe, wenn man mit Git arbeitet. In diesem Lab lernst du, wie du zu einem vorhandenen Branch in einem Git-Repository wechselst.
