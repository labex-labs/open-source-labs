Git es un sistema de control de versiones popular que permite a los desarrolladores rastrear los cambios realizados en su repositorio de código. Una de las características esenciales de Git es el área de preparación, que permite a los desarrolladores elegir de manera selectiva qué cambios se van a confirmar. En este laboratorio, aprenderás cómo agregar archivos al área de preparación utilizando Git.
