Git 은 개발자가 코드베이스에 적용된 변경 사항을 추적할 수 있도록 하는 널리 사용되는 버전 관리 시스템입니다. Git 의 필수 기능 중 하나는 개발자가 커밋할 변경 사항을 선택적으로 선택할 수 있도록 하는 스테이징 영역 (staging area) 입니다. 이 랩에서는 Git 을 사용하여 파일을 스테이징 영역에 추가하는 방법을 배우게 됩니다.
