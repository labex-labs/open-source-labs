Git を使用する際には、リポジトリに対して行われたすべてのコミットの要約を表示できることが重要です。ただし、マージコミットが出力を混乱させ、実際に行われた変更を確認しにくくなることがあります。このチャレンジでは、マージコミットを除外したすべてのコミットの短い要約を表示する方法を学びます。
