# Просмотр коммитов за определенный период времени

Ваша задача - просмотреть все коммиты за определенный период времени с использованием Git. Вы можете использовать конкретную дату или относительную дату (например, "12 недель назад").

## Задачи

Для завершения этого испытания вам понадобится использовать репозиторий `https://github.com/labex-labs/git-playground`.

1. Перейдите в директорию репозитория.
2. Просмотрите все коммиты между 25 апреля 2023 года и 27 апреля 2023 года.
3. Просмотрите все коммиты, сделанные за последние двенадцать недель.
