# Consulter les commits dans une plage de dates spécifique

Votre tâche consiste à consulter tous les commits dans une plage de dates spécifique à l'aide de Git. Vous pouvez utiliser soit une date spécifique soit une date relative (par exemple, "il y a 12 semaines").

## Tâches

Pour compléter ce défi, vous devrez utiliser le référentiel `https://github.com/labex-labs/git-playground`.

1. Accédez au répertoire du référentiel.
2. Consultez tous les commits entre le 25 avril 2023 et le 27 avril 2023.
3. Consultez tous les commits effectués au cours des douze dernières semaines.
