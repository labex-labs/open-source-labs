Git 에서 stash 는 아직 커밋할 준비가 되지 않은 변경 사항을 저장하는 방법입니다. Stash 는 다른 브랜치로 전환하거나 다른 작업을 해야 하지만 아직 변경 사항을 커밋하고 싶지 않을 때 유용합니다. 하지만 때로는 더 이상 필요하지 않은 stash 를 삭제해야 할 수도 있습니다. 이 랩에서는 Git 에서 특정 stash 를 삭제하는 방법을 배우게 됩니다.
