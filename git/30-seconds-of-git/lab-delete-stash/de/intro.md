In Git ist ein Stash eine Möglichkeit, Änderungen zu speichern, die noch nicht zum Committen bereit sind. Stashes sind nützlich, wenn Sie zu einer anderen Branch wechseln müssen oder an einer anderen Aufgabe arbeiten, aber Ihre Änderungen noch nicht committen möchten. Manchmal müssen Sie jedoch einen Stash löschen, der nicht mehr benötigt wird. In diesem Lab werden Sie lernen, wie Sie einen bestimmten Stash in Git löschen.
