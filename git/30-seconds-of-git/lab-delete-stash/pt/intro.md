No Git, um _stash_ é uma forma de salvar alterações que ainda não estão prontas para serem commitadas. Os _stashes_ são úteis quando você precisa mudar para um branch diferente ou trabalhar em uma tarefa diferente, mas ainda não quer commitar suas alterações. No entanto, às vezes você pode precisar deletar um _stash_ que não é mais necessário. Neste laboratório, você aprenderá como deletar um _stash_ específico no Git.
