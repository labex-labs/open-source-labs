# Настройка текстового редактора Git

Вы хотите настроить текстовый редактор, используемый Git, на свой предпочитаемый редактор.

## Задачи

Для этого испытания давайте используем репозиторий из `https://github.com/labex-labs/git-playground`.

1. Перейдите в клонированный репозиторий и настройте идентификацию.
2. Настройте Git для использования вашего предпочитаемого текстового редактора (в этом примере мы будем использовать vim).
3. Обновите содержимое файла `hello.txt` на "Hello, Git" и подготовьте его к коммиту.
4. Сделайте коммит изменений.
5. Ваш предпочитаемый текстовый редактор (в этом примере vim) должен открыться с сообщением коммита. Запишите ваше сообщение коммита "Update hello.txt" и сохраните файл.
6. Закройте текстовый редактор. Коммит будет сделан с сообщением, которое вы написали.

Вот конечный результат:

```shell
commit 1f19499s5387a1549f1bb5286d3d0a2b993f87e0 (HEAD -> master)
Author: xiaoshengyunan <@users.noreply.github.com>
Date:   Fri Jul 21 19:26:57 2023 +0800

    Update hello.txt
```
