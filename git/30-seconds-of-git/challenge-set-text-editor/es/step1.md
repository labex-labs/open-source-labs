# Configurar el editor de texto de git

Quieres configurar el editor de texto utilizado por Git al editor que prefieres.

## Tareas

Para este desafío, vamos a utilizar el repositorio de `https://github.com/labex-labs/git-playground`.

1. Navega hasta el repositorio clonado y configura la identidad.
2. Configura Git para que use tu editor de texto preferido (en este ejemplo, usaremos vim).
3. Actualiza el contenido del archivo `hello.txt` a "Hello, Git" y preparalo para el commit.
4. Realiza el commit del cambio.
5. Tu editor de texto preferido (en este ejemplo, vim) debería abrirse con el mensaje de commit. Escribe tu mensaje de commit "Update hello.txt" y guarda el archivo.
6. Cierra el editor de texto. El commit se realizará con el mensaje que escribiste.

Este es el resultado final:

```shell
commit 1f19499s5387a1549f1bb5286d3d0a2b993f87e0 (HEAD -> master)
Author: xiaoshengyunan <@users.noreply.github.com>
Date:   Fri Jul 21 19:26:57 2023 +0800

    Update hello.txt
```
