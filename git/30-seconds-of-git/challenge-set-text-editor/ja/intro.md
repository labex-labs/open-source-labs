Git を使用するとき、コミットメッセージを書いたり、テキストエディタで他の変更を加えることがあります。デフォルトでは、Git はシステムの既定のテキストエディタを使用しますが、それは必ずしも好きなエディタではない場合があります。このチャレンジでは、Git が使用するテキストエディタを構成する方法を学びます。
