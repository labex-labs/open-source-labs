# git のテキストエディタを構成する

Git が使用するテキストエディタを好きなエディタに構成したい。

## タスク

このチャレンジでは、`https://github.com/labex-labs/git-playground` のリポジトリを使用します。

1. クローンしたリポジトリに移動して ID を構成します。
2. Git を構成して好きなテキストエディタ（この例では vim を使用します）を使用するようにします。
3. `hello.txt` ファイルの内容を "Hello, Git" に更新してコミットのためにステージングします。
4. 変更をコミットします。
5. 好きなテキストエディタ（この例では vim）がコミットメッセージ付きで開かれるはずです。コミットメッセージ "Update hello.txt" を書き留めてファイルを保存します。
6. テキストエディタを閉じます。書いたメッセージでコミットが行われます。

これが完成した結果です：

```shell
commit 1f19499s5387a1549f1bb5286d3d0a2b993f87e0 (HEAD -> master)
Author: xiaoshengyunan <@users.noreply.github.com>
Date:   Fri Jul 21 19:26:57 2023 +0800

    Update hello.txt
```
