# Konfiguriere den Git-Texteditor

Sie möchten den von Git verwendeten Texteditor auf Ihren bevorzugten Editor konfigurieren.

## Aufgaben

Für diese Herausforderung verwenden wir das Repository von `https://github.com/labex-labs/git-playground`.

1. Navigieren Sie zum geklonten Repository und konfigurieren Sie die Identität.
2. Konfigurieren Sie Git, um Ihren bevorzugten Texteditor zu verwenden (in diesem Beispiel verwenden wir vim).
3. Aktualisieren Sie den Inhalt der Datei `hello.txt` auf "Hello, Git" und bereiten Sie ihn vor, um ihn zu committen.
4. Committen Sie die Änderung.
5. Ihr bevorzugter Texteditor (in diesem Beispiel vim) sollte mit der Commit-Nachricht geöffnet werden. Notieren Sie Ihre Commit-Nachricht "Update hello.txt" und speichern Sie die Datei.
6. Schließen Sie den Texteditor. Der Commit wird mit der Nachricht durchgeführt, die Sie geschrieben haben.

Dies ist das fertige Ergebnis:

```shell
commit 1f19499s5387a1549f1bb5286d3d0a2b993f87e0 (HEAD -> master)
Author: xiaoshengyunan <@users.noreply.github.com>
Date:   Fri Jul 21 19:26:57 2023 +0800

    Update hello.txt
```
