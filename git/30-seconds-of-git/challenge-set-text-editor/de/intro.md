Wenn Sie Git verwenden, müssen Sie manchmal eine Commit-Nachricht schreiben oder andere Änderungen in einem Texteditor vornehmen. Standardmäßig verwendet Git den standardmäßigen Texteditor des Systems, der möglicherweise nicht der von Ihnen bevorzugte ist. In dieser Herausforderung lernen Sie, wie Sie den von Git verwendeten Texteditor konfigurieren.
