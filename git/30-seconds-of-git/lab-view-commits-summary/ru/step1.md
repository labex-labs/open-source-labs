# Просмотр краткого свода коммитов

Вашей работе в качестве разработчика посвящена проект, в котором участвуют несколько участников. Вам необходимо просмотреть сводку всех коммитов, сделанных в проекте, чтобы понять внесенные изменения и выявить возможные проблемы. Однако вы не хотите тратить много времени на просмотр всех сообщений коммитов, чтобы найти нужную информацию.

Чтобы просмотреть краткую сводку всех коммитов, сделанных в репозитории Git, вы можете использовать команду `git log --oneline`. Например, предположим, что вы работаете над проектом, размещенным на GitHub и называемым `git-playground`.

1. Вы можете склонировать репозиторий на свою локальную машину с помощью следующей команды:

```shell
git clone https://github.com/labex-labs/git-playground.git
```

2. После того, как вы склонировали репозиторий, перейдите в каталог проекта и выполните следующую команду, чтобы просмотреть краткую сводку всех коммитов:

```shell
cd git-playground
git log --oneline
```

В результате будет выведен список всех коммитов, сделанных в репозитории, а также краткая сводка каждого сообщения коммитов. Например:

```shell
d22f46b (HEAD -> master, origin/master, origin/feature-branch, origin/HEAD) Added file2.txt
cf80005 Added file1.txt
b00b937 Initial commit
```
