Git est un puissant système de contrôle de versions qui permet aux développeurs de suivre les modifications apportées à leur base de code au fil du temps. L'une des fonctionnalités les plus utiles de Git est la possibilité de visualiser un résumé de tous les commits effectués dans un référentiel. Cela peut être utile lorsqu'il s'agit de comprendre l'histoire d'un projet ou d'identifier quand une modification particulière a été effectuée.
