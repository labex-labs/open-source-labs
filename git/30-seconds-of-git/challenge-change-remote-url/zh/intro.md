在 Git 中，远程仓库是指托管在服务器或另一台计算机上的仓库版本。当你克隆一个仓库时，Git 会自动创建一个名为 `origin` 的远程仓库，它指向原始仓库。然而，有时你可能需要更改远程仓库的 URL。本挑战将测试你使用 Git 命令更改远程 URL 的能力。
