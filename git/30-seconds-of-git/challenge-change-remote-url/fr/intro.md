En Git, un référentiel distant est une version de votre référentiel hébergée sur un serveur ou un autre ordinateur. Lorsque vous clonez un référentiel, Git crée automatiquement un référentiel distant appelé `origin` qui pointe vers le référentiel original. Cependant, parfois vous devrez peut-être changer l'URL du référentiel distant. Ce défi vous testera la capacité de changer l'URL du référentiel distant à l'aide de commandes Git.
