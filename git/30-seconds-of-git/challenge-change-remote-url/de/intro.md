In Git ist ein Remote eine Version Ihres Repositorys, das auf einem Server oder einem anderen Computer gehostet wird. Wenn Sie ein Repository klonen, erstellt Git automatisch ein Remote namens `origin`, das auf das ursprüngliche Repository verweist. Manchmal müssen Sie jedoch die URL des Remote-Repositorys ändern. Diese Herausforderung wird Ihre Fähigkeit testen, die Remote-URL mit Git-Befehlen zu ändern.
