Git では、リモートはサーバーまたは別のコンピュータにホストされているリポジトリのバージョンです。リポジトリをクローンすると、Git は自動的に `origin` と呼ばれるリモートを作成し、それが元のリポジトリを指します。ただし、時々はリモートリポジトリの URL を変更する必要がある場合があります。このチャレンジでは、Git コマンドを使ってリモート URL を変更する能力をテストします。
