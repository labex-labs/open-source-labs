En Git, un repositorio remoto es una versión de tu repositorio que se hospeda en un servidor o en otra computadora. Cuando clonas un repositorio, Git crea automáticamente un repositorio remoto llamado `origin` que apunta al repositorio original. Sin embargo, a veces es posible que necesites cambiar la URL del repositorio remoto. Este desafío probará tu capacidad para cambiar la URL remota utilizando comandos de Git.
