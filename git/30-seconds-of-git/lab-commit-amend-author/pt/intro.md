Git é um poderoso sistema de controle de versão que permite aos desenvolvedores rastrear as alterações feitas em seu código-base. Uma das funcionalidades do Git é a capacidade de modificar o autor do último commit sem alterar seu conteúdo. Isso pode ser útil em situações onde o autor de um commit precisa ser atualizado para fins de precisão ou responsabilidade.
