Git ist ein beliebiges Versionsverwaltungssystem, das von Entwicklern verwendet wird, um ihren Code zu verwalten. Einer der ersten Schritte bei der Verwendung von Git ist es, ein neues Repository zu erstellen. In diesem Lab werden Sie lernen, wie Sie ein neues Git-Repository initialisieren.
