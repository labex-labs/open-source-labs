No Git, um commit fixup é um tipo especial de commit que é usado para corrigir um commit anterior. Ele é tipicamente usado quando você deseja fazer uma pequena alteração em um commit que já foi feito, sem ter que criar um novo commit. Os commits fixup são especialmente úteis quando você está trabalhando em um projeto grande com muitos contribuidores, pois eles permitem que você faça pequenas alterações sem interromper o trabalho de outras pessoas.
