Git は、開発者が効率的にプロジェクトで共同作業できる強力なバージョン管理システムです。Git の機能の 1 つは、ファストフォワードマージで、これにより Git は、分岐したコミットがないブランチを自動的にマージできます。ただし、この機能は時々問題を引き起こす場合があり、特に複数の貢献者がいる大規模なプロジェクトで作業する場合です。この実験では、デフォルトでファストフォワードマージを無効にする方法を学びます。
