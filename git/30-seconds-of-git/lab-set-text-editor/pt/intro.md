Ao usar o Git, às vezes é necessário escrever uma mensagem de commit ou fazer outras alterações em um editor de texto. Por padrão, o Git usa o editor de texto padrão do sistema, que pode não ser o que você prefere. Neste laboratório, você aprenderá como configurar o editor de texto usado pelo Git.
