Git 을 사용할 때, 커밋 메시지를 작성하거나 텍스트 편집기에서 다른 변경 사항을 적용해야 하는 경우가 있습니다. 기본적으로 Git 은 시스템의 기본 텍스트 편집기를 사용하며, 이는 사용자가 선호하는 편집기가 아닐 수 있습니다. 이 랩에서는 Git 에서 사용하는 텍스트 편집기를 구성하는 방법을 배우게 됩니다.
