In Git werden Branches verwendet, um Funktionen zu entwickeln oder Änderungen an einem Projekt zu isolieren. Manchmal müssen Sie herausfinden, welche Branches einen bestimmten Commit enthalten. Diese Herausforderung wird Ihre Fähigkeit testen, Git-Befehle zu verwenden, um Branches zu finden, die einen Commit enthalten.
