En Git, las ramas se utilizan para desarrollar funciones o aislar los cambios realizados en un proyecto. A veces, es posible que necesites averiguar qué ramas contienen un commit específico. Este reto probará tu capacidad para utilizar comandos de Git para encontrar las ramas que contienen un commit.
