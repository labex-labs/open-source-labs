# 特定のコミットを含むブランチを見つける

`https://github.com/labex-labs/git-playground` という名前の Git リポジトリが与えられています。 あなたのタスクは、コミットメッセージが "Added file2.txt" のハッシュを含むすべてのブランチを見つけることです。

## タスク

1. リポジトリディレクトリに移動します。
2. コミットメッセージが "Added file2.txt" のハッシュを含むすべてのブランチを見つけます。

出力は以下のようになるはずです。

```shell
* master
new-branch
new-branch-1
new-branch-2
```
