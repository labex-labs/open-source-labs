Git ist ein beliebiges Versionsverwaltungssystem, das von Entwicklern verwendet wird, um ihre Codeänderungen zu verwalten. Eine der wesentlichen Funktionen von Git ist die Möglichkeit, Commits zu erstellen, die Momentaufnahmen des Codes zu einem bestimmten Zeitpunkt sind. In diesem Lab werden Sie lernen, wie ein Git-Commit erstellt wird.
