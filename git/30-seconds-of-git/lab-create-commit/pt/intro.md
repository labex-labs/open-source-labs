Git é um sistema de controle de versão (version control system) popular usado por desenvolvedores para gerenciar as alterações em seu código. Uma das funcionalidades essenciais do Git é a capacidade de criar commits, que são snapshots do código em um determinado ponto no tempo. Neste laboratório, você aprenderá como criar um commit Git.
