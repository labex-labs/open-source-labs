Git es un popular sistema de control de versiones utilizado por los desarrolladores para administrar sus cambios de código. Una de las características esenciales de Git es la capacidad de crear commits, que son instantáneas del código en un momento determinado. En este laboratorio, aprenderás cómo crear un commit de Git.
