Git est un puissant système de contrôle de versions qui permet aux développeurs de suivre les modifications apportées à leur base de code. L'une des fonctionnalités de Git est la possibilité de modifier l'auteur du dernier commit sans changer son contenu. Cela peut être utile dans des situations où l'auteur d'un commit doit être mis à jour pour des raisons de précision ou de responsabilité.
