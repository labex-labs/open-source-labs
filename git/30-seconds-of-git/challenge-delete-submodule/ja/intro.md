Git では、サブモジュールは別のリポジトリにサブディレクトリとして含まれるリポジトリです。サブモジュールを使用すると、より大きなプロジェクト内の特定のプロジェクト用に別個のリポジトリを維持できます。ただし、リポジトリからサブモジュールを削除する必要がある場合もあります。このチャレンジでは、Git リポジトリからサブモジュールを削除する方法を学びます。
