In Git ist ein Submodul ein Repository, das als Unterverzeichnis in einem anderen Repository enthalten ist. Submodule ermöglichen es Ihnen, ein separates Repository für ein bestimmtes Projekt innerhalb eines größeren Projekts zu verwalten. Es können jedoch Situationen auftreten, in denen Sie ein Submodul aus Ihrem Repository löschen müssen. In dieser Herausforderung lernen Sie, wie Sie ein Submodul aus einem Git-Repository löschen.
