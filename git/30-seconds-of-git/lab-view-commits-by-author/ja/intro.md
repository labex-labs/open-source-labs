# はじめに

Git は、開発者がコードベースに対して行った変更を追跡できる強力なバージョン管理システムです。Git の最も便利な機能の 1 つは、特定の作者によって作成されたコミットを表示する機能です。これは、特定のチームメンバーによって行われた変更を追跡したり、コードのコントリビューションをレビューする際に役立ちます。
