Git ist ein beliebtes Versionsverwaltungssystem, das von Entwicklern verwendet wird, um ihren Code zu verwalten. Einer der ersten Schritte bei der Verwendung von Git ist es, ein neues Repository zu erstellen. In dieser Herausforderung lernst du, wie du ein neues Git-Repository initialisierst.
