Git 은 개발자가 코드베이스의 변경 사항을 추적할 수 있도록 하는 버전 관리 시스템입니다. 때때로 개발자는 나중에 유지하고 싶지 않은 코드 변경 사항을 만들 수 있습니다. 이러한 경우, 이러한 변경 사항을 폐기하고 마지막으로 커밋된 코드 버전으로 되돌려야 합니다. 이 랩에서는 Git 에서 커밋되지 않은 변경 사항을 폐기하는 데 중점을 둡니다.
