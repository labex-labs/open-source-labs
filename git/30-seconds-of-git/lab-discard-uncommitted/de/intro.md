Git ist ein Versionsverwaltungssystem, das es Entwicklern ermöglicht, Änderungen in ihrer Codebasis zu verfolgen. Manchmal können Entwickler Änderungen an ihrem Code vornehmen, von denen sie später entscheiden, dass sie nicht beibehalten möchten. In solchen Fällen ist es notwendig, diese Änderungen abzulehnen und auf die letzte committete Version des Codes zurückzusetzen. In diesem Lab wird sich auf das Abwerfen von nicht committeten Änderungen in Git konzentrieren.
