Git では、リポジトリは Git によって追跡されるファイルとフォルダのコレクションです。ファイルを変更すると、Git はファイルの新しいバージョンを作成してリポジトリに保存します。時間の経過とともに、リポジトリには古いファイルバージョンやその他の不要なデータがたまり、Git の動作が遅くなり、リポジトリを操作するのが難しくなる場合があります。このチャレンジでは、ローカル リポジトリを最適化してパフォーマンスを向上させる方法を学びます。
