在 Git 中，仓库是由 Git 跟踪的文件和文件夹的集合。当你对文件进行更改时，Git 会创建该文件的新版本并将其存储在仓库中。随着时间的推移，你的仓库可能会因旧版本的文件和其他不必要的数据而变得杂乱。这可能会减慢 Git 的速度，并使处理仓库变得更加困难。在这个挑战中，你将学习如何优化本地仓库以提高其性能。
