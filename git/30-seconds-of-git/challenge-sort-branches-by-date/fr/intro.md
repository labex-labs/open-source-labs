Lorsque vous travaillez avec Git, il est courant d'avoir plusieurs branches dans un référentiel. Cependant, il peut être difficile de suivre toutes les branches et leurs dates de dernier commit. Dans ce défi, vous allez apprendre à utiliser Git pour trier les branches par date, ce qui facilitera la gestion de votre référentiel.
