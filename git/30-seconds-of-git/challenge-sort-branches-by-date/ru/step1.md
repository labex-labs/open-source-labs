# Сортировка ветвей Git по дате

У вас есть репозиторий Git с несколькими ветвями, и вы хотите отсортировать их по дате. Это позволит вам увидеть, какие ветви были недавно обновлены, а какие нет. Сортировка ветвей по дате также может помочь вам определить ветви, которые могут потребовать внимания или слияния.

## Задачи

Для этого испытания давайте используем репозиторий из `https://github.com/labex-labs/git-playground`.

1. Склоняйте репозиторий, перейдите в директорию и настройте личность.
2. Создайте ветвь под названием `one`, создайте файл с именем `hello.txt` и зафиксируйте его с комментарием к коммиту "hello.txt".
3. Переключитесь на ветвь с именем `master` и создайте ветвь с именем `two`.
4. Теперь отсортируйте ветви по дате.

Это покажет список всех локальных ветвей и отсортирует их в зависимости от даты их последнего коммита. Вы можете использовать стрелки клавиатуры для навигации по списку и нажать <kbd>Q</kbd>, чтобы выйти.
