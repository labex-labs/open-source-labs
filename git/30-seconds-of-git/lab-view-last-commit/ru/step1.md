# Просмотр последнего коммита

Вы работаете над проектом в команде разработчиков, и вам нужно просмотреть последний коммит, сделанный в репозитории Git проекта. Вы хотите увидеть детали коммита, включая сообщение коммита, автора и дату.

Чтобы просмотреть последний коммит, сделанный в репозитории Git, следуйте шагам:

1. Откройте терминал на своем компьютере.
2. Перейдите в директорию, где находится репозиторий Git:

```shell
cd git-playground
```

3. Просмотрите последний коммит:

```shell
git log -1
```

Вывод покажет вам детали последнего коммита, включая сообщение коммита, автора и дату:

```shell
commit d22f46ba8c2d4e07d773c5126e9c803933eb5898 (HEAD -> master, origin/master, origin/feature-branch, origin/HEAD)
Author: Hang <huhuhang@users.noreply.github.com>
Date:   Wed Apr 26 14:16:25 2023 +0800

    Added file2.txt
```
