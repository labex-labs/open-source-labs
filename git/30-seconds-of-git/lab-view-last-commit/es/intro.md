Git es un sistema de control de versiones popular que permite a los desarrolladores rastrear los cambios en su código base. Una de las características esenciales de Git es la capacidad de ver el último commit realizado en un repositorio. En este desafío, aprenderás cómo ver el último commit realizado en un repositorio Git.
