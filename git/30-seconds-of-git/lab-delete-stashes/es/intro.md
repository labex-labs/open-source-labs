En Git, las "stashes" se utilizan para guardar temporalmente cambios que aún no están listos para ser confirmados. Sin embargo, a veces puede que desees eliminar todas las "stashes" para comenzar de nuevo. En este laboratorio, aprenderás cómo eliminar todas las "stashes" en un repositorio Git.
