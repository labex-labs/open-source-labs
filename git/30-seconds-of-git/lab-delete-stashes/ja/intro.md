Git では、スタッシュはコミットする準備ができていない変更を一時的に保存するために使用されます。ただし、時々は、すべてのスタッシュを削除して新しく始めたい場合があります。この実験では、Git リポジトリ内のすべてのスタッシュを削除する方法を学びます。
