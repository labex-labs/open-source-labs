No Git, os _stashes_ são usados para salvar temporariamente as alterações que ainda não estão prontas para serem _commitadas_. No entanto, às vezes você pode querer deletar todos os _stashes_ para começar do zero. Neste laboratório, você aprenderá como deletar todos os _stashes_ em um repositório Git.
