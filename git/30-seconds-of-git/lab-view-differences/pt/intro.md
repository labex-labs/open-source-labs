Git é um sistema de controle de versão (version control system) popular que permite aos desenvolvedores rastrear as alterações feitas em seu código-base. Uma das funcionalidades essenciais do Git é a capacidade de visualizar as diferenças entre as alterações feitas em um arquivo. Essa funcionalidade é útil quando você deseja ver quais alterações foram feitas em um arquivo ou comparar as alterações entre diferentes versões de um arquivo.
