Git es un popular sistema de control de versiones que permite a los desarrolladores rastrear los cambios realizados en su repositorio de código. Una de las características esenciales de Git es la capacidad de ver las diferencias entre los cambios realizados en un archivo. Esta característica es útil cuando quieres ver qué cambios has realizado en un archivo o comparar los cambios entre diferentes versiones de un archivo.
