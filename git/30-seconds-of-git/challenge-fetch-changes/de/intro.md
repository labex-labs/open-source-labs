Git ist ein beliebiges Versionsverwaltungssystem, das zur Verfolgung von Änderungen im Code verwendet wird. Eine der wichtigsten Funktionen von Git ist die Möglichkeit, mit Remote-Repositorys zu arbeiten. In dieser Herausforderung lernst du, wie du mit Git die neuesten Änderungen aus einem Remote-Repository abholen kannst.
