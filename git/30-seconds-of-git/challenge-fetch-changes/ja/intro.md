Git は、コードの変更を追跡するために使用される人気のあるバージョン管理システムです。Git の主な機能の 1 つは、リモート リポジトリとの作業能力です。このチャレンジでは、Git を使用してリモート リポジトリから最新の変更を取得する方法を学びます。
