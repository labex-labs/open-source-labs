Git est un système de contrôle de versions populaire utilisé pour suivre les modifications dans le code. L'une des principales fonctionnalités de Git est la capacité de travailler avec des dépôts distants. Dans ce défi, vous allez apprendre à récupérer les dernières modifications d'un dépôt distant à l'aide de Git.
