In Git werden Branches verwendet, um Funktionen zu entwickeln oder Änderungen am Codebase zu isolieren. Manchmal müssen Sie während des Arbeitens an einem Projekt zwischen verschiedenen Branches wechseln. In diesen Fällen ist es wichtig, zu wissen, wie man zurück zum vorherigen Branch wechselt.
