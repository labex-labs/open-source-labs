No Git, as branches (ramificações) são usadas para desenvolver funcionalidades ou isolar as alterações feitas no codebase (código base). Às vezes, ao trabalhar em um projeto, pode ser necessário alternar entre diferentes branches. Nesses casos, torna-se importante saber como voltar para a branch anterior.
