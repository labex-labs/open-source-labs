Git es un popular sistema de control de versiones que permite a los desarrolladores rastrear los cambios en su código base. Una de las características esenciales de Git es la capacidad de ver el último commit realizado en un repositorio. En este desafío, aprenderás a ver el último commit realizado en un repositorio Git.
