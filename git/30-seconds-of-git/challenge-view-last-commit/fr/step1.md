# Voir la dernière modification

Vous travaillez sur un projet avec une équipe de développeurs et vous devez voir la dernière modification effectuée dans le référentiel Git du projet. Vous voulez voir les détails de la modification, y compris le message de modification, l'auteur et la date.

## Tâches

Pour voir la dernière modification effectuée dans un référentiel Git, suivez ces étapes :

1. Ouvrez le terminal de votre ordinateur.
2. Accédez au répertoire où se trouve le référentiel Git.
3. Voir la dernière modification.

La sortie vous montrera les détails de la dernière modification, y compris le message de modification, l'auteur et la date :

```shell
commit d22f46ba8c2d4e07d773c5126e9c803933eb5898 (HEAD -> master, origin/master, origin/feature-branch, origin/HEAD)
Author: Hang <huhuhang@users.noreply.github.com>
Date:   Wed Apr 26 14:16:25 2023 +0800

    Added file2.txt
```
