Git は、開発者がコードベースの変更を追跡できる人気のあるバージョン管理システムです。Git の重要な機能の 1 つは、リポジトリに対して行われた最後のコミットを表示する機能です。このチャレンジでは、Git リポジトリに対して行われた最後のコミットを表示する方法を学びます。
