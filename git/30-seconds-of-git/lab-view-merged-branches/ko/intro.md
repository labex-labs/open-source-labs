Git 에서 브랜치는 기능을 개발하거나 코드베이스에 적용된 변경 사항을 격리하는 데 사용됩니다. 브랜치가 목적을 달성한 후에는 메인 브랜치로 병합됩니다. 그러나 어떤 브랜치가 병합되었고 어떤 브랜치가 병합되지 않았는지 추적하는 것이 중요합니다. 이 챌린지는 병합된 모든 로컬 브랜치의 목록을 보는 능력을 테스트합니다.
