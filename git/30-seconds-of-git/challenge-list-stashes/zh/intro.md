在 Git 中，贮藏（stashing）是一种临时保存尚未准备好提交的更改的方法。贮藏允许你切换分支或处理不同的任务，而不会丢失当前的进度。一旦你准备好继续处理这些更改，可以应用贮藏并从上次中断的地方继续。在这个挑战中，你将学习如何列出 Git 仓库中的所有贮藏。
