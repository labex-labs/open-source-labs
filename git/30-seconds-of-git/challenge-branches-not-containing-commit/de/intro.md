Git ist ein beliebiges Versionskontrollsystem, das es Entwicklern ermöglicht, Änderungen in ihrer Codebasis zu verfolgen. Eine nützliche Funktion von Git ist die Möglichkeit, Branches zu erstellen, die im Wesentlichen separate Kopien der Codebasis sind, die unabhängig voneinander modifiziert werden können. Manchmal kann es jedoch schwierig sein, zu verfolgen, welche Branches bestimmte Commits enthalten. In dieser Herausforderung lernen Sie, wie Sie Branches finden, die einen bestimmten Commit nicht enthalten.
