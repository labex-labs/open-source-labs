Cuando trabajas con Git, es común tener cambios no rastreados en tu directorio de trabajo. Estos cambios son archivos que Git no está rastreando actualmente. A veces, es posible que desees descartar estos cambios y comenzar de nuevo. En este desafío, aprenderás cómo descartar todos los cambios no rastreados en la rama actual.
