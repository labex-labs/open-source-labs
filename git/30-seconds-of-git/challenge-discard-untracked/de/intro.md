Wenn Sie mit Git arbeiten, ist es üblich, unüberwachte Änderungen in Ihrem Arbeitsverzeichnis zu haben. Diese Änderungen sind Dateien, die Git derzeit nicht verfolgt. Manchmal möchten Sie diese Änderungen verwerfen und von vorne beginnen. In dieser Herausforderung lernen Sie, wie Sie alle unüberwachten Änderungen an der aktuellen Branch verwerfen.
