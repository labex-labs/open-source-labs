Git を使用して作業する際、作業ディレクトリにトラックされていない変更があることはよくあります。これらの変更は、現在 Git がトラックしていないファイルです。時々、これらの変更を破棄して新しく始めたい場合があります。このチャレンジでは、現在のブランチのすべてのトラックされていない変更を破棄する方法を学びます。
