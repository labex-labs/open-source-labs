Git 에서 브랜치 (branch) 는 커밋 (commit) 을 가리키는 가볍고 이동 가능한 포인터입니다. 브랜치는 기능을 개발하고, 변경 사항을 격리하며, 저장소 (repository) 의 다른 부분에 영향을 주지 않고 실험하는 데 사용됩니다. 원격 브랜치 (remote branch) 는 원격 저장소 (remote repository) 의 브랜치 상태를 참조합니다. 이는 다른 개발자의 작업 진행 상황을 추적하고 프로젝트를 공동 작업하는 데 사용됩니다.
