No Git, um _branch_ (ramificação) é um ponteiro leve e móvel para um _commit_ (confirmação). As ramificações são usadas para desenvolver _features_ (funcionalidades), isolar alterações e experimentar sem afetar outras partes do repositório. _Remote branches_ (ramificações remotas) são referências ao estado das ramificações em repositórios remotos. Elas são usadas para acompanhar o progresso do trabalho de outros desenvolvedores e para colaborar em projetos.
