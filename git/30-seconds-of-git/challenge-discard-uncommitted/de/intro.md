Git ist ein Versionsverwaltungssystem, das es Entwicklern ermöglicht, Änderungen in ihrer Codebasis zu verfolgen. Manchmal können Entwickler Änderungen an ihrem Code vornehmen, von denen sie später feststellen, dass sie sie nicht beibehalten möchten. In solchen Fällen ist es erforderlich, diese Änderungen abzulehnen und auf die letzte committete Version des Codes zurückzusetzen. In dieser Herausforderung geht es darum, uncommittete Änderungen in Git abzulehnen.
