サブモジュールは、1 つの Git リポジトリを別の Git リポジトリのサブディレクトリとして含める方法です。これは、プロジェクトに第三者のライブラリや共有コードベースを含めたい場合に便利です。この実験では、Git リポジトリにサブモジュールを追加する方法を学びます。
