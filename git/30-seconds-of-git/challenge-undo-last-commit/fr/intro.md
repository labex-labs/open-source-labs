Git est un puissant système de contrôle de versions qui permet aux développeurs de suivre les modifications apportées à leur base de code. L'une des tâches les plus courantes dans Git est d'annuler un commit. Ce défi vous apprendra à annuler le dernier commit sans réécrire l'historique.
