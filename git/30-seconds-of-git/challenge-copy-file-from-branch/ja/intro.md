Git では、ブランチを使用して開発作業を分離し、リポジトリ内の他のブランチに影響を与えることなく作業できます。 時々、別のブランチからファイルをコピーして現在のブランチに追加する必要がある場合があります。このチャレンジでは、Git を使って別のブランチからファイルをコピーする能力をテストします。
