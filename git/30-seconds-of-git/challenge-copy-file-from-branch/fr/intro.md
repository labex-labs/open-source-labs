En Git, les branches sont utilisées pour isoler le travail de développement sans affecter les autres branches du référentiel. Parfois, vous devrez peut-être copier un fichier d'une autre branche vers la branche actuelle. Ce défi vous permettra de tester votre capacité à copier un fichier d'une autre branche à l'aide de Git.
