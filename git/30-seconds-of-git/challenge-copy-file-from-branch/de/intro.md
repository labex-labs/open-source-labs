In Git werden Branches verwendet, um die Entwicklung ohne Auswirkungen auf andere Branches im Repository zu isolieren. Manchmal muss man eine Datei von einer anderen Branch in die aktuelle Branch kopieren. Diese Herausforderung wird deine Fähigkeit testen, eine Datei von einer anderen Branch mit Git zu kopieren.
