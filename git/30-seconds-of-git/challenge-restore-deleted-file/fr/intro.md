Git est un système de contrôle de versions populaire qui permet aux développeurs de suivre les modifications apportées à leur code au fil du temps. L'un des avantages d'utiliser Git est qu'il vous permet de restaurer des fichiers supprimés. Dans ce défi, vous allez apprendre à restaurer un fichier qui a été supprimé dans un commit spécifique.
