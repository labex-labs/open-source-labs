Git は、開発者がコードの変更履歴を追跡できる人気のあるバージョン管理システムです。Git を使用する利点の 1 つは、削除されたファイルを復元できることです。このチャレンジでは、特定のコミットで削除されたファイルを復元する方法を学びます。
