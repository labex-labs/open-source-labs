Git es un popular sistema de control de versiones que permite a los desarrolladores rastrear los cambios realizados en su código a lo largo del tiempo. Una de las ventajas de utilizar Git es que te permite restaurar archivos eliminados. En este desafío, aprenderás cómo restaurar un archivo que fue eliminado en un commit específico.
