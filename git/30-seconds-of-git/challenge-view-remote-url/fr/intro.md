Git est un système de contrôle de versions populaire qui permet aux développeurs de gérer et de suivre les modifications apportées à leur code. L'une des principales fonctionnalités de Git est la capacité de travailler avec des dépôts distants, hébergés sur un serveur et accessibles par plusieurs utilisateurs. Dans ce défi, vous allez apprendre à afficher l'URL d'un dépôt distant à l'aide de Git.
