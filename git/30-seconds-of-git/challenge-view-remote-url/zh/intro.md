Git 是一种流行的版本控制系统，它允许开发者管理和跟踪代码的更改。Git 的关键特性之一是能够使用远程仓库，这些仓库托管在服务器上，多个用户可以访问。在这个挑战中，你将学习如何使用 Git 查看远程仓库的 URL。
