En Git, un commit est une capture instantanée des modifications apportées à un référentiel. Chaque commit a un message qui décrit les modifications effectuées. Parfois, vous devrez peut-être créer un commit vide sans modifications pour servir de point d'arrêt ou pour déclencher un processus de génération. Dans ce laboratoire, vous allez apprendre à créer un commit vide dans Git.
