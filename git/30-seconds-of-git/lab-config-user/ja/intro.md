Git は、開発者がコードを管理するために使用される人気のあるバージョン管理システムです。リポジトリに変更を加えると、Git はそれらの変更を行った人を追跡します。これを行うには、Git には名前やメールアドレスなどのユーザー情報が必要です。この実験では、Git のユーザー情報を構成する方法を学びます。
