Git 은 개발자가 코드를 관리하기 위해 사용하는 널리 사용되는 버전 관리 시스템입니다. 리포지토리 (repository) 에 변경 사항을 적용하면 Git 은 누가 변경했는지 추적합니다. 이를 위해 Git 은 이름과 이메일 주소와 같은 사용자 정보를 알아야 합니다. 이 랩 (lab) 에서는 Git 에 대한 사용자 정보를 구성하는 방법을 배우게 됩니다.
