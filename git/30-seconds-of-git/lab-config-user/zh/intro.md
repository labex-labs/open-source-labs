Git 是开发者用于管理代码的流行版本控制系统。当你对仓库进行更改时，Git 会跟踪是谁做出了这些更改。为此，Git 需要知道你的用户信息，例如你的姓名和电子邮件地址。在本实验中，你将学习如何为 Git 配置你的用户信息。
