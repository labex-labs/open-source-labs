# ローカルブランチを表示する

`https://github.com/labex-labs/git-playground` という名前の Git リポジトリで作業してきました。このリポジトリ内のすべてのローカルブランチの一覧を表示したいと思います。

## タスク

1. クローンしたリポジトリに移動します。
2. すべてのローカルブランチの一覧を表示します。

リポジトリ内のすべてのローカルブランチの一覧が表示されます。矢印キーを使って一覧を移動し、<kbd>Q</kbd> キーを押して終了します。

これが最終結果です。
