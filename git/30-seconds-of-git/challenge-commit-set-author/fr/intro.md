Git est un système de contrôle de versions populaire utilisé par les développeurs pour gérer leur base de code. L'une des fonctionnalités essentielles de Git est la capacité de créer des commits, qui sont des instantanés du code à un moment précis. Par défaut, Git utilise le nom et l'adresse e-mail de l'utilisateur pour identifier l'auteur d'un commit. Cependant, il peut y avoir des situations où vous devrez créer un commit avec un autre auteur. Dans ce défi, vous allez apprendre à créer un commit avec un autre auteur dans Git.
