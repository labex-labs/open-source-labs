In Git ist der Staging-Bereich der Ort, an dem Änderungen an Dateien vor ihrer Übermittlung an das Repository vorbereitet werden. Manchmal fügen Sie versehentlich Dateien in den Staging-Bereich hinzu, die Sie nicht committen möchten. In dieser Herausforderung lernen Sie, wie Sie Dateien aus dem Staging-Bereich mit Git entfernen.
