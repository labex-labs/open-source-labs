Les sous-modules sont un moyen d'inclure un référentiel Git comme un sous-répertoire d'un autre référentiel Git. Cela peut être utile lorsque vous voulez inclure une bibliothèque tierce ou une base de code partagée dans votre projet. Dans ce défi, vous allez apprendre à ajouter un sous-module à un référentiel Git.
