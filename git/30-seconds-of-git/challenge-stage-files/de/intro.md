Git ist ein beliebtes Versionsverwaltungssystem, das es Entwicklern ermöglicht, Änderungen an ihrer Codebasis zu verfolgen. Eine der wesentlichen Funktionen von Git ist der Staging-Bereich, der es Entwicklern ermöglicht, selektiv zu wählen, welche Änderungen committet werden sollen. In dieser Herausforderung lernst du, wie du Dateien mit Git in den Staging-Bereich hinzufügst.
