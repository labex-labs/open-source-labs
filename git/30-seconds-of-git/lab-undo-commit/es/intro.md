Git es un sistema de control de versiones poderoso que permite a los desarrolladores rastrear los cambios en su código base. Una de las características más útiles de Git es la capacidad de deshacer los cambios que se han realizado. En este laboratorio, aprenderás cómo deshacer un commit usando Git.
