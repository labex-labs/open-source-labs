Git 에서 stash 는 아직 커밋할 준비가 되지 않은 변경 사항을 저장하는 방법입니다. 변경 사항을 임시로 저장하고 다른 브랜치로 전환하거나 다른 작업을 할 수 있게 해줍니다. 변경 사항 작업을 다시 시작할 준비가 되면 stash 를 적용하고 중단했던 부분부터 계속 작업할 수 있습니다.
