Git ist ein beliebtes Versionsverwaltungssystem, das es Entwicklern ermöglicht, Änderungen an ihrem Code im Laufe der Zeit zu verfolgen. Ein Vorteil bei der Verwendung von Git ist, dass es Ihnen ermöglicht, gelöschte Dateien wiederherzustellen. In diesem Lab werden Sie lernen, wie Sie eine Datei wiederherstellen, die in einem bestimmten Commit gelöscht wurde.
