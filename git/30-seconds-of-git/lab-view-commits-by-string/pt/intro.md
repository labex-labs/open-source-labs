Git é um poderoso sistema de controle de versão que permite aos desenvolvedores rastrear as alterações feitas em seu código ao longo do tempo. Uma das funcionalidades mais úteis do Git é a capacidade de visualizar o histórico de alterações feitas em um arquivo específico ou até mesmo em uma linha específica de código. Neste desafio, você aprenderá como usar o Git para encontrar todos os commits que manipularam uma determinada string.
