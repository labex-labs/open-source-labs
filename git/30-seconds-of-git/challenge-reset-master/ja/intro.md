Git を使用する際、ローカルブランチがリモートブランチと同期していないことがよくあります。これは、ローカルブランチに反映されていないリモートブランチに変更が加えられた場合に発生します。このような場合、ローカルブランチをリモートブランチに合わせるためにリセットする必要があります。このチャレンジでは、ローカルの `master` ブランチをリモートのブランチに合わせるための手順を案内します。
