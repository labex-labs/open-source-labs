No Git, um _branch_ (ramificação) é um ponteiro leve e móvel para um _commit_. Criar um novo _branch_ permite que você trabalhe em uma nova funcionalidade ou correção de _bug_ sem afetar a base de código principal. Neste laboratório, você aprenderá como criar um novo _branch_ no Git.
