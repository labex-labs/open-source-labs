En Git, une branche est un pointeur léger et mobile vers un commit. Créer une nouvelle branche vous permet de travailler sur une nouvelle fonctionnalité ou de corriger un bogue sans affecter la base de code principale. Dans ce laboratoire, vous allez apprendre à créer une nouvelle branche dans Git.
