Git es un sistema de control de versiones poderoso que permite a los desarrolladores rastrear los cambios en su repositorio de código. Una de las características clave de Git es la capacidad de modificar el historial de commits. En este laboratorio, aprenderás cómo cambiar el mensaje del último commit sin cambiar su contenido.
