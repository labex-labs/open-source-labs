No Git, as branches (ramificações) são usadas para desenvolver funcionalidades ou isolar as alterações feitas em um projeto. Às vezes, pode ser necessário descobrir quais branches contêm um commit específico. Este laboratório testará sua capacidade de usar comandos Git para encontrar branches que contêm um commit.
