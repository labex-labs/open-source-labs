Lorsque vous travaillez avec Git, il est important de pouvoir visualiser un résumé de tous les commits effectués sur un référentiel. Cependant, parfois les commits de fusion peuvent encombrer la sortie et rendre difficile la visualisation des changements réels effectués. Dans ce laboratoire, vous allez apprendre à visualiser un résumé court de tous les commits, en excluant les commits de fusion.
