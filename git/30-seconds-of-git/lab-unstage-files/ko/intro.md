Git 에서 스테이징 영역은 파일 변경 사항을 리포지토리에 커밋하기 전에 준비하는 곳입니다. 때로는 커밋하고 싶지 않은 파일을 실수로 스테이징 영역에 추가할 수 있습니다. 이 랩에서는 Git 을 사용하여 스테이징 영역에서 파일을 제거하는 방법을 배우게 됩니다.
