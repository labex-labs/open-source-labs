Git 은 개발자가 코드베이스의 변경 사항을 추적할 수 있도록 하는 널리 사용되는 버전 관리 시스템입니다. Git 의 주요 기능 중 하나는 로컬 변경 사항을 원격 저장소로 푸시하는 기능입니다. 이를 통해 여러 개발자가 동일한 코드베이스에서 협업하고 최신 상태를 유지할 수 있습니다.
