Git é um sistema de controle de versão (version control system) popular que permite aos desenvolvedores rastrear as alterações em seu código-base. Uma das principais características do Git é a capacidade de enviar (push) as alterações locais para um repositório remoto. Isso permite que vários desenvolvedores colaborem no mesmo código-base e o mantenham atualizado.
