Git 是一种流行的版本控制系统，它允许开发者高效地管理他们的代码库。Git 的一个基本特性是创建和管理分支的能力。分支使开发者能够同时处理不同的功能或修复漏洞，而不会相互干扰工作。然而，在某些时候，你可能需要比较两个分支之间的差异。在这个挑战中，你将学习如何使用 Git 查看两个分支之间的差异。
