Git では、ブランチを使用して開発作業を分離し、リポジトリ内の他のブランチに影響を与えることなく作業を行うことができます。時々、別のブランチからファイルをコピーして現在のブランチに追加する必要がある場合があります。この実験では、Git を使用して別のブランチからファイルをコピーする能力をテストします。
