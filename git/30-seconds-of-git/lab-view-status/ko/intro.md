Git 은 개발자가 코드베이스의 변경 사항을 추적할 수 있도록 하는 널리 사용되는 버전 관리 시스템입니다. Git 의 가장 중요한 기능 중 하나는 작업 트리 (working tree) 의 현재 상태를 볼 수 있는 기능입니다. 이 랩에서는 `git status` 명령을 사용하여 Git 저장소 (repository) 의 현재 상태를 확인하는 방법을 배우게 됩니다.
