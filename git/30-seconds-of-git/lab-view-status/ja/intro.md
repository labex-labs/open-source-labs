Git は、開発者がコードベースの変更を追跡できる人気のあるバージョン管理システムです。Git の最も重要な機能の 1 つは、ワーキングツリーの現在の状態を表示する機能です。この実験では、`git status` コマンドを使用して Git リポジトリの現在の状態を表示する方法を学びます。
