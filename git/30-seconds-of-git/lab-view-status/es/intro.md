Git es un sistema de control de versiones popular que permite a los desarrolladores rastrear los cambios en su repositorio de código. Una de las características más importantes de Git es la capacidad de ver el estado actual del árbol de trabajo. En este laboratorio, aprenderás a usar el comando `git status` para ver el estado actual de tu repositorio de Git.
