Git ist ein beliebiges Versionsverwaltungssystem, das es Entwicklern ermöglicht, Änderungen in ihrer Codebasis zu verfolgen. Eine der wichtigsten Funktionen von Git ist die Möglichkeit, den aktuellen Status des Arbeitsbaums anzuzeigen. In diesem Lab werden Sie lernen, wie Sie den Befehl `git status` verwenden, um den aktuellen Status Ihres Git-Repositorys anzuzeigen.
