Git é um sistema de controle de versão (version control system) popular que permite aos desenvolvedores rastrear as alterações em seu código-base. Uma das características mais importantes do Git é a capacidade de visualizar o status atual da árvore de trabalho (working tree). Neste laboratório, você aprenderá como usar o comando `git status` para visualizar o status atual do seu repositório Git.
