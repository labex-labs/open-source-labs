Git est un système de contrôle de versions populaire qui permet aux développeurs de suivre les modifications de leur base de code. L'une des fonctionnalités les plus importantes de Git est la capacité à afficher l'état actuel de l'arbre de travail. Dans ce laboratoire, vous allez apprendre à utiliser la commande `git status` pour afficher l'état actuel de votre référentiel Git.
