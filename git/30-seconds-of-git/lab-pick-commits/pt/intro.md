Git é um poderoso sistema de controle de versão que permite aos desenvolvedores gerenciar seu código-base de forma eficiente. Uma das funcionalidades mais úteis do Git é a capacidade de fazer _cherry-pick_ (seleção pontual) de alterações de um ou mais _commits_. Isso permite que os desenvolvedores apliquem alterações específicas ao seu código-base sem ter que mesclar (merge) ramos (branches) inteiros.
