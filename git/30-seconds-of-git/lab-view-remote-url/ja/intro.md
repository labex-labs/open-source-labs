Git は、開発者がコードの変更を管理して追跡できる人気のあるバージョン管理システムです。Git の主な機能の 1 つは、サーバーにホストされ、複数のユーザーがアクセスできるリモート リポジトリとの作業能力です。この実験では、Git を使用してリモート リポジトリの URL を表示する方法を学びます。
