Git es un sistema de control de versiones popular que permite a los desarrolladores administrar y rastrear los cambios en su código. Una de las características clave de Git es la capacidad de trabajar con repositorios remotos, que se hospedan en un servidor y pueden ser accedidos por múltiples usuarios. En este laboratorio, aprenderás cómo ver la URL de un repositorio remoto utilizando Git.
