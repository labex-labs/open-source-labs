Git é um sistema de controle de versão popular que permite aos desenvolvedores gerenciar e rastrear as alterações em seu código. Uma das principais características do Git é a capacidade de trabalhar com repositórios remotos, que são hospedados em um servidor e podem ser acessados por vários usuários. Neste laboratório, você aprenderá como visualizar a URL de um repositório remoto usando o Git.
