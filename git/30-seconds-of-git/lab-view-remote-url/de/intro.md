Git ist ein beliebiges Versionsverwaltungssystem, das es Entwicklern ermöglicht, Änderungen an ihrem Code zu verwalten und zu verfolgen. Eine der wichtigsten Funktionen von Git ist die Möglichkeit, mit Remote-Repositorys zu arbeiten, die auf einem Server gehostet sind und von mehreren Benutzern zugegriffen werden können. In diesem Lab werden Sie lernen, wie Sie die URL eines Remote-Repositorys mit Git anzeigen.
