# はじめに

<div class="alert alert-warning" role="alert">
<p>🧑‍💻 Git や LabEx の初心者ですか？私たちは、<b><a style="color: unset;text-decoration: underline;" href="https://labex.io/courses/quick-start-with-git" target="_blank">Git のクイックスタート</a></b> コースから始めることをお勧めします。</p>
</div>

Git では、ブランチを使用して機能を開発したり、コードベースに対して行われた変更を分離したりします。ブランチが目的を果たした後は、メインブランチにマージし直されます。ただし、どのブランチがマージされているか、どのブランチがマージされていないかを追跡することが重要です。このチャレンジでは、すべてのマージ済みのローカルブランチの一覧を表示する能力をテストします。

<div class="text-xs text-gray-500 dark:text-gray-400 mt-4 border-t border-l-2 border-gray-300 dark:border-gray-600 pt-2 pl-4">
これは Challenge です。Guided Lab とは異なり、学習のためにラボの手順に従うのではなく、チャレンジタスクを独立して完了する必要があります。Challenge は通常、少し難しい課題です。難しいと感じた場合は、Labby と相談したり、解決策を確認したりできます。
</div>
