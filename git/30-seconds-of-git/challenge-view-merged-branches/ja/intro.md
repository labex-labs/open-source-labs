# はじめに

Git では、ブランチを使用して機能を開発したり、コードベースに対して行われた変更を分離したりします。ブランチが目的を果たした後は、メインブランチにマージし直されます。ただし、どのブランチがマージされているか、どのブランチがマージされていないかを追跡することが重要です。このチャレンジでは、すべてのマージ済みのローカルブランチの一覧を表示する能力をテストします。
