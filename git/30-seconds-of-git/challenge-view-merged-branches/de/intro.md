In Git werden Branches verwendet, um Funktionen zu entwickeln oder Änderungen am Codebase zu isolieren. Nachdem ein Branch seinen Zweck erfüllt hat, wird er in den Hauptbranch zurückgemerged. Es ist jedoch wichtig, zu verfolgen, welche Branches gemerged wurden und welche nicht. Diese Herausforderung wird deine Fähigkeit testen, eine Liste aller gemerged lokalen Branches anzuzeigen.
