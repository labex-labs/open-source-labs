En Git, las ramas se utilizan para desarrollar funciones o aislar los cambios realizados en la base de código. Una vez que una rama ha cumplido su función, se fusiona de nuevo en la rama principal. Sin embargo, es importante llevar un registro de qué ramas se han fusionado y cuáles no. Este desafío probará tu capacidad para ver una lista de todas las ramas locales fusionadas.
