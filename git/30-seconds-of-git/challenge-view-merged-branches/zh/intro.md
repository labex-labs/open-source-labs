在 Git 中，分支用于开发功能或隔离对代码库所做的更改。当一个分支完成其使命后，它会被合并回主分支。然而，跟踪哪些分支已被合并以及哪些分支尚未合并非常重要。本挑战将测试你查看所有已合并本地分支列表的能力。

这是一个挑战（Challenge），与实验（Lab）不同，您需要独立完成挑战任务，而不是按照实验的步骤学习。挑战通常有一点难度。如果您觉得困难，可以与 Labby 讨论或查看解决方案。
