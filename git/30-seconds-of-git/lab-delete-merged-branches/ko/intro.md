Git 으로 프로젝트를 진행할 때, 브랜치를 생성하고 병합하는 것은 일반적인 일입니다. 하지만 시간이 지남에 따라 이러한 브랜치들이 쌓여 로컬 저장소를 어지럽힐 수 있습니다. 병합된 브랜치를 삭제하는 것은 저장소를 깨끗하고 정리된 상태로 유지하기 위한 좋은 방법입니다.
