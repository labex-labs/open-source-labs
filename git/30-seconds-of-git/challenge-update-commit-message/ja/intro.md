Git は、開発者がコードベースの変更を追跡できる強力なバージョン管理システムです。Git の主な機能の 1 つは、コミット履歴を変更する機能です。このチャレンジでは、コミットの内容を変更することなく、最後のコミットのメッセージを変更する方法を学びます。
