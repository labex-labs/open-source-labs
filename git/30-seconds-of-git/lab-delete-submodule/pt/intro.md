No Git, um _submódulo_ (submodule) é um repositório que está incluído em outro repositório como um subdiretório. Submódulos permitem que você mantenha um repositório separado para um projeto específico dentro de um projeto maior. No entanto, pode haver situações em que você precise deletar um submódulo do seu repositório. Neste laboratório, você aprenderá como deletar um submódulo de um repositório Git.
