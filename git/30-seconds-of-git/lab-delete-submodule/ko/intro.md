Git 에서 서브모듈 (submodule) 은 다른 저장소에 하위 디렉토리로 포함된 저장소입니다. 서브모듈을 사용하면 더 큰 프로젝트 내에서 특정 프로젝트에 대한 별도의 저장소를 유지할 수 있습니다. 그러나 저장소에서 서브모듈을 삭제해야 하는 상황이 발생할 수 있습니다. 이 랩에서는 Git 저장소에서 서브모듈을 삭제하는 방법을 배우게 됩니다.
