Git ist ein leistungsstarkes Versionsverwaltungssystem, das es Entwicklern ermöglicht, ihre Codebasis effizient zu verwalten. Eine der nützlichsten Funktionen von Git ist die interaktive Rebase, die Entwicklern ermöglicht, den Commit-History eines Branches zu modifizieren. In dieser Herausforderung lernst du, wie du eine interaktive Rebase mit Git durchführst.
