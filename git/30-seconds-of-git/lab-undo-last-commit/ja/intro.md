Git は、開発者がコードベースに対して行った変更を追跡できる強力なバージョン管理システムです。Git で最も一般的なタスクの 1 つは、コミットを取り消すことです。この実験では、履歴を書き換えることなく最後のコミットを取り消す方法を学びます。
