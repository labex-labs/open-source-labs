En Git, les emplacements de sauvegarde temporaire (stashes) sont utilisés pour enregistrer temporairement des modifications qui ne sont pas encore prêtes à être validées. Cependant, parfois, vous souhaiterez peut-être supprimer tous les emplacements de sauvegarde temporaire pour repartir de zéro. Dans ce défi, vous allez apprendre à supprimer tous les emplacements de sauvegarde temporaire dans un référentiel Git.
