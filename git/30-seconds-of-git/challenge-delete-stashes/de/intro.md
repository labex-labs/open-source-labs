In Git werden Stashes verwendet, um Änderungen vorübergehend zu speichern, die noch nicht zum Committen bereit sind. Manchmal möchten Sie jedoch alle Stashes löschen, um von vorne loszulegen. In dieser Herausforderung lernen Sie, wie Sie alle Stashes in einem Git-Repository löschen.
