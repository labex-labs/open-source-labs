# Einführung

In diesem Lab werden wir lernen, wie man ein Python Flask-Projekt installierbar macht. Dadurch wird das Projekt mit standardmäßigen Python-Tools bereitgestellt und verwaltbar. Dieser Prozess hat den Vorteil, dass das Projekt in verschiedenen Umgebungen installiert werden kann, Abhängigkeiten verwaltet werden können und Testumgebungen isoliert werden können.

> Hinweis: Sie müssen die Code-Datei selbst erstellen und sie in der Umgebung ausführen. Sie können den Status des Flask-Diensts auf Web 5000 vorab anzeigen.
