# Ajouter des règles CSS

Ensuite, copiez les règles CSS suivantes dans le fichier `style.css`. Ces règles vont styliser divers éléments HTML de notre application.

```css
/* flaskr/static/style.css */

html {
  font-family: sans-serif;
  background: #eee;
  padding: 1rem;
}
body {
  max-width: 960px;
  margin: 0 auto;
  background: white;
}
/* Plus de règles CSS... */
```
