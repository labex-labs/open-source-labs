# Добавление CSS - правил

Далее, скопируйте следующие CSS - правила в файл `style.css`. Эти правила будут стилизовать различные HTML - элементы в нашем приложении.

```css
/* flaskr/static/style.css */

html {
  font-family: sans-serif;
  background: #eee;
  padding: 1rem;
}
body {
  max-width: 960px;
  margin: 0 auto;
  background: white;
}
/* Дополнительные CSS - правила... */
```
