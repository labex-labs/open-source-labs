# Adicionar regras CSS

Em seguida, copie as seguintes regras CSS para o arquivo `style.css`. Essas regras estilizarão vários elementos HTML em nossa aplicação.

```css
/* flaskr/static/style.css */

html {
  font-family: sans-serif;
  background: #eee;
  padding: 1rem;
}
body {
  max-width: 960px;
  margin: 0 auto;
  background: white;
}
/* Mais regras CSS... */
```
