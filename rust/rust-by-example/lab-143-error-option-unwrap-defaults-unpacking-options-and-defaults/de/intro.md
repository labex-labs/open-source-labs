# Einführung

In diesem Lab werden wir verschiedene Möglichkeiten zur Entpackung eines `Option` untersuchen und auf einen Standardwert zurückgreifen, wenn es `None` ist, wobei Faktoren wie eager oder lazy Evaluation sowie die Frage, ob der leere Wert in situ geändert oder unberührt gelassen werden soll, berücksichtigt werden.

> **Hinweis:** Wenn im Lab kein Dateiname angegeben ist, können Sie einen beliebigen Dateinamen verwenden. Beispielsweise können Sie `main.rs` verwenden und es mit `rustc main.rs &&./main` kompilieren und ausführen.
