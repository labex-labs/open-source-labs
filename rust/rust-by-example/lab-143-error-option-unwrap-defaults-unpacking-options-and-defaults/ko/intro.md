# 소개

이 랩에서는 `Option`을 언팩 (unpack) 하고, 값이 `None`인 경우 기본값으로 대체하는 다양한 방법을 살펴봅니다. 여기에는 즉시 평가 (eager evaluation) 또는 지연 평가 (lazy evaluation) 와 빈 값을 제자리에서 수정할지 또는 그대로 유지할지 여부와 같은 요소를 고려합니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우, 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
