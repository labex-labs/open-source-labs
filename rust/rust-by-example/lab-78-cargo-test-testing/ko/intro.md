# 소개

이 실습에서는 Rust 를 사용하여 소프트웨어 개발에서 테스트의 중요성을 탐구하고, 단위 테스트와 통합 테스트와 같은 다양한 유형의 테스트를 작성하는 방법을 살펴봅니다. 또한 Rust 프로젝트에서 테스트를 구성하고 `cargo test` 명령어를 사용하여 테스트를 실행하는 방법을 배울 것입니다. 또한, 테스트를 동시에 실행할 때 발생할 수 있는 잠재적인 문제점을 논의하고, 이를 설명하기 위한 예제를 제공할 것입니다.

> **참고:** 실습에서 파일 이름을 지정하지 않은 경우 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs` 파일을 사용하고 `rustc main.rs && ./main` 명령어로 컴파일 및 실행할 수 있습니다.
