# はじめに

この実験では、Rustを使ったソフトウェア開発におけるテストの重要性と、ユニットテストや統合テストなどのさまざまな種類のテストを書く方法を探ります。また、Rustプロジェクトでテストを整理し、`cargo test`コマンドを使って実行する方法についても学びます。さらに、同時にテストを実行することで発生する可能性のある問題について議論し、これを示す例を挙げます。

> **注:** 実験でファイル名が指定されていない場合、好きなファイル名を使うことができます。たとえば、`main.rs`を使って、`rustc main.rs &&./main`でコンパイルして実行することができます。
