# Introducción

En este laboratorio, aprenderá sobre el Rust Playground, una interfaz web que le permite experimentar con código Rust. También descubrirá cómo utilizarlo con `mdbook` para hacer que los ejemplos de código sean reproducibles y editables. Además, explorará cómo utilizarlo con la documentación y habilitar el botón "Ejecutar" utilizando el atributo `html_playground_url` en el código Rust.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede utilizar cualquier nombre de archivo que desee. Por ejemplo, puede utilizar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
