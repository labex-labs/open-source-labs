# 소개

이 실습에서는 Rust Playground, 즉 Rust 코드를 실험할 수 있는 웹 인터페이스에 대해 배웁니다. 또한 `mdbook`을 사용하여 코드 예제를 실행 가능하고 편집 가능하게 만드는 방법을 알아봅니다. 추가적으로, Rust 코드에서 `html_playground_url` 속성을 사용하여 문서와 함께 "실행" 버튼을 활성화하는 방법을 탐색합니다.

> **참고:** 실습에서 파일 이름을 지정하지 않은 경우 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
