# Introducción

En este laboratorio, aprenderemos sobre los identificadores crudos en Rust, que nos permiten usar palabras clave como identificadores en situaciones donde normalmente no se les permite, como nombres de variables o funciones, especialmente cuando ediciones antiguas de Rust entran en conflicto con palabras clave más recientes.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilarlo y ejecutarlo con `rustc main.rs &&./main`.
