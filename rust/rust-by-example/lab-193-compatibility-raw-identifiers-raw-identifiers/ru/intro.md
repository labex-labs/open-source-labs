# Введение

В этом практическом занятии мы узнаем о необработанных идентификаторах в Rust, которые позволяют использовать ключевые слова в качестве идентификаторов в ситуациях, где это обычно не допускается, например, в качестве имен переменных или функций, особенно когда старые версии Rust конфликтуют с новыми ключевыми словами.

> **Примечание:** Если практическое занятие не задает имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
