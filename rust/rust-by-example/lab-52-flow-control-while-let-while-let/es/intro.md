# Introducción

En este laboratorio, se demuestra el uso de `while let` en Rust como una alternativa más concisa y eficiente al uso de secuencias `match` al incrementar variables o iterar sobre tipos `Option`.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
