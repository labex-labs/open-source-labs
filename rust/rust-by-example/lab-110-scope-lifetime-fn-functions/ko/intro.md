# 소개

이 랩에서는 Rust 에서 생명주기 (lifetime) 를 가진 함수 시그니처 (function signature) 를 소개합니다. 여기서 모든 참조 (reference) 는 주석 처리된 생명주기를 가져야 하며, 반환되는 모든 참조는 입력과 동일한 생명주기를 가지거나 `static`이어야 합니다. 입력 없이 참조를 반환하는 것은 유효하지 않은 데이터에 대한 참조를 반환하는 결과를 초래할 경우 금지된다는 점에 유의해야 합니다. 제공된 예제는 생명주기를 가진 함수의 유효한 형태를 보여줍니다. 여기에는 하나의 입력 참조를 가진 함수, 가변 참조 (mutable reference) 를 가진 함수, 여러 요소와 서로 다른 생명주기를 가진 함수, 그리고 매개변수로 전달된 참조를 반환하는 함수가 포함됩니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우, 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
