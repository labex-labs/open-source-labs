# 소개

이 랩에서는 `Box` 타입을 사용하여 원래의 오류를 래핑 (wrapping) 함으로써 보존하고, 동적 오류 처리를 가능하게 하는 방법을 보여주는 코드를 제공합니다. 또한, `Std` 라이브러리의 `From` 트레이트는 `Error` 트레이트를 구현하는 모든 타입을 트레이트 객체 `Box<Error>`로 변환하는 데 도움을 줍니다. 여기에는 사용자 정의 오류 타입과 함께 `Box`를 사용하여 오류를 변환하고 처리하는 예제가 포함되어 있습니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우, 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
