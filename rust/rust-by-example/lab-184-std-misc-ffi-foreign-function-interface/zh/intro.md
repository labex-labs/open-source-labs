# 简介

在本实验中，我们将学习Rust的外部函数接口（FFI），它允许通过在 `extern` 块中声明外部函数，并使用包含外部库名称的 `#[link]` 属性对其进行注释，从而与C库进行交互。代码示例展示了如何使用FFI调用 `libm` 库中的外部函数，比如计算单精度复数的平方根以及计算复数的余弦值。在这些不安全的外部函数调用周围通常会使用安全包装器。本实验还包括单精度复数的最小实现，并演示了如何调用围绕不安全操作包装的安全API。

> **注意**：如果实验未指定文件名，你可以使用任何你想要的文件名。例如，你可以使用 `main.rs`，并通过 `rustc main.rs &&./main` 进行编译和运行。
