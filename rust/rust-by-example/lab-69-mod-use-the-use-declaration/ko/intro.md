# 소개

이 실습에서는 Rust 의 `use` 선언을 탐색합니다. `use` 선언은 전체 경로를 새로운 이름으로 바인딩하여 쉽게 접근할 수 있게 해주며, `as` 키워드와 함께 사용하여 가져온 항목을 다른 이름으로 바인딩할 수 있습니다. 예제 코드 스니펫은 중첩된 경로 내의 함수와 모듈에 접근하는 것을 단순화하기 위해 `use` 선언을 사용하는 방법을 보여줍니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
