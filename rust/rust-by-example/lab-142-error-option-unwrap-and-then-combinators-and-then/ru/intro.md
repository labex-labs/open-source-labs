# Введение

В этом лабораторном задании комбинатор `and_then()` представляется в качестве цепляемого альтернативы `map()` при работе с `Option<T>`, так как он избавляет от необходимости вложения `Option<Option<T>>` и упрощает читаемость кода.

> **Примечание:** Если в лабораторном задании не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
