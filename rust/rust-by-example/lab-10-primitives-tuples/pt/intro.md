# Introdução

Neste laboratório, exploramos as tuplas em Rust. Tuplas são coleções de valores de diferentes tipos e são construídas usando parênteses. Elas podem ser usadas como argumentos de funções e valores de retorno, permitindo que funções retornem múltiplos valores. Tuplas também podem ser usadas como membros de outras tuplas. Rust fornece indexação de tuplas para acessar valores em uma tupla. Tuplas são imprimíveis e podem ser desestruturadas para criar ligações (bindings). Adicionalmente, aprendemos como adicionar o trait `fmt::Display` a uma struct para personalizar seu formato de impressão. Finalmente, recebemos uma atividade para implementar uma função `transpose` que troca dois elementos em uma matriz.

> **Nota:** Se o laboratório não especificar um nome de arquivo, você pode usar qualquer nome de arquivo que desejar. Por exemplo, você pode usar `main.rs`, compilar e executá-lo com `rustc main.rs && ./main`.
