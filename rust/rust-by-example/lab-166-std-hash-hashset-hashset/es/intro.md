# Introducción

En este laboratorio, exploramos las funcionalidades de la estructura de datos HashSet en Rust, que garantiza elementos únicos y puede realizar operaciones como unión, diferencia, intersección y diferencia simétrica en conjuntos.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
