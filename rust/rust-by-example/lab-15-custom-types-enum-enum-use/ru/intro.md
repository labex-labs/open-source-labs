# Введение

В этом лабораторном задании используется объявление `use` для устранения необходимости вручную определять области видимости в Rust и для доступности имен без явного указания пространства имен.

> **Примечание:** Если лабораторная работа не уточняет имя файла, вы можете использовать любое имя файла. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
