# Einführung

In diesem Lab wird das Konzept der Aliasing im Kontext der Rust-Programmiersprache untersucht. Aliasing bezieht sich auf die Situation, in der mehrere Referenzen auf die gleichen Daten erstellt werden, entweder als unveränderliche oder veränderliche Entleihen.

> **Hinweis:** Wenn das Lab keinen Dateinamen angibt, können Sie einen beliebigen Dateinamen verwenden. Beispielsweise können Sie `main.rs` verwenden, es mit `rustc main.rs &&./main` kompilieren und ausführen.
