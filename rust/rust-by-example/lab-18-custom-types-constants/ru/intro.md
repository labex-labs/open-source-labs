# Введение

В этом лабе мы познакомимся с концепцией констант в Rust, которые можно объявить с использованием ключевых слов `const` или `static`, с явными аннотациями типов, и которые можно использовать в любом скоупе, включая глобальный скоуп.

> **Примечание:** Если в лабе не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
