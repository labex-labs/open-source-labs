# Константы

В Rust есть два разных типа констант, которые можно объявить в любом скоупе, включая глобальный. Оба типа требуют явной аннотации типа:

- `const`: неизменяемое значение (обычный случай).
- `static`: возможно изменяемая переменная с временем жизни `'static`. Жизненный цикл `static` выводится автоматически и не нужно его указывать. Доступ к изменяемой статической переменной или её модификация являются `unsafe` операциями.

```rust
// Глобальные переменные объявляются вне всех других скоупов.
static LANGUAGE: &str = "Rust";
const THRESHOLD: i32 = 10;

fn is_big(n: i32) -> bool {
    // Доступ к константе внутри какой-то функции
    n > THRESHOLD
}

fn main() {
    let n = 16;

    // Доступ к константе в главном потоке
    println!("Это {}", LANGUAGE);
    println!("Порог равен {}", THRESHOLD);
    println!("{} является {}", n, if is_big(n) { "большим" } else { "маленьким" });

    // Ошибка! Нельзя изменить `const`.
    THRESHOLD = 5;
    // FIXME ^ Закомментируйте эту строку
}
```
