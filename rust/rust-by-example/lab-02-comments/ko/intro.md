# 소개

이 랩에서는 일반 주석과 문서 주석을 포함하여 Rust 에서 지원하는 다양한 유형의 주석에 대해 배웁니다. 일반 주석은 두 개의 슬래시로 시작하여 줄 끝까지 이어지는 줄 주석이거나, `/*`와 `*/` 사이에 묶여 중첩될 수 있는 블록 주석일 수 있습니다. 반면에 문서 주석은 HTML 라이브러리 문서를 생성하는 데 사용되며 `///` 또는 `//!`로 시작합니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
