# Введение

В этом лабораторном задании мы узнаем, что в Rust можно использовать предложение `where` для выражения ограничений для обобщенных типов отдельно от их объявления, что позволяет получить более ясный синтаксис, и можно также применять ограничения к произвольным типам, а не только к параметрам типа. Предложение `where` особенно полезно, когда ограничения более выразительны, чем обычный синтаксис, как показано в примере, связанном с треитом `PrintInOption`.

> **Примечание:** Если лабораторное задание не задает имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
