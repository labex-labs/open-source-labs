# Introdução

Neste laboratório, você aprenderá como implementar o trait `fmt::Display` em Rust para personalizar a aparência da saída de uma estrutura. Você também explorará a diferença entre `fmt::Display` e `fmt::Debug`, bem como as limitações de `fmt::Display` para tipos de contêiner genéricos. Finalmente, você terá uma atividade para implementar o trait `fmt::Display` para uma nova estrutura `Complex` e imprimi-la em um formato específico.

> **Nota:** Se o laboratório não especificar um nome de arquivo, você pode usar qualquer nome de arquivo que desejar. Por exemplo, você pode usar `main.rs`, compilar e executá-lo com `rustc main.rs && ./main`.
