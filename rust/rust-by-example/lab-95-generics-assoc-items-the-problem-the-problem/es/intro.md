# Introducción

En este laboratorio, tenemos un trato llamado `Contains` que es genérico sobre su tipo de contenedor y requiere la especificación explícita de sus tipos genéricos cuando se implementa para el tipo `Container`.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
