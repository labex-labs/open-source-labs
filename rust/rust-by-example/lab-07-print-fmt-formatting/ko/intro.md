# 소개

이 랩에서는 Rust 의 형식 지정 (formatting) 에 대해 배우고, `format!` 매크로를 사용하여 변수의 형식을 지정하는 방법을 익혔습니다. 형식 지정은 형식 문자열 (format string) 을 사용하여 지정되며, 서로 다른 인수 유형 (argument types) 을 사용하여 동일한 변수를 다양한 방식으로 형식 지정할 수 있다는 것을 확인했습니다. 가장 일반적인 형식 지정 트레이트 (formatting trait) 는 `Display`로, 인수 유형이 지정되지 않은 경우를 처리합니다. `City` 구조체 (struct) 에 대해 `Display` 트레이트를 구현하는 예시를 살펴보았으며, 위도와 경도 값을 형식 지정했습니다. 또한 `Color` 구조체의 예시를 보았고, RGB 값과 16 진수 표현을 표시하기 위해 `Display` 트레이트를 구현하는 과제를 수행했습니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우, 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
