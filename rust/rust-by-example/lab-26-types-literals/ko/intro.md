# 소개

이 실습에서는 Rust 에서 리터럴과 접미사를 사용하여 타입을 명시하는 방법을 배웁니다. 접미사가 있는 리터럴은 초기화 시 타입이 결정되는 반면, 접미사가 없는 리터럴의 타입은 사용 방법에 따라 달라집니다. `size_of_val` 함수는 변수의 크기를 바이트 단위로 결정하는 데 사용되며, 전체 경로 `std::mem::size_of_val`로 호출됩니다. `size_of_val` 함수는 `mem` 모듈에 정의되어 있으며, `mem` 모듈은 `std` 크레이트에 정의되어 있습니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
