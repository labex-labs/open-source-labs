# Introdução

Neste laboratório, exploramos como usar combinadores como `map` e `and_then` para o tipo `Result` em Rust para lidar com erros de forma mais elegante, conforme demonstrado nos exemplos fornecidos.

> **Nota:** Se o laboratório não especificar um nome de arquivo, você pode usar qualquer nome de arquivo que desejar. Por exemplo, você pode usar `main.rs`, compilar e executá-lo com `rustc main.rs && ./main`.
