# Introducción

En este laboratorio, exploramos cómo usar combinadores como `map` y `and_then` para el tipo `Result` en Rust para manejar errores de manera más elegante, como se demuestra en los ejemplos proporcionados.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
