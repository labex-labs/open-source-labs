# 式

Rust プログラムは（主に）一連の文で構成されています。

```rust
fn main() {
    // 文
    // 文
    // 文
}
```

Rust にはいくつかの種類の文があります。最も一般的な 2 つは、変数束縛を宣言することと、式に `;` を付けることです。

```rust
fn main() {
    // 変数束縛
    let x = 5;

    // 式;
    x;
    x + 1;
    15;
}
```

ブロックも式であるため、代入で値として使用できます。ブロック内の最後の式は、ローカル変数などの場所式に割り当てられます。ただし、ブロックの最後の式がセミコロンで終わる場合、戻り値は `()` になります。

```rust
fn main() {
    let x = 5u32;

    let y = {
        let x_squared = x * x;
        let x_cube = x_squared * x;

        // この式が `y` に割り当てられます
        x_cube + x_squared + x
    };

    let z = {
        // セミコロンによりこの式が抑制され、`()` が `z` に割り当てられます
        2 * x;
    };

    println!("x is {:?}", x);
    println!("y is {:?}", y);
    println!("z is {:?}", z);
}
```
