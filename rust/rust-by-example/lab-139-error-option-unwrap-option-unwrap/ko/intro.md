# 소개

이 랩에서는 Rust 의 `std` 라이브러리에 있는 `Option` 열거형에 대해 배우며, 이는 부재 가능성을 처리하는 데 사용됩니다. `Option`은 두 가지 옵션을 제공합니다: 타입 `T`의 요소가 발견될 때 `Some(T)`를, 요소가 발견되지 않을 때 `None`을 사용합니다. 이러한 경우는 `match`를 사용하여 명시적으로 처리하거나 `unwrap`을 사용하여 암시적으로 처리할 수 있습니다. 명시적 처리는 더 큰 제어력과 의미 있는 출력을 가능하게 하는 반면, `unwrap`은 내부 요소를 반환하거나 패닉을 유발할 수 있습니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우, 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
