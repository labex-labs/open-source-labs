# Introducción

En este laboratorio, aprendemos sobre el atributo `#[derive]` en Rust, que permite al compilador proporcionar implementaciones básicas para ciertos tratos como `Eq`, `PartialEq`, `Ord`, `PartialOrd`, `Clone`, `Copy`, `Hash`, `Default` y `Debug`. Estos tratos también se pueden implementar manualmente si se necesita un comportamiento más complejo. El laboratorio proporciona código de ejemplo que muestra el uso de estos tratos en diferentes structs de tupla como `Centimeters`, `Inches` y `Seconds`.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
