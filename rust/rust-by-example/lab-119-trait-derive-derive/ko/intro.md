# 소개

이 랩에서는 Rust 의 `#[derive]` 속성에 대해 배우게 됩니다. 이 속성은 컴파일러가 `Eq`, `PartialEq`, `Ord`, `PartialOrd`, `Clone`, `Copy`, `Hash`, `Default`, 그리고 `Debug`와 같은 특정 트레이트에 대한 기본적인 구현을 제공하도록 합니다. 이러한 트레이트는 더 복잡한 동작이 필요할 경우 수동으로 구현할 수도 있습니다. 이 랩에서는 `Centimeters`, `Inches`, 그리고 `Seconds`와 같은 다양한 튜플 구조체에 이러한 트레이트를 사용하는 예제 코드를 제공합니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우, 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
