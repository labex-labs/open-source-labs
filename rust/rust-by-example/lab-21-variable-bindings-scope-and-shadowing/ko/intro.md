# 소개

이 실습에서는 Rust 에서 변수 바인딩, 그 범위, 그리고 그림자화 (shadowing) 개념을 학습합니다. 변수 바인딩은 중괄호로 묶인 문장들의 집합인 블록 내에서 제한됩니다. 이러한 개념을 설명하기 위해 두 가지 예제가 제공됩니다. 첫 번째 예제는 블록 내에서 선언된 변수 바인딩이 해당 블록의 범위에 국한되고 블록 외부에서는 접근할 수 없음을 보여줍니다. 두 번째 예제는 블록 내에서 동일한 이름을 가진 새로운 바인딩이 선언되어 외부 바인딩을 효과적으로 그림자화하는 변수 그림자화를 보여줍니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
