# 소개

이 실습에서는 Rust 에서 연관된 타입 (associated types) 의 개념을 탐구합니다. 연관된 타입은 특정 트레이트 내에서 출력 타입으로 내부 타입을 로컬하게 정의하여 코드 가독성을 향상시키는 기능을 제공합니다. 이는 트레이트 정의 내에서 `type` 키워드를 사용하여 달성됩니다. 연관된 타입을 사용하면 트레이트를 사용하는 함수가 더 이상 타입 `A`와 `B`를 명시적으로 표현할 필요가 없어 코드를 더 간결하고 유연하게 만들 수 있습니다. 이전 섹션의 예제를 연관된 타입을 사용하여 다시 작성하여 실제 사용법을 보여줍니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
