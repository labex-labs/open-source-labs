# 소개

이 실습에서는 Rust 에서 `TryFrom`과 `TryInto`의 사용법을 살펴봅니다. 이들은 타입 간의 오류 가능한 변환을 수행하고 `Result` 타입을 반환하는 데 사용되는 제네릭 트레이트입니다. `i32`를 사용자 정의 `EvenNumber` 구조체로 변환하는 `TryFrom` 구현을 보여주는 예제 코드를 제공하고, `TryFrom`과 `TryInto`를 사용하여 변환을 수행하고 가능한 오류를 처리하는 방법을 보여줍니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
