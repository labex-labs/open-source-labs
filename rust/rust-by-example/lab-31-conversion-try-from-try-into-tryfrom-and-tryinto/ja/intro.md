# はじめに

この実験では、Rust における`TryFrom`と`TryInto`の使い方を調べます。これらは、型間のエラーが発生する変換に使用されるジェネリックトレイトで、`Result`型を返します。`i32`をカスタムの`EvenNumber`構造体に変換するための`TryFrom`の実装を示すコードスニペットを提供し、その後、`TryFrom`と`TryInto`を使って変換を行い、発生する可能性のあるエラーを処理する方法を紹介します。

> **注：** 実験でファイル名が指定されていない場合、好きなファイル名を使うことができます。たとえば、`main.rs`を使って、`rustc main.rs &&./main`でコンパイルして実行することができます。
