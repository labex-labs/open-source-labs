# Introducción

En este laboratorio, se demuestra el uso de la estructura `std::Child` para representar un proceso hijo en ejecución y su interacción a través de tuberías con los manejadores `stdin`, `stdout` y `stderr`.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
