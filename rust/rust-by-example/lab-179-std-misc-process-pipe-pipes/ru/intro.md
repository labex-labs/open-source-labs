# Введение

В этом лабораторном задании демонстрируется использование структуры `std::Child` для представления запущенного дочернего процесса и его взаимодействие через трубы с дескрипторами `stdin`, `stdout` и `stderr`.

> **Примечание:** Если лабораторная работа не уточняет имя файла, вы можете использовать любое имя, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
