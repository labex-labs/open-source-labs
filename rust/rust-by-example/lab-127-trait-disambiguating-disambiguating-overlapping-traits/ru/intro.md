# Введение

В этом лабораторном задании рассматривается проблема устранения неоднозначности перекрывающихся характеристик, когда две характеристики имеют методы с одинаковым именем и потенциально разными типами возвращаемых значений.

> **Примечание:** Если лабораторное задание не задает имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
