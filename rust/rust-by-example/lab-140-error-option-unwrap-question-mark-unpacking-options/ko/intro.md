# 소개

이 랩에서는 Rust 의 `?` 연산자를 사용하여 중첩된 `match` 문 없이도 `Option` 값을 쉽게 언패킹하는 방법을 살펴봅니다. `?` 연산자는 `Option`이 `Some`인 경우 기본 값을 빠르게 반환하고, `Option`이 `None`인 경우 함수를 종료하고 `None`을 반환하는 데 사용될 수 있습니다. 이 연산자는 코드를 더 읽기 쉽고 간결하게 만들기 위해 함께 연결될 수 있습니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
