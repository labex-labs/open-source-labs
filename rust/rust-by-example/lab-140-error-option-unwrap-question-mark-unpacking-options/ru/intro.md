# Введение

В этом лабораторном задании мы исследуем использование оператора `?` в Rust, который позволяет легко извлекать значения `Option` без необходимости вложенных инструкций `match`. Оператор `?` можно использовать для быстрого возврата внутреннего значения, если `Option` имеет значение `Some`, или завершения функции и возврата `None`, если `Option` имеет значение `None`. Этот оператор можно объединять в цепочки, чтобы сделать код более читаемым и компактным.

> **Примечание:** Если лабораторное задание не задает имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
