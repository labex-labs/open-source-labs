# Introdução

Neste laboratório, para ligar um _crate_ a uma nova biblioteca em Rust, pode utilizar o sinalizador `--extern` do comando `rustc` e importar todos os seus itens sob um módulo com o mesmo nome da biblioteca.

> **Nota:** Se o laboratório não especificar um nome de ficheiro, pode utilizar qualquer nome de ficheiro que desejar. Por exemplo, pode utilizar `main.rs`, compilá-lo e executá-lo com `rustc main.rs && ./main`.
