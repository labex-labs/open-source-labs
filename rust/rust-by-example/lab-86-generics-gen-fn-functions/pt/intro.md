# Introdução

Neste laboratório, você aprenderá como definir funções genéricas em Rust. Para tornar uma função genérica, você precisa prefixar o tipo `T` com `<T>`. Às vezes, você pode precisar especificar explicitamente os parâmetros de tipo ao chamar uma função genérica, o que pode ser feito usando a sintaxe `fun::<A, B, ...>()`. O código fornecido demonstra o uso de funções genéricas em Rust e inclui exemplos de funções que são genéricas e não genéricas.

> **Nota:** Se o laboratório não especificar um nome de arquivo, você pode usar qualquer nome de arquivo que desejar. Por exemplo, você pode usar `main.rs`, compilar e executá-lo com `rustc main.rs && ./main`.
