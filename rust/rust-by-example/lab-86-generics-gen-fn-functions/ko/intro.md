# 소개

이 실습에서는 Rust 에서 제네릭 함수를 정의하는 방법을 배웁니다. 함수를 제네릭으로 만들려면 타입 `T`를 `<T>`와 함께 사용해야 합니다. 때로는 제네릭 함수를 호출할 때 타입 매개변수를 명시적으로 지정해야 할 수 있으며, 이는 `fun::<A, B, ...>()` 구문을 사용하여 수행할 수 있습니다. 제공된 코드는 Rust 에서 제네릭 함수를 사용하는 방법을 보여주며, 제네릭 함수와 일반 함수의 예제를 포함합니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
