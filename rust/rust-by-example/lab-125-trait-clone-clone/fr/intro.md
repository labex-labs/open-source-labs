# Introduction

Dans ce laboratoire, nous apprenons à utiliser le trait `Clone` en Rust pour créer des copies de ressources, telles que des structs, en utilisant la méthode `.clone()`. Le trait `Clone` nous permet de créer des copies indépendantes de ressources, et nous pouvons utiliser la méthode `.clone()` pour créer une nouvelle instance avec les mêmes valeurs.

> **Note** : Si le nom de fichier n'est pas spécifié dans le laboratoire, vous pouvez utiliser n'importe quel nom de fichier que vous voulez. Par exemple, vous pouvez utiliser `main.rs`, le compiler et l'exécuter avec `rustc main.rs &&./main`.
