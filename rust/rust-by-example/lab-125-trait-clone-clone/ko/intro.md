# 소개

이 랩에서는 Rust 에서 `Clone` 트레이트를 사용하여 구조체 (struct) 와 같은 리소스를 `.clone()` 메서드를 통해 복사하는 방법을 배웁니다. `Clone` 트레이트를 사용하면 리소스의 독립적인 복사본을 생성할 수 있으며, `.clone()` 메서드를 사용하여 동일한 값을 가진 새로운 인스턴스를 만들 수 있습니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
