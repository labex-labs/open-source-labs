# 简介

在本实验中，我们将探索 Rust 中字符串的概念。Rust 有两种类型的字符串：`String` 和 `&str`。

`String` 是一个在堆上分配的、可增长的字符串，它保证是一个有效的 UTF-8 序列。另一方面，`&str` 是一个指向有效 UTF-8 序列的切片，可用于查看 `String`。

在 Rust 中，字符串字面量可以用不同的方式编写，包括使用转义字符来表示特殊字符。例如，`\x3F` 表示问号字符，`\u{211D}` 表示一个 Unicode 代码点。如果你想原样编写字符串而不使用转义字符，也可以使用原始字符串字面量。

如果你需要处理字节字符串，Rust 提供了使用 `b` 前缀的字节字符串字面量。字节字符串可以有字节转义，但不能有 Unicode 转义。原始字节字符串也可以以类似于原始字符串字面量的方式使用。

需要注意的是，`str` 和 `String` 必须始终是有效的 UTF-8 序列。如果你需要处理不同编码的字符串，可以使用像 `encoding` 这样的外部 crate 进行字符编码之间的转换。
