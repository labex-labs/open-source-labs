# 소개

이 실습에서는 파일을 쓰기 전용 모드로 열고, 파일이 이미 존재하면 기존 내용을 삭제하고 새 파일을 생성하는 `create` 함수를 사용합니다. 이 함수는 Rust 의 표준 라이브러리를 사용하여 파일 작업을 처리합니다. 제공된 예제는 `create` 함수를 사용하여 정적 `LOREM_IPSUM` 문자열의 내용을 "lorem_ipsum.txt"라는 파일에 쓰는 방법을 보여줍니다. 출력은 성공적인 쓰기 작업 확인을 보여주며, `cat` 명령어를 사용하여 파일 내용을 표시합니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
