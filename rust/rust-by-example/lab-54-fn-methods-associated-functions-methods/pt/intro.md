# Introdução

Neste laboratório, definimos uma estrutura `Point` com funções associadas `origin()` e `new()`, e uma estrutura `Rectangle` com métodos `area()`, `perimeter()` e `translate()`. Também introduzimos uma estrutura `Pair` com um método `destroy()`. Demonstramos como usar essas funções e métodos criando instâncias das estruturas e chamando as funções e métodos respectivos nelas.

> **Nota:** Se o laboratório não especificar um nome de ficheiro, pode usar qualquer nome de ficheiro que desejar. Por exemplo, pode usar `main.rs`, compilá-lo e executá-lo com `rustc main.rs && ./main`.
