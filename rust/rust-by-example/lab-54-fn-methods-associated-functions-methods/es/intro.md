# Introducción

En este laboratorio, definimos una estructura `Point` con las funciones asociadas `origin()` y `new()`, y una estructura `Rectangle` con los métodos `area()`, `perímetro()` y `translate()`. También introducimos una estructura `Pair` con un método `destroy()`. Demostramos cómo utilizar estas funciones y métodos creando instancias de las estructuras y llamando a las funciones y métodos respectivos sobre ellas.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede utilizar cualquier nombre de archivo que desee. Por ejemplo, puede utilizar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
