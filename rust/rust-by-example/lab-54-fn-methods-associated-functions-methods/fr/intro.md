# Introduction

Dans ce laboratoire, nous définissons une structure `Point` avec les fonctions associées `origin()` et `new()`, et une structure `Rectangle` avec les méthodes `area()`, `perimeter()` et `translate()`. Nous introduisons également une structure `Pair` avec une méthode `destroy()`. Nous montrons comment utiliser ces fonctions et méthodes en créant des instances des structures et en appelant les fonctions et méthodes respectives sur celles-ci.

> **Note** : Si le laboratoire ne spécifie pas de nom de fichier, vous pouvez utiliser n'importe quel nom de fichier que vous voulez. Par exemple, vous pouvez utiliser `main.rs`, le compiler et l'exécuter avec `rustc main.rs &&./main`.
