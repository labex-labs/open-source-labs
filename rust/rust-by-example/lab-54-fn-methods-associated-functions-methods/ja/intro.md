# はじめに

この実験では、関連付けられた関数 `origin()` と `new()` を持つ `Point` 構造体と、メソッド `area()`、`perimeter()`、および `translate()` を持つ `Rectangle` 構造体を定義します。また、メソッド `destroy()` を持つ `Pair` 構造体も紹介します。構造体のインスタンスを作成し、それらに対してそれぞれの関数とメソッドを呼び出すことで、これらの関数とメソッドの使い方を示します。

> **注:** 実験でファイル名が指定されていない場合、好きなファイル名を使うことができます。たとえば、`main.rs` を使って、`rustc main.rs &&./main` でコンパイルして実行することができます。
