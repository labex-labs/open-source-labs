# 소개

이 실습에서는 Point 구조체와 관련 함수 origin() 및 new(), Rectangle 구조체와 메서드 area(), perimeter(), translate(), 그리고 Pair 구조체와 메서드 destroy() 를 정의합니다. 이러한 구조체의 인스턴스를 생성하고 각 함수와 메서드를 호출하여 사용하는 방법을 보여줍니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
