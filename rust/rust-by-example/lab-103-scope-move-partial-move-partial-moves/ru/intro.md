# Введение

В этом лабораторном задании мы изучаем частичные перемещения при деструктуризации одной переменной, при этом одновременно можно использовать как `by-move`, так и `by-reference` паттерны связывания. Это приводит к частичному перемещению переменной, при котором некоторые части могут быть перемещены, а другие по-прежнему могут быть использованы для обращения. Если родительская переменная частично перемещена, то после этого ее нельзя использовать в целом, но части, которые только используются для обращения и не перемещаются, по-прежнему могут быть использованы.

> **Примечание:** Если в лабораторном задании не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
