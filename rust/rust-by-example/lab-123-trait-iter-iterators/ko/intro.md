# 소개

이 랩에서는 Rust 의 `Iterator` 트레이트를 배우고, 이는 배열과 같은 컬렉션에 대한 이터레이터를 구현하는 데 사용됩니다. `Iterator` 트레이트는 이터레이터에 대해 `next` 메서드를 정의해야 하며, `impl` 블록에서 수동으로 구현하거나 배열 및 범위에 대해 자동으로 정의할 수 있습니다. `for` 구문을 사용하여 `.into_iter()` 메서드를 사용하여 일부 컬렉션을 편리하게 이터레이터로 변환할 수 있습니다. 이 랩에서는 `next` 메서드를 정의하고 `Iterator` 트레이트를 사용하는 방법을 보여주는 이터레이터로서 `Fibonacci` 수열 생성기의 예시 구현을 제공합니다. 또한, 이터레이터를 조작하기 위한 `take` 및 `skip` 메서드의 사용법과 배열을 반복하기 위한 `iter` 메서드를 보여줍니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우, 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
