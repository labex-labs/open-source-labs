# Introducción

En este laboratorio, aprendemos sobre la visibilidad de los elementos en los módulos de Rust, incluyendo la visibilidad privada predeterminada, el uso del modificador `pub` para anular la visibilidad y los diferentes niveles de visibilidad como `pub(in path)`, `pub(self)`, `pub(super)` y `pub(crate)`. También exploramos los módulos anidados y las restricciones para acceder a elementos privados dentro de los módulos.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
