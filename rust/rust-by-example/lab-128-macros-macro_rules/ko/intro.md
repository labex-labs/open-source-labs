# 소개

이 랩에서는 Rust 가 제공하는 강력한 매크로 시스템을 탐구할 것입니다. 이 시스템은 매크로를 추상 구문 트리 (Abstract Syntax Trees, AST) 로 확장하여 메타 프로그래밍을 가능하게 합니다. `macro_rules!` 매크로는 매크로를 생성하는 데 사용되며, 함수와는 달리 느낌표 `!`로 끝나는 특징을 가지고 있습니다. 매크로는 코드 중복을 피하고, 도메인 특화 언어 (Domain-Specific Languages, DSL) 를 만들며, 가변 인자를 받는 함수를 위한 가변 인터페이스를 정의하는 데 유용합니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우, 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
