# Введение

В этом лабораторном задании мы будем изучать использование встроенного ассемблера в Rust с использованием макроса `asm!`. Мы рассмотрим базовое использование встроенного ассемблера, входы и выходы, операнды позднего вывода, явные операнды регистров, замаскированные регистры, операнды символов и ABI-замаскировки, модификаторы шаблонов регистров, операнды адресов памяти, метки и параметры для оптимизации кода ассемблера.

> **Примечание:** Если в лабораторном задании не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
