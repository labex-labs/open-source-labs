# 소개

이 실습에서는 Rust 에서 `asm!` 매크로를 사용하여 인라인 어셈블리를 사용하는 방법을 탐색합니다. 인라인 어셈블리의 기본 사용법, 입력 및 출력, 늦은 출력 피연산자, 명시적인 레지스터 피연산자, 덮어쓰기 레지스터, 심볼 피연산자 및 ABI 덮어쓰기, 레지스터 템플릿 수정자, 메모리 주소 피연산자, 레이블 및 어셈블리 코드 최적화 옵션을 다룰 것입니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
