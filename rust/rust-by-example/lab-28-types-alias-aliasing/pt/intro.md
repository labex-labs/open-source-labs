# Introdução

Neste laboratório, aprendemos sobre aliases em Rust, que nos permitem dar um novo nome a um tipo existente usando a declaração `type`. Os aliases devem seguir certas convenções de nomenclatura e podem ser usados para criar novos nomes para tipos primitivos ou tipos definidos pelo utilizador. O principal propósito dos aliases é reduzir a duplicação de código e melhorar a legibilidade.

> **Nota:** Se o laboratório não especificar um nome de ficheiro, pode usar qualquer nome de ficheiro que desejar. Por exemplo, pode usar `main.rs`, compilá-lo e executá-lo com `rustc main.rs && ./main`.
