# Псевдонимы

Инструкция `type` может использоваться для присвоения нового имени существующему типу. Типы должны иметь имена в стиле `UpperCamelCase`, иначе компилятор выдаст предупреждение. Исключение из этого правила составляют примитивные типы: `usize`, `f32` и т.д.

```rust
// `NanoSecond`, `Inch` и `U64` - новые имена для `u64`.
type NanoSecond = u64;
type Inch = u64;
type U64 = u64;

fn main() {
    // `NanoSecond` = `Inch` = `U64` = `u64`.
    let nanoseconds: NanoSecond = 5 as U64;
    let inches: Inch = 2 as U64;

    // Обратите внимание, что псевдонимы *не* обеспечивают дополнительной безопасности типов, потому что
    // псевдонимы *не* являются новыми типами
    println!("{} наносекунд + {} дюймов = {} единиц?",
             nanoseconds,
             inches,
             nanoseconds + inches);
}
```

Основное применение псевдонимов - сокращение избыточного кода; например, тип `io::Result<T>` является псевдонимом для типа `Result<T, io::Error>`.
