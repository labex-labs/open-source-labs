# Введение

В этом лабораторном задании рассматривается реализация трейта `Add` с параметром типа-призраком с использованием примера кода на Rust.

> **Примечание:** Если в лабораторном задании не указано имя файла, вы можете использовать любое имя файла. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
