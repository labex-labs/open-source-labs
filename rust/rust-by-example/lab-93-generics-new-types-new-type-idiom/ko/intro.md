# 소개

이 실습에서는 `newtype` 관용구를 탐색합니다. `newtype` 관용구는 기본 타입과는 다른 새로운 타입을 생성하여 컴파일 시점 보장을 제공합니다. 예제에서는 나이를 연 단위로 나타내는 `Years` 구조체와 일 단위로 나타내는 `Days` 구조체를 사용하는 것을 보여줍니다. `newtype` 관용구를 사용하면 `Years` 타입의 값이 필요한 나이 확인 함수 `old_enough`와 같이 프로그램에 올바른 타입의 값이 제공되도록 보장할 수 있습니다. 또한 튜플 또는 구조 분해 구문을 사용하여 `newtype`의 값을 기본 타입으로 얻는 방법을 배웁니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs` 파일을 사용하고 `rustc main.rs && ./main` 명령으로 컴파일 및 실행할 수 있습니다.
