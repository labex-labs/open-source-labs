# Введение

В этом практическом занятии рассматривается модуль `std::fs` в Rust, который предоставляет функции для операций с файловой системой. В практическом занятии приводятся примеры различных операций с файловой системой, включая создание директорий, создание файлов, чтение содержимого файлов, создание символических ссылок, перечисление содержимого директории, удаление файлов и удаление директорий. В примерах кода показано, как выполнять эти операции с использованием функций модуля `std::fs`, и для каждой операции представлен ожидаемый вывод. Кроме того, упоминается альтернативная реализация функции `cat` с использованием нотации `?` для обработки ошибок.

> **Примечание:** Если практическое занятие не уточняет имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью команд `rustc main.rs &&./main`.
