# Аргументы программы

## Стандартная библиотека

Аргументы командной строки можно получить с помощью `std::env::args`, которая возвращает итератор, выдающий `String` для каждого аргумента:

```rust
use std::env;

fn main() {
    let args: Vec<String> = env::args().collect();

    // Первым аргументом является путь, по которому вызывалась программа.
    println!("Мой путь - {}.", args[0]);

    // Остальные аргументы - это переданные параметры командной строки.
    // Вызовите программу так:
    //   $./args arg1 arg2
    println!("Я получил {:?} аргументов: {:?}.", args.len() - 1, &args[1..]);
}
```

```shell
$./args 1 2 3
My path is./args.
I got 3 arguments: ["1", "2", "3"].
```

## Пакеты

В качестве альтернативы существуют многочисленные пакеты, которые могут обеспечить дополнительную функциональность при создании командных приложений. В \[Rust Cookbook\] показаны наилучшие практики по использованию одного из более популярных пакетов для обработки аргументов командной строки - `clap`.
