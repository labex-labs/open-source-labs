# Introducción

En este laboratorio, exploramos el concepto de rasgos en Rust, que son una colección de métodos definidos para un tipo desconocido llamado `Self`. Los rasgos se pueden implementar para cualquier tipo de datos, lo que permite el uso de los métodos definidos dentro del rasgo.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
