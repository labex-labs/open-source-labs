# Введение

В этом лабе мы исследуем концепцию трейтов в Rust, которые представляют собой коллекцию методов, определенных для неизвестного типа, называемого `Self`. Трейты могут быть реализованы для любого типа данных, что позволяет использовать методы, определенные внутри трейта.

> **Примечание:** Если в лабе не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
