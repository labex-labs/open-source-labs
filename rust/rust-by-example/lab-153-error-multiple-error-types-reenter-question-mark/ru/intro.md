# Введение

В этом лабе объясняется, что оператор `?` в Rust может быть использован для автоматического преобразования ошибок в желаемый тип возврата, устраняя необходимость явной обработки ошибок с использованием `map_err`.

> **Примечание:** Если лаб не задает имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
