# Introducción

En este laboratorio, se explica que el operador `?` en Rust se puede utilizar para convertir automáticamente errores en el tipo de retorno deseado, eliminando la necesidad de manejar errores explícitamente utilizando `map_err`.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede utilizar cualquier nombre de archivo que desee. Por ejemplo, puede utilizar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
