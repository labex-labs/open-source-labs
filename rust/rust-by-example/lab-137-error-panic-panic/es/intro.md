# Introducción

En este laboratorio, aprendemos sobre el mecanismo de manejo de errores `panic` en Rust. Es un mecanismo simple que imprime un mensaje de error, deshace la pila y generalmente sale del programa. El laboratorio proporciona un ejemplo donde se llama explícitamente a `panic` cuando la bebida es "limonada".

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilarlo y ejecutarlo con `rustc main.rs &&./main`.
