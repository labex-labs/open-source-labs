# Введение

В этом лабораторном задании мы изучаем механизм обработки ошибок `panic` в Rust. Это простой механизм, который выводит сообщение об ошибке, разматывает стек и обычно завершает программу. В лабораторном задании приводится пример, в котором явно вызывается `panic`, когда напиток - это «лимонад».

> **Примечание:** Если лабораторное задание не задает имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
