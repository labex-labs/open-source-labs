# Introducción

En este laboratorio, aprenderá sobre el concepto de retornar desde bucles en Rust, que implica reintentar una operación hasta que tenga éxito y pasar el valor resultante al resto del código colocándolo después de la declaración `break` dentro de la expresión `loop`.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
