# Возвращение из циклов

Одно из применений цикла `loop` - это повторение операции до тех пор, пока она не будет успешной. Однако, если операция возвращает значение, вам может потребоваться передать его в остальную часть кода: поместите его после `break`, и оно будет возвращено выражением `loop`.

```rust
fn main() {
    let mut counter = 0;

    let result = loop {
        counter += 1;

        if counter == 10 {
            break counter * 2;
        }
    };

    assert_eq!(result, 20);
}
```
