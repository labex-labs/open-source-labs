# Введение

В этом практическом занятии вы узнаете о концепции возврата из циклов в Rust, которая включает повторение операции до тех пор, пока она не будет успешной, и передачу результирующего значения в остальную часть кода, поместив его после инструкции `break` внутри выражения `loop`.

> **Примечание:** Если в практическом занятии не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
