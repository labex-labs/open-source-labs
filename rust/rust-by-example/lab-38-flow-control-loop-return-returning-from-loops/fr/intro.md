# Introduction

Dans ce laboratoire, vous allez découvrir le concept de retour de boucles en Rust, qui consiste à réessayer une opération jusqu'à ce qu'elle réussisse et à passer la valeur résultante au reste du code en la plaçant après l'instruction `break` à l'intérieur de l'expression `loop`.

> **Note** : Si le nom de fichier n'est pas spécifié dans le laboratoire, vous pouvez utiliser n'importe quel nom de fichier que vous voulez. Par exemple, vous pouvez utiliser `main.rs`, le compiler et l'exécuter avec `rustc main.rs &&./main`.
