# 소개

이 실습에서는 Rust 의 `HashMap`에서 대체/사용자 지정 키 유형을 사용하는 방법을 살펴봅니다. 이 유형에는 `Eq` 및 `Hash` 트레이트를 구현하는 유형, 예를 들어 `bool`, `int`, `uint`, `String`, `&str`이 포함될 수 있습니다. 또한 `#[derive(PartialEq, Eq, Hash)]` 속성을 사용하여 사용자 지정 유형에 이러한 트레이트를 구현할 수 있으며, 이를 `HashMap`의 키로 사용할 수 있습니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
