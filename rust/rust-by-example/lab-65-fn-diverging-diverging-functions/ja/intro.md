# はじめに

この実験では、Rust で `!` でマークされた発散関数について学びます。発散関数は決して戻りませんし、その戻り値の型は空の型です。これは、ただ1つの可能な値しか持たない `()` 型とは異なります。発散関数は、`match` ブランチのように、他の任意の型にキャストする必要がある場合に便利です。また、無限ループまたはプロセスを終了する関数の戻り値の型でもあります。

> **注:** 実験でファイル名が指定されていない場合、好きなファイル名を使うことができます。たとえば、`main.rs` を使って、`rustc main.rs &&./main` でコンパイルして実行することができます。
