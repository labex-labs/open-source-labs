# Introduction

Dans ce laboratoire, vous pouvez découvrir comment les tableaux et les slices peuvent être décomposés en Rust, démontré à travers un exemple de code qui met en évidence différents modèles pour correspondre des valeurs dans le tableau et les lier à des variables.

> **Note** : Si le laboratoire ne spécifie pas de nom de fichier, vous pouvez utiliser n'importe quel nom de fichier que vous voulez. Par exemple, vous pouvez utiliser `main.rs`, le compiler et l'exécuter avec `rustc main.rs &&./main`.
