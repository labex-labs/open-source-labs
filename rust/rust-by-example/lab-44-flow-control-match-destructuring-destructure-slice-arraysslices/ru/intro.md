# Введение

В этом лабе вы можете узнать, как в Rust можно деструктурировать массивы и срезы, продемонстрировано на примере кода, который показывает разные паттерны для сопоставления значений в массиве и связывания их с переменными.

> **Примечание:** Если в лабе не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
