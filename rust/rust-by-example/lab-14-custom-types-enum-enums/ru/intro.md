# Введение

В этом практическом занятии мы изучаем перечисления (`enum`) в Rust, которые позволяют создавать тип, который может иметь несколько вариантов, в том числе варианты, похожие на единицы (`unit`), кортежные структуры (`tuple structs`) и структуры, похожие на C (`c-like structures`). Мы также узнаем, как определить варианты перечисления и использовать сопоставление с образцом (`pattern matching`) для обработки различных вариантов. Кроме того, мы исследуем псевдонимы типов (`type aliases`), которые предоставляют способ ссылаться на каждый вариант перечисления через псевдоним, делая код более компактным и читаемым.

> **Примечание:** Если практическое занятие не задает имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
