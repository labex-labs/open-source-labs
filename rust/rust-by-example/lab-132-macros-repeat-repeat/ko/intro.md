# 소개

이 랩에서는 인자 목록에서 반복을 나타내기 위해 `+` 및 `*` 기호를 매크로와 함께 사용하는 방법을 배우게 됩니다. 특히, 제공된 예제는 쉼표로 구분된 하나 이상의 표현식을 일치시키기 위해 `$(...),+`를 사용하는 방법을 보여줍니다. `find_min!` 매크로는 가변 개수의 인자 중에서 최소값을 계산합니다.

> **참고:** 랩에서 파일 이름을 지정하지 않은 경우, 원하는 파일 이름을 사용할 수 있습니다. 예를 들어, `main.rs`를 사용하고, `rustc main.rs && ./main`으로 컴파일하고 실행할 수 있습니다.
