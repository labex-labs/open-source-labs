# 반복 (Repeat)

매크로는 인자 목록에서 `+`를 사용하여 인자가 최소 한 번 이상 반복될 수 있음을 나타내거나, `*`를 사용하여 인자가 0 번 이상 반복될 수 있음을 나타낼 수 있습니다.

다음 예제에서 매처를 `$(...),+`로 묶으면 쉼표로 구분된 하나 이상의 표현식과 일치합니다. 또한 마지막 경우에 세미콜론은 선택 사항입니다.

```rust
// `find_min!` 은 임의의 수의 인자 중 최소값을 계산합니다.
macro_rules! find_min {
    // 기본 사례:
    ($x:expr) => ($x);
    // `$x` 다음에 하나 이상의 `$y,` 가 옵니다.
    ($x:expr, $($y:expr),+) => (
        // 꼬리 부분 `$y` 에 대해 `find_min!` 을 호출합니다.
        std::cmp::min($x, find_min!($($y),+))
    )
}

fn main() {
    println!("{}", find_min!(1));
    println!("{}", find_min!(1 + 2, 2));
    println!("{}", find_min!(5, 2 * 3, 4));
}
```
