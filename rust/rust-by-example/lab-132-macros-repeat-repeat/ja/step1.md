# 繰り返し

マクロは、引数リストにおいて `+` を使用して引数が少なくとも 1 回繰り返すことができることを示すか、または `*` を使用して引数が 0 回以上繰り返すことができることを示すことができます。

次の例では、マッチャを `$(...),+` で囲むことで、コンマで区切られた 1 つ以上の式がマッチします。また、最後のケースではセミコロンは省略可能であることにも注意してください。

```rust
// `find_min!` は任意の数の引数の最小値を計算します。
macro_rules! find_min {
    // 基本ケース：
    ($x:expr) => ($x);
    // `$x` の後に少なくとも 1 つの `$y,`
    ($x:expr, $($y:expr),+) => (
        // 末尾の `$y` に対して `find_min!` を呼び出す
        std::cmp::min($x, find_min!($($y),+))
    )
}

fn main() {
    println!("{}", find_min!(1));
    println!("{}", find_min!(1 + 2, 2));
    println!("{}", find_min!(5, 2 * 3, 4));
}
```
