# Введение

В этом практическом занятии вы узнаете о мутабельности в Rust и о том, как переменные можно объявить изменяемыми с помощью модификатора `mut`. В практическом занятии приводится пример объявления неизменяемых и изменяемых связей, демонстрируется, как можно изменить изменяемую связь, и подчеркивается ошибка, возникающая при попытке присвоить новое значение неизменяемой переменной.

> **Примечание:** Если практическое занятие не уточняет имя файла, вы можете использовать любое имя, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью команды `rustc main.rs &&./main`.
