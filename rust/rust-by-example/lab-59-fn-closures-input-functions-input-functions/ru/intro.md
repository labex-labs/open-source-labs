# Введение

В этом лабе вы узнаете, как функции могут принимать замыкания в качестве параметров, что позволяет использовать любую функцию, которая удовлетворяет ограничению типа замыкания, в качестве аргумента. Трейты `Fn`, `FnMut` и `FnOnce` определяют, как замыкание захватывает переменные из окружающей области видимости.

> **Примечание:** Если в лабе не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
