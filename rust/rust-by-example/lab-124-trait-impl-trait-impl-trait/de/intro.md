# Einführung

In diesem Lab werden wir die Verwendung von `impl Trait` in Rust untersuchen, das als Argumenttyp oder Rückgabetyp verwendet werden kann, was die Funktionsdeklarationen vereinfacht und für flexiblere und präzisere Code ermöglicht.

> **Hinweis:** Wenn das Lab keinen Dateinamen angibt, können Sie einen beliebigen Dateinamen verwenden. Beispielsweise können Sie `main.rs` verwenden und es mit `rustc main.rs &&./main` kompilieren und ausführen.
