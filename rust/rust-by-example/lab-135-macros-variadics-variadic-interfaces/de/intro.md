# Einführung

In diesem Lab untersuchen wir variadische Schnittstellen, das sind Schnittstellen, die beliebig viele Argumente akzeptieren können. Als Beispiel ändern wir die `calculate!`-Makro aus dem vorherigen Abschnitt, um es variadisch zu machen, sodass es mehrere Ausdrücke gleichzeitig auswerten kann.

> **Hinweis:** Wenn das Lab keinen Dateinamen angibt, können Sie einen beliebigen Dateinamen verwenden. Beispielsweise können Sie `main.rs` verwenden und es mit `rustc main.rs &&./main` kompilieren und ausführen.
