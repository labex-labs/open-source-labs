# Einführung

In diesem Lab werden Sie über die `Option<T>`-Enumeration in Rust lernen, die zwei Varianten (`None` und `Some(value)`) bietet, um das Versagen oder das Fehlen eines Werts in einem Programm zu behandeln.

> **Hinweis:** Wenn im Lab kein Dateiname angegeben ist, können Sie einen beliebigen Dateinamen verwenden. Beispielsweise können Sie `main.rs` verwenden und es mit `rustc main.rs &&./main` kompilieren und ausführen.
