# Introduction

Dans ce laboratoire, nous explorons l'énumération `Result` en Rust, qui nous permet d'exprimer le succès ou l'échec d'une opération en renvoyant soit `Ok` avec une valeur soit `Err` avec une explication de l'échec.

> **Note** : Si le laboratoire ne spécifie pas de nom de fichier, vous pouvez utiliser n'importe quel nom de fichier que vous voulez. Par exemple, vous pouvez utiliser `main.rs`, le compiler et l'exécuter avec `rustc main.rs &&./main`.
