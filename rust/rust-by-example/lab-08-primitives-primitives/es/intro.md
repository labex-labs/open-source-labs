# Introducción

En este laboratorio, aprenderá sobre las diversas primitivas que ofrece Rust, incluyendo tipos escalares como enteros firmados y no firmados, números de punto flotante, valores escalares Unicode, valores booleanos y el tipo unitario. También explorará tipos compuestos como arrays y tuplas, y aprenderá cómo anotar variables con sus respectivos tipos o permitir que Rust infiera los tipos a partir del contexto.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
