# Введение

В этом лабораторном задании вы узнаете о асинхронных каналах Rust, которые позволяют осуществлять однонаправленную коммуникацию между потоками с использованием точек доступа `Sender` и `Receiver`, как показано в предоставленном фрагменте кода на Rust.

> **Примечание:** Если в лабораторном задании не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
