# Introduction

Dans ce laboratoire, vous allez découvrir les canaux asynchrones de Rust, qui permettent une communication unidirectionnelle entre les threads en utilisant les points de terminaison `Sender` et `Receiver`, comme le montre le extrait de code Rust fourni.

> **Note** : Si le laboratoire ne spécifie pas de nom de fichier, vous pouvez utiliser n'importe quel nom de fichier que vous voulez. Par exemple, vous pouvez utiliser `main.rs`, le compiler et l'exécuter avec `rustc main.rs &&./main`.
