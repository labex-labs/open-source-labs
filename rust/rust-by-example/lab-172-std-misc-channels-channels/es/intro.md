# Introducción

En este laboratorio, aprenderá sobre los canales asincrónicos de Rust, que permiten la comunicación unidireccional entre hilos utilizando los puntos finales `Sender` y `Receiver`, como se demuestra en el fragmento de código de Rust proporcionado.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
