# Einführung

In diesem Lab werden Sie über Rusts asynchrone Kanäle lernen, die eine einseitige Kommunikation zwischen Threads über die `Sender`- und `Receiver`-Endpunkte ermöglichen, wie im bereitgestellten Rust-Codesnippet gezeigt.

> **Hinweis:** Wenn im Lab kein Dateiname angegeben ist, können Sie einen beliebigen Dateinamen verwenden. Beispielsweise können Sie `main.rs` verwenden und es mit `rustc main.rs &&./main` kompilieren und ausführen.
