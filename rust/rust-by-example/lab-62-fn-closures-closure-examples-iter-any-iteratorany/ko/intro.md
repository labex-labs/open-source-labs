# 소개

이 실습에서는 반복자 (iterator) 를 입력으로 받아 반복자 내의 어떤 요소가 주어진 조건 (predicate) 을 만족하는지 여부를 판별하여 `true` 또는 `false`를 반환하는 `Iterator::any` 함수에 대해 설명합니다. 이 함수는 Rust 표준 라이브러리의 트레이트 메서드로 정의되어 있으며 `Iterator` 트레이트를 구현하는 모든 형식에 사용할 수 있습니다. 이 함수는 반복자의 각 요소에 적용할 조건을 결정하는 클로저를 인수로 받습니다. 클로저는 `FnMut` 트레이트로 정의되어 있으며, 캡처된 변수를 수정할 수 있지만 소비할 수는 없습니다. `any` 함수는 반복자 내의 어떤 요소가 조건을 만족하는지 여부를 나타내는 부울 값을 반환합니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs` 파일을 사용하고 `rustc main.rs && ./main` 명령으로 컴파일 및 실행할 수 있습니다.
