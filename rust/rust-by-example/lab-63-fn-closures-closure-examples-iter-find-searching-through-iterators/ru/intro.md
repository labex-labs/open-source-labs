# Введение

В этом лабе мы узнаем о функции `Iterator::find` в Rust, которая перебирает итератор и ищет первое значение, удовлетворяющее заданному условию. Если ни одно из значений не удовлетворяет условию, возвращается `None`.

> **Примечание:** Если в лабе не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
