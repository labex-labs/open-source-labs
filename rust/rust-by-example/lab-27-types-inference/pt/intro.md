# Introdução

Neste laboratório, o motor de inferência de tipos é destacado por sua capacidade de deduzir o tipo de uma variável com base em seu uso e atribuição de valor, demonstrado em um exemplo avançado utilizando a linguagem de programação Rust.

> **Nota:** Se o laboratório não especificar um nome de arquivo, você pode usar qualquer nome de arquivo desejado. Por exemplo, você pode usar `main.rs`, compilar e executar com `rustc main.rs && ./main`.
