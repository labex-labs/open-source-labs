# Связывание переменных

Rust обеспечивает безопасность типов за счет статической типизации. Связывания переменных могут быть аннотированы по типу при объявлении. Однако, в большинстве случаев компилятор сможет вывести тип переменной из контекста, значительно уменьшая нагрузку на аннотацию.

Значения (например, литералы) могут быть связаны с переменными с использованием связывания `let`.

```rust
fn main() {
    let an_integer = 1u32;
    let a_boolean = true;
    let unit = ();

    // копируем `an_integer` в `copied_integer`
    let copied_integer = an_integer;

    println!("An integer: {:?}", copied_integer);
    println!("A boolean: {:?}", a_boolean);
    println!("Meet the unit value: {:?}", unit);

    // Компилятор предупреждает о неиспользуемых связаниях переменных; эти предупреждения можно отключить, добавив перед именем переменной знак подчеркивания
    let _unused_variable = 3u32;

    let noisy_unused_variable = 2u32;
    // FIXME ^ Добавьте знак подчеркивания перед именем, чтобы отключить предупреждение
    // Обратите внимание, что предупреждения могут не отображаться в браузере
}
```
