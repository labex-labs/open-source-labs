# Введение

В этом практическом занятии мы изучаем связывание переменных в Rust, где значения могут быть присвоены переменным с использованием связывания `let`, а также, как можно использовать аннотации типов, которые часто выводятся компилятором.

> **Примечание:** Если практическое занятие не задает имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
