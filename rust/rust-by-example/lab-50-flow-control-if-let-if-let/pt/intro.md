# Introdução

Neste laboratório, exploramos o uso da construção `if let` em Rust, que permite um código mais limpo ao corresponder a enumerações e fornece a capacidade de especificar opções de falha. Também demonstramos como `if let` pode ser usado para corresponder a qualquer valor de enumeração, incluindo variantes não parametrizadas. Além disso, fornecemos um exemplo de correção de código que tenta comparar dois valores de enumeração usando `if let` em vez disso.

> **Nota:** Se o laboratório não especificar um nome de arquivo, você pode usar qualquer nome de arquivo que desejar. Por exemplo, você pode usar `main.rs`, compilar e executar com `rustc main.rs && ./main`.
