# 소개

이 실습에서는 Rust 에서 `if let` 구문의 사용법을 탐구합니다. `if let`은 열거형 (enum) 과 일치할 때 코드를 더욱 깔끔하게 만들고, 실패 옵션을 지정할 수 있는 기능을 제공합니다. 또한, 매개변수가 없는 열거형 변형을 포함하여 모든 열거형 값과 일치하는 방법을 보여줍니다. 마지막으로, 두 열거형 값을 비교하려고 시도하는 코드를 수정하는 예제를 제공하여 `if let`을 사용하는 방법을 보여줍니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
