# 소개

이 실습에서는 Rust 의 `Path` 구조체를 탐색합니다. `Path` 구조체는 기본 파일 시스템에서 파일 경로를 나타냅니다. UNIX 계열 시스템에서는 `posix::Path`, Windows 에서는 `windows::Path`로 구분됩니다. `Path`는 `OsStr`에서 생성될 수 있으며, 경로가 가리키는 파일 또는 디렉터리의 정보를 얻기 위한 다양한 메서드를 제공합니다. `Path`는 불변이며, 소유 버전은 `PathBuf`로, 자리에서 변경될 수 있습니다. `Path`와 `PathBuf`의 관계는 `str`과 `String`의 관계와 유사합니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
