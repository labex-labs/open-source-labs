# Введение

В этом лабе демонстрируется изменяемость данных с использованием языка программирования Rust, показывая, как изменяется владение и изменяемость, когда коробка перемещается из неизменяемого состояния в изменяемое.

> **Примечание:** Если в лабе не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
