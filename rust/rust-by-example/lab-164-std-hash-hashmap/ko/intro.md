# 소개

이 실습에서는 키를 사용하여 값을 저장하는 Rust 의 `HashMap`에 대해 배웁니다. `HashMap` 키는 부울, 정수, 문자열 또는 `Eq` 및 `Hash` 트레이트를 구현하는 다른 모든 유형을 포함할 수 있습니다. `HashMap`은 요소 수에 따라 동적으로 크기가 조절될 수 있습니다. `HashMap::with_capacity(uint)`를 사용하여 특정 용량으로 `HashMap`을 생성하거나 `HashMap::new()`를 사용하여 기본 초기 용량을 가진 `HashMap`을 얻을 수 있습니다. 제공된 코드 예제는 연락처 이름과 전화번호를 저장하고 삽입, 검색, 수정 및 삭제와 같은 작업을 수행하여 `HashMap`의 사용법을 보여줍니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
