# Introduction

Dans ce laboratoire, nous explorerons les concepts de déstructuration et de déréférencement en Rust, en mettant en évidence les différences dans leur utilisation par rapport à des langages tels que C/C++. La déstructuration consiste à utiliser `&`, `ref` et `ref mut` pour assigner et correspondre des références, tandis que le déréférencement est effectué en utilisant `*` pour accéder à la valeur pointée par une référence.

> **Note** : Si le laboratoire ne spécifie pas de nom de fichier, vous pouvez utiliser n'importe quel nom de fichier que vous voulez. Par exemple, vous pouvez utiliser `main.rs`, le compiler et l'exécuter avec `rustc main.rs &&./main`.
