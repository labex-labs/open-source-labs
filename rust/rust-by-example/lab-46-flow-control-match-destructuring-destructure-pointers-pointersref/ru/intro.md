# Введение

В этом лабе мы исследуем концепции деструктуризации и разыменовывания в Rust, подчеркивая различия в их использовании по сравнению с такими языками, как C/C++. Деструктуризация заключается в использовании `&`, `ref` и `ref mut` для присвоения и сопоставления ссылок, а разыменовывание выполняется с использованием `*` для доступа к значению, на которое указывает ссылка.

> **Примечание:** Если в лабе не указано имя файла, вы можете использовать любое имя файла, которое хотите. Например, вы можете использовать `main.rs`, скомпилировать и запустить его с помощью `rustc main.rs &&./main`.
