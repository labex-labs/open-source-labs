# 소개

이 실습에서는 Rust 에서 무한 루프를 생성하는 데 사용되는 `loop` 키워드의 사용법을 살펴봅니다. `break` 문을 사용하여 언제든지 루프를 종료하고 `continue` 문을 사용하여 나머지 반복을 건너뛸 수 있습니다. 제공된 예제 코드는 카운터를 증가시키고 특정 조건에 도달할 때까지 값을 출력하는 방법을 보여주며, `continue`를 사용하여 한 반복을 건너뛰고 `break`를 사용하여 루프를 완전히 종료합니다.

> **참고:** 실습에서 파일 이름을 지정하지 않으면 원하는 파일 이름을 사용할 수 있습니다. 예를 들어 `main.rs`를 사용하고 `rustc main.rs && ./main`으로 컴파일 및 실행할 수 있습니다.
