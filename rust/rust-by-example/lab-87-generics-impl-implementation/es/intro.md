# Introducción

En este laboratorio, aprenderemos a implementar tipos y métodos genéricos en Rust, lo que nos permitirá especificar diferentes parámetros de tipo cuando usamos la estructura o llamamos a los métodos.

> **Nota:** Si el laboratorio no especifica un nombre de archivo, puede usar cualquier nombre de archivo que desee. Por ejemplo, puede usar `main.rs`, compilar y ejecutarlo con `rustc main.rs &&./main`.
