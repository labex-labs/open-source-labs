# Introduction

Bienvenue dans **Traits: Définir un Comportement Partagé**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous explorons les traits comme un moyen de définir un comportement partagé dans un type et de spécifier des contraintes de trait pour les types génériques.
