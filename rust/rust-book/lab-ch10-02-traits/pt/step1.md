# Traits: Definindo Comportamento Compartilhado

Uma _trait_ define a funcionalidade que um tipo específico possui e pode compartilhar com outros tipos. Podemos usar traits para definir comportamento compartilhado de forma abstrata. Podemos usar _limites de trait_ (trait bounds) para especificar que um tipo genérico pode ser qualquer tipo que tenha determinado comportamento.

> Nota: Traits são semelhantes a um recurso frequentemente chamado de _interfaces_ em outras linguagens, embora com algumas diferenças.
