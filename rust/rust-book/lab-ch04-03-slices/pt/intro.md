# Introdução

Bem-vindo(a) ao **The Slice Type**. Este laboratório faz parte do [Rust Book](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, resolveremos um problema de programação escrevendo uma função que recebe uma string de palavras separadas por espaços e retorna a primeira palavra que encontra nessa string. Em seguida, discutiremos as limitações de usar índices para representar substrings e a solução para este problema usando _string slices_ em Rust.
