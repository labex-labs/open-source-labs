# Introdução

Bem-vindo(a) a **Leitura de um Arquivo**. Este laboratório faz parte do [Livro de Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, adicionamos funcionalidade para ler o arquivo especificado no argumento `file_path` usando o método `fs::read_to_string` e, em seguida, imprimir o conteúdo do arquivo.
