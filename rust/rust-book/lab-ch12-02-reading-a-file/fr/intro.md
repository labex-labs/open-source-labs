# Introduction

Bienvenue dans **Reading a File**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous ajoutons une fonctionnalité pour lire le fichier spécifié dans l'argument `file_path` en utilisant la méthode `fs::read_to_string`, puis imprimons le contenu du fichier.
