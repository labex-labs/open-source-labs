# Introducción

Bienvenido a **Recoverable Errors With Result**. Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, aprenderemos a manejar errores recuperables con el enum `Result` en Rust, que nos permite interpretar y responder a errores sin terminar el programa.
