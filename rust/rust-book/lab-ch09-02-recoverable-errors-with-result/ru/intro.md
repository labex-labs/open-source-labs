# Введение

Добро пожаловать в **Recoverable Errors With Result**. Этот лаба является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этом лабе мы узнаем, как обрабатывать восстанавливаемые ошибки с помощью перечисления `Result` в Rust, которое позволяет нам интерпретировать и реагировать на ошибки без завершения программы.
