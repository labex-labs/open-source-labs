# Introduction

Bienvenue dans **Accepting Command Line Arguments**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons créer un nouveau projet appelé "minigrep" qui accepte deux arguments de ligne de commande : une chaîne de caractères à rechercher et un chemin de fichier.
