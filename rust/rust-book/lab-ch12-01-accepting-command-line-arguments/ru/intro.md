# Введение

Добро пожаловать в **Прием аргументов командной строки**. Эта лабораторная работа является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этой лабораторной работе мы создадим новый проект под названием "minigrep", который принимает два аргумента командной строки: строку для поиска и путь к файлу.
