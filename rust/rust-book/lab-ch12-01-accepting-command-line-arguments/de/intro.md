# Einführung

Willkommen zu **Akzeptieren von Befehlszeilenargumenten**. Dieser Lab ist ein Teil des [Rust Buchs](https://doc.rust-lang.org/book/). Du kannst deine Rust-Fähigkeiten in LabEx üben.

In diesem Lab werden wir ein neues Projekt namens "minigrep" erstellen, das zwei Befehlszeilenargumente akzeptiert: einen String, nach dem gesucht werden soll, und einen Dateipfad.
