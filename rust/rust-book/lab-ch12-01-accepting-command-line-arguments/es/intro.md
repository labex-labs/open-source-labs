# Introducción

Bienvenido a **Aceptando Argumentos de Línea de Comandos**. Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, crearemos un nuevo proyecto llamado "minigrep" que acepte dos argumentos de línea de comandos: una cadena de texto a buscar y una ruta de archivo.
