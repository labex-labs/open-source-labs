# Introdução

Bem-vindo(a) a **Aceitando Argumentos da Linha de Comando**. Este laboratório faz parte do [Livro de Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, criaremos um novo projeto chamado "minigrep" que aceita dois argumentos da linha de comando: uma string para pesquisar e um caminho de arquivo.
