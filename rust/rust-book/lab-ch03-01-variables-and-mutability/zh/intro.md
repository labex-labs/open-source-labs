# 简介

欢迎来到「变量与可变性」实验。本实验是 [《Rust 程序设计语言》](https://doc.rust-lang.org/book/) 的一部分。你可以在 LabEx 中练习 Rust 技能。

在本实验中，我们将探讨 Rust 中的可变性概念，讨论变量默认情况下是不可变的，以及如何使用 `mut` 关键字使其可变，并强调不可变性对于安全性和并发性的重要性，同时也承认可变性在某些情况下的有用性。
