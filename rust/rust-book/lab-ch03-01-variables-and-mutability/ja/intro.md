# はじめに

**変数と不変性**へようこそ。この実験は、[Rust Book](https://doc.rust-lang.org/book/)の一部です。LabEx で Rust のスキルを練習することができます。

この実験では、Rust における不変性の概念を探り、変数がデフォルトでは不変である方法と、`mut`キーワードを使用して変数を変更可能にする方法について説明します。また、不変性が安全性と並行性にとって重要であることを強調しながら、特定の状況における可変性の有用性も認識します。
