# Введение

Добро пожаловать в **Использование объектов-примесей, которые допускают значения разных типов**. Эта лабораторная работа является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этой лабораторной работе мы исследуем, как использовать объекты-примеси для допуска значений разных типов в библиотеке, конкретно в контексте графического интерфейса пользователя (GUI) инструмента.
