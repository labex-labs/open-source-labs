# Introdução

Bem-vindo(a) a **Usando Objetos Trait que Permitem Valores de Diferentes Tipos**. Este laboratório faz parte do [Livro do Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, exploraremos como usar objetos trait para permitir valores de diferentes tipos em uma biblioteca, especificamente no contexto de uma ferramenta de interface gráfica do usuário (GUI).
