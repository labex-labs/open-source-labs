# Введение

Добро пожаловать в **Преобразование нашего однопоточного сервера в многопоточный сервер**. Этот лаба является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этом лабе мы преобразуем наш однопоточный сервер в многопоточный сервер, чтобы повысить его эффективность при обработке нескольких запросов одновременно.
