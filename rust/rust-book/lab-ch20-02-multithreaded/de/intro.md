# Einführung

Willkommen zu **Turning Our Single-Threaded Server Into a Multithreaded Server**. Dieser Lab ist Teil des [Rust Buchs](https://doc.rust-lang.org/book/). Du kannst deine Rust-Fähigkeiten in LabEx üben.

In diesem Lab werden wir unseren ein-threadigen Server in einen multi-threadigen Server umwandeln, um seine Effizienz bei der gleichzeitigen Verarbeitung mehrerer Anfragen zu verbessern.
