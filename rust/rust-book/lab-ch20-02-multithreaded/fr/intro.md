# Introduction

Bienvenue dans **Turning Our Single-Threaded Server Into a Multithreaded Server**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons transformer notre serveur mono-fil en un serveur multi-fils pour améliorer son efficacité dans le traitement de plusieurs requêtes simultanément.
