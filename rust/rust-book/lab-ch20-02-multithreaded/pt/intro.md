# Introdução

Bem-vindo a **Transformando Nosso Servidor de Thread Única em um Servidor Multithread**. Este laboratório faz parte do [Livro de Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, transformaremos nosso servidor de thread única em um servidor multithread para melhorar sua eficiência no processamento de múltiplas requisições simultaneamente.
