# Introduction

Bienvenue dans **What Is Ownership?**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, vous allez découvrir la notion de propriété en Rust, un ensemble de règles qui régissent la manière dont un programme gère la mémoire et comment cela affecte le comportement et les performances du langage.
