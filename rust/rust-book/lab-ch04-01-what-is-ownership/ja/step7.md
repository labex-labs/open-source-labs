# Cloneによる変数とデータの相互作用

`String`のヒープデータだけでなく、スタックデータも深くコピーしたい場合は、`clone`と呼ばれる一般的なメソッドを使用できます。第5章でメソッド構文について説明しますが、メソッドは多くのプログラミング言語の共通機能であるため、おそらく以前に見たことがあるでしょう。

以下は、`clone`メソッドの動作例です。

```rust
let s1 = String::from("hello");
let s2 = s1.clone();

println!("s1 = {s1}, s2 = {s2}");
```

これはうまく機能し、ヒープデータがコピーされる（図4-3に示されている動作）ことが明示的に行われます。

`clone`の呼び出しを見ると、何らかの任意のコードが実行されており、そのコードは高価である可能性があることがわかります。これは、何か異なることが起こっていることを視覚的に示すものです。
