# 변수 범위

이제 기본적인 Rust 구문을 지나갔으므로 예제에 모든 `fn main() {` 코드를 포함하지 않겠습니다. 따라서 따라하고 있다면 다음 예제를 수동으로 `main` 함수 안에 넣으십시오. 결과적으로 예제가 조금 더 간결해져서 상용구 코드보다는 실제 세부 사항에 집중할 수 있습니다.

소유권의 첫 번째 예제로, 일부 변수의 *범위 (scope)*를 살펴보겠습니다. 범위는 프로그램 내에서 항목이 유효한 범위입니다. 다음 변수를 예로 들어보겠습니다.

```rust
let s = "hello";
```

변수 `s`는 문자열 리터럴을 참조하며, 문자열의 값은 프로그램 텍스트에 하드 코딩되어 있습니다. 변수는 선언된 시점부터 현재 *범위 (scope)*의 끝까지 유효합니다. 목록 4-1 은 변수 `s`가 유효한 위치에 주석이 있는 프로그램을 보여줍니다.

    {                      // s는 아직 선언되지 않았으므로 여기서는 유효하지 않습니다.
        let s = "hello";   // s는 이 시점부터 유효합니다.

        // s로 작업 수행
    }                      // 이 범위가 종료되었고 s는 더 이상 유효하지 않습니다.

목록 4-1: 변수와 해당 변수가 유효한 범위

다시 말해, 여기에는 두 가지 중요한 시점이 있습니다.

- `s`가 범위 _내로_ 들어오면 유효합니다.
- 범위 _밖으로_ 나갈 때까지 유효합니다.

이 시점에서 범위와 변수가 유효한 시점 간의 관계는 다른 프로그래밍 언어의 관계와 유사합니다. 이제 `String` 타입을 소개하여 이러한 이해를 바탕으로 구축해 보겠습니다.
