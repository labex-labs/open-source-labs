# 변수와 데이터의 상호 작용: Clone

`String`의 힙 데이터를 스택 데이터뿐만 아니라 _정말로_ 깊이 복사하고 싶다면 `clone`이라는 일반적인 메서드를 사용할 수 있습니다. 5 장에서 메서드 구문에 대해 논의하겠지만, 메서드는 많은 프로그래밍 언어에서 흔히 사용되는 기능이므로 이전에 본 적이 있을 것입니다.

다음은 `clone` 메서드의 예입니다.

```rust
let s1 = String::from("hello");
let s2 = s1.clone();

println!("s1 = {s1}, s2 = {s2}");
```

이것은 잘 작동하며 힙 데이터가 _정말로_ 복사되는 그림 4-3 에 표시된 동작을 명시적으로 생성합니다.

`clone` 호출을 보면 임의의 코드가 실행되고 있으며 해당 코드가 비용이 많이 들 수 있음을 알 수 있습니다. 이는 다른 일이 일어나고 있음을 시각적으로 나타내는 지표입니다.
