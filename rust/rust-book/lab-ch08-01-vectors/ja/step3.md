# ベクトルの更新

ベクトルを作成してから要素を追加するには、`push`メソッドを使用できます。これは、リスト8-3に示すようになります。

```rust
let mut v = Vec::new();

v.push(5);
v.push(6);
v.push(7);
v.push(8);
```

リスト8-3：`push`メソッドを使用してベクトルに値を追加する

他の変数と同様に、値を変更できるようにするには、第3章で説明した`mut`キーワードを使用して可変にする必要があります。入れる数字はすべて`i32`型で、Rustはデータからこれを推論するので、`Vec<i32>`の注釈は必要ありません。
