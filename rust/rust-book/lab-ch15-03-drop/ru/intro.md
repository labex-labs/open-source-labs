# Введение

Добро пожаловать в **Запуск кода при очистке с помощью трейта Drop**. Эта лабораторная работа является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этой лабораторной работе мы исследуем трейт `Drop` в Rust, который позволяет настроить действия по очистке, когда значение выходит за пределы области видимости, обычно используется в умных указателях, и может быть реализован путём предоставления метода `drop`, который автоматически вызывается Rust, избавляя от необходимости явного кода по очистке.
