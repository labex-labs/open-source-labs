# Introdução

Bem-vindo a **Executando Código na Limpeza com a Trait Drop**. Este laboratório faz parte do [Livro Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, exploramos a trait `Drop` em Rust, que permite a personalização de ações de limpeza quando um valor sai do escopo, tipicamente usado em ponteiros inteligentes (smart pointers), e pode ser implementada fornecendo um método `drop` que é automaticamente chamado pelo Rust, evitando a necessidade de código de limpeza explícito.
