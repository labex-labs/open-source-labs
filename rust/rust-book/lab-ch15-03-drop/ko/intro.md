# 소개

**Drop Trait 을 사용한 정리 작업에서 코드 실행**에 오신 것을 환영합니다. 이 랩은 [Rust Book](https://doc.rust-lang.org/book/)의 일부입니다. LabEx 에서 Rust 기술을 연습할 수 있습니다.

이 랩에서는 Rust 의 `Drop` 트레이트를 살펴봅니다. 이 트레이트는 값이 스코프 밖으로 벗어날 때 정리 작업을 사용자 정의할 수 있도록 하며, 일반적으로 스마트 포인터에서 사용됩니다. `drop` 메서드를 제공하여 구현할 수 있으며, Rust 에 의해 자동으로 호출되어 명시적인 정리 코드가 필요하지 않게 됩니다.
