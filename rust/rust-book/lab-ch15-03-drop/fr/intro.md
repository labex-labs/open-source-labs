# Introduction

Bienvenue dans **Exécution de code lors de la nettoyage avec le trait Drop**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous explorons le trait `Drop` en Rust, qui permet de personnaliser les actions de nettoyage lorsqu'une valeur sort de portée, généralement utilisé dans les pointeurs intelligents, et peut être implémenté en fournissant une méthode `drop` qui est automatiquement appelée par Rust, évitant ainsi la nécessité de code de nettoyage explicite.
