# Введение

Добро пожаловать в **Разработка функциональности библиотеки с использованием тестируемого разработки**. Этот лаба является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этом лабе мы разработаем функциональность библиотеки с использованием тестируемого разработки, чтобы добавить логику поиска в программу.
