# Écrire du code pour passer le test

Actuellement, notre test échoue car nous retournons toujours un vecteur vide. Pour corriger cela et implémenter `search`, notre programme doit suivre les étapes suivantes :

1. Itérer sur chaque ligne du contenu.
2. Vérifier si la ligne contient notre chaîne de requête.
3. Si c'est le cas, l'ajouter à la liste des valeurs que nous retournons.
4. Si ce n'est pas le cas, ne rien faire.
5. Retourner la liste des résultats qui correspondent.

Travaillons étape par étape, en commençant par l'itération sur les lignes.
