# Introduction

Bienvenue dans **Développement de la fonctionnalité de la bibliothèque avec le développement guidé par les tests**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons développer la fonctionnalité de la bibliothèque en utilisant le développement guidé par les tests pour ajouter une logique de recherche au programme.
