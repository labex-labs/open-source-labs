# Introducción

Bienvenido a **Desarrollando la funcionalidad de la biblioteca con Desarrollo Guiado por Pruebas**. Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, desarrollaremos la funcionalidad de la biblioteca utilizando el desarrollo guiado por pruebas para agregar lógica de búsqueda al programa.
