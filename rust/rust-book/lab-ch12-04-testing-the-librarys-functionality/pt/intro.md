# Introdução

Bem-vindo a **Desenvolvimento da Funcionalidade da Biblioteca com Desenvolvimento Orientado a Testes**. Este laboratório faz parte do [Livro Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, desenvolveremos a funcionalidade da biblioteca usando desenvolvimento orientado a testes para adicionar lógica de busca ao programa.
