# Introdução

Bem-vindo ao **Usando Passagem de Mensagens para Transferir Dados Entre Threads**. Este laboratório faz parte do [Livro Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, exploramos a passagem de mensagens como uma abordagem segura de concorrência, usando canais na biblioteca padrão do Rust para enviar e receber dados entre threads.
