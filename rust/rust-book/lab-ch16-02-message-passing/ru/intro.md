# Введение

Добро пожаловать в **Использование передачи сообщений для передачи данных между потоками**. Эта лабораторная работа является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этой лабораторной работе мы исследуем передачу сообщений как безопасный подход к параллелизации, используя каналы из стандартной библиотеки Rust для отправки и приема данных между потоками.
