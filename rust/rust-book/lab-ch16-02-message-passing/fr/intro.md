# Introduction

Bienvenue dans **Utiliser la communication par messages pour transférer des données entre des threads**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous explorons la communication par messages comme une approche de concurrence sécurisée, en utilisant les canaux de la bibliothèque standard de Rust pour envoyer et recevoir des données entre des threads.
