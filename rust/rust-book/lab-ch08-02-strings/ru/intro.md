# Введение

Добро пожаловать в **Сохранение текста в кодировке UTF-8 с помощью строк**. Этот практикум является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этом практикуме мы обсудим сложности работы со строками в Rust, особенно в отношении кодировки UTF-8, а также операции и различия типа String по сравнению с другими коллекциями.
