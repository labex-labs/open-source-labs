# Introducción

Bienvenido a **Instalación de Binarios con Cargo Install**. Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, aprenderás a usar el comando `cargo install` para instalar cajas binarias localmente y hacerlas ejecutables agregando el directorio de instalación a tu `$PATH`.
