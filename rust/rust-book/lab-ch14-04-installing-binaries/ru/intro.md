# Введение

Добро пожаловать в **Установка бинарников с помощью cargo install**. Эта лабораторная работа является частью [Книги по Rust](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этой лабораторной работе вы научитесь использовать команду `cargo install` для локальной установки бинарных крейтов и сделать их исполняемыми, добавив директорию установки в ваш `$PATH`.
