# Introdução

Bem-vindo a **Instalando Binários com Cargo Install**. Este laboratório faz parte do [Livro Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, você aprenderá como usar o comando `cargo install` para instalar crates binários localmente e torná-los executáveis, adicionando o diretório de instalação ao seu `$PATH`.
