# Introduction

Bienvenue dans **Building a Single-Threaded Web Server**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons construire un serveur web mono-fil qui utilise les protocoles HTTP et TCP pour traiter les requêtes des clients et fournir des réponses.
