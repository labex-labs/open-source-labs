# Введение

Добро пожаловать в **Building a Single-Threaded Web Server**. Этот лаба является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этом лабе мы построим однопоточный веб-сервер, который использует протоколы HTTP и TCP для обработки запросов клиентов и предоставления ответов.
