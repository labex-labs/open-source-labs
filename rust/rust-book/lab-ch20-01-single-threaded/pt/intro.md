# Introdução

Bem-vindo(a) a **Construindo um Servidor Web de Thread Única**. Este laboratório faz parte do [Livro Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, construiremos um servidor web de thread única que utiliza os protocolos HTTP e TCP para lidar com as requisições dos clientes e fornecer respostas.
