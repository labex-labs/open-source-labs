# Introducción

Bienvenido a **Construyendo un servidor web de un solo hilo**. Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, construiremos un servidor web de un solo hilo que utiliza los protocolos HTTP y TCP para manejar las solicitudes de los clientes y proporcionar respuestas.
