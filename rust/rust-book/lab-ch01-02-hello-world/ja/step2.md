# プロジェクトディレクトリの作成

まずは、Rustコードを保存するディレクトリを作成しましょう。コードがどこにあるかはRustにとっては問題ありませんが、この本の演習やプロジェクトでは、ホームディレクトリに`project`ディレクトリを作成し、そこにすべてのプロジェクトを保管することをおすすめします。

ターミナルを開き、以下のコマンドを入力して、`project`ディレクトリと、その中にある「Hello, world!」プロジェクト用のディレクトリを作成します。

Linux、macOS、およびWindowsのPowerShellの場合、これを入力します。

```bash
cd ~/project
mkdir hello_world
cd hello_world
```
