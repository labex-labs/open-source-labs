# Hello, World

Rustをインストールしたので、最初のRustプログラムを書く時が来ました。新しい言語を学ぶ際には、画面に「Hello, world!」という文字列を表示する小さなプログラムを書くのが慣例ですので、ここでも同じことをしましょう！

> 注：この本ではコマンドラインに関する基本的な知識を前提としています。Rustはコードの編集やツール、コードの保存場所について特定の要求をしません。なので、コマンドラインの代わりに統合開発環境（IDE）を使う方が好きな場合は、好きなIDEを使って構いません。現在、多くのIDEにはある程度のRustサポートがあります。詳細はIDEのドキュメントを参照してください。Rustチームは`rust-analyzer`を通じて素晴らしいIDEサポートを可能にすることに焦点を当ててきました。詳細は付録Dを参照してください。
