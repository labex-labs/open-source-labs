# Hello, World

Rust をインストールしたので、最初の Rust プログラムを書く時が来ました。新しい言語を学ぶ際には、画面に「Hello, world!」という文字列を表示する小さなプログラムを書くのが慣例ですので、ここでも同じことをしましょう！

> 注：この本ではコマンドラインに関する基本的な知識を前提としています。Rust はコードの編集やツール、コードの保存場所について特定の要求をしません。なので、コマンドラインの代わりに統合開発環境（IDE）を使う方が好きな場合は、好きな IDE を使って構いません。現在、多くの IDE にはある程度の Rust サポートがあります。詳細は IDE のドキュメントを参照してください。Rust チームは`rust-analyzer`を通じて素晴らしい IDE サポートを可能にすることに焦点を当ててきました。詳細は付録 D を参照してください。
