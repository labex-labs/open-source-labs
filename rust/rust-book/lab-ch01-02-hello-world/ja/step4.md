# Rustプログラムの構成要素

この「Hello, world!」プログラムを詳細に見てみましょう。まずは、問題の最初の部分です。

```rust
fn main() {

}
```

これらの行は、`main`という名前の関数を定義しています。`main`関数は特別です。すべての実行可能なRustプログラムでは、常に最初に実行されるコードです。ここで、最初の行は、パラメータを持たず、何も返さない`main`という名前の関数を宣言しています。パラメータがある場合、それらは丸括弧`()`の中に入ります。

関数本体は`{}`で囲まれています。Rustでは、すべての関数本体の周りに波括弧が必要です。関数宣言と同じ行に始めの波括弧を置き、その間に1つのスペースを入れるのが良いスタイルです。

> 注：Rustプロジェクト全体で標準的なスタイルを守りたい場合は、`rustfmt`と呼ばれる自動フォーマッタツールを使って、コードを特定のスタイルにフォーマットできます（付録Dで`rustfmt`についてもっと詳しく説明します）。Rustチームは、`rustc`と同じように、このツールを標準のRust配布に含めています。なので、あなたのコンピュータにはすでにインストールされているはずです！

`main`関数の本体には、次のコードがあります。

```rust
    println!("Hello, world!");
```

この行が、この小さなプログラムのすべての作業を行っています。画面に文字列を表示します。ここで注意すべき重要な点は4つあります。

まず、Rustのスタイルは、タブではなく4つのスペースでインデントすることです。

第二に、`println!`はRustのマクロを呼び出します。関数を呼び出した場合、`println`（`!`なし）と入力します。第19章でRustのマクロについてもっと詳しく説明します。今のところ、`!`を使うことは、通常の関数ではなくマクロを呼び出していることを意味し、マクロは常に関数と同じ規則に従うわけではないことだけを知っておけば良いです。

第三に、`"Hello, world!"`という文字列を見ることができます。この文字列を引数として`println!`に渡し、文字列が画面に表示されます。

第四に、行の末尾にセミコロン（`;`）があります。これは、この式が終了し、次の式が始まる準備ができていることを示しています。ほとんどのRustコードの行はセミコロンで終わります。
