# 简介

欢迎来到**你好，世界**。本实验是[《Rust 程序设计语言》](https://doc.rust-lang.org/book/)的一部分。你可以在 LabEx 中练习 Rust 技能。

在本实验中，你将编写你的第一个 Rust 程序，它会在屏幕上打印文本“你好，世界！”。
