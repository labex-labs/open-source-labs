```bash
$ cargo run

    Finished dev [unoptimized + debuginfo] target(s) in 6.44s
     Running `target/debug/guessing_game`
Guess the number!
Please input your guess.
6
You guessed: 6
```

この時点で、ゲームの最初の部分は完了です。キーボードから入力を受け取り、それを表示しています。
