# 简介

欢迎来到**猜数字游戏编程**。本实验是[《Rust 程序设计语言》](https://doc.rust-lang.org/book/)的一部分。你可以在 LabEx 中练习 Rust 技能。

在本实验中，我们将用 Rust 实现一个猜数字游戏，程序会生成一个随机数并提示玩家猜测，根据猜测结果给出反馈，提示猜测过高或过低，如果玩家猜对则表示祝贺。
