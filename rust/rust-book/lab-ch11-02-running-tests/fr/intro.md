# Introduction

Bienvenue dans **Contrôler la façon dont les tests sont exécutés**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, vous allez apprendre à contrôler le comportement des exécutions de tests en Rust en utilisant les options de ligne de commande pour `cargo test` et le binaire de test résultant.
