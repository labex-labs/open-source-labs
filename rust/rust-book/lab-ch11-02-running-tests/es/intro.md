# Introducción

Bienvenido a **Controlling How Tests Are Run** (Controlar cómo se ejecutan las pruebas). Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, aprenderás a controlar el comportamiento de las ejecuciones de pruebas en Rust utilizando opciones de línea de comandos para `cargo test` y el binario de prueba resultante.
