# Introdução

Bem-vindo a **Controlando Como os Testes são Executados**. Este laboratório faz parte do [Livro Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, você aprenderá como controlar o comportamento das execuções de testes em Rust usando opções de linha de comando para `cargo test` e o binário de teste resultante.
