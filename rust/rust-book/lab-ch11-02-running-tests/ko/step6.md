# 여러 테스트 실행을 위한 필터링

테스트 이름의 일부를 지정할 수 있으며, 해당 값과 일치하는 이름을 가진 모든 테스트가 실행됩니다. 예를 들어, 테스트 이름 중 두 개에 `add`가 포함되어 있으므로 `cargo test add`를 실행하여 해당 두 테스트를 실행할 수 있습니다.

```bash
[object Object]
```

이 명령은 이름에 `add`가 포함된 모든 테스트를 실행하고 `one_hundred`라는 이름의 테스트를 필터링했습니다. 또한 테스트가 나타나는 모듈이 테스트 이름의 일부가 되므로, 모듈 이름을 기준으로 필터링하여 모듈의 모든 테스트를 실행할 수 있습니다.
