# Introduction

Bienvenue dans **RefCell<T> et le modèle de mutabilité interne**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous explorerons le concept de mutabilité interne en Rust et la manière dont il est implémenté à l'aide du type `RefCell<T>`.
