# Introducción

Bienvenido a **Eliminación de duplicados mediante extracción de una función**. Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, aprenderemos cómo eliminar la duplicación de código mediante la extracción de una función y el uso de genéricos para operar en tipos abstractos.
