# Tipos de Dados Genéricos

Usamos genéricos para criar definições para itens como assinaturas de funções ou structs, que podemos então usar com muitos tipos de dados concretos diferentes. Vamos primeiro analisar como definir funções, structs, enums e métodos usando genéricos. Em seguida, discutiremos como os genéricos afetam o desempenho do código.
