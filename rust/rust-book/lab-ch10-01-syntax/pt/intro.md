# Introdução

Bem-vindo(a) a **Removendo Duplicação por Extração de uma Função**. Este laboratório faz parte do [Livro de Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, aprenderemos como remover a duplicação de código extraindo uma função e usando genéricos para operar em tipos abstratos.
