# Generic Data Types

Nous utilisons des génériques pour créer des définitions pour des éléments tels que les signatures de fonctions ou les structs, que nous pouvons ensuite utiliser avec de nombreux types de données concrètes différents. Regardons tout d'abord comment définir des fonctions, des structs, des enums et des méthodes à l'aide de génériques. Ensuite, nous discuterons de la manière dont les génériques affectent les performances du code.
