# Introduction

Bienvenue dans **Removing Duplication by Extracting a Function**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons apprendre à éliminer la duplication de code en extrayant une fonction et en utilisant des génériques pour opérer sur des types abstraits.
