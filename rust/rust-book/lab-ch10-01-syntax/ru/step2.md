# Обобщенные типы данных

Мы используем обобщения для создания определений для элементов, таких как сигнатуры функций или структуры, которые мы можем затем использовать с многими различными конкретными типами данных. Давайте сначала рассмотрим, как определить функции, структуры, перечисления и методы с использованием обобщений. Затем мы обсудим, как обобщения влияют на производительность кода.
