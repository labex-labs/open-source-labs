# Введение

Добро пожаловать в раздел **Синтаксис шаблонов (Pattern Syntax)**. Этот практикум является частью [Книги по Rust](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки программирования на Rust в LabEx.

В этом практикуме мы рассмотрим допустимый синтаксис шаблонов и приведем примеры случаев, когда и почему вы можете захотеть использовать каждый из них.
