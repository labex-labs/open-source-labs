# 構造体とタプルの分解

私たちは、さらに複雑な方法で分解パターンを組み合わせ、マッチさせ、ネストさせることができます。次の例は、タプルの中に構造体とタプルをネストさせ、すべての基本型の値を分解する複雑な分解を示しています。

```rust
let ((feet, inches), Point { x, y }) =
    ((3, 10), Point { x: 3, y: -10 });
```

このコードは、複雑な型をその構成要素に分解し、私たちが興味のある値を個別に使用できるようにします。

パターンを使った分解は、構造体の各フィールドの値のような値の一部を互いに独立して使用する便利な方法です。
