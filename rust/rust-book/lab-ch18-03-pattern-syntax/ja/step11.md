# パターン内の値を無視する

パターン内の値を無視することが時に便利であることをご覧いただきました。たとえば、`match` の最後のアームでは、何も実際に行わないキャッチオールとして、すべての残りの可能性のある値を処理するために、パターン内の値を無視することができます。パターン内の値全体または値の一部を無視する方法はいくつかあります。`_` パターン（既にご覧いただいた通り）を使用すること、別のパターン内で `_` パターンを使用すること、アンダースコアで始まる名前を使用すること、または値の残りの部分を無視するために `..` を使用することです。これらのパターンをそれぞれどのように使用するか、そしてなぜ使用するかを見ていきましょう。
