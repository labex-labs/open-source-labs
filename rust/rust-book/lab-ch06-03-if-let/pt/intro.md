# Introdução

Bem-vindo ao **Fluxo de Controle Conciso com if let**. Este laboratório faz parte do [Livro Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, exploramos o fluxo de controle conciso da sintaxe `if let`, que permite combinar `if` e `let` de uma forma mais curta e legível para lidar com valores que correspondem a um padrão, ignorando o restante.
