# Introduction

Bienvenue dans **Concise Control Flow With if Let**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous explorons le flux de contrôle concis de la syntaxe `if let`, qui permet de combiner `if` et `let` d'une manière plus courte et plus lisible pour traiter les valeurs qui correspondent à un motif tout en ignorant le reste.
