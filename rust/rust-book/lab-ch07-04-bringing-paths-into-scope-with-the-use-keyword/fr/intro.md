# Introduction

Bienvenue dans **Bringing Paths Into Scope With the Use Keyword**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous apprenons à porter des chemins en portée en utilisant le mot-clé `use` pour créer des raccourcis pour appeler des fonctions et des modules.
