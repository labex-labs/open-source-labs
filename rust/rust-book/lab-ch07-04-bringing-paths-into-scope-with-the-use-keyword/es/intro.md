# Introducción

Bienvenido a **Bringing Paths Into Scope With the Use Keyword**. Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, aprenderemos cómo traer rutas al alcance utilizando la palabra clave `use` para crear atajos para llamar a funciones y módulos.
