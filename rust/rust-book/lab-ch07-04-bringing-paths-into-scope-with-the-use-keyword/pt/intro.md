# Introdução

Bem-vindo a **Bringing Paths Into Scope With the Use Keyword**. Este laboratório faz parte do [Rust Book](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, aprenderemos como trazer caminhos para o escopo usando a palavra-chave `use` para criar atalhos para chamar funções e módulos.
