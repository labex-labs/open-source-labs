# Introduction

Bienvenue dans **Utiliser Box<T> pour pointer vers des données sur le tas**. Ce laboratoire est une partie du [Livre Rust](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons apprendre à utiliser les pointeurs intelligents Box<T> pour stocker des données sur le tas plutôt que sur la pile, dans des situations où la taille du type est inconnue à la compilation, lors du transfert de la propriété de grandes quantités de données pour éviter la copie, ou lorsqu'on possède une valeur qui implémente un trait particulier.
