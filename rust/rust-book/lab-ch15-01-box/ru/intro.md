# Введение

Добро пожаловать в **Использование Box<T> для указания на данные в куче**. Эта лабораторная работа является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этой лабораторной работе мы узнаем, как использовать умные указатели Box<T> для хранения данных в куче, а не на стеке, в ситуациях, когда размер типа неизвестен на этапе компиляции, при передаче владения большими объемами данных для избежания копирования или при владении значением, которое реализует определенный трейт.
