# Introdução

Bem-vindo(a) ao **Usando Box<T> para Apontar para Dados no Heap**. Este laboratório faz parte do [Livro de Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, aprenderemos sobre o uso de smart pointers `Box<T>` para armazenar dados no heap, em vez da stack, em situações onde o tamanho do tipo é desconhecido em tempo de compilação, ao transferir a propriedade de grandes quantidades de dados para evitar cópias, ou ao possuir um valor que implementa um trait específico.
