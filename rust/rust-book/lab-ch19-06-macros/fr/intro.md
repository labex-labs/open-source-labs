# Introduction

Bienvenue dans **Macros**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous explorons le concept de macros en Rust, y compris les macros déclaratives avec `macro_rules!` et trois types de macros procédurales : les macros personnalisées `#[derive]`, les macros ressemblant à des attributs et les macros ressemblant à des fonctions.
