# 매크로 (Macros)

이 책 전체에서 `println!`과 같은 매크로를 사용했지만, 매크로가 무엇이며 어떻게 작동하는지 완전히 탐구하지는 않았습니다. *매크로 (macro)*라는 용어는 Rust 의 여러 기능을 통칭합니다: `macro_rules!`를 사용한 _선언적 (declarative)_ 매크로와 세 가지 종류의 _절차적 (procedural)_ 매크로가 있습니다.

- 구조체 (struct) 와 열거형 (enum) 에 사용되는 `derive` 속성으로 추가되는 코드를 지정하는 사용자 정의 `#[derive]` 매크로
- 모든 항목에 사용할 수 있는 사용자 정의 속성을 정의하는 속성 유사 매크로
- 함수 호출처럼 보이지만 인수로 지정된 토큰 (token) 에 대해 작동하는 함수 유사 매크로

각각에 대해 차례로 이야기하겠지만, 먼저 함수가 이미 있는데 왜 매크로가 필요한지 살펴보겠습니다.
