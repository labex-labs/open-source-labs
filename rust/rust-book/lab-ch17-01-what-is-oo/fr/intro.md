# Introduction

Bienvenue dans **Caractéristiques des langages orientés objet**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons explorer les caractéristiques des langages orientés objet, y compris les objets, l'encapsulation et l'héritage, et examiner si Rust prend en charge ces fonctionnalités.
