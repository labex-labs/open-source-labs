# Introduction

Bienvenue dans **Refactoring to Improve Modularity and Error Handling**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons refactoriser le programme pour améliorer la modularité et la gestion des erreurs en séparant les tâches, en regroupant les variables de configuration, en fournissant des messages d'erreur significatifs et en consolidant le code de gestion des erreurs.
