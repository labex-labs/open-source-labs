# Введение

Добро пожаловать в **Refactoring to Improve Modularity and Error Handling**. Эта лабораторная работа является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этой лабораторной работе мы будем рефакторить программу для улучшения модульности и обработки ошибок, разделяя задачи, группируя переменные конфигурации, предоставляя осмысленные сообщения об ошибках и консолидируя код обработки ошибок.
