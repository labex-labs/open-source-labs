# Introdução

Bem-vindo(a) a **Refatoração para Melhorar a Modularidade e o Tratamento de Erros**. Este laboratório faz parte do [Livro do Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, vamos refatorar o programa para melhorar a modularidade e o tratamento de erros, separando tarefas, agrupando variáveis de configuração, fornecendo mensagens de erro significativas e consolidando o código de tratamento de erros.
