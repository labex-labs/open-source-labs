# Introducción

Bienvenido a **Storing Keys With Associated Values in Hash Maps** (Almacenar claves con valores asociados en mapas hash). Este laboratorio es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades en Rust en LabEx.
