# Introdução

Bem-vindo(a) a **Armazenando Chaves com Valores Associados em Hash Maps**. Este laboratório faz parte do [Livro de Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, exploraremos o conceito de hash maps (mapas de hash) e como eles podem ser usados para armazenar chaves com valores associados.
