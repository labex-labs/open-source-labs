# Einführung

Willkommen bei **Storing Keys With Associated Values in Hash Maps** (Speichern von Schlüsseln mit zugehörigen Werten in Hash-Tabellen). Dieser Lab-Abschnitt ist Teil des [Rust Book](https://doc.rust-lang.org/book/). Sie können Ihre Rust-Fähigkeiten in LabEx üben.

In diesem Lab werden wir das Konzept von Hash-Tabellen (Hash Maps) untersuchen und erfahren, wie sie verwendet werden können, um Schlüssel mit zugehörigen Werten zu speichern.
