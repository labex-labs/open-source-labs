# Introduction

Bienvenue dans **Packages and Crates**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous aborderons les packages et les crates, où une crate est la plus petite unité de code que le compilateur Rust considère et peut être une crate binaire ou une crate bibliothèque, et un package est une collection d'une ou plusieurs crates qui fournit un ensemble de fonctionnalités.
