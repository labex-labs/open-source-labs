# Introducción

Bienvenido a **Rc<T>, el puntero inteligente con conteo de referencias**. Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, exploraremos el uso de Rc<T> (conteo de referencias) en Rust para permitir que múltiples propietarios tengan una misma valor, mediante el seguimiento del número de referencias a ese valor y asegurándonos de que solo se elimine cuando ya no hay propietarios.
