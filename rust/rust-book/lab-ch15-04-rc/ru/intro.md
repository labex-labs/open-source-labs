# Введение

Добро пожаловать в **Rc<T>, умный указатель с подсчетом ссылок**. Эта лабораторная работа является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этой лабораторной работе мы будем изучать использование Rc<T> (подсчет ссылок) в Rust для обеспечения нескольких владельцев значения путем отслеживания количества ссылок на значение и гарантии того, что оно будет очищено только в том случае, если не будет ни одного владельца.
