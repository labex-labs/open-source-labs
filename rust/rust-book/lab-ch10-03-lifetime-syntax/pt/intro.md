# Introdução

Bem-vindo a **Validando Referências com Lifetimes**. Este laboratório faz parte do [Livro do Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, discutiremos _lifetimes_ (tempos de vida) e como eles garantem que as referências sejam válidas pelo tempo necessário, e embora os _lifetimes_ possam parecer estranhos, abordaremos as formas comuns com que você pode encontrar a sintaxe de _lifetime_ para ajudá-lo a se sentir confortável com o conceito.
