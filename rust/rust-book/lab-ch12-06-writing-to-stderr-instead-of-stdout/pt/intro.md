# Introdução

Bem-vindo a **Escrevendo Mensagens de Erro para Saída de Erro Padrão em Vez de Saída Padrão**. Este laboratório faz parte do [Livro Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, precisamos modificar nosso código para que as mensagens de erro sejam escritas na saída de erro padrão (`stderr`) em vez da saída padrão (`stdout`).
