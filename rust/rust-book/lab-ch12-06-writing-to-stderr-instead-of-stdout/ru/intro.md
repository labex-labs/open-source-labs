# Введение

Добро пожаловать в **Запись сообщений об ошибках в стандартный вывод ошибок вместо стандартного вывода**. Эта лабораторная работа является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этой лабораторной работе нам нужно изменить наш код так, чтобы сообщения об ошибках записывались в стандартный вывод ошибок (`stderr`), а не в стандартный вывод (`stdout`).
