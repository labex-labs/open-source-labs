# Запись сообщений об ошибках в стандартный вывод ошибок вместо стандартного вывода

В настоящее время мы выводим все свои данные в терминал с помощью макроса `println!`. В большинстве терминалов есть два типа вывода: _стандартный вывод_ (`stdout`) для общих сведений и _стандартный вывод ошибок_ (`stderr`) для сообщений об ошибках. Это отличие позволяет пользователям направлять успешный вывод программы в файл, но по-прежнему выводить сообщения об ошибках на экран.

Макрос `println!` способен только выводить в стандартный вывод, поэтому нам нужно использовать что-то другое для вывода в стандартный вывод ошибок.
