# Einführung

Willkommen zu **Schreiben von Fehlermeldungen an die Standardfehlerausgabe statt an die Standardausgabe**. Dieser Lab ist ein Teil des [Rust Buchs](https://doc.rust-lang.org/book/). Du kannst deine Rust-Fähigkeiten in LabEx üben.

In diesem Lab müssen wir unseren Code so ändern, dass Fehlermeldungen an die Standardfehlerausgabe (`stderr`) statt an die Standardausgabe (`stdout`) geschrieben werden.
