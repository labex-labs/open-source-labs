# Introdução

Bem-vindo(a) a **Functions** (Funções). Este laboratório faz parte do [Rust Book](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, você aprenderá como definir e chamar funções em Rust usando a palavra-chave `fn` e a convenção de nomenclatura snake case.
