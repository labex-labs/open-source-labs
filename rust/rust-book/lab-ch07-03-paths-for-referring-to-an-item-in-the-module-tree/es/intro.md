# Introducción

Bienvenido a **Rutas para Referirse a un Elemento en el Árbol de Módulos**. Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, aprendemos que las rutas se utilizan en Rust para referirse a elementos en el árbol de módulos y pueden tener la forma de rutas absolutas o rutas relativas.
