# Einführung

Willkommen zu **Paths for Referring to an Item in the Module Tree**. Dieser Lab ist ein Teil des [Rust Buchs](https://doc.rust-lang.org/book/). Du kannst deine Rust-Fähigkeiten in LabEx üben.

In diesem Lab lernen wir, dass Pfade in Rust verwendet werden, um auf Elemente im Modultree zu verweisen, und dass sie in der Form von absoluten Pfaden oder relativen Pfaden auftreten können.
