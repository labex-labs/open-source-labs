# crates.io 에 크레이트 게시하기

우리는 프로젝트의 종속성으로 *https://crates.io*의 패키지를 사용했지만, 자신의 패키지를 게시하여 다른 사람들과 코드를 공유할 수도 있습니다. *https://crates.io*의 크레이트 레지스트리는 패키지의 소스 코드를 배포하므로 주로 오픈 소스 코드를 호스팅합니다.

Rust 와 Cargo 는 게시된 패키지를 사람들이 더 쉽게 찾고 사용할 수 있도록 하는 기능을 가지고 있습니다. 다음에서 이러한 기능 중 일부에 대해 이야기하고 패키지를 게시하는 방법을 설명하겠습니다.
