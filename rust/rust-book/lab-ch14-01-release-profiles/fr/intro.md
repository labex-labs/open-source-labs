# Introduction

Bienvenue dans **Personnaliser les builds avec des profils de publication**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons explorer la manière de personnaliser les profils de publication de Rust en ajoutant des sections `[profile.*]` au fichier `Cargo.toml` et en substituant les paramètres par défaut tels que le `niveau d'optimisation` pour l'optimisation.
