# Введение

Добро пожаловать в **Настройку сборок с помощью профилей выпуска**. Эта лабораторная работа является частью [Rust Book](https://doc.rust-lang.org/book/). Вы можете практиковать свои навыки Rust в LabEx.

В этой лабораторной работе мы изучим, как настраивать профили выпуска Rust, добавляя разделы `[profile.*]` в файл `Cargo.toml` и переопределяя параметры по умолчанию, такие как `opt-level` для оптимизации.
