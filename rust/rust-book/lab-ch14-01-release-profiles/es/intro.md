# Introducción

Bienvenido a **Personalizando compilaciones con perfiles de lanzamiento**. Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, exploraremos cómo personalizar los perfiles de lanzamiento de Rust agregando secciones `[profile.*]` al archivo `Cargo.toml` y reemplazando configuraciones predeterminadas como el `opt-level` para la optimización.
