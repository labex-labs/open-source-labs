# Introdução

Bem-vindo a **Customizando Builds com Perfis de Release**. Este laboratório faz parte do [Livro do Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, exploraremos como personalizar os perfis de release do Rust adicionando seções `[profile.*]` ao arquivo `Cargo.toml` e substituindo as configurações padrão, como o `opt-level` para otimização.
