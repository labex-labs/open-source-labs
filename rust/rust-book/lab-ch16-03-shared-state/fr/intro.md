# Introduction

Bienvenue dans **Shared-State Concurrency**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous explorons le concept de concurrence en mémoire partagée et pourquoi les partisans de la communication par message s'y opposent.
