# 소개

**Refutability: 패턴이 일치하지 않을 수 있는지 여부**에 오신 것을 환영합니다. 이 랩은 [Rust Book](https://doc.rust-lang.org/book/)의 일부입니다. LabEx 에서 Rust 기술을 연습할 수 있습니다.

이 랩에서는 패턴의 반박 가능성 (refutability) 개념과 이것이 Rust 의 매칭 프로세스에 미치는 영향에 대해 배우고, `let` 문 및 `if let` 표현식과 같은 다양한 구조에서 반박 가능한 (refutable) 패턴과 반박 불가능한 (irrefutable) 패턴의 차이점과 이를 올바르게 사용하는 방법을 배웁니다.
