# Introduction

Bienvenue dans **Refutabilité : La possibilité qu'un motif ne corresponde pas**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons découvrir le concept de réfutabilité dans les motifs et comment cela affecte le processus de correspondance en Rust, y compris la distinction entre les motifs réfutables et irréfutables et la manière de les utiliser correctement dans différents constructs tels que les instructions `let` et les expressions `if let`.
