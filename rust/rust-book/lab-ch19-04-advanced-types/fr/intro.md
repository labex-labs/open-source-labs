# Introduction

Bienvenue dans **Advanced Types**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons discuter des newtypes, des alias de type, du type `!` et des types de taille dynamique dans le système de types Rust.
