# Introdução

Bem-vindo a **Tipos Avançados**. Este laboratório faz parte do [Livro do Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, discutiremos newtypes, type aliases (alias de tipos), o tipo `!`, e tipos de tamanho dinâmico no sistema de tipos do Rust.
