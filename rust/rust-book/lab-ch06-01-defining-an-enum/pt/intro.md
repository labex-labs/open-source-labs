# Introdução

Bem-vindo a **Definindo um Enum**. Este laboratório faz parte do [Livro Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, definiremos um enum chamado `IpAddrKind` para representar os possíveis tipos de endereços IP, incluindo a versão quatro (`V4`) e a versão seis (`V6`).
