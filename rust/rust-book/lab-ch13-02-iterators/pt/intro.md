# Introdução

Bem-vindo(a) a **Processamento de uma Série de Itens com Iteradores**. Este laboratório faz parte do [Livro do Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, exploraremos como processar uma série de itens usando iteradores, que são _lazy_ (preguiçosos) e nos permitem iterar sobre uma sequência de itens sem ter que reimplementar a lógica por conta própria.
