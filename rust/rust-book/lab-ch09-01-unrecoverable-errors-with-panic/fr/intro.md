# Introduction

Bienvenue dans **Unrecoverable Errors With Panic**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons découvrir les panics en Rust et comment elles peuvent être causées par des actions dans notre code ou en appelant explicitement la macro `panic!`, et comment utiliser les traces de pile pour localiser la source de la panique.
