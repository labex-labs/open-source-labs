# Introduction

Bienvenue dans **Extensible Concurrency With the Send and Sync Traits**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous explorons les deux concepts de concurrence en Rust - les traits `Send` et `Sync`, qui offrent des capacités de concurrence extensibles au-delà de la bibliothèque standard.
