# はじめに

**インストール**へようこそ。この実験は、[Rust Book](https://doc.rust-lang.org/book/) の一部です。LabEx で Rust のスキルを練習することができます。

この実験では、Rust のバージョンと関連ツールを管理するコマンドラインツールである `rustup` を使って、インターネット接続を通じてダウンロードすることで Rust をインストールする必要があります。
