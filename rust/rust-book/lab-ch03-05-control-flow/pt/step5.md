# Repetição com Loops

É frequentemente útil executar um bloco de código mais de uma vez. Para esta tarefa, Rust fornece vários _loops_ (laços), que percorrerão o código dentro do corpo do loop até o final e, em seguida, começarão imediatamente de volta ao início. Para experimentar com loops, vamos criar um novo projeto chamado _loops_.

Rust tem três tipos de loops: `loop`, `while` e `for`. Vamos experimentar cada um.
