# Repetición con Bucles

A menudo es útil ejecutar un bloque de código más de una vez. Para esta tarea, Rust proporciona varios _bucles_, que ejecutarán el código dentro del cuerpo del bucle hasta el final y luego comenzarán de inmediato nuevamente desde el principio. Para experimentar con los bucles, creemos un nuevo proyecto llamado _loops_.

Rust tiene tres tipos de bucles: `loop`, `while` y `for`. Intentemos cada uno.
