# Повторение с помощью циклов

Часто полезно выполнять блок кода более одного раза. Для этой задачи Rust предоставляет несколько _циклов_, которые будут выполнять код внутри тела цикла до конца и затем сразу начинать сначала. Чтобы поэкспериментировать с циклами, давайте создадим новый проект под названием _loops_.

В Rust есть три вида циклов: `loop`, `while` и `for`. Давайте попробуем каждый из них.
