# Répétition avec des boucles

Il est souvent utile d'exécuter un bloc de code plusieurs fois. Pour cette tâche, Rust propose plusieurs _boucles_, qui exécuteront le code à l'intérieur du corps de la boucle jusqu'à la fin puis recommenceront immédiatement au début. Pour expérimenter avec les boucles, créons un nouveau projet appelé _boucles_.

Rust a trois types de boucles : `loop`, `while` et `for`. Essayons chacun d'eux.
