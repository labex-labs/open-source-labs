# Wiederholung mit Schleifen

Es ist oft nützlich, einen Codeblock mehr als einmal auszuführen. Für diese Aufgabe bietet Rust mehrere _Schleifen_, die den Code innerhalb des Schleifenkörpers bis zum Ende ausführen und dann sofort wieder am Anfang beginnen. Um mit Schleifen zu experimentieren, erstellen wir ein neues Projekt namens _loops_.

Rust hat drei Arten von Schleifen: `loop`, `while` und `for`. Probieren wir jede aus.
