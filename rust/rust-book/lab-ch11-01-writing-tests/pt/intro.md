# Introdução

Bem-vindo a **Como Escrever Testes**. Este laboratório faz parte do [Livro Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, aprenderemos sobre como escrever testes em Rust usando atributos, macros e asserções (assertions).
