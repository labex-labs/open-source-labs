# Trabalhando com Variáveis de Ambiente

Vamos aprimorar o `minigrep` adicionando um recurso extra: uma opção para pesquisa sem distinção entre maiúsculas e minúsculas que o usuário pode ativar por meio de uma variável de ambiente. Poderíamos tornar esse recurso uma opção de linha de comando e exigir que os usuários o insiram toda vez que desejarem aplicá-lo, mas, em vez disso, tornando-o uma variável de ambiente, permitimos que nossos usuários definam a variável de ambiente uma vez e que todas as suas pesquisas sejam sem distinção entre maiúsculas e minúsculas naquela sessão de terminal.
