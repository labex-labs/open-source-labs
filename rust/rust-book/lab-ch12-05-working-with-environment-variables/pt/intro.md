# Introdução

Bem-vindo(a) a **Trabalhando com Variáveis de Ambiente**. Este laboratório faz parte do [Livro de Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, aprimoraremos o `minigrep` permitindo que os usuários habilitem a pesquisa sem distinção entre maiúsculas e minúsculas por meio de uma variável de ambiente, fornecendo uma maneira conveniente de aplicar esse recurso a todas as suas pesquisas em uma sessão de terminal.
