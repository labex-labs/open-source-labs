# Introduction

Bienvenue dans **Working With Environment Variables**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons améliorer `minigrep` en permettant aux utilisateurs d'activer la recherche insensible à la casse via une variable d'environnement, offrant ainsi un moyen pratique d'appliquer cette fonctionnalité à toutes leurs recherches dans une session terminale.
