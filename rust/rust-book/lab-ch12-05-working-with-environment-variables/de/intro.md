# Einführung

Willkommen zu **Working With Environment Variables**. Dieser Lab ist Teil des [Rust Buchs](https://doc.rust-lang.org/book/). Du kannst deine Rust-Fähigkeiten in LabEx üben.

In diesem Lab werden wir `minigrep` verbessern, indem wir Benutzern ermöglichen, eine case-insensitive Suche über eine Umgebungsvariable zu aktivieren, was eine bequeme Möglichkeit bietet, diese Funktion auf alle ihre Suchanfragen in einer Terminalsitzung anzuwenden.
