# Arbeiten mit Umgebungsvariablen

Wir werden `minigrep` verbessern, indem wir ein zusätzliches Feature hinzufügen: eine Option für eine case-insensitive Suche, die der Benutzer über eine Umgebungsvariable aktivieren kann. Wir könnten dieses Feature als Befehlszeilenoption machen und erfordern, dass die Benutzer es jedes Mal eingeben, wenn sie es anwenden möchten, aber indem wir es stattdessen als Umgebungsvariable machen, erlauben wir unseren Benutzern, die Umgebungsvariable einmal einzustellen und alle ihre Suchanfragen in dieser Terminalsitzung case-insensitive zu machen.
