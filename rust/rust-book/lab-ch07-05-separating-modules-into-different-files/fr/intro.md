# Introduction

Bienvenue dans **Séparer les modules dans des fichiers différents**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, vous allez apprendre à séparer les modules dans des fichiers différents en les extrayant dans leurs propres fichiers et en les organisant dans une structure d'arborescence de modules.
