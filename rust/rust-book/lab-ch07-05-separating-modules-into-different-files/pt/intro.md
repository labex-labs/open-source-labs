# Introdução

Bem-vindo a **Separando Módulos em Arquivos Diferentes**. Este laboratório faz parte do [Livro do Rust](https://doc.rust-lang.org/book/). Você pode praticar suas habilidades em Rust no LabEx.

Neste laboratório, você aprenderá como separar módulos em arquivos diferentes, extraindo-os para seus próprios arquivos e organizando-os em uma estrutura de árvore de módulos.
