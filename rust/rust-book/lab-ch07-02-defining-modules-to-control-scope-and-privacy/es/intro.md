# Introducción

Bienvenido a **Definiendo Módulos para Controlar el Alcance y la Privacidad**. Esta práctica es parte del [Rust Book](https://doc.rust-lang.org/book/). Puedes practicar tus habilidades de Rust en LabEx.

En esta práctica, aprenderemos sobre los módulos y cómo nos permiten organizar el código dentro de un crat, controlar la privacidad de los elementos y proporcionar una forma de navegar por el árbol de módulos.
