# Introduction

Bienvenue dans **Définir des modules pour contrôler la portée et la confidentialité**. Ce laboratoire est une partie du [Rust Book](https://doc.rust-lang.org/book/). Vous pouvez pratiquer vos compétences Rust dans LabEx.

Dans ce laboratoire, nous allons apprendre à connaître les modules et la manière dont ils nous permettent d'organiser le code dans une boîte à outils, de contrôler la confidentialité des éléments et de fournir un moyen de naviguer dans l'arbre de modules.
