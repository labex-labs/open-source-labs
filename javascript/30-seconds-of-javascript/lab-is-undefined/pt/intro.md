# Introdução

Neste laboratório, exploraremos o conceito de valor em JavaScript e como verificar se um valor especificado é `undefined`. Este laboratório fornecerá experiência prática na escrita de código para determinar a presença de valores `undefined` usando o operador de igualdade estrita. Ao final deste laboratório, você terá uma sólida compreensão de como usar a função `isUndefined()` para verificar valores `undefined` em seus programas JavaScript.
