# Verificando Valores `Undefined`

Para verificar se um valor é `undefined`, abra o Terminal/SSH e digite `node`.

- Use o operador de igualdade estrita para verificar se `val` é igual a `undefined`.

```js
const isUndefined = (val) => val === undefined;
```

```js
isUndefined(undefined); // true
```

Este código verificará se o valor especificado é `undefined`.
