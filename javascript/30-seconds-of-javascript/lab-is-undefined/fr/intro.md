# Introduction

Dans ce laboratoire, nous allons explorer le concept de valeur en JavaScript et découvrir comment vérifier si une valeur spécifiée est `undefined`. Ce laboratoire vous donnera une expérience pratique dans la rédaction de code pour déterminer la présence de valeurs `undefined` en utilisant l'opérateur d'égalité stricte. À la fin de ce laboratoire, vous aurez une compréhension solide de la manière d'utiliser la fonction `isUndefined()` pour vérifier les valeurs `undefined` dans vos programmes JavaScript.
