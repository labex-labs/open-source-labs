# 소개

이 랩에서는 JavaScript 의 값 (value) 개념과 특정 값이 undefined 인지 확인하는 방법을 살펴봅니다. 이 랩을 통해 엄격한 동등 연산자 (strict equality operator) 를 사용하여 undefined 값의 존재 여부를 판단하는 코드를 직접 작성하는 경험을 얻을 수 있습니다. 이 랩을 마치면 JavaScript 프로그램에서 `isUndefined()` 함수를 사용하여 undefined 값을 확인하는 방법에 대한 확실한 이해를 갖게 될 것입니다.
