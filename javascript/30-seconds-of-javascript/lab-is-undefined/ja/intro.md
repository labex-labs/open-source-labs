# はじめに

この実験では、JavaScript における値の概念と、指定された値が `undefined` であるかどうかを確認する方法を探ります。この実験では、厳密な等値演算子を使用して `undefined` 値の存在を判断するコードを書く実践的な経験を得ます。この実験が終了するとき、JavaScript プログラムで `undefined` 値をチェックするための `isUndefined()` 関数の使い方を十分に理解しているでしょう。
