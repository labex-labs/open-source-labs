# Überprüfen auf `undefined`-Wert

Um zu überprüfen, ob ein Wert `undefined` ist, öffnen Sie das Terminal/SSH und geben Sie `node` ein.

- Verwenden Sie den strikt gleichen Operator, um zu überprüfen, ob `val` gleich `undefined` ist.

```js
const isUndefined = (val) => val === undefined;
```

```js
isUndefined(undefined); // true
```

Dieser Code wird überprüfen, ob der angegebene Wert `undefined` ist.
