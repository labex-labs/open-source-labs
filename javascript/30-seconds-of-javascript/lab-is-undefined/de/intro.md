# Einführung

In diesem Lab werden wir das Konzept des Werts in JavaScript untersuchen und lernen, wie man überprüft, ob ein bestimmter Wert `undefined` ist. In diesem Lab erhalten Sie praktische Erfahrungen beim Schreiben von Code, um mithilfe des strikt gleichen Operators festzustellen, ob `undefined`-Werte vorhanden sind. Am Ende dieses Labs werden Sie eine solide Vorstellung davon haben, wie die `isUndefined()`-Funktion verwendet wird, um `undefined`-Werte in Ihren JavaScript-Programmen zu überprüfen.
