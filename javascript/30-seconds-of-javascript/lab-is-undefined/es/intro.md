# Introducción

En este laboratorio, exploraremos el concepto de valor en JavaScript y cómo comprobar si un valor especificado es `undefined`. Este laboratorio te dará experiencia práctica en escribir código para determinar la presencia de valores `undefined` utilizando el operador de igualdad estricta. Al final de este laboratorio, tendrás una comprensión sólida de cómo utilizar la función `isUndefined()` para comprobar valores `undefined` en tus programas JavaScript.
