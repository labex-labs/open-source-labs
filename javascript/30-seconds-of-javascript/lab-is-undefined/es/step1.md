# Comprobando un valor `undefined`

Para comprobar si un valor es `undefined`, abre la Terminal/SSH y escribe `node`.

- Utiliza el operador de igualdad estricta para comprobar si `val` es igual a `undefined`.

```js
const isUndefined = (val) => val === undefined;
```

```js
isUndefined(undefined); // true
```

Este código comprobará si el valor especificado es `undefined`.
