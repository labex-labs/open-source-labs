# Introdução

Neste laboratório, vamos mergulhar no tópico de análise (parsing) de cookies HTTP em JavaScript. O objetivo deste laboratório é fornecer uma experiência prática de como extrair e organizar dados de cookies a partir de uma string de cabeçalho HTTP. Ao final deste laboratório, você terá uma compreensão clara de como usar a função `parseCookie` para criar um objeto contendo todos os pares nome-valor dos cookies.
