# Einführung

In diesem Lab werden wir das Konzept des Überprüfens, ob ein gegebener String in Kleinbuchstaben geschrieben ist, mit JavaScript erkunden. Wir werden uns mit der `String.prototype.toLowerCase()`-Methode befassen, die verwendet wird, um einen gegebenen String in Kleinbuchstaben umzuwandeln. Am Ende dieses Labs sollten Sie in der Lage sein, eine Funktion zu schreiben, die überprüft, ob ein String in Kleinbuchstaben geschrieben ist oder nicht.
