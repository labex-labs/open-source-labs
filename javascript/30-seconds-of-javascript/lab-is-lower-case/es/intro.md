# Introducción

En este laboratorio, exploraremos el concepto de comprobar si una cadena de texto dada está en minúsculas o no utilizando JavaScript. Aprenderemos sobre el método `String.prototype.toLowerCase()`, que se utiliza para convertir una cadena de texto dada a minúsculas. Al final de este laboratorio, serás capaz de escribir una función que compruebe si una cadena está en minúsculas o no.
