# Introdução

Neste laboratório, exploraremos o conceito de verificar se uma determinada string está em minúsculas ou não, usando JavaScript. Aprenderemos sobre o método `String.prototype.toLowerCase()`, que é usado para converter uma string em minúsculas. Ao final deste laboratório, você será capaz de escrever uma função que verifica se uma string está em minúsculas ou não.
