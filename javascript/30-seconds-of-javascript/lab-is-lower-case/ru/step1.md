# JavaScript-функция для проверки, является ли строка строчной

Чтобы проверить, является ли данная строка строчной, вы можете использовать следующую JavaScript-функцию. Во - первых, преобразуйте строку в строчные буквы с использованием `String.prototype.toLowerCase()`, а затем сравните ее с исходной строкой с использованием строгой равенства (`===`).

```js
const isLowerCase = (str) => str === str.toLowerCase();
```

Вот пример использования:

```js
isLowerCase("abc"); // true
isLowerCase("a3@$"); // true
isLowerCase("Ab4"); // false
```

Для использования этой функции откройте Терминал/SSH и введите `node`.
