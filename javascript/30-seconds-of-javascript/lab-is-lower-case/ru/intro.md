# Введение

В этом практическом занятии мы изучим концепцию проверки того, является ли данная строка строчными буквами, с использованием JavaScript. Мы узнаем о методе `String.prototype.toLowerCase()`, который используется для преобразования заданной строки в строчные буквы. В конце этого практического занятия вы сможете написать функцию, которая проверяет, является ли строка строчными буквами или нет.
