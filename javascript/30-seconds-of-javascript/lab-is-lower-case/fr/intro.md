# Introduction

Dans ce laboratoire, nous allons explorer le concept de vérification pour savoir si une chaîne de caractères donnée est en minuscules ou non en utilisant JavaScript. Nous allons apprendre à utiliser la méthode `String.prototype.toLowerCase()`, qui est utilisée pour convertir une chaîne de caractères donnée en minuscules. À la fin de ce laboratoire, vous serez capable d'écrire une fonction qui vérifie si une chaîne est en minuscules ou non.
