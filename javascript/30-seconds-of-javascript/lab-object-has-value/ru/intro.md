# Введение

В этом практическом занятии мы изучим, как проверить, содержит ли объект определенное значение с использованием JavaScript. Мы узнаем, как использовать метод `Object.values()` для извлечения всех значений объекта и как использовать метод `Array.prototype.includes()` для проверки наличия целевого значения в массиве значений. Эти знания помогут нам эффективно проверять значения в JSON-объектах в наших проектах на JavaScript.
