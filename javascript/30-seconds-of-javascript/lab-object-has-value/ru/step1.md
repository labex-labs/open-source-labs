# Функция для проверки наличия определенного значения в объекте

Для проверки наличия определенного значения в объекте используйте следующую функцию:

```js
const hasValue = (obj, value) => Object.values(obj).includes(value);
```

Для использования этой функции передайте в качестве аргументов объект, в котором вы хотите выполнить поиск, и целевой объект. Функция вернет `true`, если объект содержит значение, и `false`, если не содержит.

Вот пример:

```js
const obj = { a: 100, b: 200 };
console.log(hasValue(obj, 100)); // true
console.log(hasValue(obj, 999)); // false
```

Для начала работы с кодом откройте Терминал/SSH и введите `node`.
