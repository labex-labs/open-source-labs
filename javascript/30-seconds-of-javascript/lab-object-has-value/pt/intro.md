# Introdução

Neste laboratório, exploraremos como verificar se um objeto possui um valor específico usando JavaScript. Aprenderemos como usar o método `Object.values()` para extrair todos os valores de um objeto e como usar o método `Array.prototype.includes()` para verificar se um valor alvo existe no array de valores. Este conhecimento nos ajudará a verificar eficientemente valores em objetos JSON em nossos projetos JavaScript.
