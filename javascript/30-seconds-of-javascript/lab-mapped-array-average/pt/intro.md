# Introdução

Neste laboratório, exploraremos o conceito de mapeamento (mapping) e redução (reducing) de arrays em JavaScript. Especificamente, aprenderemos como usar a função `averageBy()` para calcular a média de um array após mapear cada elemento para um valor usando uma função fornecida. Através de exercícios práticos e exemplos, obteremos uma compreensão mais profunda desses poderosos métodos de array e como eles podem ser usados para otimizar nosso código.
