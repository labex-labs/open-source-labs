# Introduction

Dans ce laboratoire, nous allons explorer le concept de suppression d'éléments d'un tableau en JavaScript. Nous utiliserons la méthode `slice()` pour créer un nouveau tableau en supprimant un nombre spécifié d'éléments au début. Grâce à ce laboratoire, vous apprendrez à manipuler les tableaux en JavaScript et à mieux comprendre le fonctionnement de la méthode `slice()`.
