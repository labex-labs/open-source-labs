# JavaScript 에서 배열 요소 제거 방법

JavaScript 에서 배열의 시작 부분에서 요소를 제거하려면 다음 단계를 따르세요.

1. 터미널 또는 SSH 를 열고 `node`를 입력하여 코딩 연습을 시작합니다.
2. `Array.prototype.slice()` 메서드를 사용하여 시작 부분에서 `n`개의 요소가 제거된 새로운 배열을 생성합니다.
3. 아래 코드 조각에서 `take` 함수를 사용하여 로직을 구현합니다.

```js
const take = (arr, n = 1) => arr.slice(0, n);
```

`take` 함수를 사용하는 예시는 다음과 같습니다.

```js
take([1, 2, 3], 5); // [1, 2, 3]
take([1, 2, 3], 0); // []
```

첫 번째 예시에서 `take([1, 2, 3], 5)`는 `[1, 2, 3]`을 반환합니다. 배열에 요소가 3 개뿐이기 때문입니다. 두 번째 예시에서 `take([1, 2, 3], 0)`은 `[]`을 반환합니다. 배열의 시작 부분에서 아무런 요소도 가져오지 않기 때문입니다.
