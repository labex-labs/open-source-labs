# Einführung

In diesem Lab werden wir das Konzept des Entfernens von Elementen aus einem Array in JavaScript erkunden. Wir werden die `slice()`-Methode verwenden, um ein neues Array zu erstellen, aus dem eine bestimmte Anzahl von Elementen am Anfang entfernt wird. Durch dieses Lab lernen Sie, wie Sie Arrays in JavaScript manipulieren, und erhalten ein tieferes Verständnis davon, wie die `slice()`-Methode funktioniert.
