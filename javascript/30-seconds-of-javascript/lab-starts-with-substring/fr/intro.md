# Introduction

Dans ce laboratoire, nous allons explorer comment vérifier si une chaîne de caractères donnée commence par une sous-chaîne d'une autre chaîne de caractères à l'aide de JavaScript. Nous allons apprendre à utiliser la boucle `for...in` et `String.prototype.slice()` pour obtenir chaque sous-chaîne d'un mot donné, et `String.prototype.startsWith()` pour vérifier si la sous-chaîne actuelle correspond au texte. À la fin de ce laboratoire, vous serez capable de trouver efficacement si une chaîne de caractères commence par une sous-chaîne spécifique en JavaScript.
