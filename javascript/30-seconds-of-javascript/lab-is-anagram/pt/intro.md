# Introdução

Neste laboratório, exploraremos como verificar se uma determinada string é um anagrama de outra string. Um anagrama é uma palavra ou frase formada pela reorganização das letras de outra, como "iceman" e "cinema". Usaremos JavaScript para criar uma função que recebe duas strings como argumentos e retorna um valor booleano indicando se elas são anagramas uma da outra ou não.
