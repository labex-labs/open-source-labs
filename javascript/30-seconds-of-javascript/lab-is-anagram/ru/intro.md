# Введение

В этом практическом занятии мы изучим, как проверить, является ли данная строка анаграммой другой строки. Анаграмма - это слово или фраза, образованные путём перестановки букв другой, например, "iceman" и "cinema". Мы будем использовать JavaScript для создания функции, которая принимает две строки в качестве аргументов и возвращает логическое значение, указывающее, являются ли они анаграммами друг друга или нет.
