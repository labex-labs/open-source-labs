# Как использовать логический Nor в JavaScript

Для начала кодирования на JavaScript откройте Терминал/SSH и введите `node`. Логический Nor проверяет, являются ли истинными ни один из заданных аргументов. Чтобы вернуть обратное значение логического или двух значений, используйте логический оператор отрицания (`!`). Вот пример:

```js
const nor = (a, b) => !(a || b);
```

И вот некоторые результаты:

```js
nor(true, true); // false
nor(true, false); // false
nor(false, false); // true
```
