# Introdução

Neste laboratório, exploraremos o tópico da manipulação de arrays em JavaScript. Especificamente, focaremos na tarefa de dividir um array em pedaços menores de um tamanho especificado. Através deste laboratório, você aprenderá como usar métodos JavaScript embutidos para manipular arrays de forma eficiente e eficaz. Ao final deste laboratório, você terá adquirido uma compreensão mais profunda da manipulação de arrays em JavaScript e será capaz de aplicar essas habilidades para resolver problemas do mundo real.
