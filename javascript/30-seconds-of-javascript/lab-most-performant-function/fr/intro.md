# Introduction

Dans ce laboratoire, nous allons explorer et pratiquer la manière d'écrire une fonction performante qui renvoie l'index de la fonction s'exécutant le plus rapidement à partir d'un tableau de fonctions. Le laboratoire se concentre sur l'utilisation de `performance.now()` et de `Array.prototype.map()` pour mesurer précisément le temps d'exécution des fonctions et les comparer. À la fin du laboratoire, vous aurez une meilleure compréhension de la manière d'optimiser votre code pour obtenir les meilleures performances.
