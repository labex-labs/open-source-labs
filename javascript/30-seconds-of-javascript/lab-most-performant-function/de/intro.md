# Einführung

In diesem Lab werden wir untersuchen und üben, wie man eine leistungsfähige Funktion schreibt, die den Index der am schnellsten ausführenden Funktion aus einem Array von Funktionen zurückgibt. Das Lab konzentriert sich auf die Verwendung von `performance.now()` und `Array.prototype.map()` zur genauen Messung der Ausführungszeit von Funktionen und zum Vergleich dieser. Am Ende des Labs werden Sie ein besseres Verständnis dafür haben, wie Sie Ihren Code für die maximale Leistung optimieren können.
