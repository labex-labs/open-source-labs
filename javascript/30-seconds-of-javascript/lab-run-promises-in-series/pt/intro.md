# Introdução

Neste laboratório, exploraremos como executar um array de promises em série usando JavaScript. Usaremos o método `Array.prototype.reduce()` para criar uma cadeia de promises, onde cada promise retorna a próxima promise quando resolvida. Ao final deste laboratório, você terá uma melhor compreensão de como executar promises sequencialmente, elevando suas habilidades de programação a um novo nível.
