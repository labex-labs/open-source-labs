# Einführung

In diesem Lab werden wir untersuchen, wie man in JavaScript eine Reihe von Promises nacheinander ausführt. Wir werden die `Array.prototype.reduce()`-Methode verwenden, um eine Promise-Kette zu erstellen, bei der jedes Promise beim Auflösen das nächste Promise zurückgibt. Am Ende dieses Labs werden Sie ein besseres Verständnis dafür haben, wie man Promises sequentiell ausführt und Ihre Programmierfähigkeiten auf die nächste Stufe heben.
