# Introduction

Dans ce laboratoire, nous allons apprendre à exécuter des fonctions à long terme de manière asynchrone en utilisant un Web Worker. Le but du laboratoire est d'enseigner aux développeurs comment exécuter des fonctions dans un fil séparé, leur permettant d'éviter de bloquer l'interface utilisateur pendant que la fonction est en cours d'exécution. Le laboratoire fournit un exemple pratique de création d'un Web Worker et de son utilisation pour exécuter une fonction.
