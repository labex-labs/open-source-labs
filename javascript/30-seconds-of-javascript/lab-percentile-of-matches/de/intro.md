# Einführung

In diesem Lab werden wir das Konzept des Berechnens des Percentils von Übereinstimmungen in einem Array mit JavaScript erkunden. Mit Hilfe der `Array.prototype.reduce()`-Methode werden wir eine Funktion entwickeln, die den Prozentsatz der Zahlen in einem Array bestimmen kann, die kleiner oder gleich einem angegebenen Wert sind. Dieses Lab wird dazu beitragen, Ihr Verständnis von JavaScript-Arraymethoden und deren praktischen Anwendungen zu stärken.
