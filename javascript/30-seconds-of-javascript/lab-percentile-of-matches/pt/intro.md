# Introdução

Neste laboratório, exploraremos o conceito de cálculo da percentila de correspondências em um array usando JavaScript. Através do uso do método `Array.prototype.reduce()`, desenvolveremos uma função que pode determinar a porcentagem de números em um array que são menores ou iguais a um determinado valor. Este laboratório ajudará a reforçar sua compreensão dos métodos de array JavaScript e suas aplicações práticas.
