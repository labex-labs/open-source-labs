# Введение

В этом лабе мы будем изучать концепцию вычисления процентиля совпадений в массиве с использованием JavaScript. С использованием метода `Array.prototype.reduce()` мы разработаем функцию, которая сможет определить процент чисел в массиве, которые меньше или равны заданному значению. Этот лаб поможет укрепить ваше понимание методов массивов JavaScript и их практических приложений.
