# Introduction

Dans ce laboratoire, nous allons explorer le concept de calcul du centile de correspondances dans un tableau à l'aide de JavaScript. En utilisant la méthode `Array.prototype.reduce()`, nous allons développer une fonction qui peut déterminer le pourcentage de nombres dans un tableau qui sont inférieurs ou égaux à une valeur donnée. Ce laboratoire vous aidera à renforcer votre compréhension des méthodes de tableaux JavaScript et de leurs applications pratiques.
