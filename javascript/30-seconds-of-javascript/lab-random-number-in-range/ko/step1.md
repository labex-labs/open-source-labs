# JavaScript 를 사용하여 주어진 범위 내에서 난수 생성하는 방법

JavaScript 를 사용하여 지정된 범위 내에서 난수를 생성하려면 다음 단계를 따르세요.

1. 터미널/SSH 를 열고 `node`를 입력하여 코딩 연습을 시작합니다.
2. `Math.random()` 메서드를 사용하여 임의의 값을 생성합니다.
3. 생성된 값을 곱셈을 사용하여 원하는 범위에 매핑합니다.
4. 다음 코드를 사용하여 주어진 범위 내에서 난수를 생성하는 함수를 만듭니다.

```js
const randomNumberInRange = (min, max) => Math.random() * (max - min) + min;
```

5. 함수를 사용하려면 원하는 범위의 최소값과 최대값을 인수로 전달합니다. 예를 들어:

```js
randomNumberInRange(2, 10); // 6.0211363285087005
```

이러한 단계를 따르면 JavaScript 를 사용하여 주어진 범위 내에서 난수를 쉽게 생성할 수 있습니다.
