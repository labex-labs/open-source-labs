# Introducción

En este laboratorio, exploraremos el concepto de crear un generador de números aleatorios en JavaScript. Utilizaremos el método `Math.random()` para generar un valor aleatorio y lo mapearemos a un rango deseado mediante multiplicación. Al final de este laboratorio, tendrás una mejor comprensión de cómo generar números aleatorios en JavaScript y cómo utilizarlos en tus proyectos.
