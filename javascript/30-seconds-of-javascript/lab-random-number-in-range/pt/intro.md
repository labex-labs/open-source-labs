# Introdução

Neste laboratório, exploraremos o conceito de criação de um gerador de números aleatórios em JavaScript. Usaremos o método `Math.random()` para gerar um valor aleatório e mapeá-lo para um intervalo desejado usando multiplicação. Ao final deste laboratório, você terá uma melhor compreensão de como gerar números aleatórios em JavaScript e como usá-los em seus projetos.
