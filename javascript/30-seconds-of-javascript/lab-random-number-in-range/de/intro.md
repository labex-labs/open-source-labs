# Einführung

In diesem Lab werden wir das Konzept des Erstellens eines Zufallszahlengenerators in JavaScript erkunden. Wir werden die `Math.random()`-Methode verwenden, um einen Zufallswert zu generieren und ihn mithilfe von Multiplikation auf einen gewünschten Bereich abzubilden. Am Ende dieses Labs werden Sie ein besseres Verständnis dafür haben, wie Sie in JavaScript Zufallszahlen generieren und wie Sie sie in Ihren Projekten verwenden.
