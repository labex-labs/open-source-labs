# JavaScript を使って指定された范囲内で乱数を生成する方法

JavaScript を使って指定された范囲内で乱数を生成するには、次の手順に従います。

1. ターミナル/SSH を開き、コーディングの練習を始めるために `node` と入力します。
2. `Math.random()` メソッドを使って乱数を生成します。
3. 生成された値を乗算を使って目的の範囲にマッピングします。
4. 次のコードを使って、指定された範囲内で乱数を生成する関数を作成します。

```js
const randomNumberInRange = (min, max) => Math.random() * (max - min) + min;
```

5. 関数を使用するには、目的の範囲の最小値と最大値を引数として渡します。たとえば：

```js
randomNumberInRange(2, 10); // 6.0211363285087005
```

これらの手順に従えば、JavaScript を使って指定された範囲内で簡単に乱数を生成できます。
