# Введение

В этом практическом занятии мы изучим концепцию создания генератора случайных чисел на JavaScript. Мы будем использовать метод `Math.random()` для генерации случайного значения и отображать его в желаемый диапазон с использованием умножения. В конце этого практического занятия вы будете лучше понимать, как генерировать случайные числа на JavaScript и как использовать их в своих проектах.
