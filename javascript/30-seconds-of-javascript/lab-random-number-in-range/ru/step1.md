# Как сгенерировать случайное число в заданном диапазоне с использованием JavaScript

Чтобы сгенерировать случайное число в заданном диапазоне с использованием JavaScript, следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте метод `Math.random()` для генерации случайного значения.
3. Отобразите сгенерированное значение в желаемый диапазон с использованием умножения.
4. Используйте следующий код, чтобы создать функцию, которая генерирует случайное число в заданном диапазоне:

```js
const randomNumberInRange = (min, max) => Math.random() * (max - min) + min;
```

5. Чтобы использовать функцию, передайте в качестве аргументов минимальное и максимальное значения желаемого диапазона. Например:

```js
randomNumberInRange(2, 10); // 6.0211363285087005
```

Следуя этим шагам, вы можете легко сгенерировать случайное число в заданном диапазоне с использованием JavaScript.
