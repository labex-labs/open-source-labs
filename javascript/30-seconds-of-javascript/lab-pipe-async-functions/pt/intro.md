# Introdução

Neste laboratório, exploraremos o conceito de composição de funções em programação assíncrona usando JavaScript. Aprenderemos como usar a função `pipeAsyncFunctions` para compor uma série de funções assíncronas que podem aceitar um único argumento e retornar uma combinação de valores normais, `Promise`s, ou ser `async`. Ao final deste laboratório, você terá uma sólida compreensão de como usar a composição de funções para escrever código assíncrono mais eficiente e legível.
