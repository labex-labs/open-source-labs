# Introdução

Neste laboratório, exploraremos uma função JavaScript que verifica se uma data fornecida é válida ou não. A função utiliza o operador spread (spread operator) e o construtor `Date` para criar um novo objeto `Date` a partir dos argumentos fornecidos. Em seguida, ela usa o método `valueOf()` e `Number.isNaN()` para verificar se o objeto é válido. Este laboratório é uma ótima oportunidade para praticar o trabalho com funções JavaScript e objetos de data.
