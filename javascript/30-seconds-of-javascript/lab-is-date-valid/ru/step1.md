# Как проверить, является ли дата действительной

Для проверки того, является ли дата действительной, следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковать программирование.
2. Используйте оператор расширения (`...`), чтобы передать массив аргументов конструктору `Date`.
3. Используйте `Date.prototype.valueOf()` и `Number.isNaN()`, чтобы проверить, можно ли создать действительный объект `Date` из заданных значений.

Вот пример кода:

```js
const isDateValid = (...val) => !Number.isNaN(new Date(...val).valueOf());
```

Вы можете протестировать функцию с разными значениями, как показано ниже:

```js
isDateValid("December 17, 1995 03:24:00"); // true
isDateValid("1995-12-17T03:24:00"); // true
isDateValid("1995-12-17 T03:24:00"); // false
isDateValid("Duck"); // false
isDateValid(1995, 11, 17); // true
isDateValid(1995, 11, 17, "Duck"); // false
isDateValid({}); // false
```
