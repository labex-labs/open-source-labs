# Как сгенерировать случайное булево значение в JavaScript

Чтобы сгенерировать случайное булево значение в JavaScript, следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте метод `Math.random()` для генерации случайного числа.
3. Проверьте, больше ли случайное число или равно `0.5`.
4. Возвращайте `true`, если число больше или равно `0.5`, в противном случае возвращайте `false`.

Вот краткое реализация кода:

```js
const randomBoolean = () => Math.random() >= 0.5;
```

Вы можете протестировать функцию, вызвав `randomBoolean()`, которая вернет либо `true`, либо `false`.
