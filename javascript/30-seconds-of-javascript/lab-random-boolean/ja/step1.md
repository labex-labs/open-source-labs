# JavaScriptでランダムなブール値を生成する方法

JavaScriptでランダムなブール値を生成するには、次の手順に従います。

1. ターミナル/SSHを開き、コーディングの練習を始めるために `node` と入力します。
2. `Math.random()` メソッドを使用してランダムな数値を生成します。
3. 生成されたランダムな数値が `0.5` 以上であるかどうかを確認します。
4. 数値が `0.5` 以上の場合は `true` を返し、それ以外の場合は `false` を返します。

以下は、コードの簡潔な実装例です。

```js
const randomBoolean = () => Math.random() >= 0.5;
```

`randomBoolean()` を呼び出すことで関数をテストでき、`true` または `false` が返されます。
