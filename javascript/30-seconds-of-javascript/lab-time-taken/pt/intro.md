# Introdução

Neste laboratório, exploraremos como medir o tempo de execução de uma função usando JavaScript. Usaremos os métodos `console.time()` e `console.timeEnd()` para medir a diferença entre os tempos de início e fim de uma função e determinar quanto tempo levou para ser executada. Essa habilidade é útil para otimizar código e melhorar o desempenho no desenvolvimento web.
