# Introdução

Neste laboratório, exploraremos como gerar uma _query string_ a partir de pares chave-valor de um objeto usando JavaScript. Usaremos o método `Array.prototype.reduce()` em `Object.entries()` para criar a _query string_ e determinar o símbolo apropriado a ser usado com base no comprimento da `queryString`. Ao final deste laboratório, você será capaz de gerar eficientemente uma _query string_ para suas aplicações web.
