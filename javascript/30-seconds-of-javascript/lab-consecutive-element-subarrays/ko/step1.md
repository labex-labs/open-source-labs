# 연속 요소 하위 배열

코딩 연습을 위해 터미널/SSH 를 열고 `node`를 입력하십시오. 다음 코드는 연속적인 요소의 `n`-튜플 배열을 생성합니다.

```js
const aperture = (n, arr) =>
  n > arr.length ? [] : arr.slice(n - 1).map((v, i) => arr.slice(i, i + n));
```

함수를 사용하려면:

- `n`을 연속 요소의 수로, `arr`을 숫자 배열로 하여 `aperture(n, arr)` 함수를 호출합니다.
- 이 함수는 `arr`에서 연속적인 요소의 `n`-튜플 배열을 반환합니다.
- `n`이 `arr`의 길이보다 크면, 함수는 빈 배열을 반환합니다.

사용 예시:

```js
aperture(2, [1, 2, 3, 4]); // [[1, 2], [2, 3], [3, 4]]
aperture(3, [1, 2, 3, 4]); // [[1, 2, 3], [2, 3, 4]]
aperture(5, [1, 2, 3, 4]); // []
```
