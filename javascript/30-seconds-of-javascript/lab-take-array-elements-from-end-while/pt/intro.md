# Introdução

Neste laboratório, trabalharemos em um desafio de programação JavaScript que envolve a manipulação de arrays. O objetivo deste laboratório é escrever uma função que remove elementos do final de um array até que uma condição específica seja atendida, e retorna os elementos removidos. Você usará vários métodos de array JavaScript e uma função de callback para completar este desafio.
