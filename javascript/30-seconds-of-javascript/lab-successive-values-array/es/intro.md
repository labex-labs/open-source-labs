# Introducción

En este laboratorio, exploraremos cómo usar la función `reduceSuccessive()` para aplicar una función dada contra un acumulador y cada elemento de una matriz, devolviendo una matriz de valores reducidos sucesivamente. Aprenderemos cómo usar `Array.prototype.reduce()` para aplicar la función a la matriz y almacenar cada nuevo resultado. Al final de este laboratorio, tendrás una mejor comprensión de cómo usar estos métodos para manipular matrices en JavaScript.
