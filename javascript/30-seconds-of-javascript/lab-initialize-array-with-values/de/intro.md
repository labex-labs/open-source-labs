# Einführung

In diesem Lab werden wir untersuchen, wie man in JavaScript ein Array initialisiert und mit bestimmten Werten füllt. Wir werden den `Array()`-Konstruktor und die `Array.prototype.fill()`-Methode verwenden, um das Array zu erstellen und zu befüllen. Am Ende des Labs wirst du ein tieferes Verständnis dafür haben, wie man in JavaScript Arrays erstellt und manipuliert.
