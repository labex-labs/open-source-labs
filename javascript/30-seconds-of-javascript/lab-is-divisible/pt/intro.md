# Introdução

Neste laboratório, mergulharemos no mundo da programação JavaScript e aprenderemos a criar uma função que verifica se um número é divisível por outro número. Através deste laboratório, você obterá uma compreensão mais profunda do operador módulo (modulo operator) e como ele pode ser usado para realizar cálculos em JavaScript. Ao final deste laboratório, você terá uma solução funcional e eficiente para determinar se um número é divisível por outro.
