# Введение

В этом практическом занятии мы углубимся в мир программирования на JavaScript и научимся создавать функцию, которая проверяет, делится ли одно число на другое. В ходе этого практического занятия вы глубже познакомитесь с оператором модуля и узнаете, как его можно использовать для выполнения вычислений в JavaScript. В конце практического занятия у вас будет функциональное и эффективное решение для определения, делится ли одно число на другое.
