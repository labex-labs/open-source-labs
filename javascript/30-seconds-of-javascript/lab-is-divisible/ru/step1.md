# Проверить, делится ли число

Чтобы проверить, делится ли число на другое число в JavaScript, следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте оператор модуля (`%`), чтобы проверить, равно ли остаток от деления `0`. Если да, то число делится.

Вот пример функции, которая проверяет, делится ли первый числовой аргумент на второй:

```js
const isDivisible = (dividend, divisor) => dividend % divisor === 0;
```

Вы можете протестировать эту функцию с помощью `isDivisible(6, 3)`, что должно вернуть `true`, так как 6 делится на 3.
