# Introducción

En este laboratorio, profundizaremos en el mundo de la programación de JavaScript y aprenderemos cómo crear una función que verifique si un número es divisible por otro número. A través de este laboratorio, obtendrás una comprensión más profunda del operador módulo y de cómo se puede utilizar para realizar cálculos en JavaScript. Al final de este laboratorio, tendrás una solución funcional y eficiente para determinar si un número es divisible por otro.
