# Introduction

Dans ce laboratoire, nous plongerons dans le monde de la programmation JavaScript et apprendrons à créer une fonction qui vérifie si un nombre est divisible par un autre nombre. Grâce à ce laboratoire, vous comprendrez mieux l'opérateur modulo et comment il peut être utilisé pour effectuer des calculs en JavaScript. À la fin de ce laboratoire, vous aurez une solution fonctionnelle et efficace pour déterminer si un nombre est divisible par un autre.
