# 숫자가 나누어 떨어지는지 확인하기

JavaScript 에서 한 숫자가 다른 숫자로 나누어 떨어지는지 확인하려면 다음 단계를 따르세요.

1. 터미널/SSH 를 열고 `node`를 입력하여 코딩 연습을 시작합니다.
2. 모듈로 연산자 (`%`) 를 사용하여 나눗셈의 나머지가 `0`과 같은지 확인합니다. 만약 그렇다면, 해당 숫자는 나누어 떨어지는 것입니다.

다음은 첫 번째 숫자 인수가 두 번째 인수로 나누어 떨어지는지 확인하는 예시 함수입니다.

```js
const isDivisible = (dividend, divisor) => dividend % divisor === 0;
```

`isDivisible(6, 3)`으로 이 함수를 테스트할 수 있으며, 6 이 3 으로 나누어 떨어지므로 `true`를 반환해야 합니다.
