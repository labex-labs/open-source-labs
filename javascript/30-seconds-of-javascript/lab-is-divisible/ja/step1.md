# 数が割り切れるかどうかを確認する

JavaScript である数が別の数で割り切れるかどうかを確認するには、次の手順に従います。

1. ターミナル/SSH を開き、コーディングの練習を始めるために `node` と入力します。
2. モジュロ演算子 (`%`) を使用して、除算の余りが `0` に等しいかどうかを確認します。もし等しい場合、その数は割り切れます。

以下は、最初の数値引数が 2 番目の引数で割り切れるかどうかを確認する関数の例です。

```js
const isDivisible = (dividend, divisor) => dividend % divisor === 0;
```

この関数を `isDivisible(6, 3)` でテストすることができます。6 は 3 で割り切れるため、`true` が返されるはずです。
