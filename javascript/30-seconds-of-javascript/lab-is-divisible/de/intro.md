# Einführung

In diesem Lab werden wir uns in die Welt der JavaScript-Programmierung stürzen und lernen, wie man eine Funktion erstellt, die überprüft, ob eine Zahl durch eine andere Zahl teilbar ist. Durch dieses Lab werden Sie ein tieferes Verständnis des Modulo-Operators und dessen Verwendung für Berechnungen in JavaScript erlangen. Am Ende dieses Labs werden Sie eine funktionierende und effiziente Lösung haben, um zu bestimmen, ob eine Zahl durch eine andere teilbar ist.
