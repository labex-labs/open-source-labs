# Проверить, является ли число степенью двойки

Для проверки того, является ли число степенью двойки, следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте побитовый оператор AND (`&`) для определения, является ли число (`n`) степенью `2`.
3. Кроме того, проверьте, что `n` не ложно.
4. Следующий код функционально проверяет, является ли `n` степенью двойки:

```js
const isPowerOfTwo = (n) => !!n && (n & (n - 1)) == 0;
```

Вот несколько примеров использования функции `isPowerOfTwo`:

```js
isPowerOfTwo(0); // false
isPowerOfTwo(1); // true
isPowerOfTwo(8); // true
```
