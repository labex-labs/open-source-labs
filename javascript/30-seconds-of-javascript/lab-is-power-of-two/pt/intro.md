# Introdução

Neste laboratório, exploraremos uma função JavaScript que determina se um determinado número é uma potência de `2`. Ao usar o operador AND binário bit a bit (bitwise binary AND operator) e verificar se o número não é falsy, podemos determinar com precisão se um número é uma potência de `2`. Este laboratório proporcionará uma oportunidade para praticar o uso de operadores bit a bit e operadores lógicos em JavaScript.
