# Introduction

Dans ce laboratoire, nous allons explorer une fonction JavaScript qui détermine si un nombre donné est une puissance de `2`. En utilisant l'opérateur binaire ET bit à bit et en vérifiant que le nombre n'est pas faux, nous pouvons déterminer avec précision si un nombre est une puissance de `2`. Ce laboratoire vous offrira l'occasion de pratiquer l'utilisation d'opérateurs bit à bit et d'opérateurs logiques en JavaScript.
