# Introducción

En este laboratorio, exploraremos una función de JavaScript que determina si un número dado es una potencia de `2`. Al utilizar el operador AND binario bit a bit y comprobar que el número no es falso, podemos determinar con precisión si un número es una potencia de `2`. Este laboratorio ofrecerá la oportunidad de practicar el uso de operadores bit a bit y operadores lógicos en JavaScript.
