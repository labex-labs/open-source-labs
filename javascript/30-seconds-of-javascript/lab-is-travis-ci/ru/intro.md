# Введение

В этом практическом занятии мы будем изучать основы языка программирования JavaScript с помощью серии практических упражнений. Цель этого практического занятия - помочь вам получить фундаментальное понимание синтаксиса JavaScript, типов данных, операторов, управляющих конструкций, функций и объектов. В конце этого практического занятия у вас будут знания и навыки, необходимые для написания базовых программ на JavaScript и создания простых приложений.
