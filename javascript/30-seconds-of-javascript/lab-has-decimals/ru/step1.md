# Как проверить, имеет ли число десятичные цифры

Чтобы проверить, имеет ли число десятичные цифры, можно использовать оператор модуля в JavaScript. Следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте оператор модуля (`%`), чтобы проверить, делится ли число на `1`.
3. Если результат не равен нулю, то число имеет десятичные цифры.

Вот пример кода для проверки наличия десятичных цифр в числе:

```js
const hasDecimals = (num) => num % 1 !== 0;
```

Вы можете протестировать функцию, вызвав ее с разными числами, например, так:

```js
hasDecimals(1); // false
hasDecimals(1.001); // true
```
