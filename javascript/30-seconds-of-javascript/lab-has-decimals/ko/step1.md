# 숫자에 소수 자릿수가 있는지 확인하는 방법

숫자에 소수 자릿수가 있는지 확인하려면 JavaScript 에서 모듈로 연산자 (modulo operator) 를 사용할 수 있습니다. 다음 단계를 따르세요:

1. 터미널/SSH 를 열고 `node`를 입력하여 코딩 연습을 시작합니다.
2. 모듈로 연산자 (`%`) 를 사용하여 숫자가 `1`로 나누어 떨어지는지 확인합니다.
3. 결과가 0 이 아니면 숫자에 소수 자릿수가 있는 것입니다.

다음은 숫자에 소수 자릿수가 있는지 확인하는 예제 코드입니다:

```js
const hasDecimals = (num) => num % 1 !== 0;
```

다음과 같이 다른 숫자로 함수를 호출하여 테스트할 수 있습니다:

```js
hasDecimals(1); // false
hasDecimals(1.001); // true
```
