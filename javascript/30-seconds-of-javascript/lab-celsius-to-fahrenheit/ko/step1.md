# 섭씨를 화씨로 변환하기

코딩을 사용하여 섭씨를 화씨로 변환하려면 다음 단계를 따르세요.

1. 터미널/SSH 를 엽니다.
2. `node`를 입력합니다.
3. 변환 공식 `F = 1.8 * C + 32`를 사용합니다.
4. 다음 코드를 사용하여 공식을 구현합니다.

```js
const celsiusToFahrenheit = (degrees) => 1.8 * degrees + 32;
```

5. 섭씨 온도 값을 입력하여 함수를 테스트합니다. 예를 들어 다음과 같습니다.

```js
celsiusToFahrenheit(33); // 91.4
```

이렇게 하면 해당 화씨 온도 값이 출력됩니다.
