# Introduction

Dans ce laboratoire, nous allons explorer l'implémentation de la fonction `differenceBy` en JavaScript. Cette fonction nous permet de trouver la différence entre deux tableaux en appliquant une fonction fournie à chaque élément des deux tableaux. Nous allons apprendre à utiliser `Set`, `Array.prototype.map()` et `Array.prototype.filter()` pour comparer et filtrer efficacement les tableaux selon un critère spécifique.
