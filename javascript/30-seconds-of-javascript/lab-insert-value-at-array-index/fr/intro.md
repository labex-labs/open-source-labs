# Introduction

Dans ce laboratoire, nous allons explorer le concept d'insérer des valeurs dans un tableau à un index spécifié en utilisant JavaScript. Vous allez apprendre à utiliser la méthode `Array.prototype.splice()` pour insérer des valeurs après un index donné avec un compte de suppression de `0`. Ce laboratoire vous procurera une expérience pratique dans la manipulation de tableaux en JavaScript et vous aidera à comprendre comment modifier le tableau original tout en insérant de nouvelles valeurs.
