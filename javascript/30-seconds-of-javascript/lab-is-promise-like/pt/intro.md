# Introdução

Neste laboratório, exploraremos o conceito de Promises em JavaScript e aprenderemos como identificar se um objeto é semelhante a uma Promise usando a função `isPromiseLike`. Através de exemplos práticos, entenderemos como as Promises funcionam e como aproveitá-las para programação assíncrona. Ao final deste laboratório, você terá uma sólida compreensão de Promises e como identificá-las em seu código.
