# Einführung

In diesem Lab werden wir das Konzept von Promises in JavaScript erkunden und lernen, wie man mithilfe der `isPromiseLike`-Funktion feststellt, ob ein Objekt promise-ähnlich ist. Anhand von praktischen Beispielen werden wir verstehen, wie Promises funktionieren und wie wir sie für asynchrone Programmierung nutzen können. Am Ende dieses Labs werden Sie ein solides Verständnis von Promises und davon haben, wie Sie sie in Ihrem Code identifizieren können.
