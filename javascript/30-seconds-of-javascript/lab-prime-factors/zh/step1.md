# 如何使用试除法找到一个数的质因数

要使用试除法找到给定数字的质因数，请遵循以下步骤：

- 打开终端/SSH 并输入`node`以开始练习编码。
- 使用`while`循环遍历所有可能的质因数，从`2`开始。
- 如果当前因数`f`能整除`n`，则将`f`添加到因数数组中，并将`n`除以`f`。否则，将`f`加一。
- 函数`primeFactors`接受一个数字`n`作为输入，并返回其质因数的数组。
- 要测试该函数，请调用`primeFactors(147)`，它将返回`[3, 7, 7]`。

以下是 JavaScript 代码：

```js
const primeFactors = (n) => {
  let a = [],
    f = 2;
  while (n > 1) {
    if (n % f === 0) {
      a.push(f);
      n /= f;
    } else {
      f++;
    }
  }
  return a;
};
```

请记住将`147`替换为你想要找到其质因数的数字。
