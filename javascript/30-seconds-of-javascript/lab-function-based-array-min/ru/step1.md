# Функция для возврата минимального значения массива

Для начала практики программирования откройте Терминал/SSH и введите `node`.

Эта функция возвращает минимальное значение массива, основываясь на предоставленной функции.

Для этого она использует `Array.prototype.map()`, чтобы сопоставить каждый элемент с значением, возвращаемым функцией. Затем она использует `Math.min()`, чтобы получить минимальное значение.

```js
const minBy = (arr, fn) =>
  Math.min(...arr.map(typeof fn === "function" ? fn : (val) => val[fn]));
```

Вы можете использовать эту функцию, передав массив и функцию. Например:

```js
minBy([{ n: 4 }, { n: 2 }, { n: 8 }, { n: 6 }], (x) => x.n); // 2
minBy([{ n: 4 }, { n: 2 }, { n: 8 }, { n: 6 }], "n"); // 2
```
