# Introdução

Neste laboratório, exploraremos a função `minN()` em JavaScript, que retorna os `n` elementos mínimos de um array. Aprenderemos como usar os métodos `Array.prototype.sort()` e `Array.prototype.slice()` para criar um clone raso (shallow clone) do array, ordená-lo em ordem crescente e obter o número especificado de elementos. Ao final deste laboratório, você terá uma melhor compreensão de como manipular arrays em JavaScript usando esses métodos.
