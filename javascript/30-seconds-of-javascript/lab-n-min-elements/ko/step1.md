# 배열에서 N 개의 최소 요소 반환 함수

코딩 연습을 위해 터미널/SSH 를 열고 `node`를 입력하세요. `minN` 함수를 사용하여 배열에서 `n`개의 최소 요소를 반환합니다.

함수 사용법은 다음과 같습니다.

- `Array.prototype.sort()`와 스프레드 연산자 (`...`) 를 사용하여 배열의 얕은 복사본을 생성하고 오름차순으로 정렬합니다.
- `Array.prototype.slice()`를 사용하여 지정된 수의 요소를 가져옵니다.
- 두 번째 인수인 `n`을 생략하면 함수는 단일 요소 배열을 반환합니다.
- `n`이 제공된 배열의 길이보다 크거나 같으면 함수는 오름차순으로 정렬된 원래 배열을 반환합니다.

```js
const minN = (arr, n = 1) => [...arr].sort((a, b) => a - b).slice(0, n);
```

다음은 몇 가지 예시입니다.

```js
minN([1, 2, 3]); // [1]
minN([1, 2, 3], 2); // [1, 2]
```
