# Introduction

Dans ce laboratoire, nous allons explorer le concept d'enlever des éléments d'un tableau jusqu'à ce qu'une certaine condition soit remplie. Nous utiliserons la fonction `takeUntil`, qui parcourt un tableau et enlève des éléments jusqu'à ce qu'une condition spécifiée soit vraie. Grâce à ce laboratoire, vous comprendrez mieux comment manipuler les tableaux en JavaScript.
