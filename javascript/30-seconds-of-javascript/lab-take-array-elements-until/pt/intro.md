# Introdução

Neste laboratório, exploraremos o conceito de remoção de elementos de um array até que uma determinada condição seja atendida. Usaremos a função `takeUntil`, que itera sobre um array e remove elementos até que uma condição especificada seja verdadeira. Através deste laboratório, você obterá uma compreensão mais profunda de como manipular arrays em JavaScript.
