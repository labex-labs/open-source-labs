# Introducción

En este laboratorio, exploraremos el concepto de eliminar elementos de un arreglo hasta que se cumpla una cierta condición. Utilizaremos la función `takeUntil`, que recorre un arreglo y elimina elementos hasta que una condición especificada sea verdadera. A través de este laboratorio, obtendrás una comprensión más profunda de cómo manipular arreglos en JavaScript.
