# Introdução

Neste laboratório, exploraremos o conceito de encontrar o último índice de inserção em um array ordenado. Aprenderemos como verificar se o array está ordenado em ordem decrescente e como usar `Array.prototype.reverse()` e `Array.prototype.findIndex()` para encontrar o último índice apropriado onde o elemento deve ser inserido. Ao final deste laboratório, você terá uma melhor compreensão de como manter a ordem de classificação de um array.
