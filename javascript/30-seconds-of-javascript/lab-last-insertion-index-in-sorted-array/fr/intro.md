# Introduction

Dans ce laboratoire, nous allons explorer le concept de recherche de l'index d'insertion dernier dans un tableau trié. Nous allons apprendre à vérifier si le tableau est trié dans l'ordre décroissant et à utiliser `Array.prototype.reverse()` et `Array.prototype.findIndex()` pour trouver l'index approprié où l'élément devrait être inséré. À la fin de ce laboratoire, vous aurez une meilleure compréhension de la manière de maintenir l'ordre de tri d'un tableau.
