# Инструкции по преобразованию Map в объект на JavaScript

Для преобразования JavaScript `Map` в объект следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте метод `Map.prototype.entries()`, чтобы преобразовать `Map` в массив пар ключ-значение.
3. Используйте метод `Object.fromEntries()`, чтобы преобразовать массив в объект.

Ниже представлен пример кода для преобразования `Map` в объект:

```js
const mapToObject = (map) => Object.fromEntries(map.entries());
```

Для тестирования функции вы можете выполнить:

```js
mapToObject(
  new Map([
    ["a", 1],
    ["b", 2]
  ])
); // {a: 1, b: 2}
```
