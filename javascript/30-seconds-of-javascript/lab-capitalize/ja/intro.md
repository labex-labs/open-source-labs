# はじめに

この実験では、文字列の先頭の文字を大文字に変換する `capitalize` と呼ばれる JavaScript 関数を調べます。この関数は、配列の分解構文と `String.prototype.toUpperCase()` を使って、望ましい結果を達成します。必要に応じて文字列の残りを小文字に変換するために `lowerRest` 引数をどのように使うかも見てみましょう。
