# Introdução

Neste laboratório, exploraremos o conceito de verificar se um determinado valor é um objeto simples (plain object) ou não, usando JavaScript. Usaremos a função `isPlainObject()` para verificar se o valor fornecido é um objeto criado pelo construtor `Object`. Ao final deste laboratório, você terá uma melhor compreensão de como determinar se um objeto é simples ou não em JavaScript.
