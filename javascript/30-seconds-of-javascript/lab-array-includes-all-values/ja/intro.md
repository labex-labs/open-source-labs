# はじめに

この実験では、与えられた配列のすべての要素が別の配列に含まれているかどうかをチェックする JavaScript のプログラミング演習を探ります。この演習を通じて、`Array.prototype.every()` と `Array.prototype.includes()` を使ってこの問題を効率的に解決する方法を学びます。この実験は、JavaScript の配列と配列メソッドに対する理解を深めるのに役立ちます。
