# Introdução

Neste laboratório, exploraremos como ordenar um array de objetos com base em uma propriedade específica e em um array de ordem personalizado. Usaremos uma combinação de `Array.prototype.reduce()` e `Array.prototype.sort()` para criar um novo array que é ordenado com base na ordem fornecida no array `order`. Este laboratório é uma ótima oportunidade para aprender a manipular arrays de objetos em JavaScript.
