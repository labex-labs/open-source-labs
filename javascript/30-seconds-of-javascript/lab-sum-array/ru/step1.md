# Вот как найти сумму массива

Для нахождения суммы массива чисел следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать программирование.
2. Используйте метод `Array.prototype.reduce()`, чтобы добавить каждое значение к накопителю, который должен быть инициализирован значением `0`.
3. Вот код, который вы можете использовать для нахождения суммы массива:

```js
const sum = (...arr) => [...arr].reduce((acc, val) => acc + val, 0);
```

4. Чтобы протестировать функцию `sum`, используйте следующие примеры кода:

```js
sum(1, 2, 3, 4); // 10
sum(...[1, 2, 3, 4]); // 10
```

Следуя этим шагам, вы можете легко найти сумму массива чисел с использованием JavaScript.
