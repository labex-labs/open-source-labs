# Введение

В этом практическом занятии мы изучим применение метода `Array.prototype.reduce()` для вычисления суммы двух и более чисел/массивов. Мы узнаем, как инициализировать накопитель значением `0` и как использовать оператор расширения для передачи массива в качестве отдельных аргументов. В конце этого практического занятия вы будете лучше понимать, как использовать метод `reduce()` в JavaScript.
