# Введение

В этом практическом занятии мы узнаем, как генерировать UUID в Node.js. UUID - это уникальные идентификаторы, которые широко используются в распределенных системах для уникального идентификации сущностей без необходимости централизованной координации. Мы будем использовать модуль `crypto` в Node.js для генерации UUID, соответствующего стандарту RFC4122 версии 4.
