# Как найти минимальную дату в JavaScript

Чтобы найти минимальное значение даты в JavaScript, можно использовать расширение ES6 с `Math.min()` и конструктором `Date`. Вот пример кода:

```js
const minDate = (...dates) => new Date(Math.min(...dates));
```

Для использования этой функции создайте массив объектов `Date` и передайте его в `minDate()` с использованием расширения. Вот пример:

```js
const dates = [
  new Date(2017, 4, 13),
  new Date(2018, 2, 12),
  new Date(2016, 0, 10),
  new Date(2016, 0, 9)
];
minDate(...dates); // Возвращает объект `Date`, представляющий 2016-01-08T22:00:00.000Z
```

С использованием этого кода вы можете легко найти минимальное значение даты в JavaScript.
