# Introdução

Neste laboratório, exploraremos os fundamentos da programação JavaScript. Através de uma série de exercícios e desafios, abordaremos tópicos como variáveis, tipos de dados, funções, fluxo de controle e muito mais. Ao final deste laboratório, você terá uma sólida compreensão da sintaxe JavaScript e será capaz de escrever programas básicos nesta linguagem de programação versátil.
