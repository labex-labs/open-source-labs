# Introdução

Neste laboratório, exploraremos o conceito de Funções Juxtapostas (Juxtapose Functions) em JavaScript. Aprenderemos como criar uma função que recebe múltiplas funções como argumentos e retorna uma nova função que aplica essas funções ao mesmo conjunto de argumentos, retornando um array com os resultados. Ao final deste laboratório, você terá uma melhor compreensão de como usar Funções Juxtapostas para simplificar seu código e torná-lo mais eficiente.
