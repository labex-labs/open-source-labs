# Как вычислить логарифм в конкретном основании

Для вычисления логарифма заданного числа в конкретном основании следуйте этим шагам:

1. Откройте Терминал или SSH.
2. Введите `node`, чтобы начать практиковаться в написании кода.
3. Используйте следующий код для вычисления логарифма:

```js
const logBase = (n, base) => Math.log(n) / Math.log(base);
```

4. Замените `n` на заданное число, а `base` на конкретное основание.
5. Запустите код, чтобы получить результат.

Вот несколько примеров:

```js
logBase(10, 10); // 1
logBase(100, 10); // 2
```
