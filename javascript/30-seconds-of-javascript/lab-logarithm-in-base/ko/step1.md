# 특정 밑의 로그 계산 방법

주어진 숫자의 특정 밑에 대한 로그를 계산하려면 다음 단계를 따르세요.

1. 터미널 또는 SSH 를 엽니다.
2. 코딩 연습을 시작하려면 `node`를 입력합니다.
3. 다음 코드를 사용하여 로그를 계산합니다.

```js
const logBase = (n, base) => Math.log(n) / Math.log(base);
```

4. `n`을 주어진 숫자로, `base`를 특정 밑으로 바꿉니다.
5. 코드를 실행하여 결과를 얻습니다.

다음은 몇 가지 예시입니다.

```js
logBase(10, 10); // 1
logBase(100, 10); // 2
```
