# Einführung

In diesem Lab werden wir das Konzept der Logarithmen in bestimmten Basen in JavaScript erkunden. Wir werden lernen, wie man den Logarithmus einer gegebenen Zahl in einer bestimmten Basis mithilfe der `Math.log()`-Funktion und einiger grundlegender mathematischer Operationen berechnet. Durch dieses Lab werden wir ein besseres Verständnis von logarithmischen Funktionen und wie man sie in unseren JavaScript-Programmen implementiert, erlangen.
