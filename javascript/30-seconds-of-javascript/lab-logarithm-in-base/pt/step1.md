# Como Calcular o Logaritmo em uma Base Específica

Para calcular o logaritmo de um número dado em uma base específica, siga estes passos:

1.  Abra o Terminal ou SSH.
2.  Digite `node` para começar a praticar a codificação.
3.  Use o seguinte código para calcular o logaritmo:

```js
const logBase = (n, base) => Math.log(n) / Math.log(base);
```

4.  Substitua `n` pelo número dado e `base` pela base específica.
5.  Execute o código para obter o resultado.

Aqui estão alguns exemplos:

```js
logBase(10, 10); // 1
logBase(100, 10); // 2
```
