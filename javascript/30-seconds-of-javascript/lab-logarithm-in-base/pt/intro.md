# Introdução

Neste laboratório, exploraremos o conceito de logaritmos em bases específicas em JavaScript. Aprenderemos como calcular o logaritmo de um número dado em uma base específica usando a função `Math.log()` e algumas operações matemáticas básicas. Através deste laboratório, obteremos uma melhor compreensão das funções logarítmicas e como implementá-las em nossos programas JavaScript.
