# Cómo calcular el logaritmo en una base específica

Para calcular el logaritmo de un número dado en una base específica, siga estos pasos:

1. Abra la Terminal o SSH.
2. Escriba `node` para comenzar a practicar la codificación.
3. Utilice el siguiente código para calcular el logaritmo:

```js
const logBase = (n, base) => Math.log(n) / Math.log(base);
```

4. Reemplace `n` con el número dado y `base` con la base específica.
5. Ejecute el código para obtener el resultado.

Aquí hay algunos ejemplos:

```js
logBase(10, 10); // 1
logBase(100, 10); // 2
```
