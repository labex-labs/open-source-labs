# Introducción

En este laboratorio, exploraremos el concepto de logaritmos en bases específicas en JavaScript. Aprenderemos cómo calcular el logaritmo de un número dado en una base específica utilizando la función `Math.log()` y algunas operaciones matemáticas básicas. A través de este laboratorio, obtendremos una mejor comprensión de las funciones logarítmicas y cómo implementarlas en nuestros programas de JavaScript.
