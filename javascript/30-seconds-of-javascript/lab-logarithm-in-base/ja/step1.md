# 特定の基数の対数を計算する方法

特定の基数の与えられた数の対数を計算するには、次の手順に従います。

1. ターミナルまたは SSH を開きます。
2. コーディングの練習を始めるために `node` を入力します。
3. 対数を計算するには、次のコードを使用します。

```js
const logBase = (n, base) => Math.log(n) / Math.log(base);
```

4. `n` を与えられた数に置き換え、`base` を特定の基数に置き換えます。
5. コードを実行して結果を取得します。

以下はいくつかの例です。

```js
logBase(10, 10); // 1
logBase(100, 10); // 2
```
