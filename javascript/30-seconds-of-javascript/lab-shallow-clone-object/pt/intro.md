# Introdução

Neste laboratório, exploraremos o conceito de _shallow cloning_ (clonagem rasa) em JavaScript. A clonagem rasa cria um novo objeto com todas as propriedades do objeto original, mas as próprias propriedades não são clonadas. Em vez disso, elas são copiadas por referência, o que significa que quaisquer alterações feitas nas propriedades do objeto original também serão refletidas no objeto clonado. Através deste laboratório, entenderemos como criar clones rasos de objetos usando o método `Object.assign()` em JavaScript.
