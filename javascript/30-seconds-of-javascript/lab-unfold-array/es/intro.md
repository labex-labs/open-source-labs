# Introducción

En este laboratorio, exploraremos la función `unfold()` en JavaScript. Esta función nos permite crear un array llamando repetidamente a una función iteradora con un valor semilla inicial hasta que la función devuelva `false`. A través de este laboratorio, aprenderemos cómo usar la función `unfold()` y cómo se puede utilizar para simplificar ciertas tareas de programación.
