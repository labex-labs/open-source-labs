# Einführung

In diesem Lab werden wir die grundlegenden Konzepte der JavaScript-Programmierung erkunden. Durch eine Reihe von praktischen Übungen lernen Sie, wie Sie die grundlegende Syntax schreiben, Variablen manipulieren und Kontrollflussstrukturen nutzen. Am Ende des Labs werden Sie einen soliden Grundlagenwissen in der JavaScript-Programmierung haben und bereit sein, fortgeschrittene Themen anzugehen.
