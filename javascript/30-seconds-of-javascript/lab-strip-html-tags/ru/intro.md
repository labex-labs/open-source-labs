# Введение

В этом практическом занятии мы будем изучать основные концепции программирования на JavaScript. С помощью серии практических упражнений вы научитесь писать базовый синтаксис, манипулировать переменными и использовать конструкции управления потоком. К концу практического занятия у вас будет твердый фундамент в программировании на JavaScript и вы будете готовы приступать к изучению более продвинутых тем.
