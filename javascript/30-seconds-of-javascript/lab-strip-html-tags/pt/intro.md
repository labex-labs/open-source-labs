# Introdução

Neste laboratório, exploraremos os conceitos fundamentais da programação JavaScript. Através de uma série de exercícios práticos, você aprenderá a escrever a sintaxe básica, manipular variáveis e utilizar estruturas de controle de fluxo. Ao final do laboratório, você terá uma base sólida em programação JavaScript e estará pronto para abordar tópicos mais avançados.
