# Введение

В этом практическом занятии мы будем углубляться в концепцию проверки, является ли заданное значение null, с использованием JavaScript. Мы будем изучать применение строгого оператора равенства для определения, равно ли значение заданной переменной null. В конце этого практического занятия вы будете иметь лучшее понимание того, как написать функцию, которая проверяет значения на null в JavaScript.
