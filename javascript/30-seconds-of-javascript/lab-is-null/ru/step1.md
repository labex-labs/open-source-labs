# Проверка, является ли значение null

Для проверки, является ли значение `null` в JavaScript, используйте строгий оператор равенства (`===`). Вот пример кода, который определяет функцию под названием `isNull`, которая возвращает `true`, если заданное значение равно `null`, и `false` в противном случае.

```js
const isNull = (val) => val === null;
```

Для тестирования этой функции вы можете вызвать ее с значением, которое вы хотите проверить, в качестве аргумента. Например, `isNull(null)` вернет `true`.
