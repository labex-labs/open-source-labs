# Comprobando si un valor es nulo

Para comprobar si un valor es `null` en JavaScript, utiliza el operador de igualdad estricta (`===`). Aquí hay un fragmento de código de ejemplo que define una función llamada `isNull` que devuelve `true` si el valor dado es `null` y `false` en caso contrario.

```js
const isNull = (val) => val === null;
```

Para probar esta función, puedes llamarla con el valor que quieres comprobar como argumento. Por ejemplo, `isNull(null)` devolverá `true`.
