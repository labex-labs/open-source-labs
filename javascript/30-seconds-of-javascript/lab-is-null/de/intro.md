# Einführung

In diesem Lab werden wir uns mit dem Konzept beschäftigen, ob ein bestimmter Wert in JavaScript null ist. Wir werden die Verwendung des strikten Gleichheitsoperators untersuchen, um zu bestimmen, ob der Wert einer gegebenen Variable null ist. Am Ende dieses Labs werden Sie einen besseren Überblick darüber haben, wie Sie eine Funktion schreiben, die in JavaScript nach null-Werten sucht.
