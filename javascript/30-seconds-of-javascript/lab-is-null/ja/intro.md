# はじめに

この実験では、JavaScript を使って指定された値が null かどうかを確認する概念を深く掘り下げます。与えられた変数の値が null と等しいかどうかを判断するために、厳密な等価演算子の使い方を探ります。この実験が終わるとき、JavaScript で null 値をチェックする関数を書く方法をより深く理解しているでしょう。
