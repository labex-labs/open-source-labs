# Introduction

Dans ce laboratoire, nous allons approfondir le concept de vérification si une valeur spécifiée est nulle en utilisant JavaScript. Nous allons explorer l'utilisation de l'opérateur d'égalité stricte pour déterminer si la valeur d'une variable donnée est égale à null. À la fin de ce laboratoire, vous aurez une meilleure compréhension de la manière d'écrire une fonction qui vérifie les valeurs nulles en JavaScript.
