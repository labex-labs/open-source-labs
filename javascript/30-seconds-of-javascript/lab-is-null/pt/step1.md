# Verificando se um Valor é Nulo

Para verificar se um valor é `null` em JavaScript, use o operador de igualdade estrita (`===`). Aqui está um trecho de código de exemplo que define uma função chamada `isNull` que retorna `true` se o valor fornecido for `null` e `false` caso contrário.

```js
const isNull = (val) => val === null;
```

Para testar esta função, você pode chamá-la com o valor que deseja verificar como seu argumento. Por exemplo, `isNull(null)` retornará `true`.
