# Introdução

Neste laboratório, vamos aprofundar o conceito de verificação se um valor especificado é nulo usando JavaScript. Exploraremos o uso do operador de igualdade estrita para determinar se o valor de uma determinada variável é igual a `null`. Ao final deste laboratório, você terá uma melhor compreensão de como escrever uma função que verifica valores nulos em JavaScript.
