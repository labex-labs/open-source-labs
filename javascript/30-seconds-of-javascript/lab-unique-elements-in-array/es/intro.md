# Introducción

En este laboratorio, profundizaremos en el concepto de arrays en JavaScript y aprenderemos cómo encontrar valores únicos en un array utilizando el objeto `Set` y el operador de propagación. Exploraremos un enfoque simple y eficiente para eliminar duplicados de un array y devolver solo los elementos únicos. Este laboratorio es perfecto para aquellos que deseen mejorar su comprensión de los arrays y la manipulación de datos en JavaScript.
