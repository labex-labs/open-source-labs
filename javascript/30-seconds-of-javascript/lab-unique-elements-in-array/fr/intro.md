# Introduction

Dans ce laboratoire, nous allons plonger dans le concept d'arrays en JavaScript et apprendre à trouver les valeurs uniques dans un tableau en utilisant l'objet `Set` et l'opérateur de répandage. Nous allons explorer une approche simple et efficace pour supprimer les doublons d'un tableau et ne renvoyer que les éléments uniques. Ce laboratoire est parfait pour ceux qui souhaitent améliorer leur compréhension des tableaux et de la manipulation de données en JavaScript.
