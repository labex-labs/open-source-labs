# Как найти уникальные значения в массиве с помощью JavaScript

Чтобы найти все уникальные значения в массиве, вы можете выполнить следующие шаги в JavaScript:

1. Создайте `Set` из заданного массива, чтобы удалить дублированные значения.
2. Используйте оператор расширения (`...`), чтобы преобразовать `Set` обратно в массив.

Вот пример кода:

```js
const getUniqueValues = (arr) => [...new Set(arr)];
```

Вы можете вызвать функцию и передать ей массив, как это:

```js
getUniqueValues([1, 2, 2, 3, 4, 4, 5]); // [1, 2, 3, 4, 5]
```

Это вернет массив со всеми уникальными значениями из исходного массива без каких-либо дубликатов.
