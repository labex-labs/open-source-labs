# JavaScript を使って配列内の一意の値を見つける方法

配列内のすべての一意の値を見つけるには、JavaScript で次の手順を実行できます。

1. 与えられた配列から `Set` を作成して、重複する値を除外します。
2. スプレッド演算子 (`...`) を使って、`Set` を再び配列に変換します。

以下はコードの例です。

```js
const getUniqueValues = (arr) => [...new Set(arr)];
```

この関数を呼び出して配列を渡すことができます。

```js
getUniqueValues([1, 2, 2, 3, 4, 4, 5]); // [1, 2, 3, 4, 5]
```

これにより、元の配列のすべての一意の値が含まれた配列が返され、重複はありません。
