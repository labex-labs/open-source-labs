# JavaScript 로 배열에서 고유한 값 찾기

배열에서 모든 고유한 값을 찾으려면 JavaScript 에서 다음 단계를 따를 수 있습니다.

1. 주어진 배열에서 중복된 값을 제거하기 위해 `Set`을 생성합니다.
2. spread operator (`...`) 를 사용하여 `Set`을 다시 배열로 변환합니다.

다음은 예시 코드 조각입니다.

```js
const getUniqueValues = (arr) => [...new Set(arr)];
```

다음과 같이 함수를 호출하고 배열을 전달할 수 있습니다.

```js
getUniqueValues([1, 2, 2, 3, 4, 4, 5]); // [1, 2, 3, 4, 5]
```

이렇게 하면 중복 항목 없이 원래 배열의 모든 고유한 값을 가진 배열이 반환됩니다.
