# Introdução

Neste laboratório, mergulharemos no conceito de arrays em JavaScript e aprenderemos como encontrar valores únicos em um array usando o objeto `Set` e o operador spread. Exploraremos uma abordagem simples e eficiente para remover duplicatas de um array e retornar apenas os elementos únicos. Este laboratório é perfeito para aqueles que buscam aprimorar sua compreensão de arrays e manipulação de dados em JavaScript.
