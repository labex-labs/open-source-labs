# Introdução

Neste laboratório, exploraremos a função `transform` em JavaScript, que nos permite aplicar uma função especificada em um acumulador e em cada chave de um objeto. Ao usar `Object.keys()` e `Array.prototype.reduce()`, podemos facilmente iterar sobre cada chave em um objeto e realizar a transformação desejada. Este laboratório fornecerá experiência prática com esta função útil e demonstrará suas aplicações práticas na programação JavaScript.
