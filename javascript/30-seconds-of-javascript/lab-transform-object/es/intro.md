# Introducción

En este laboratorio, exploraremos la función `transform` en JavaScript que nos permite aplicar una función especificada contra un acumulador y cada clave en un objeto. Al usar `Object.keys()` y `Array.prototype.reduce()`, podemos iterar fácilmente sobre cada clave en un objeto y realizar la transformación deseada. Este laboratorio proporcionará experiencia práctica con esta función útil y demostrará sus aplicaciones prácticas en la programación de JavaScript.
