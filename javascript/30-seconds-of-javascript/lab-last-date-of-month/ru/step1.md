# Функция для возврата последней даты месяца

Для начала работы с кодом откройте Терминал/SSH и введите `node`.

Эта функция возвращает последнюю дату месяца для заданной даты.

Для этого следуйте шагам:

1. Используйте `Date.prototype.getFullYear()` и `Date.prototype.getMonth()`, чтобы получить текущий год и месяц из заданной даты.
2. Создайте новую дату с заданным годом и месяцем, увеличенным на `1`, и днем, установленным в `0` (последний день предыдущего месяца). Для этого можно использовать конструктор `Date`.
3. Если в функцию не передается аргумент, по умолчанию она будет использовать текущую дату.
4. Возвращаем последнюю дату месяца в формате строкового представления даты.

Вот код функции:

```js
const getLastDateOfMonth = (date = new Date()) => {
  let lastDate = new Date(date.getFullYear(), date.getMonth() + 1, 0);
  return lastDate.toISOString().split("T")[0];
};
```

Вы можете протестировать функцию, вызвав ее с объектом даты, например, так:

```js
getLastDateOfMonth(new Date("2015-08-11")); // '2015-08-30'
```
