# Проверка, является ли значение числом в JavaScript

Чтобы проверить, является ли значение числом в JavaScript, вы можете использовать оператор `typeof`, чтобы определить, классифицируется ли значение как примитивный тип `number`. Чтобы избежать проблем с `NaN`, которое имеет `typeof`, равный `number`, и не равно самому себе, вы также можете проверить, равно ли значение самому себе, используя `val === val`.

Вот пример функции, которая проверяет, является ли заданное значение числом:

```js
const isNumber = (val) => typeof val === "number" && val === val;
```

Вы можете использовать эту функцию так:

```js
isNumber(1); // true
isNumber("1"); // false
isNumber(NaN); // false
```
