# Введение

В этом практическом занятии мы сосредоточимся на создании JavaScript-функции, которая проверяет, является ли заданное значение числом. Мы будем использовать оператор `typeof` и защиту от `NaN`, чтобы убедиться, что функция возвращает `true` только для допустимых чисел. Это практическое занятие поможет вам лучше понять типы данных JavaScript и проверку типов.
