# Получение названия дня недели из объекта даты

Для получения названия дня недели из объекта `Date` следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте `Date.prototype.toLocaleDateString()` с параметром `{ weekday: 'long' }`, чтобы получить день недели.
3. Вы можете использовать необязательный второй аргумент, чтобы получить название на определенном языке, или опустить его, чтобы использовать стандартную локаль.

Вот пример реализации:

```js
const dayName = (date, locale) =>
  date.toLocaleDateString(locale, { weekday: "long" });
```

Вы можете использовать эту функцию так:

```js
dayName(new Date()); // 'Saturday'
dayName(new Date("09/23/2020"), "de-DE"); // 'Samstag'
```
