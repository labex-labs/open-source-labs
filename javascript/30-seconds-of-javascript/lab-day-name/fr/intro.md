# Introduction

Dans ce laboratoire, nous allons explorer comment récupérer le nom du jour de la semaine à partir d'un objet de date donné en utilisant JavaScript. Nous utiliserons la méthode `Date.prototype.toLocaleDateString()` avec l'option `{ weekday: 'long' }` pour obtenir le nom du jour de la semaine. De plus, nous pouvons spécifier un nom spécifique à une langue en utilisant le deuxième argument optionnel ou utiliser la locale par défaut.
