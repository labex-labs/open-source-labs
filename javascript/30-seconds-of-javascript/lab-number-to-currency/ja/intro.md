# はじめに

この実験では、`Intl.NumberFormat` メソッドを使用して数値を通貨形式の文字列にフォーマットする方法を学びます。このメソッドを使うことで、国や言語形式に応じた通貨形式に数値をフォーマットすることができます。さまざまな通貨や言語形式でこのメソッドを使って、さまざまな形式の通貨文字列を表示する方法を検討します。
