# Функция для вычисления разницы между датами в минутах

Для вычисления разницы (в минутах) между двумя датами используйте следующую функцию:

```js
const getMinutesDiffBetweenDates = (dateInitial, dateFinal) =>
  (dateFinal - dateInitial) / (1000 * 60);
```

Просто вычтите два объекта `Date` и разделите на количество миллисекунд в минуте, чтобы получить разницу (в минутах) между ними.

Вот пример использования этой функции:

```js
getMinutesDiffBetweenDates(
  new Date("2021-04-24 01:00:15"),
  new Date("2021-04-24 02:00:15")
); // 60
```

Для начала практики программирования откройте Терминал/SSH и введите `node`.
