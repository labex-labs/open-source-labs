# Introduction

Dans ce laboratoire, nous allons explorer une fonction JavaScript qui convertit une chaîne de caractères de couleur `rgb()` en un objet avec les valeurs de chaque couleur. Cette fonction utilise une combinaison de manipulation de chaînes de caractères et de décomposition d'objets pour analyser les valeurs numériques à partir de la chaîne d'entrée et les stocker dans un nouvel objet. Ce laboratoire vous aidera à comprendre comment travailler avec les chaînes de caractères, les tableaux et les objets en JavaScript.
