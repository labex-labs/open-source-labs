# Introdução

Neste laboratório, exploraremos uma função JavaScript que converte uma string de cor `rgb()` em um objeto com os valores de cada cor. Esta função utiliza uma combinação de manipulação de strings e destruturação de arrays para analisar os valores numéricos da string de entrada e armazená-los em um novo objeto. Este laboratório ajudará você a entender como trabalhar com strings, arrays e objetos em JavaScript.
