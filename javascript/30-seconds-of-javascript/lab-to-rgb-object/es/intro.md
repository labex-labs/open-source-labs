# Introducción

En este laboratorio, exploraremos una función de JavaScript que convierte una cadena de color `rgb()` en un objeto con los valores de cada color. Esta función utiliza una combinación de manipulación de cadenas y desestructuración de arrays para analizar los valores numéricos de la cadena de entrada y almacenarlos en un nuevo objeto. Este laboratorio te ayudará a entender cómo trabajar con cadenas, arrays y objetos en JavaScript.
