# Введение

В этом практическом занятии мы изучим функцию JavaScript, которая преобразует строку цвета `rgb()` в объект с значениями каждого цвета. Эта функция использует комбинацию манипуляций со строками и деструктуризации массивов для разбора числовых значений из входной строки и сохранения их в новом объекте. Это практическое занятие поможет вам понять, как работать со строками, массивами и объектами в JavaScript.
