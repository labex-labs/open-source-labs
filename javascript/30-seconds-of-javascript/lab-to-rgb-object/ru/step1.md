# Преобразование RGB в объект

Для преобразования строки цвета `rgb()` в объект с значениями каждого цвета следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте `String.prototype.match()`, чтобы получить массив из 3 строк с числовыми значениями.
3. Используйте `Array.prototype.map()` в сочетании с `Number`, чтобы преобразовать их в массив числовых значений.
4. Используйте деструктуризацию массива, чтобы сохранить значения в именованные переменные и создать из них соответствующий объект.

Вот код, который вы можете использовать:

```js
const toRGBObject = (rgbStr) => {
  const [red, green, blue] = rgbStr.match(/\d+/g).map(Number);
  return { red, green, blue };
};

toRGBObject("rgb(255, 12, 0)"); // {red: 255, green: 12, blue: 0}
```
