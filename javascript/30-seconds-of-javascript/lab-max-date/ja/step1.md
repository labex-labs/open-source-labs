# 最大日付の検索

与えられた日付の配列から最大の日付値を見つけるには、次の手順に従います。

1. ターミナルまたはSSHを開きます。
2. コーディングの練習を始めるには `node` と入力します。
3. `Math.max()` とES6のスプレッド構文を使って最大の日付値を見つけます。
4. `Date` コンストラクタを使って最大の日付値を `Date` オブジェクトに変換します。

以下はコードの例です。

```js
const maxDate = (...dates) => new Date(Math.max(...dates));

const dates = [
  new Date(2017, 4, 13),
  new Date(2018, 2, 12),
  new Date(2016, 0, 10),
  new Date(2016, 0, 9)
];

maxDate(...dates); // Returns "2018-03-11T22:00:00.000Z"
```

これらの手順に従って提供されたコードを使えば、与えられた日付の配列から最大の日付値を簡単に見つけることができます。
