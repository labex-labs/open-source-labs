# Генерация случайного шестнадцатеричного кода цвета в Терминале/SSH

Для генерации случайного шестнадцатеричного кода цвета в Терминале/SSH выполните следующие шаги:

1. Откройте Терминал/SSH.
2. Введите `node`.
3. Используйте следующий код для генерации случайного 24-разрядного (6 \* 4 разряда) шестнадцатеричного числа:

```js
const randomHexColorCode = () => {
  let n = (Math.random() * 0xfffff * 1000000).toString(16);
  return "#" + n.slice(0, 6);
};
```

4. Чтобы сгенерировать случайный шестнадцатеричный код цвета, вызовите функцию `randomHexColorCode()`.

Пример:

```js
randomHexColorCode(); // '#e34155'
```

Это сгенерирует случайный шестнадцатеричный код цвета, который вы можете использовать в своих проектах по программированию.
