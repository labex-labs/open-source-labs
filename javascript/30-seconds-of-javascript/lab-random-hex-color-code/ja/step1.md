# ターミナル/SSH でランダムな 16 進数カラーコードを生成する

ターミナル/SSH でランダムな 16 進数カラーコードを生成するには、以下の手順に従ってください。

1. ターミナル/SSH を開きます。
2. `node` と入力します。
3. 次のコードを使用して、ランダムな 24 ビット (6 \* 4 ビット) の 16 進数を生成します。

```js
const randomHexColorCode = () => {
  let n = (Math.random() * 0xfffff * 1000000).toString(16);
  return "#" + n.slice(0, 6);
};
```

4. ランダムな 16 進数カラーコードを生成するには、`randomHexColorCode()` 関数を呼び出します。

例:

```js
randomHexColorCode(); // '#e34155'
```

これにより、コーディング プロジェクトで使用できるランダムな 16 進数カラーコードが生成されます。
