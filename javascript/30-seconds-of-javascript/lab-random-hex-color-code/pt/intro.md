# Introdução

Neste laboratório, exploraremos os fundamentos da linguagem de programação JavaScript. O objetivo deste laboratório é fornecer uma experiência prática para iniciantes em JavaScript, a fim de aprender e compreender os fundamentos de conceitos de programação, como variáveis, tipos de dados, operadores e instruções de controle. Ao final deste laboratório, você terá uma base sólida em programação JavaScript e será capaz de criar programas simples por conta própria.
