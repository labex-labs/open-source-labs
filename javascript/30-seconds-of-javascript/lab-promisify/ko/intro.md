이 랩에서는 `promisify` 함수를 사용하여 비동기 함수를 프로미스 (Promise) 를 반환하도록 변환하는 방법을 배웁니다. 이를 통해 콜백 함수를 피함으로써 더 깔끔하고 읽기 쉬운 코드를 작성할 수 있습니다. 또한 JavaScript 에서 커링 (currying) 과 나머지 연산자 (rest operator) 의 사용법도 살펴봅니다.
