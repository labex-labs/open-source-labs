# Introdução

Neste laboratório, aprenderemos como converter uma função assíncrona para retornar uma _promise_ usando a função `promisify`. Isso nos permitirá escrever um código mais limpo e legível, evitando funções de _callback_. Também exploraremos o uso de _currying_ e do operador _rest_ em JavaScript.
