# Introducción

En este laboratorio, exploraremos el concepto de funciones asíncronas en JavaScript. Aprenderemos a reconocer y diferenciar entre funciones regulares y funciones asíncronas utilizando el método `isAsyncFunction()`. Al final de este laboratorio, tendrás una comprensión clara de cómo trabajar con funciones asíncronas y cómo se pueden utilizar para mejorar el rendimiento de tu código.
