# Einführung

In diesem Lab werden wir das Konzept asynchroner Funktionen in JavaScript erkunden. Wir werden lernen, wie man reguläre Funktionen und asynchrone Funktionen mithilfe der Methode `isAsyncFunction()` erkennen und unterscheiden kann. Am Ende dieses Labs werden Sie einen klaren Überblick darüber haben, wie man mit asynchronen Funktionen umgeht und wie diese verwendet werden können, um die Leistung Ihres Codes zu verbessern.
