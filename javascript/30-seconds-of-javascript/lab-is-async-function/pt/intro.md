# Introdução

Neste laboratório, exploraremos o conceito de funções assíncronas em JavaScript. Aprenderemos a reconhecer e diferenciar entre funções regulares e funções assíncronas usando o método `isAsyncFunction()`. Ao final deste laboratório, você terá uma compreensão clara de como trabalhar com funções assíncronas e como elas podem ser usadas para melhorar o desempenho do seu código.
