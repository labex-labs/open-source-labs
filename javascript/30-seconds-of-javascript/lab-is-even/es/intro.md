# Introducción

En este laboratorio, profundizaremos en el mundo de la programación de JavaScript y aprenderemos a escribir una función para determinar si un número dado es par o impar. El laboratorio te guiará a través de los pasos para usar el operador módulo para comprobar si un número es divisible por 2, y luego devolver `true` si es par, o `false` si es impar. Al final de este laboratorio, tendrás una mejor comprensión de los conceptos básicos de programación en JavaScript y cómo escribir funciones para tareas simples.
