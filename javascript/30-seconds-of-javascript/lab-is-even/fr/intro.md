# Introduction

Dans ce laboratoire, nous allons plonger dans le monde de la programmation JavaScript et apprendre à écrire une fonction pour déterminer si un nombre donné est pair ou impair. Le laboratoire vous guidera tout au long des étapes d'utilisation de l'opérateur modulo pour vérifier si un nombre est divisible par 2, puis de renvoyer true s'il est pair, ou false s'il est impair. À la fin de ce laboratoire, vous aurez une meilleure compréhension des concepts de base de la programmation en JavaScript et de la manière d'écrire des fonctions pour des tâches simples.
