# 숫자가 짝수인지 확인하기

코딩 연습을 위해 터미널/SSH 를 열고 `node`를 입력하세요. 다음 코드를 사용하여 숫자가 짝수인지 홀수인지 확인합니다.

```js
const isEven = (num) => num % 2 === 0;
```

위 코드는 나머지 연산자 (modulo operator, `%`) 를 사용하여 숫자가 홀수인지 짝수인지 확인합니다. 숫자가 짝수인 경우 함수는 `true`를 반환합니다. 홀수인 경우 함수는 `false`를 반환합니다.

`isEven` 함수를 사용하는 예시는 다음과 같습니다.

```js
isEven(3); // returns false
```
