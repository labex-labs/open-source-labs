# Введение

В этом лабе мы углубимся в мир программирования на JavaScript и научимся писать функцию для определения, является ли заданное число четным или нечетным. Лабка проведет вас по шагам по использованию оператора модуля для проверки, делится ли число на 2, и затем возвращать true, если оно четное, или false, если оно нечетное. В конце этого лабара вы будете иметь лучшее понимание основных концепций программирования на JavaScript и о том, как писать функции для простых задач.
