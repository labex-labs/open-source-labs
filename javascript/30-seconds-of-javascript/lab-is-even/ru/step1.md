# Проверить, является ли число четным

Для практики программирования откройте Терминал/SSH и введите `node`. Используйте следующий код, чтобы проверить, является ли число четным или нечетным:

```js
const isEven = (num) => num % 2 === 0;
```

В приведенном выше коде используется оператор модуля (`%`), чтобы проверить, является ли число нечетным или четным. Если число четное, функция возвращает `true`. Если число нечетное, функция возвращает `false`.

Вот пример использования функции `isEven`:

```js
isEven(3); // возвращает false
```
