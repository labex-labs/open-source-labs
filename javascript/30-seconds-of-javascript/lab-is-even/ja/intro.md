# はじめに

この実験では、JavaScript プログラミングの世界に入り、与えられた数が偶数か奇数かを判断する関数を書く方法を学びます。この実験では、剰余演算子を使って数が 2 で割り切れるかどうかを確認し、偶数の場合は true を返し、奇数の場合は false を返す手順を案内します。この実験が終わるとき、JavaScript の基本的なプログラミング概念と、簡単なタスクのための関数の書き方をより深く理解しているでしょう。
