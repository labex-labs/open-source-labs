# Introdução

Neste laboratório, vamos mergulhar no mundo da programação JavaScript e aprender a escrever uma função para determinar se um número dado é par ou ímpar. O laboratório irá guiá-lo através dos passos de utilização do operador módulo para verificar se um número é divisível por 2, e então retornar `true` se for par, ou `false` se for ímpar. Ao final deste laboratório, você terá uma melhor compreensão dos conceitos básicos de programação em JavaScript e de como escrever funções para tarefas simples.
