# Verificar se um Número é Par

Para praticar a codificação, abra o Terminal/SSH e digite `node`. Use o seguinte código para verificar se um número é par ou ímpar:

```js
const isEven = (num) => num % 2 === 0;
```

O código acima utiliza o operador módulo (`%`) para verificar se um número é ímpar ou par. Se o número for par, a função retorna `true`. Se for ímpar, a função retorna `false`.

Aqui está um exemplo de como usar a função `isEven`:

```js
isEven(3); // returns false
```
