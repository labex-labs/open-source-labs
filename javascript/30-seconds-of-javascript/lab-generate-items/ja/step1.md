# コードの練習

コーディングの練習を始めるには、ターミナル/SSH を開き、`node`と入力します。その後、`generateItems`関数を使用して、特定の数の要素を持つ配列を生成できます。

- 生成したい要素の数と、要素を生成するために使用する関数を引数に`generateItems`を呼び出します。
- `generateItems`は`Array.from()`を使用して指定された長さの空の配列を作成し、新しく作成された各要素のインデックスを引数にして提供された関数を呼び出します。
- 提供された関数は 1 つの引数を持ちます。各要素のインデックスです。

```js
const generateItems = (n, fn) => Array.from({ length: n }, (_, i) => fn(i));
```

以下は、`generateItems`を使用して 10 個の乱数の配列を生成する例です：

```js
generateItems(10, Math.random);
// [0.21, 0.08, 0.40, 0.96, 0.96, 0.24, 0.19, 0.96, 0.42, 0.70]
```
