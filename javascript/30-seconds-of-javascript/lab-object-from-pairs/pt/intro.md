# Introdução

Neste laboratório, exploraremos como criar um objeto a partir de pares chave-valor em JavaScript. Usaremos o método `Array.prototype.reduce()` para combinar os pares em um único objeto. Adicionalmente, aprenderemos sobre o método `Object.fromEntries()`, que oferece funcionalidade semelhante. Ao final do laboratório, você terá uma melhor compreensão de como trabalhar com pares chave-valor em JavaScript.
