# Введение

В этом практическом занятии мы изучим, как создавать объект из пар ключ-значение в JavaScript. Мы будем использовать метод `Array.prototype.reduce()`, чтобы объединить пары в единый объект. Кроме того, мы узнаем о методе `Object.fromEntries()`, который предоставляет аналогичную функциональность. В конце практического занятия у вас будет лучше понимание, как работать с парами ключ-значение в JavaScript.
