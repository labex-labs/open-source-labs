# Введение

В этом практическом занятии (лабораторной работе) мы рассмотрим, как преобразовать все ключи объекта в верхний регистр на JavaScript. Вы узнаете, как использовать методы `Object.keys()` и `Array.prototype.reduce()` для создания нового объекта с всеми ключами, преобразованными в верхний регистр. Эта техника может быть полезна в различных сценариях, когда вам нужно стандартизировать ключи объекта для обеспечения согласованности и удобства использования.
