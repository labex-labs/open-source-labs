# Introdução

Neste laboratório, exploraremos a funcionalidade do Node.js e aprenderemos como determinar se nosso código está sendo executado em um ambiente Node.js. Ao usar o objeto global `process`, podemos verificar se nosso código está sendo executado dentro de um ambiente de tempo de execução Node.js e realizar ações específicas de acordo. Este laboratório fornecerá experiência prática com Node.js e aprofundará sua compreensão de suas capacidades.
