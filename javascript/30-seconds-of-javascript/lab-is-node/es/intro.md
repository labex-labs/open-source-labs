# Introducción

En este laboratorio, exploraremos la funcionalidad de Node.js y aprenderemos a determinar si nuestro código se está ejecutando dentro de un entorno de Node.js. Al utilizar el objeto global `process`, podemos comprobar si nuestro código se está ejecutando dentro de un entorno de tiempo de ejecución de Node.js y realizar acciones específicas en consecuencia. Este laboratorio proporcionará experiencia práctica con Node.js y profundizará su comprensión de sus capacidades.
