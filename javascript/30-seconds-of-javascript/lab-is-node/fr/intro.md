# Introduction

Dans ce laboratoire, nous allons explorer la fonctionnalité de Node.js et apprendre à déterminer si notre code est exécuté dans un environnement Node.js. En utilisant l'objet global `process`, nous pouvons vérifier si notre code est exécuté dans un environnement d'exécution Node.js et effectuer des actions spécifiques en conséquence. Ce laboratoire vous procurera une expérience pratique avec Node.js et approfondira votre compréhension de ses capacités.
