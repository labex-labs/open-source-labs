# Einführung

In diesem Lab werden wir die Funktionalität von Node.js erkunden und lernen, wie wir feststellen können, ob unser Code innerhalb einer Node.js-Umgebung ausgeführt wird. Mit dem globalen Objekt `process` können wir überprüfen, ob unser Code innerhalb einer Node.js-Laufzeitumgebung ausgeführt wird, und dementsprechend spezifische Aktionen ausführen. In diesem Lab erhalten Sie praktische Erfahrungen mit Node.js und vertiefen Ihr Verständnis seiner Möglichkeiten.
