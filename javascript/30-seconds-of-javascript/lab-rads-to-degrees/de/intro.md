# Einführung

In diesem Lab werden wir das Konzept des Umwandels von Winkeln von Radiant in Grad in JavaScript erkunden. Sie werden lernen, wie man die Formel für die Umrechnung von Radiant in Grad und die integrierte Konstante `Math.PI` verwendet, um Winkel umzurechnen. Durch praktische Übungen werden Sie praktische Erfahrungen mit diesem Umrechnungsprozess sammeln und Ihr Verständnis der Trigonometrie in JavaScript verbessern.
