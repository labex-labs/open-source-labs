# Преобразование радиан в градусы

Для преобразования угла из радиан в градусы следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте следующую формулу: `градусы = радианы * (180 / Math.PI)`
3. Замените `радианы` в формуле на значение, которое вы хотите преобразовать.
4. Результат будет в градусах.

Вот пример:

```js
const radsToDegrees = (rad) => (rad * 180.0) / Math.PI;
radsToDegrees(Math.PI / 2); // 90
```

Это преобразует `π/2` радианы в `90` градусов.
