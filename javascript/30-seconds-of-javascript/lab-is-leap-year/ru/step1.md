# Код для проверки високосного года

Для проверки того, является ли заданный `год` високосным, следуйте следующим шагам:

1. Откройте Терминал/SSH.
2. Введите `node`, чтобы начать программирование.
3. Используйте конструктор `Date` и установите дату на 29 февраля заданного `года`.
4. Проверьте, равно ли число месяца `1`, используя `Date.prototype.getMonth()`.
5. Используйте следующий фрагмент кода, чтобы проверить, является ли год високосным:

```js
const isLeapYear = (year) => new Date(year, 1, 29).getMonth() === 1;
```

Вот пример использования этого кода:

```js
isLeapYear(2019); // false
isLeapYear(2020); // true
```
