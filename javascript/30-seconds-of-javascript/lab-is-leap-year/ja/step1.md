# うるう年をチェックするコード

与えられた `year` がうるう年かどうかをチェックするには、次の手順に従います。

1. ターミナル/SSH を開きます。
2. コーディングを開始するには `node` と入力します。
3. `Date` コンストラクタを使用して、与えられた `year` の 2 月 29 日に日付を設定します。
4. `Date.prototype.getMonth()` を使用して、月が `1` に等しいかどうかを確認します。
5. 次のコード スニペットを使用して、年がうるう年かどうかを確認します。

```js
const isLeapYear = (year) => new Date(year, 1, 29).getMonth() === 1;
```

このコードの使用方法の例を次に示します。

```js
isLeapYear(2019); // false
isLeapYear(2020); // true
```
