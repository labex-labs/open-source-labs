# Introduction

Dans ce laboratoire, nous allons explorer les concepts fondamentaux de la programmation JavaScript. Au travers d'une série d'exercices, vous allez apprendre à écrire des programmes JavaScript de base, y compris les variables, les types de données, les fonctions et les structures de contrôle de flux. À la fin du laboratoire, vous aurez une solide base en programmation JavaScript et serez en mesure d'appliquer vos connaissances pour résoudre des problèmes du monde réel.
