# Проверка, является ли строка в формате ISO

Чтобы проверить, является ли заданная строка в упрощенном расширенном формате ISO (ISO 8601), следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в программировании.
2. Используйте конструктор `Date` для создания объекта `Date` из заданной строки.
3. Проверьте, является ли созданный объект даты допустимым, используя `Date.prototype.valueOf()` и `Number.isNaN()`.
4. Сравните представление даты в формате ISO в виде строки с исходной строкой, используя `Date.prototype.toISOString()`.
5. Если строки совпадают и дата допустима, верните `true`. В противном случае верните `false`.

Вот пример фрагмента кода:

```js
const isISOString = (val) => {
  const d = new Date(val);
  return !Number.isNaN(d.valueOf()) && d.toISOString() === val;
};

isISOString("2020-10-12T10:10:10.000Z"); // true
isISOString("2020-10-12"); // false
```

Эта функция вернет `true`, если строка находится в формате ISO, и `false` в противном случае.
