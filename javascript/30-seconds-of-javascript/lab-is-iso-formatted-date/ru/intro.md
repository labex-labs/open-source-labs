# Введение

В этом практическом занятии (lab) мы рассмотрим, как определить, является ли заданная строка допустимой строкой даты в упрощенном расширенном формате ISO (ISO 8601). Мы будем использовать конструктор `Date` и связанные с ним методы для создания объекта `Date` из строки и проверки его допустимости. К концу практического занятия вы лучше поймете, как работать с датами в JavaScript и как проверять их допустимость с использованием формата ISO.
