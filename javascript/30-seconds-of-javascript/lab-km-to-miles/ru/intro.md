# Введение

В этом практическом занятии мы изучим программирование на JavaScript и научимся писать код, который переводит километры в мили. Практическое занятие будет посвящено изучению формулы перевода и методам ее реализации в коде на JavaScript. В конце практического занятия вы будете лучше понимать, как использовать JavaScript для выполнения вычислений и решения задач.
