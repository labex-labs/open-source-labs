# Перевод километров в мили

Для перевода километров в мили с использованием кода следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте формулу перевода `mi = km * 0.621371`.
3. Напишите следующий фрагмент кода:

```js
const kmToMiles = (km) => km * 0.621371;
```

4. Вызовите функцию с желаемым значением километров в качестве аргумента, например:

```js
kmToMiles(8.1); // 5.0331051
```

Это вернет преобразованное значение в милях.
