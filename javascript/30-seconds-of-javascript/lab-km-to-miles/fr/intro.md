# Introduction

Dans ce laboratoire, nous allons explorer la programmation JavaScript et apprendre à écrire du code qui convertit les kilomètres en miles. Le laboratoire se concentrera sur l'enseignement de la formule de conversion et sur la manière de l'implémenter dans le code à l'aide de JavaScript. À la fin du laboratoire, vous aurez une meilleure compréhension de la manière d'utiliser JavaScript pour effectuer des calculs et résoudre des problèmes.
