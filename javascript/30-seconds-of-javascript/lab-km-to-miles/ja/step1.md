# キロメートルからマイルへの変換

コードを使ってキロメートルをマイルに変換するには、次の手順に従います。

1. ターミナル/SSH を開き、コードを練習するために `node` と入力します。
2. 変換公式 `mi = km * 0.621371` を使用します。
3. 次のコード スニペットを書きます。

```js
const kmToMiles = (km) => km * 0.621371;
```

4. 関数に変換したいキロメートルの値を引数として渡します。たとえば：

```js
kmToMiles(8.1); // 5.0331051
```

これにより、マイル単位の変換値が返されます。
