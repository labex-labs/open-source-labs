# 킬로미터에서 마일로의 변환

코드를 사용하여 킬로미터를 마일로 변환하려면 다음 단계를 따르세요.

1. 터미널/SSH 를 열고 `node`를 입력하여 코딩 연습을 시작합니다.
2. 변환 공식 `mi = km * 0.621371`을 사용합니다.
3. 다음 코드 스니펫을 작성합니다.

```js
const kmToMiles = (km) => km * 0.621371;
```

4. 원하는 킬로미터 값을 인수로 사용하여 함수를 호출합니다. 예를 들어:

```js
kmToMiles(8.1); // 5.0331051
```

이렇게 하면 마일 단위로 변환된 값이 반환됩니다.
