# Conversión de kilómetros a millas

Para convertir kilómetros a millas utilizando código, sigue estos pasos:

1. Abre la Terminal/SSH y escribe `node` para comenzar a practicar la codificación.
2. Utiliza la fórmula de conversión `mi = km * 0.621371`.
3. Escribe el siguiente fragmento de código:

```js
const kmToMiles = (km) => km * 0.621371;
```

4. Llama a la función con el valor de kilómetros deseado como argumento, por ejemplo:

```js
kmToMiles(8.1); // 5.0331051
```

Esto devolverá el valor convertido en millas.
