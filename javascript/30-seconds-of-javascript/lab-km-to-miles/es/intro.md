# Introducción

En este laboratorio, exploraremos la programación en JavaScript y aprenderemos a escribir código que convierta kilómetros a millas. El laboratorio se centrará en enseñar la fórmula de conversión y cómo implementarla en código utilizando JavaScript. Al final del laboratorio, tendrás una mejor comprensión de cómo utilizar JavaScript para realizar cálculos y resolver problemas.
