# Einführung

In diesem Lab werden wir die JavaScript-Programmierung erkunden und lernen, wie man Code schreibt, der Kilometer in Meilen umrechnet. Das Lab wird sich auf das Vermitteln der Umrechnungsformel und auf die Implementierung dieser in Code mit JavaScript konzentrieren. Am Ende des Labs werdet ihr ein besseres Verständnis dafür haben, wie ihr JavaScript zur Durchführung von Berechnungen und zur Lösung von Problemen verwendet.
