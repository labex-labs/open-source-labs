# Kilometer in Meilen umrechnen

Um Kilometer in Meilen umzurechnen, folge diesen Schritten:

1. Öffne das Terminal/SSH und tippe `node`, um mit der Code-Praxis zu beginnen.
2. Verwende die Umrechnungsformel `mi = km * 0.621371`.
3. Schreibe den folgenden Codeausschnitt:

```js
const kmToMiles = (km) => km * 0.621371;
```

4. Rufe die Funktion mit dem gewünschten Kilometerwert als Argument auf, z. B.:

```js
kmToMiles(8.1); // 5.0331051
```

Dies wird den umgerechneten Wert in Meilen zurückgeben.
