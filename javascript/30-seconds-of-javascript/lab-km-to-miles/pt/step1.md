# Conversão de Quilômetros para Milhas

Para converter quilômetros em milhas usando código, siga estes passos:

1.  Abra o Terminal/SSH e digite `node` para começar a praticar a codificação.
2.  Use a fórmula de conversão `mi = km * 0.621371`.
3.  Escreva o seguinte trecho de código:

```js
const kmToMiles = (km) => km * 0.621371;
```

4.  Chame a função com o valor de quilômetros desejado como argumento, por exemplo:

```js
kmToMiles(8.1); // 5.0331051
```

Isso retornará o valor convertido em milhas.
