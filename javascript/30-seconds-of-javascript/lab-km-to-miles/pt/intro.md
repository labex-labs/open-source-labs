# Introdução

Neste laboratório, exploraremos a programação JavaScript e aprenderemos a escrever código que converte quilômetros em milhas. O laboratório se concentrará em ensinar a fórmula para a conversão e como implementá-la em código usando JavaScript. Ao final do laboratório, você terá uma melhor compreensão de como usar JavaScript para realizar cálculos e resolver problemas.
