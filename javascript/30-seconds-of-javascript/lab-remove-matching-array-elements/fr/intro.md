# Introduction

Dans ce laboratoire, nous allons explorer comment supprimer les éléments correspondants d'un tableau à l'aide de JavaScript. Nous utiliserons la méthode `Array.prototype.filter()` pour trouver les éléments qui correspondent à la condition donnée et la méthode `Array.prototype.reduce()` pour les supprimer du tableau original. À la fin de ce laboratoire, vous aurez une meilleure compréhension de la manière de manipuler les tableaux en JavaScript.
