# Introdução

Neste laboratório, exploraremos o conceito de calcular o ponto médio entre dois pares de pontos (x, y) usando JavaScript. Aprenderemos como desestruturar um array para obter as coordenadas x e y dos pontos finais e calcular o ponto médio para cada dimensão, dividindo a soma dos dois pontos finais por 2. Este laboratório foi projetado para aprimorar nossa compreensão das operações matemáticas básicas em JavaScript e aplicá-las para resolver problemas do mundo real.
