# 두 쌍의 (x, y) 점 사이의 중점 계산 지침:

두 쌍의 (x, y) 점 사이의 중점을 계산하려면 다음 단계를 따르세요.

1. 배열을 분해하여 `x1`, `y1`, `x2` 및 `y2`를 얻습니다.
2. 두 끝점의 합을 `2`로 나누어 각 차원의 중점을 계산합니다.

다음은 중점 계산 함수를 구현하는 코드 스니펫 예시입니다.

```js
const midpoint = ([x1, y1], [x2, y2]) => [(x1 + x2) / 2, (y1 + y2) / 2];
```

중점 좌표를 얻으려면 다음 매개변수를 사용하여 `midpoint` 함수를 호출할 수 있습니다.

```js
midpoint([2, 2], [4, 4]); // [3, 3]
midpoint([4, 4], [6, 6]); // [5, 5]
midpoint([1, 3], [2, 4]); // [1.5, 3.5]
```

# 코딩 시작하기:

코딩 연습을 시작하려면 다음 단계를 따르세요.

1. 터미널/SSH 를 엽니다.
2. `node`를 입력하여 Node.js 환경을 시작합니다.
