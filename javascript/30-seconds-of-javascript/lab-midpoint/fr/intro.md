# Introduction

Dans ce laboratoire, nous allons explorer le concept de calcul du point milieu entre deux paires de points (x,y) à l'aide de JavaScript. Nous apprendrons à extraire les composants d'un tableau pour obtenir les coordonnées x et y des extrémités et à calculer le point milieu pour chaque dimension en divisant la somme des deux extrémités par 2. Ce laboratoire est conçu pour améliorer notre compréhension des opérations mathématiques de base en JavaScript et les appliquer pour résoudre des problèmes du monde réel.
