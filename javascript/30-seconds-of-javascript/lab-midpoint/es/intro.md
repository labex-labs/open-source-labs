# Introducción

En este laboratorio, exploraremos el concepto de calcular el punto medio entre dos pares de puntos (x,y) utilizando JavaScript. Aprenderemos cómo desestructurar un array para obtener las coordenadas x e y de los extremos y calcular el punto medio para cada dimensión dividiendo la suma de los dos extremos por 2. Este laboratorio está diseñado para mejorar nuestra comprensión de las operaciones matemáticas básicas en JavaScript y aplicarlas para resolver problemas del mundo real.
