# 2つの(x,y)座標のペアの中点を計算するための手順

2つの(x,y)座標のペアの中点を計算するには、次の手順に従います。

1. 配列を分解して、`x1`、`y1`、`x2`、`y2`を取得します。
2. 各次元の中点を、2つの端点の合計を`2`で割ることにより計算します。

以下は、中点計算関数を実装したコードのサンプルです。

```js
const midpoint = ([x1, y1], [x2, y2]) => [(x1 + x2) / 2, (y1 + y2) / 2];
```

中点座標を取得するには、次のパラメータで`midpoint`関数を呼び出します。

```js
midpoint([2, 2], [4, 4]); // [3, 3]
midpoint([4, 4], [6, 6]); // [5, 5]
midpoint([1, 3], [2, 4]); // [1.5, 3.5]
```

# コーディングの始め方

コーディングの練習を始めるには、次の手順に従います。

1. ターミナル/SSHを開きます。
2. `node`と入力して、Node.js環境を起動します。
