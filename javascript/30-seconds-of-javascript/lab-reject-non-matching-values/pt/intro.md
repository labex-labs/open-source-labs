# Introdução

Neste laboratório, exploraremos como filtrar os valores de um array com base em uma função predicado, retornando apenas os valores para os quais a função predicado retorna `false`. Usaremos `Array.prototype.filter()` em combinação com a função predicado para alcançar isso. Ao final deste laboratório, você terá uma boa compreensão de como implementar essa técnica em seu código JavaScript.
