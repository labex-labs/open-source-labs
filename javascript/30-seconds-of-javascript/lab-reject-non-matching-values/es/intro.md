# Introducción

En este laboratorio, exploraremos cómo filtrar los valores de un array en función de una función predicado, devolviendo solo los valores para los cuales la función predicado devuelve `false`. Utilizaremos `Array.prototype.filter()` en combinación con la función predicado para lograr esto. Al final de este laboratorio, tendrás una buena comprensión de cómo implementar esta técnica en tu código JavaScript.
