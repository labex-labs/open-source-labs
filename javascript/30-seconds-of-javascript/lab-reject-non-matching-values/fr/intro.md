# Introduction

Dans ce laboratoire, nous allons explorer comment filtrer les valeurs d'un tableau en fonction d'une fonction prédicat, en ne renvoyant que les valeurs pour lesquelles la fonction prédicat renvoie `false`. Nous allons utiliser `Array.prototype.filter()` en combinaison avec la fonction prédicat pour y arriver. À la fin de ce laboratoire, vous aurez une bonne compréhension de la manière de mettre en œuvre cette technique dans votre code JavaScript.
