# Фильтрация значений массива

Для фильтрации массива на основе предикатной функции и возврата только тех значений, для которых предикатная функция возвращает false, следуйте следующим шагам:

1. Используйте `Array.prototype.filter()` в сочетании с предикатной функцией `pred`.
2. Метод фильтрации вернет только те значения, для которых предикатная функция возвращает `false`.
3. Чтобы отфильтровать несовпадающие значения, передайте предикатную функцию и массив в функцию `reject()`.

```js
const reject = (pred, array) => array.filter((...args) => !pred(...args));
```

Ниже приведены примеры использования функции `reject()`:

```js
reject((x) => x % 2 === 0, [1, 2, 3, 4, 5]); // [1, 3, 5]
reject((word) => word.length > 4, ["Apple", "Pear", "Kiwi", "Banana"]);
// ['Pear', 'Kiwi']
```

Следуя этим шагам, вы можете легко отфильтровать массив на основе предикатной функции и отбросить несовпадающие значения.
