# Введение

В этом практическом занятии мы изучим, как фильтровать значения массива на основе предикатной функции, возвращая только те значения, для которых предикатная функция возвращает `false`. Для этого мы будем использовать `Array.prototype.filter()` в сочетании с предикатной функцией. В конце этого практического занятия вы хорошо освоите, как реализовать эту технику в своем коде на JavaScript.
