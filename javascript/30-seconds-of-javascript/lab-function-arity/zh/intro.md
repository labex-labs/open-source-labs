# 简介

在本实验中，我们将探索 JavaScript 中的「函数元数（Function Arity）」概念。本实验的目的是创建一个高阶函数，该函数接受一个函数和一个数字 `n` 作为参数，并返回一个新函数，该新函数最多只接受 `n` 个参数，忽略任何额外的参数。这将帮助我们理解如何在 JavaScript 中限制传递给函数的参数数量。
