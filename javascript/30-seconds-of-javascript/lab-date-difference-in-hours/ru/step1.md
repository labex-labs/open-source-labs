# JavaScript-функция для вычисления разницы дат в часах

Чтобы вычислить разницу между двумя датами в часах с использованием JavaScript, следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.

2. Используйте следующую JavaScript-функцию, чтобы получить разницу (в часах) между двумя объектами `Date`:

```js
const getHoursDiffBetweenDates = (dateInitial, dateFinal) =>
  (dateFinal - dateInitial) / (1000 * 3600);
```

3. Вызовите функцию с двумя датами в качестве аргументов, чтобы получить разницу в часах:

```js
getHoursDiffBetweenDates(
  new Date("2021-04-24 10:25:00"),
  new Date("2021-04-25 10:25:00")
); // 24
```
