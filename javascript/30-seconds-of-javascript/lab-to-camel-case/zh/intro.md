# 简介

在本实验中，我们将学习如何使用 JavaScript 将字符串转换为驼峰命名法。驼峰命名法是编程中使用的一种命名约定，其中多单词标识符的书写方式是，除第一个单词外，每个单词都以大写字母开头，单词之间没有任何空格或下划线。本实验中提供的 `toCamelCase()` 函数使用正则表达式和数组方法将字符串转换为驼峰命名法。
