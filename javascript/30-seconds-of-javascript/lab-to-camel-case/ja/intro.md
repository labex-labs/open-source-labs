# はじめに

この実験では、JavaScript を使って文字列をキャメルケースに変換する方法を学びます。キャメルケースは、プログラミングで使用される命名規則であり、複数語の識別子は、最初の単語を除いて各単語が大文字で始まり、それらの間にスペースやアンダースコアがない書き方で記述されます。この実験で提供されている `toCamelCase()` 関数は、正規表現と配列メソッドを使って文字列をキャメルケースに変換します。
