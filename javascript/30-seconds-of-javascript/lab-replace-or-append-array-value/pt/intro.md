# Introdução

Neste laboratório, trabalharemos em um exercício de programação JavaScript que envolve a substituição ou adição de valores em um array com base em uma função de comparação. Através deste exercício, você aprenderá como usar o operador spread (`...`), `Array.prototype.findIndex()`, `Array.prototype.push()`, e `Array.prototype.splice()` para manipular arrays em JavaScript. Este laboratório o ajudará a aprimorar suas habilidades em trabalhar com arrays e funções em JavaScript.
