# Функция форматирования числа

Для форматирования числа с использованием местного формата чисел следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте метод `Number.prototype.toLocaleString()`, чтобы преобразовать число с использованием разделителей местного формата чисел.
3. Передайте число, которое вы хотите отформатировать, в качестве аргумента функции.

Вот пример реализации:

```js
const formatNumber = (num) => num.toLocaleString();
```

Вот несколько примеров использования функции:

```js
formatNumber(123456); // '123,456' в `en-US`
formatNumber(15675436903); // '15.675.436.903' в `de-DE`
```
