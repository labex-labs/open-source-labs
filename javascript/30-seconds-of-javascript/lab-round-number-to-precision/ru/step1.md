# Вот как округлить число до заданной точности в JavaScript:

```
const round = (n, decimals = 0) =>
  Number(`${Math.round(`${n}e${decimals}`)}e-${decimals}`);
```

- Используйте `Math.round()` и шаблонные литералы для округления числа до заданного количества цифр.
- Если вы хотите округлить до целого числа, опустите второй аргумент, `decimals`.
- Чтобы начать практиковаться в программировании, откройте Терминал/SSH и введите `node`.
- Например, `round(1.005, 2)` вернет `1.01`.
