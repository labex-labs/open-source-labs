# Introduction

Dans ce laboratoire, nous allons explorer la manière de arrondir des nombres à une précision donnée en utilisant JavaScript. Nous allons apprendre à utiliser la méthode `Math.round()` et les littéraux de gabarit pour arrondir un nombre à un nombre spécifié de chiffres ou à une valeur entière. Cette connaissance sera utile dans des situations où nous devons afficher des nombres dans un format plus lisible, comme lorsqu'il s'agit de traiter des devises ou des mesures.
