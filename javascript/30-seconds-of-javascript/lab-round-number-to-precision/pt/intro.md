# Introdução

Neste laboratório, exploraremos como arredondar números para uma precisão específica usando JavaScript. Aprenderemos como usar o método `Math.round()` e _template literals_ para arredondar um número para um número especificado de dígitos ou para um valor inteiro. Este conhecimento será útil em situações onde precisamos exibir números em um formato mais legível, como ao lidar com moeda ou medições.
