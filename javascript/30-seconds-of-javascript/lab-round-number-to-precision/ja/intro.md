# はじめに

この実験では、JavaScript を使って数値を指定された精度に丸める方法を探ります。`Math.round()` メソッドとテンプレートリテラルを使って、数値を指定された桁数または整数値に丸める方法を学びます。この知識は、通貨や計測値など、より読みやすい形式で数値を表示する必要がある状況で役立ちます。
