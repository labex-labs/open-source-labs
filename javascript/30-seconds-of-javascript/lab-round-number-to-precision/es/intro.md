# Introducción

En este laboratorio, exploraremos cómo redondear números a una precisión dada utilizando JavaScript. Aprenderemos a utilizar el método `Math.round()` y los literales de plantilla para redondear un número a un número específico de dígitos o a un valor entero. Este conocimiento será útil en situaciones donde necesitemos mostrar números en un formato más legible, como al trabajar con divisas o medidas.
