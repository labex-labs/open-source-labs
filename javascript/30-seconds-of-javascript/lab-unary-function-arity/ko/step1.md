# 단항 함수 아리티 이해

코딩을 시작하려면 터미널/SSH 를 열고 `node`를 입력하세요.

단항 함수 아리티 (Unary function arity) 는 추가 인자를 무시하고 하나의 인자만 받는 함수를 의미합니다.

제공된 함수 `fn`은 첫 번째 인자만 사용하여 호출할 수 있습니다. 단항 함수를 만들려면 다음 코드를 사용하세요.

```js
const unary = (fn) => (val) => fn(val);
```

`parseInt` 함수와 함께 `unary`를 사용하는 예는 다음과 같습니다.

```js
["6", "8", "10"].map(unary(parseInt)); // [6, 8, 10]
```
