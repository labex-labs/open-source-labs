# Introdução

Neste laboratório, exploraremos o conceito de "Aridade de Função Unária" (Unary Function Arity) em JavaScript. Aprenderemos como criar uma função que aceita apenas um argumento e ignora quaisquer argumentos adicionais. Este laboratório também demonstrará como chamar a função fornecida com apenas o primeiro argumento fornecido. Ao final deste laboratório, você será capaz de implementar esta técnica em seu código para melhorar sua eficiência e legibilidade.
