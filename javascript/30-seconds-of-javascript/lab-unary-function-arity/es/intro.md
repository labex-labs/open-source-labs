# Introducción

En este laboratorio, exploraremos el concepto de "Unary Function Arity" en JavaScript. Aprenderemos cómo crear una función que acepte solo un argumento y ignore cualquier argumento adicional. Este laboratorio también demostrará cómo llamar a la función proporcionada solo con el primer argumento suministrado. Al final de este laboratorio, serás capaz de implementar esta técnica en tu código para mejorar su eficiencia y legibilidad.
