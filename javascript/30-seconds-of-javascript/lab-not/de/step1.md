# Das Verwenden des logischen Negationsoperators in JavaScript

Um zu beginnen, JavaScript zu programmieren, öffnen Sie das Terminal/SSH und geben Sie `node` ein.

Der logische Negationsoperator (`!`) gibt die Umkehrung des angegebenen Werts zurück. Im folgenden finden Sie eine Beispielimplementierung der logischen Not-Funktion in JavaScript:

```js
const not = (a) => !a;
```

Sie können die `not`-Funktion verwenden, um die Umkehrung eines angegebenen Werts zurückzugeben. Hier sind einige Beispiele:

```js
not(true); // false
not(false); // true
```
