# Usando el operador lógico no en JavaScript

Para comenzar a practicar la programación en JavaScript, abre la Terminal/SSH y escribe `node`.

El operador lógico no (`!`) devuelve el inverso del valor dado. A continuación, se muestra una implementación de ejemplo de la función lógica no en JavaScript:

```js
const not = (a) => !a;
```

Puedes usar la función `not` para devolver el inverso de un valor dado. Aquí hay algunos ejemplos:

```js
not(true); // false
not(false); // true
```
