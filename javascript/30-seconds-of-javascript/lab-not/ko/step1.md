# JavaScript 에서 논리 부정 연산자 사용하기

JavaScript 코딩 연습을 시작하려면 터미널/SSH 를 열고 `node`를 입력하세요.

논리 부정 연산자 (`!`) 는 주어진 값의 반대 값을 반환합니다. 다음은 JavaScript 에서 논리 부정 함수를 구현한 예시입니다.

```js
const not = (a) => !a;
```

`not` 함수를 사용하여 주어진 값의 반대 값을 반환할 수 있습니다. 몇 가지 예시는 다음과 같습니다.

```js
not(true); // false
not(false); // true
```
