# Введение

В этом лабораторном задании мы будем изучать концепцию логического оператора отрицания в JavaScript. В частности, мы создадим функцию, которая принимает логическое значение и возвращает его обратное с использованием оператора `!`. С помощью этого упражнения вы получите более глубокое понимание того, как работают логические операции в JavaScript и как они могут быть использованы в повседневных задачах программирования.
