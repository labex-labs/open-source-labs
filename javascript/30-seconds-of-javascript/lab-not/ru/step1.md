# Использование логического оператора отрицания в JavaScript

Для начала практики программирования на JavaScript откройте Терминал/SSH и введите `node`.

Логический оператор отрицания (`!`) возвращает обратное значение заданному. Ниже приведен пример реализации логической функции отрицания в JavaScript:

```js
const not = (a) => !a;
```

Вы можете использовать функцию `not`, чтобы вернуть обратное значение заданному. Вот несколько примеров:

```js
not(true); // false
not(false); // true
```
