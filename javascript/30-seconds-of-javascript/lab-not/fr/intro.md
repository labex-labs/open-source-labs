# Introduction

Dans ce laboratoire, nous allons explorer le concept de l'opérateur logique non (`not`) en JavaScript. Plus précisément, nous allons créer une fonction qui prend une valeur booléenne et renvoie son inverse en utilisant l'opérateur `!`. Grâce à cet exercice, vous comprendrez mieux comment les opérations logiques fonctionnent en JavaScript et comment elles peuvent être utilisées dans les tâches de programmation quotidiennes.
