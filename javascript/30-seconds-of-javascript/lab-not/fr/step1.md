# Utilisation de l'opérateur logique non (`not`) en JavaScript

Pour commencer à pratiquer la programmation en JavaScript, ouvrez le Terminal/SSH et tapez `node`.

L'opérateur logique non (`!`) renvoie l'inverse de la valeur donnée. Voici une implémentation de l'exemple de fonction logique non en JavaScript :

```js
const not = (a) => !a;
```

Vous pouvez utiliser la fonction `not` pour renvoyer l'inverse d'une valeur donnée. Voici quelques exemples :

```js
not(true); // false
not(false); // true
```
