# Usando o Operador Lógico "Not" em JavaScript

Para começar a praticar a codificação em JavaScript, abra o Terminal/SSH e digite `node`.

O operador lógico "not" (`!`) retorna o inverso do valor fornecido. Abaixo está um exemplo de implementação da função lógica "not" em JavaScript:

```js
const not = (a) => !a;
```

Você pode usar a função `not` para retornar o inverso de um valor fornecido. Aqui estão alguns exemplos:

```js
not(true); // false
not(false); // true
```
