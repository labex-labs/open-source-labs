# Introdução

Neste laboratório, exploraremos o conceito do operador lógico "not" em JavaScript. Especificamente, criaremos uma função que recebe um valor booleano e retorna seu inverso usando o operador `!`. Através deste exercício, você obterá uma compreensão mais profunda de como as operações lógicas funcionam em JavaScript e como elas podem ser usadas em tarefas de programação do dia a dia.
