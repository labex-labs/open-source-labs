# Как вычислить медиану массива чисел

Для вычисления медианы массива чисел следуйте этим шагам:

1. Найдите середину массива.
2. Используйте `Array.prototype.sort()`, чтобы отсортировать значения.
3. Если `Array.prototype.length` нечетное, верните число в середине. Если четное, верните среднее арифметическое двух центральных чисел.
4. Чтобы начать практиковать программирование и использовать `node`, откройте Терминал/SSH и введите `node`.

Вот пример кода, который реализует эту логику:

```js
const median = (arr) => {
  const mid = Math.floor(arr.length / 2),
    nums = [...arr].sort((a, b) => a - b);
  return arr.length % 2 !== 0 ? nums[mid] : (nums[mid - 1] + nums[mid]) / 2;
};
```

Вы можете вызвать эту функцию с массивом чисел, как показано ниже:

```js
median([5, 6, 50, 1, -5]); // 5
```
