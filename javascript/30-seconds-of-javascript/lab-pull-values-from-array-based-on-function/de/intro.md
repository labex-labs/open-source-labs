# Einführung

In diesem Lab werden wir die `pullBy()`-Funktion in JavaScript erkunden. Diese Funktion ermöglicht es uns, bestimmte Werte aus einem Array basierend auf einer angegebenen Iterationsfunktion zu filtern. Am Ende dieses Labs werden Sie verstehen, wie Sie `pullBy()` verwenden, um Arrays zu manipulieren und unerwünschte Werte effektiv zu filtern.
