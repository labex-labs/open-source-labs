# Введение

В этом практическом занятии мы изучим функцию `pullBy()` в JavaScript. Эта функция позволяет нам фильтровать определенные значения из массива на основе заданной итераторной функции. В конце этого практического занятия вы поймете, как использовать `pullBy()` для манипуляции массивами и эффективного фильтрации нежелательных значений.
