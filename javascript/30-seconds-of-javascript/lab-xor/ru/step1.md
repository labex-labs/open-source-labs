# Логическое исключающее ИЛИ (XOR)

Для начала практики в программировании откройте Терминал/SSH и введите `node`. Логическое исключающее ИЛИ проверяет, является ли истинным только один из аргументов. Чтобы создать логическое исключающее ИЛИ, используйте операторы логического ИЛИ (`||`), логического И (`&&`) и логическое отрицание (`!`) для двух заданных значений. Вот пример кода для этого:

```js
const xor = (a, b) => (a || b) && !(a && b);
```

Вот выходные значения:

```js
xor(true, true); // false
xor(true, false); // true
xor(false, true); // true
xor(false, false); // false
```
