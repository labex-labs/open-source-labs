# 論理排他的論理和

コーディングの練習を始めるには、ターミナル/SSH を開いて`node`と入力します。論理排他的論理和は、引数のうち一方のみが`true`であるかどうかを確認します。論理排他的論理和を作成するには、与えられた 2 つの値に対して論理和（`||`）、論理積（`&&`）、および否定（`!`）演算子を使用します。以下はそれに対するサンプルコードです：

```js
const xor = (a, b) => (a || b) && !(a && b);
```

以下は出力値です：

```js
xor(true, true); // false
xor(true, false); // true
xor(false, true); // true
xor(false, false); // false
```
