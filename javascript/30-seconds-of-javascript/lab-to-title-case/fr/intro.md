# Introduction

Dans ce laboratoire, nous allons explorer la manière de convertir une chaîne de caractères en cas titre à l'aide de JavaScript. Nous utiliserons des expressions régulières pour diviser la chaîne en mots puis mettre en majuscule la première lettre de chaque mot. À la fin de ce laboratoire, vous aurez une meilleure compréhension de la manière de manipuler les chaînes de caractères en JavaScript et d'appliquer cette technique pour formater le texte dans vos applications web.
