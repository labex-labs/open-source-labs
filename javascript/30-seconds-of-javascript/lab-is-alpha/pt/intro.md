# Introdução

Neste laboratório, exploraremos como determinar se uma determinada string contém apenas caracteres alfabéticos usando JavaScript. Usaremos um padrão de expressão regular para testar a string de entrada e retornar um valor booleano indicando se a string contém apenas caracteres alfabéticos ou não. Este laboratório visa ajudá-lo a fortalecer sua compreensão de expressões regulares e seu uso em JavaScript.
