# Einführung

In diesem Lab werden wir untersuchen, wie man mit JavaScript feststellt, ob ein gegebener String ausschließlich alphabetische Zeichen enthält. Wir werden ein regulärer Ausdrucksmuster verwenden, um die Eingabezeichenfolge zu testen und einen booleschen Wert zurückzugeben, der angibt, ob der String ausschließlich alphabetische Zeichen enthält oder nicht. Ziel dieses Labs ist es, Ihr Verständnis von regulären Ausdrücken und ihrer Verwendung in JavaScript zu stärken.
