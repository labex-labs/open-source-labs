# Функция для проверки, является ли строка алфавитной

Для проверки того, содержит ли строка только алфавитные символы:

- Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
- Используйте `RegExp.prototype.test()`, чтобы проверить, соответствует ли данная строка шаблону регулярного выражения для алфавитных символов.
- Функция `isAlpha` принимает строку в качестве аргумента и возвращает `true`, если строка содержит только алфавитные символы, и `false` в противном случае.

Вот пример:

```js
const isAlpha = (str) => /^[a-zA-Z]*$/.test(str);
```

```js
isAlpha("sampleInput"); // true
isAlpha("this Will fail"); // false
isAlpha("123"); // false
```
