# Введение

В этом практическом занятии мы изучим, как определить, содержит ли данная строка только алфавитные символы, используя JavaScript. Мы будем использовать регулярное выражение для проверки входной строки и возвращать логическое значение, указывающее, содержит ли строка только алфавитные символы или нет. Эта лабораторная работа направлена на то, чтобы помочь вам углубить свое понимание регулярных выражений и их применения в JavaScript.
