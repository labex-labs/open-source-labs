# Introducción

En este laboratorio, exploraremos cómo determinar si una cadena dada contiene solo caracteres alfabéticos utilizando JavaScript. Utilizaremos un patrón de expresión regular para probar la cadena de entrada y devolver un valor booleano que indique si la cadena contiene solo caracteres alfabéticos o no. Este laboratorio tiene como objetivo ayudarte a fortalecer tu comprensión de las expresiones regulares y su uso en JavaScript.
