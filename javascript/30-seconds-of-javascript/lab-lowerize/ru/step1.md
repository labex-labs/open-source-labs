# Преобразование ключей объекта в нижний регистр

Для преобразования всех ключей объекта в нижний регистр выполните следующие шаги:

1. Откройте Терминал/SSH, чтобы начать практиковать программирование, и введите `node`.
2. Используйте метод `Object.keys()`, чтобы получить массив ключей объекта.
3. Используйте метод `Array.prototype.reduce()`, чтобы преобразовать массив в новый объект.
4. Используйте метод `String.prototype.toLowerCase()`, чтобы перевести ключи в нижний регистр.

Вот пример кода, реализующего эти шаги:

```js
const lowerize = (obj) =>
  Object.keys(obj).reduce((acc, k) => {
    acc[k.toLowerCase()] = obj[k];
    return acc;
  }, {});
```

Затем вы можете вызвать функцию `lowerize()` с объектом в качестве аргумента, чтобы получить новый объект с всеми ключами в нижнем регистре. Например:

```js
lowerize({ Name: "John", Age: 22 }); // { name: 'John', age: 22 }
```
