# Введение

В этом лабораторном занятии (LabEx) мы научимся преобразовывать все ключи объекта в нижний регистр с использованием JavaScript. Мы будем использовать методы `Object.keys()` и `Array.prototype.reduce()` для создания нового объекта с преобразованными ключами и метод `String.prototype.toLowerCase()` для перевода ключей в нижний регистр. Этот подход может быть полезен при работе с объектами, в которых регистр ключей не является единообразным.
