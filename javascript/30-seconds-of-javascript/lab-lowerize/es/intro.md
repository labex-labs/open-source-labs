# Introducción

En este laboratorio, aprenderemos cómo convertir todas las claves de un objeto a minúsculas utilizando JavaScript. Utilizaremos `Object.keys()` y `Array.prototype.reduce()` para mapear las claves a un objeto y `String.prototype.toLowerCase()` para convertirlas a minúsculas. Esta técnica puede ser útil cuando se trabaja con objetos que tienen una capitalización de claves inconsistente.
