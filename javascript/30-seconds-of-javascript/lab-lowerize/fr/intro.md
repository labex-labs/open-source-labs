# Introduction

Dans ce laboratoire (lab), nous allons apprendre à convertir toutes les clés d'un objet en minuscules à l'aide de JavaScript. Nous utiliserons `Object.keys()` et `Array.prototype.reduce()` pour mapper les clés sur un objet et `String.prototype.toLowerCase()` pour les convertir en minuscules. Cette technique peut être utile lorsqu'on manipule des objets dont la casse des clés est incohérente.
