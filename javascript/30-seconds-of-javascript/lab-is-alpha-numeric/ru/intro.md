# Введение

В этом практическом занятии (лабораторной работе) мы рассмотрим, как проверить, содержит ли заданная строка только алфавитно-цифровые символы с использованием JavaScript. Мы будем использовать шаблон регулярного выражения для сопоставления входной строки с алфавитно-цифровыми символами и возвращения булевого значения, указывающего, является ли строка алфавитно-цифровой или нет. Это практическое занятие поможет вам понять, как работать с регулярными выражениями в JavaScript и как использовать их для решения практических задач.
