# Проверка, является ли строка алфавитно-цифровой

Если вы хотите практиковать программирование, откройте Терминал/SSH и введите `node`. Вот функция, которая проверяет, содержит ли строка только алфавитно-цифровые символы:

```js
const isAlphaNumeric = (str) => /^[a-z0-9]+$/gi.test(str);
```

Чтобы использовать эту функцию, вызовите `isAlphaNumeric`, передав строку в качестве аргумента. Функция вернет `true`, если строка содержит только алфавитно-цифровые символы, и `false` в противном случае.

Например:

```js
isAlphaNumeric("hello123"); // true
isAlphaNumeric("123"); // true
isAlphaNumeric("hello 123"); // false (содержит пробел)
isAlphaNumeric("#$hello"); // false (содержит неалфавитно-цифровые символы)
```

Метод `RegExp.prototype.test()` используется для проверки, соответствует ли входная строка алфавитно-цифровому шаблону, который представлен регулярным выражением `/^[a-z0-9]+$/gi`. Этот шаблон соответствует любой последовательности из одного или нескольких строчных букв или цифр, а флаги `g` и `i` делают сопоставление регистронезависимым.
