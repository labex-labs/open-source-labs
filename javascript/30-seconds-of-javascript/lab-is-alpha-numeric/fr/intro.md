# Introduction

Dans ce laboratoire, nous allons explorer comment vérifier si une chaîne de caractères donnée ne contient que des caractères alphanumériques à l'aide de JavaScript. Nous utiliserons un motif d'expression régulière pour comparer la chaîne d'entrée avec les caractères alphanumériques et renvoyer une valeur booléenne indiquant si la chaîne est alphanumérique ou non. Ce laboratoire vous aidera à comprendre comment travailler avec les expressions régulières en JavaScript et comment les utiliser pour résoudre des problèmes pratiques.
