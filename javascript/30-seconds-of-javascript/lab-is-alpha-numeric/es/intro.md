# Introducción

En este laboratorio, exploraremos cómo verificar si una cadena dada contiene solo caracteres alfanuméricos utilizando JavaScript. Utilizaremos un patrón de expresión regular para comparar la cadena de entrada con caracteres alfanuméricos y devolver un valor booleano que indique si la cadena es alfanumérica o no. Este laboratorio te ayudará a entender cómo trabajar con expresiones regulares en JavaScript y cómo utilizarlas para resolver problemas prácticos.
