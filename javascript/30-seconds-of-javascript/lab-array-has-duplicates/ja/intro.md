# はじめに

この実験では、JavaScript を使って配列に重複する要素があるかどうかを確認する方法を探ります。配列内の一意の値を取得するために `Set` オブジェクトを使い、その後、これらの一意の値の数を元の配列の長さと比較します。この実験が終わるとき、JavaScript を使って配列内の重複を効率的に確認する方法をより深く理解しているでしょう。
