# Практика программирования: получение случайных элементов из массива

Для практики программирования откройте Терминал/SSH и введите `node`. Следующий код использует алгоритм Фишера-Йейтса для перемешивания массива и получения `n` случайных, уникальных элементов по уникальным ключам из массива, до размера массива.

```js
const sampleSize = ([...arr], n = 1) => {
  let m = arr.length;
  while (m) {
    const i = Math.floor(Math.random() * m--);
    [arr[m], arr[i]] = [arr[i], arr[m]];
  }
  return arr.slice(0, n);
};
```

Для использования этого кода вызовите `sampleSize()` с массивом и необязательным числом `n` элементов для извлечения. Если `n` не указано, функция вернет только один случайный элемент из массива.

```js
sampleSize([1, 2, 3], 2); // [3, 1]
sampleSize([1, 2, 3], 4); // [2, 3, 1]
```
