# Introdução

Neste laboratório, exploraremos como selecionar elementos aleatórios de um array usando JavaScript. Aprenderemos como embaralhar um array usando o algoritmo Fisher-Yates e usá-lo para selecionar aleatoriamente um ou mais elementos de um array. Este laboratório foi projetado para aprimorar sua compreensão da manipulação de arrays e melhorar suas habilidades de resolução de problemas em JavaScript.
