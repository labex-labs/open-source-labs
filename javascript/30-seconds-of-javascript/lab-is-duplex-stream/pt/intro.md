# Introdução

Neste laboratório, exploraremos como determinar se um determinado argumento é um stream duplex (legível e gravável) em JavaScript. Usaremos uma combinação de verificações `typeof` e validações de propriedades para criar uma função que possa identificar com precisão streams duplex. Ao final deste laboratório, você terá uma melhor compreensão de como os streams funcionam em JavaScript e como trabalhar com eles em seu próprio código.
