# Introduction

Dans ce laboratoire, nous allons explorer la manière de déterminer si un argument donné est un flux duplex (lisible et écrivable) en JavaScript. Nous utiliserons une combinaison de vérifications `typeof` et de validations de propriétés pour créer une fonction qui peut identifier précisément les flux duplex. À la fin de ce laboratoire, vous aurez une meilleure compréhension de la manière dont les flux fonctionnent en JavaScript et de la manière de les utiliser dans votre propre code.
