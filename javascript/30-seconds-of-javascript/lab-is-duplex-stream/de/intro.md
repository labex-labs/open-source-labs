# Einführung

In diesem Lab werden wir untersuchen, wie man in JavaScript feststellt, ob ein gegebener Argument ein duplex (lesbar und beschreibbar) Stream ist. Wir werden eine Kombination aus `typeof`-Prüfungen und Eigenschaftsvalidierungen verwenden, um eine Funktion zu erstellen, die duplex Streams genau identifizieren kann. Am Ende dieses Labs werden Sie ein besseres Verständnis dafür haben, wie Streams in JavaScript funktionieren und wie Sie mit ihnen in Ihrem eigenen Code umgehen können.
