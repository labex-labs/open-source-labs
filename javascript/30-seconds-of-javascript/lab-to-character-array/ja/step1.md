# JavaScript で文字列を文字配列に変換する方法

JavaScript で文字列を文字配列に変換するには、次の手順に従います。

1. ターミナル/SSH を開き、コーディングを練習するために `node` と入力します。
2. スプレッド演算子 (`...`) を使用して、文字列を文字配列に変換します。
3. `toCharArray` という関数を定義し、文字列引数を取り、その文字の配列を返します。
4. 変換したい文字列引数として `toCharArray` 関数を呼び出します。
5. 関数は文字の配列を返します。

コードは次のとおりです。

```js
const toCharArray = (s) => [...s];

toCharArray("hello"); // ['h', 'e', 'l', 'l', 'o']
```
