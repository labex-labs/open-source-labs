# Как преобразовать строку в массив символов в JavaScript

Для преобразования строки в массив символов в JavaScript следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковать программирование.
2. Используйте оператор расширения (`...`), чтобы преобразовать строку в массив символов.
3. Определите функцию под названием `toCharArray`, которая принимает строку в качестве аргумента и возвращает массив ее символов.
4. Вызовите функцию `toCharArray` с аргументом, который является строкой, которую вы хотите преобразовать.
5. Функция вернет массив символов.

Вот код:

```js
const toCharArray = (s) => [...s];

toCharArray("hello"); // ['h', 'e', 'l', 'l', 'o']
```
