# Einführung

In diesem Lab werden wir untersuchen, wie man in JavaScript eine Zeichenfolge in ein Array von Zeichen umwandelt. Dies ist eine häufige Aufgabe bei der Programmierung, wenn es um Zeichenfolgen geht, da es ermöglicht, einzelne Zeichen innerhalb der Zeichenfolge zu manipulieren. Mit dem Spread-Operator können wir eine Zeichenfolge in ein Array von Zeichen in nur wenigen Codezeilen leicht umwandeln.
