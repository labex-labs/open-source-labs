# Introducción

En este laboratorio, exploraremos cómo convertir una cadena en una matriz de caracteres en JavaScript. Esta es una tarea común en la programación al trabajar con cadenas, ya que permite manipular cada carácter dentro de la cadena. Utilizando el operador de propagación, podemos convertir fácilmente una cadena en una matriz de caracteres con solo unas pocas líneas de código.
