# Einführung

In diesem Lab werden wir untersuchen, wie man in JavaScript ein Objekt mit Pfaden für Schlüssel entflacht. Wir werden die geschachtelten Methoden `Array.prototype.reduce()` und `String.prototype.split()` verwenden, um jeden Schlüssel mit einem Punkt als Trennzeichen zu teilen und gegen die Schlüssel Objekte hinzuzufügen. Am Ende dieses Labs werden Sie eine solide Vorstellung davon haben, wie man effizient ein flaches Objekt in ein geschachteltes umwandelt.
