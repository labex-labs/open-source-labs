# Introdução

Neste laboratório, exploraremos o conceito de pluralização de strings em JavaScript. Você aprenderá como usar uma closure (fechamento) para definir uma função que pode retornar a forma singular ou plural de uma palavra com base em um número de entrada. Além disso, você terá a oportunidade de praticar o uso de argumentos opcionais e dicionários para personalizar o comportamento da função. Ao final deste laboratório, você terá uma sólida compreensão de como lidar com a pluralização em seus projetos JavaScript.
