# Введение

В этом практическом занятии мы изучим, как создавать массив пар ключ-значение из объекта с использованием JavaScript. Мы будем использовать метод `Object.entries()`, чтобы извлечь ключи и значения объекта и вернуть их в виде массива пар. Это умение полезно при работе с API или базами данных, которые возвращают данные в формате объекта.
