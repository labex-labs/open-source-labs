# Преобразование объекта в массив пар ключ-значение

Для преобразования объекта в массив пар ключ-значение вы можете следовать следующим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковать программирование.
2. Используйте метод `Object.entries()`, чтобы получить массив массивов пар ключ-значение из объекта.
3. Создайте функцию под названием `objectToPairs`, которая принимает объект в качестве аргумента и возвращает массив пар ключ-значение.
4. Вызовите функцию `objectToPairs` с примером объекта, чтобы проверить вывод.

Вот пример реализации:

```js
const objectToPairs = (obj) => Object.entries(obj);
objectToPairs({ a: 1, b: 2 }); // [ ['a', 1], ['b', 2] ]
```

Следуя этим шагам, вы можете легко преобразовать объект в массив пар ключ-значение с использованием JavaScript.
