# Einführung

In diesem Lab werden wir untersuchen, wie man aus einem Objekt ein Array von Schlüssel-Wert-Paaren mit JavaScript erstellt. Wir werden die `Object.entries()`-Methode verwenden, um die Schlüssel und Werte des Objekts zu extrahieren und als Array von Paaren zurückzugeben. Diese Fähigkeit ist nützlich, wenn man mit APIs oder Datenbanken arbeitet, die Daten im Objektformat zurückgeben.
