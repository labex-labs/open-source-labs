# Introdução

Neste laboratório, exploraremos como criar um array de pares chave-valor a partir de um objeto usando JavaScript. Usaremos o método `Object.entries()` para extrair as chaves e os valores do objeto e retorná-los como um array de pares. Essa habilidade é útil ao trabalhar com APIs ou bancos de dados que retornam dados em formato de objeto.
