# Introduction

Dans ce laboratoire, nous allons explorer comment créer un tableau de paires clé-valeur à partir d'un objet en utilisant JavaScript. Nous utiliserons la méthode `Object.entries()` pour extraire les clés et les valeurs de l'objet et les retourner sous forme d'un tableau de paires. Cette compétence est utile lorsqu'on travaille avec des API ou des bases de données qui renvoient des données au format objet.
