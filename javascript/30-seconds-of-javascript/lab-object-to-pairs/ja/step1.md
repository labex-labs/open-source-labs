# オブジェクトをキーと値のペアの配列に変換する

オブジェクトをキーと値のペアの配列に変換するには、次の手順を実行できます。

1. ターミナル/SSH を開き、コーディングを練習するために `node` と入力します。
2. `Object.entries()` メソッドを使用して、オブジェクトからキーと値のペアの配列の配列を取得します。
3. `objectToPairs` という関数を作成し、オブジェクトを引数として受け取り、キーと値のペアの配列を返します。
4. 出力をテストするために、サンプル オブジェクトを使用して `objectToPairs` 関数を呼び出します。

以下は、例としての実装です。

```js
const objectToPairs = (obj) => Object.entries(obj);
objectToPairs({ a: 1, b: 2 }); // [ ['a', 1], ['b', 2] ]
```

これらの手順に従えば、JavaScript を使ってオブジェクトをキーと値のペアの配列に簡単に変換できます。
