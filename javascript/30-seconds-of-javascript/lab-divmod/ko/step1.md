# 코드 연습: 나눗셈의 몫과 나머지

코딩 연습을 위해 터미널/SSH 를 열고 `node`를 입력하세요. 이 코드는 주어진 숫자의 몫과 나머지로 구성된 배열을 반환합니다.

`x / y` 나눗셈의 몫을 구하려면 `Math.floor()`를 사용합니다. `x / y` 나눗셈의 나머지를 구하려면 모듈로 연산자 (`%`) 를 사용합니다.

```js
const divmod = (x, y) => [Math.floor(x / y), x % y];
```

예를 들어:

```js
divmod(8, 3); // [2, 2]
divmod(3, 8); // [0, 3]
divmod(5, 5); // [1, 0]
```
