# Практика программирования: Частное и остаток от деления

Для практики программирования откройте Терминал/SSH и введите `node`. Этот код возвращает массив, состоящий из частного и остатка от деления заданных чисел.

Для получения частного от деления `x / y` используйте `Math.floor()`. Для получения остатка от деления `x / y` используйте оператор модуля (`%`).

```js
const divmod = (x, y) => [Math.floor(x / y), x % y];
```

Например:

```js
divmod(8, 3); // [2, 2]
divmod(3, 8); // [0, 3]
divmod(5, 5); // [1, 0]
```
