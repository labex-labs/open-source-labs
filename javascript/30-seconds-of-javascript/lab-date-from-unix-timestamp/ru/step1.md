# Как создать объект Date из Unix-метки времени

Для создания объекта `Date` из Unix-метки времени следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Умножьте метку времени на `1000`, чтобы преобразовать ее в миллисекунды.
3. Используйте конструктор `Date`, чтобы создать новый объект `Date`.

Вот пример кода:

```js
const fromTimestamp = (timestamp) => new Date(timestamp * 1000);
```

Вы можете использовать эту функцию для преобразования Unix-метки времени в объект `Date` следующим образом:

```js
fromTimestamp(1602162242); // 2020-10-08T13:04:02.000Z
```
