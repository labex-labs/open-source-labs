# Инструкции по выбору ключей объекта

Для выбора определенных пар ключ-значение из объекта используйте функцию `pick(obj, arr)`.

- Передайте объект в качестве первого аргумента и массив ключей для выбора в качестве второго аргумента.
- Функция возвращает новый объект с только парами ключ-значение, соответствующими заданным ключам.

Вот пример использования `pick()`:

```js
const pick = (obj, arr) =>
  arr.reduce((acc, curr) => (curr in obj && (acc[curr] = obj[curr]), acc), {});

pick({ a: 1, b: "2", c: 3 }, ["a", "c"]); // { 'a': 1, 'c': 3 }
```
