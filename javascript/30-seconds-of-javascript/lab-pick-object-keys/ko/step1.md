# 객체 키 선택 지침

객체에서 특정 키 - 값 쌍을 선택하려면 `pick(obj, arr)` 함수를 사용하십시오.

- 객체를 첫 번째 인수로, 선택할 키 배열을 두 번째 인수로 전달합니다.
- 이 함수는 주어진 키에 해당하는 키 - 값 쌍만 포함하는 새로운 객체를 반환합니다.

`pick()`을 사용하는 방법의 예는 다음과 같습니다.

```js
const pick = (obj, arr) =>
  arr.reduce((acc, curr) => (curr in obj && (acc[curr] = obj[curr]), acc), {});

pick({ a: 1, b: "2", c: 3 }, ["a", "c"]); // { 'a': 1, 'c': 3 }
```

코딩 연습을 시작하려면 터미널/SSH 를 열고 `node`를 입력하십시오.
