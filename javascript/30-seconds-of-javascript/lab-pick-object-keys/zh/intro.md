# 简介

在本实验中，我们将探索如何使用 JavaScript 中的 `pick` 函数从对象中提取特定的键值对。我们将学习如何将一个对象和一个键数组传递给 `pick` 函数，以及它如何使用 `Array.prototype.reduce()` 来过滤并仅从对象中返回指定的键值对。本实验将帮助你理解如何在 JavaScript 中处理对象，以及如何仅从对象中提取必要的信息。
