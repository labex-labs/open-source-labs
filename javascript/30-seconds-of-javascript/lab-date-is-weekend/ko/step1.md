# 날짜가 주말인지 확인하기

주어진 날짜가 주말인지 확인하려면 다음 단계를 따르세요.

- 터미널/SSH 를 열고 `node`를 입력하여 코딩 연습을 시작합니다.
- `Date.prototype.getDay()` 메서드를 사용하여 요일을 숫자로 가져옵니다 (0-6). 일요일은 0, 토요일은 6 입니다.
- 모듈로 연산자 (`%`) 를 사용하고 0 또는 6 과 비교하여 해당 요일이 주말인지 확인합니다.
- 인자 `d`를 생략하면 현재 날짜를 기본값으로 사용합니다.

다음은 사용할 수 있는 코드 스니펫 예시입니다.

```js
const isWeekend = (d = new Date()) => d.getDay() % 6 === 0;
```

함수를 테스트하려면 인자 없이 호출하기만 하면 됩니다.

```js
isWeekend(); // true 또는 false (현재 날짜에 따라 다름)
```

현재 날짜가 주말 (토요일 또는 일요일) 이면 `true`를 반환하고, 그렇지 않으면 `false`를 반환합니다.
