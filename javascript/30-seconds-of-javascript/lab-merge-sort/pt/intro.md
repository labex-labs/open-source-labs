# Introdução

Neste laboratório, exploraremos o algoritmo Merge Sort em JavaScript. Merge Sort é um algoritmo de ordenação popular do tipo "dividir e conquistar" (divide-and-conquer) que é eficiente e comumente usado na prática. Ao final deste laboratório, você terá uma sólida compreensão de como o Merge Sort funciona e como implementá-lo em seus próprios projetos JavaScript.
