# Introducción

En este laboratorio, exploraremos cómo convertir una cadena de color `hsl()` en una matriz de valores utilizando JavaScript. Utilizaremos una combinación de expresiones regulares y métodos de matriz para extraer los valores numéricos de la cadena y convertirlos en una matriz de valores numéricos. Este laboratorio te ayudará a entender cómo trabajar con valores de color en JavaScript y a mejorar tus habilidades con expresiones regulares y manipulación de matrices.
