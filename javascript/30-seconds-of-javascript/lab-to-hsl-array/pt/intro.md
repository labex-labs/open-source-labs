# Introdução

Neste laboratório, exploraremos como converter uma string de cor `hsl()` em um array de valores usando JavaScript. Usaremos uma combinação de expressões regulares e métodos de array para extrair os valores numéricos da string e convertê-los em um array de valores numéricos. Este laboratório ajudará você a entender como trabalhar com valores de cor em JavaScript e aprimorar suas habilidades com expressões regulares e manipulação de arrays.
