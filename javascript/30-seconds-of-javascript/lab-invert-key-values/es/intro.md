# Introducción

En este laboratorio, exploraremos cómo invertir los pares clave-valor de un objeto sin mutar el objeto original. Utilizaremos el método `Object.keys()` y `Array.prototype.reduce()` para invertir el objeto. Además, aprenderemos cómo aplicar una función a las claves invertidas y devolver una matriz de claves responsables de generar el valor invertido.
