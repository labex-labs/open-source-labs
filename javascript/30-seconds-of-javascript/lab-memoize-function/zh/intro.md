# 简介

在本实验中，我们将探索 JavaScript 中的记忆化（memoization）概念。记忆化是一种通过缓存昂贵函数调用的结果，并在再次出现相同输入时返回缓存结果来优化函数性能的技术。通过本实验，我们将学习如何使用 `Map` 对象在 JavaScript 中实现记忆化。
