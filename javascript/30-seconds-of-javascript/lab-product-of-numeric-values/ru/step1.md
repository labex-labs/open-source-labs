# Как вычислить произведение числовых значений в JavaScript

Для вычисления произведения двух или более чисел или массивов в JavaScript следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте метод `Array.prototype.reduce()`, чтобы умножать каждое значение на накопитель, который должен быть инициализирован значением `1`.
3. Определите функцию под названием `prod`, которая принимает любое количество аргументов с использованием оператора расширения (`...`). Внутри функции примените метод `reduce()` к массиву аргументов, полученному с помощью оператора расширения.
4. Функция возвращает результат умножения.

Вот пример использования функции `prod`:

```js
const prod = (...arr) => [...arr].reduce((acc, val) => acc * val, 1);

prod(1, 2, 3, 4); // 24
prod(...[1, 2, 3, 4]); // 24
```

В приведенном выше примере `prod(1, 2, 3, 4)` и `prod(...[1, 2, 3, 4])` возвращают оба `24`.
