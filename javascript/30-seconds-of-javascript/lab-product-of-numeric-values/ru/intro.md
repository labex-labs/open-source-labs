# Введение

В этом практическом занятии мы изучим, как вычислять произведение двух или более чисел/массивов с использованием JavaScript. Мы будем использовать метод `Array.prototype.reduce()`, чтобы умножать каждое значение на накопитель, который инициализируется значением `1`. В конце этого практического занятия вы будете лучше понимать, как использовать метод reduce для решения задач, связанных с умножением значений.
