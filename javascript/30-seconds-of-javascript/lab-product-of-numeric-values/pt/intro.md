# Introdução

Neste laboratório, exploraremos como calcular o produto de dois ou mais números/arrays usando JavaScript. Faremos uso do método `Array.prototype.reduce()` para multiplicar cada valor com um acumulador, que é inicializado com o valor `1`. Ao final deste laboratório, você terá uma melhor compreensão de como usar o método reduce para resolver problemas envolvendo a multiplicação de valores.
