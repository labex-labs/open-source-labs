# Introduction

Dans ce laboratoire, nous allons explorer comment calculer le produit de deux ou plusieurs nombres / tableaux à l'aide de JavaScript. Nous utiliserons la méthode `Array.prototype.reduce()` pour multiplier chaque valeur par un accumulateur, qui est initialisé avec une valeur de `1`. À la fin de ce laboratoire, vous aurez une meilleure compréhension de la manière d'utiliser la méthode reduce pour résoudre des problèmes impliquant la multiplication de valeurs.
