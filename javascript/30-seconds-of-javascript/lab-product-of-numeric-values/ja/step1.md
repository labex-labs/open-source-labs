# JavaScript で数値の積を計算する方法

JavaScript で 2 つ以上の数値または配列の積を計算するには、次の手順に従います。

1. ターミナル/SSH を開き、コーディングを練習するために `node` と入力します。
2. `Array.prototype.reduce()` メソッドを使って、各値を 1 で初期化されたアキュムレーターと乗算します。
3. スプレッド演算子 (`...`) を使って任意の数の引数を受け取る `prod` という関数を定義します。関数の中で、引数のスプレッド配列に `reduce()` メソッドを適用します。
4. 関数は乗算の結果を返します。

`prod` 関数を使う方法の例を次に示します。

```js
const prod = (...arr) => [...arr].reduce((acc, val) => acc * val, 1);

prod(1, 2, 3, 4); // 24
prod(...[1, 2, 3, 4]); // 24
```

上記の例では、`prod(1, 2, 3, 4)` と `prod(...[1, 2, 3, 4])` の両方が `24` を返します。
