# Introdução

Neste laboratório, trabalharemos em um exercício de programação JavaScript que se concentra no mapeamento dos valores de um objeto usando uma função fornecida. O objetivo deste exercício é ajudá-lo a obter uma melhor compreensão de como usar `Object.keys()` e `Array.prototype.reduce()` para criar um novo objeto com as mesmas chaves e valores mapeados. Ao final deste laboratório, você será capaz de aplicar este conhecimento para resolver problemas mais complexos em seus próprios projetos.
