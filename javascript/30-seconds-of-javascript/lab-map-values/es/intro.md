# Introducción

En este laboratorio, trabajaremos en un ejercicio de programación en JavaScript que se centra en mapear los valores de un objeto utilizando una función proporcionada. El propósito de este ejercicio es ayudarte a comprender mejor cómo utilizar `Object.keys()` y `Array.prototype.reduce()` para crear un nuevo objeto con las mismas claves y valores mapeados. Al final de este laboratorio, serás capaz de aplicar este conocimiento para resolver problemas más complejos en tus propios proyectos.
