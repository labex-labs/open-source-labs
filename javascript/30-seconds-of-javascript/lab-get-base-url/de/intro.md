# Einführung

In diesem Lab werden wir die Grundlagen der Programmiersprache JavaScript erkunden. Sie werden lernen, wie man die grundlegende Syntax schreibt, Datentypen manipuliert, bedingte Anweisungen, Schleifen und Funktionen verwendet. Am Ende dieses Labs werden Sie einen soliden Grundlagenwissen in JavaScript haben und einfache Programme erstellen können.
