# Introdução

Neste laboratório, exploraremos como ordenar caracteres em uma string em ordem alfabética usando JavaScript. Usaremos o operador spread (spread operator), `Array.prototype.sort()` e `String.prototype.localeCompare()` para alcançar este objetivo. Ao final do laboratório, você terá uma boa compreensão de como manipular strings em JavaScript usando esses métodos.
