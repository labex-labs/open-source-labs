# Introduction

Dans ce laboratoire, nous allons explorer la manière de trier les caractères d'une chaîne de caractères par ordre alphabétique à l'aide de JavaScript. Nous utiliserons l'opérateur de propagation, `Array.prototype.sort()` et `String.prototype.localeCompare()` pour y parvenir. À la fin du laboratoire, vous aurez une bonne compréhension de la manière de manipuler les chaînes de caractères en JavaScript à l'aide de ces méthodes.
