# Введение

В этом практическом занятии мы изучим, как сортировать символы в строке в алфавитном порядке с использованием JavaScript. Для этого мы будем использовать оператор расширения, `Array.prototype.sort()` и `String.prototype.localeCompare()`. В конце практического занятия вы хорошо освоите методы манипуляции строками в JavaScript.
