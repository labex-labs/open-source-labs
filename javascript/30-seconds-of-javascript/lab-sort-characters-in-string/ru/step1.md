# Вот, как сортировать символы в строке:

Используйте следующий код, чтобы отсортировать символы в строке в алфавитном порядке:

```js
const sortCharactersInString = (str) =>
  [...str].sort((a, b) => a.localeCompare(b)).join("");
```

Для начала откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.

Пример использования:

```js
sortCharactersInString("cabbage"); // 'aabbceg'
```
