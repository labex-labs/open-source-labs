# JavaScript-функция для убирания заглавной буквы в начале строки

Для убирания заглавной буквы в начале строки используйте следующую JavaScript-функцию:

```js
const decapitalize = ([first, ...rest], upperRest = false) => {
  return (
    first.toLowerCase() +
    (upperRest ? rest.join("").toUpperCase() : rest.join(""))
  );
};
```

Для использования этой функции откройте Терминал/SSH и введите `node`. Затем вызовите функцию `decapitalize`, передав в качестве первого аргумента строку, в которой вы хотите убрать заглавную букву в начале.

По желанию вы можете установить второй аргумент `upperRest` в значение `true`, чтобы преобразовать остальную часть строки в верхний регистр. Если аргумент `upperRest` не указан, по умолчанию он равен `false`.

Ниже приведены некоторые примеры:

```js
decapitalize("FooBar"); // 'fooBar'
decapitalize("FooBar", true); // 'fOOBAR'
```
