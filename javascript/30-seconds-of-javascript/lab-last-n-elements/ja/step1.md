# JavaScript で配列の最後の N 要素を取得する方法

JavaScript で配列の最後の `n` 要素を取得するには、次の手順に従います。

1. ターミナル/SSH を開き、コーディングを練習するために `node` と入力します。

2. 配列の最後の `n` 要素を取得するには、開始値を `-n` として `Array.prototype.slice()` を使用します。

配列の最後の `n` 要素を取得する JavaScript コードは次のとおりです。

```js
const lastN = (arr, n) => arr.slice(-n);
```

コードをテストするには、配列と取得したい要素数を指定して `lastN()` 関数を呼び出します。例えば：

```js
lastN(["a", "b", "c", "d"], 2); // ['c', 'd']
```
