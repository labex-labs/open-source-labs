# Introducción

En este laboratorio, exploraremos la función `lastN` que se utiliza para extraer los últimos `n` elementos de una matriz. Aprenderemos cómo utilizar el método `Array.prototype.slice()` para lograr esto y veremos cómo se puede implementar en nuestros programas de JavaScript. Al final de este laboratorio, tendrás una mejor comprensión de cómo manipular matrices en JavaScript y extraer elementos específicos de ellas.
