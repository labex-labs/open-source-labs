# Einführung

In diesem Lab werden wir die `lastN`-Funktion untersuchen, die verwendet wird, um die letzten `n` Elemente eines Arrays zu extrahieren. Wir werden lernen, wie die `Array.prototype.slice()`-Methode dazu verwendet werden kann und wie sie in unseren JavaScript-Programmen implementiert werden kann. Am Ende dieses Labs werden Sie ein besseres Verständnis dafür haben, wie Sie in JavaScript Arrays manipulieren und bestimmte Elemente daraus extrahieren.
