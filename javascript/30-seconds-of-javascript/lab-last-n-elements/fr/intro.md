# Introduction

Dans ce laboratoire, nous allons explorer la fonction `lastN` qui est utilisée pour extraire les derniers `n` éléments d'un tableau. Nous allons apprendre à utiliser la méthode `Array.prototype.slice()` pour y parvenir et voir comment elle peut être implémentée dans nos programmes JavaScript. À la fin de ce laboratoire, vous aurez une meilleure compréhension de la manière de manipuler les tableaux en JavaScript et d'en extraire des éléments spécifiques.
