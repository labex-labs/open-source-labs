# Introdução

Neste laboratório, exploraremos a função `lastN`, que é usada para extrair os últimos `n` elementos de um array. Aprenderemos como usar o método `Array.prototype.slice()` para alcançar este objetivo e veremos como ele pode ser implementado em nossos programas JavaScript. Ao final deste laboratório, você terá uma melhor compreensão de como manipular arrays em JavaScript e extrair elementos específicos deles.
