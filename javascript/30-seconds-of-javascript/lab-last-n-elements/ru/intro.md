# Введение

В этом лабе мы исследуем функцию `lastN`, которая используется для извлечения последних `n` элементов массива. Мы узнаем, как использовать метод `Array.prototype.slice()` для этого и увидим, как его можно реализовать в наших JavaScript-программах. В конце этого лабе вы будете лучше понимать, как манипулировать массивами в JavaScript и извлекать из них определенные элементы.
