# Как получить последние n элементов массива в JavaScript

Чтобы получить последние n элементов массива в JavaScript, следуйте этим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковать программирование.

2. Используйте `Array.prototype.slice()` со значением начала `-n`, чтобы получить последние n элементов массива.

Вот JavaScript-код для получения последних n элементов массива:

```js
const lastN = (arr, n) => arr.slice(-n);
```

Чтобы протестировать код, вызовите функцию `lastN()` с массивом и количеством элементов, которое вы хотите получить, вот так:

```js
lastN(["a", "b", "c", "d"], 2); // ['c', 'd']
```
