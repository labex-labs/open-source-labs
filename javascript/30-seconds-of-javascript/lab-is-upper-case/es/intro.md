# Introducción

En este laboratorio, exploraremos cómo comprobar si una cadena dada está en mayúsculas utilizando JavaScript. Crearemos una función que convierte la cadena a mayúsculas utilizando el método `String.prototype.toUpperCase()`, y luego la compararemos con la cadena original. Al final de este laboratorio, tendrás una mejor comprensión de la manipulación de cadenas en JavaScript y cómo comprobar cadenas en mayúsculas.
