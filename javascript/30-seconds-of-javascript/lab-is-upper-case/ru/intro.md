# Введение

В этом практическом занятии мы изучим, как проверить, является ли заданная строка полностью заглавными, используя JavaScript. Мы создадим функцию, которая преобразует строку в верхний регистр с использованием метода `String.prototype.toUpperCase()`, а затем сравним ее с исходной строкой. В конце этого практического занятия у вас будет лучше понимание манипуляций со строками в JavaScript и способов проверки строк на наличие только заглавных букв.
