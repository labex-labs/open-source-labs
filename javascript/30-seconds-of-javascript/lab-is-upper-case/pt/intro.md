# Introdução

Neste laboratório, exploraremos como verificar se uma determinada string está em maiúsculas usando JavaScript. Criaremos uma função que converte a string para maiúsculas usando o método `String.prototype.toUpperCase()`, e então a compararemos com a string original. Ao final deste laboratório, você terá uma melhor compreensão da manipulação de strings em JavaScript e de como verificar strings em maiúsculas.
