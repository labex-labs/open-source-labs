# Einführung

In diesem Lab werden wir untersuchen, wie man mithilfe von JavaScript überprüft, ob ein gegebener String in Großbuchstaben geschrieben ist. Wir werden eine Funktion erstellen, die den String mithilfe der `String.prototype.toUpperCase()`-Methode in Großbuchstaben umwandelt und ihn anschließend mit dem ursprünglichen String vergleicht. Am Ende dieses Labs werden Sie ein besseres Verständnis von der Zeichenkettenmanipulation in JavaScript und davon haben, wie man auf Strings in Großbuchstaben prüft.
