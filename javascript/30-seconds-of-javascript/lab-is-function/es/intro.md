# Introducción

En este laboratorio, exploraremos el concepto de funciones en JavaScript. Aprenderemos cómo definir y llamar a funciones, pasar argumentos y devolver valores. Además, practicaremos el uso de la función `isFunction()` para comprobar si un valor dado es una función. Al final de este laboratorio, tendrás una comprensión sólida de cómo funcionan las funciones en JavaScript y cómo utilizarlas efectivamente en tu código.
