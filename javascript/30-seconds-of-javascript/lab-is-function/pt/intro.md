# Introdução

Neste laboratório, exploraremos o conceito de funções em JavaScript. Aprenderemos como definir e chamar funções, passar argumentos e retornar valores. Adicionalmente, praticaremos o uso da função `isFunction()` para verificar se um determinado valor é uma função. Ao final deste laboratório, você terá uma sólida compreensão de como as funções funcionam em JavaScript e como usá-las efetivamente em seu código.
