# Проверка, является ли значение функцией

Для проверки того, является ли значение функцией, вы можете использовать оператор `typeof` с примитивом `function`.

Вот пример функции, которая проверяет, является ли заданное значение функцией:

```js
const isFunction = (val) => typeof val === "function";
```

Вы можете использовать ее так:

```js
isFunction("x"); // false
isFunction((x) => x); // true
```
