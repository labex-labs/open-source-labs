# Introdução

Neste laboratório, exploraremos como encontrar o elemento mais frequente em um array usando JavaScript. Usaremos o método `reduce()` para mapear valores únicos para as chaves de um objeto e, em seguida, usaremos `Object.entries()` e `reduce()` para determinar o valor mais frequente no array. Ao final deste laboratório, você terá uma compreensão sólida de como encontrar eficientemente o elemento mais frequente em um array.
