# Introduction

Dans ce laboratoire, nous allons explorer le concept d'ajout de caractères pour transformer un nombre donné en une chaîne de caractères d'une longueur spécifiée en utilisant JavaScript. Nous utiliserons la méthode `String.prototype.padStart()` pour ajouter des caractères au nombre et le convertir en chaîne de caractères. Grâce à ce laboratoire, vous comprendrez mieux comment manipuler les chaînes de caractères et les nombres en JavaScript.
