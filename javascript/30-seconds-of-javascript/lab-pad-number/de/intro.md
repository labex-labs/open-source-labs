# Einführung

In diesem Lab werden wir das Konzept des Auffüllens einer gegebenen Zahl auf eine bestimmte Länge mit JavaScript erkunden. Wir werden die `String.prototype.padStart()`-Methode verwenden, um die Zahl aufzufüllen und sie in einen String umzuwandeln. Durch dieses Lab werden Sie ein besseres Verständnis dafür erlangen, wie Sie in JavaScript Strings und Zahlen manipulieren.
