# Как дополнить число в JavaScript

Чтобы дополнить число в JavaScript, следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте метод `String.prototype.padStart()`, чтобы дополнить число до заданной длины, предварительно преобразовав его в строку.
3. Функция `padNumber()` ниже демонстрирует этот подход.
4. Передайте число и желаемую длину в качестве аргументов функции.
5. Функция возвращает дополненное число в виде строки.

```js
const padNumber = (n, l) => `${n}`.padStart(l, "0");
```

Пример использования:

```js
padNumber(1234, 6); // '001234'
```
