# JavaScript における数値のパディング方法

JavaScript において数値をパディングするには、次の手順に従います。

1. ターミナル/SSH を開き、コーディングの練習を始めるために `node` と入力します。
2. 数値を文字列に変換した後、`String.prototype.padStart()` メソッドを使用して数値を指定された長さにパディングします。
3. 以下の `padNumber()` 関数はこのアプローチを示しています。
4. 数値と望ましい長さを引数として関数に渡します。
5. 関数はパディングされた数値を文字列として返します。

```js
const padNumber = (n, l) => `${n}`.padStart(l, "0");
```

使用例：

```js
padNumber(1234, 6); // '001234'
```
