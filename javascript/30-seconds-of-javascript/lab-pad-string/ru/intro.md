# Введение

В этом лабе мы будем изучать функцию `pad()` в JavaScript. Функция `pad()` позволяет дополнять строку с обеих сторон определенным символом, чтобы достичь определенной длины. Мы узнаем, как использовать методы `String.prototype.padStart()` и `String.prototype.padEnd()`, чтобы это сделать.
