# Функция для дополнения строки

Чтобы дополнить строку с обеих сторон определенным символом, если она короче указанной `длины`, используйте следующую функцию:

```js
const pad = (str, length, char = " ") =>
  str.padStart((str.length + length) / 2, char).padEnd(length, char);
```

Функция использует `String.prototype.padStart()` и `String.prototype.padEnd()`, чтобы дополнить обе стороны заданной строки. Вы можете опустить третий аргумент, `char`, чтобы использовать символ пробела в качестве символа дополнения по умолчанию.

Вот несколько примеров использования функции `pad()`:

```js
pad("cat", 8); // '  cat   '
pad(String(42), 6, "0"); // '004200'
pad("foobar", 3); // 'foobar'
```
