# Введение

В этом лабе мы будем изучать концепцию примитивных значений в JavaScript. Мы узнаем, как различать примитивные и не примитивные значения с использованием функции `isPrimitive()`, которая сравнивает переданное значение с объектом, созданным из него. Через серию упражнений мы углубим наше понимание этой фундаментальной концепции в JavaScript.
