# Проверка на примитивные значения

Для практики программирования откройте Терминал или SSH и введите `node`. После этого вы можете проверить, является ли значение примитивным, следуя следующим шагам:

1. Создайте объект из значения, которое вы хотите проверить, используя `Object(val)`.
2. Сравните созданный объект с исходным значением с использованием строгого оператора неравенства `!==`.
3. Если два значения не равны, исходное значение является примитивным.

Вот код для функции `isPrimitive`:

```js
const isPrimitive = (val) => Object(val) !== val;
```

Вы можете протестировать эту функцию с помощью следующих значений:

```js
isPrimitive(null); // true
isPrimitive(undefined); // true
isPrimitive(50); // true
isPrimitive("Hello!"); // true
isPrimitive(false); // true
isPrimitive(Symbol()); // true
isPrimitive([]); // false
isPrimitive({}); // false
```

Если значение, которое вы хотите проверить, является примитивным, функция вернет `true`. В противном случае она вернет `false`.
