# Introduction

Dans ce laboratoire, nous allons apprendre à inverser une chaîne de caractères à l'aide de JavaScript. Nous utiliserons l'opérateur de propagation et les méthodes `Array.prototype` pour inverser l'ordre des caractères dans une chaîne de caractères donnée. Ce laboratoire est conçu pour vous aider à comprendre les concepts fondamentaux de JavaScript et à améliorer vos compétences en résolution de problèmes.
