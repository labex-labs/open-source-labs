# Introdução

Neste laboratório, aprenderemos como inverter uma string usando JavaScript. Usaremos o operador spread (spread operator) e os métodos `Array.prototype` para inverter a ordem dos caracteres em uma string fornecida. Este laboratório foi projetado para ajudá-lo a entender os conceitos fundamentais do JavaScript e a aprimorar suas habilidades de resolução de problemas.
