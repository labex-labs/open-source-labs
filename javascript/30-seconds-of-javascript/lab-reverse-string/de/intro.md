# Einführung

In diesem Lab werden wir lernen, wie man eine Zeichenkette mit JavaScript umkehren kann. Wir werden den Spread-Operator und `Array.prototype`-Methoden verwenden, um die Reihenfolge der Zeichen in einer gegebenen Zeichenkette umzukehren. Dieses Lab ist dazu gedacht, Ihnen die grundlegenden Konzepte von JavaScript zu vermitteln und Ihre Problemlösefähigkeiten zu verbessern.
