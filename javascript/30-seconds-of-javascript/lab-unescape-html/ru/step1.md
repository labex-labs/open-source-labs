# Раскодирование HTML

Эта функция раскодирует закодированные HTML-символы. Чтобы использовать ее, следуйте шагам:

1. Откройте Терминал/SSH.
2. Введите `node`.
3. Скопируйте и вставьте следующий код:

```js
const unescapeHTML = (str) =>
  str.replace(
    /&amp;|&lt;|&gt;|&#39;|&quot;/g,
    (tag) =>
      ({
        "&amp;": "&",
        "&lt;": "<",
        "&gt;": ">",
        "&#39;": "'",
        "&quot;": '"'
      })[tag] || tag
  );
```

4. Вызовите функцию `unescapeHTML` и передайте ей строку с закодированными символами.
5. Функция вернет раскодированную строку.

Пример использования:

```js
unescapeHTML("&lt;a href=&quot;#&quot;&gt;Me &amp; you&lt;/a&gt;");
// '<a href="#">Me & you</a>'
```
