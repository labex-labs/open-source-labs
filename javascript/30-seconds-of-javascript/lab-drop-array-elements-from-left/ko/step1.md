# 왼쪽에서 배열 요소 제거하기

코딩 연습을 시작하려면 터미널/SSH 를 열고 `node`를 입력하세요.

다음은 왼쪽에서 지정된 수의 요소를 제거하여 새로운 배열을 생성하는 함수입니다.

```js
const drop = (arr, n = 1) => arr.slice(n);
```

이 함수는 `Array.prototype.slice()`를 사용하여 왼쪽에서 지정된 수의 요소를 제거합니다. 마지막 인수인 `n`을 생략하면 함수는 기본값 `1`을 사용합니다.

다음은 `drop` 함수를 사용하는 몇 가지 예입니다.

```js
drop([1, 2, 3]); // [2, 3]
drop([1, 2, 3], 2); // [3]
drop([1, 2, 3], 42); // []
```
