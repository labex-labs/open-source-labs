# 配列の左側の要素を削除する

コーディングを練習するには、ターミナル/SSH を開いて `node` と入力します。

以下は、左側から指定された数の要素を削除した新しい配列を作成する関数です。

```js
const drop = (arr, n = 1) => arr.slice(n);
```

この関数は、`Array.prototype.slice()` を使って左側から指定された数の要素を削除します。最後の引数 `n` を省略すると、関数はデフォルト値 `1` を使用します。

以下は、`drop` 関数を使用するいくつかの例です。

```js
drop([1, 2, 3]); // [2, 3]
drop([1, 2, 3], 2); // [3]
drop([1, 2, 3], 42); // []
```
