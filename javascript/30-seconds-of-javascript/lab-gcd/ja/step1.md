# 最大公約数を計算する方法

コードを使って 2 つ以上の数または配列の最大公約数を計算するには、次の手順に従います。

1. ターミナル/SSH を開き、コーディングを練習するために `node` と入力します。

2. 次のコードを使用します。

```js
const gcd = (...arr) => {
  const _gcd = (x, y) => (!y ? x : gcd(y, x % y));
  return [...arr].reduce((a, b) => _gcd(a, b));
};
```

3. `gcd` 関数は再帰を使用しています。

4. ベースケースは `y` が `0` に等しい場合です。この場合、関数は `x` を返します。

5. それ以外の場合、関数は `y` と `x / y` の除算の余りの最大公約数を返します。

6. 関数をテストするには、次のコードを使用します。

```js
gcd(8, 36); // 4
gcd(...[12, 8, 32]); // 4
```
