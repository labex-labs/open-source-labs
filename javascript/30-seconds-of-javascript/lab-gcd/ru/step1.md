# Как вычислить наибольший общий делитель

Для вычисления наибольшего общего делителя между двумя или более числами/массивами с использованием кода, следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.

2. Используйте следующий код:

```js
const gcd = (...arr) => {
  const _gcd = (x, y) => (!y ? x : gcd(y, x % y));
  return [...arr].reduce((a, b) => _gcd(a, b));
};
```

3. Функция `gcd` использует рекурсию.

4. Базовым случаем является когда `y` равно `0`. В этом случае функция возвращает `x`.

5. В противном случае функция возвращает НОД `y` и остаток от деления `x / y`.

6. Чтобы протестировать функцию, используйте следующий код:

```js
gcd(8, 36); // 4
gcd(...[12, 8, 32]); // 4
```
