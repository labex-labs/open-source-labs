# Conversion des miles en kilomètres

Pour convertir les miles en kilomètres, suivez les étapes suivantes :

1. Ouvrez le Terminal/SSH et tapez `node` pour commencer à pratiquer la programmation.
2. Utilisez la formule de conversion `km = mi * 1.609344`.
3. Implémentez la formule à l'aide du code JavaScript ci-dessous :

```js
const milesToKm = (miles) => miles * 1.609344;
```

4. Testez la fonction en l'appelant avec le nombre de miles souhaité, comme ceci :

```js
milesToKm(5); // ~8.04672
```

Cela renverra la distance équivalente en kilomètres.
