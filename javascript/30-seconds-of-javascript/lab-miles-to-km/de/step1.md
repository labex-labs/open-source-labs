# Umrechnung von Meilen in Kilometer

Um Meilen in Kilometer umzurechnen, folgen Sie diesen Schritten:

1. Öffnen Sie das Terminal/SSH und geben Sie `node` ein, um mit der Programmierung zu beginnen.
2. Verwenden Sie die Umrechnungsformel `km = mi * 1.609344`.
3. Implementieren Sie die Formel mit dem folgenden JavaScript-Code:

```js
const milesToKm = (miles) => miles * 1.609344;
```

4. Testen Sie die Funktion, indem Sie sie mit der gewünschten Anzahl von Meilen aufrufen, wie folgt:

```js
milesToKm(5); // ~8.04672
```

Dies wird die entsprechende Entfernung in Kilometern zurückgeben.
