# Einführung

In diesem Lab werden wir untersuchen, wie man Meilen in Kilometer umrechnet, indem man JavaScript verwendet. Das Ziel dieses Labs ist es, Ihnen die Umrechnungsformel zu erklären und anwendbar zu machen. Am Ende dieses Labs sollten Sie in der Lage sein, eine Funktion zu schreiben, die Meilen in Kilometer umrechnet, was in einer Vielzahl von Anwendungen nützlich sein kann.
