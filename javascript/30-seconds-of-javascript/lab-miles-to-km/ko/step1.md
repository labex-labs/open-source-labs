# 마일을 킬로미터로 변환하기

마일을 킬로미터로 변환하려면 다음 단계를 따르세요.

1. 터미널/SSH 를 열고 `node`를 입력하여 코딩 연습을 시작합니다.
2. 변환 공식 `km = mi * 1.609344`를 사용합니다.
3. 아래 JavaScript 코드를 사용하여 공식을 구현합니다.

```js
const milesToKm = (miles) => miles * 1.609344;
```

4. 원하는 마일 수로 함수를 호출하여 테스트합니다. 예:

```js
milesToKm(5); // ~8.04672
```

이렇게 하면 킬로미터 단위의 해당 거리가 반환됩니다.
