# Преобразование миль в километры

Для преобразования миль в километры следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте формулу преобразования `км = мили * 1,609344`.
3. Реализуйте формулу с использованием JavaScript кода ниже:

```js
const milesToKm = (miles) => miles * 1.609344;
```

4. Протестируйте функцию, вызвав ее с нужным количеством миль, как показано ниже:

```js
milesToKm(5); // ~8.04672
```

Это вернет эквивалентное расстояние в километрах.
