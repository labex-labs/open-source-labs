# Conversão de Milhas para Quilômetros

Para converter milhas em quilômetros, siga estes passos:

1.  Abra o Terminal/SSH e digite `node` para começar a praticar a codificação.
2.  Use a fórmula de conversão `km = mi * 1.609344`.
3.  Implemente a fórmula usando o código JavaScript abaixo:

```js
const milesToKm = (miles) => miles * 1.609344;
```

4.  Teste a função chamando-a com o número desejado de milhas, assim:

```js
milesToKm(5); // ~8.04672
```

Isso retornará a distância equivalente em quilômetros.
