# Conversión de millas a kilómetros

Para convertir millas a kilómetros, sigue estos pasos:

1. Abre la Terminal/SSH y escribe `node` para comenzar a practicar la codificación.
2. Utiliza la fórmula de conversión `km = mi * 1.609344`.
3. Implementa la fórmula utilizando el código de JavaScript que se muestra a continuación:

```js
const milesToKm = (miles) => miles * 1.609344;
```

4. Prueba la función llamándola con el número deseado de millas, como se muestra a continuación:

```js
milesToKm(5); // ~8.04672
```

Esto devolverá la distancia equivalente en kilómetros.
