# Introducción

En este laboratorio, exploraremos cómo convertir millas a kilómetros utilizando JavaScript. El propósito de este laboratorio es ayudarte a entender la fórmula de conversión y aplicarla a un escenario del mundo real. Al final de este laboratorio, serás capaz de escribir una función que convierta millas a kilómetros, lo cual puede ser útil en una variedad de aplicaciones.
