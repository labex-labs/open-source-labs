# Verificar se um Número é Ímpar

Para praticar a codificação, abra o Terminal/SSH e digite `node`. Use o operador módulo (`%`) para verificar se um número é ímpar ou par. Se o resultado for 1, o número é ímpar, e se o resultado for 0, o número é par. Retorne `true` se o número for ímpar e `false` se o número for par.

```js
const isOdd = (num) => num % 2 === 1;
```

Exemplo de uso:

```js
isOdd(3); // true
```
