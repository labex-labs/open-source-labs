# Introdução

Neste laboratório, exploraremos o conceito de determinar se um número dado é ímpar ou par em JavaScript. Usaremos o operador módulo para verificar o resto quando o número é dividido por 2. Este laboratório ajudará você a entender os fundamentos dos operadores aritméticos em JavaScript e como usá-los para escrever programas simples.
