# Введение

В этом практическом занятии мы изучим концепцию определения, является ли заданное число нечетным или четным, на JavaScript. Мы будем использовать оператор модуля (%) для проверки остатка при делении числа на 2. Это практическое занятие поможет вам понять основы арифметических операторов в JavaScript и как ими пользоваться для написания простых программ.
