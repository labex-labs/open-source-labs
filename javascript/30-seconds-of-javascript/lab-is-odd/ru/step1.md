# Проверить, является ли число нечетным

Для практики программирования откройте Терминал/SSH и введите `node`. Используйте оператор модуля (`%`), чтобы проверить, является ли число нечетным или четным. Если результат равен 1, число нечетное, а если результат равен 0, число четное. Возвращайте `true`, если число нечетное, и `false`, если число четное.

```js
const isOdd = (num) => num % 2 === 1;
```

Пример использования:

```js
isOdd(3); // true
```
