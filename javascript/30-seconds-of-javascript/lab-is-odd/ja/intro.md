# はじめに

この実験では、JavaScript で与えられた数が奇数か偶数かを判断する概念を探ります。数を 2 で割ったときの余りを確認するために剰余演算子を使用します。この実験は、JavaScript の算術演算子の基本と、それらを使って簡単なプログラムを書く方法を理解するのに役立ちます。
