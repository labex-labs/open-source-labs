# Einführung

In diesem Lab werden wir das Konzept des Bestimmens, ob eine gegebene Zahl ungerade oder gerade ist, in JavaScript erkunden. Wir werden den Modulo-Operator verwenden, um den Rest zu überprüfen, wenn die Zahl durch 2 geteilt wird. Dieses Lab wird Ihnen helfen, die Grundlagen der arithmetischen Operatoren in JavaScript zu verstehen und wie Sie sie verwenden, um einfache Programme zu schreiben.
