# Überprüfen, ob eine Zahl ungerade ist

Um das Programmieren zu üben, öffnen Sie das Terminal/SSH und geben Sie `node` ein. Verwenden Sie den Modulo- (`%`) Operator, um zu überprüfen, ob eine Zahl ungerade oder gerade ist. Wenn das Ergebnis 1 ist, ist die Zahl ungerade, und wenn das Ergebnis 0 ist, ist die Zahl gerade. Geben Sie `true` zurück, wenn die Zahl ungerade ist, und `false`, wenn die Zahl gerade ist.

```js
const isOdd = (num) => num % 2 === 1;
```

Beispielverwendung:

```js
isOdd(3); // true
```
