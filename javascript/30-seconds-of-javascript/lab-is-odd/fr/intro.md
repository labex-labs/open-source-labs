# Introduction

Dans ce laboratoire, nous allons explorer le concept de détermination si un nombre donné est impair ou pair en JavaScript. Nous utiliserons l'opérateur modulo pour vérifier le reste lorsqu'un nombre est divisé par 2. Ce laboratoire vous aidera à comprendre les bases des opérateurs arithmétiques en JavaScript et à utiliser ces derniers pour écrire des programmes simples.
