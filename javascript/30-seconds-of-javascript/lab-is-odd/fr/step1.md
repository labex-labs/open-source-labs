# Vérifier si un nombre est impair

Pour pratiquer la programmation, ouvrez le Terminal/SSH et tapez `node`. Utilisez l'opérateur modulo (`%`) pour vérifier si un nombre est impair ou pair. Si le résultat est 1, le nombre est impair, et si le résultat est 0, le nombre est pair. Retournez `true` si le nombre est impair et `false` si le nombre est pair.

```js
const isOdd = (num) => num % 2 === 1;
```

Utilisation exemple :

```js
isOdd(3); // true
```
