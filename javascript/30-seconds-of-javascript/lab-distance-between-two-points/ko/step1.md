# 두 점 사이의 거리 계산

두 점 사이의 거리를 계산하려면 다음 단계를 따르세요.

1. 터미널/SSH 를 열고 `node`를 입력하여 코딩 연습을 시작합니다.
2. `Math.hypot()`을 사용하여 두 점 사이의 유클리드 거리 (Euclidean distance) 를 계산합니다.
3. 아래 코드를 구현하고, `x0`, `y0`, `x1`, `y1` 값을 여러분의 점 좌표로 바꿉니다.

```js
const distance = (x0, y0, x1, y1) => Math.hypot(x1 - x0, y1 - y0);
```

이 함수를 사용하는 방법의 예는 다음과 같습니다.

```js
distance(1, 1, 2, 3); // ~2.2361
```

이는 점 `(1, 1)`과 `(2, 3)` 사이의 거리를 출력합니다.
