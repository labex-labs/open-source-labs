# Introdução

Neste laboratório, exploraremos como verificar se um determinado argumento é um fluxo gravável (writable stream) em JavaScript. Usaremos uma função que verifica se o valor é diferente de `null`, se possui uma propriedade `pipe` do tipo `function`, e se também possui as propriedades `_write` e `_writableState` do tipo `function` e `object`, respectivamente. Através deste laboratório, obteremos uma melhor compreensão de como trabalhar com fluxos (streams) em JavaScript.
