# Utilisation de l'opérateur logique OU en JavaScript

Pour commencer la pratique de codage, ouvrez le Terminal/SSH et entrez `node`. L'opérateur logique OU (`||`) vérifie si au moins l'un des arguments donnés est `true`.

Voici un exemple d'utilisation de l'opérateur logique OU :

```js
const or = (a, b) => a || b;
```

Et voici quelques exemples de sortie lors de l'utilisation de l'opérateur :

```js
or(true, true); // true
or(true, false); // true
or(false, false); // false
```
