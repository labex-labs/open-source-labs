# Introducción

En este laboratorio, exploraremos el operador lógico o (`||`) en JavaScript. El objetivo de este laboratorio es entender cómo funciona el operador lógico o y cómo se puede utilizar para comprobar si al menos uno de los argumentos dados es `true`. A través de ejemplos prácticos, aprenderemos cómo implementar el operador lógico o en nuestro código.
