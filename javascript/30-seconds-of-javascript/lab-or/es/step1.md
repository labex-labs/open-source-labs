# Usando el operador lógico O en JavaScript

Para comenzar la práctica de codificación, abre la Terminal/SSH y escribe `node`. El operador lógico o (`||`) comprueba si al menos uno de los argumentos dados es `true`.

Aquí hay un ejemplo de uso del operador lógico o:

```js
const or = (a, b) => a || b;
```

Y aquí hay algunos ejemplos de la salida al usar el operador:

```js
or(true, true); // true
or(true, false); // true
or(false, false); // false
```
