# Использование логического оператора "или" в JavaScript

Для начала практики программирования откройте Терминал/SSH и введите `node`. Логический оператор "или" (`||`) проверяет, является ли хотя бы один из заданных аргументов истинным (`true`).

Вот пример использования логического оператора "или":

```js
const or = (a, b) => a || b;
```

Вот несколько примеров вывода при использовании оператора:

```js
or(true, true); // true
or(true, false); // true
or(false, false); // false
```
