# Usando o Operador Lógico OR em JavaScript

Para começar a praticar a codificação, abra o Terminal/SSH e digite `node`. O operador lógico OR (`||`) verifica se pelo menos um dos argumentos fornecidos é `true`.

Aqui está um exemplo de como usar o operador lógico OR:

```js
const or = (a, b) => a || b;
```

E aqui estão alguns exemplos da saída ao usar o operador:

```js
or(true, true); // true
or(true, false); // true
or(false, false); // false
```
