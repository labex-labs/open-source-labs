# Einführung

In diesem Lab werden wir den logischen Oder (`||`) -Operator in JavaScript erkunden. Ziel dieses Labs ist es, zu verstehen, wie der logische Oder -Operator funktioniert und wie er verwendet werden kann, um zu überprüfen, ob mindestens ein der angegebenen Argumente `true` ist. Anhand von praktischen Beispielen lernen wir, wie wir den logischen Oder -Operator in unserem Code implementieren.
