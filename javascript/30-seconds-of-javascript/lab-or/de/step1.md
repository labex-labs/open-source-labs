# Das Verwenden des logischen Oder -Operators in JavaScript

Um mit der Code - Praxis zu beginnen, öffnen Sie das Terminal/SSH und geben Sie `node` ein. Der logische Oder -Operator (`||`) überprüft, ob mindestens eines der angegebenen Argumente `true` ist.

Hier ist ein Beispiel für das Verwenden des logischen Oder -Operators:

```js
const or = (a, b) => a || b;
```

Und hier sind einige Beispiele für die Ausgabe, wenn der Operator verwendet wird:

```js
or(true, true); // true
or(true, false); // true
or(false, false); // false
```
