# Introdução

Neste laboratório, aprenderemos como criar uma _promise_ (promessa) que resolve para um valor dado após um período de tempo especificado usando JavaScript. Usaremos o construtor `Promise` para criar uma nova _promise_ e o método `setTimeout()` para atrasar a resolução da _promise_. Esta habilidade é essencial ao trabalhar com operações assíncronas em JavaScript.
