# Введение

В этом практическом занятии мы научимся создавать промис, который будет разрешаться с заданным значением через определенное время, используя JavaScript. Мы будем использовать конструктор `Promise` для создания нового промиса и метод `setTimeout()` для задержки разрешения промиса. Это умение является важным при работе с асинхронными операциями в JavaScript.
