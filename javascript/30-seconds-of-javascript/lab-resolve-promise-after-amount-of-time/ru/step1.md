# Создание промиса с задержкой

Для создания нового промиса, который будет разрешаться через определенное время, следуйте этим шагам:

1. Используйте конструктор `Promise` для создания нового промиса.
2. Внутри функции исполнителя промиса используйте `setTimeout()` для вызова функции `resolve` промиса с предоставленным `value` после указанной `задержки`.

Вот пример реализации `resolveAfter()`:

```js
const resolveAfter = (value, delay) =>
  new Promise((resolve) => {
    setTimeout(() => resolve(value), delay);
  });
```

Теперь вы можете вызвать `resolveAfter()` для получения промиса, который будет разрешаться с предоставленным значением после указанной задержки:

```js
resolveAfter("Hello", 1000);
// Возвращает промис, который разрешится в 'Hello' через 1 секунду
```

Для начала практики по программированию откройте Терминал или SSH и введите `node`.
