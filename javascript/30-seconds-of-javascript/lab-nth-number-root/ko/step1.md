# 숫자의 N 번째 근 계산 방법

숫자의 N 번째 근을 계산하려면:

1. 터미널/SSH 를 열고 `node`를 입력하여 코딩 연습을 시작합니다.
2. `Math.pow(x, 1/n)` 공식을 사용하여 `x`의 `1/n` 제곱을 계산합니다.
3. 이 계산의 결과는 `x`의 N 번째 근과 같습니다.

다음은 예시 코드 조각입니다:

```js
const nthRoot = (x, n) => Math.pow(x, 1 / n);
nthRoot(32, 5); // Output: 2
```

이 코드는 32 의 N 번째 근 (여기서 n 은 5) 을 계산하고 결과를 2 로 반환합니다.
