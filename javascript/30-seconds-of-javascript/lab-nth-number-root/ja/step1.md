# 数値の n 乗根を計算する方法

数値の n 乗根を計算するには：

1. ターミナル/SSH を開き、コーディングの練習を始めるために `node` と入力します。
2. `Math.pow(x, 1/n)` の式を使用して、`x` の `1/n` 乗を計算します。
3. この計算の結果は、`x` の n 乗根に等しくなります。

以下はコードの例です：

```js
const nthRoot = (x, n) => Math.pow(x, 1 / n);
nthRoot(32, 5); // 出力：2
```

このコードは、32 の n 乗根（n は 5）を計算し、出力として 2 を返します。
