# Введение

В этом лабе мы будем изучать концепции программирования на JavaScript и применять их к решениям реальных задач. Цель этого лабара - помочь студентам создать твердый фундамент в JavaScript, научиться основным программировательским методам и получить практический опыт в решении задач по программированию. В конце этого лабара студенты будут иметь более глубокое понимание программирования на JavaScript и способность создавать функциональные приложения.
