# Как вычислить n-ую степень корня из числа

Для вычисления n-ой степени корня из числа:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в программировании.
2. Используйте формулу `Math.pow(x, 1/n)`, чтобы вычислить `x` в степени `1/n`.
3. Результат этого вычисления равен n-ой степени корня из `x`.

Вот пример кода:

```js
const nthRoot = (x, n) => Math.pow(x, 1 / n);
nthRoot(32, 5); // Output: 2
```

Этот код вычислит n-ую степень корня из 32 (где n равно 5) и вернет результат 2.
