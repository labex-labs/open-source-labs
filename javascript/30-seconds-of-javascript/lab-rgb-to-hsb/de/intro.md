# Einführung

In diesem Lab werden wir die Grundlagen der Programmiersprache JavaScript erkunden. Das Lab ist so konzipiert, dass die Teilnehmer*innen einen soliden Verständnis der Grundlagen von JavaScript erlangen, einschließlich Variablen, Datentypen, Steuerstrukturen, Funktionen und vieles mehr. Durch eine Reihe von praktischen Übungen und Beispielen lernen die Teilnehmer*innen, wie man in JavaScript Code schreibt und einfache Programme erstellt. Dieses Lab eignet sich für Einsteiger mit wenig oder gar keinem vorherigen Programmiererfahrung.
