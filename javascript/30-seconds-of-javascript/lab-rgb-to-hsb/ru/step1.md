# Преобразование RGB в HSB

Для преобразования кортежа цветов RGB в формат HSB вы можете следовать следующим шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте [формулу преобразования RGB в HSB](https://en.wikipedia.org/wiki/HSL_and_HSV#From_RGB), чтобы преобразовать кортеж цветов RGB в соответствующий формат HSB.
3. Диапазон входных параметров составляет [0, 255], в то время как результирующие значения имеют диапазон:

- H: [0, 360]
- S: [0, 100]
- B: [0, 100]

Вот функция на JavaScript:

```js
const RGBToHSB = (r, g, b) => {
  r /= 255;
  g /= 255;
  b /= 255;
  const v = Math.max(r, g, b),
    n = v - Math.min(r, g, b);
  const h =
    n === 0
      ? 0
      : n && v === r
        ? (g - b) / n
        : v === g
          ? 2 + (b - r) / n
          : 4 + (r - g) / n;
  return [60 * (h < 0 ? h + 6 : h), v && (n / v) * 100, v * 100];
};
```

Вы можете вызвать функцию так:

```js
RGBToHSB(252, 111, 48);
// [18.529411764705856, 80.95238095238095, 98.82352941176471]
```
