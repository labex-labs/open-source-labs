# Функция для преобразования чисел в порядковые суффиксы

Для преобразования числа в порядковый суффикс используйте функцию `toOrdinalSuffix`.

- Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
- Функция принимает число в качестве входных данных и возвращает его в виде строки с правильным порядковым индикатором суффикса.
- Используйте оператор модуля (`%`), чтобы найти значения единичных и десятковых разрядов.
- Найдите, с каким шаблоном порядкового числа соответствуют разряды.
- Если цифра находится в шаблоне чисел от 13 до 19, используйте соответствующее порядковое число.

```js
const toOrdinalSuffix = (num) => {
  const int = parseInt(num),
    digits = [int % 10, int % 100],
    ordinals = ["st", "nd", "rd", "th"],
    oPattern = [1, 2, 3, 4],
    tPattern = [11, 12, 13, 14, 15, 16, 17, 18, 19];
  return oPattern.includes(digits[0]) && !tPattern.includes(digits[1])
    ? int + ordinals[digits[0] - 1]
    : int + ordinals[3];
};
```

Вот пример использования функции `toOrdinalSuffix`:

```js
toOrdinalSuffix("123"); // '123rd'
```
