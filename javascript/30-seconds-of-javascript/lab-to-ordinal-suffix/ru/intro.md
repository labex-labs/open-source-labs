# Введение

В этом практическом занятии мы изучим функцию на JavaScript, которая принимает число и возвращает его в виде строки с правильным суффиксом порядкового индикатора. Функция будет использовать оператор модуля для нахождения значений единичных и десятковых разрядов, а затем определять, с каким шаблоном порядкового числа соответствуют эти разряды. Мы также будем обрабатывать особый случай шаблонов разрядов для чисел от 13 до 19 и использовать соответствующие порядковые числа для этих значений.
