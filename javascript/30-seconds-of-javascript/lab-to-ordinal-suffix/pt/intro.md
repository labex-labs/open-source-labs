# Introdução

Neste laboratório, exploraremos uma função JavaScript que recebe um número e o retorna como uma string com o sufixo ordinal correto. A função usará o operador módulo para encontrar os valores dos dígitos únicos e das dezenas, e então determinar qual padrão ordinal os dígitos correspondem. Também lidaremos com o caso especial dos dígitos do padrão "teens" e usaremos o ordinal apropriado para esses números.
