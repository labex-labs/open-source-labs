# Генерация массива случайных целых чисел в заданном диапазоне

Для генерации массива случайных целых чисел в заданном диапазоне следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте `Array.from()` для создания пустого массива заданной длины.
3. Используйте `Math.random()` для генерации случайных чисел и сопоставьте их с заданным диапазоном. Используйте `Math.floor()` для преобразования их в целые числа.
4. Функция `randomIntArrayInRange()` принимает три аргумента: `min`, `max` и необязательный аргумент `n` (значение по умолчанию равно 1).
5. Вызовите функцию `randomIntArrayInRange()` с заданными значениями `min`, `max` и `n`, чтобы сгенерировать массив случайных целых чисел.

Вот код:

```js
const randomIntArrayInRange = (min, max, n = 1) =>
  Array.from(
    { length: n },
    () => Math.floor(Math.random() * (max - min + 1)) + min
  );
```

Пример использования:

```js
randomIntArrayInRange(12, 35, 10); // [ 34, 14, 27, 17, 30, 27, 20, 26, 21, 14 ]
```
