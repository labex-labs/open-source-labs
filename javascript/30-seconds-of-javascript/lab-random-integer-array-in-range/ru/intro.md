# Введение

В этом практическом занятии мы изучим, как с помощью JavaScript генерировать массив целых чисел, выбранных случайным образом, в заданном диапазоне. Мы будем использовать метод `Array.from()` для создания пустого массива и заполнять его случайными целыми числами с использованием `Math.random()` и `Math.floor()`. В конце практического занятия вы глубоко освоите методы генерации случайных целых чисел в JavaScript и сможете применить эти знания в своих проектах.
