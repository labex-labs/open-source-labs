# Introducción

En este laboratorio, exploraremos cómo generar una matriz de enteros aleatorios en un rango específico utilizando JavaScript. Utilizaremos el método `Array.from()` para crear una matriz vacía y llenarla con enteros generados aleatoriamente utilizando `Math.random()` y `Math.floor()`. Al final de este laboratorio, tendrás una comprensión sólida de cómo generar enteros aleatorios en JavaScript y aplicar este conocimiento a tus propios proyectos.
