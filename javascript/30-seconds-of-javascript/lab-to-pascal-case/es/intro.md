# Introducción

En este laboratorio (lab), exploraremos cómo convertir una cadena de texto (string) a formato Pascal case utilizando JavaScript. Pascal case es una convención de nomenclatura en la que cada palabra en un identificador de varias palabras comienza con una letra mayúscula y las letras restantes son minúsculas. Este laboratorio proporcionará una guía paso a paso sobre cómo utilizar expresiones regulares y métodos de arrays para transformar una cadena de texto a formato Pascal case.
