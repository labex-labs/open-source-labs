# Einführung

In diesem Lab werden wir untersuchen, wie man mithilfe von JavaScript einen String in Pascal Case umwandelt. Pascal Case ist eine Benennungskonvention, bei der jedes Wort in einem mehrwortigen Bezeichner mit einem Großbuchstaben beginnt und die übrigen Buchstaben Kleinbuchstaben sind. In diesem Lab erhalten Sie Schritt-für-Schritt-Anleitungen dazu, wie Sie reguläre Ausdrücke und Array-Methoden verwenden, um einen String in Pascal Case zu transformieren.
