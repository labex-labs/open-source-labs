# Введение

В этом практическом занятии мы изучим, как разбить элементы массива на группы на основе заданной функции. Мы будем использовать функцию `unzipWith()`, которая принимает массив, полученный с помощью функции `zip()`, и применяет заданную функцию для разбивки элементов на группы. В конце этого практического занятия вы будете лучше понимать, как манипулировать массивами и выполнять пользовательские операции с их элементами.
