# Introdução

Neste laboratório, exploraremos como desagrupar elementos de um array com base em uma função fornecida. Usaremos a função `unzipWith()`, que recebe um array produzido pela função `zip()` e aplica a função fornecida para desagrupar os elementos. Ao final deste laboratório, você terá uma melhor compreensão de como manipular arrays e realizar operações personalizadas em seus elementos.
