# Вот как преобразовать объект в Map

Для преобразования объекта в `Map` следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Используйте `Object.entries()`, чтобы преобразовать объект в массив пар ключ-значение.
3. Используйте конструктор `Map`, чтобы преобразовать массив в `Map`.

Вот пример кода:

```js
const objectToMap = (obj) => new Map(Object.entries(obj));
```

Вы можете использовать функцию `objectToMap()` для преобразования объекта в `Map`. Например:

```js
objectToMap({ a: 1, b: 2 }); // Map {'a' => 1, 'b' => 2}
```
