# Introducción

En este laboratorio, exploraremos cómo determinar si un número dado es una potencia de `10` o no utilizando JavaScript. Utilizaremos la función `Math.log10()` y el operador módulo para comprobar si el número es una potencia de `10`. Al final de este laboratorio, tendrás una mejor comprensión de cómo trabajar con funciones logarítmicas y el operador módulo en JavaScript.
