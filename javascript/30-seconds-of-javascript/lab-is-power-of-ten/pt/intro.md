# Introdução

Neste laboratório, exploraremos como determinar se um determinado número é uma potência de `10` ou não, usando JavaScript. Usaremos a função `Math.log10()` e o operador módulo para verificar se o número é uma potência de `10`. Ao final deste laboratório, você terá uma melhor compreensão de como trabalhar com funções logarítmicas e o operador módulo em JavaScript.
