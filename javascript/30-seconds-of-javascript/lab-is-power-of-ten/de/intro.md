# Einführung

In diesem Lab werden wir untersuchen, wie man mit JavaScript bestimmt, ob eine gegebene Zahl eine Potenz von `10` ist oder nicht. Wir werden die `Math.log10()`-Funktion und den Modulo-Operator verwenden, um zu überprüfen, ob die Zahl eine Potenz von `10` ist. Am Ende dieses Labs werdet ihr ein besseres Verständnis dafür haben, wie man mit logarithmischen Funktionen und dem Modulo-Operator in JavaScript umgeht.
