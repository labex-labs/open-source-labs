# Функция для определения квартала года

Для определения квартала года используйте функцию `quarterOfYear()`. Эта функция принимает необязательный аргумент `date` и возвращает массив с кварталом и годом, к которому относится переданная дата.

Для использования этой функции откройте Терминал/SSH и введите `node`. Затем скопируйте и вставьте следующий код:

```js
const quarterOfYear = (date = new Date()) => [
  Math.ceil((date.getMonth() + 1) / 3),
  date.getFullYear()
];
```

Функция `quarterOfYear()` использует следующие шаги для вычисления квартала и года:

- Использует `Date.prototype.getMonth()`, чтобы получить текущий месяц в диапазоне (0, 11), добавляет `1`, чтобы сопоставить его с диапазоном (1, 12).
- Использует `Math.ceil()` и делит месяц на `3`, чтобы получить текущий квартал.
- Использует `Date.prototype.getFullYear()`, чтобы получить год из заданной `date`.
- Пропускает аргумент `date`, чтобы по умолчанию использовать текущую дату.

Вот несколько примеров использования функции `quarterOfYear()`:

```js
quarterOfYear(new Date("07/10/2018")); // [ 3, 2018 ]
quarterOfYear(); // [ 4, 2020 ]
```
