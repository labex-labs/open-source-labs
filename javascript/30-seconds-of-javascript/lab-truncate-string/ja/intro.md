# はじめに

この実験では、JavaScript における文字列を切り詰める概念を探ります。文字列を切り詰めるとは、与えられた文字列の長さを指定された長さまで短縮することを意味します。さまざまな演習と例を通じて、文字列を切り詰めて切り詰められた文字列の末尾に省略記号を追加する関数を実装する方法を学びます。
