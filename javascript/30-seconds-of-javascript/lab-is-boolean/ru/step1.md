# Проверка, является ли значение булевым

Для проверки, является ли значение примитивным булевым значением в JavaScript, используйте оператор `typeof` вместе с оператором сравнения `===`.

```js
const isBoolean = (val) => typeof val === "boolean";
```

Вот пример того, как использовать функцию `isBoolean()` для проверки, является ли значение булевым:

```js
isBoolean(null); // возвращает false
isBoolean(false); // возвращает true
```
