# Introdução

Neste laboratório, vamos mergulhar no conceito de valores booleanos em JavaScript. Exploraremos como verificar se um determinado valor é um primitivo booleano ou não, usando o operador `typeof`. Ao final deste laboratório, você terá uma sólida compreensão de como identificar valores booleanos em JavaScript.
