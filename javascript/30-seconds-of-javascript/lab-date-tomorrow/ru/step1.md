# Получение даты завтрашнего дня

Для практики программирования вы можете начать, открыв Терминал/SSH и введя `node`. После этого вы можете получить дату завтрашнего дня с помощью следующих шагов:

1. Используйте конструктор `Date`, чтобы получить текущую дату.
2. Увеличьте ее на единицу с использованием `Date.prototype.getDate()`.
3. Установите значение в результат с использованием `Date.prototype.setDate()`.
4. Используйте `Date.prototype.toISOString()`, чтобы вернуть строку в формате `yyyy-mm-dd`.

Вот код, который вы можете использовать:

```js
const tomorrow = () => {
  let currentDate = new Date();
  currentDate.setDate(currentDate.getDate() + 1);
  return currentDate.toISOString().split("T")[0];
};
```

После ввода этого кода вы можете получить дату завтрашнего дня, вызвав функцию `tomorrow()`. Например, если сегодняшняя дата 2018-10-18, вывод будет `2018-10-19`.
