# RGB から 16 進数に変換する

RGB 値を 16 進数の色コードに変換するには：

1. ターミナル/SSH を開き、コーディングを練習するために `node` と入力します。
2. 次の関数を使用します：

```js
const RGBToHex = (r, g, b) =>
  ((r << 16) + (g << 8) + b).toString(16).padStart(6, "0");
```

3. RGB 値を引数として関数を呼び出して、6 桁の 16 進数値を取得します。

例：

```js
RGBToHex(255, 165, 1); // 'ffa501'
```
