# Introdução

Neste laboratório, mergulharemos no mundo da programação JavaScript e exploraremos o conceito de distância vetorial. O objetivo do laboratório é ajudá-lo a entender como usar `Array.prototype.reduce()`, `Math.pow()` e `Math.sqrt()` para calcular a distância euclidiana entre dois vetores. Ao final do laboratório, você terá adquirido experiência prática na implementação desta função útil em JavaScript.
