# Введение

В этом лабе мы углубимся в мир программирования на JavaScript и исследуем концепцию расстояния между векторами. Цель лабы - помочь вам понять, как использовать `Array.prototype.reduce()`, `Math.pow()` и `Math.sqrt()` для вычисления евклидового расстояния между двумя векторами. В конце лабы вы получите практический опыт в реализации этой полезной функции на JavaScript.
