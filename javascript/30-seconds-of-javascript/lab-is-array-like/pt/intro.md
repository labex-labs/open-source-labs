# Introdução

Neste laboratório, exploraremos o conceito de objetos semelhantes a arrays (array-like objects) em JavaScript e aprenderemos como determinar se um objeto é iterável. Usaremos a função `isArrayLike` para verificar se um objeto possui um iterador e, portanto, pode ser iterado usando um loop `for...of`. Ao final deste laboratório, você terá uma melhor compreensão de como trabalhar com objetos semelhantes a arrays em JavaScript.
