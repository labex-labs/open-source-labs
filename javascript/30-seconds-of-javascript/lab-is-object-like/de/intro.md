# Einführung

In diesem Lab werden wir untersuchen, wie man mithilfe von JavaScript überprüft, ob ein Wert objektähnlich ist. Wir werden die `isObjectLike`-Funktion verwenden, um zu bestimmen, ob ein gegebener Wert ein Objekt oder ein Array ist, und wir werden auch lernen, wie man es von anderen Datentypen wie null oder Funktionen unterscheidet. Am Ende dieses Labs werden Sie ein besseres Verständnis von objektähnlichen Werten und wie man mit ihnen in Ihrem JavaScript-Code umgeht haben.
