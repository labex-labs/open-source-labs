# Introducción

En este laboratorio, exploraremos cómo comprobar si un valor es similar a un objeto utilizando JavaScript. Utilizaremos la función `isObjectLike` para determinar si un valor dado es un objeto o un array, y también aprenderemos a diferenciarlo de otros tipos de datos, como null o funciones. Al final de este laboratorio, tendrás una mejor comprensión de los valores similares a objetos y de cómo trabajar con ellos en tu código JavaScript.
