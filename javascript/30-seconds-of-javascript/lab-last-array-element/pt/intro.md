# Introdução

Neste laboratório, exploraremos uma função JavaScript que nos ajuda a encontrar o último elemento de um array. Aprenderemos como verificar se um array é válido e possui uma propriedade `length`, e como usar a propriedade `length` para calcular o índice do último elemento. Também aprenderemos como retornar `undefined` se o array estiver vazio ou inválido.
