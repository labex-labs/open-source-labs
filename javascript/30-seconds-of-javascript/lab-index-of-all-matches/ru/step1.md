# Все индексы совпадений

Для нахождения всех индексов `val` в массиве используйте `Array.prototype.reduce()`, чтобы пройти по элементам и сохранить индексы для совпадающих элементов. Если `val` никогда не встречается, возвращается пустой массив.

```js
const indexOfAll = (arr, val) =>
  arr.reduce((acc, el, i) => (el === val ? [...acc, i] : acc), []);
```

Пример использования:

```js
indexOfAll([1, 2, 3, 1, 2, 3], 1); // [0, 3]
indexOfAll([1, 2, 3], 4); // []
```
