# はじめに

この実験では、JavaScript でプロミスを返すデバウンス関数を作成する方法を探ります。指定された時間が経過するまで関数の呼び出しを遅らせる方法と、この間に返される複数のプロミスを処理してすべてが同じデータを返すようにする方法を学びます。この実験が終わるとき、JavaScript プロジェクトでプロミスを使ったデバウンスの実装方法をしっかりと理解しているでしょう。
