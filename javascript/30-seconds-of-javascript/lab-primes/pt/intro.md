# Introdução

Neste laboratório, vamos mergulhar no fascinante mundo da programação JavaScript. Este laboratório foi projetado para ajudá-lo a obter experiência prática na resolução de problemas de programação usando JavaScript. Você aprenderá como gerar números primos até um determinado número usando o algoritmo Crivo de Eratóstenes (Sieve of Eratosthenes).
