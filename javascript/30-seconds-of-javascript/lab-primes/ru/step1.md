# Генерация простых чисел с использованием Решета Эратосфена

Чтобы сгенерировать простые числа до заданного числа с использованием Решета Эратосфена, следуйте шагам:

1. Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.
2. Создайте массив, содержащий числа от `2` до заданного числа.
3. Используйте `Array.prototype.filter()`, чтобы отфильтровать значения, которые делятся на любое число от `2` до квадратного корня из заданного числа.
4. Верните результирующий массив, содержащий простые числа.

Вот код на JavaScript для генерации простых чисел до заданного числа:

```js
const generatePrimes = (num) => {
  let arr = Array.from({ length: num - 1 }).map((x, i) => i + 2),
    sqrt = Math.floor(Math.sqrt(num)),
    numsTillSqrt = Array.from({ length: sqrt - 1 }).map((x, i) => i + 2);
  numsTillSqrt.forEach(
    (x) => (arr = arr.filter((y) => y % x !== 0 || y === x))
  );
  return arr;
};
```

Вы можете вызвать функцию `generatePrimes()`, передав в качестве аргумента желаемое число. Например:

```js
generatePrimes(10); // [2, 3, 5, 7]
```
