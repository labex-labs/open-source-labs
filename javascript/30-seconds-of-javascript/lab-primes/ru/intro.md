# Введение

В этом практическом занятии мы углубимся в увлекательный мир программирования на JavaScript. Это практическое занятие предназначено для того, чтобы вы приобрели практический опыт в решении задач программирования с использованием JavaScript. Вы научитесь генерировать простые числа до заданного числа с использованием алгоритма «Решето Эратосфена».
