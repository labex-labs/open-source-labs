# Einführung

In diesem Lab werden wir untersuchen, wie man aus einem Objekt in JavaScript ein Array von Schlüssel-Wert-Paar-Arrays erstellt. Das Lab bietet Schritt-für-Schritt-Anleitungen dazu, wie man `Object.keys()` und `Array.prototype.map()` verwendet, um über die Schlüssel eines Objekts zu iterieren und ein Array von Schlüssel-Wert-Paaren zu erzeugen. Wir werden auch über die integrierte `Object.entries()`-Methode lernen, die eine ähnliche Funktionalität bietet.
