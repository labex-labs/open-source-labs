# Introdução

Neste laboratório, exploraremos como criar um array de arrays de pares chave-valor a partir de um objeto em JavaScript. O laboratório fornecerá orientação passo a passo sobre como usar `Object.keys()` e `Array.prototype.map()` para iterar sobre as chaves de um objeto e produzir um array de pares chave-valor. Também aprenderemos sobre o método embutido `Object.entries()` que fornece funcionalidade semelhante.
