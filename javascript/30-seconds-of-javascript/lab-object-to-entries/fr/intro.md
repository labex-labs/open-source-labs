# Introduction

Dans ce laboratoire, nous allons explorer comment créer un tableau d'objets clé-valeur à partir d'un objet en JavaScript. Le laboratoire fournira des instructions étape par étape sur la manière d'utiliser `Object.keys()` et `Array.prototype.map()` pour itérer sur les clés d'un objet et produire un tableau de paires clé-valeur. Nous allons également découvrir la méthode intégrée `Object.entries()` qui offre une fonctionnalité similaire.
