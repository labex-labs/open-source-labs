# Fonction pour Calculer la Somme des Nombres Jusqu'à N

Pour calculer la somme des nombres compris entre 1 et `n`, vous pouvez utiliser la formule `(n * (n + 1)) / 2`. Voici un exemple de code en JavaScript :

```js
const sumN = (n) => (n * (n + 1)) / 2;
```

Pour utiliser cette fonction, passez la valeur de `n` en tant qu'argument. Par exemple, `sumN(100)` renverra `5050`. Pour commencer à pratiquer la programmation, ouvrez le Terminal/SSH et tapez `node`.
