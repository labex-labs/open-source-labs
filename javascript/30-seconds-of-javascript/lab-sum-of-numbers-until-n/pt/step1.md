# Função para Calcular a Soma de Números Até N

Para calcular a soma dos números entre 1 e `n`, você pode usar a fórmula `(n * (n + 1)) / 2`. Aqui está um exemplo de código em JavaScript:

```js
const sumN = (n) => (n * (n + 1)) / 2;
```

Para usar esta função, passe o valor de `n` como um argumento. Por exemplo, `sumN(100)` retornará `5050`. Para começar a praticar a codificação, abra o Terminal/SSH e digite `node`.
