# Función para Calcular la Suma de Números Hasta N

Para calcular la suma de los números entre 1 y `n`, puedes utilizar la fórmula `(n * (n + 1)) / 2`. Aquí hay un ejemplo de código en JavaScript:

```js
const sumN = (n) => (n * (n + 1)) / 2;
```

Para utilizar esta función, pasa el valor de `n` como argumento. Por ejemplo, `sumN(100)` devolverá `5050`. Para comenzar a practicar la codificación, abre la Terminal/SSH y escribe `node`.
