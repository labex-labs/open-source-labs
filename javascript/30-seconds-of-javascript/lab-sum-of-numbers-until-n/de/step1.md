# Funktion zur Berechnung der Summe der Zahlen bis zu N

Um die Summe der Zahlen zwischen 1 und `n` zu berechnen, können Sie die Formel `(n * (n + 1)) / 2` verwenden. Hier ist ein Beispielcode in JavaScript:

```js
const sumN = (n) => (n * (n + 1)) / 2;
```

Um diese Funktion zu verwenden, übergeben Sie den Wert von `n` als Argument. Beispielsweise wird `sumN(100)` `5050` zurückgeben. Um mit der Programmierung zu beginnen, öffnen Sie das Terminal/SSH und geben Sie `node` ein.
