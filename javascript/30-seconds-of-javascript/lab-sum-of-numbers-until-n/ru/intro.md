# Введение

В этом лабораторном задании мы будем изучать концепцию суммирования всех чисел от 1 до заданного числа `n`. Мы будем использовать формулу `(n * (n + 1)) / 2` для вычисления суммы чисел. С помощью этого лабораторного задания вы лучше поймете, как решать математические задачи с использованием программирования.
