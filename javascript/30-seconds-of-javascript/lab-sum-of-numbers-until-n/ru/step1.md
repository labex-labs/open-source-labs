# Функция для вычисления суммы чисел до N

Для вычисления суммы чисел от 1 до `n` можно использовать формулу `(n * (n + 1)) / 2`. Вот пример кода на JavaScript:

```js
const sumN = (n) => (n * (n + 1)) / 2;
```

Для использования этой функции передайте значение `n` в качестве аргумента. Например, `sumN(100)` вернет `5050`. Чтобы начать практиковаться в программировании, откройте Терминал/SSH и введите `node`.
