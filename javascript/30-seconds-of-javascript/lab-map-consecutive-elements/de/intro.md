# Einführung

In diesem Lab werden wir die `mapConsecutive`-Funktion in JavaScript erkunden. Diese Funktion ermöglicht es uns, aufeinanderfolgende Elemente eines Arrays mit einer angegebenen Funktion abzubilden. Wir werden lernen, wie diese Funktion verwendet werden kann, um Arrays auf eine präzise und effiziente Weise zu manipulieren.
