# Introduction

Dans ce laboratoire, nous allons explorer comment désagréger des éléments dans un tableau de tableaux à l'aide de JavaScript. Nous utiliserons une combinaison de fonctions JavaScript intégrées telles que `reduce`, `map` et `forEach` pour extraire les tableaux individuels d'un seul tableau de tableaux. À la fin de ce laboratoire, vous aurez une compréhension solide de la manière de manipuler les tableaux en JavaScript pour obtenir la sortie souhaitée.
