# Introduction

Dans ce laboratoire, nous allons explorer le concept de fonctions partielles en JavaScript et apprendre à créer une fonction de haut niveau qui nous permet d'ajouter facilement des arguments à une fonction donnée. Par des exemples pratiques et des exercices, nous comprendrons comment les fonctions partielles fonctionnent et comment elles peuvent être utilisées pour simplifier notre code et le rendre plus modulaire.
