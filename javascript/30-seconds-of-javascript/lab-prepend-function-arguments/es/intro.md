# Introducción

En este laboratorio, exploraremos el concepto de funciones parciales en JavaScript y aprenderemos cómo crear una función de orden superior que nos permita agregar fácilmente argumentos a una función dada. A través de ejemplos prácticos y ejercicios, entenderemos cómo funcionan las funciones parciales y cómo se pueden utilizar para simplificar nuestro código y hacerlo más modular.
