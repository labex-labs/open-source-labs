# Introduction

Dans ce laboratoire, nous allons explorer le concept de vérification de type en JavaScript. Plus précisément, nous nous concentrerons sur l'opérateur `typeof` et sur la manière dont il peut être utilisé pour vérifier si une valeur donnée est une primitive chaîne de caractères. Grâce à une série d'exercices et de défis, vous acquerrez une compréhension plus approfondie de cet aspect fondamental de la programmation JavaScript.
