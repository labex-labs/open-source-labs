# Введение

В этом практическом занятии мы изучим концепцию проверки типов в JavaScript. В частности, мы сосредоточимся на операторе `typeof` и том, как его можно использовать для проверки того, является ли заданное значение примитивной строкой. С помощью серии упражнений и задач вы получите более глубокое понимание этого фундаментального аспекта программирования на JavaScript.
