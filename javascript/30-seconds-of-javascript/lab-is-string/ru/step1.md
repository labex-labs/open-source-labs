# Проверка, является ли значение строкой

Для проверки того, является ли значение строкой, используйте ключевое слово `typeof`, за которым следует значение, которое вы хотите проверить. Этот метод работает только для примитивных строк.

Вот пример кода, который проверяет, является ли заданное значение строкой:

```js
const isString = (val) => typeof val === "string";
```

Вы можете использовать функцию `isString` следующим образом:

```js
isString("10"); // true
```
