# Introdução

Neste laboratório, exploraremos o conceito de verificação de tipos (type checking) em JavaScript. Especificamente, focaremos no operador `typeof` e como ele pode ser usado para verificar se um determinado valor é uma string primitiva. Através de uma série de exercícios e desafios, você obterá uma compreensão mais profunda deste aspecto fundamental da programação JavaScript.
