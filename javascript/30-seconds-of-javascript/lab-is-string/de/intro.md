# Einführung

In diesem Lab werden wir das Konzept der Typüberprüfung in JavaScript erkunden. Insbesondere werden wir uns auf den `typeof`-Operator konzentrieren und wie er verwendet werden kann, um zu überprüfen, ob ein bestimmter Wert ein primitiver String ist. Durch eine Reihe von Übungen und Herausforderungen werden Sie ein tieferes Verständnis dieses grundlegenden Aspekts der JavaScript-Programmierung erlangen.
