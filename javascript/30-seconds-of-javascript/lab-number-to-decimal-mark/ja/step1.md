# 数値を小数記法形式に変換する方法

数値を小数記法形式に変換するには、次の手順に従います。

1. ターミナル/SSH を開き、コーディングの練習を始めるために `node` と入力します。
2. `Number.prototype.toLocaleString()` を使用して、数値を小数記法形式に変換します。
3. このプロセスには次のコードを使用できます。

```js
const toDecimalMark = (num) => num.toLocaleString("en-US");
```

この関数の使用方法の例を次に示します。

```js
toDecimalMark(12305030388.9087); // '12,305,030,388.909'
```

これにより、数値 `12305030388.9087` が小数記法形式の文字列 `'12,305,030,388.909'` に変換されます。
