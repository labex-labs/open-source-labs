# Introduction

Dans ce laboratoire, nous allons approfondir la fonction `nthElement` en JavaScript qui renvoie l'élément d'indice `n` d'un tableau. Vous allez apprendre à utiliser `Array.prototype.slice()` pour extraire un élément d'un tableau et à gérer correctement les indices hors limites. À la fin de ce laboratoire, vous aurez une compréhension solide de la fonction `nthElement` et de son utilité dans vos projets JavaScript.
