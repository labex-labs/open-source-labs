# 配列の n 番目の要素を見つける

配列の n 番目の要素を見つけるには、次の手順に従います。

1. ターミナル/SSH を開き、コーディングを練習するために `node` と入力します。
2. `Array.prototype.slice()` を使って、n 番目の要素を含む新しい配列を作成します。
3. インデックスが範囲外の場合、`undefined` を返します。
4. 配列の最初の要素を取得するには、2 番目の引数 `n` を省略します。

これを実装するサンプルコードは次のとおりです。

```js
const nthElement = (arr, n = 0) =>
  (n === -1 ? arr.slice(n) : arr.slice(n, n + 1))[0];
```

この関数を次の例でテストできます。

```js
nthElement(["a", "b", "c"], 1); // 出力：'b'
nthElement(["a", "b", "b"], -3); // 出力：'a'
```

これらの手順に従えば、JavaScript を使って配列の n 番目の要素を簡単に見つけることができます。
