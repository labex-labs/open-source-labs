# Introducción

En este laboratorio, profundizaremos en la función `nthElement` de JavaScript que devuelve el enésimo elemento de un array. Aprenderás cómo utilizar `Array.prototype.slice()` para extraer un elemento de un array y cómo manejar adecuadamente los índices fuera de los límites. Al final de este laboratorio, tendrás una comprensión sólida de la función `nthElement` y de cómo puede ser útil en tus proyectos de JavaScript.
