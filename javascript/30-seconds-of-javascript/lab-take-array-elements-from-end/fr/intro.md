# Introduction

Dans ce laboratoire, nous allons explorer comment supprimer des éléments de la fin d'un tableau en JavaScript. Nous utiliserons la méthode `Array.prototype.slice()` pour créer un nouveau tableau avec `n` éléments pris à partir de la fin du tableau original. À la fin de ce laboratoire, vous aurez une meilleure compréhension de la manière de manipuler les tableaux en JavaScript.
