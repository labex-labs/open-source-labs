# Введение

В этом практическом занятии мы изучим, как удалить элементы из конца массива в JavaScript. Мы будем использовать метод `Array.prototype.slice()`, чтобы создать новый массив, содержащий `n` элементов, взятых из конца исходного массива. В конце этого практического занятия вы будете лучше понимать, как манипулировать массивами в JavaScript.
