# Как удалить элементы из конца массива в JavaScript

Чтобы удалить элементы из конца массива в JavaScript, вы можете использовать метод `Array.prototype.slice()`. Вот, как это можно сделать:

```js
const takeRight = (arr, n = 1) => arr.slice(arr.length - n, arr.length);
```

Эта функция создает новый массив с последними `n` элементами исходного массива. Вот, как вы можете использовать ее:

```js
takeRight([1, 2, 3], 2); // [ 2, 3 ]
takeRight([1, 2, 3]); // [3]
```
