# Introducción

En este laboratorio, profundizaremos en el concepto de comprobar si un valor especificado es `null` o `undefined`. Exploraremos la función `isNil()`, que utiliza el operador de igualdad estricta para realizar la comprobación, y veremos cómo se puede utilizar en escenarios prácticos para mejorar la confiabilidad y robustez de nuestro código.
