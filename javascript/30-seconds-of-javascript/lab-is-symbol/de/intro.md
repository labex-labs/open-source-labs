# Einführung

In diesem Lab werden wir das Konzept von Symbolen in JavaScript erkunden. Symbole sind ein neuer primitiver Datentyp, der in ECMAScript 6 eingeführt wurde und einzigartige Identifikatoren repräsentiert. Wir werden lernen, wie Symbole erstellt werden, wie sie als Eigenschaftsschlüssel in Objekten verwendet werden und wie überprüft werden kann, ob ein gegebener Wert ein Symbol ist.
