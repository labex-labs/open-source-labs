# Введение

В этом лабе мы изучим концепцию символов в JavaScript. Символы - это новый примитивный тип, введенный в ECMAScript 6, который представляет уникальные идентификаторы. Мы узнаем, как создавать символы, использовать их в качестве ключей свойств в объектах и проверять, является ли данное значение символом.
