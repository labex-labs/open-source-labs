# JavaScript 에서 두 배열의 합집합 구하는 방법

JavaScript 에서 두 배열의 합집합을 구하려면 다음 단계를 따르세요.

1. 터미널/SSH 를 열고 `node`를 입력하여 코딩 연습을 시작합니다.

2. 두 배열의 합집합은 두 배열 중 하나 이상에 존재하는 모든 요소를 반환합니다.

3. 두 배열의 합집합을 구하려면, `a`와 `b`의 모든 값을 사용하여 `Set`을 생성하고, `Array.from()` 메서드를 사용하여 배열로 변환합니다.

다음은 이를 구현하는 예시입니다.

```js
const union = (a, b) => Array.from(new Set([...a, ...b]));

console.log(union([1, 2, 3], [4, 3, 2])); // Output: [1, 2, 3, 4]
```

위 예시에서 `union()` 함수는 두 개의 배열 `[1, 2, 3]`과 `[4, 3, 2]`를 인수로 받아 두 배열의 합집합을 배열 `[1, 2, 3, 4]`로 반환합니다.
