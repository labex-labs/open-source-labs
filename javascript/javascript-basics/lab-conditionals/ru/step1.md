# Условные операторы

> Откройте Терминал/SSH и введите `node`, чтобы начать практиковаться в написании кода.

Условные операторы - это структуры кода, используемые для проверки, возвращает ли выражение значение `true` или нет. Очень распространенной формой условных операторов является оператор `if...else`. Например:

```js
let iceCream = "chocolate";
if (iceCream === "chocolate") {
  console.log("Yay, I love chocolate ice cream!");
} else {
  console.log("Awwww, but chocolate is my favorite…");
}
```

Выражение внутри `if ()` - это тест. В этом случае используется оператор строгого равенства (описанный выше) для сравнения переменной `iceCream` со строкой `chocolate`, чтобы определить, равны ли они. Если эта сравнение возвращает `true`, выполняется первый блок кода. Если сравнение не дает `true`, выполняется второй блок кода - после оператора `else`.
