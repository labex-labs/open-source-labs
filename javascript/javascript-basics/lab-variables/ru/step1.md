# Переменные

> Откройте Терминал/SSH и введите `node`, чтобы начать практиковать программирование.

Переменные - это контейнеры, которые хранят значения. Вы начинаете с объявления переменной с помощью ключевого слова `let`, за которым следует имя, которое вы даете переменной:

```js
let myVariable;
```

Точка с запятой в конце строки указывает, где заканчивается оператор. Она требуется только тогда, когда вам нужно разделить операторы в одной строке. Однако некоторые люди считают, что хорошей практикой является использование точек с запятой в конце каждого оператора. Существуют и другие правила, когда следует и когда не следует использовать точки с запятой.

Вы можете назвать переменную практически любым именем, но есть некоторые ограничения. Если вы не уверены, вы можете [проверить имя переменной](https://mothereff.in/js-variables), чтобы узнать, является ли оно допустимым.

JavaScript чувствителен к регистру. Это означает, что `myVariable` не то же самое, что `myvariable`. Если у вас есть проблемы в коде, проверьте регистр!

После объявления переменной вы можете присвоить ей значение:

```js
myVariable = "Bob";
```

Также вы можете выполнить оба этих действия в одной строке:

```js
let myVariable = "Bob";
```

Вы получаете значение, вызывая имя переменной:

```js
myVariable;
```

После присвоения значения переменной вы можете изменить его позже в коде:

```js
let myVariable = "Bob";
myVariable = "Steve";
```

Обратите внимание, что переменные могут хранить значения различных [типов данных](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures):

| Переменная                                                           | Пояснение                                                                                                                                            | Пример                                                                                                                       |
| -------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| [String](https://developer.mozilla.org/en-US/docs/Glossary/String)   | Это последовательность текста, известная как строка. Чтобы обозначить, что значение является строкой, заключите его в одинарные или двойные кавычки. | `let myVariable = 'Bob';` или `let myVariable = "Bob";`                                                                      |
| [Number](https://developer.mozilla.org/en-US/docs/Glossary/Number)   | Это число. Числа не заключаются в кавычки.                                                                                                           | `let myVariable = 10;`                                                                                                       |
| [Boolean](https://developer.mozilla.org/en-US/docs/Glossary/Boolean) | Это значение "Истина/Ложь". Слова `true` и `false` - это специальные ключевые слова, которые не требуют кавычек.                                     | `let myVariable = true;`                                                                                                     |
| [Array](https://developer.mozilla.org/en-US/docs/Glossary/Array)     | Это структура, которая позволяет вам хранить несколько значений в одной ссылке.                                                                      | `let myVariable = [1,'Bob','Steve',10];` Обращайтесь к каждому элементу массива так: `myVariable[0]`, `myVariable[1]` и т.д. |
| [Object](https://developer.mozilla.org/en-US/docs/Glossary/Object)   | Это может быть что угодно. Весь JavaScript представляет собой объекты, которые могут быть сохранены в переменных. Имейте это в виду при изучении.    | `let myVariable = document.querySelector('h1');` Также все вышеприведенные примеры.                                          |

Так зачем нам нужны переменные? Переменные необходимы для выполнения любых интересных действий в программировании. Если значения не могли бы изменяться, то вы не могли бы выполнять ничего динамического, например, персонализировать приветственное сообщение или изменить отображаемое изображение в галерее.
