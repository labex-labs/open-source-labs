# 소개

jQuery 문서에 오신 것을 환영합니다! 이 랩에서는 jQuery 에 대한 소개를 제공합니다.

jQuery 를 시작하는 것은 JavaScript, HTML, CSS 및 전반적인 프로그래밍 개념에 대한 경험에 따라 쉬울 수도 있고 어려울 수도 있습니다.

알아두어야 할 중요한 점은 jQuery 가 단순한 JavaScript 라이브러리라는 것입니다. jQuery 의 모든 기능은 JavaScript 를 통해 액세스되므로, 코드를 이해하고, 구조화하고, 디버깅하려면 JavaScript 에 대한 강력한 이해가 필수적입니다. jQuery 를 정기적으로 사용하면 시간이 지남에 따라 JavaScript 숙련도가 향상될 수 있지만, JavaScript 의 내장 구조와 구문에 대한 실질적인 지식 없이는 jQuery 를 작성하는 데 어려움을 겪을 수 있습니다. 따라서 JavaScript 를 처음 접하는 경우 Mozilla Developer Network (MDN) 의 JavaScript 기본 튜토리얼을 확인하는 것이 좋습니다.
