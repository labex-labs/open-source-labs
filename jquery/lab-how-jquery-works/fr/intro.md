# Introduction

Bienvenue dans la documentation jQuery! Ce laboratoire vous présentera jQuery.

Commencer avec jQuery peut être facile ou difficile, selon votre expérience en JavaScript, HTML, CSS et en programmation en général.

Une chose importante à savoir est que jQuery est simplement une bibliothèque JavaScript. Toutes les fonctionnalités de jQuery sont accessibles via JavaScript, il est donc essentiel de maîtriser fortement JavaScript pour comprendre, structurer et déboguer votre code. Bien que travailler régulièrement avec jQuery puisse, au fil du temps, améliorer votre compétence en JavaScript, il peut être difficile de commencer à écrire du jQuery sans une connaissance pratique des constructions et de la syntaxe intégrées de JavaScript. Par conséquent, si vous êtes nouveau en JavaScript, nous vous recommandons de consulter le tutoriel de base JavaScript sur le Réseau de Développeurs Mozilla (MDN).
