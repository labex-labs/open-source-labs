# 简介

欢迎来到 jQuery 文档！本实验将向你介绍 jQuery 事件。

jQuery 提供了简单的方法来为选中的元素附加事件处理程序。当事件发生时，所提供的函数将被执行。在函数内部，this 指向触发事件的 DOM 元素。

事件处理函数可以接收一个事件对象。该对象可用于确定事件的性质，并阻止事件的默认行为。
